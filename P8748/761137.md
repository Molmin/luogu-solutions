一道模拟题，挺适合初学者的。

对于一个毫秒数 $n$，我们先忽略大于前面的年月日，让 $n$ 只表示从今天 00:00:00 到现在的毫秒数。

然后将 $n$ 除以 $1000$，让 $n$ 从毫秒数变成秒数。

最后输出，小时数是 ```n/3600```，分钟数是 ```n%3600/60```，秒数是 ```n%60```。不足两位的补足两位。

参考代码：

```cpp
#include<iostream>
using namespace std;
long long n;//开long long
int main(){
	cin>>n;
	n%=60*60*24*1000;//变成今天的毫秒数
	n/=1000;//变成今天的秒数
	if(n/3600<=9)cout<<"0";//判断补零，下面同理
   cout<<n/3600<<":";//输出，下面同理
   if(n%3600/60<=9)cout<<"0";
   cout<<n%3600/60<<":";
   if(n%60<=9)cout<<"0";
   cout<<n%60;
	return 0;
}
```
