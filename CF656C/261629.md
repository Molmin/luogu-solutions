### 这题就是简单的字符串模拟，差不多红题难度。
题目有点绕，大致意思其实就是输入一个字符串a并循环判断其中字符的大小写。  

- 如果是大写，则计算出这个字符在字母表中的序列（即ai - 'A' + 1)  

- 如果是小写，则同样计算出这个字符在字母表中的序列（即ai - 'a' + 1)  

最后将第一个计算出的数减去第二个计算出的数，得出答案。  
## AC代码
------------
```cpp
#include <bits/stdc++.h>
using namespace std;
string a;
int i,j,k,ans = 0,x = 0,y = 0; 
int main(){
	cin>>a;//输入字符串(不能用gets()) 
	for(i = 0;i < a.size();i++){//循环判断字符 
		x = 0;
		y = 0;
		if(a[i] > '@' && a[i] < '['){//判断其中字符的大小写
			x = a[i] - 'A' + 1;//计算出这个字符在字母表中的序列 
		}
		if(a[i] > '`' && a[i] < '{'){
			y = a[i] - 'a' + 1;
		}
		ans += x - y;//计算出答案 
	}
	cout<<ans;//输出 
	return 0;
}
 
```
