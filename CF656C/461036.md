## 题意：
给出一个字符串，遍历每个字符，如果是大写字母，累加器 $s$ 加上这个字母在字母表的表序，如果是小写字母，累加器 $s$ 减去这个字母在字母表的表序。
### 样例2分析：
样例输入：
```cpp
APRIL.1st
```
样例输出：
```cpp
17
```
开始遍历， $A,P,R,I,L$ 都是大写字母，且表序分别为$1,16,18,9,12$ ，那么它们的和为 $56$ ， $s,t$ 是小写字母，表序分别为 $19,20$ ，和为 $39$ ，其余的字符既不是小写字母，也不是大写字母，所以不管它。因为 $56-39=17$ ，所以输出 $17$ 。
## 思路：
因为每个字符都有其对应的值，遍历时判断一下就可以了。本蒟蒻更喜欢边读入边处理，接下来直接**模拟**！
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	char a;
	int s=0;
	while(cin>>a)//边读入边处理
	{
		if(a>='A'&&a<='Z')//大写字母判断
			s+=(int)(a-'A'+1);
		if(a>='a'&&a<='z')//小写字母判断
			s-=(int)(a-'a'+1);
	}
	cout<<s;//输出
	return 0;
}
```
