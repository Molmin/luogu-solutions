给大家推荐一种硬核走迷宫方法:

1.打开画图

2.将迷宫的入口和出口封住

3.在迷宫的入口处染色

例如：

![](https://cdn.luogu.com.cn/upload/image_hosting/98rvk5ml.png)

4.如果迷宫出口处被染上色，说明这个迷宫可以走通（例如$A$,$C$），反之则不行（如$B$,$D$）

用如上方法可以得出答案，输出即可

程序如下：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
    puts("ACEFGHNPQSYZ");
    return 0;
}
```
