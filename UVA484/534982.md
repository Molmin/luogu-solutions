 - ## 题意分析
   输入一串数字,按照输入顺序依次输出数字与它的个数(好像也没有什么要分析的)。
 - ## 那么思路
   一直输入 $n$ ,如果 $n$ 第一次出现,那么把它储存到 $b$ 当中( $b$ 是按照输入顺序储存数字的),再把 $n$ 储存到 $a$ 这个桶中,如果它不止一次出现,那么直接 $a[n]++$ 即可。
   ```cpp
   #include<bits/stdc++.h>
   using namespace std;
   int a[100000000],b[100000000],sum=1; //a是桶,b是储存出现的数字 
   int main(){
    	int n;
    	while(cin>>n){   //一直输入n 
    		if(a[n]==0){ //当n第一次输入,那么把它储存到b当中, a[n]++ 
	    		b[sum++]=n;
    			a[n]++;
	    	}
		    else{
	     		a[n]++;  //如果不是第一次输入,直接加! 
		    }
    	}
	    for(int i=1;i<=sum-1;i++) cout<<b[i]<<" "<<a[b[i]]<<endl; //最后按照顺序输出即可 
    	return 0;
    }
    ```
