### 这道题有3种情况
一定要考虑全面！

------------
**第一种情况：直接睡饱**     
因为是要睡$a$分钟，定了一个$b$分钟的闹钟，也就是睡$b$分钟。      
所以，假如$a≤b$，那么，不用再定闹钟，直接就可以睡饱，也就是睡$b$分钟。    
**代码实现：**
```cpp
if(a<=b) 
{
	cout<<b<<endl;
	continue;
}
```


------------
**第二种情况：永不起床**      
排除了第一种情况后，就要定第二次闹钟，而定第二次闹钟后要$d$分钟后才睡得着。   
第二次闹钟是定$c$分钟。那么就有一种情况，就是每次小$P$还没睡着，闹钟就响了。     
这就是永不起床的情况。

**代码实现：**
```cpp
if(c<=d) 
{
	cout<<"-1"<<endl;
	continue;
}
```


------------
**第三种情况：定二次闹钟**    
既然要定第二次闹钟那就要定($a$-$b$)/($c$-$d$)轮    
每一轮耗时$c$。
我们就只用$a*c+b$就可以了   
这就是定二次闹钟的情况

**代码实现：**
```cpp
a=a-b;
a=ceil(a*1.0/(c-d));
cout<<a*c+b<<endl;
```

### 补习：关于ceil函数
格式：ceil（number）;   
在这里，是将number向上取整，**注意！不是四舍五入！**    
也就是：   
ceil(4.1)=5     
ceil(4.6)=5    
ceil(4)=4     

**懂了吗？**  

P.S. 想要四舍五入请使用round()函数
