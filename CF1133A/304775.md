# 1、题目大意

给你两个时刻，让你输出这两个事件的中间时刻。

输入时刻形式为`ab` `:` `cd`  

注意事项：

- 输入可能会有前导零（不过无关紧要）。

- 输出也要注意关于前导零的处理。

# 2、主要思想

输入这两个时刻后，利用时间关系全转化成分钟形式，然后再利用公式求出中间时刻的分钟形式，最后转成标准格式。

例：

输入：
```
10:00
11:00
```
输出：
```
10:30
```

先将`10:00`转成$600$分钟，将`11:00`转为$660$分钟，求出中间时刻为$630$分钟，转回`10:30`。

# 3、代码实现

详见代码注释。

```cpp
#include <cstdio>
#include <iostream>
#include <cstring>
using namespace std;
int main()
{
	int h1,h2,m1,m2;//为起始小时分钟和终止小时分钟 
	char nothing;//用于处理输入的':' 
	cin>>h1>>nothing>>m1>>h2>>nothing>>m2;
	int t1=h1*60+m1;
	int t2=h2*60+m2;
	//将两个时刻转为分钟形式 
	int midt=(t1+t2)/2;//求出中间时间 
	if(midt/60<10)//若不到十小时，则要输出前导零 
		cout<<0;
	cout<<midt/60<<':';
	if(midt%60<=9)//同上 
		cout<<0;
	cout<<midt%60;
	return 0;
}
```
