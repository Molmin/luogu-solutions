~~大水题~~

本蒟蒻的思路是：把两个时间都转换成分钟，取平均数再换回去

输入不需要字符串，冒号用一个字符变量$x$解决就可以了

$Code:$

```
#include<bits/stdc++.h>
using namespace std;
int main (){
	int a1,a2,b1,b2;//a1,b1是小时，a2,b2是分钟
	char x;//这个没什么用
	cin>>a1>>x>>a2>>b1>>x>>b2;//不用读入字符串
   //c++会自动区分数字和普通字符
	int t1,t2;//这两个数字代表这一天过去的分钟数
	t1=a1*60+a2;//换算一下
	t2=b1*60+b2;
	int t=(t1+t2)/2;//取平均
	int h=t/60,m=t%60;//算出平均时间的小时和分钟数
	if (h<10)//注意前导0，但如果是00的话不用再额外特判
   //前导0：例如h=1，需要输出01
	cout<<0;
	cout<<h<<":";//正常输出即可
	if (m<10)//同上
	cout<<0;
	cout<<m<<endl;
	return 0;
}
```
