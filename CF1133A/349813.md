### **多么精美的题目！**  


------------
### **读题**  
 输入两个时间，需要你输出这两个时间的中间时刻

注意：题目保证这两个时间在**同一天**，且第一个时间会早于第二个时间，且为24小时计数法。  


------------
### **思路：**    
  小学生都学过，$a$,$b$的中点可以表示为$\frac{a+b}{2}$（具体不证明了）  
  先算出两个时刻加起来
经过了多少分钟（用$n$表示），再算出分钟的中位数$\frac{n}{2}$,用第一个时间加上$\frac{n}{2}$，再转换为24小时计数法的正确格式输出即可。  
  

------------
### **注意：**  
 1. 不能输出如"8:9"这样的时间，必须输出"08:09"这样的时间  
 2. 注意中间的冒号是":"，而不是“：”.  


------------
### **上代码：**  
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int t1=0,t2=0,n=0;
	string s1,s2;
	cin>>s1>>s2;
	t1=((s1[0]-'0')*10+(s1[1]-'0'))*60+(s1[3]-'0')*10+(s1[4]-'0');
	t2=((s2[0]-'0')*10+(s2[1]-'0'))*60+(s2[3]-'0')*10+(s2[4]-'0');
	n=(t1+t2)/2;
	if(n/60<=9) 
		cout<<"0"<<n/60<<":";
	else 
		cout<<n/60<<":";
	if(n%60<=9) 
		cout<<"0"<<n%60;
	else 
		cout<<n%60;
	return 0;
}
```
