可以先算出比赛持续时间分钟数/2$m=((h2 - h1) * 60 + m2 - m1) / 2$，中间时刻就是`起始时间小时数+m包含的小时数+m包含的分钟数与起始时间的分钟数相加进位的小时数:m包含的分钟数与起始时间的分钟数相加后包含的分钟数`，即
$h1 + m / 60 + (m \% 60 + m1) / 60 : (m \% 60 + m1) \% 60$，最后直接输出即可。注意如果输出小时数或分钟数不足两位需在前面补$0$，可用$printf$的$\%02d$实现。  
代码如下：  
```cpp
#include <cstdio>

int main()
{
    int h1, m1, h2, m2;
    scanf("%d:%d%d:%d", &h1, &m1, &h2, &m2);
    int m = ((h2 - h1) * 60 + m2 - m1) / 2; // 比赛持续时间/2
    printf("%02d:%02d\n", h1 + m / 60 + (m % 60 + m1) / 60, (m % 60 + m1) % 60);
    return 0;
}
```
