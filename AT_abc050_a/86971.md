首先在讨论区里，本蒟蒻知道了题目大意：  
输入一个表达式，求值（符号只有+或-）那么，我们就可以枚举所有的可能。  
下面献上switch代码：  
```
#include <stdio.h>
int main()
{
	int a,b;//定义没啥好说的。
	char c;
	scanf("%d %c %d",&a,&c,&b);//读入表达式。
	switch(c)
	{
		case '+':printf("%d\n",a+b);break;//如果是“+”的话，输出a+b，注意后面要加“break”
		case '-':printf("%d\n",a-b);break;//也一样
	}
}//完美的手场23333333
```
