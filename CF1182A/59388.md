对于一个$3 \times n$矩阵，有两列（$3 \times 2$）可以这样拼：
```
++
+.
..
```
或这样拼：
```
++
.+
..
```
共两种。
但是如果某两行（$3 \times 2$）被覆盖，则第三行无法覆盖：
```
++.
+..
***
```
（*为未覆盖）

所以如果$n$是$2$的倍数则可以两行两行地覆盖，每两行有$2$种方案，有$2^{\frac{n}{2}}$种方案（乘法原理）。否则无法满足，输出0.

Code（给了和没给大家一样会做吧）：
```cpp
#include<cstdio>
int N;
int main()
{
	scanf(" %d",&N);
	if(N&1)printf("0\n");//奇数，无解
	else printf("%d\n",1<<(N>>1));//偶数就是这样
	return 0;
}
```
