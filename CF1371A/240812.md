首先，由于木棍的长度各不相同，必然需要拼接。

因为所有木棍的和是一定的，所以拼接后长度要尽可能平均，（且最多两两拼接）。

可以构造出最优的方法：

对于 $n$ 是偶数的情况：

将第 $1$ 根和第 $n$ 根拼接，第 $2$ 根和第 $n-1$ 根拼接，$\dots$

一共可以组成 $\frac n 2$ 根长度都为 $n+1$ 的木棍。

对于 $n$ 是奇数的情况：

将第 $1$ 根和第 $n-1$ 根拼接，第 $2$ 根和第 $n-2$ 根拼接，$\dots$ 剩下 第 $n$ 根不拼接（孤独终老）。

一共可以组成 $\frac {n-1} 2+1$ 根长度都为 $n$ 的木棍。

利用~~c艹~~c++除法下取整的特殊性，答案可以直接写成 $\frac {n+1} 2$ 。（想一想，为什么）

喜闻乐见的代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int t;
int main()
{
	scanf("%d",&t);
	while(t--)
	{
		int n;
		scanf("%d",&n);
		cout<<(n+1)/2<<endl;//简洁明了
	}	
} 
```
