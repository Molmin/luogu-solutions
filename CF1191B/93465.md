$\large\textbf{2.做法}$

前面其实可以当做来水字数的，下面进入正题。

首先读进来的三块麻将明显可以用字符串$\rm Hash$的技巧转化成数字。这个处理过程在我的`read`函数里面。  
你可以发现转换后的数字会**将连号表现为相邻数字的形式**。这为我们的暴力判断提供了便利。

设这三个数为 $a,b,c$ 且 $a\leq b\leq c$ ，现在我们需要大力`if`一下：

 1. $a=b=c$ 时输出 $0$ （全部相同）；
 2. $a+1=b$ 且 $b+1=c$ 时输出 $0$ （构成连号）；
 3. $a+1=b$ 时输出 $1$ （把 $c$ 换成 $b+1$ ）；
 4. $a+2=b$ 时输出 $1$ （把 $c$ 换成 $a+1$ ）；
 5. $a+1=c$ 时输出 $1$ （理由与3类似）；
 6. $a+2=c$ 时输出 $1$ （理由与4类似）；
 7. $b+1=c$ 时输出 $1$ （理由与3类似）；
 8. $b+2=c$ 时输出 $1$ （理由与4类似）；
 9. $a=b$ 或 $b=c$ 或 $a=c$ 时输出 $1$ （可构成连号）；  
 (10.)若以上条件均不满足，输出 $2$ 。
 
**注意，若在依次判断的过程中发现真命题则输出并立即停止，不再判断后面的命题。**  

我的代码中采取的判断顺序是 $123456789$ ，其实可以改成 $1293478$ ，请读者自行思考原因

$\large\textbf{3.代码}$

```cpp
#include<cstdio>
 
inline void swap(int &a,int &b) { a^=b; b^=a; a^=b; }
//交换两个数的值

inline int read() { //读入+Hash
    char ch=getchar(); int x=(ch^48);
    ch=getchar(); x=x+(ch-'a')*10;
    getchar(); return x;
}
int main() {
    int a=read(),b=read(),c=read();
    if (a==b&&b==c) return puts("0"),0; //1
    if (a>b) swap(a,b);
    if (b>c) swap(b,c);
    if (a>b) swap(a,b);
    //以上三行本质上是冒泡排序
    if (a+1==b&&b+1==c) return puts("0"),0; //2
    if (a+1==b||a+2==b) return puts("1"),0; //3,4 
    if (a+1==c||a+2==c) return puts("1"),0; //5,6
    if (b+1==c||b+2==c) return puts("1"),0; //7,8
    if (a==b||b==c||a==c) return puts("1"),0; //9
    return puts("2"),0; //10
}
```
