## 题目描述
$Leo$开发了一种新的编程语言$c+=$。在$c+=$=中，只能通过“+=”操作来更改整型变量，该操作将右侧值添加到左侧变量。例如，当$a=2$时执行“a += b”，$b=3$将$a$的值更改为$5$ ($b$的值不变)。

在原型程序中，$Leo$有两个整型变量$a$和$b$，初始化为一些正值。他可以执行任意数量的操作“a += b”或“b += a”。$Leo$想测试如何处理大整数，所以他想让$a$或$b$的值严格大于给定的$n$值。他要做的最小的手术数是多少?
## 输入格式
第一行包含单个整数$T(1≤T≤100)$--测试用例的数量。

下面的每一行$T$描述一个单独的测试用例，并且包含三个整数$a,b,n(1 \leq a,b \leq n \leq 10^9)$−−$a$和$b$的初始值，以及其中一个变量必须分别超过的值。
## 输出格式
对于每个测试用例，打印一个整数$——$所需操作的最小数目。用换行符分隔答案。
## 说明/提示
在第一种情况下，我们不能让一个变量在一次操作中超过$3$。在两个操作中实现这一点的一种方法是执行两次“b += a”。

理解完题意，我们就可以知道：

* 循环输入
* 每一次计数要重置
* 判断$a$和$b$哪个大，就用哪个去作计算的加数

了解完这些后我们就可以做题啦！

```
#include<bits/stdc++.h>//火车头
using namespace std;
int main(){
   ios::sync_with_stdio(false);//加速黑科技
	cin.tie(0);
	cout.tie(0);
	int a,b,m,n,sum;
	cin>>n;
	for(int i=0;i<n;i++){//循环输入
		cin>>a>>b>>m;
		sum=0;//每一次重置计数
        while(a<=m&&b<=m){//判断是否可以输出
            if(a>b){//如果a大
              b+=a;//用a作加数
            }
            else{//如果b大
            	a+=b;//用b作加数
            }
            sum++;//每一次加一
        }cout<<sum<<endl;//输出
	}
    return 0;
}
```
