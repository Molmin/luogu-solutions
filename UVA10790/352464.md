### 题目翻译

给定两条直线，输入中每行两个数，第一个数代表在上面的直线上的点数，第二个数代表下面直线上的点数；
输出连接的所有线段中一共有多少交点。
**本题多测！**

---
### 具体思路

思路其实就很简单，上面随便挑两个点，下面随便挑两个点，
计算 $C_n^2\times C_m^2$ 就行了。

具体化简后就得出 $\dfrac{mn(m-1)(n-1)}{4}$ ~~上过小学应该都会~~

---
### 代码完成

其实这个读入后直接输出就能完成，但是代码会很乱。所以我们就用宏语句定义一个运算，直接使用就行。

```cpp
#define s(a,b) a*b*(a-1)*(b-1)/4
```
后面套用`s`就行了。

但是要记住输出格式，`case i` 要从 $1$ 开始。

最后完整代码：

```cpp
#include<bits/stdc++.h>
#define s(a,b) a*b*(a-1)*(b-1)/4
using namespace std;
int main()
{
	long long a,b,i=1;
	while(cin>>a>>b&&a!=0&&b!=0)
	{
		cout<<"Case "<<i<<": "<<s(a,b)<<endl;
		i++;
	}
}
```
