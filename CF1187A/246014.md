## 题解 CF1187A 【Stickers and Toys】

### 题意

+ 有 $n$ 个蛋，里面共有 $s$ 个棍子和 $t$ 个玩具；
+ 每个蛋里有三种可能：一个棍子、一个玩具或棍子玩具各一个；
+ 求最少买几个蛋时，能保证至少有一个棍子和一个玩具。

### 做法

考虑先拿只有棍子的蛋。

因为一个蛋里玩具个数只可能是 $0$ 或 $1$，且若个数为 $0$ 一定有一个棍子，则可得只有棍子的蛋有 $n-t$ 个。

先拿了这 $n-t$ 个蛋，此时再拿一个，就能保证至少有一个棍子和一个玩具，也就是一共拿 $n-t+1$ 个蛋。

同理，先拿 $n-s$ 个只有玩具的蛋，再拿一个蛋，就能保证，共拿 $n-s+1$ 个蛋。

因为要**保证**，其中较小的次数显然无法保证，所以最后答案是 $\max(n-t+1,n-s+1)$。

### 代码

```cpp
#include <cstdio>
#include <algorithm>

using namespace std;

int main() {
	int t;
	scanf("%d", &t);

	while (t--) {
		int n, a, b;
		scanf("%d%d%d", &n, &a, &b);

		printf("%d\n", n - min(a, b) + 1);
	}
}
```
