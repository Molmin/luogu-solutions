这一题比较水，但也需要动点脑筋。
### 思路
想要两个数相加为 $5$ 的倍数，那它们除 $5$ 的**余数**相加必须等于 $5$。例如，一个**余数**是 $1$ 的数加一个**余数**是 $4$ 等于 $5$,那这两个数相加就等于 $5$。题目还说 ```a+b, b+a视为不同的数对```~~，这样还可以让我们不用考虑太多~~，但如果余数为 $0$ 的话，就不用乘 $2$ 了。
### 代码
```
#include<bits/stdc++.h>
using namespace std;
long long n,m;
long long a[5],b[5];
int main(){
	cin>>n>>m;
	for(int i=0;i<5;i++)
	{
		a[i]=n/5; 
		b[i]=m/5;
		if(n%5>=i&&i!=0)a[i]++;
		if(m%5>=i&&i!=0)b[i]++;
	} //算出余数
	cout<<a[0]*b[0]+a[1]*b[4]+a[2]*b[3]+a[3]*b[2]+a[4]*b[1]; //乘法原理算出结果
	
	
	return 0; //完结撒花
}
```
