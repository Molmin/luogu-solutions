## 题意简述
小 S 从左上角走到右下角，小 M 从左下角走到右上角。小 M 走过的每个格子之间在小 S 走过去的时候都能相互瞬间到达，求两个人最少一共要走多少步。

## 解题思路
首先观察样例，注意到样例的第四个 testcase 很有特点，答案正好是 $n + m - 2 + n$ 或者 $n + m - 2 + m$。那么我们尝试证明这个式子。

注意到小 M 在走的过程中一定只会向上或者向右走，且不会尝试穿越自己留下的传送门，因为传送门能够到达的地方对他接近右上角没有帮助，因此他所需要的移动次数是固定的 $n + m - 2$。

对于小 S，小 M 在走到右上角的过程中显然在每一行和每一列都至少留下了一个传送门。显然，小 S 想要最快走到右下角，他应该选择通过传送门穿过 $\max(n, m)$ 的距离，且最多只能经过这段距离，如果想要同时在另一个方向上有节省的距离，小 M 在放传送门的时候一定向左或者向下走了，这时候小 M 需要的能量就更多了，不是更优解

因此需要小 S 自己走的能量为 $(\min(n, m) - 1) + 1$ 点（传送门需要 $1$ 点能量）

因此总共最少需要 $n + m - 2 + \min(n, m)$ 点能量。

需要特判一下 $n$ 和 $m$ 都是 $1$ 的情况，这时候不需要能量，但是用上面的公式会算出需要传送门的那一点能量

## AC Code
```cpp
#include <bits/stdc++.h>
using namespace std;


int main(){
	int t, n, m;
	scanf("%d", &t);
	while (t--){
		scanf("%d%d", &n, &m);
		if (n == 1 && m == 1) printf("0\n");
		else printf("%d\n", n + m - 2 + min(n, m));
	}
	return 0;
}
```
