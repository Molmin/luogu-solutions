### 题意
给了一块 n$\times$m 的地，要让第一个第二个人分别要从左上角到右下角，从左下角到右上角。第二个人走过的地方都会有传送门,第一个人就可以使用第二个人留下的传送门，让它们一共走的路程最短

### 我的思路
要让第一个人走过尽量多的第二个人留下的传送门，这样就可以节约两人总共走过的路程，也就是题目中的能量 让第一个人只往右或者往下走，让第二个人只能往左或往上走，这样可以避免走重复路径。
![](https://cdn.luogu.com.cn/upload/image_hosting/yb0pd9yi.png)

上面这个是一个图，会更直观。 红色是第一个人，蓝色是第二个人。 这里蓝色的部分就是有传送门的部分而第一个人沿着红色路线走，走到这条水平的蓝色路线时，就可以从开始传送到结束

### 注意点
因为传送也睡会消耗掉 1 能量，所以直线就是最优解，如果有拐弯，就一定会是能量增加

### 这里是代码（这样子代码也就很容易啦）
~~~
#include <bits/stdc++.h>
using namespace std;
int t;
int main()
{
    cin>>t;
        int n,m;
    while(t--)
   {
        cin>>n>>m;
        if (n==1&&m == 1) 
            printf("0\n");
        else 
            printf("%d\n",n+m-2+min(n,m));
    }
    return 0;
}
