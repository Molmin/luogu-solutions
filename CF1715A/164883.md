# 分析
这道题很显然看一下样例解释就出来了，我们让其中一个人直接**沿着边线**(或者像样例解释里一样走一个Z形也可以)**走到终点**，而另一个人就可以借助第一个人的标记直接跨过一条边，显然我们让他**跨过长边**更优。
![在这里插入图片描述](https://img-blog.csdnimg.cn/2b2a7726ba6641d9b0c1ade101635b1c.png)
(上图来自CF)

### 证明

由于两个人的起点和终点位于不同对角线上，因此无论其中一个人如何走，另一个人都**最多**只能通过上一个人的标记通过一条边。也就是说，即使前一个人把路走的弯弯曲曲，后一个人也只可能跨过其中一条边（指矩形的某条边）。因此显然我们让他跨过最长的边最优。

# 代码实现
```cpp
//省略快读和头文件
int T;
int n, m;

int main()
{
	T = inpt();
	while(T--) {
		n = inpt(), m = inpt();
		if(n == 1 && m == 1)
			puts("0");
		else
			printf("%d\n", n - 1 + m - 1 + min(n, m));
			//第一个人需要走(n - 1) + (m - 1)
			//第二个人只需要走一条短边就可以了，但记得跨越也需要一点能量
	}
	return 0;
}

```
