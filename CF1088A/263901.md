~~超级无敌大水题~~

**题意：**

给定一个数 $x$ ,求满足以下条件的两个数 $a$ 和 $b$ ;

条件： 

**1.**  1$<$ $a$ , $b$ < $x$ ;

**2.**  $a$ 能被 $b$ 整除 ;

**3.** $a$ $*$ $b$ $>$ $x$ ;

**4.** $a$ $/$ $b$ $<$ $x$ ;

**思路：**

首先要注意第二个条件：$a$ 能被 $b$ 整除而不是 $a$ 能整除 $b$ ;
所以 ，千万不要被坑了，样例1是错误滴，样例输出应该是 4 4；

接下来说方法：我们只要按照题意说的条件暴力枚举就可以了，用两个循环，一个用来枚举 $a$, 另一个用来枚举 $b$ ，如果四个条件都符合，就输出这两个数，如果没有符合条件的数，就输出-1。

**上代码**
```cpp
#include <bits/stdc++.h>
using namespace std;
int main()
{
    int x;
    cin>>x;//输入就不用我说了吧 
    for(int i=1;i<=x;i++)//枚举a 
    {
    	for(int j=1;j<=x;j++)//枚举b 
    	{
    		if((i%j==0)&&(j*i>x)&&((i/j)<x))//条件 
            {
            	cout<<i<<" "<<j;//输出符合条件的两个数 
            	return 0;//退出循环；千万不要忘了写这句话，否则 他会把所有符合的数都输出，而题目只用输出一组 
			}
		}     
	}
    cout<<"-1";//如果没有符合条件的数，输出-1； 
    return 0;//完美结束 
}
```
求通过。
