### 思路：

定义一个变量now表示此时车上的人数，遇到上车的人就加，遇到下车的人就减，在每一个状态取最大值（题目中要求最低容量就得求人数最多的时候电车装了多少人）。

___

### 注意：

- 第一个变量是下车人数，第二个变量才是上车人数
- 题目中要求输出的是电车的最低容量（及所有的状态中的最大值），不是最后电车上有多少人（~~给没读题的人说的~~）

___

### code（~~代码最短~~）:

```cpp
#include <bits/stdc++.h>//万能头 
using namespace std;
int n, x, y, now, maxn;//x是下车人数，y才是上车人数(不要错了)，now是此时车上有多少人，maxn是车上最多的时候有多少人(及题目中的最低容量) 
int main()
{
	scanf("%d", &n);//输入有几次上下车
	for(register int i = 1; i <= n; ++i)//采用边读边做的方法 
	{
		scanf("%d %d", &x, &y);//输入下车人数和上车人数
		now = now + y - x;//减去上车人数加上上车人数(再强调一遍，x是下车人数，y才是上车人数) 
		maxn = max(maxn, now);//取max 
	}
	printf("%d", maxn);//输出车的最小容量(车上人数最多的人数)
	return 0;//结束
}
```
