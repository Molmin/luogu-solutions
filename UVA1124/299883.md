### 正文：
本题很适合萌新学习输入输出（所以本题解也是纯新手向题解）。

不用多说什么了，原样打印标准输出即可。

#### FOR 新手

这里介绍一个C语言中的好东西——`scanf`。它不仅可以读入整数，还可以可以只读入一个字符（包括换行和空格），将它存为 `char` 类型并且通过 `printf` 输出。食用方法如下：
```cpp
char c;
scanf("%c",&c);
```

而且 `scanf` 也和 `cin` 一样有返回值，只是读到了多少东西就会返回多少。我们就利用以上特性轻松AC本题。

### 程序如下：
```cpp
#include<cstdio>//用来使用scanf
char c;//定义一个字符变量储存输入的字符
int main(){
    while(scanf("%c",&c)==1)printf("%c",c);//当读到一个字符时，输出那个字符
    return 0;//结束
}
```
