题目中提到是同一天，没有跨天的情况，所以只用分减分，时减时。考虑到分那里可能会有负数（时是不可能的，因为在同一天），所以要从时退位

```cpp
#include<cstdio>//头文件
int main()
{
    int h1,h2,m1,m2;
    scanf("%d %d %d %d",&h1,&m1,&h2,&m2);
    if(m2<m1)//如果不够减
    {
        m2+=60;//退位
        h2--;
    }
    m1=m2-m1;//分减分
    h1=h2-h1;/;/时减时
    printf("%d %d",h1,m1);
    return 0;
}

```
