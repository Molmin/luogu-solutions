### 题意简述

有一个式子 $a^3=b^3+c^3+d^3$，求出当 $a≤100$ 时，满足这个式子的所有三元组 $(b,c,d)$。

### 思路分析

这题数据非常小，四重循环暴力枚举是完全可行的，但由于样例中的 $c,d$ 都是小于 $b$ 的，所以我们在枚举 $c$ 的值时要从 $b$ 的值开始枚举。同样，枚举 $d$ 的值的时候从 $c$ 的值开始枚举，这样，程序计算的实际规模只有 $O(a^3)$ 左右，所以不用担心心暴力枚举会 TLE。
 
注意：由于本题的输出较为复杂，所以建议用 `printf` 输出，这样会更方便一些，同时还会降低出错的可能（~~主要是因为 `printf` 比 `cout` 快~~）。

### 代码实现

```cpp
#include <iostream>
using namespace std;

int main()
{
    for (int i = 2; i <= 100; i++) //直接枚举，本题没有输入
	{
		for (int j = 2; j <= 100; j++)
		{
			for (int k = j; k <= 100; k++)
			{
				for (int l = k; l <= 100; l++)
				{
					if (i * i * i == j * j * j + k * k * k + l * l * l) //判断
					{
						printf("Cube = %d, Triple = (%d,%d,%d)\n", i, j, k, l); // 按格式输出
					} 
				} 
			}
		}
	} 
	return 0;
}
```
