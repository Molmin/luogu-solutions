**思路**：题意大概为给 $n$ 条赛道， $m$ 个运动员，分别给定其划过的路程，所用时间及其赌的卢布，让我们求出每个运动员最短时间所能获得的最大收益收益。

**做法**：我们用 $a$ 来记录这个运动员的位置，存储完之后记录时就不用再求， $b$ 来记录最短时间所能获得的最多对应的卢布，用 $  t $ 来找最短的时间，然后将它的时间和最短时间内能获得的最多的卢布记录下来然后最后用 $ans$ 跑一遍记录总获得的卢布的最大利润，然后累加起来就可以结束了。

代码呈上：
```cpp
#include<bits/stdc++.h>

long long n,m,ans,a[1001],b[1001],l,r,t,c;

			if(t<x[k])//分别记录每一个赛道跑的最快的人，记录位置和赌资 
			{
				a[k]=t;
				b[k]=c;
			}
		}
	}
	for(int i=1;i<=n;i++) ans=ans+b[i];//前面已经记录可以直接累加 
	cout<<ans;//输出
	return 0;//比赛一定要加	
}
```
