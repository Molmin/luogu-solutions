找规律才是最好的方法！！！

我们都知道最小的偶数是 $2$，所以第 $1$ 小偶数也是 $2$。

没事，后面加了一个“位”字也没有关系。 

我们来列一个表格：

| $n$ | 第 $n$ 小的偶数位的回文数 |
| :----------: | :----------: |
| $1$ | $11$ |
| $2$ | $22$ |
| $3$ | $33$ |
| $4$ | $44$ |
| $5$ | $55$ |
| $6$ | $66$ |
| $7$ | $77$ |
| $8$ | $88$ |
| $9$ | $99$ |
| $10$ | $1001$ |


------------
**如果要偶数尽可能小，那么位数要小。**

观察上表可以发现，$1 \sim 9$（一位数） 对应 $11 \sim 99$（两位数），而 $10$（两位数） 对应 $1001$（四位数）。

而正好 $1 \times 2 = 2$，$2 \times 2 = 4$。

而另一半正好是这个数字倒过来。

由于数字较大，建议用字符串读入、存储、输出。 

------------
假设一个数字 ${s_0,s_1,s_2,...,s_n}$（$s_0$、$s_1$ 等表示某一位上的数字），那么整个第 $s$ 小的数就是： 
${s_0,s_1,s_2,...,s_n,s_n,s_{n-1},s_{n-2},s_{n-3},...,s_2,s_1,s_0}$。 



------------
给大家看一下我写的 **浓缩版** C++ 代码：
```cpp
#include <iostream>
using namespace std;
int main(){
	string s; //数字较大，建议用字符串存储。 
	cin>>s;
	for(int i=0;i<s.size();i++)cout<<s[i];
	for(int i=s.size()-1;i>=0;i--)cout<<s[i];
	return 0;
}
```
