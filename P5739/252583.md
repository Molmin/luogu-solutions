# 题目链接：
[P5739 【深基7.例7】计算阶乘](https://www.luogu.com.cn/problem/P5739)

求 
n!(n≤12)n!(n\le12)
n!(n≤12)，也就是 
1×2×3...×n1×2×3...×n
1×2×3...×n。
挑战：尝试不使用循环语句（for、while）完成这个任务。

### 既然说了不用循环，就~~不~~用循环！！
计算n的阶乘，就是计算n*(n-1)*(n-2)......*1

这里是循环版本（解释在CODE里）
```cpp
#include<cstdio>
/*头文件，定义scanf和printf等 
如果不会用这个可以用iostream流 */ 
using namespace std;//名字空间 
int main()
{
	int n,ans=1;//这里用于算乘积的ans一定要初始化为1，因为任何数*0都等于0 
	scanf("%d",&n);//输入不解释 
	for(int i=1;i<=n;i++)//最重要的循环，从1开始计算乘积直到n 
		ans*=i;//累加乘积 
	printf("%d",ans);//输出 
	return 0;//必要的结尾 
} 
```

而题意要求我们尽量不用循环。。

怎么避免循环呢？

给大家介绍一种方法——递归！！！
```
1.在函数的定义中调用函数自身的方法叫递归，这样的函数叫递归函数。
2.递归：无限调用自身这个函数，每次调用总会改动一个关键变量，直到这个关键变量达到边界的时候，不再调用。 
3.如果一个函数在调用自己， 将会无限循环下去， 所以递归函数中肯定有办法在某种情况下停止调用自己，即停止递归， 一般来说使用if语句实现在满足特定的条件下终止递归。 
4.所以递归函数一定要包含两个要素： 
（1）终止条件  
（2）执行内容 
```

上CODE
```cpp
#include<cstdio>//头文件不解释
using namespace std;//名字空间不解释
int jx(int n)//阶乘递归函数
{
	if(n==1)//停止条件，如果n=1，就返回1
		return 1;
	return jx(n-1)*n;//如果n不等于1，进行递归
  //即计算n*（n-1）直至n=1
}
int main()
{
	int n;
	scanf("%d",&n);//输入
	printf("%d",jx(n));//调用函数并输出
	return 0;//结束
}
```
