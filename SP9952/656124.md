由于 $n$ 的范围非常大，所以直接排除模拟这种方法。

那本题就成为了构造数列 $w$ 和找规律得出 $w$ 通项公式的题。

$w_n$ 为 $S(n^2)$ 的数字和。

我们就列出数列找。

数列为：```1 4 9 16 25 36 49 64 81 82 85 90 97...```

然后发现 $81$ 前面的数字是完全平方数，后面就是 $81+$ 一个平方数。

列出通项公式：$n\div 9 \times 81+(n \bmod 9) ^2$。

其实只要推出了通项公式这题就是个大水题，代码非常简单：

```c
#include <bits/stdc++.h>
using namespace std;
int main()
{
	int n;
	while(~scanf("%d",&n))
	{
		int s=(n/9*81)+(n%9)*(n%9);
		printf("%d\n",s);
	} 
	return 0;
}
```
