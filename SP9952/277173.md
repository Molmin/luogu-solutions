本题直接推公式会有点困难，所以可以先写个生成器找找规律。

```1 4 9 16 25 36 49 64 81 82 85 90 97...```

发现前几个数都是完全平方数，后面是$81+$完全平方数，于是可以~~猜~~出公式:$n/9*81+(n$%$9)^2$.

然后代码就很简单啦：
```cpp
#include<bits/stdc++.h>//不要抄！！
using namespace std;
int n;
long long s;
int main()
{
    while(cin>>n)//多组数据
    {
    	s=(n/9*81)+(n%9)*(n%9);
    	cout<<s<<"\n";
	 }
    return 0;
}
```
