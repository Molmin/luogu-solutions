## 总的来说这题还是比较简单的……

## 本蒟蒻用123D建了个模就出来了

	只需要比较一下：上下有两数对应相等的话就写“YES”
	没有相等的话就写“NO”

# 但为啥呢？

首先，两只苍蝇所在的位置只可能在立方体行星的顶点上

所以它们所在的面就是它们所在顶点相邻的三个面

如果它们有长、宽、高之一相等，**它们所在的面中就有两个面重叠**

## 换句话说，两只虫在同一面上了

个人技术原因，~~123D里建的模没弄下来~~

大家看看[这篇题解](https://www.luogu.org/blog/Stylistic-Flowering/solution-cf39d)，里面有个手绘的图，可以拿来看一下

算法解释完毕，上代码：

```
#include<bits/stdc++.h>
using namespace std;
int main(){
    int a[4],b[4],i;  //开个数组，看起来方便
    for ( i = 1 ; i <= 3 ; i++ )
        cin >> a[i];
    for ( i = 1 ; i <= 3 ; i++ )
        cin >> b[i];
    //这些读入就不说了吧
    //一个一个吧两只苍蝇的坐标读经来罢了
    for ( i = 1 ; i <= 3 ; i++ )  //判断是否有相同
    {
        if ( a[i] == b[i] )  //找到相同了
        {
            cout << "YES";
            return 0;
            //直接写return 0可以少开一个bool
            //由于已经找到了，后面的咋样其实就没有意义了
            //所以直接return 0就好了
            //当然，用bool判断找没找到也可以【滑稽】
        }
    }
    cout << "NO";  //没找到
    return 0;
}
```
