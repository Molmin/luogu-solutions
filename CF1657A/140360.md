题意：多次询问，每次给出平面直角坐标系中的一点 $(x,y)$，其中 $0 \le x,y \le 50$。你的起点是 $(0,0)$，如果两点间的欧几里得距离是整数，则你可以一点移动到另一点。问最少移动几次可达 $(x,y)$。

考虑对目标点进行分类讨论。

+ 目标点为 $(0,0)$

答案为 $0$，没啥好说的。

+ 目标点到 $(0,0)$ 的距离为整数

显然一步到位是最优的。判断距离是否为整数，相当于判断距离的平方是否为完全平方数，开桶预处理即可。

+ 其余情况

考虑到平行于坐标轴的运动一定是合法的。对于 $(x,y)$，第一步走到 $(x,0)$，第二步走到 $(x,y)$，答案为 $2$。

代码：

```
#include<bits/stdc++.h>
using namespace std;
bool b[10010];
int main(){
	int T;
	cin>>T;
	for(int i=0;i<=100;i++)
		b[i*i]=1;
	while(T--){
		int x,y;
		cin>>x>>y;
		if(x==0&&y==0){
			cout<<0<<endl;
			continue;
		}
		if(b[x*x+y*y])
			cout<<1<<endl;
		else
			cout<<2<<endl;
	}
	return 0;
}
```
