# 题意

一个 $n$ 位数，要求只由 $0$ 和 $1$ 组成，且 $0$ 不相邻，问有几个满足要求的数。

# 思路

首先在 $n \leq 10000$ 的情况下暴力枚举每一位是 $1$ 还是 $0$ 显然会超时，那么我们就要考虑组合数的解法。

首先我们考虑有 $k$ 个 $1$ 的情况，那么显然有 $n-k$ 个 $0$。这 $k$ 个 $1$ 之间（或前后）会有 $k+1$ 个间隔，现在的问题就被转化为**在 $k+1$ 个空中选取 $n-k$ 个空来放 $0$**。

可以用组合数解决，即计算 $C_{k+1}^{n-k}$。还有一个优化，就是如果 $n-k>k+1$ 的时候，就意味着无论如何都要分两个 $0$ 到同一个间隔之中，题目要求无法满足，所以出现 $n-k>k+1$ 的时候可以直接不计算。

另外这题**没有取模**，使用 ``C++`` 解决需要用高精度，所以代码很难写。
