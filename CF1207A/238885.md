在你的餐厅里有两种汉堡：牛肉汉堡和鸡肉汉堡。每个牛肉汉堡需要$2$片面包和$1$片牛肉，一个鸡肉汉堡需要$2$片面包和$1$个鸡排。

一个牛肉汉堡卖$h$元，一个鸡肉汉堡卖$c$元。

你有$b$片面包，$p$片牛肉和$f$块鸡排。求最大收益。

**解题思路：** 对于这道题，我们可以先将面包数/$2$,便于计算，那个的价格高就多做那个，直到没有肉和面包。若还有面包，继续做收益低的即可。

**$Code$**
```cpp
#include<iostream>
#include<bits/stdc++.h>
#define min(a,b)a<b?a:b//自定义min函数，STL自带的很慢
using namespace std;
int main(){//二面一牛，二面一鸡 
	int T;
	cin>>T;//数据组数T
	int ny,jy,ans;//分别一份牛汉堡的收益，一份鸡汉堡的收益，答案。
	int m,n,j;//面包，牛肉，鸡肉的份数
	for(int i=1;i<=T;++i){
		ans=0;//重置答案累加器为零
		cin>>m>>n>>j;//输入不多说
		m/=2;//除以2计算方便
		cin>>ny>>jy;
		if(ny>jy){//若牛汉堡收益大于鸡汉堡收益
			ans+=min(m,n)*ny;//+牛汉堡的最大收益
			m-=min(m,n);//面包数减掉
			ans+=min(m,j)*jy;//牛汉堡计算之后算鸡汉堡的最大收益
		}
		else{若鸡汉堡收益大于牛汉堡收益
			ans+=min(m,j)*jy;//刚好与上面相反，不多说。
			m-=min(m,j);
			ans+=min(m,n)*ny;
		}
		cout<<ans<<'\n';//输出别忘换行
	}
	return 0;
}
```

最后讲讲**三目运算符**

简单来说就是**条件？ 条件成立返回值：条件不成立返回值**

再看代码中 **$a<b?a:b$** 的意思为： **$a$小于$b$吗？小于返回$a$,不小于返回$b$**
