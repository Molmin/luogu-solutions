这题是真的水啊。

可以直接枚举，反正这三个符号随便放位置(带括号！)也只有$P_{3}^{3}=6$种情况啊。

所以直接暴力，再用个打擂台求六个答案的最大值就好了。
~~尽管大多数情况下肯定是三数乘积最大了。~~

然后就没有然后了。

放代码:
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int a[4],i,sum[10],ma/*ma是迭代器,也是擂主，待会儿循环求最大值的过程就相当于打擂台*/;
	for(i=1;i<=3;++i)
		cin>>a[i];
	ma=a[1];
	sum[1]=a[1]+a[2]+a[3];
	sum[2]=a[1]+a[2]*a[3];
	sum[3]=(a[1]+a[2])*a[3];
	sum[4]=a[1]*a[2]+a[3];
	sum[5]=a[1]*a[2]*a[3];
	sum[6]=a[1]*(a[2]+a[3]);
	for(i=2;i<=6;++i)
		if(ma<sum[i])ma=sum[i];
	/*此部分还有一种写法:
		int ma=0或者-1e9;
		for(i=1;i<=6;++i)
			if(ma<sum[i])ma=sum[i];
		这个就用到了最大值最小の什么东西
	*/
	cout<<ma;
}
```
