### 解题须知的判定方法：

#### 基础判定：

1. 判断一个图形是不是四边形，根据欧氏几何公理（两点之间线段最短）可得：若任意三边长度之和均大于第四边，即为四边形

2. 判断一个图形是不是正方形：根据正方形的定义，四边长度相等的矩形就是正方形。 

3. 判断一个图形是不是矩形：矩形的对边长度相等，判断若出现其中两边长度相等，另外两边长度也相等，即为矩形。 

但是可以有 **优化**。

可以在输入是对四条边的大小排序。

代码实现（用 STL 库）：

```cpp
sort(a+1,a+5)//即将 a[1] 到 a[5] 之间从小到大排序。
```

#### 优化后的判定：

1. 判断一个图形是不是四边形，即最短的三边之和大于最长边。

2. 判断是否为正方形，即最长边等于最短边。

3. 判断是否为矩形，即最短的两边相等，最长的两边相等。

注意数据范围：

$0 \le a_i \le 2^{30}$

虽然没爆 int 但 **最好开 long long**。

### 代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
long long t,a[5];
int main()
{
	scanf("%lld",&t);//t 组数据 
	while(t--)
	{
		for(int i=1;i<=4;i++)
			scanf("%lld",&a[i]);
		sort(a+1,a+5);					//排序 
		if(a[1]+a[2]+a[3]<a[4])			//判断是否为四边形 
			printf("banana\n");
		else if(a[1]==a[4])				//判断是否为正方形 
			printf("square\n");
		else if(a[1]==a[2]&&a[3]==a[4])	//判断是否为矩形 
			printf("rectangle\n");
		else							//不是四边形 
			printf("quadrangle\n");
	}
	return 0;
}
```
