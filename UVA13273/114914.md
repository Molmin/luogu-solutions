#### Description

> - 给定 $N$，要求组建一个团队，要求里面必须包含团队负责人，首席开发人员，首席测试人员，营销经理四个职位，可以一人多职，不担任职务的其他人都是普通人。
>- 求能组建出的不同的团队个数，两个团队是不同的即满足下面任意一个定义即可：
>    1. 两个团的参赛者个数不同
>    2. 两个团的参赛者个数相同，至少有一个选手不同
>    3. 两个团的选手相同，至少有一个选手的职业不同。
>- 答案对 $10^8+7$ 取模。
>- 本题多测，测试数据个数在 $[1,10^4+1]$ 的范围内。
>- $0 < N < 10^7+1$。
>- 翻译者：@一只书虫仔。

#### Solution

萌新不会 AKrry 的强强做法，但是这题一脸找规律，所以只好来打表了。

通过样例和会 T 掉的暴力，可以得到 $N=1,2,3,4,5$ 的情况：

- $N=1$，答案为 $1$
- $N=2$，答案为 $18$
- $N=3$，答案为 $132$
- $N=4$，答案为 $680$
- $N=5$，答案为 $2880$

然后，对着这些毫无头绪的数字，根本没思路。

这时，要用到我们的好东西：[Oeis](https://oeis.org/) 了！

搜查结果：[Link](https://oeis.org/search?q=1%2C+18%2C+132%2C+680%2C+2880&language=english&go=Search)，就会发现一个数列 [A058649](https://oeis.org/A058649)，规律为：

$$2^{n-4} \times n \times (n+1) \times (n^2+ 5 \times n - 2)$$

然后我们化简一下式子：

$$\begin{aligned}2^{n-4}\times n \times (n+1) \times (n^2+5 \times n-2)&=2^{n-4}\times(n^2+n)\times (n^2+5 \times n-2)\\&=2^{n-4}\times (n^4+6 \times n^3+3 \times n^2-2n)\end{aligned}$$

然后用快速幂算出 $2^{n-4}$，直接算出后面的式子即可，有的地方溢出记得取模。
