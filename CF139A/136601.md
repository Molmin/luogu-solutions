### 大致翻译（机翻）
```
一个星期天，皮特去了一家书店，买了一本关于体育节目的新书。这本书整整有N页。

皮特决定从第二天开始阅读，也就是从周一开始。佩特的日程安排很紧，他知道一周中的每一天他能读多少页。有些日子太忙了，彼得什么都没时间看书。然而，我们知道他每周至少能读一页。

假设Petr不会跳过几天的时间，每天都会尽可能多地阅读，那么就确定他将在一周中的哪一天阅读这本书的最后一页。

第一个输入行包含单个整数n（1<=n<=1000）-书籍中的页数。

第二行包含7个不超过1000的非负空格分隔整数-这些整数表示Petr可以相应地在周一、周二、周三、周四、周五、周六和周日读取多少页。可以保证这些数字中至少有一个大于零。

打印一个数字-一周中的某一天，彼特将读完这本书。一周中的几天从自然顺序开始编号：周一、周二、周三、周四、周五、周六、周日。

在第一周的星期一，皮特将阅读第一页。在第二周的星期一，皮特将阅读第二页，并完成这本书的阅读。
```
### 算法分析
```
由于n的范围极小，这题可以直接暴力模拟。只要定义一个int类型的p代表天数，当天数增加时p=p%7+1，只要当前页数和小于n就一直读书，最后输出p即可
```
### 代码（防作弊）
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a[8],p,sum;
int main(){
	cin>>n;
	for(int i=1;i<=7;i++) cin>>a[i];
	while(1){
		if(sum>n) break;//一直循环直到当前页数和大于n
		p=p%7+1;//天数增加，（如果是（p+1）%7就会出现0）
		sum+=a[p];//页数和继续累加
	}
	cout<<p;//输出最终答案
}
```
