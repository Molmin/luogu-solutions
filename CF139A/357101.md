# 翻译
题面（机翻）：
```cpp
一个星期天，彼得去一家书店买了一本关于体育节目的新书。这本书正好有n页。

彼得决定从第二天（即星期一）开始阅读。彼得的时间表很紧，在一周的每一天，他都知道他当天可以阅读多少页。某些日子太忙了，Petr将没有时间阅读任何东西。但是，我们知道他每周至少可以阅读一页。

假设彼得不会跳过几天，并且每天会尽可能多地阅读，请确定他将在一周的哪一天阅读本书的最后一页。
```

# 题意

Petr有本$n$页的书，给出一本书的页数和一周每天看多少页，求看完的那天是星期几。

数据范围$1 \leq n \leq 1000$ .

# 分析

本题 $n$ 的范围较小，直接模拟就能过。用一个 $i$  来表示现在的天数，每过完一天后，用 $n$ 减去当天看的页数，之后 $i++$ ，直到$n \leq 0$ 为止。如果 $i \geq7$ ,用 $i$ 在对 $7$ 取模。因为使用for，输出时 $i$ 会自动加 $1$ ,不过要特判 $i=0$ 的情况。

# 代码

```cpp
#include<cstdio>
using namespace std;
int a[7],n,i;//a为每周看的页数，i为周几
int main()
{
    scanf("%d",&n);
    for(i=0;i<7;i++
    	)scanf("%d",&a[i]);
    for(i=0;n>0;i=(i+1)%7)
	//i=(i+1)%7同时完成了i++与i%7
        n-=a[i];
    printf("%d",(!i?7:i));//特判i=0的情况
    return 0;
}
```
