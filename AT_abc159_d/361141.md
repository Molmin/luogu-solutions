## AT4816 [ABC159D] Banned K

题目大意：

给一个长度为 $n$ 的序列，问每次删去 $a[i]$ 后，有多少对两两相等的数。

思路：

用一个 $sum$ 表示一共有几对相同的数字，即先用一个桶保存每一个数的出现次数，然后用**全排列公式**算出总地情况即可。最后在减一下就可以了。

所以该部分的代码为：


```cpp
	for(int i=0;i<=n;i++)
	{
		if(t[i]>0)
		{
			sum+=(t[i]-1)*t[i]/2;
			b[i]=t[i]-1;
		}
	}
```
值得注意，$i$ 应该 $0$ 开始，为什么自己想一想。
