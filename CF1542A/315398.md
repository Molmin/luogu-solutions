题意：给你一个集合，问你是否能将这个集合分成两两一对，使每对两个数字之和为奇数。能就输出```Yes```，不能就输出```No```。  
~~这是我昨天在考场上唯一做出来的题。~~  
学过~~小学~~数学的人都知道，如果两个数相加，只有两数一奇一偶，和才会是奇数，而且自然数中除了奇数外就是偶数。     
既然要求分成两两一对，使每对之和为奇数，那么只需要判断奇数和偶数数量是否相等即可。只有这样，才可以保证每一对都是一奇一偶。  
根据自然数中除了奇数外就是偶数，因此偶数可以通过总数减去奇数的数量来求。所以，如果奇数和偶数一样的话，那么奇数的数量应该占总数的一半。           
code：

```cpp
#include<bits/stdc++.h>
using namespace std;
int T,n,i,s,a[10001];
int main(){
	scanf("%d",&T);//读入
	while (T--){//T组数据
		scanf("%d",&n);
		s=0;
		for (i=1;i<=2*n;i++)//长度为2n
        	scanf("%d",&a[i]),s+=a[i]%2;//统计奇数的个数。应为奇数%2=1。
		if (s==n)//如果奇数和偶数一样的话，那么奇数的数量应该占总数的一半
        printf("Yes\n");//可以分
		else
        printf("No\n");//不可以分
	}
	return 0;
}

```
