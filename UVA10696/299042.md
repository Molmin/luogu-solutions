本题涉及到一个很著名的数学函数——91函数

可能有同学不懂为什么叫做91函数。我来举个栗子：

`1 -> 12 -> 23 -> 34 -> 45 -> 56 -> 67 -> 78 > 89 -> 100 -> 111 -> 101 -> 91`

`102 -> 92 -> 103 -> 93 -> 104 -> 94 -> 105 ...... -> 109 -> 99 -> 110 -> 100 -> 90 -> 101 -> 91`

由此得出，无论 $x$ 是多少，最后总是会等于 $91$ ，所以叫做91函数。

所以，我们不用一层一层递归，直接返回 `91` 就行了
```cpp
#include <cstdio>

using namespace std;

int n;

int f91(int x) // 91函数
{
	if(x <= 100)
	{
		return 91; // 不需要递归
	}
	else
	{
		return n - 10;
	}
}

int main()
{
	while(scanf("%d", &n) && n)
	{
		printf("f91(%d) = %d\n", n, f91(n)); //计算并输出
	}
	return 0;
}
```
