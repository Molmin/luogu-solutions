这道题是一个世界上经典函数：

# $f91$函数

他的原理其实和**冰雹猜想**差不多，因为**不管在100以下的数如何变化，最终都会变回91**，所以叫做$f91$函数。

了解了这一点，这一题就是一道真真正正的大水题了。

至于参与运算的数$N$要是大于100的话，直接输出$N-10$就行了。

~~好吧……如果说这道题有一点点难点的话，那就是如何判定读入0的时候结束程序了……~~

## 上代码：

```cpp
#include<iostream>
using namespace std;
int main()
{
	int a;
	while(cin>>a&&a!=0)//当读到了a且a不等于0时继续运行
	if(a<=100)printf("f91(%d)=%d\n",a,91);//当a小于100时输出91
		else printf("f91(%d)=%d\n",a,a-10);//否则输出a-10
	return 0;
}
```
