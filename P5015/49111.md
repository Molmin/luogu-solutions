# EOF函数的用法

EOF函数是End Of File的缩写，表示的是输入的结束。在输入文件中，总会有一个特殊的、你看不到的字符来表示输入文件的结束。一个终端的输入永远不会有结束，除非你断掉电源。而EOF函数正是能让终端输入分为多个小文件的函数。

下面我们来看代码，边看代码边理解：
```
#include<cstdio>
#include<cstring>
int main()
{
	char s[10];
	int n=0;
	while(scanf("%s",s+1)!=EOF)n+=strlen(s+1);
	printf("%d",n);
}
```
实际上，中间的
```
while(scanf("%s",s+1)!=EOF)n+=strlen(s+1)
```
可以这么理解：
```
while(1)
{
	scanf("%s",s+1);
    if(s!=EOF)
    {
    	n+=strlen(s+1);
	}
    else
    {
    	break;
	}
}
```
