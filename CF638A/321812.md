## 前言：

一道明显的数论

------------

## 题意：

你在一个街道的头，一边的房子编号为
$1$,$3$,$5$,$7$,$9$………另一边编号为
$n$,$n-2$,$n-4$,$n-6$,$n-8$………且你到第一个房子和每个房子之间的距离为$1$，给你你家的门牌号为$a$，求距离

------------


## 分析
分为两种情况：
1. 门牌号从$1$开始到$a$,两个数之间差为$2$
1. 门牌号从$n$开始到$a$,两个数之间差为$2$

可以得出
**用等差数列的公式**：

**$（头-尾）/公差+1$**~~（小学数学不会不会吧？）~~

------------


## 思路
分类讨论$a$的情况

1. 当$a$为偶数时 ，头为$n$,尾为$a$,代入公式为$\dfrac{(n-a)}{2}+1$
2. 当$a$为奇数时 ，头为$a$,尾为$1$，代入公式为$\dfrac{(a-1)}{2}+1$

------------


## 代码
```
#include<bits/stdc++.h>//万能头好评 
using namespace std;
int n,a;
int main() {
	cin>>n>>a;
	if(a%2==0)//当在偶数边时 
	{
		cout<<(n-a)/2+1;//公式
		return 0;//提前结束 
	}
	else//当在奇数边时 
	cout<<(a-1)/2+1;//公式
	return 0;//好习惯  ^_^ 
}
```
