## 思路
显然，此题博弈。

我们将此题题意进行化简，发现可以浓缩如下：

给定两个整数，A 去掉一个整数，若剩下一个整数可以**平均分成两份**，则 A 胜利，否则，B 胜利。

我们注意到，粗体部分的条件实际上就是表示剩下的这个整数是个偶数，这样才能平均分成两份。

那么，对于先手 A，要尽可能地让剩下的一个数是一个偶数，这样，A 可以完成将次数分成两份的任务，A 即可胜利。

那么就有以下三种情况：

1. 若给定的数为两个偶数，则不论 A 取走哪个，他都必定可以令剩下的数是个偶数，A 必胜。

2. 若给定的数是一个偶数与一个奇数，则只要取走奇数，剩下的仍是偶数，A 亦必胜。

3. 若给定的数是两个奇数，则不论 A 取走哪个，剩下的数都是个奇数，A 必败，则 B 必胜。

那么，我们即可实现如下代码通过此题——

## Code
```cpp
#include <bits/stdc++.h>
using namespace std;
int t;
int main() {
    cin >> t;
    while(t--) {
        int c1,c2;
        cin >> c1 >> c2;
        if (c1%2==1&&c2%2==1) cout << "Ramesh\n";//若是两个奇数，则后手必胜
        else cout << "Suresh\n";//否则，先手必胜
    }
    return 0;
}
```
