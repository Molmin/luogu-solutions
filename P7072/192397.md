# 1 一句话题意
给你一个**无序**序列$a_1,a_2......a_n$和一个数$w$，请你对于每个$i(1≤i≤n)$，给出$a_1,a_2......a_i$中第$\left\lfloor\dfrac{i·w}{100}\right\rfloor$大的数。

# 2 解法一 暴力
拿到这个排在第二题的题目，马上就会想到可以用暴力解。

(据我所知，往年的第二问都是太模拟）

```cpp
for(int i=0;i<n;i++){
	cin>>a[i];
  	sort(a,a+i,cmp);
  	/*
   	bool cmp(int a,int b){
  	    return a>b;
  	}
  	*/
	cout<<a[i*w/100];
}
```
我写完以后，转念一想，**会不会TLE**?

吓得我赶紧提起笔算了算复杂度：

快排是$O(nlogn)$，执行了$n$次，所以总的复杂度就是$O(n^2logn)$次。

然后$n$最大可能是$10^6$，爆了。

赶紧想正解。

# 3 解法二 桶
那么题目要求的复杂度，应该是$O(nlogn)$以下。

想了一会儿，以为是$DP$，但是写了一会儿不会写了。只好另想方法。

然后我又转念一想，能不能按照数个数的方法，按计划获奖人数来往下数，数到哪条分数线就是哪一个？

有道理啊。

题干里面还说同学们的分数都是600分以下，这是一个线索。

那么，不就是用桶的方法了吗？

把每个人的分数按分数存起来，挨个加起来。

详见代码。

```
for(int i=0;i<n;i++){
	cin>>a[i];
  	buck[a[i]]++;//x分的人有buck[x]个
  	int sum=0,cnt=601;
  	while(sum<=(i*w)/100){
        cnt--;
        sum+=buck[cnt];
    }
    cout<<cnt+1;
}
```
