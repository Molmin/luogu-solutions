根据题目需求，按要求输出一个序列，判断几个条件即可：

1、求出最左边的页码和最右边的页码。如果最左边的页码在第一页前面，最左边的页码就是1；如果最右边的页码在第$n$页前面，最右边的页码就是$n$。

2、判断最左边的页码是否为1，如果不是就输出<<。

3、从最左边的页码开始，到最右边的页码。当到第$p$页时，加括号输出。

2、判断最右边的页码是否为$n$，如果不是就输出>>。
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int n,p,k,l,r;
	cin>>n>>p>>k;
	l=max(p-k,0); //最左边的页码
	r=min(p+k,n); //最右边的页码
	if(l!=1) cout<<"<< "; //不是第一页就输出<<
	for(register int i=l;i<=r;i++) //最左边的页码到最右边的页码
	{
		if(i==p) cout<<'('<<i<<')'<<' '; //第p页，加括号输出
		else cout<<i<<' ';
	}
	if(r!=n) cout<<">>"; //不是第n页就输出>>
	cout<<endl; //换行
	return 0;
}
```
