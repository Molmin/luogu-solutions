一到有关于三角形的CF水题。

**题目大意**

有一个升序排列的数列 $a$。$(a_i\le a_{i+1})$

问是否存在三个数 $i,j,k$，满足：

- $1\le i<j<k\le n$。
- 以 $a_i,a_j,a_k$ 为三边长的三角形不存在。(比如 $3,4,5$ 可以构成一个三角形，但是 $3,4,7$ 不能。)

如果存在，依次输出 $i,j,k$ 的值；

如果不存在，输出 $\mathtt{-1}$。

**题目思路**

三角形的三条边的关系：任意两边之和大于第三边。（不会的重修小三数学）

我们可以直接取最小的两个和最大的一个，如果这都不行的话那么就没有办法了。

比如序列为：$[1,2,5,7,8,9,21,45]$，我们可以取 $[1,2,45]$，可以组成符合要求的三角形。

举个反例：$[5,5,5,7]$，我们取 $[5,5,7]$，发现不满足 $1\le i<j<k\le n$ 的要求，不可以组成符合要求的三角形。

程序就比较显而易见了：

```cpp
#include <bits/stdc++.h>
using namespace std;
int T, a[500000], n;

int main()
{
    cin >> T;
    while (T--)
    {
        cin >> n;
        for (int i = 0; i < n; i++)
        {
            cin >> a[i];
        }
        if (a[0] + a[1] > a[n - 1] && a[0] + a[n - 1] > a[1] && a[1] + a[0] > a[n - 1]) //按照我们找出来的规律取数，如果不行的话输出 -1。
        {
            cout << -1 << endl;
        }
        else //输出可以的方案
        {
            cout << 1 << " " << 2 << " " << n << endl;
        }
    }
    return 0;
}
```
