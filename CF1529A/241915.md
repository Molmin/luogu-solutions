## 题意简述：

本题有多组数据。

给定一个长度为 $n$ 的数组，你一次操作可以从中选取一个子序列，将其中**大于子序列平均值**的数删去，问最多能删去几个数。

## 方法：

通过题干我们不难发现，**只有数列中最小的那几个数无法删去**，其他都可以。

**为什么呢？**

我们现在先找到这个数列中的最小值，然后依次将他**相邻**的数消掉（因为已经是最小值，所以肯定消得掉），最后就只会剩下数列中最小的那几个数。

所以我们只要找出数组中不是最小值的数的个数就行了。

code：

```cpp
#include <bits/stdc++.h>
using namespace std;
int a[105],n,t,Min,ans; 
int main()
{
	cin >> t;
	while(t--)
	{
		cin >> n;
		Min = 105,ans = 0;
		for (int i = 1; i <= n; i++)
			{
				cin >> a[i];
				Min = min(Min,a[i]);//找到最小值。
			}
		for (int i = 1; i <= n; i++)
			{
				if (a[i] != Min) ans++;
                //找出数组中不是最小值的数的个数。
				}	
		cout << ans << endl;		
	}
	return 0;
}

```
