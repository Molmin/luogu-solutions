# 书归正传
根据本题题意，我们可以得知：这是一道用程序来控制输出图形的题。关键是，他一共有两种图形，分别为X和"."。若只用一种变量控制图形，则会出现与目标图形不相符的形状，即每一列的图形是交错的。此题注意点如下：

1. 需要两个变量控制。
1. 输出的图形是什么要搞清楚。
1. 一定要加回车！

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int a,b,c,d,e=1,f=1;//创建变量
	cin>>a>>b>>c>>d;
	for(int i=1;i<=a*c;i++){
		e=f;//控制第一个出来的（每列的第一个）
		for(int j=1;j<=b*d;j++){
			if(e){
				cout<<"X";
			}else{
				cout<<".";
			}
			if(j%d==0){//判断输出的是X还是.
				if(e){
					e=0;
				}else{
					e=1;
				}
			}
		}
		cout<<endl;
		if(i%c==0){//判断第一个输出的是X还是.
			if(f){
				f=0;
			}else{
				f=1;
			}
		}
	}
	return 0;
}

```
