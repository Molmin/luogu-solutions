**题目大意**

给定一个长度为奇数的由小写字母组成的字符串 $s$ 和小写字母 $c$，每次操作可以在 $s$ 中删除相邻的 $2$ 个字母，求经过若干次操作后是否可以将 $s$ 变成 $c$。

**思路**

既然每次只能删除相邻 $2$ 个字母，那么 $c$ 最终在 $s$ 中的位置的奇偶性不变。而在 $c$ 本身中，它自身的位置可以视为 $1$。也就是说，只需判断 $c$ 在 $s$ 中的位置是否是奇数即可。

**MAIN CODE**
```cpp
for(int i=0;i<s.length();i++)
	if(s[i]==c&&!(i&1)){//注意字符串的下标从 0 开始
		flag=1;
		break;
	}
```
