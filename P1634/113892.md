##### 其实挺easy，小学奥数就学过类似分裂呀什么的题。
### 但是注意：这不是分裂，是传染呀，原来的也得加上呢……  
#####  总之，先看代码吧

------------

```cpp
#include <iostream>

using namespace std;

int main()
{
	int n,x;
	cin>>x>>n;
	int ans=1;
	int i;
	for(i=1;i<=n;i++)//传染开始
	{
		ans+=ans*x;
	 } 
	 cout<<ans;
	return 0;//完美的结尾
}

```
