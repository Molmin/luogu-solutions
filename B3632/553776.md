一些集合之间的关系：

- 当两个集合完全一致，那么两个集合**相等**。记作 $A=B$。

- 当集合 $A$ 是集合 $B$ 的一部分，$A$ 是 $B$ 的子集，记作 $A \subseteq B$。


- 当两个集合 $A \ B$ 共有的部分，被称为交集，记作 $A \cap B = \{3,7\}$。

- 当两个集合 $A \ B$ 合并起来，成为并集，记作 $A \cup B = \{...\}$。

例如：

- $A = \{ 1,3,7,5 \} \ \  B = \{ 1,5,7,3\}$，那么 $A=B$；

- $A = \{ 3,5\} \ \  B = \{ 1,3,5,7\}$，那么 $A \subseteq B$；

- $A = \{ 1,3,5,7\} \ \  B = \{ 0,3,6,7\}$，那么 $ A \cap B = \{3,7\}$；

- $A = \{ 1,3,5,7\} \ \  B = \{ 0,3,6,7\}$，那么 $ A \cup B = \{0,1,3,5,6,7\}$。

由于数据范围很小，所以我们可以这么做：

- 开两个桶，读入集合，如果读入了 $i$ 就将 $a_i$ 设为 $1$，说明这个集合有 $i$。
- 因为 $|A|=n$，所以先直接输出 $n$;
- 题目中说明集合内只会出现 $0$ 到 $63$ 之间的数字，所以我们从 $1$ 枚举到 $63$，如果 $a_i = b_i = 1$ 说明这个数字在两个集合都出现过，如果 $a_i = 1  \ \texttt{or} \ b_i =1$，则说明这个数字出现在集合中。
