## 题意
华佗去看病时会带药，他总把药串成串，每串放进一个葫芦中。为了避免忘记，如果他带了 $A$ , $B$ 两种药，他就会把药串成形如 $A$ - $B$ - $A$ 的形式放入葫芦。

### 输入 

第一行,输入 $n$ 表示葫芦数。

下面 $n$ 行 每行一个数，表示华佗在第 $i$ 个葫芦里装的药的种类数。

### 输出

共 $n$ 行，输出第 $i$ 个葫芦中药的个数。

输入输出格式见样例。

## 思路
我们发现，每个葫芦里装的药构成一个回文串，而回文中心就是第 $n$ 个数。

所以，此题转换为：给定回文半径，求回文串长度。

因为此串有回文中心，所以长度公式要减 $1$ ，就是 $ans=n\times2-1$ ，所以读入后直接输出即可。
## 代码
 
 ```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a;
int main(){
	cin>>n;
    for(int i=0;i<n;i++){
        cin>>a;
        cout<<"Case #"<<i+1<<": "<<a*2-1<<endl;
    } 
} 
```
