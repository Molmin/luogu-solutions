考虑一个作轴对称的性质。

对于操作 ```1```，直接开两个全局变量 $l$ 和 $r$，把 $x$ 和 $y$ 分别累加进去即可。

```cpp
if(opt==1){
	scanf("%lld",&y);
	l+=x,r+=y;
}
```


对于操作 ```2```，手玩一下可以发现就是把 $x_{i}$ 变成 $x_{i}+|x_{i}-y_{i}|$，把 $y_{i}$ 变成 $y_{i}-|x_{i}-y_{i}|$。**注意：这里要先把 $x_{i}$ 与 $y_{i}$ 加上 $l$ 和 $r$ 再进行变换操作，详情见代码。**

```cpp
if(opt==2){
	int ABS=b[x]+r-a[x]-l;
	a[x]+=ABS,b[x]-=ABS;
}
```
对于操作 ```3```，由于我们已经把棘手的 ```2``` 操作处理了，直接分别加上 $l$ 与 $r$ 即可。

```cpp
if(opt==3) printf("%lld %lld\n",a[x]+l,b[x]+r);
```
