## 题意

有一个持续两天的活动，第 $1$ 天有 $A$ 人参加，第 $2$ 天有 $B$ 人参加。所有参加者中，参加 $1$ 天及以上的人数最少是多少人，最多是多少人？

## 解析

结合三组样例分析一下，我们不难发现，参加 $1$ 天及以上的人数最少，即为 $A$ 和 $B$ 中的最大值，最多则为 $A$ 与 $B$ 的和。 

例如，样例一中：

`input:`

```cpp
  4 3
```


`output:`

```cpp
  4 7
```

- 第一天来了 $4$ 个人，很显然，如果第二天参加的 $3$ 个人也都是他们 $4$ 个人中的任意 $3$ 个，即有 $3$ 个人重复出现了，则参加 $1$ 天及以上的人数为 $4$，为最少情况。

- 反之，当两天来的人中不存在重复的情况时，答案为 $4+3=7$，也是参加 $1$ 天及以上的人数的最多情况。

综上所述，我们可以直接用内置的 $\max$ 函数来实现输出最大值，接着输出两数之和。


## AC Code：
```cpp
#include <iostream>
using namespace std;
int a,b;
int main()
{
	cin>>a>>b;//输入 
	cout<<max(a,b)<<' '<<a+b<<"\n";//输出(岛国题记得换行) 
	return 0;//好习惯 
}


```
