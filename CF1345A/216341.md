### 题目大意   
* 给定 $n$ 和 $m$ 。
* 你需要求出能否用题目中所给出的拼图拼出 $n$ 行 $m$ 列的大拼图。 
* 要求小拼图之间的凸凹刚好是吻合的。   
### 思路  
这题看似不容易，实际上稍微推一下就可以解决。  
先考虑 $n$ 或 $m$ 等于$1$时的情况，显而易见是可行的。  
可以从样例中给出的 $n = 1,m = 3$ 这种情况入手，因为每一个小拼图至少有一个凸凹面，所以可以一个接着一个拼下去。  
再考虑 $n = m = 2$ 的情况，题目中已经给出，是可行的。  
可是此时拼出的大拼图每一面都是凸面朝外，如果要拼出$n = 2,m = 3$的大拼图就一定会产生矛盾的情况(两凸面相对)  
综上所述，只有$m=1$ 或 $n=1$ 或 $m=n=2$ 时可以拼出。  
### 代码  
代码的实现比较容易，这里就不多加说明了~~其实是我懒~~。  
```cpp
#include <bits/stdc++.h>
using namespace std;
int n;
int a,b;
int main(){
	cin>>n;
    for(int i = 0;i < n;i++){
    	cin>>a>>b;
    	if((a==1||b==1)||(a==2&&b==2))cout<<"YES\n";
    	else cout<<"NO\n";
	}
    return 0;
}
```
