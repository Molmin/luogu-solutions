典型的一道**前缀和**题目嘛。

这道题目的意思很明确：

查询**静态区间中**相邻**两个字符相等**的**个数**。

设**前缀和数组**为 $sum$ ，每位 $sum_i$ 保存着 $1$ 到 $i-1$ 之间的符合**条件（相邻两个字符相等）** 的 $i$ 的个数。

式子就是：$sum_i=sum_{i-1}+$ 这是不是一个**合法的** $i$ 。

然后，坑点来了——

假设我要区间查询 $l$ 到 $r$ 之间**合法数的个数**，但是我们要把前缀和式子

$$s_r-s_{l-1}$$

改一下，变成

$$s_{r-1}-s_{l-1}$$

其中的原因**自己思考**。

int main()
{
	while((a[1]=getchar())!='#'&&a[1]!='.');
	while((a[++n]=getchar())=='#'||a[n]=='.');
	//读入字符串
	in(m);n--;
	Fu(i,1,n-1)s[i]=s[i-1]+((a[i]==a[i+1])?1:0);
	Fu(i,1,m)in(t1,t2),out(s[t2-1]-s[t1-1]);//上面的公式
	return 0;
}
