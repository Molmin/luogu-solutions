思路：

1. 贪心思想：

因为走五步移动的最快，所以首先只走五步，直到剩余的步数小于五，那么走$1$或$2$或$3$或$4$步。

公式：
$$\text{n mod 5=0 : n/5}$$

$$\text{n mod 5!=0 : n/5+1}$$

2. 也可以用另一种思路想：

假如只走$5$步，那么直接用$n$除以$5$，如果有小数说明还有需要走的路程，于是可以用向上取整。

公式：

$$\lfloor\frac{n}{5}\rfloor$$

------------
代码：
```python
import math;print(math.ceil(int(input())/5))
```
