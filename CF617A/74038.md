这道题可以使用贪心的思想解决。为了移动步数最少，就应当使得走五步的次数最多。这时有两种情况:

1. 所需移动的步数N恰好是5的倍数，此时全走五步即可，总步数为N/5。

2. 所需移动的步数N不是5的倍数，此时可以先全走五步，最后一步因为肯定大于1小于5，因此只需要再移动一步即可，总步数为int(N/5)+1。

这个公式可以归结为ceil(N/5.0)，也就是求不小于N/5.0的最小整数。注意，由于C++处理整数除法的方法，必须通过将5改写为5.0以将整数除法运算改为浮点数除法运算，否则将导致错误。

代码:

```
#include <iostream>
#include <cmath>
using namespace std; //头文件和命名空间
int main(){
	ios::sync_with_stdio(false); //输入输出加速
	int iLen; //长度
	cin >> iLen; //读入长度
	cout << ceil(iLen / 5.0); //计算并向上取整
	return 0; //结束
}
```
