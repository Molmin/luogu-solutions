### P1 题意

规定一串数列的长度为 $n(n\le 50)$，这串数列由 $0$ 和 $1$ 组成。在这数列中，不能有两个 $1$ 放在一起，请问有多少种方案。

### P2 题解

首先找一下规律：

- $n=1$ 时，串可以为 `0`，`1`。
- $n=2$ 时，串可以为 `00`，`01`，`10`。
- $n=3$ 时，串可以为 `000`，`001`，`010`，`100`，`101`。
- $n=4$ 时，串可以为 `0000`，`0001`，`0010`，`0100`，`1000`，`1010`，`0101`，`1001`。

- $......$

由这些规律可以发现：

- $ans_1=2$
- $ans_2=3$
- $ans_3=5$
- $ans_4=8$
- $......$

哇，这是啥？经典的斐波那契数列！那就可以写代码了~

### P3 代码

```cpp
#include <cstdio>
#define int long long

const int N = 100 + 7;

int a[N], cases;

inline void Solve() {
    int n; scanf("%lld", &n);
    printf("Scenario #%lld:\n", ++cases);
    printf("%lld\n", a[n]); // 记得用 %lld 输出~
} // 按要求调用 & 输出

signed main() {
    a[0] = 1; a[1] = 2;
    for(int i=2; i<=60; i++)
        a[i] = a[i-1] + a[i-2]; // 计算斐波那契数列
    int T; scanf("%lld", &T);
    while(T--) 
        Solve();
}
```
