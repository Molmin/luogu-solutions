只因数的定义是将一个正整数分解质因数后只要有一个质因数是偶数就是只因数

一看就明白，是偶数的质因数只有 $2$ 呀！

这就解决了

对于每一个数，只需要判断每一个数是否被 $2$ 整除，如果被 $2$ 整除就代表它一定是只因数，所以时间复杂为线性的，是 $\Theta(T)$ 

注意数据范围： $2\le n\le10^{18}$

给出代码如下

```cpp
#include<iostream>
using namespace std;
int main(){
	int t;
	long long a;//注意数据范围
	cin>>t;
	for(int i=1;i<=t;i++){//线性复杂度
		cin>>a;
		if(a%2==0){//判断是否是偶数，即它的质因数里有没有偶数
			cout<<"No";
		}
		else cout<<"Yes";
		cout<<endl;
	}
}
```
