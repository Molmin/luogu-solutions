这题的题意其实很明确，就是让你输出合法的括号序列，~~只不过这道题好像没翻译~~，很多人可能看到这道题第一时间会想到写暴搜，当然可以，但本题解将提供一种模拟思路。

首先,在纸上枚举一下 $1$ 到 $5$，你会发现括号之间是有规律的，这里以 $5$ 为例：

```
5

()()()()()
(())()()()
((()))()()
(((())))()
((((()))))
```

怎么样，有没有发现上述 $5$ 种情况中所包含的规律。没错上述情况的规律就是每次有后面的一个括号保住前面的扩号，从第一次全独立，一直到全部包住，显然正好 $5$ 种情况。

那么我们可不可全按这种规律输出呢？答案显然是可以的，根据上面的例子，横容易知道无论 n 是多少，只要按这种规律输出完，正好 n 种情况。

那么如何实现呢？我们只要用一个 for 从 $2$ 枚举到 $2n+2$，每次 $i+2$，然后输出 $i/2$ 个前括号与 $i/2$ 个后括号，最后输出其他的一一配对的括号就大功告成了！

以下是代码实现：

```
#include<bits/stdc++.h>
using namespace std;
int main(){
  int T,n;
  scanf("%d",&T);
  while(T--) {
    scanf("%d",&n);
    int l=(n<<1)+2;
    for(int i=2;i<l;i+=2) {//从 2 枚举到 2n+2
      int j=i>>1;//前括号的个数
      for(int f=0;f<j;f++) printf("(");
      for(int f=j;f<i;f++) printf(")");
      for(int f=i;f<l-2;f+=2) printf("()");输出独立的括号
      puts("");
    }
  }
  return 0;
}
```
