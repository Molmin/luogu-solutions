题意：在一个 $2 \times 2$ 的矩阵中，有的数是 $1$，有的数是 $0$，定义一次操作为：选择一个点，把与这个点横坐标或纵坐标相同的点变为 $0$，问至少需要几次操作才能把图上的所有点变为 $0$。

因为矩阵大小为 $2 \times 2$，不是很大，所以我们可以用分类讨论的方法来做这道题。

我们先想可能的答案中最大的答案，那就是矩阵中所有的点都是 $1$，那么我们只需要选择在矩阵的左上方和右下方的点即可。

然后再想可能的答案中最小的答案，就是矩阵中全是 $0$，如果这样，我们就不需要放点。

最后再想其它的可能，如果矩阵中值为 $1$ 的点只有 $1$ 或 $2$ 或 $3$，那么我们只需要放一个点即可，因为在这个矩阵中一个点可以把它本身以及与它横坐标或纵坐标相同的点变为 $0$，最多有三个点可以从 $1$ 变为 $0$。

代码：

```cpp
#include<cstdio>

using namespace std;

int t;
int a[10][10];

int main(){
	scanf("%d",&t);
	while(t--){
		scanf("%d%d",&a[1][1],&a[1][2]);
		scanf("%d%d",&a[2][1],&a[2][2]);
		if(a[1][1]==1&&a[1][2]==1&&a[2][1]==1&&a[2][2]==1){
			printf("2\n");//矩阵中全是1时的情况
			continue;
		}
		if(a[1][1]==0&&a[1][2]==0&&a[2][1]==0&&a[2][2]==0){
			printf("0\n");//矩阵中全是0时的情况
			continue;
		}
		puts("1");//剩下的方案
	}
	return 0;
}
```
