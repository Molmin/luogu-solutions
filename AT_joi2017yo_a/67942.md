# 或许是目前最短的代码了吧（笑）
## 这是一道纯模拟题，按照题意模拟即可。
### 上代码：
```cpp
#include<cstdio>
main(){
	int a,b,c,d,e;
	scanf("%d%d%d%d%d",&a,&b,&c,&d,&e);
	printf("%d\n",a<=0?-a*c+d+b*e:(b-a)*e);
    /*若a<=0，则需要在零度以下加热-a次，每次需要c秒，在零度需要d秒，
      在零度以上需要加热b次，每次需要e秒。
      若a>0，则需要加热b-a次，每次需要e秒。
      此处使用三目运算符，省去if语句，所以码量少（笑）*/
}
```
