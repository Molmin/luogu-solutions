暴力枚举因为它是$  100 $~ $105 $，太大了，于是……完全可以/=100后~~再暴力枚举嘛~~

我去……都知道/=100了还枚举个啥啊……

那么$ x/=100 $后这个数就是我们的饭团个数啊

于是剩下了$ x  mod  100$这么多钱

~~我们不难发现~~ ，当$  100 $~ $105$变为$  0 $~ $5$时，如果$ x  mod  100$比我们能够承受的还多或者$ =$ ，再见，0走人。否则就是1

代码如下：

```
#include<bits/stdc++.h>
using namespace std;
long long x;
int main()
{
	cin>>x;
	int xa=x/100;
	int xb=x%100;
	if(xa*5>=xb)cout<<1<<endl;
	else cout<<0<<endl;
	return 0;
}
```
