### 2.解法:
看到题目中给出的 $n$ 非常大 $(\le 2\times10^9)$，所以大概率存在 $O(1)$ 做法。

看到每个人可以取 $1\sim9$ 个石子，考虑匹配。

简单思考后发现可以分成两类进行讨论：

第一种情况, $n \ \bmod 10 = 0$ 时，后手有必胜策略，即先手取 $x$ 个石子，后手取 $10 - x$ 个石子，这样一回合下一共拿走了 $ 10$ 个石子，又 $n \ \bmod 10 = 0$，所以一直可以取到最后的石子，故后手必胜。

第二种情况，$n \ \bmod 10 \neq 0$ 时，先手有必胜策略，即先手取 $n \ \bmod 10$ 个石子，就可以先手变为后手，回到情况一。

故思路如下，先判断 $n$ 是否可以被 $10$ 整除，如果可以，输出"2"；如果不可以，输出"1"，再输出 $n \ \bmod 10$。

### 3.代码：
``` cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
	int n;
	cin >> n;
	int tmp = n % 10;
	if(tmp)
	{
		cout << 1 << endl << tmp;
	}else
	{
		cout << 2;		
	}
 	return 0;
}

```
