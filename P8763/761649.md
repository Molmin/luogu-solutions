这个题目如果单纯模拟去做的话，大概过一半的样例可以通过，但剩下的就会超时。这种情况往往是有规律可循的，我们拿几个较短的字符串去模拟这个过程：

输入 $n=5,t=8$，不难发现，从第 4 次就开始循环了。

$0. 01101$

$1. 01011$

$2. 01110$

$3. 01001$

$4. 01101 \leftarrow$

$5. 01011$

$6. 01110$

$7. 01001$

$8. 01101 \leftarrow$

那么，最终的规律是当大于等于 $n$ 的一个 2 的整数次幂情况下，必定存在循环。

也就是满足 $x=2^m$，$x \geq n$，在这个 $x$ 范围内，必定存在一个循环。

AC CODE

```c
#include<stdio.h>
long long n,t;
int x=1;
char s[10005];
int main(){
	scanf ("%lld%lld",&n,&t);
	scanf ("%s",s);
	while(x<n)x<<=1;t=t%x;
	for(int i=0;i<t;i++)
		for(int j=n-1;j>=1;j--)
			s[j]=(s[j]-'0')^(s[j-1]-'0')+'0';
	printf ("%s",s);
	return 0;
}
```
