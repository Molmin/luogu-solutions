[传送门](https://www.luogu.com.cn/problem/CF1030C)

题目大意：给定一个的数列，要求是否可以将数列分成两段或多段，使得这些段相加都相等。

读完题，我们不难发现，为了把数列分成若干段，且每段是数字之和相等，我们可以用数字总和去除以段数，就能求出每段数的数字和。而段数就可以一一枚举，再详细判断。用第一个样例举例：

```cpp
5
73452
```
我们首先统计数字和,再枚举段数。

| 段数 | 2 | 3 | 4 | 5 | 6 | 7 |
| :----------: | :----------: | :----------: | :----------: | :----------: | :----------: | :----------: |
| 每段和 | 10.5 | 7 | 5.25 | 4.2 | 3.5 | 3 |
| 是否成立 | × | √ | × | × | × | × |

枚举完后，我们发现有一种方法可行，所以输出对。

# Code：

```cpp
#include <bits/stdc++.h> //万能头
using namespace std;
int n, c[105];
string a; //数列
int main() {
	cin >> n;
	cin >> a;
	int sum = 0;
	for (int i = 0; i < n; i++)c[i + 1] = int(a[i]) - 48; //转化为数组
	for (int i = 1; i <= n; i++)sum += c[i]; //总和
	for (int i = 2; i <= n; i++) {
		if (sum % i == 0) {
			int mdh = 0,md = sum / i;
			for (int j = 1; j <= n; j++) {
				mdh += c[j];
				if (mdh == md)mdh = 0; //达到段和，重新开始
				if (mdh > md)break; //超过段和，直接排除
				if (j == n) { //全部分完，符合题意
					cout << "YES";
					return 0;
				}
			}
		}
	}
	cout << "NO";
	return 0;
}
```
