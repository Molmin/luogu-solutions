题目转换：
有一条数轴，上面有 $N$ 个点。现求一点 $M$，使得所有点到其的距离最短。

思路：此题就是求中位数。推导如下：

$1$：设三点 $A$，$B$，$C$ 表示 $1$，$2$，$3$，则 $M$ 位于 $2$ 的位置。

$2$：设四点 $A$，$B$，$C$，$D$ 表示 $1$，$2$，$3$，$4$，则 $M$ 位于 $2.5$ 的位置。

$3$：设十点 $A$，$B$，$C$，$D$，$E$，……，$J$ 表示 $1$，$2$，$3$，$4$，$5$，……$10$，则 $M$ 位于 $5.5$ 的位置。

……

则我们可以推出以下结论：在一条数轴上，对于表示 $a_{1}$，$a_{2}$，$a_{3}$，……$a_{n}$ 的 $n$ 个点（$a_{1}\le a_{2}\le a_{3}\le ……\le a_{n}$），其对应的 $M$ 值为 $a_{(1+n)/2}$。

code:
```cpp
#include<bits/stdc++.h>//万能头
using namespace std;
int a[505];
int main(){
	int t;
	scanf("%d",&t);
	while(t--){//t组数据
		int n;
		scanf("%d",&n);
		for(int i=1;i<=n;i++)scanf("%d",&a[i]);
		sort(a+1,a+n+1);//排序是为了找最中间的数（即中位数）
		int mid=(1+n)/2;
		int ans=0;
		for(int i=1;i<=n;i++)ans+=abs(a[mid]-a[i]);//累加答案
		printf("%d\n",ans);//输出
	}
	return 0;
}
```
