一、题意

这道题提供一下大致翻译：有一个人（姑且叫他小明）和他的亲戚们住在一条大街上，因为小明会经常去拜访他所有的亲戚，所以他想尽可能缩短与亲戚们的总距离。

二、思路

首先我们要对输入进来的位置进行一个排序。利用 sort 函数，我们就可以得到这个序列的从小到大的顺序，方便我们操作。

这道题最关键的地方就在于小明在哪里？在数学中，我们知道一条直线上有 $n$ 个点，要使一个点和其他点的总距离最短，那么就是这些点的中间 $1$ 个或 $2$ 个（取决于点的数量），具体见[这个](https://zhidao.baidu.com/question/586230811.html)。通过这个突破口，我们就可以计算总距离了，计算方法为遍历数组，用这个中位数去减每一个数，记得用绝对值，否则会出现负数。最后输出总距离 $sum$ 即可。


三、代码实现

```cpp
#include<iostream>
#include<cmath>
#include<algorithm>
using namespace std;
int main(){
    int a,n,b[508] = {0};//注意数组要初始化为0
	cin>>a;
	for(int i=1;i<=a;i++){
		cin>>n;
		for(int i=1;i<=n;i++){
		    cin>>b[i];    
		}
		sort(b+1,b+n+1);
		int mid=(n+1)/2;//找出中位数
		int sum=0;
		for(int i=1;i<=n;i++){
		    sum+=abs(b[mid]-b[i]);//注意要用绝对值，不然可能是负数
		}
		cout<<sum;
	}
	return 0;//好习惯
}
```
