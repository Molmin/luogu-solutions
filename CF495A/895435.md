**一道水题**

根据乘法原理，总情况数就等于十位情况数乘各位情况数。

我们还要枚举下每种数字的所有情况

$0$ 有 $0$，$8$。

$1$ 有 $1$，$3$，$4$，$7$，$8$，$9$，$0$。

$2$ 有 $2$，$8$。

$3$ 有 $3$，$8$，$9$。

$4$ 有 $4$，$8$，$9$。

$5$ 有 $5$，$6$，$8$，$9$。

$6$ 有 $6$，$8$。

$7$ 有 $0$，$3$，$7$，$8$，$9$。

$8$ 有 $8$。

$9$ 有 $8$，$9$。

## 代码

```cpp
#include <cstdio>
using namespace std;
int n,a[10]={2,7,2,3,3,4,2,5,1,2};
int main(){
	scanf("%d",&n);
	printf("%d",a[n%10]*a[n/10]);
	return 0;
}
```
