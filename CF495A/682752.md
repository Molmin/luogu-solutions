**题目大意：**

给你一个电梯上显示的两位数，有一些发光条坏掉了，求原来数字有多少种可能。

**分析：**

这道题比较简单，不难发现：

$0$ 只能变成 $0$ 或 $8$，共 $2$ 种；

$1$ 可以变成 $0$ 或 $1$ 或 $3$ 或 $4$ 或 $7$ 或 $8$ 或 $9$，共 $7$ 种；

以此类推，$\cdots \cdots$

**代码：**

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,f[10]={2,7,2,3,3,4,2,5,1,2};//打表得到每一个数字可变成的数字个数
int main(){
	cin >>n;
	cout<<f[n%10]*f[n/10];
	return 0;
}
```
