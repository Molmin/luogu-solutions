# 题意
电梯坏掉了，几个地方可能没发光，请你算出楼层一共有多少种可能。
# 分析
由枚举可以得出：

$0$ 可能是 $0$，$8$。

$1$ 可能是 $1$，$3$，$4$，$7$，$8$，$9$，$0$。

$2$ 可能是 $2$，$8$。

$3$ 可能是 $3$，$8$，$9$。

$4$ 可能是 $4$，$8$，$9$。

$5$ 可能是 $5$，$6$，$8$，$9$。

$6$ 可能是 $6$，$8$。

$7$ 可能是 $0$，$3$，$7$，$8$，$9$。

$8$ 可能是 $8$。

$9$ 可能是 $8$，$9$。

于是只用将十位和个位两个数字能够变成其他数字的可能数相乘即可得出答案。

# Code
```
#include<bits/stdc++.h>
using namespace std;
int n,a[11]={2,7,2,3,3,4,2,5,1,2};
int main(){
	cin>>n;
	cout<<a[n%10]*a[n/10];
	return 0;
}
```
