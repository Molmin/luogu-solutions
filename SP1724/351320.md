### 思路：

![](https://cdn.luogu.com.cn/upload/image_hosting/7xl78n92.png)

如上图。

图一只有一个三角形，我们把它看成 $1^2$，算式：$1^2=1$。

图二共有五个三角形，其中有四个是 $2^2$，再加上（图一），算式：$1^2+2^2=5$。

图三共有十三个三角形，其中有九个小三角是 $3^2$，加上（图二），再减去一个不完整的倒三角，正好是十三个。

图四共有二十七个三角形。其中有十六个三角形是 $4^2$，加上（图三），再减去两个倒三角。


由此我们可以找出规律正三角个数就是平方和，即 ```n(n+1)(n+2)/6```，而倒三角分两种：

当层数为奇数时：倒三角个数为 ```(n-1)(n+1)(2n+3)/24```（层数为$n$）。

当层数为偶数时：倒三角个数为 ```n(n+1)(4n+5n)/6```（层数为$n$）。

代码就很好打了。

### $Code$：

```cpp
#include <bits/stdc++.h>
using namespace std;
long long t;
long long n;
long long ans=0;
int main() {
	scanf("%lld",&t);
	while(t--) {
		scanf("%lld",&n);
		ans += n*(n+1)*(n+2)/6;
		if (n%2==1) ans += (n-1)*(n+1)*(2*n+3)/24;
		else ans += n*(n+2)*(2*n-1)/24;
		printf("%lld\n",ans);
		ans=0;
	}
	return 0;
}
```
