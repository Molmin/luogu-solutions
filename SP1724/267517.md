```
#include<bits/stdc++.h>
using namespace std;
long long n,s;
int main() {
	scanf("%ld",&n);
	for(long long i=1; i<=n; ++i) {
		long long a;
		cin>>a;
		if(a%2==0)
			cout<<a*(a+2)*(a*2+1)/8<<endl;
		else cout<<(a+1)*(2*a*a+3*a-1)/8<<endl;

	}
	return 0;
}
```
总之一句话：分正立和倒立的三角形处理

正立的三角形个数：$n*(n+1)*(n+2)/6$~~这不是小学奥数的平方和公式吗~~

倒立的：

- 当n为奇数时，sum=$[2*3+4*5+6*7+...+n*(n-1)]/2$=$[(n-1)*(n+1)(2n+3)]/24$

- 当n为偶数时，sum=$[1*2+3*4+5*6+...+n*(n-1)]/2$=$[n*(n+2)(2n-1)]/24$

再把奇数和偶数的sum分别与正立的三角形加起来,得到：
```cpp
a*(a+2)*(a*2+1)/8<<endl;//偶数
(a+1)*(2*a*a+3*a-1)/8<<endl;//奇数
```
