## 首先，仔细看题
是不是感觉似曾相识
您的任务非常简单。您要做的就是计算最大三角形$n$中的所有三角形。这是不是小学奥数？是的

公式：
$n*(n+1)*(n+2)/6n∗(n+1)∗(n+2)/6$；这是为正立的
倒立

 1.当$n$为奇数时，$sum=[2*3+4*5+6*7+...+n*(n-1)]/2[2∗3+4∗5+6∗7+...+n∗(n−1)]/2=[(n-1)*(n+1)(2n+3)]/24[(n−1)∗(n+1)(2n+3)]/24$



 2.当$n$为偶数时，$sum=[1*2+3*4+5*6+...+n*(n-1)]/2[1∗2+3∗4+5∗6+...+n∗(n−1)]/2=[n*(n+2)(2n-1)]/24[n∗(n+2)(2n−1)]/24$

知道公式也就简单多了

### 代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
long long n,s;
int main() {
	scanf("%ld",&n);
	for(long long i=1; i<=n; ++i) {
		long long a;
		cin>>a;
		if(a%2==0)
			cout<<a*(a+2)*(a*2+1)/8<<endl;
		else cout<<(a+1)*(2*a*a+3*a-1)/8<<endl;

	}
	return 0;
}
```
