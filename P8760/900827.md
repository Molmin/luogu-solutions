# 问题A：
只需将 $11111111$ 将二进制转换成十进制即可，得到答案为 $255$。
# 问题B：
从网上找到1MB = 8Mbps，所以200Mbps = 25MB；答案为 $25$。
# 问题C:
思路很简单，就是枚举 $11$到 $20210605$ 所有的数，判断其是否为质数，再进行数位分离。答案为 $1903$。
# 问题D：
直接枚举 $2001-01-01$ 到 $2021-12-31$，每次进行数位分离，并判断数位分离后的和是否为完全平方数，答案为 $977$。
# 问题E：
简单的动态规划，千万别以为是二叉树，直接推公式就行，答案为得出答案 $2653631372$。

$CODE$:

```
#include<iostream>
using namespace std;
int main() {
    string ans [] = {
        "255",
        "25",
        "1903",
        "977",
        "2653631372",
    };
    char T;
    cin >> T;
    cout << ans[T - 'A'] << endl;
    return 0;
}
```
