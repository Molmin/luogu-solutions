这道题有一点点的复杂，如果你想让代码尽可能地短，那么你的**逻辑必须要清晰**。不然需要调试很多次！！！

1. 输入n
1. 先从第1行开始，**运算2*n+1行**
1. **每一行先输出空格，再输出数字，记住数字大小变化是凸字状的！！**

### 这道题我用了初中知识：绝对值

大家都知道，**绝对值的几何意义是点到点在数轴上的距离，那么我们可以利用这一点来解这道题！！！**

比如：
|  0| 1 | 2 |3  |2  |1  |0  |
| -----------: | -----------: | -----------: | -----------: | -----------: | -----------: | -----------: |
| 1 |2  |3  |4  |5  |6  |7  |


我们可以先找到最中间的数字指针为4，计算每一个数字到最中间数字的距离，即
| 0 |1  |2  |3  | 2 | 1 |0  |
| -----------: | -----------: | -----------: | -----------: | -----------: | -----------: | -----------: |
|1  |2  | 3 |  4| 5 | 6 | 7 |
|3  |2  |1  |0  |1  | 2 |  3|
再用4-1=3减去：
| 0 | 1 | 2 |3  |2 |1  |0  |
| -----------: | -----------: | -----------: | -----------: | -----------: | -----------: | -----------: |
| 1 |2  | 3 |4  |  5| 6 |7  |
| 3 | 2 | 1 |0  |1  |2  | 3 |
|0  | 1 |  2| 3 | 2 | 1 | 0 |



下面奉上AC代码以及解说：
```cpp
#include <iostream>
#include <cmath>
using namespace std;
int main(){
	int i,j,n;
	cin>>n;
	for(i=1;i<=2*n+1;i++){//我卡了好多次就是一开始写成pow(2,n)+1了，监介
		for(j=1;j<=abs(n-i+1)*2;j++)cout<<" ";//计算空格，也运用绝对值的方法
		for(j=1;j<=(n-abs(n-i+1)+1)*2-1;j++){//运用绝对值的方法
			if(j<=((n-abs(n-i+1)+1)*2-1)/2)cout<<j-1<<" ";//此处需要判断是否为中间数之前还是中间数之后
			else{
			    if((n-abs(n-i+1)+1)*2-1-j>0)cout<<(n-abs(n-i+1)+1)*2-1-j<<" "; //判断是否是最后一位
			    else cout<<0;//最后一位输出0，不加空格
			}
		}
		cout<<endl;//换行
	}
	return 0;
}
```
