### 思路
---     
我将输出分 $2$ 个部分,因为输出的数字从少变多又变少。  
放在一起比较麻烦，所以我们分开输出,第一部分：（上面一半及中间一行）。  
空格越来越少，而数字越来越多。   
循环判断每行需要输出的数字。  
行数乘以 $2$ 加 $1$ 就是需要输出数字的个数。  
模拟输出空格，以及间隔空格。    
再分 $4$ 种情况：   
列数小于输入行数并且不等于行数乘以 $2$，输出：列数及间隔空格。  
列数小于等于行数，输出：列数。  
列数不等于行数乘以 $2$，输出：行数乘以 $2$ 减去列数及间隔空格。  
其他所有情况，输出：行数乘以 $2$ 减去列数。    
第二部分：（下面一半）。  
与第一部分同理。     
#### 注意点 ：  
每两个输出之间有间隔空格。    
在输出数字时要 $+1$。  
循环判断时：  
乘 $2$：我们要输出递增和递减 $2$ 种情况，加 $1$ 是中间的那个数字。   
### 全代码
---
```
#include<bits/stdc++.h>
using namespace std;
int main(){
   int n;
   cin>>n;
   for(int i=0 ; i<n+1; i++){
       for(int j=0; j<i*2+1; j++){
            for(int k=i*2 ; j==0&&k<n*2;k++{ 
                cout<<" ";
            }
            if(j<=i&&j!=i*2){
               cout<<j<<" ";
            }else if(j<=i){
               cout<<j;	
            }else if(j!=i*2){
               cout<<i*2-j<<" ";
            }else {
               cout<<i*2-j;
            }
       }
       cout<<endl;
   }
   for(int i=0 ; i<n; i++){
       for(int j=0; j<(n-i)*2-1; j++){
           for(int k=0;j==0&&k<(i+1)*2;k++){
               cout<<" ";
           }
           if(j<=n-i-1 && j!= (n-i)*2-2){
              cout<<j<<" ";
           }else if(j<=n-i-1){
              cout<<j;
           }else if(j!=(n-i)*2-2){
              cout<<(n-i -1)*2 -j <<" ";
           }else{
              cout<<(n-i-1)*2-j;
           } 
       }
       cout<<endl;
   }
   return 0;
}
```
