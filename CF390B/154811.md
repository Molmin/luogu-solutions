------------

这道题的意思大致是：两个小朋友演奏音符，给你一个数n和两个数组，每个数组中有n个数，第一个数组中，第i个数a[i]代表两个人能弹出的最高音调。**如果两个人弹出的音符x,y（x,y<=a[i]）的总和等于第二个数组对应的音调b[i]，他们能获得x\*y点愉悦值，反之，他们的愉悦值减一**。输出最大的愉悦值。


------------

这道题有很多要注意的地方,详见代码：

```cpp
#include<iostream>
#include<cctype>//快读中isdigit头文件
#define ULL unsigned long long//简化unsigned long long,方便定义
#define LL long long//简化long long
const int N=10000001;
int a[N],b[N];//定义数组
LL s=0;//注意点1：最终的结果是要开long long的（不然你就会在第六个测试点WA）
int pan(int x,int y){//判断弹出的音符能否达到标准音调
    if(y==1)//注意点2：两个人每个人都必须要弹！所以在标准音符为1的情况下是不可能实现的（因为不可能弹出0的音调，相当于没弹）。
    return 0;
    else if(x*2<y)//因为x是两个人能弹出的最高音调，如果两个人的最高音调加起来（也就是乘以2）都不能达到标准音调，返回0
    return 0;
    else return 1;//如果可以弹出标准音调，返回1
}
inline int read(){//快读，比cin和scanf都快
    int X=0,w=0; char ch=0;
    while(!isdigit(ch)) {w|=ch=='-';ch=getchar();}
    while(isdigit(ch)) X=(X<<3)+(X<<1)+(ch^48),ch=getchar();
    return w?-X:X;
}
int main(){
    int n=read();//用快读读入n
    for(int i=1;i<=n;i++)
    a[i]=read();//读入a数组
    for(int i=1;i<=n;i++)
    b[i]=read();//读入b数组
    for(int i=1;i<=n;i++){
        if(pan(a[i],b[i])==1){//如果两个人弹出的音调能达到标准音符
            ULL k=b[i]/2;//注意点3：因为要求最大的愉悦度，而两个人弹出的音调都不能超过标准音调，这时最大的愉悦度为(b[i]/2)*(b[i]/2),考虑到奇数，所以写成(b[i]/2)*(b[i]-b[i]/2)。
            s+=(k*(b[i]-k));//加上最大的愉悦度
        }
        else --s;//否则，愉悦度减少一
    }
    std::cout<<s;//输出最大愉悦度
    return 0;//养成好习惯
}
```


------------

注：可能有很多人会问：在判断能不能弹出标准音调这里，题目要求等于标准音调才有愉悦感，为什么不是2*x!=y返回0,超过y也应该返回0啊。因为题目说a数组里储存的是两个小朋友能弹出的**最大音调**（注意是最大），意味着在这个数之前的音调都能弹出来（除零以外）。如果两个小朋友能弹出的最高音调之和超过标准音调，这代表他们是有能力弹出标准音符的，只要某个人的音调减少一点，他们照样可以弹出标准音符。
