题意：给一个数，如果能整除十七输出一，不然输出零。

看到数据范围就一定是道高精题了，那么对于我这种不会写高精的废物来讲，只能用自带高精了，我用的是比较好写也易懂的~~蟒蛇~~，那么此题就变成一道入门题了，只需读入，然后判断一下输出即可( 

既然题目这么简单，那么代码也不会长。建议评为橙题。

```python
n = int(input())#读入n，转为int类型
while n != 0:#读到0为止
    if n % 17 == 0:#如果整除输出一
        print(1)
    else:
        print(0)#不然输出0
    n = int(input())#继续读入
```
