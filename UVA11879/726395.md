## **思路**
有一个神奇的算法：每次把输入的数字末尾砍掉，把剩下的数减去砍掉的末尾的 $5$ 倍，若剩下的数处理完后被 $17$ 整除，则原数也被 $17$ 整除。下面是核心代码：
```cpp
int a[105] , len = 0;
while ( cin >> a[len] )
len++;
for ( int i = len - 1; i > 1; i++ )
{
    int h = 0;
    if ( a[i - 1] >= ( a[i] * 5 ) % 10 )
    a[i - 1] -= ( a[i] * 5 ) % 10;
    else
    {
        a[i - 1] -= ( a[i] * 5 ) % 10 - 10;
        h++;
    }
    a[i - 2] -= 10 - ( ( a[i] * 5 ) / 10 + h );
}
int ans = abs ( a[0] * 10 + a[1] );
if ( ans % 17 == 0 )
cout << 0 << endl;
else
cout << 1 << endl;
```
