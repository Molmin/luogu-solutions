## 分析

看完题面，在看数据，数据说 $n$ 小于最小的$10$位质数....

~~结论题，确定！~~

看到这道题目可能最开始无从下手，没关系，我们来臆想一下。

假设现在有 $n$ 个~~摔跤~~拳击手，要求用最短的比赛次数，选出冠军和亚军。

（是不是很有画面感了）

我们怎么做最优呢？一定是把其中每两个人给组起来，让他们 ~~摔跤~~PK，然后选出优胜者。

我们再从优胜者里面选，每两个人组起来，再让他们 PK， 再选出优胜者，以此类推....

很明显，这样只需要比较 $log(n-1)$ 次就可以找出冠军。但是我们还要找亚军，这样就还得在剩下的 $n-1$ 个人之间比较，这样才能找出亚军。

显然，最后的答案就为

$$Ans = n - 1 + log_2(n - 1)$$

而$c++$的$cmath$库里正好友情提供了$log2$函数，直接拿出来用就好了。而且要记得给$log2$强转 $int$ ，不然会吃大亏（别问我怎么知道的qaq）

## $Code$

```cpp
#include<bits/stdc++.h> 
#define orz 0
#define inf 0x3f3f3f3f
using namespace std;

int n;

int main()
{
    while (~scanf("%d", &n))
        printf("%d\n", n - 1 + (int)log2(n - 1));
    return orz;
}
```
