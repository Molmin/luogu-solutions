# 思路——数学

## 题面描述

有 $n$ 个确定不变但未知的正整数，你每次询问可以选取两个数并得知两数的大小关系。

问在最坏的情况下，至少需要几次询问，才能知道最大数和次大数是哪两个数。

## 申明

本文中使用 $\log_n(x)$ 意思为找到 $n^y=x$ 中的 $y$。

而 $⌊ y ⌋$ 是去比 $y$ 小的最大整数，高斯函数，比如在 c++ 中 ```floor(y)```。

## Part 1

- 求出最大数

我们每次判断，如果 $i<j$ 那么我们就可以把 $i$ 删除，所以最后的次数为 $⌊\log_2{(n-1)}⌋$。

## Part 2

- 求出次大数

我们就需要知道每个数的数量关系，所以为 $n-1$。

# AC Code

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int x;
	while(cin>>x){
        cout<<floor(log2(x-1))+x-1<<endl;
    }
	return 0;
}
```
