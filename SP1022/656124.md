## 题目大意
最近，许多战士在一场比赛中丧生。战斗不再被认为是残酷的，所以所有的战士都应该在天堂里有一席之地。然而，由于空间短缺，他们可能无法在天堂获得一席之地。战士们仍然对彼此怀恨在心，所以一个战士不能被放在一个他可以看到其他战士的房间里。战斗机只能看到四个基本方向(北、南、东、西)。他不能朝对角线或其他任何方向看。找到最大数量的战士谁可以有一个天堂的房间。题目会给出 T 组数据，每组数据第一行包含两个数 $n$ 和 $m$，接下来会读入 $n$ 行 $m$ 列的字符矩阵，字符一定是以下 $3$ 种之一:

1. H，表示此位置是空的。

2. A，表示此位置已经被天使占据，人们无法看到在这些位置后的位置。

3. D，表示此位置是地狱，不能容纳任何东西。

## 分析
依次枚举位置，如果这个位置可以容纳人且不会被别人看到，就将人数加 $1$，并标记四个方向可看到的位置（看到 A 之前的位置）。

## 上代码
```c
#include<bits/stdc++.h>
using namespace std;
int t,n,m,b[1010][1010];
char a[1010][1010];
void bj(int x,int y)
{
    for(int i=1;i<=n;i++)
	{
        if(a[i][y]=='A') break;
        b[i][y]=1;
    }
    for(int i=1;i<=m;i++)
	{
        if(a[x][i]=='A') break;
        b[x][i]=1;
    }
}
int main()
{
    cin>>t;
    while(t--)
	{
        cin>>n>>m;
        for(int i=1;i<=n;i++) 
			for(int j=1;j<=m;j++)
				cin>>a[i][j];
        int gs=0;
        for(int i=1;i<=n;i++)
        {
        	for(int j=1;j<=m;j++)
                if(b[i][j]==0&&a[i][j]=='H')
				{
                    bj(i,j);
                    gs++;
                }
        }            
        cout<<gs<<endl;
        memset(b,0,sizeof(b));
    }
    return 0;
}
```
