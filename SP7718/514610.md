题意：给定两个整数，求他们的公因数个数。

思路：由于题目的时间限制只有仅仅六百毫秒，所以我们可以先求两数的最大公因数，然后枚举。但是不能用两层循环，否则会超时。我们可以只枚举一个变量，若最大公因数可以被它整除，那么答案就加二。但是要判断取余它后是否也等于它，如果是，那么答案就只加一。这里求最大公约数时使用的函数是：

``__gcd(a,b)``

含义是求这两个数的最大公约数。

注意：注意输入输出（见代码）。

代码（附解释）：

```cpp
#include<bits/stdc++.h>
using namespace std;
int t,a,b,ans;
int main()
{
    scanf("%d",&t);
    while(t--)
    {
        ans=0;
        scanf("%d%d",&a,&b); // 读速较快，避免超时，下同
        int tp=__gcd(a,b);
        for(int i=1;i*i<=tp;i++)
            if(tp%i==0) // 若可被整除
                if(i!=(tp/i)) ans+=2; //而且值不相同，答案加二
	        else ans++; // 否则只加一
        printf("%d\n",ans);
    }
    return 0;
}
```
