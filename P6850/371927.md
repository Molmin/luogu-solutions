此题的关键就是在题目的理解，我们要搞清楚如何计算得分，才能判断进没进队。总得分有好几部分，我们一个个来看。

首先，笔试分。因笔试每答对一题给一分，况且又五十分底分，我们仅需把第一个输入的数，也就是答对的题目数与底分加起来，就完成了第一部分的得分，笔试分的计算。注意！一定要加底分！

我们接下来看第二部分，题目得分。题目得分没什么好说的，直接把第一天一二三题的得分与第二天一二三题的得分加起来即可，题目也说得很清楚。

第三部分是 $A$ 类的分，仅需一个判断决定是否加五分。

最后，将几部分分加起来如大于进队的分就输出`AKIOI`，反之亦然。

```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int d1,d2,d3,d4,d5,d6,d7,f,i,ans=0;
int main(){
    cin>>d1>>d2>>d3>>d4>>d5>>d6>>d7>>f>>i;//输入
    ans=d1+d2+d3+d4+d5+d6+d7+50;//计算每部分答案
    if(f==1)//A类
        ans=ans+5;
    if(ans>=i)//判断是否进队
        printf("AKIOI");
    else
        printf("AFO");
    return 0;
}

```
