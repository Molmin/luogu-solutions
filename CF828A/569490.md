## CF828A 题解

本题其实并不难，按照题意进行模拟即可。

先定义两个变量 $ a $ ， $ b $ 。表示有 $ a $ 个单人桌和 $ b $ 个双人桌。然后再定义三个变量 $ c $ ， $ d $ ， $ e $ ，分别表示还有 $ c $ 张空的单人桌， $ d $ 张还剩一个座位的双人桌， $ e $ 张空的双人桌。然后我们再按操作进行模拟即可。

这些已经有些题解写过了，我再说说部分在第 12 个点 WA 掉的人是怎么错的。

很多人只定义了两个变量，一个表示空的单人桌还剩几个，一个表示空的双人桌，而还剩一人的双人桌就变成了单人桌，但是空的双人桌的优先级比还剩一人的双人桌的单人桌高，可是一旦合并空的双人桌的优先级却比还剩一人的双人桌的单人桌优先级低了。

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a,b,c,d,e,come,s;//c张单人桌，d张还剩一人的双人桌，e张空的双人桌 
int main(){
	scanf("%d%d%d",&n,&a,&b);
	c=a,e=b;
	for(int i=1;i<=n;i++){
		scanf("%d",&come);
		if(come==1){//当只来了一人
			if(c)c--;
			else if(e)e--,d++;
			else if(d)d--;
			else s++;
		}
		else{//来了两人
			if(e)e--;
			else s+=2;
		}
	}
	printf("%d",s);
	return 0;
}
```
