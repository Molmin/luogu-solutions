# 题目大意
给你 $T$ 组样例，每组样例中有一个正整数 $n$，现在让你求出满足   $34^n+30n+32+m \equiv 0\pmod {11}$ 最小的自然数 $m$。


------------
# 分析
这道题因为 $1 \leq n \leq 10^{18}$ 这个数据范围太变态了，而且高精度也行不通，所以我们只能使用数学方法了！首先我们先看看 $34^n$ 除以 $11$ 的余数，根据余数的乘方性，可以得出余数为 $(34\bmod11)^n$，即为 $1$；然后再看看 $30n$，它的余数为 $(30\bmod11)\times (n\bmod11)$；
最后 $32\bmod11$ 也就是 $10$ 了！最后将所有余数通过余数的可加性累加起来，并用 $11$ 减去总余数即为答案，但要注意**如果余数为 $0$ 的话那么答案也为 $0$**。

------------
# 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int T;
long long n;
int main(){
	cin>>T;
	while(T--){
		cin>>n;
		long long r=(1+(30%11)*(n%11)+32%11)%11; //求余数 
		cout<<(r==0?0:11-r)<<'\n'; //判断余数是否为0 
	}
	return 0;
} 
```
