### 一、思路

首先只考虑第一个字母。

显然，以任意一个字母开头的词，共有 $25$ 个。

因此，我们先看看在字典序中，这个词语的第一个字母之前有多少个字母，设为 $a$。

比如，如果第一个字母为 `c`，那么之前有 $2$ 个字母，分别是 `a` 和 `b`。$a=2$。

然后，我们再看看第二个字母。这就需要分类讨论了。

如果第二个字母的编号小于第一个字母，那么将 $b$ 设为这个字母在所有小写字母中排第几位。否则，将 $b$ 设为这个字母在所有小写字母中的排位**减一**。

### 二、完整代码

```cpp
#include<cstdio>
using namespace std;
int t;
char s[5];
int main()
{
	scanf("%d",&t);
	while(t--)
	{
		scanf("%s",s+1);
		int ans=0;
		ans+=(s[1]-'a')*25;
		if(s[2]<s[1])
		{
			ans+=(s[2]-'a'+1);
		}
		else
		{
			ans+=(s[2]-'a');
		}
		printf("%d\n",ans);
	}
	return 0;
}
```
