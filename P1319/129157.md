现简介一下题目：

 	先输入n，n代表接下来输出矩阵的边长。然后是若干个数字，直到按
    回车为止。除去输入的n，第一个输入的数字便代表输出0的个数，第
    二个输入的数字便代表输出1的个数，第三个输入的数字便代表输出0
    的个数......以此类推，我们便得出“第奇数个输入的数字便代表输
    出0的个数，第偶数个输入的数字便代表输出1的个数”。知道了这点
    之后，打代码的思路就清晰多了。
	
顺便附上c++代码

-----------------------------------------我是~~华丽的~~分割线-----------------------------------------

```cpp
#include<iostream>
#include<cstring>
#include<cstdio>
using namespace std;
int main()
{
    int n,i=0,j,t=0,a;    	  //定义变量。
    cin>>n;             	  //输入n。
    while(scanf("%d",&a)!=EOF)    //while循环到输入完。
    {
        i++;                      //累加。
        for(j=1;j<=a;j++)	  //for循环a次。
        {
            if(t==n)              //判断是否要换行。
            {
            	cout<<endl;
            	t=0;		  //别忘了计数器归0。
	    }
	    if(i%2==0){           //判断输出1还是0.
                cout<<"1";
                t++;
	    }
            else {
		cout<<"0";
		t++;
	    }
        }
    }
    return 0;         		  //结束啦！
}
```
