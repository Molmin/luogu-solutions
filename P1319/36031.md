**【分析】**
------------
各位可以发现，假设将题目中的第一个变量(N)输入后，每次输入都要改变一次字符，并刷新次数。

假设我们初始化一个变量m维护输入的其他数据，变量b维护当前需要输出的字符。

那么，当m!=0时，我们输出字符，并m--

当m==0时，我们重新输入m，修改输出的字符，并输出

考虑到输出的字符无非就是0或1，且每次更新后就修改字符类型，因此，可以考虑使用"非"运算(b=!b)。若b之前为0，则修改为1；b之前为1，则修改为0

**重点来了，题目中并没有说明m是否可以为0，如果m为0，我们得循环执行m==0时的操作**

综上，如果我们不考虑回车的情况下，只要数据输入进来，程序就可以直接处理输出。

~~那么问题来了......回车怎么办......~~

实际上评测系统中输入和输出是分开的，也就是说，我们的输入和输出是两个文件，所以我们根本不需要考虑他们会不会串在一起。也就是说我们是可以直接输出的。

**【代码】**
------------

那本蒟蒻就放代码了，其他一些细节在代码内有解释。

```cpp
#include<cstdio>
#include<cctype>
using namespace std;
int read(){
	int abs=0;char c=getchar();while(!isdigit(c)) c=getchar();
	while(isdigit(c)) abs=abs*10+c-'0',c=getchar();
	return abs;
}//读入优化，考虑到输入量不确定多少
int main(){
	int n=read(),m=0;
	bool b=1;
	for(int i=0;i<n;i++){
		for(int j=0;j<n;j++){
			while(!m) m=read(),b=!b;
            //m==0时持续更新字符
			putchar(b?'1':'0');//b==1时输出1，否则为0
			m--;
		}
		putchar('\n');//满n个要输出一次回车
	}
	return 0;
}
```
