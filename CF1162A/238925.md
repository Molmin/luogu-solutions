### 题意：

有 $n$ 个建筑,每个建筑有一个最大的高度 $h$ ,接下来有 $m$ 个要求，每个要求需要让 $l$ 到 $r$ 这一段的建筑高度不超过 $x$ ,求最后所有建筑高度平方的最大值.

### 思路：

先将每一个建筑的高度都赋值为 $h$ ,这样就确定了每个建筑的最大最大值.

因为 $n<=50$ $,$所以接下来的 $m$ 个要求,我们都可以暴力去解决.每一次都让 $l$ 到 $r$ 这一段的建筑高度变为原来的和这次要求的更小的一个,即为$min(a[i],x);$

枚举一个样例 $:$

$4$ $10$ $2$

$2$ $3$ $8$

$3$ $4$ $7$

$1:$ $a$ $10$ $10$ $10$ $10$ 

$2:$ $a$ $10$ $8$ $8$ $10$ 

$3:$ $a$ $10$ $8$ $7$ $7$ 

则对后答案最大$:10*10+8*8+7*7+7*7=262$

$Code:$
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,h,m,l,r,x,ans,a[55];
int main(){
	scanf("%d%d%d",&n,&h,&m);
	for(int i=1;i<=n;i++)a[i]=h;
	for(int i=1;i<=m;i++){
		scanf("%d%d%d",&l,&r,&x);
		for(int j=l;j<=r;j++)a[j]=min(a[j],x); 
	} 
	for(int i=1;i<=n;i++)ans+=a[i]*a[i];
	printf("%d\n",ans);
	return 0;
} 
```
