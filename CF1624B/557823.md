这道题大意是指输入 $a,b,c$ 三个数，其中一个数乘上一个整数 $m$，让 $a,b,c$ 三个数变为等差数列。

既然是等差序列，不妨设公差为 $d$ ，首项为 $x$ 。那第二项就是 $x + d$ 了，第三项就是 $x + 2d$ 了。可以得到首项加第三项是第二项的两倍，可以随意固定两个数，进行求值。

讨论：

设未确定的数为 $t$ 。

当首项、第二项固定时，第三项就是两倍第二项减去首项，$2(x+d)-x=x+2d$，即 $t =2b-a$；

当首项、第三项固定时，第二项就是首项加第三项除以 2，$(2 x + 2 d ) \div 2 = x + d $，即 $t =( c + a )\div 2$；

当第二项、第三项固定时，首项就是两倍第二项减去第三项，$2 ( x + d ) - x - 2 d = x$ ，即 $t = 2b - c$ ；

这个时候只要判断 $t$ 是否整除相对应的 $a$ 或 $b$ 或 $c$ 了。


附上代码：
```cpp
#include <iostream>
using namespace std;
int main(){
    int t;
    cin >>t;
    while(t--){
        int a,b,c;
        cin >>a>>b>>c;
        if((2*b-c)>0&&(2*b-c)%a==0){
            cout<<"yes"<<endl;
        }else if((a+c)%(2*b)==0&&(a+c)>0){
            cout<<"yes"<<endl;
        }else if((2*b-a)>0&&(2*b-a)%c==0){
            cout<<"yes"<<endl;
        }else{
            cout<<"no"<<endl;
        }//精华部分
    }
}
```
