### 分析题目：

题目大意：判断长度为 $3$ 的序列是否存在一个正整数 $m$，序列中的某一个数乘上 $m$，使该数列成为等差数列。

所以这道题跟等差数列有关，主要涉及到等差数列的知识，比如 $a$，$b$，$c$，三个数，$(a+c) = 2 \times b$。

**做题思路：** 利用这个，我们就可以做题了，既然是其中一个数加上 $m$，那么不妨分类讨论，假设 $a$，是要乘上 $m$ 的数，那么就用 $2 \times b - c$ 来表示需改成等差数列以后的 $a$，判断这个数除以原本的 $a$ 以后求出的 $m$ 是不是整数，如果是就输出 `YES`,否则就试试是不是修改其他的数，如果都不行，最后输出 `NO`。 

**注意：**  
1. 求出修改完后的数之后还要判断，这个是不是正数。
2. 判断 $b$ 是否是修改的数时，求出 $b$ 修改完后的数，要取余 $2 \times b$ 这样得出的才是修改 $b$ 时的 $m$。

蒟蒻代码：

```cpp
#include<bits/stdc++.h>
using namespace std ;

int main(){
	int t ; 
	cin >> t ;
	while(t--){
		int a , b , c ;
		cin >> a >> b >> c ;
		if(b - a == c - b) cout << "YES" << endl ; //小特判，其实没有应该也没关系
		else if((2 * b - a) % c == 0 && (2 * b) - a > 0) cout << "YES" << endl ; //如果是修改c
		else if((a + c) % (2 * b) == 0 && (a + c) > 0) cout << "YES" << endl ; //如果是修改b
		else if((2 * b - c) % a == 0 && (2 * b) - c > 0) cout << "YES" << endl ; //如果是修改a
		else cout << "NO" << endl ; //如果都不行，就输出 NO 
	}
	return 0 ; 
}
```
