首先设这种数为n，所以$n^{3}$末尾三位为888，我们可以设$n^{3}$=A×1000+888，其中A为一个大整数（至于究竟有多大......），
根据样例，第一个满足要求的数为192，不妨设$192^{3}$为B×1000+888
因为它们末尾三位都是888，应该不难想到做相减操作，就能得到式子 
$n^{3}$-$192^{3}$=（A-B）×1000,
为1000即$2^{3}$×$5^{3}$的倍数。

根据立方差公式$a^{3}$-$b^{3}$=(a-b)×($a^{2}$+a×b+$b^{2}$)，又可以得到

$n^{3}$-$192^{3}$=(n-192)×($n^{2}$+192×n+$192^{2}$)=(A-B)×1000

接下来我们枚举一下n除以5的余数。

1、n$\equiv$0(mod5) ,此时 $n^{2}$+192×n+$192^{2}$ $\equiv$4(mod5)

2、n$\equiv$1(mod5) ,此时 $n^{2}$+192×n+$192^{2}$ $\equiv$2(mod5)

3、n$\equiv$2(mod5) ,此时 $n^{2}$+192×n+$192^{2}$ $\equiv$1(mod5)

4、n$\equiv$3(mod5) ,此时 $n^{2}$+192×n+$192^{2}$ $\equiv$4(mod5)

5、n$\equiv$4(mod5) ,此时 $n^{2}$+192×n+$192^{2}$ $\equiv$3(mod5)

好的，经过一番艰苦卓绝、锲而不舍、含辛茹苦.......的枚举，我们会发现这样一个惊人的事实：无论n除以5余几，$n^{2}$+192×n+$n^{2}$
除以5都不为0，也就是说，无论n取什么值，
$n^{2}$+192×n+$192^{2}$
都不为5的倍数，但是满足条件的n要使（n-192)×（$n^{2}$+192×n+$192^{2}$）
为1000的倍数，而1000是125也就是$5^{3}$的倍数，所以被125整除这样光荣（雾）的任务就交给了 n-192 ，也就是说满足条件的 n 必能满足 125|n-192 。

什么？你问我这就讨论完了吧，没可能的 ~~楼上找规律大佬都告诉你要讨论到250呀~~，毕竟1000=$5^{3}$×$2^{3}$,所以还要讨论除以2的余数。

我们假设n为奇数，那么 n-192就是奇数-偶数=奇数,$n^{2}$+192×n+$192^{2}$就是 
奇数+偶数+偶数=奇数。

那么此时（n-192)×（$n^{2}$+192×n+$192^{2}$）就是奇数，不可能为偶数1000的倍数，矛盾。

所以，n要为偶数，因为192是偶数，所以n-192也是偶数，又有 125|n-192 ,所以有 250|n-192 时满足条件，也就是说，第k个这种数可以表示为 250×（k-1)+192.

嗯，经过了我们的长篇大论，我们终于可以写出这样短小精悍的代码啦：
```cpp
//Tips:记得开long long。
#include<cstdio>
using namespace std;
long long k;
int main(){
	int t;
	scanf("%d",&t);
	while(t--){
		scanf("%lld",&k);
		printf("%lld\n",192+(k-1)*250); 
	}
}
```
