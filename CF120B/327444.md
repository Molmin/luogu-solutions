### 题目大意

给定一个 $n$ 个数的环，求第 $k$ 个数往后找到的第一个 $1$ 的位置。

### 思路分析

模拟一遍。如果过程中发现了当前的 $k>n$，就说明回到了原点，要将 $k$ 减去 $n$。如果当前位置为 $1$，就说明找到答案了，就输出下标即可。

### 温馨提示

这道题居然要用文件操作。

```cpp
freopen("input.txt","r",stdin);
freopen("output.txt","w",stdout);
```

### 核心代码

```cpp
while(!a[k])
{
	if(k>n)
		k-=n;
	k++;
}
```
