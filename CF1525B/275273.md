## 思路

简单结论题。

显然，如果数组已经排好序了，答案为 $0$；

如果第一个元素为 $1$ 或者最后一个元素为 $n$，答案为 $1$。例如：$1,3,5,2,4$，无论第 $2$ 到第 $5$ 个元素是按怎样的顺序排列，我们都可以将其变为 $2,3,4,5$。

如果最后一个元素为 $1$ 且第 $1$ 个元素为 $n$，答案为 $3$。例如：$5,3,4,2,1$ --> $2,3,4,5,1$ --> $2,1,3,4,5$ --> $1,2,3,4,5$。

否则答案为 $2$。例如：$2,5,3,1,4$ --> $1,2,3,5,4$ --> $1,2,3,4,5$。

        bool flag = 1;
        for (int i = 1; i <= n; ++i) {

            }

            cout << 0 << endl;
            continue;
        }

            if (a[1] != n) {

            } else {
                cout << 3 << endl;
            }
