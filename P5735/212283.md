首先，题目将提供 $3$ 个点的坐标，我们需要求出这个三角形的周长。乍一看这题，码量似乎不小，但是我们将它化为几个子问题，就简单了。

首先，介绍一个公式：

定义两个点 $a$ 和 $b$ , 坐标分别为 $x1,y1$ 和 $x2,y2$ 。那么 $a$ 到 $b$ 的距离即 $|a-b|$ 的值等于 $\sqrt{{(x1-x2)^2+{(y1+y2)^2}}}$

整理一下：

$\large|a-b|=\sqrt{{(x1-x2)^2+{(y1+y2)^2}}}$


那么，由于使用 $pow()$ 函数相对比较不稳定，我们采取最基本的方式：$a\times a=a^2$ ;

代码如下：

```
#include<bits/stdc++.h>
using namespace std;
double a,b,c,d,e,f;
double s1,s2,s3;
int main()
{
	cin>>a>>b>>c>>d>>e>>f;
	s1=sqrt((a-c)*(a-c)+(b-d)*(b-d));
	s2=sqrt((a-e)*(a-e)+(b-f)*(b-f));
	s3=sqrt((c-e)*(c-e)+(d-f)*(d-f));
	printf("%.2lf\n",s1+s2+s3);
	return 0;
}
```

当中的重点在于坐标值的一一对应，一开始我就 $Wa$ 了，就是因为 $a~f$ 的坐标值没有对应清楚。
