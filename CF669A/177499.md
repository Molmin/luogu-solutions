### **我们一起来找一下规律**
### 在题目中，我们可以得到以下的信息
### 问每次给个数,相邻的数不能一样,和为n,怎么才能使次数最多,最多为多少次;
### 正常人的第一反应是n*2/3
### 但由于题目中要的是最多方案
### ~~注意细节~~我们需要再加1
### 即为 n*2+1/3
# 代码如下
```cpp
//C++代码
#include<iostream>
using namespace std;
int main()
{
    double a;//正常情况下int也行，double以防万一
    cin>>a;   //输入
    a*=2;   //进行计算
    a+=1;  //C++的一种专门计算 eg：a+=1 == a=a+1
    a/=3;
    cout<<a;  //输出
    return 0;//NOIP别忘了
}
```
