这道题的意思想必大家已经清楚了，那这道题应该怎么做呢？

首先，题目给的 $7$ 个数是按单调不减排列的，所以前两个数一定是最小的那两个数。并且由于数组 $a$ 中的三个元素全是正整数，所以最小的两个数一定是数组 $a$ 中的元素（且为数组 $a$ 中最小的两个元素），最大的那个数一定是 $a$ 数组中 $3$ 个元素的和。

而由于最小的两个数一定是数组 $a$ 中的元素，所以通过给出的最小的两个数就可以知道数组 $a$ 中的两个元素。至于最后一个元素，就只需要将给出的最大的那个数减去已知的数组 $a$ 中的那两个数就可以求出来啦！

那么下面，就是大家最期待的代码环节啦！

```c++
#include<bits/stdc++.h>
using namespace std;
int n,i,j,a[12],b[8];
int main(){
    cin >> n;
    for(i = 1; i <= n; i++){
    	for(j = 1; j <= 7; j++){
    		cin >> a[j];
    	}
    	b[1] = a[1];//a 数组的第一个元素
    	b[2] = a[2];//a 数组的第二个元素
    	b[3] = a[7] - a[1] - a[2];//a 数组的第三个元素
    	for(j = 1; j <= 3; j++){
    		cout<< b[j] << " " ;
    	}
    	cout<< endl;
    }
    return 0;
}
```
