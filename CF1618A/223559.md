### 解题思路：

~~现在看来，似乎每一个人的方法都不尽相同。~~

由于每一个数都是非负的，所以最后一个数一定是三个数的和，所以找出两个数后就可以根据最后一个数直接算出结果。

然后发现第一个数和第二个数一定是原来就在序列中的数，因为假设这个数是由某两个或者三个数相加得到的，那么一定存在比这两个数小的数，与条件矛盾。

所以直接输出就行了。

---
### 代码：

```cpp
#include<cstdio>
using namespace std;
int T,a[10];
int main(){
	scanf("%d",&T);
	while(T--){
		for(int i=1;i<=7;i++)scanf("%d",&a[i]);
		printf("%d %d %d\n",a[1],a[2],a[7]-a[1]-a[2]);
	}
	return 0;
}
```
