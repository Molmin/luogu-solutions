# Emmm...
## 这个题的话。。。
### 首先没有翻译
但是~~聪明的~~同学们可以发现：其实只用看输出格式就可以了。

对于每个测试用例，单独在一行上输出以下技能测试问题的答案:
 $n$ 乘以 $567$ ，然后除以 $9$ ，然后加上 $7492$ ，再乘以 $235$，然后除以 $47$ ，然后减去 $498$ 。十位的数字是多少?
 
这么一来，就好做多了。  
首先，求一个数（程序中为 $n$ ）十位上的数字的操作很简单，代码如下：
```cpp
abs(n/10%10)
```
Emmm...

十位上的数字肯定是大于 $0$ 的。

所以一定要加abs（绝对值）。

至于运算，看了翻译，就不用我说了吧。

贴上代码：
```cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
	int t;
	cin >> t;
	while ( t-- )  //数据组数
	{
		long long n;
		scanf ( "%lld", &n );
        printf ( "%lld\n", abs(((n*567/9+7492)*235/47-498)/10%10) );  //进行以上操作
	}
	return 0;
}
```
