#### 题意简述

某国有 $2$ ， $4$ ， $8$ ， $16$ ， $32$ 面值的货币，大面值货币可以兑换成小面值货币，给定一个价格的货物，求能否买到。

#### 题目分析

我们可以考虑，那些情况买不到：

1. 该价格为奇数时是买不到的（没法兑换出奇数面值的货币）。

2. 货币面值之和不够买该货物时，也是不可行的。

为什么没有其它情况呢？我们可以发现，所有的面值都是 $2$ 的倍数，也就是说，所有货币都可以兑换为面值为 $2$ 的货币，所以非前两种情况的货物，是绝对可以买到的。

代码：

```cpp
#include<iostream>
using namespace std;
int main(){
    int a,b,c,d,e,w,sum;
    cin>>a>>b>>c>>d>>e>>w;
    sum=a*2+b*4+c*8+d*16+e*32;
    if((w&1==1)||sum<w){
        cout<<"NO";
        return 0;
    }
    cout<<"YES";
    return 0;
}
```
