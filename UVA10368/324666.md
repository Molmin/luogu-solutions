**题意**

自己看双倍经验里面的吧，懒得写了。

**思路**

首先，假设当前的 $2$ 个数分别为 $x$ 和 $y$，令 $x \ge y$。

### 若 $x=y$

显然先手必胜。

### 若 $x>y$

#### 若 $x \ge 2y$

在这种情况下，不妨令 $x=ky+r$，其中 $r<y$。若此时 $(y,r)$ 为必败状态，那么先手只需要取到 $(y,r)$ 即可取胜。否则先手可以取到 $(y,y+r)$ 的状态，从而使得后手只能取到 $(y,r)$，先手取胜。

综上，这种情况下先手必胜。

#### 若 $x < 2y$

此时显然先手只能取到 $(y,x-y)$，一直递归求解即可。

**MAIN CODE**
```cpp
bool work(int x,int y){
	if(x==y||x/y>=2)
		return 1;
	return !work(y,x-y);
}
```
