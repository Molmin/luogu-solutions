这道题要我们判断是否**恰好**有一门课不及格。

我们的思路大概是这样哒：

```
if 语文<60 数学>=60 英语>=60
或
语文>=60 数学<60 英语>=60
或
语文>=60 数学>=60 英语<60

输出 1;

else 输出 0;
```

好啦，这里给大家补充一个知识，“或”和“与”。

或在英文中是`or`，没错，C++中你也可以用`or`表示或！

`||`是或的另一种表示方法。

与就是`and`，还可以表示成`&&`（很像对不对？）。

下面是标程：

```cpp
#include <iostream>
using namespace std;
int main(){
    int c, m, e; // 语数英
    cin >> c >> m >> e; // 输入
    if (c < 60 && m >= 60 && e >= 60 // 主要代码
        ||
        c >= 60 && m < 60 && e >= 60 // 感谢@flysnow的纠错
        ||
        c >= 60 && m >= 60 && e < 60) cout << "1" << endl;
    else cout << "0" << endl;
    return 0;
}
```
