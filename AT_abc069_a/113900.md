题目中所谓“四条街包围一个街区”其实就是两条街之间夹一个街区而已；

n条街有几条两两相邻呢？答案是n-1条；

m条街有几条两两相邻呢？答案是m-1条；

那么n,m条街就有(n-1)(m-1)个街区；

## 注意：

n与m可能是0与0，(n-1)(m-1)=(-1)(-1)=1。

要加特判！！！

AC代码（~~你们最想要的~~）

```
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n,m;
    cin>>n>>m;
    if(n==0&&m==0)cout<<"0";//特判
    else {//使用公式(n-1)(m-1)
        --n;--m;
        cout<<n*m;
    }
    return 0;//好习惯
}
```
