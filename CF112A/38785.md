P党福利
---------------
这一题的做法楼下已经讲得很明白了，但对于P党而言，如果能学会使用upcase函数，这一题就可以写得更加简单。

-------------

upcase函数用于字符型变量，可以**将小写字母转化为大写字母**，但它**也可以用于字符串**。它能**将整个字符串中的小写字母转化为大写字母**。在这一题中，我们可以用这个函数把字符串中的字符统一为大写字母，然后就可以直接比较了。

--------------

代码：
```
var
    s,t:string;
begin
    readln(s); readln(t); //读入两个字符串
    s:=upcase(s); t:=upcase(t); //统一为大写字母
    if s>t then writeln(1) //直接比较
    else if s=t then writeln(0)
            else writeln(-1);
end.
```
