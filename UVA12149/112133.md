题意是求一个$n*n$的正方形中包含多少个正方形，不难想到枚举正方形的边长在求和，记$a_i$表示边长为i的正方形的个数，则答案为

$$\sum_{i=1}^na_i$$

那么怎么求$a_i$呢？~~那不是轻轻松松吗~~

我们从$a_1$开始：

~~是个人都能想到~~ 不难想到$a_1=n^2$

再看$a_2$:

边长为2的正方形一行可以放下$(n-1)$个,可以放$(n-1)$行，所以$a_2=(n-1)^2$

诶嘿，发现了个好东西：

对于边长为$i(1\le i\le n)$的正方形，在这个$n*n$的正方形中可以放$(n-i+1)^2$个

那原来的答案：
$$\sum_{i=1}^na_i$$

就变为：
$$\sum_{i=1}^n(n-i+1)^2$$

也就是：
$$\sum_{i=1}^ni^2$$

$O(n)$复杂度已经绰绰有余了，但最后这个平方和还可以继续化简吗？

### ~~废话~~

把这个式子展开：

$\sum_{i=1}^ni^2=1^2+2^2+……+(n-1)^2+n^2$

继续展开：

$1^2+2^2+……+(n-1)^2+n^2$

$=1*2+2*3+……+(n-1)*n+(1+2+3+……+n)$

先搞前面这一大坨

$1*2+2*3+……+(n-1)*n$

$=1/3*[1*2*3-0*1*2+2*3*4-1*2*3+……+(n-1)*n*(n+1)-(n-2)*(n-1)*n]$

$=1/3*(n-1)*n*(n+1)$

后面这个就用高斯求和就行了

$1+2+3+……+n=(n+1)*n/2$

提个公因数$n*(n+1)$再加起来，就得到了下面这个式子：

$1^2+2^2+……+(n-1)^2+n^2=n*(n+1)*[(n-1)/3+1/2]$

把中括号里的东西搞一搞：

$(n-1)/3+1/2=(2n-2)/6+3/6=(2n+1)/6$

正解已经呼之欲出！

$\sum_{i=1}^ni^2=n*(n+1)*(2n+1)/6$

这其实就是个平方和数列求和公式，结论附加证明已经在上面了，下面放代码：
```cpp
#include<bits/stdc++.h>

namespace solve{
	int n;
	void work(){
		while(scanf("%d",&n)){
			if(n==0)return;
			printf("%d\n",n*(n+1)/2*(2*n+1)/3);
		}
	}
}

int main(void){
	solve::work();
	return 0;
}
```
