简而言之就是，只需要定义一个足够大的数组，然后循环输入直到结束，再循环输出即可。

这里介绍一下本题的输入方式：

```cpp
int n[10000001]; //先定义一个足够大的数组储存输入值。
int num=0;//这个变量表示总共有多少个输入值。

for (int i=1; cin>>n[i]&&n[i]!=0; i++) num++;
//在 for 循环中定义i，使得每次输入不等于 0 的 n[i] 时增加一个输入次数，而当输入为零的时候退出循环。
```

同理，输出即为：

```cpp
for(int i=1; i<=num; i++) cout<<n[i]*(n[i]+1)*(2*n[i]+1)/6<<endl; 

//当 i 小于总输入次数 num 时，输出对应的结果并换行。
```


那么，话不多说，上代码：

### 代码：

```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
	int n[10000001],num=0;
	for (int i=1; cin>>n[i]&&n[i]!=0; i++) num++; //先输入。
	for(int i=1; i<=num; i++) cout<<n[i]*(n[i]+1)*(2*n[i]+1)/6<<endl; //再输出。
	return 0;
}
```

这样，我们就成功达成了分批输入输出的目的啦。
