题目大意：输入n个数，$x_1$ , $x_2$..., $x_n$ 将每个数分成a+b的形式，保证 $a>b$ ，输出有几种分法。 

想一想，如果 $x_{i}$ 除以 $2$ 的余数为$1$，则分法为 $x_i/2$ 向下取整 

示例：$9=1+8=2+7=3+6=4+5$ 共四种 

若$x_i$除以 $2$ 的余数为 $0$ ,则分法为 $x_i/2-1$ 

示例：$12=1+11=2+10=3+9=4+8=5+7$ 共五种 

同时还要特判 $0$，$1$，$2$，问题就迎刃而解了

代码（c++）
```cpp

#include<bits/stdc++.h>
using namespace std;
int main()
{
	long long n[10005],t;//t为数据组数
	cin>>t;
	for(int i=1;i<=t;i++)
		cin>>n[i];
	for(int i=1;i<=t;i++)
	{
		if(n[i]/2-1>=0)//相当于n[i]>2
		{
			if(n[i]%2==0)
				cout<<n[i]/2-1;
			else
				cout<<n[i]/2;//特判，思路见上
		}
		else
			cout<<"0";//若n[i]<=2,没有
		cout<<endl;
	}
    return 0;
}
```
