## 解法
一眼望过去这是道打表题

所以菜鸡LSC打了个表

>当x为```1```时```ans = 1```
>
>当x为```2```时```ans = 11```
>
>当x为```3```时```ans = 21```
>
>当x为```11```时```ans = 13```
>
>当x为```22```时```ans = 23```
>
>当x为```33```时```ans = 33```

以此类推

所以，只要计算出组成x的数字c
和c的个数

我们就能从菜鸡LSC的表中找到ans

## 上代码

```cpp
#include <iostream>

using namespace std;

const int biao[9][4]= //标准的打表
{
	{1 ,  3,  6, 10},
	{11, 13, 16, 20},
	{21, 23, 26, 30},
	{31, 33, 36, 40},
	{41, 43, 46, 50},
	{51, 53, 56, 60},
	{61, 63, 66, 70},
	{71, 73, 76, 80},
	{81, 83, 86, 90}
};

int n;
int a;
int ans;

int main() {
	cin >> n;
	for (int i = 1; i <= n; i++){
		ans = 0;
		cin >> a;
		for (int j = 1; j <= 4; j++){ //计算组成数字a和个数ans
			if (a < 10)
				break;
			else ans++;
			a/=10;
		}
		cout << biao[a-1][ans] << endl; //从菜鸡的表中输出
	}
	return 0; //标准のreturn
}
```
