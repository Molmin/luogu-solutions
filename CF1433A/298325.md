题意：

一宿舍楼，有 $10000$ 间房子，分别编号为 $1...10000$。一个无聊的房间就是编号中所有数字都是一样的数，比如：$7777, 9999,666,22,3$ 这都是无聊的房子。有一个无聊的人，他要拨打所有无聊房间的号码，直到有人接电话，依照这样的顺序：

- 先打数字为1组成的无聊编码，即：$1,11,111,1111$
- 然后打数字为2组成的无聊编码，即：$2,22,222,2222$
- **以此类推**

住在无聊房间编号为 $x$ 的居民接了电话，则这个无聊的人不再打电话。这个无聊的人想要知道，他拨打到 $x$，一共按键了多少次？

譬如，$x=22$，则这个无聊的人按了 $1,11,111,1111,2,22$ 这些键的电话。算出来就是 $1+2+3+4+1+2=13$。

思路：

给出了 $x$ ，我们取 $x$ 中的一个数字就知道拨打的是几组成的无聊号码。我们可以用字符串形式存。因为字符串可以获得长度，设长度为 $l$，并且这个号码由 $c$ 组成，列出公式为：$(l+1)\times l\div2+(1+2+3+4)\times (c-1)$，算出来的数就是答案。为什么是 $(1+2+3+4)\times (c-1)$ 呢？我们求的是上一个数码无聊数列按键和，比如数字 $33$，前面有 $1,11,111,1111...2222$，要求出 $(1+2+3+4)$ (一组无聊数列按键总数) $\times 2$ (前面有多少组无聊数列)。

得出代码：

```
int T;
    cin >> T;
    while (T--) { //多组数据
        string s;
        cin >> s;
        int l = s.length(); //字符串长度
        int c = s[0] - '0'; //转换数字
        cout << (l + 1) * l / 2 + (1 + 2 + 3 + 4) * (c - 1) << '\n';
    }	
```
