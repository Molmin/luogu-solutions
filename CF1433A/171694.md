### 题目大意：


------------

在序号为$1-10000$栋楼房中，序号**只由同一个数字组成**的楼房，叫做“无聊楼”。

比如：$2,77,999$就是“无聊楼”。

现在，麻烦制造者要按一定顺序给无聊楼打电话。顺序是这样的：$1，11，111，1111，2，22，222，2222$......

问题是：麻烦制造者**所摁次数的和**。


------------

### 思路


------------


可以看到，从$1$位到$4$位，所摁次数分别为$1，2，3，4$；所以关于数字$1111$，摁$1+2+3+4=10$次。

对应地，$1，11，111$分别摁$1，3，6$次。

再来看2；当麻烦制造者摁到$2$时，他已经摁完了$1，11，111，$和$1111$。

所以关于2，摁的次数为$1+2+3+4+1=11$。

相似地，$3$为$21$，$444$为$36$，$77$为$63$。

>对于n构成的数字，**一定有答案为**$10*(n - 1)+(1/3/6/10)$
>
>数位为1时，+1；
>
>数位为2时，+2；
>
>数位为3时，+3；
>
>数位为4时，+4；


------------


### 代码


------------


**出示蒟蒻的代码**

```cpp

#include<bits/stdc++.h>
using namespace std;
int main(){
	int n,a[100],ans[100];
	cin>>n;//输入
	for(int i=1;i<=n;i++){
	    cin>>a[i];//输入
	    if(a[i]%1111==0) ans[i]=a[i]/1111*10;//数位为4时
	    if(a[i]<1000) ans[i]=(a[i]/111-1)*10+6;//数位为3时
	    if(a[i]<100) ans[i]=(a[i]/11-1)*10+3;//数位为2时
	    if(a[i]<10) ans[i]=(a[i]-1)*10+1;//数位为1时
    }
	for(int i=1;i<=n;i++) cout<<ans[i]<<"\n";//输出
	return 0;
}

```
