首先，我们会发现，**只要数组中最大的数小于等于 $d$，或者数组中最小的两个数的和小于等于 $d$，那么就可以，否则就不可以**。因为要让最大值最小，我们就得把这个最大值变成最小的两个数的和与自己比较起来更小的数。

照这个样子，其它的每个数都得变成最小的两个数之和与它比起来更小的数。比如 `2 3 4 7` ，我们可以知道，$4$ 比最小的数和第二小的数 $5$ 小，于是它不变。可是，$7$ 比最小的数和第二小的数 $5$ 大，于是 $7$ 就得变成 $5$。

最后，如果最大的数都可以变成小于等于 $d$ 的数了，那么其它的数肯定都能变成小于等于 $d$ 的数了，所以不用再判断其它的数了。
## Code:
```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int t;
    cin >> t;
    while(t--){
        int n, d;
        cin >> n >> d;
        int a[n];
        for(int i = 0;i < n;i++){
            cin >> a[i];
        }
        sort(a, a + n);
        if(a[0] + a[1] <= d || a[n - 1] <= d){
            cout << "YES" << endl;
        }
        else{
            cout << "NO" << endl;
        }
    }
    return 0;
}
```
