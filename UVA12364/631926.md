### 题目大意

给定若干个字符串，将其从数字转换为盲文或从盲文转换为数字。

![](https://pic.imgdb.cn/item/64b4e17f1ddac507ccfdb668.jpg)

### 思路分析

本题很水，但稍微有点坑。

先打表记录每个数字对应的盲文。

```cpp
string a[3][10] = {
	{".*", "*.", "*.", "**", "**", "*.", "**", "**", "*.", ".*"},
	{"**", "..", "*.", "..", ".*", ".*", "*.", "**", "**", "*."},
	{"..", "..", "..", "..", "..", "..", "..", "..", "..", ".."}
};
```

注意图片是先给 $1,2,\cdots,9$ 的盲文，再给 $0$ 的盲文的，需要注意调整。

#### Part 1 数字转盲文

这个还是比较简单的，用一个字符串 $ans$ 记录答案。对于数字字符串 $s$，枚举每一位，再将每一位每一行的盲文加上空格拼接在 $ans$ 后面。

```cpp
if (c == 'S') {
	ans = "";
	cin >> s1;
	for (int t = 0; t < 3; t++) {
		for (int i = 0; i < n; i++)
			ans += a[t][s1[i] - '0'] + (i == n - 1 ? "" : " ");
		ans += '\n';
	}
	cout << ans;
}
```

`(i == n - 1 ? "" : " ")` 是为了防止行末的空格导致 Presentation Error 的。

#### Part 2 盲文转数字

这个稍微复杂一点，用二维数组 $s_{i,j}$ 记下整个盲文，那么每一列的盲文正好可以与前面打表的 $a$ 数组的每一列一一对应。

```cpp
if (c == 'B') {
	for (int t = 0; t < 3; t++)
		for (int i = 1; i <= n; i++)
			cin >> s[t][i];
	for (int i = 1; i <= n; i++) {
		for (int j = 0; j < 10; j++)
			if (s[0][i] == a[0][j] && s[1][i] == a[1][j])
				cout << j;
	// s[0][i] 是第 i 位盲文的第 1 行。
	// a[0][j] 是 j 所表示的盲文的第 1 行。
	// s[1][i], a[1][j] 同理。
	// 只要 s[0][i] == a[0][j] 且 s[1][i] == a[1][j] 配对即可判定数字为 j。
	}
	cout << endl;
}
```

注意格式要与样例完全一致，否则会 Presentation Error。

### AC Code

```cpp
#include <bits/stdc++.h>
using namespace std;
#define int long long

string a[3][10] = {
	{".*", "*.", "*.", "**", "**", "*.", "**", "**", "*.", ".*"},
	{"**", "..", "*.", "..", ".*", ".*", "*.", "**", "**", "*."},
	{"..", "..", "..", "..", "..", "..", "..", "..", "..", ".."}
};
int n;
char c;
string s1, s[3][102], ans;

signed main() {
	while (cin >> n) {
		if (n == 0)
			break; // 特判不要忘了。（虽然不加好像也没什么事）
		cin >> c;
		if (c == 'S') {
			ans = "";
			cin >> s1;
			for (int t = 0; t < 3; t++) {
				for (int i = 0; i < n; i++)
					ans += a[t][s1[i] - '0'] + (i == n - 1 ? "" : " ");
				ans += '\n';
			}
			cout << ans;
		}
		if (c == 'B') {
			for (int t = 0; t < 3; t++)
				for (int i = 1; i <= n; i++)
					cin >> s[t][i];
			for (int i = 1; i <= n; i++) {
				for (int j = 0; j < 10; j++)
					if (s[0][i] == a[0][j] && s[1][i] == a[1][j])
						cout << j;
			}
			cout << endl;
		}
	}
	return 0;
}
```
