## Part 1
考虑我们有如下串
```
B A B A B A
1 2 3 4 5 6
```
很明显最优的方案是：

将第2、3个移动到-1、0上变成这样：
```
 A B B _ _ A B A
-1 0 1 2 3 4 5 6
```
然后将第5、6个移动到2、3上：
```
 A B B B A A
-1 0 1 2 3 4
```
最后把3、4移动到-3、-2上：
```
 A  A  A B B B
-3 -2 -1 0 1 2
```
当然也有其他的移动方案(其实这里的方案和下面的有些出入 ~~请不要在意qwq~~)，但总之最少的步数为$3$

## Part 3
对于更长的串我们总是可以将其最后的一串AB移动到-1和0位处再把第3位(注意不是从-1位开始)移动至先前被移动的那一串的位置

中间的那一串是否变成了与最初的那一串相似但长度不同的一串？于是我们针对于这一个子问题继续处理，然后一直拆分为子问题

现在我们发现这一长串还是有点小问题，对于每一个问题，在它的子问题处理完后还需要在进行$2$步操作才能符合题目要求，由于挺容易想的，所以就请自行思考
