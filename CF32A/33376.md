首先注意到我们n的范围在1000之内，所以说我们用O（n^2）的算法是不会超时的，那么我们可以从1~n枚举i，j；如果说i不等于j并且a[i]与a[j]的差小于等于d，那么这一次的选择就是符合条件的，于是总答案++。然后输出总答案即可

最后说一个本题中可不注意，但做其他的题需要注意一下的细节，就是本题总方案数最多是n*(n-1)/2，这个方案数即使在n=1000时也不会超过int的范围，故可以用int。

奉上代码：
```
#include<bits/stdc++.h>
using namespace std;
int n,m,s=0,a[1001];//s为总方案数
int main()
{
	cin>>n>>m;//习惯，d改作m
	for(int i=1;i<=n;i++)
	cin>>a[i];//读入
	for(int i=1;i<=n;i++)
	for(int j=1;j<=n;j++)
	if(i-j&&abs(a[i]-a[j])<=m) s++;//如果说i不等于j
    //即不是同一个人，那么如果差的绝对值小于等于d，就+1
	cout<<s;//输出答案
	return 0;
}
```
