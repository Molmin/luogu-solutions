先介绍几种关于**等差数列**的公式：   

**文字**描述：   
末项 $=$ 首项 $+$ $($项数 $-1$ $)$ $\ast$ 公差   
项数 $=$ $($末项 $-$ 首项$)$ $/$公差 $+1$    
首项 $=$末项 $-($ 项数 $-1$ $)$ $\ast$ 公差    
和 $=$ $($ 首项 $+$ 末项 $)$ $\ast$ 项数 $/2$   

**数学**语言：   
$a_n = a_1 + ( n - 1 ) \ast d$   
$n = ( a_n - a_1 ) / d + 1$   
$a_1 = a_n - ( n - 1 ) \ast d$   
$sum = ( a_1 + a_n ) \ast n /2$
