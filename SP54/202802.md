# Problems
给出两个人的苹果个数的和与差，确定他们的苹果个数。
# Answer
先推公式。

设第一个人的苹果数为$x$个，第二个人的苹果数为$y$个。

$x+y=$和

$x-y=$差

将这两个式子相加，得$x=($和$+$差$)\div2$；

带入其中一个式子得$y=($和$-$差$)\div2$

再看数据范围：$10^{100}$

于是可以用$python$。
# Code
```python
for i in range(10): # 循环
    a=int(input()) # 读入a，其中a为整形
    b=int(input()) # 读入b，其中b为整形
    print((a+b)//2) # 套公式，求除第一个人的苹果数
    print((a-b)//2) # 套公式，求除第一个人的苹果数
```
