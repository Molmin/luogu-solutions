## UVA11044 Searching for Nessy（寻找尼斯湖水怪） 题解

#### [题目传送门](https://www.luogu.com.cn/problem/UVA11044)

### 题目大意

本题共有多组数据，对于每组数据，提供给出 $n$ 和 $m$，代表一个方格的长和宽，即 $n×m$ 的方格，现在有一个 $3×3$ 的正方形，问需要多少个正方形才能把放个铺满（如果有正方形摆不下去的地方，不要管它）。

### 样例解释

#### 样例 #1 解释

共有 $3$ 组数据。

第一组数据，$n=6,m=6$，即方格的长宽分别为 $6$ 和 $6$，需要 $4$ 个 $3×3$ 的正方形就能铺满。

第二组数据，$n=7,m=7$，即方格的长宽分别为 $7$ 和 $7$，需要 $4$ 个 $3×3$ 的正方形，但是会有部分铺不满。

第三组数据，$n=9,m=12$，即方格的长宽分别为 $9$ 和 $13$，需要 $12$ 个 $3×3$ 的正方形，但是会有部分铺不满。

### 思路

上过小学的人应该知道，这种情况不能用 $\frac{S_a}{S_b}$，我们应该用  $\lfloor\frac{n}{3}\rfloor\cdot\lfloor\frac{m}{3}\rfloor$，由于把两个 $int$ 类型的数相除，自动向下取整，所以我们直接输出就可以了。

### 代码+注释

```cpp
#include <bits/stdc++.h>
using namespace std;
int n, a, b;
int main(){
	cin >> n;// 注意是多组数据
	for (int i = 1; i <= n; i++){
		cin >> a >> b;// 读入长和宽
		cout << (a / 3) * (b / 3) << "\n";// 输出个数
	}
	return 0;
}
```
