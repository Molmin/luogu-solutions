首先，这是一道比较水但是~~有一点思考难度~~的题。

主要思路：

	先算出总和，如果所有的糖果不能被平均分配的话，输出-1.
    其他情况，就先算出平均值，之后再拿每一个糖果包减去平均值，
    加起来，除以二，就是答案了。
    
PS:这道题会莫名奇妙的CE，本地机能过，但是SPoj好像炸了。

注意一个坑：前面的“file”是没有的。
数据范围：t<=10000，这点值得注意。
n=0 时，也要直接退出。

```cpp
#include<cstdio>
int main(){
    int n,a[10001],s,t;
    while(scanf("%d",&n) && n && n!=-1){//输入，注意有坑
        s=0,t=0;//一定要注意清空
        for(int i=0;i<n;i++){
            scanf("%d",&a[i]);//输入
            s+=a[i];//糖果的总和
        }
        if(s%n!=0) printf("-1\n");//不能被平均分配的情况
        else{
            s/=n;
            for(int i=0;i<n;i++){
                if(a[i]>s) t+=a[i]-s;
                else t+=s-a[i];//这里用abs函数也可以，记得加头文件及using namespace std;
            }
            printf("%d\n",t/2);//输出答案
        }
    }
    return 0;//完美答案！
}
```
