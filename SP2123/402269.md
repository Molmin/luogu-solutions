这是一道模拟题，虽然题简单，但还是有点坑的。
+ 坑1：输入输出样例貌似有问题（file是多余的）
+ 坑2：是以``` -1 ```当做输入结束的符号的
+ 坑3：糖果与平均值求绝对值后，还要除以$2$
+ 坑4：题目中没有说输入数据范围，所以数组不要设太小
### 由于题很简单，所以直接上代码：

```cpp
#includ<bits/stdc++.h>
using namespace std;`
int a[1000005], n, sum = 0;
int mian(){
	while(cin >> n){//如果还有数据没有输入，cin >> n 为真，同时输入n。全都输完了则cin >> n 为假，跳出循环 
		if(n == -1) break;//如果n为-1，那么跳出循环（也可以写return 0） 
		sum = 0;//将sum清零 
		for(int i = 0; i < n; i++) cin >> a[i], sum += a[i];//输入n个数，并且求和 
		if(sum % n != 0) puts("-1");//如果总和不是n的倍数，那么无解。所以直接输出-1 
		else{//如果有解 
			int pj = sum / n, cnt = 0;//pj代表n个数的平均值，cnt代表n个数与平均值的差 
			for(int i = 0; i < n; i++){
				cnt += abs(pj - a[i]);//此代码同等于：cnt += max(pj, a[i]) - min(pj, a[i]) ; 
			}
			cout << cnt / 2 << endl; //由于每移动n个糖果可以改变2n个糖果，所以要将与平均值的差再除以二 
		}
	} 
	retrun 0;
}
```
