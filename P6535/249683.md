## 字符串+数组
这是一个新手练习**字符串与数组**的比较好的题目  
### Part 1：审题
拿到什么题目，首先都要**审好题**  
**题目大意**:读入n个字符串，并输出**字符串开头字母的个数大于等于5**的字母，如果没有则输出“**PREDAJA**”  
### Part 2：判断字符串开头字母的个数  
依据题目，我们需要判断**字符串开头字母的个数**   
怎么实现呢？  
我们可以先读入一个**字符串**，然后将**第一个字母**转换为**整数**并存入相应的**数组**里  
**代码实现如下：**
```cpp
    for(int i=1; i<=n; i++)//循环n次
    {
        cin>>s;//读入字符串
        a[s[0]-'a']++;
        //s[0]即为该字符串的首字母
        //s[0]-'a'则为当前首字母所对应的数字
        //例如a-'a'=0,c-'a'=2
        //然后存入数组
    }
```
### Part 3：判断是否大于等于5  
将26个数组**遍历一遍**，如果里面存的字母数量**大于等于5**，我们就直接**输出**  
否则输出“**PREDAJA**”；  
**代码实现如下：**
```cpp
    for(int i=0; i<26; i++)//0~25遍历一遍
    {
        if(a[i]>=5) cout<<char(i+'a'), k=false;
        //如果数量大于等于5，那么直接输出，同时将bool变量设为false；
    }
    if(k==true) cout<<"PREDAJA"<<endl;
    //如果bool变量仍是true，那么说明并没有大于等于5的字母
    //那么就输出"PREDAJA"

```

然后我们就可以顺利地**AC**这道题了！
