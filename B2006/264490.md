直入主题，题目想让地球能够持续发展，这是什么意思？

地球原本是有资源的，每年地球的资源会增长。

如果人每年消耗的资源比地球每年增长的资源多了，那么人就会消耗地球原本的资源。

如果人类每年都会对地球原本的资源造成消耗，地球的资源就一定会越来越小，直至没有。

所以，我们要做到的事情就是让地球原本的资源不受消耗，反过来说，**我们要做到的事情是让人每年消耗的资源不超过地球每年增长的资源**。

而我们需要做到人数最大化，咋办？

人每年消耗的资源不超过地球每年增长的资源，那我们就让他等于。

而地球每年增长的资源咋算？

我们设 $c$ 为地球原本资源， $d$ 为地球每年增长的资源，人类每年消耗的资源为 $1$ 份。

那么:

1式： $xa=c+ad$

2式： $yb=c+bd$

1式-2式： $(xa-yb)=d(a-b)$

由1式-2式得： $(xa-yb)/(a-b)=d$

而**人类数量=地球每年增长资源/人类每年消耗资源**

所以人类数量= $d/1$ = $d$ = $(xa-yb)/(a-b)$

至此，式子推完了，摆上代码。

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int x,a,y,b;
	double z;
	cin>>x>>a>>y>>b;
	z=(a*x-b*y)/(a-b);
	printf("%0.2f",z);
	return 0;
}
```
