这次又是拿出一个八进制来吓唬人的  
废话不多说，详情见代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n,s;
    s=0;
    cin>>n;//输入
    while(n)
    {
        if(n%8==1)//重点是这里，见后面解析
            s++;
        n/=8;
    }
    cout<<s;
}
```
为什么十进制转八进制想要求一，直接将十进制mod8呢  
因为转换过程就是将一个数一直mod8    
例如407，八进制为57  
先除8，得50余7  
50取5，得57，所以这里是没有1存在的  
相反将57 转为十进制，
7*（8的零次方）+50*8可得答案
