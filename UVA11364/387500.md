## UVA11364 Parking
[原题链接](https://www.luogu.com.cn/problem/UVA11364) 首篇题解。

## 题意
在某街上购物时，你通常在路边随便找个地方停车，然后走路去店里。某街是一条直线，所有的店家及收费停车位都在直线的整数位置上。你要去很多家店，却只想付一次的停车费。你可以给自己找个停车位，好让你能够少走两步吗？询问你如果车停得好，你最少需要走的距离。

多组数据，每组数据中包含一个整数 $n$，表示你要去 $n$ 家店，接下来是 $n$ 个整数，表示 $n$ 个店在街上的位置。

输出每组数据中你最少需要走的距离。

## 分析
不难发现，如果只有两个店，车停在两个店之间时需要走的距离最短。那么当存在多个店时，车停在最左边的店和最右边的店之间时距离两店最短。

由于你需要来回走一趟才能在保证每个店都去过的基础上回到车的位置，所以最左和最右的店中间的店全部可以忽略不计。最后答案便是来回走一趟的距离。得出式子：$(max-min)\times 2$。

注意，由于存在多组数据，最大值和最小值变量在每次循环中读入数据前需要初始化。

## Code
```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int a,maxa,mina,n,t;
int main(){
    cin>>t;
    while(t--){
        cin>>n;
        maxa=0;mina=100; //最大值和最小值初始化 
        for(int i=1;i<=n;i++){
            cin>>a;
            maxa=max(maxa,a); //取最右边的店位置 
            mina=min(mina,a); //取最左边的店位置 
        }
        cout<<(maxa-mina)*2<<endl;
    }
    return 0;
}
```
