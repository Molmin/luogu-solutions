### 题意翻译

$T$ 组数据，每组数据给定 $n, k$，在由整数 $1 - n$ 中选取尽可能多的数组成集合 $S$，满足 $S$ 中任何一个元素都不是另一个元素的 $k$ 倍。求 $|S|$。

### 算法思路

考虑贪心。

如何才能选到尽可能多的数？

由于 $k$ 的倍数是每隔 $k$ 个才有一个，且若先前在较小的数中选取了某一个区间内的数，那么在相对应的区间内也仅仅是隔 $k$ 个才有一个不能选而已。

但是若在较小的区间内选择了一部分数，那么这部分数有可能同时还会影响到更小的某段区间中的数的选择。

因此先从大的数中选，可是选哪一段呢？答：直接选择 $(\frac{n}{k},n]$ 中的所有整数。为什么？

因为要在某个数本身和 $k$ 倍中做出选择，而我们知道选择一个较小的数是会同时影响到更小的数和更大的数的选择的，因此凡是需要作出选择的，我们都选较大的。

而又知道当 $(\frac{n}{k^2},\frac{n}{k}]$ 不被选择时，在小于 等于 $\frac{n}{k^2}$ 的数中又有一部分能被选择。

这样从 $n$ 开始，每次除以 $k$，就出现了交替选于不选的情况，并且这样的解一定不是更劣的。循环求解。

时间复杂度 $\Theta(\log_kn)$

### Code

```cpp
#include<bits/stdc++.h>
using namespace std;
int main() {
	int T;
	cin >> T;
	while(T--) {
		int n, k, ans = 0;
		cin >> n >> k;
		while(n) ans += n - (n / k), n /= k, n /= k;
		cout << ans << endl;	
	}
	return 0;
}
```
