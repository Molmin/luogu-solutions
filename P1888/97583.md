这题目还是很简单的

只要学过三角函数的人应该都知道，最小角所对的即为最小边，而题目中已经给出的就是直角三角形，正弦为对边比斜边。所以只需找到最小边与最大边就OK啦。

这题稍微考验初学者的是要求约分。其实只需使用欧几里得算法（辗转相除法）。

直接上代码。。。

```
#include<iostream>
using namespace std;
int main(){
    int a,b,c,k,l;   //**k代表最大数，l代表最小数**
	int i,p[3];   //用数组做可能会快一点
	for(i=0;i<3;i++)   //输入数据
	cin>>p[i];
	k=p[0];   //初始化数据
	l=p[0];
	for(i=0;i<3;i++){   //以下是找最大最小值步骤
		if(p[i]>=k)   //只要循环中有数大于原k，就更新k的值
		k=p[i]; 
		if(p[i]<=l)   //只要循环中有数小于原l，就更新l的值
		l=p[i];
	}   //寻找最大最小值结束，此时k是最大，l是最小
    for(i=1;i<=l;i++){   //以下是约分步骤
		if(l%i==0&&k%i==0){   //从1循环到l，寻找公约数并不断更新k与l的值
		l=l/i;
		k=k/i;
	}
	}   //约分步骤结束
    cout<<l<<"/"<<k;   //题目中只要求分数，直接输出最小值/最大值
	
    return 0;
} 

```

这题也可以不用数组做，因为已经限定只有三个数。

还有，特别注意一下，约分步骤循环一定只能从1开始，因为后面的步骤l=l/i;     k=k/i;

中i的值不能为0，否则运行会出问题。
