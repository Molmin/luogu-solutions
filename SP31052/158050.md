## 题目描述
有两个人玩游戏（$Hemlata$和$Ragini$），有n个石子，现在要轮流取石子，每人能取1个或2个或5个。由$Hemlata$先取，谁没法取了就输了。

## 样例分析
首先我们来看一下样例：1,2的话，第一个人可以直接取完。如果是3，无论第一个人怎样取，都会使对面一次性取完，即第二个人赢。

## 做法分析
看到这里，我们可以先试试枚举。其实如果要赢，我们只要使取完后达成必输状态即可，如果怎样取都会回到必胜状态，那么我们必输。

4：可回到2,3。其中3必输，所以4必胜。

5：可直接取光，必胜。

6：可回到1,4,5。均必胜，所以6必败。

7：可回到2,5,6。其中6必输，所以7必胜。

...

当我们枚举到较大的数后，便可以发现：3的倍数必败，其他必胜。我们就可以根据这个写出代码了。代码很简单，只需注意n开longlong即可。

## 代码
```cpp
#include<iostream> 
using namespace std;
int main(){
	long long n,t;
	cin>>t;
	while(t--){
		cin>>n;
		if(n%3==0)cout<<"Ragini"<<endl;
		else cout<<"Hemlata"<<endl;
	}
	return 0;
}
```

## 结语
这里稍微解释一下为什么只有三的倍数输：$3n+1$与$3n+2$可以分别取走1,2个石子，就会达到必输状态，即均必胜。而$3n-1,3n-2,3n-5$除以3分别余2,1,1，均为必胜状态，所以3的倍数必输。
