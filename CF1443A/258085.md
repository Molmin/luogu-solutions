题意：给定 $n$，求一个正整数序列a，使得a的元素两两之间不能满足如下其中一种情况（设该二数为a,b）：

1. $\gcd(a,b)=1 $或是
2. a能被b整除或b能被a整除

且每个元素最大不能超过4n。输出任意满足上述情况的序列。

----
首先解决 $\gcd(a,b)=1$：

观察样例输出：
```
6 4
4 6 10
14 10 12 8
```
发现都是2的倍数。~~妙————————啊！~~

因为文中给的范围是4n，挺宽裕，所以我们只要使每个数都是2的倍数即可，那么$\gcd(a_i,a_j)=2$ 了。

再解决整除的问题。也即不能有一个数是另一个数的倍数。

我们不妨试着从 $4n$ 开始，依次枚举n个2的倍数：

$4n,4n-2,4n-4,...,2n+2$

然后 $2(2n+2)=4n+4$ 故在序列中不存在 $2n+2$ 的倍数。

最小的数的两倍都不存在，那么其他数也不会存在两倍及更高的倍数了。

那么就输出 $4n$ 到 $2n+2$ 即可。

----
总结：观察样例也是一个解题的好方法。

Code:
```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int T,n;
int main(){
	scanf("%d",&T);
	while(T--){
		scanf("%d",&n);
		for(int i=2*n;i>n;i--) printf("%d ",i*2);
		puts("");
	}
	return 0;
}
```
