[题目传送门](https://www.luogu.com.cn/problem/AT_wupc2019_a)
## 题意
- 从开头看字符串 $s$，如果连续的两个字符是`WA`，则将其替换为`AC`字符串。

- 在进行了一次置换的情况下，从字符串的开头再次进行上述的行动，在没有进行置换的情况下结束。
- 最后输出 $s$。

## 分析
1. 用`cin`输入字符串 $s$。
2. 从前往后遍历每个字符，如果出现`WA`，替换为`AC`。

**为什么要从后往前遍历：如果这个字符是`A`，前一个字符是`W`，那么将这个字符替换为`C`，前一个字符替换为`A`。这样如果再往前的那个字符为`W`，就可以再进行一次置换，达到前一个字符也可以参与到下一次替换之中的效果。**

## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	string s;
	cin>>s;
	for(int i=s.size()-1;i>=0;i--)
	{
		if(s[i]=='A'&&s[i-1]=='W')
		{
			s[i]='C';
			s[i-1]='A';
		}
	}
	cout<<s<<endl;
	return 0;
}
```
