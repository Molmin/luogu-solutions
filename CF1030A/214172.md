其实啊，我们在做这道题时有一个小小的技巧，那就是：

在while()语句这个括号里，是**可以写读入的代码**的，并且只要确实有输入的数据，编译器这个表达式恒等于**真**，否则就是假（这时候就会跳出循环）。

那么，我们就很高兴了~

依次读入每个字符，读到1就输出`HARD`然后`return 0`，如果字符都读完了也没有1就直接输出`EASY`。

至于有的萌新担心不读完全部数据会不会导致错误等等，都是莫须有的担心，毕竟人家评测机只是比较你的答案和标答（就是`.ans`文件是否一致）是否一致，只要一致，就是`AC`！

以下为代码（亲测可过）

```
#include <bits/stdc++.h>
using namespace std;
int main()
{
	int c;
	cin>>c;//覆盖掉第一个输入的n
  	while(cin>>c)//循环读入
   		if(c)//若c=1，那么c就为真，就会执行下面的语句
   		{
   			cout<<"HARD"<<endl;//输出
   			return 0; //直接结束程序
		}
	cout<<"EASY"<<endl;//不然输出EASY
	return 0;
}
```
