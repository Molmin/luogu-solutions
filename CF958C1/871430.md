#### [题 目 传 送 门](https://www.luogu.com.cn/problem/CF958C1)


### 【题意分析】
题意已经十分清晰，对于一个序列，求一个整点将序列分成两个部分，需要求出序列前半部分的和加上序列后半部分的和最大是多少。

### 【思路解析】

首先可以发现，如果暴力枚举，对于每一个整点求出前半部分的和加上后半部分的和，找出最大的，时间复杂度显然会超时，那么我们需要想一些优化的方法。

每一次枚举前半部分的和以及后半部分的和，可以现将数据存入一个前缀和数组，需要的时候直接调用，这样，查询区间和的时间复杂度就降到了 $ O ( n ) $。

再看一下题目：**多次查询序列中连续一段的和**。

显然，这是部分和。

### 【代码实现】


```cpp
#include<bits/stdc++.h>
using namespace std;
int n,p,s[1000010];
int main(){
  	cin>>n>>p;
	for(int i=1;i<=n;i++)cin>>s[i],s[i]+=s[i-1];
	for(int i=1;i<n;i++)ans=max(ans,s[i]%p+(s[n]-s[i])%p);
	cout<<ans;
	return 0;
}
```
