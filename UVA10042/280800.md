## 小小地翻译一下
### 题目背景
美国有一位数字家名叫阿尔伯特·威兰斯基，他姐夫史密斯非常喜欢研究数学，所以两人经常在一起研讨各种数学问题。
### 题目描述
有时，两人碰不到一起，就习惯性地用电话交流。

两人刚结束电话交谈，史密斯突然灵感来临，对威兰斯基的电话号码“$4937775$”产生了兴趣，总觉得这是个特别的数。

可它的特殊之处究竟在哪儿呢？史密斯开始思索考证起来，他先把 $4937775$ 分解质因数：$4937775=3×5×5×65837$，然后再把 $4937775$ 所有质因数各位上的数字相加得：$3+5+5+6+5+8+3+7=42$，接着他又把 $4937775$ 各位上的数字相加得：$4+9+3+7+7+7+5=42$，秘密终于找到了，原来这两个和相等。这真有意思，难道是巧合么？有没有其他的数也有此特点呢？结果发现，所有质数都是具有如此特点。

后来的数学家们把这样的数叫做“史密斯数”，而且还决定质数（简单不复杂）不叫斯密斯数。除质数之外还有许多数具有这样独特的性质，其中最小的数是 $4$。

大家不妨检查一下，$4=2×2$，$2+2=4$。类似有：$22=2×11$,$2+2=2+1+1$；$27=3×3×3$，$2+7=3+3+3$。

你的任务是寻找最接近而且大于给定的数的斯密斯数。
## 小小地分析一下
简单地理解了一下，就是把一个数每个位上的数加起来得到$sum_1$，然后把这个数分解质因数之后，对所有因数都进行相同操作——把每个位上的数合起来，得到$sum_{a1},sum_{a2}...sum_a{k}$，最后得到$sum_2=\sum_{i=1}^{k}$，如果$sum_1$等于$sum_2$，且这个数**不是质数**，那么就说它是Smith数。

可以发现，在上面过程中，主要只有两个步骤：
1. 得到这个数的$sum_1$
2. 分解质因数

第一个步骤很简单了：
```cpp
inline int Sum(int n){
	int res=0;
	while(n){
		res+=n%10;
		n/=10;
	}
	return res;
}
```
第二个步骤其实和埃氏筛有点像了：
```cpp
inline int Dec(int k){
	int res=0;
	for(int i=2;i*i<=k;++i){
		if(k%i == 0)			//如果能被i整除
		while(k%i == 0){		//把所有有关i的因子捞出来
			k/=i;				//先除掉
			res+=Sum(i);		//然后加上这个因子
		}
		//由于我们每次都把i的因子和其倍数除掉了，所以如果之后还有i的倍数，就会忽略掉。所以说和埃氏筛很像，因为这样我们就能保证以后每次晒出来的因子都是质数。
	}
	return res+(k==1?0:Sum(k));	//如果还剩一点没筛完，那么可以断定剩下的这一点是质数。质数的质因数就是它自己啦。
}
```
你可能会问：
> 怎么断定这剩下的一个没筛完的是质数？啊？啊？

好问题。可以想象一下，什么时候循环会退出？

没错，就是当$i\times i>k$时（废话），转换一下就是$i>\sqrt{k}$。没错，这就是我们经常用反证法的判质数！因为在对于所有$i(i\le\sqrt{k})$的情况下，$k$中没有$i$的的因数，那么就可以断定这个数是质数了！
## 完整代码
```cpp
#include <cstdio>

int n;

inline int read(){
	int x=0,f=1;
	char ch=getchar();
	while(ch>'9' || ch<'0'){
		if(ch=='-')f=-1;
		ch=getchar();
	}
	while(ch>='0' && ch<='9'){
		x=x*10+ch-48;
		ch=getchar();
	}
	return x*f;
}

inline bool Isp(int k){
	for(int i=2;i*i<=k;++i)
	if(k%i == 0)
	return false;
	return true;
}

inline int Sum(int k){
	int res=0;
	while(k){
		res+=k%10;
		k/=10;
	}
	return res;
}

inline int Dec(int k){
	int res=0;
	for(int i=2;i*i<=k;++i){
		if(k%i == 0)
		while(k%i == 0){
			k/=i;
			res+=Sum(i);
		}
	}
	return res+(k==1?0:Sum(k));
}
int main(){
    int Kace=read();
	while(Kace--){
        n=read();
    	for(int i=n+1;true;++i){
    		if(Isp(i))continue;
    		if(Sum(i)==Dec(i)){
    			printf("%d\n",i);
    			break;
    		}
    	}
    }
	return 0;
}
```
