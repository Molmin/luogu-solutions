#### 思路：
把这个环形拆成两条序列进行**冒泡**，我们按照奇偶性进行讨论：当$n$为偶数时，我们把它分为两条长度为$n/2$的序列，所以次数为$\frac{n}{2}×(\frac{n}{2}-1)$；当$n$为奇数时，我们拆成长度为$\frac{n}{2}$和$\frac{n}{2}+1$的两条序列，然后分别乘以$\frac{\frac{n}{2}-1}{2}$和$\frac{\frac{n}{2}+1}{2}$，最终两个结果再相加即可。

还有两种特殊情况还需要**特判**，也就是$n=1$或$n=2$时，我们不需要次数就可以完成操作。

#### Code:
```
while(T--)
{
	int n;
	cin>>n;
	if(n==1||n==2)
		cout<<0;
	int ans=0;
	else
	{
		if(n%2==0)
        	ans=(n/2)*(n/2-1);
		if(n%2==1)
        	ans=(n/2)*(n/2-1)/2+(n/2)*(n/2+1)/2;
		cout<<ans;
	}
	puts("");
}
```
