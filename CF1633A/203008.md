由于 $10>7$，所以在十进制下，最多修改一位，也一定能把一个数变成 $7$ 的倍数。

而修改个位不会导致出现前导零什么的，所以我们选择修改个位，把整个数减去它除以 $7$ 的余数，就一定能把它变成 $7$ 的倍数，但是如果修改后，除了个位以外，还有其他位变了，那我们就把这个数再加上 $7$，仍然是 $7$ 的倍数，而且可以发现，这样只有个位变了。

```cpp
#include<stdio.h>
int main(){
    int t,n;
    for(scanf("%d",&t);t--;){
        scanf("%d",&n);
        int ans=n-n%7;
        if(ans<(n/10)*10)ans+=7;
        printf("%d\n",ans);
    }
    return 0;
}
```
