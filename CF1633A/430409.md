# 题目大意

给定一个数 $n$，用最少的操作次数来改变这个数各个数位上的数使得这个数为 $7$ 的倍数。

# 题目分析

可以发现性质：只需改变个位上的数字，就可以将一个数变为 $7$ 的倍数。

于是我们可以枚举个位，修改后的数字如果是 $7$ 的倍数，那么就是这个数。

操作次数为 $1$ 当且仅当原数不是 $7$ 的倍数，故我们需要特判原数就为 $7$ 的倍数的情况。

# 代码

```cpp
inline void solve()
{
	n=read();

	if(n%7==0)
	{
		printf("%d\n",n);
	}

	else
	{
		for(register int i=0;i<=9;i++)
		{
			if((n-n%10+i)%7==0)
			{
				printf("%d\n",n-n%10+i);

				break;
			}
		}
	}
}
```
