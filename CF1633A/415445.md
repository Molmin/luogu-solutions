# 题意简述
- 给定一个整数 $n$。
- 你需要尽可能少改变数位，使改变后的数 $n'$ 为 $7$ 的倍数（如 $n$ 已为 $7$ 的倍数，直接输出 $n$）。

答案不唯一时，你只需要输出其中一种。

每个测试点可能存在多组数据。对于所有数据，满足 $1\leq t\leq 990,10\leq n\leq 999$。

# 题目分析
显然，无论 $n$ 为何值，只变动个位，就一定可以使 $n'$ 为 $7$ 的倍数。

以下是证明：

令 $n$ 的个位为 $p$，$\frac{n-p}{10}=t$，则 $n=10t+p$。

于是，我们只需证明存在小于等于 $10$ 的自然数 $q$，使得 $7\mid (10t+q)$。

显然总有整数 $p'$ 满足 $7\mid (10t+p')$。此时，$p''_{k}=p'+7k(k\in \mathbb{Z} )$ 也一定满足 $7\mid (10t+p'')$。

因为 $7\lt 10$，所以总能找到满足上文的、小于等于 $10$ 的自然数 $p''_{n}$。当 $q=p''_{n}$ 时，满足题目。

证毕。

毕竟个位只有 $0$ 至 $9$ 十种取值，可以直接枚举个位。

# 代码
```cpp
#include <iostream>
using namespace std;

int t,n,ans;

int main(){
    cin >> t;
    for (int i = 1;i <= t;i++){
        cin >> n;
        if (n % 7 == 0){
            ans = n;
        }
        else{
            for (int j = 0;j <= 9;j++){
                if ((n - n % 10 + j) % 7 == 0){
                    ans = n - n % 10 + j;
                    break;
                }
            }
        }
        cout << ans << endl;
    }
    return 0;
}
```
