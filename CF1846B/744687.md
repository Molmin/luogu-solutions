### 分析

一共有八种连线的方式：横着的三行、竖着的三列和两条对角线。

那我们依次判断这八种方式是否连成线即可。

要注意这八种连线方式的线上的点不能有 `.`，即不能把空行也算为答案。

```c++
string s[4];
void solve(){
	cin>>s[0]>>s[1]>>s[2];
	for(int i=0;i<3;i++){
		if(s[i][0]==s[i][1]&&s[i][1]==s[i][2]&&s[i][1]!='.'){
			cout<<s[i][1]<<endl;
			return;
		}
	}
	for(int i=0;i<3;i++){
		if(s[0][i]==s[1][i]&&s[1][i]==s[2][i]&&s[1][i]!='.'){
			cout<<s[1][i]<<endl;
			return;
		}
	}
	if(s[0][0]==s[1][1]&&s[1][1]==s[2][2]&&s[1][1]!='.'){
		cout<<s[1][1]<<endl;
		return;
	}
	if(s[2][0]==s[1][1]&&s[1][1]==s[0][2]&&s[1][1]!='.'){
		cout<<s[1][1]<<endl;
		return;
	}
	cout<<"DRAW\n";
}
```
