## 我又来发题解了-----无力的声音
这道题就用到数位分离的知识了
不明白的这里有[链♂接](https://zhidao.baidu.com/question/1541245027684425907.html)
### 代码如下
```
#include<iostream>//头文件 
using namespace std;
int main()
{
	int a,b,c;//定义 
	cin>>a;
	b=a%10;//求出个位数字 
	c=a/10;//算出十位数字 
	cout<<b+c<<endl;//输出个位和十位的和 
	return 0;
}
```
