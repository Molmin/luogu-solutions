# CF1527A 题解

**一句话题意**：

给定一个整数 $n$ , 要我们求出最大的 $k$ ，使得 n &  (n - 1) &  (n - 2) & …… & k = 0 。


**解题思路** ：

乍一看挺复杂，吓到了我，仔细一看不过是对与运算的一点理解而已。

举个栗子吧，没有栗子不好说。

假设 $ n = 13 $ ，由于是要做与运算，所以说我们将其化成二进制： $13 = 1101$

这时我们将这个数分为两部分：首位的1和后面的所有东西。

为了让结果为 $0$，我们的算式中必定含有首位为 $0$ 的数，为了使 $k$ 最大，我们选 $0111$ ，这个数是 $7$ 。

同时 , 我们将这个数加一 ，就会发现，它变成了 $10000$ ，这时候后面的部分也全部满足了。

所以，$k$ 最大就是将原数化成二进制数后，将首位以后的所有位都改成 $0$ 的那个数减一。

那我们怎么求出这个数呢？

他就是 $a^{\left\lfloor\ log(n) \right\rfloor} -  1$；

综上，代码已出：

```cpp
#include <bits/stdc++.h>
using namespace std;
int t;
int x;
int main(){
 	cin >> t;
	while(t--){
		cin >> x;
		printf("%d\n",(1 << int(log2(x))) - 1);
	}
	return 0;
}
```
