### 题意：
输出200内$a^3=b^3+c^3+d^3$的所有结果。

### 思路：
1.先四重循环：
```
for(int a=6;a<=200;++a)
        for(int b=2;b<=200;++b)
            for(int c=b+1;c<=200;++c)
                for(int d=c+1;d<=200;++d)
```
2.进行直接判断：
```
if(!(x[a]-x[b]-x[c]-x[d]))
     cout<<"Cube = "<<a<<", Triple = ("<<b<<","<<c<<","<<d<<")<<endl;
```
时间复杂度相当于$O(1)$，可以AC。
