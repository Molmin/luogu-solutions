### 思路：
我们输入字符，直接进行判断。

只需要统计 $A$、$K$、$Q$、$J$ 的个数，因为 $X$ 的分值为 $0$，不管有多少张都不计入总分中。

最终输入的时候将字母的个数乘以相应的分值即可。

### 核心代码：
```
	for(int i=1;i<=n;i++) {
		for(int j=1;j<=13;j++) {
			cin>>s;
			if(s=='A') {
				A++;
			}
			if(s=='K') {
				K++;
			}
			if(s=='Q') {
				Q++;
			}
			if(s=='J') {
				J++;
			}
		}
	}	
	printf("%lld",A*4+K*3+Q*2+J);
```
