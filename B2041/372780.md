### 题目大意：
输入两个数 $a$ 和 $b$，如果 $a\geq10$ 或 $b\geq20$ 就输出 $1$ 否则输出 $0$。

---
### 前言：

这是一道很经典的**分支结构**题。

分支结构看起来很简单，但又十分重要，学好它可以为以后的学习打下良好的基础。

现在来介绍两种很基础的分支结构的格式。

#### 分支结构格式1（单分支结构）：
```
if(条件表达式）
    Do something.
```

**功能：** 就是如果条件表达式成立，就会运行它下方的那一行语句，如果不成立就不会运行。

#### 分支结构格式2（双分支结构）：
```
if(条件表达式）
    Do something.
else
    Do something.
```
**功能：** 就是如果条件表达式成立，就会运行它下方的那一行语句，如果不成立就会运行 `else` 下方的那一行语句。

---
### 正文：

这道题需要用到双分支结构，但是，它还需要一个东西，就是逻辑运算符 `||` 。

这个 `||` 就是“或”的意思，它是这样用的：

```
条件表达式1||条件表达式2
```
**功能：** 只要条件表达式 $1$ 或 $2$ 中有一个成立，那这句就都成立。

### 参考代码：
```cpp
#include<bits/stdc++.h>//万能头
using namespace std;
int main(){
	int a,b;
	cin>>a>>b;
	if(a>=10||b>=20)
		cout<<1;
	else
		cout<<0; 
	return 0;//这是个好习惯
}//不抄题解的代码也是个好习惯
```
