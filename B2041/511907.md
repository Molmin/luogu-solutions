[原题传送门](https://www.luogu.com.cn/problem/B2041)

对于这道题，我们可以不用`if...else...`语句，直接输出`a>=10||b>=20`。（用`a`代表印着“幸运”的瓶盖数，用`b`代表印着“鼓励”的瓶盖数。）因为`a>=10||b>=20`本来就是布尔表达式，如果满足条件，结果就是`1`；否则就是`0`。

那么为什么用或运算符`||`呢？因为以下两个条件：

1. `a>=10`

2. `b>=20`

只要满足其中一个就行了，所以用或运算符`||`。

代码如下：

```cpp
#include<iostream>
using namespace std;
int a,b;
int main(){
	cin>>a>>b;
	cout<<(a>=10||b>=19);
	return 0;
}

```
