#### [题 目 传 送 门](https://www.luogu.com.cn/problem/CF1197A)
#### [题意解析]

题目翻译人是真良心awa

题目的翻译几乎直接将思路告诉了我们。

对于 $ n $ 个数，从中选出 $ k+2 $ 个数，我们需要这 $ k+2 $ 个数中的第二大的数大于 $ k $，求出最大的 $k$ 是多少。

#### [思路分析]

对于选出的 $k+2$ 个数，我们只需要知道第二大的数是多少。由于第二大的数要大于 $k$ ，所以我们想要找出最大的 $k$，就要找出最大的第二大的数。由于第二大的数只小于第一大的数，所以想要第二大的数尽可能的大，就要最大的数尽可能的大，那么让最大的数尽可能的大十分简单，只要找出数组中最大的数就好了。

#### [贴上码码]

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,t,a[100010];
int main()
{
	cin>>t;
	while(t--){
		cin>>n;
		for(int i=1;i<=n;i++)
			cin>>a[i];
		sort(a+1,a+n+1);
		if(a[n-1]==1)cout<<"0\n";
		else cout<<min(n-2,a[n-1]-1)<<"\n";
	}
	return 0;
}
```
