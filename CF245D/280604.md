# 一、题意
有一个矩阵 $b$，满足：
- 若 $i\neq j$，则 $b_{i,j}=  a_i \ \&\  a_j$；
- 若 $i= j$，则 $b_{i,j}=-1$。

请你根这个矩阵找到满足要求的 $a$ 数列。

# 二、分析
首先，我们需要了解一下按位与。按位与就是把两个数二进制对齐，同一个位置上都是 $1$ 的话得出的结果该位置就是 $1$，否则就是 $0$。

那么 $b_{i,j}$ 包括了 $a_i$ 和 $a_j$ 所有的位置相同的 $1$。我们先来看一行，不妨设 $i=1$。此时我们发现对于所有不等负一的 $b_{1,j}$，都有 $b_{1,j}=a_1\&a_j$，那么 $b_{1,j}$ 一定是 $a_1$ 里面所有 $1$ 的一个子序列，所以如果任何一个 $b_{1,j}$ 某个位置上有一个 $1$，$a_1$ 那个位置上一定有一个 $1$。这样的话，我们直接把 $b_{1,j}$ **按位或**就能得出 $a_1$ 了。

我们还要再想想按位或了之后要不要再给 $a_1$ 加点东西，我们发现，如果再在 $a_1$ 的某个位置加上一个 $1$ 的话，这个位置的 $1$ 有更大的概率会和 $a_k$ 同一个位置的 $1$ 重复，这样就不能保证原来的结果了。所以直接按位或就行了。

# 三、代码
```cpp
#include <bits/stdc++.h>
using namespace std;
#define MAXN 102
int n,b,a[MAXN];
int main()
{
	cin>>n;
	for(int i=1;i<=n;i++)
		for(int j=1;j<=n;j++)
		{
			cin>>b;
			if(b!=-1) a[i]|=b;
		}
	for(int i=1;i<=n;i++) cout<<a[i]<<' ';
}
```
