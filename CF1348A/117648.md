## 题目大意
给你一个偶数$n$，让你将$2^1,2^2...2^n$分成$2$组(每组的数字个数一样，都是$\frac{n}{2}$)，使这两组的差的绝对值最小。

举个栗子：比如说$n=4$，那我们要把$2^1,2^2,2^3,2^4$分为$2$组，那我们肯定是将$2^1,2^4$分在一起，$2^2,2^3$分在一起，那么他们的差值就是$2^4+2^1-(2^2+2^3)=16+2-(4+8)=18-12=6$

好了，题意理解了我们开始做题。

## 思路
你要使差最小化，那你一定是用最大的那个数减去次大的，次次大的，次次次大的，然后一直这样减掉$\frac{n}{2}$个数，在把那些剩下的小数字加起来，这样才能使差最小化，对不对？那么就相当于$2^n-
\begin{matrix}\underbrace{2^{n-1}-2^{n-2}-\cdots-2^{\frac{n}{2}}}\\\frac{n}{2}\text{个数字}\end{matrix}+\begin{matrix}\underbrace{2^{\frac{n}{2}-1}+2^{\frac{n}{2}-2}+\cdots+2^1}\\\frac{n}{2}-1\text{个数字}\end{matrix}$ 

那么我们就开始求答案吧。

## code:
```
#include<bits/stdc++.h>
using namespace std;
int T;
int n;
long long a[35]={1,2,4,8,16,32,64,128,256,512,1024,2048,4096,8192,16384,32768,65536,131072,262144,524288,1048576,2097152,4194304,8388608,16777216,33554432,67108864,134217728,268435456,536870912,1073741824};
//把2的次幂打个表（其实是我怕TLE……）
int main() {
	scanf("%d",&T);
	while(T--) {
		scanf("%d",&n);
      //aa意义就是上文的最大数，然后一直减减加加的
      //bb意义就是来看剩下了多少数字
		long long aa=a[n],bb=n-1;//因为aa本身就是一个数，所以剩下n-1个数字
      //开始用aa去减了
		while(bb>=n/2) aa-=a[bb--];//减掉n/2个数
		while(bb>0) aa+=a[bb--];//然后把剩下的数字都加起来
		printf("%lld\n",aa);//不开ll=WA？
	}
	return 0;//好习惯！
}
```
