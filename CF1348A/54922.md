这道题乍一看像个贪心题，但是有一个很显然的做法。

首先，显然，$2^1+2^2+...+2^{n-1}=2^n$。

所以显然要尽量让$2^n$和小的数配成一对。

题目上要求两对个数相等。

于是，很显然只需把$2^1,2^2,...,2^{n/2-1},2^n$配成一对，另外配成一对即可。

上代码：
```cpp
#include<iostream>
using namespace std;
int main()
{
	int t,n,i,j,s1,s2;//s1,s2为计数器
	cin>>t;
	for(i=1;i<=t;i++)
	{
		cin>>n;
		s1=0;//计数器清零
		s2=0;//计数器清零
		for(j=1;j<n/2;j++)//如上,第一对
			s1+=(1<<j);
		s1+=(1<<n);//还有2^n
		for(j=n/2;j<n;j++)//第二对
			s2+=(1<<j);
		cout<<s1-s2<<endl;//s1-s2即为答案
	}
	return 0;
}
```
