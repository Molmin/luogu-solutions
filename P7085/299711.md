#   贪心的想法
------------

  想要放矩形最少，那么每一个矩形占的面积就应该最大；
  当一个矩形想占尽可能大的面积的时候（~~你是拦不住的~~）,其实是可以拦的；
  
  **当它占的长度（和宽度）比自身长度（和宽度）的两倍少一点点（都可以忽略不计）的时候；那么他占的面积必然是最大的；** 
  
  注意 ：当你占完了之后，还有位置刚好可以放，那你不得不放。
  
  代码如下：
```
#include<iostream>
using namespace std;
int n,m,n1,m1;
int main(){
   cin >> n >> m >> n1 >> m1;	
   int ansx = n / (2 * n1);
   int ansy = m / (2 * m1);
   if(n - ansx * (2 * n1) >= n1) ansx += 1;
   if(m - ansy * (2 * m1) >= m1) ansy += 1;
   cout << ansx * ansy; 
	return 0;
}
```
