### 一、题意简述

有一个 $8\times8$ 的棋盘，棋盘上有一个女王，想要去到另外一个点。

请问，这个女王最多需要使用多少个**方向**到达目的地？女王可以走八个方向。

比如，从第 $1$ 行第 $1$ 列，到第 $3$ 行第 $3$ 列，既可以先向右走 $2$ 步，再向下走 $2$ 步，使用了 $2$ 个方向；但其实可以向右下走 $2$ 步，只使用 $1$ 个方向。

### 二、思路

在我来之前，此题只有一篇题解，使用的是宽度优先搜索。但我觉得，这道题没必要使用宽度优先搜索。

首先，可以证明，答案一定不超过 $2$。

最坏情况下，先斜着走，使得自己的横坐标或纵坐标与目标一致。然后，再横着走或竖着走，即可到达目标点。

如果两个点位置一致，女王动都不用动，答案为 $0$（废话）。

如果两个点横坐标一致或纵坐标一致，女王只用横着走或竖着走，答案为 $1$。

如果两个点之间可以斜着走直达，答案为 $1$。此时一定有 $abs(x_1-x_2)=abs(y_1-y_2)$。

如果以上条件都不满足，答案为 $2$。

### 三、完整代码

```cpp
#include<cstdio>
using namespace std;
int x1,y1,x2,y2;
inline int abs(int a)
{
	return a>=0?a:-a;
}
int main()
{
	while(scanf("%d%d%d%d",&x1,&y1,&x2,&y2)&&x1&&y1&&x2&&y2)
	{
		if(x1==x2&&y1==y2)//两点重合。
		{
			printf("0\n");
			continue;
		}
		else if(x1==x2||y1==y2)//横坐标或纵坐标一致。
		{
			printf("1\n");
			continue;
		}
		else if(abs(x1-x2)==abs(y1-y2))//斜着走直达。
		{
			printf("1\n");
			continue;
		}
		else//均不符合。
		{
			printf("2\n");
			continue;
		}
	}
	return 0;//完结撒花！
}

```
