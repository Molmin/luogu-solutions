[题目](https://www.luogu.com.cn/problem/P9278)

### 解法：

首先题目声明双方都以**最优策略**玩游戏。 

最优策略：

当你移动石子导致只剩下最后一堆时，你就输了，所以要让对方移动除最后一堆以外的最后一个石子。

如果你是先手，你只要移动每一堆除最后一个以外的所有石子就行了，这样对方就不得不移动剩下的那一个，重复进行下去你就会获得胜利；

但是，如果当第一堆只有一个石子的话，你只能移动这一个，这样就相当于对方获得了先手，他按照刚才的方式便会使你失败。

所以我们分两种情况：  

1.$a=2$

>这时只要 Charlie 将第一堆的所有移走就行了。

2.$a>2$

>如果第一堆不为 1，Charlie 便会用最优策略获胜 。 

>当第一堆为 1 时，Dan 便会以他的最优策略获胜。

因此，程序思路如下：

1.输入 $a,b$ 表示一共几堆和第一堆是什么；

2.当 $a=2$ 或 $b≠1$ 时直接输出 Charlie 获胜；否则输出 Dan 获胜。

### 代码:


 ```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
	int a,b;
	cin>>a>>b;//输入一共几堆和第一堆个数 
	if(a==2||b!=1)//判断谁会获胜 
		cout<<"Charlie"<<endl;//输出获胜者 
	else
		cout<<"Dan"<<endl;
	return 0;
}
```
