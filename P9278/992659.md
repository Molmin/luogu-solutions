简单博弈论题。

首先需要判断 $n=2$ 的情况：先手可以将第一堆石子全部移动到第二堆石子，显然先手必胜。

否则，先手可以拿第一堆 $a_1 - 1$ 个石子，此时这一堆只剩下了一个石子。

那么后手就只能转移最后一个石子。

同理，先手可以移动第二堆 $a_2-1$ 个石子，这样第二堆也只剩下了一个石子。那么后手只能移动最后一个石子。先手继续移动第三堆的 $a_3-1$ 个石子……

所以先手一直按照这个策略进行下去之后就必胜。

但是如果 $a_1=1$ 那么等于是一开始 $a_1=2$ 然后后手先拿了一个石子，这个时候后手必胜。

时间复杂度 $O(n)$。

但是可以优化到 $O(1)$。

具体见代码。

```cpp
#include <bits/stdc++.h>
using namespace std;
signed main()
{
  int n, m;
  cin >> n >> m;
  cout << ((n == 2 || m != 1) ? "Charlie" : "Dan") << '\n';
}
```
