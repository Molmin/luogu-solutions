### 听说在考前发题解RP++呢


------------


### 题目大意

 有一个投球手来投球，如果他的前脚在线的前面，那这就是一个坏球。
       
 但是在有些时候，摄像机会坏掉，此时给的图像会旋转 $180\degree$ 。
     
 有 $t$ 组数据，每一组会给定一个 $5 \times 5$ 字符矩阵 ，其中只会包含 `.` `|` `<` `>` 。
 
 `.` 代表空地，`|` 代表那条线，保证所有的 `|` 都在一列上，`>` 代表投球手的前脚，
 
 `<` 也代表投球手的前脚但是此时摄像机坏了（也就是说当前的图像是原来的图像
 
 旋转 $180\degree$ 之后的样子）
 
### 解题思路

 用两个变量 $a$ , $b$ 来记录线和投球手在的那一列，如果摄像机没有坏就可以直接比较
 
 了。摄像机如果坏了，此时就需要翻转，但是不用暴力，可以列个表来找找规律。
 
 | 原来 | 翻转后 |
| -----------: | -----------: |
| 1 | 5 |
| 2 | 4 |
| 3 | 3 |
| 4 | 2 |
| 5 | 1 |
 
 可以发现原来加上翻转后的数字一定是 $6$ ，所以用 $6$ 减去其中一个数字就可以
 
 得出另外一个了。
 
### 代码

```cpp
#include<iostream>
using namespace std;
int t, a, b;
bool f;
char k[10][10];
int main()
{
	cin >> t;
	for (int s = 1; s <= t; s++)
	{
		f = false; //这里的 f 代表摄像机坏没坏
		for (int i = 1; i <= 5; i++)
		{
			for (int j = 1; j <= 5; j++)
			{
				cin >> k[i][j];
				if (k[i][j] == '|')
					a = j;
				else if (k[i][j] == '>')
					b = j;
				else if (k[i][j] == '<')
					f = true, b = j; 
			}
		}
		if (f)
		{
			a = 6 - a; //如果摄像机坏了，就翻转一下
			b = 6 - b;
		}
		cout << "Case " << s << ": ";
		if (b > a)//如果前脚在线的前面
			cout << "No Ball" << endl; 
		else
			cout << "Thik Ball" << endl;
	}
	return 0;
}
```
