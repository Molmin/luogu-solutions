# 1.分析

1. 变量：$n$是一元$n$次多项式的最高项次数，$a$用来存储多项式的各次项的系数，注意系数有$n+1$个$($包括常数项$)$；

2.输出： 

- 不输出：若系数$a[i]==0$，不输出此项$(continue)$；

- 符号：若系数为正数$(a[i]>0)$且非首项$(i!=0)($首项不要输出$"+")$，输出$"+"$；若为$-1$且非最后一项$($其它负数输出时会输出$"-"$，但$-1$非最后一项时只输出$"-")$，输出$"-"$；

- 系数：系数非$1(1$无需输出$)$也非$-1(-1$只需要输出$"-")$时$(a[i]!=1\&\&a[i]!=-1)$，输出系数；

- 常数项$($次数为$0)$：非$1$非$-1$的系数已经输出过，若为$1$或$-1$，输出，跳出循环$(break)$，不输出$x$的几次幂；

- $x$的几次幂：若次数为$1$，只需输出$x$；若次数不为$1$也不为$0$，输出$n-i$次幂。

# 2.代码

```cpp
#include <iostream>

using namespace std;

int n;      //次数
int a[105]; //系数

int main(void)
{
	cin >> n;
	
	for(int i = 0; i <= n; i++)
	{
		cin >> a[i];
	}
	
	for(int i = 0; i <= n; i++)
	{
		//系数为0，不输出
		if(a[i] == 0)
		{
			continue;
		}
		
		//符号
		if(i != 0 && a[i] > 0)
		{
			cout << "+";
		}
		if(i != n && a[i] == -1)
		{
			cout << "-";
		}
		
		//输出系数
		if(a[i] != 1 && a[i] != -1)
		{
			cout << a[i];
		}
		
		//常数项
		if(i == n)
		{
			//最后一项是1或-1，要格外输出
			if(a[i] == 1 || a[i] == -1)
			{
				cout << a[i];
			}
			break;
		}

        //输出幂
		cout << "x";
		if(n - i != 1)
		{
			cout << "^" << n - i;
		}
	}
	
	return 0;
}
```
