思路：先枚举第一次延长，如果第一次可以延长则输出延长一次后的数，因为一个数乘10除以一个数的余数是不变的，所以我们可以再输出n-1个0啦。如果第一次不可以延长，就输出“-1”了。

代码。
```c
#include<bits/stdc++.h>
using namespace std;
int main(){
	int a,b,n,i,t=1;
	cin>>a>>b>>n;
	for(i=0;i<=9;i++)//枚举第一次延长
		if((a*10+i)%b==0){//如果可以延长，则标记t为0，并进行延长和退出此层循环
			t=0;
			a=a*10+i;
			break;
		}
	if(t)cout<<"-1";//如果第一次不可以延长，就输出-1
	else{//否则输出a并输出剩下的0
		cout<<a;
		for(i=1;i<n;i++)
			cout<<0;
	}
}
```
