考察算法：贪心。


主要思路：

1. 根据题意读入（这个应该都会吧）

1. 把石头的重量排序。因为要用最少的石头让水溢出箱子，所以一定是要先挑最重的石头往里面放，就一定是最优解。（注意：如果用`sort`排序，排序的结果是小的数在前，所以后面一定要倒着循环才是从大到小）

1. 循环加入石头，从最重的开始加，如果水溢出了就输出答案，然后跳出循环。

代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a,b,c,m,x;//n:数据组数；a,b,c:箱子的长宽高；x:箱子中的水量；m:石头的数量；d:每块石头的重量
int main()
{
	cin>>n;
	for(int i=1;i<=n;i++)//注意多组数据
	{
		cin>>a>>b>>c>>x>>m;//读入
		int d[m+1];
		for(int j=1;j<=m;j++) cin>>d[j]; 
		sort(d+1,d+m+1);//按石头的重量排序
		for(int j=m;;j--)//倒着循环
		{
			x+=d[j];//放入石头
			if(x>c)//判断是否溢出（不要写成>=了）
			{
			    cout<<m-j+1<<endl;//输出（记得换行）
			    break;//跳出循环
			}
		}
	}
	return 0;
}//完美结束QWQ

```
