- 这道题的翻译有些不准，我再来翻译一下：艾伦有很多钱。他在银行有存款。出于安全原因，他想以现金的形式取款(我们在此不透露原因)。美元面值分别为$1$,$5$,$10$,$20$,$100$。艾伦取款后能收到的最小账单数是多少?

- 所以，我们只需要用一点贪心，通过$while$循环来从大到小连续取。

- 例如：$127->100* 1->27->20* 1->7->5* 1->2->1* 2->0$
所以，一共用了$1+1+1+2=5$次，输出5。


```
#include<bits/stdc++.h>
using namespace std;
int main(void){
	ios::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);
	int n,m=0;
	cin>>n;
	while(n>=100&&n!=0){//这个数必须要大于100或20或10或5或1且没有被取完（不等于0）
		n=n-100;
		m++;//每一次加一
	}
	while(n>=20&&n!=0){
		n=n-20;
		m++;
	}
	while(n>=10&&n!=0){
		n=n-10;
		m++;
	}
	while(n>=5&&n!=0){
		n=n-5;
		m++;
	}
	while(n>=1&&n!=0){
		n=n-1;
		m++;
	}
	cout<<m;//输出
	return 0;
}
```
