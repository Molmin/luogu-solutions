这个题是一个最值问题。

首先我们先来模拟一下样例。
```
125
```
为了让答案尽可能小。

我们先定$k_5$是多少。

$125/100=1……25$,那么$k_5=1,n=125-1*100=25$。

我们在来看看$k_4$是多少。

$25/20=1……5$,那么$k_4=1,n=25-1*20=5$。

我们在来看看$k_3$是多少。

$5/10=0……5$,那么$k_3=0,n=5-0*10=5$。

我们在来看看$k_2$是多少。

$5/5=1$,那么$k_2=1,n=5-1*5=0$。

我们在来看看$k_1$是多少。

$0/1=0$,那么$k_1=0,n=0-0*1=0$。

输出$k_1+k_2+k_3+k_4+k_5=3$。

思路已经很清晰了，接下来看看代码吧。
```
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int sum=0; 
	int n;
	cin>>n;
	sum+=n/100;//n从100开始才能保证答案最小。
	n=n%100;//n最后=n%100，因为被除掉部分已经不算n的了。
	sum+=n/20;//计算k4
	n=n%20;
	sum+=n/10;//计算k3
	n=n%10;
	sum+=n/5;//计算k2
	n=n%5;
	sum+=n;//注意，k1不用那么复杂，只要直接+n就可以了。
	cout<<sum; 
	return 0;
}
```
