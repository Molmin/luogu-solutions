首先，我们来分析一下题目（学过高数的大佬可直接跳过此段）。

给出一个数 $n$ ，令 $n=k_1+5\cdot k_2+10\cdot k_3+20 \cdot k_4+100\cdot k_5$ ，$k \in \mathbb{N}^*$ 。

求最小化 $\text{ans}=\sum\limits_{i=1}^5k_i$ 。

这里用了一些高等数学的符号，比如$\sum\limits_{}$，$\in$，等等符号，但是我们不用慌张，因为原题目非常简单。

艾伦有很多钱。他银行里有$n$美元。出于安全原因，他想提取现金(我们不会在此披露原因)。一美元钞票的面值是1，5，10，20，100。艾伦提取全部余额后能收到的最低账单数是多少？
(机翻)

简单来说就是$n$元，每次取钱只能提1,5,10,20,100元，这样题目就相对简单了。

这是一道数学题，所以我们不需要思前想后直接从大到小进行寻找是否可以取一次钱（贪心算法）。

代码呈上。
```cpp
#include<iostream>//我使用cin,cout，所以要用iostream库。
using namespace std;
int main()
{
	int n,ans=0;//n为原本钱数，ans为取钱次数。
	cin>>n;
	while(n>=100)//我们从大到小判断，首先看能否取100元。
	{
		n-=100;//可以取，先取出100元。
		ans++;//次数加一。 
	}
	while(n>=20)//接下来判断是否能取20元。
	{
		n-=20;//可以取，取出20元。
		ans++;//次数加一。
	}
	while(n>=10)//以上类推。 
	{
		n-=10;
		ans++;
	}
	while(n>=5)
	{
		n-=5;
		ans++;
	}
	while(n>=1)
	{
		n-=1;
		ans++;
	}
	cout<<ans;//输出答案 
	return 0;//这里在我的编译器里不用写return 0;但是最好要写，在某些编译器里会出错。 
}
```

本篇题解基本结束了，如果看这篇题解的您只是为了做出题目，到这里您就可以停止了。

我们需要再将程序加速（丧心病狂）。虽然这道题非常不卡常，但是我们还是要加速。

代码呈上。
```cpp
#include<iostream>
using namespace std;
int main()
{
	int n,ans=0;
	cin>>n;
	if(n>=100)//改成if语句不用多次循环。 
	{
		ans+=n/100;//一次取到底，不需循环。 
		n-=n/100*100;//注意，这里按照算术优先性先算n/100。int形除法自己去除小数部分。 
	}
	if(n>=20)//同理，这里不讲解了 
	{
		ans+=n/20;
		n-=n/20*20;
	}
	if(n>=10) 
	{
		ans+=n/10;
		n-=n/10*10;
	}
	if(n>=5)
	{
		ans+=n/5;
		n-=n/5*5;
	}
	if(n>=1)
	{
		ans+=n/1;
		n-=n/1*1;
	}
	cout<<ans;//同样输出 
	return 0; 
}
```
