类似于一道数学的最值问题  
所以当然是从大的（100美元）开始取然后一直减小直到1  
下面贴代码：
```cpp
#include<iostream>
using namespace std;
int main(){
	int n,ans=0;
	cin>>n;
	ans+=n/100;        //100的张数
	n%=100;            //取完100剩余的
	ans+=n/20;         //20的张数
	n%=20;             //取完20剩余的
	ans+=n/10;         //10的张数
	n%=10;             //取完10剩余的
	ans+=n/5;          //5的张数
	n%=5;              //取完5剩余的
	ans+=n;            //这里直接加就行了，因为只有1的了
	cout<<ans;
}
```
