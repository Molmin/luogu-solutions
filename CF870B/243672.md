### 题目大意：

------------

给你一个长度为 $n$ 数组，让你把他分成 $k$ 个非空子数组，问你这些非空子数组中最小值中最大是多少。

## 大体思路
主要分为三种情况，需要分类讨论。

### ① $k=1$   
把这个数组分成 $1$ 个非空子数组，相当于把原数组取最小值，**答案为最小值**。

------------


### ② $k=2$   
在这里卡了很久，其他题解并没有讲的很清晰。首先我们要对一个结论很清楚： **如果这个数在其中一个子数组，那这个子数组最小值一定小于或等于这个数** 。之所以有这个结论，是因为最小值比除他之外的每个数都更小。  

$k=2$ 时，原数组的首和尾一定在不同的子数组里。此时，便可以运用这个结论，可以得到两个子数组的最大值只能来自首和尾的两个数值。 

所以**答案就是首和尾的两个数值的最大值**。

------------

### ③ $k \ge 3$
把最大值直接切出来，**答案为最大值**。


### 代码
```
int n,k,maxi=-2e9,mini=2e9,a[100005];
cin>>n>>k;
for(int i=1;i<=n;i++)
{
	int x;
	cin>>a[i];
	maxi=max(maxi,a[i]);
	mini=min(mini,a[i]);
}	
if(k==1)
	cout<<mini;
else if(k==2)
	cout<<max(a[1],a[n]);
else
	cout<<maxi;
```
