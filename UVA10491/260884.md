**题意**
- [如果你能看懂英文题面的话，我要说一句，原题还是挺有趣的](https://www.luogu.com.cn/problem/UVA10491)。
- 在你面前有 $a+b$ 个门，其中 $a$ 扇门里是奶牛，$b$ 扇门里是轿车，你选定其中的一扇门，想要获得里面的东西（显然不是奶牛），选定后门不会立刻打开，此时主持人会另外再开 $c$ 个有奶牛的门，如果你改变你的决策，输出获得轿车的概率（保留小数点后 $5$ 位小数）。
- $0< a,b$ 且 $0\le c<a$。

**分析**
- 如果你不改变你的决策，概率显然是 $\dfrac b{a+b}$，但改变了决策答案会改变吗，答案是肯定的，让我们来用严谨的方法求出这个概率吧。

**知识**
- 记 $P(A|B)$ 表示在 $B$ 发生的前提下 $A$ 发生的概率，$P(AB)$ 表示 $A,B$ 都发生的概率。
- 全概率公式：如果将样本空间划分为 $B_1,B_2,\cdots B_n$ 一些互不相交的事件那么：
$$P(A)=\sum_{i=1}^nP(A|B_i)P(B_i)$$
- 其实没啥复杂的，举个例子：
- 比如说你知道你参加 $\text{NOI}$ 得 $\text{Au}$，$\text{Ag}$，$\text{Cu}$ 的概率分别是 $0.2,0.3,0.5$，分别被你妈表扬的概率为 $1.0,0.8,0.5$，所以你被你妈表扬的概率为 $0.2\times 1.0+0.3\times 0.8+0.5\times 0.5=0.69$。

**思路**
- 打开 $c$ 个门之后，有 $a-c$ 个牛，$b$ 个轿车，接下来的问题就是你到底一开始选了牛还是轿车。
- 记一开始选牛为 $B_1$，一开始选轿车为 $B_2$，记选到轿车为 $A$。
- $P(B_1)=\dfrac{a}{a+b}$，$P(B_2)=\dfrac{b}{a+b}$。
- $P(A|B_1)=\dfrac b{a+b-c-1}$，$P(A|B_2)=\dfrac{b-1}{a+b-c-1}$

$$P(A)=P(A|B_1)P(B_1)+P(A|B_2)P(B_2)$$
$$=\frac{b}{a+b-c-1}\cdot \frac{a}{a+b}+\frac{b-1}{a+b-c-1}\cdot \frac{b}{a+b}$$
$$=\frac{(a+b-1)b}{(a+b-c-1)(a+b)}$$
- 所以输出答案即可。

**代码实现**
```cpp
#include<fstream>
using namespace std;
int a,b,c;
int main()
{
	while(~scanf("%d%d%d",&a,&b,&c))
		printf("%.5lf\n",1.0*(a+b-1)*b/(a+b-c-1)/(a+b));
	return 0;
} 
```
**坑点**
- 后来我与同学讨论了一下，发现他认为保持和另选的概率是一样的，其实这就是不清楚 $P(A|B)$ 和 $P(A)$ 的区别。
- 举个例子：$3$ 位囚犯有两位将要被处死，囚犯甲询问法官哪位囚犯将要被处死，出于人道，法官不能透露有关甲的信息，于是法官说囚犯乙会被处死，囚犯甲感到很轻松，因为他觉得他和丙被处死的概率相等，他被处死的概率由 $\dfrac 23$ 变成 $\dfrac 12$了，囚犯甲的想法对吗？
