这题非常简单qwq。

首先我们来模拟一下样例。

```
5 50
1 2 3 4 5
```
首先我们看到有$50$份水，而$5$个瓶子里有$1,2,3,4,5$份。

然后求出他们的和为$1+2+3+4+5+50=65$份水。

然后求出平均数为$65/5=13$。

然后一个一个输出。

$13-1=12$

$13-2=11$

$13-3=10$

$13-4=9$

$13-2=8$

就可以完美的做出答案了。

但是这里还有一个小细节，就如何判断判断这个数据是无解的。

其实很简单，只要 平均数$-$这个水的数量 小于$0$的话，就删除$-1$。

接下来看看我的代码。

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int n,m;
	cin>>n>>m; 
	double sum=m;
	double a[n];
	for(int i=0;i<n;i++)
	{
		cin>>a[i];
		sum+=a[i];
	} 
	sum=(sum*1.0)/n;//求平均数。
	for(int i=0;i<n;i++)
	{
		if(sum-a[i]<0)
		{
			cout<<-1;
			return 0;
		}//如果平均数-水的数量小于0，输出-1，跳出程序。
	}
	for(int i=0;i<n;i++)
	{
		printf("%0.6f\n",sum-a[i]);//输出要加入水的分量，注意，要保留6位小数。
	}
    return 0;
}
```
