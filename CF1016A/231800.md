## 题解 CF1016A

**思路 ： 模拟**

这题简单模拟即可，即使这题那么~~水~~简单，但是还有可以优化的地方

- 优化 ：

还是从时间和空间上来说

1. 空间： 首先就是数组的问题，数组完全可以简化掉，变成一个变量 $temp$ 即可，这样就大大优化了空间的使用

2. 时间：配合上面优化过的数组，随着输入及时处理，然后立刻输出，可以优化掉时间复杂度（虽然只有常数级别的优化）

- 注意事项：

1. 这题好像不需要开 $longlong$ ，正常的 $int$ 即可

2. 内部变量要赋初值

3. 不需要快读

**下面附上仅仅10行的极简AC代码**

```
#include<iostream>
using namespace std;
int main(){
	int n , m , temp , answer = 0;
	cin >> n >> m;
	for(int i = 1;i <= n;i ++){
		cin >> temp; answer += temp;cout << answer / m << ' '; answer %= m;
	}
	return 0;
} 

```
