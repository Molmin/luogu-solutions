**题意：**

在一本书上写名字，每页上最多写 $m$ 个名字，要连续写 $n$ 天，每天可以写 $A_i$ 个名字，如果有一页没写完的话，第二天可以接着上一页继续写，求每天各翻了几页（**注：如果有一天正好把一页写完了，在这一天就需要翻页**）。

---
**分析：** 

按照题意输入 $n,m$ ，然后用 for 循环，输入今天要写的名字数，然后直接对这个数进行运算即可。

---
**代码实现：**

定义一个变量，用于存储现存的名字数（即上一页遗留下来的和现在输入的总和，可能这不太好理解，一会上代码就可以直观的看出来了），然后除以每页可容纳的名字数量，输出，最后再转换成现在的一页上剩余的名字即可。

---
上代码

```cpp
#include<iostream>
using namespace std;
int main()
{
	int n,m,mingzi=0;  //mingzi为现存名字数
	cin>>n>>m;
	for(int i=1;i<=n;i++)
	{
		int k;  // k 为每天输入的要写的名字数
		cin>>k;
		mingzi+=k;  //现存名字数加上今天要写的名字数
		cout<<mingzi/m<<" ";  // int 自带向下取整，所以输出的就是翻的页数
		mingzi%=m;  //变成现在的一页上有几个名字
	}
	return 0;
}
```
