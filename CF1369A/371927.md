此题是一道经典的找规律题，题目让我们判断多边形是否满足至少有一对轴是垂直的。我们来枚举下那些多边形满足这个要求即可。通过推到，我们可以十分惊奇的看出，四边形，八边形，十二边形，十六边形等边数为四的倍数的多边形符合要求。那么，此题就变成了一道简单的倍数判断题，用边数“模”四，把每组数据查找下即可。

| 图形 | 是否可以 |
| :----------: | :----------: |
| 三角形 | 不行 |
| 四边形 | 行 |
| 六边形 | 不行 |
| 七边形 | 不行 |
| 八边形 | 行 |
| 九边形 | 不行 |
| 十边形 | 不行|
| 十一边形 | 不行 |
| 十二边形 | 行 |
| 十三边形 | 不行 |
| 十四边形 | 不行 |
|  ... | ... |

```cpp
#include<bits/stdc++.h>
using namespace std;
int n;
int main(){
    cin>>n;//输入
    while(n--){//循环n次
        int tmp;//循环内临时变量
        cin>>tmp;
        if(tmp%4==0)//如果除以4为0，代表为4的倍数，那就输出YES
            cout<<"YES";
        else
            cout<<"NO";//否则输出NO
    }
    return 0;
}
