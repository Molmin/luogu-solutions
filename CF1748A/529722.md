# CF1748A The Ultimate Square 题解

## 文字题解

### 题目大意

有 $n$ 块木板，第 $i$ 块的大小为 $\lfloor\frac{i}{2}\rfloor$，选出若干块拼成的最大正方形的边长为几。

### 解析

每一块矩形木板的长度是：

$$\{1,1,2,2,3,3...\}$$

可以发现，规律是每两个一组，为同一个数。明显地，我们可以横着拼，将上述序列头、尾组合在一起，即 $(1,k),(2,k-1)...$，其中 $k$ 指答案。

也就是说，按这种方法组合的结果为 $\lfloor\frac{n+1}{2}\rfloor$。


**注意事项：**

- 本题多测。

### 核心代码

```cpp
int n;
cin>>n;
cout<<int((n+1)/2.0)<<endl;
```
