## 题目概括
这个题目说的已经很清楚了，纠正翻译一个错误。英文题目是向上取整，这里变成向下取整了，注意一下就行了。
## 解题思路
首先如果毫无头绪可以从代数的角度找规律。

- 输入为 1，输出为 1
- 输入为 2，输出为 1
- 输入为 3，输出为 2
- 输入为 4，输出为 2
- 输入为 5，输出为 3

（后续省略）

通过上面数据我们不难发现，1 的输出值为 1，之后没两项结果就增加 1，那我们不妨先假设这个规律成立，那么我们就可以得出结论，对于正整数 $n$ 的输出为 $\lfloor (n+1) \div 2 \rfloor$。

那我们通过代数找规律已经得出一个暂定的结论，接下来我们就从几何的角度证明他。

首先，由于我们这个是正方形，我们就需要让他的长和宽同时增加。而由于每一项的长是 $\lceil n \div 2 \rceil$，所以相邻项的长的差距不可能大于 1，是 +0+1+0+1 这样的情况，也就是说相差不大，那么我们就考虑竖着加一个，横着加一个。那么加入有10个板子（$n=10$）那么就是如图所示的操作。
![](https://img-blog.csdnimg.cn/1acfbd9f4eca47c780246de9860354c9.png)
最终我们不难发现从几何方面得出的结论也是$\lfloor (n+1) \div 2 \rfloor$。而且几何方面已经不是一个猜想，而是对他进行了证明，所以我们就能确定的得出对于正整数 $n$ 的输出为 $\lfloor (n+1) \div 2 \rfloor$。
## 通过代码
这个代码很简单，就不多些注释了。
```cpp
#include<iostream>
#include<cstdio>
using namespace std;
typedef long long ll;//为了以防万一还是开个long long 
int main()
{
	ll t,n;
	cin>>t;
	while(t--)//循环t次 
	{
		scanf("%lld",&n);//等同于cin>>n 
		cout<<(n+1)/2<<endl;//输出得出的结论 
	}
	return 0;
}
```
