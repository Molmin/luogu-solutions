### $\sf Code$

先找规律。。

```cpp
#include<bits/stdc++.h>
using namespace std;
long long n=5;
int main() {
	cout<<"1^5=5"<<endl;
	for(int i=2; i<=100; i++) {
		n*=5;
		cout<<i<<"^5="<<n<<endl;
	}
	return 0;
}
```
