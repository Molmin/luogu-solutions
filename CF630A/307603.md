发现$5^n$的后两位都是25（除$5^1$外）

而且因为$n >= 2$

所以我们直接连输入都不用，直接输出25即可。

```cpp

#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>

	cout<<25;//直接输出25
 	return 0;
}
```
