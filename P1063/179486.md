# 比较典型的区间DP
首先这道题本质来说~~不难~~（毕竟DP只是状态转移方程稍微难推一点）况且我这样的蒟蒻都写出来了...

------------
考虑到项链是环状首尾相连的，那么就会考虑使用区间DP，所以就会采用 **破环为链** 的操作，但是此处能量聚合出现了 **头标记** 和 **尾标记**：

------------
```cpp
	for(int i=1;i<=n;i++)
	{
		cin>>head[i];
		if(i!=1)//排除掉1的特殊情况;
		   tail[i-1]=head[i];//处理头标记 标记 上一个尾标记;
	}
	tail[n]=head[1];//单独处理1，n环形连接的端点;
```

------------

处理完环形的连接与标记,因为是** 破环为链** 操作，所以进行断链的位置也不知道，所以为了方便处理接下来断链后项链的完整，做下面的对当前标记数组**向后复制**：

------------
```cpp
	for(int i=1;i<=n;i++)
	{
		head[i+n]=head[i];//将头标记复制到后面;
		tail[i+n]=tail[i];//将尾标记复制到后面;
	}
```
------------
最后处理一下**自己不能与自己聚合**的情况：

------------
```cpp
	for(int i=1;i<=2*n;i++)
	    f[i][i]=0;//因为不存在聚合自身，所以处理为0;
```
------------
初始工作做完了，不妨推一下状态转移方程。
1. 首先从断点开始，将两个能量珠化为一个区间，即产生能量=左端点能量珠头标记 * 左端点能量珠尾标记（右端点能量珠头标记） * 右端点能量珠尾标记；
1. 出现第三个能量珠，此时产生能量=（左端单个能量珠区间+右边两个能量珠区间产生的能量）
1. 以此类推...

可以得到**状态转移方程**

------------
```cpp
f[l][r]=max(f[l][mid]+f[mid+1][r]+head[l]*head[mid+1]*tail[r],f[l][r]);
//左区间合并后的能量+右区间合并后的能量+两区间合并产生能量,不操作的能量;
```
------------
所以整个题大概分析完了

完整代码：


------------
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,head[1001],tail[1001];
int f[1001][1001];
int main()
{
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		cin>>head[i];
		if(i!=1)
		   tail[i-1]=head[i];
	}
	tail[n]=head[1];
	for(int i=1;i<=n;i++)
	{
		head[i+n]=head[i];
		tail[i+n]=tail[i];
	}
	for(int i=1;i<=2*n;i++)
	    f[i][i]=0;
	for(int i=1;i<=n;i++)//枚举可能的断点;
	{
		for(int l=1;l+i<=2*n;l++)//左端点;
		{
			int r=l+i-1;//右端点;
			for(int mid=l;mid<r;mid++)//中点，分为两个区间; 
                f[l][r]=max(f[l][mid]+f[mid+1][r]+head[l]*head[mid+1]*tail[r],f[l][r]);
        } 
	}
	int ans=0;
	for(int i=1;i<=n;i++)//枚举所有长度为n的项链，选出能量最大的;
	{
		ans=max(ans,f[i][i+n-1]);
	}
	cout<<ans; 
	return 0;
}
```
