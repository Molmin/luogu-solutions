好的，那么当我们点开这道题，许多人可能就会像我一样认为：啊，他不是用这 $w$ 个和 $b$ 个摆下来就可以了吗？

实际上，他是这个意思：

我们有 $w$ 个白色多米诺骨牌和 $b$ 个黑色多米诺骨牌，大小为 $1 \times 2$。问这些多米诺骨牌到底能不能放进大小为 $2 \times n$ ，第一行在最左侧有连续的 $k_1$ 个白色的单位面积，第二行在最左侧有连续的 $k_2$ 个白色的单位面积的平面内，使得所有白色多米诺骨牌都被放置在白色区域内，所有黑色多米诺骨牌都被放置在黑色区域内。

### 思路
---
像这种题，我们其实没有必要思考每一块到底是如何摆放的，因为我们稍加思索就会发现，只要总块数，也就是 $k_1 + k_ 2$ 比 $2 \times w$ 多，那么在 $k_1$ 和 $k_2$ 无论取到何值，都可以顺利摆下那 $w$ 块多米诺骨牌。

黑色同理。

既然有了思路，那我们便可以着手写代码了。

### AC Code
---
```cpp
#include<bits/stdc++.h>

using namespace std;

int main()
{
    int t;  //多组数据
    cin >> t;
    while (t--)
    {
        int n, k1, k2, w, b;
        cin >> n >> k1 >> k2 >> w >> b;
        if ((k1 + k2) / 2 >= w && (2 * n - k1 - k2) / 2 >= b) cout << "YES" << endl;  //如果能够满足如上要求，输出 YES
        else cout << "NO" << endl;  //否则输出NO
    }
    return 0;
}
```
