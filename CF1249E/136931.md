**注：题目翻译中的“1楼到每层楼”应该是“上一层楼到每一层楼”，而且输出的是从1楼到每层楼的最少时间花费。**

DP思想，对于每一层楼，计算出上一层楼到这层楼坐电梯与楼梯时间的最小值，再加上这一层楼的时间，输出即可。

对于求上一层楼到这一层楼坐电梯的时间，在上一层楼有两种决策即“走楼梯”或“走电梯”上一层楼到这一层楼坐电梯的时间就是走楼梯+等电梯时间和走电梯时间的最小值。

同理，上一层楼到这一层楼走楼梯的时间就是走楼梯时间和走电梯时间的最小值。

如果在上上层楼到上一层楼的过程中走楼梯最优，但到这一层楼的时候还需要等一个超级长的电梯时间，反而坐电梯最优，怎么办？因为输出的是到每层楼，你不需要保证到 $n+1$ 层楼的路径前 $n$ 步一定与到 $n$ 层楼的路径吻合。

$Code:$

```cpp
#include<bits/stdc++.h>
using namespace std;
int dp[200005][2],e[200005],s[200005];
int main()
{
	int n,c;
	cin>>n>>c;
	for(int i=2;i<=n;i++)cin>>s[i];
	for(int i=2;i<=n;i++)cin>>e[i];
	//e[i],s[i]代表从第i-1层走到第i层 
	cout<<"0";//第一层到第一层的时间为0
	dp[1][1]=999999999;//第一层只能走楼梯
	for(int i=2;i<=n;i++)
	{
		cout<<" ";
		dp[i][0]=min(dp[i-1][0],dp[i-1][1])+s[i];//如果走楼梯
		dp[i][1]=min(dp[i-1][0]+c,dp[i-1][1])+e[i];//如果坐电梯
		cout<<min(dp[i][0],dp[i][1]);
	}
} 
```
