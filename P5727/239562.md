这道题整体来讲还是比较简单的，直接模拟就行。不过好像大家都是用while循环做的，我就来一波for循环。至于倒序输出，我们可以应用for循环的特性。先来看看for循环是怎么用的：
```cpp
for(a;b;c){
    ......
}
```
其中a是初始化语句，只运行一次。您可以初始化一个变量,或定义一个变量,或者什么都不写，如：
```cpp
i=0

int i=0

```
而b是一个条件语句，每当循环一次就判断一次。只有这个语句为真，才可以运行......部分的代码，也可以不写，如：
```cpp
n>m

n<m

m>=n

(ans>=0)&&(m<n)

```
c语句每当循环一次就运行一次。一般与b条件有所关联，同样可以不写，如：
```cpp
i++
i--

```
而我用到的for循环有点不一样，没有a语句，c语句与b语句也没有关联。可c语句却是倒序输出的核心。上代码！
```cpp
#include <iostream>
#include <cstdio>
#include <algorithm>
#include <cstring>
#include <cmath>
using namespace std;
int main(){
    long long n=0;
    long long ans[1000];
    int i=1;             //注意这个变量！！
    cin>>n;
    ans[0]=n;
    for(;n>1;i++){       //i既作为数组下标，又作为操作次数
	if(n%2==0){          //按题意模拟
	    n/=2;
        }else{
	   n*=3;
	   n+=1;
	}
	ans[i]=n;
    }
    i-=1;
    for(;i>=0;i--){      //利用i倒序输出
	cout<<ans[i]<<' ';
    }
    return 0;
}
```
相信很多小伙伴都看不懂
```cpp
i-=1;
```
