## 题目大意
给定长度为 $n$ 的括号串，可以将相邻的括号交换位置。定义括号匹配前缀长度指最大的 $k$ 使得字符串前 $k$ 位都是合法的括号匹配串。求交换后最大括号匹配前缀。

## 思路解析
方法 1：可证，经过若干个步骤，一定能将所有可以成功匹配的字符串放在最前面。

即：一个非匹配括号串不能经过交换变成匹配括号串，那么这个字符串一定全由左括号或右括号组成。

那么我们只要求出串内左右括号数量的差，用总长减去即可。

方法 2：同样地推导结论，得出答案为 $2\times min(a,b)$。

## 具体代码
用的是方法 1
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a[100005],t,p;
string s;
int main(){
	cin>>n>>s;
	for(int i=0;i<n;i++){
		if(s[i]=='(') t++;
		else p++;
	}
	cout<<n-abs(t-p);
	return 0;
}
```
