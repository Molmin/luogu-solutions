这道题的题意不是很清楚。下面是正确题意：

> 我们称一个由 $\texttt{X,Y,Z}$ 组成的字符串是特殊的，当且仅当它有连续三个字符各不相同。例如，$\texttt{XYZXYZ}$ 是特殊的，$\texttt{YZXYY}$ 是特殊的（前三个连续字符各不相同），但 $\texttt{XXYYZ}$ 不是特殊的。
>
> 给定 $n$，在由 $\texttt{X,Y,Z}$ 组成的长度为 $n$ 的所有字符串中（考虑顺序），有多少个字符串不是特殊的？（输入包含多组数据）

这道题是一个动态规划问题。首先我们可以知道，$n=1$ 时有 $3$ 个：$\texttt{X,Y,Z}$。

$n=2$ 也比较简单，因为它根本找不到一个三个字符组成的子串，所以有 $3 \times 3=9$ 个。

接下来就是递推了。我们首先列出 $n=2$ 时所有满足条件的字符串来推 $n=3$ 的情况。

$$
\texttt{XX}\\
\texttt{XY}\\
\texttt{XZ}\\
\texttt{YX}\\
\texttt{YY}\\
\texttt{YZ}\\
\texttt{ZX}\\
\texttt{ZY}\\
\texttt{ZZ}\\
$$

因为其中每种情况可以在第三个位置放的字符种类不同（$\texttt{XX}$ 可以放 $3$ 个，但是 $\texttt{XY}$ 只能放 $2$ 个），所以我们需要分类讨论。

我们可以发现每种情况都可以在第三个位置先放 $2$ 种字符（前两个字符相同的情况可以随便选两个，例如 $\texttt{XX}$ 就先放 $\texttt{X}$ 和 $\texttt{Y}$，前两个字符不同的就分别放前两个字符，例如 $\texttt{XY}$ 只能放 $\texttt{X}$ 和 $\texttt{Y}$）。

剩下的情况怎么选呢？容易发现，可以在第三个位置放第 $3$ 种字符的情况只有 $\texttt{XX,YY,ZZ}$。我们可以将它们看成 $n=1$ 的情况（因为字符相同嘛），而需要增加的方案数也恰恰是 $n=1$ 的个数，$3$ 个。

我们可以大胆地只推到 $n=3$ 的情况。因为 $n>3$ 的情况是只用保留最后三位来判断的。为什么这么说，因为我们的递推是线性的，而在最后三位之前就不满足条件的字符串早就被前面的递推筛除了，在后面的情况中根本不需要考虑。

现在我们可以归纳出，设长度为 $n$ 的非特殊字符串个数为 $f(n)$，则有如下递推式：

$$f(n)=2\times f(n-1)+f(n-2)$$

代码就很好写了，记得开 `long long`。

```cpp
#include <bits/stdc++.h>
using namespace std;

long long dp[40];

void init() {
    dp[1] = 3, dp[2] = 9;
    for (int i = 3; i <= 30; i++) dp[i] = 2 * dp[i - 1] + dp[i - 2];
}

int main() {
    init();
    int t;
    cin >> t;
    while (t--) {
        int n;
        cin >> n;
        cout << dp[n] << endl;
    }
    return 0;
}
```
