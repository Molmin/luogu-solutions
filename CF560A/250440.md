# 两个变量就够了

看前先点个赞么~

翻译：
Gerald居住的魔幻岛Geraldion有自己的货币体系。它使用几种价值的钞票。但问题是，这个体系并不完美，有时Geraldion人不能用任何纸币来表达一定数额的钱。当然，他们可以使用任何数量的钞票。这样的话称为不幸。Gerald想知道：最小的不幸金额是多少？

一开始看这个题，感觉很复杂，不像是入门题，其实很简单。众周所知，所有的数都可以用 $1$ 构成，那么我们是不是判断数列中有没有 $1$ 就可以了？如果没有 $1$ ，即使面值再丰富，也凑不出来 $1$ .

代码实现就很简单了，输入每一个数，如果是 $1$ ，输出 $-1$ 结束，循环结束时还没有结束的就凑不成 $1$ ，然后输出 $1$ .

代码双手奉上：
```
#include<bits/stdc++.h>
using namespace std;
int main(){
	int n,m;
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>m;
		if(m==1){
			cout<<"-1";
			return 0;
		}
	} 
	cout<<"1"<<endl;
	return 0;
}
```
