一看完题，脑子里就蹦出来个东西：

$ 1 $ 可以组成任何正整数

任何除了 $1$ 的正整数都不能变成 $1$

然后代码就出来了

```
#include <bits/stdc++.h>
using namespace std;
int n,l;
int main(){
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		cin>>l;
		if(l==1)
		{
			cout<<"-1";
			return 0;\\有1就能组成任何数，直接结束
		}
	}
	cout<<"1";//没结束程序就说明没有1，其他数变不成1。1就是最小的不幸的数
	return 0;
}
```
