### 题目大意
一场比赛有 $n$ 人参加，一共设置 $m$ 道题目。每名选手的编号依次为 $1,2,...,n$，每道题目的编号依次为 $1,2,...,m$。

在这次比赛中，问题的得分随解决问题的人数的变化而变化。具体来说，当一道题被 $x$ 个人解决时，这道题目的当前分数就是 $n-x$。所以，每当有人解决题目时，所有解决过这道题的选手的得分都会发生变化。

现在有 $q$ 次查询，第 $i$ 个查询被记为 $s_i$。请按照 $i=1,2,...,q$ 的顺序处理 $s_i$。每个查询的格式为下面两种形式中的一种：

- `1 i`：询问编号为 $i$ 的选手的分数。
- `2 i j`：编号为 $i$ 的参赛者解决了编号为 $j$ 的问题。（计分时解决问题 $j$ 的人中包括编号 $i$ 的选手）

------------

### 题目分析
看到此题，我们可以用一个 `vector` 数组 $a$ 来记录每一个人做对了哪些题。一个 `int` 类型的数组 $b$ 来记录每一题的分值。

接下来我们来分析两种查询。

1. 当输入为 `1 i` 时，询问编号为 $i$ 的选手的分数。我们可以循环遍历数组 $a_i$ ，再把他们相加，最后输出这个和。特别的，当数组为空时，需要特判并输出 $0$。


2. 当输入为 `2 i j` 时，首先将 $b_j$ 减一，因为题目的要求是若多一个人解决此题就将他的分数减一。然后把 $j$ 存进 $a_i$，表示此人解决掉了此题。


------------
### 题目代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,m,q,t,x,y,k,b[100000],sum;
vector <int> a[1000000];
int main(){
	cin>>n>>m>>q;
	for(int i=1;i<=m;i++)b[i]=n;//初始化所有题的分值都为n。
	for(int i=1;i<=q;i++){
		cin>>t;
		if(t==1){
			cin>>k;
			sum=0; 
			if(a[k].empty()){//特判。
				cout<<0<<endl;
				continue;
			}
			for(int j=0;j<=a[k].size()-1;j++){
				sum+=b[a[k][j]];//求和。
			}
			cout<<sum<<endl;//输出。
		}
		else if(t==2){
			cin>>x>>y;
			b[y]--;//将y题的分值减少。
			a[x].push_back(y);//把x解决的题y存进a[x]里。
		}
	}
    return 0;
}
```
