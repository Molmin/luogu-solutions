给出 $p$ 个数和 $n$，求能不能选择一些数使其和为 $n$。

这题是01背包弱化版。其实就是按照01背包的思想去完成题目。

状态：$f_i$ 表示能不能用这些数拼出 $i$

状态转移方程：$f_i=\max\limits_{j=1}^p f_{i-a_j}$

但是这里我们可以用另一种方法，换顺序。

对于每个数 $a_i$，我们都从大到小遍历一遍，如果 $f_{j-a_i}$ 是 $1$，说明 $j-a_i$ 这个位置能被拼到，那么 $j$ 位置自然也可以被拼到了。

为什么是从大到小呢？就好比我们踢一个水瓶。

如果我们从小到大正着踢，就是这样：

![](https://cdn.luogu.com.cn/upload/image_hosting/u621wdfb.png)

这样会算 $3$ 次。而如果我们从大到小，就只会算1次，就是符合题意的。

边界条件：$f_0=1$

最终答案：$f_n$

我们就可以写下核心代码：
```cpp
memset(f,0,sizeof(f));//初始化
f[0]=1;
for(int i=1;i<=p;i++){//按照上面说的
	cin>>a;
	for(int j=n;j>=a;j--) if(f[j-a]) f[j]=1;
}
if(f[n]) cout<<"YES"<<endl;
else cout<<"NO"<<endl;
```
