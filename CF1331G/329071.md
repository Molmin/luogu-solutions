[点我看题](https://www.luogu.com.cn/problem/CF1331G)

一开始拿到这道题时，其实完全没有思路：这串数字似乎是没有规律的，自然也无从下手。

后来想起了CF的其他愚人节题目，就随便写了个输出样例的程序交了，没想到竟然过了？

思路&方法：直接输出样例即可。

细节：输出的每一行都有换行（本题唯一有点麻烦的地方）。

代码：

```cpp
#include<bits/stdc++.h>
using namespace std; 
int main() 
{ 
	puts("f(10) = MAGNA NIMIS!");//这里之所以用puts，是因为puts自带换行 
	puts("f(-9) = -3642.00");
	puts("f(-8) = -2557.17");
	puts("f(-7) = -1712.35");
	puts("f(-6) = -1077.55");
	puts("f(-5) = -622.76");
	puts("f(-4) = -318.00");
	puts("f(-3) = -133.27");
	puts("f(-2) = -38.59");
	puts("f(1) = 6.00");
	puts("f(0) = 0.00");
	return 0;//结束程序 
} 
```
