~~愚人节题目。~~

### 题意

直接输出样例。

类似 [P1000](https://www.luogu.com.cn/problem/P1000/)

### 代码

1. `PHP`

`PHP`会对非代码部分直接输出。

```php
f(10) = MAGNA NIMIS!
f(-9) = -3642.00
f(-8) = -2557.17
f(-7) = -1712.35
f(-6) = -1077.55
f(-5) = -622.76
f(-4) = -318.00
f(-3) = -133.27
f(-2) = -38.59
f(1) = 6.00
f(0) = 0.00
```

2. `C++/C`

`printf`换行输出。

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	printf("f(10) = MAGNA NIMIS!\n"
	"f(-9) = -3642.00\n"
	"f(-8) = -2557.17\n"
	"f(-7) = -1712.35\n"
	"f(-6) = -1077.55\n"
	"f(-5) = -622.76\n"
	"f(-4) = -318.00\n"
	"f(-3) = -133.27\n"
	"f(-2) = -38.59\n"
	"f(1) = 6.00\n");
	return 0;
}
```
