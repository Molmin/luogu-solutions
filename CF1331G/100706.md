本题是一道愚人节题目，题目是用[Perligata语言](http://users.monash.edu/~damian/papers/HTML/Perligata.html)写的,用了[这个算法](https://rosettacode.org/wiki/Trabb_Pardo%E2%80%93Knuth_algorithm)

输出样例就能过了

### C
```
#include<stdio.h>

int main()
{
    puts("f(10) = MAGNA NIMIS!");
    puts("f(-9) = -3642.00");
    puts("f(-8) = -2557.17");
    puts("f(-7) = -1712.35");
    puts("f(-6) = -1077.55");
    puts("f(-5) = -622.76");
    puts("f(-4) = -318.00");
    puts("f(-3) = -133.27");
    puts("f(-2) = -38.59");
    puts("f(1) = 6.00");
    puts("f(0) = 0.00");
}
```
