题目大意大概是这样的：

构造一个每一位都不同，各位之和等于 $x$ 的正整数，要求这个正整数是所有合法的整数中最小的，无解输出 $-1$，有解输出这个正整数。

通过题面我们可以得到几个显然的结论：

1. $x>45$ 时必定无解；
2. 两个合法的正整数里，位数更少的必定更小；
3. 两个合法的正整数里，高位数字小的必定更小。

又因为构造的正整数必须每一位都不相同，所以可以得出做题方法：

1. 用最少数量的不同一位正整数表示这个正整数 $x$；
2. 把这几个数升序输出。

那么就变成了一道贪心了。

下见代码。

```cpp
#include<bits/stdc++.h>
using namespace std;
int T,x,sav;
int main(){
	cin>>T;
	while(T){
		T--;
		sav=10;
		cin>>x;	
		if(x>45){
			cout<<-1<<endl;
			continue;
			//x>45时必定无解 (因为 1+2+..+9=45) 
		}
		for(int i=9;x-i>0;i--){
			//从大数开始取一定会使最终输出的正整数最小 
			x-=i;
			sav=i;
		}
		if(x>0)cout<<x;
		for(int i=sav;i<=9;i++) cout<<i;
		cout<<endl;
	}
	return 0;
}

```
