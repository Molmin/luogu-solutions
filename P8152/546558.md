## 题意简述
- 开始时有 $1$ 个正方形，进行 $k$ 次操作，每次将右下角一个正方形分成 $n\times n$ 个正方形，求最后的正方形数量。
- 答案对 $998244353$ 取模。

## 解题思路
每次操作可以看做先减去右下角的一个正方形，然后再加上 $n\times n$ 个小正方形。设操作前有 $a$ 个正方形，$1$ 次操作后就有 $a+n\times n-1$ 个。

一开始时有 $1$ 个正方形，$k$ 次操作后就有 $1+k(n^2-1)$ 个。

## 代码实现
```python
a = input().split()
n = int(a[0])
k = int(a[1])
print((1 + k*(n*n-1)) % 998244353)
```
