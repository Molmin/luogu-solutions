## 1.前置芝士 : 桶。

桶是一种数据结构，它可以统计每个数据在数据集合中一共出现了多少次。

打个比方，桶就像一排抽屉，每接到一个新的数据，就把这个数据放进相应的抽屉里，要想统计每个数据在数据集合中一共出现的次数，只需要看这个数据对应的抽屉里元素的个数就可以了。

桶不光可以统计次数，还可以用来排序—— [桶排序](https://baike.baidu.com/item/%E6%A1%B6%E6%8E%92%E5%BA%8F/4973777?fr=aladdin) 。

## 2.思路:

显然，这道题要统计这个数有没有重复，如果重复就只保留最后一个。我们可以用一个桶来统计每个数出现的次数，然后从前往后遍历，如果这个数出现的次数只有一次，就输出这个数。然后将这个数出现的次数减一。

## 3.代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[55], t[1005], b[55]; //a表示输入的数组，t表示桶，b表示去重后的数组
int main()
{
	int n;
	cin >> n; //输入
	for(int i = 1; i <= n; i++){
		cin >> a[i];
		t[a[i]]++; //将这个数出现的次数加一
	}
	int len = 0; //len表示去重后的数组的长度
	for(int i = 1; i <= n; i++){
		if(t[a[i]] == 1){ //判断是不是这个值出现的最右边的位置
			b[++len] = a[i]; //将a[i]加入去重后的数组
		}
		t[a[i]]--; //将这个数出现的次数减一
	}
	cout << len << endl; //输出
	for(int i = 1; i <= len; i++){
		cout << b[i] << ' ';
	}
	cout << endl;
	return 0;
}
```
