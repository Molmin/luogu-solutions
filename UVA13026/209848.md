## 题意：
这题没有翻译，自行机翻就能知道大概意思了，在此我还是把主要意思讲一下。

输入 $T$ 组数据，每组数据的第一行输入一个 $n$ ，后面 $n$ 行，每行一个号码，同一组数据里的号码位数相同，每组数据最后再输入一个号码，让你输出前 $n$  个号码中与最后一个号码相差不超过一个字符的所有号码（本人表达能力不强，具体看下面）。

## 思路：
很水的题~按题目说的做即可，一位一位比较过去。

### 用到的知识点：

字符串、多重循环、数组

## 坑点：
用来对比的字符串是最后输入的，不是先输入的，这就有一点点烦，先存起来即可。

## 代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int T;
int n;
string a;
string b[10000];
bool t(string a1,string a2)//返回比较结果——两个字符串是否相同或只差一个字符
{
    int s=0;//不同字符的个数
    for(int i=0;i<a1.size();i++)
        if(a1[i]!=a2[i])//如果不同
            s++;//个数++，范围小，没必要优化
    return s<2;//不相同的字符个数是否小于二
}
int main()
{
    cin>>T;//输入数据组数
    for(int i=1;i<=T;i++)
    {
        printf("Case %d:\n",i);
        cin>>n;//每组数据输入1个n
        for(int i=1;i<=n;i++)cin>>b[i];//输入，请把这行的i全换成j（大佬能不能留言为什么变量名重复了也能对，是自动忽略之前的i了吗
        cin>>a;//最后输入用来比较的字符串
        for(int i=1;i<=n;i++)//同上，请把这行的i全换成j
        {
            if(t(a,b[i]))//挨个比较，请把这行的i全换成j
                cout<<b[i]<<"\n";//符合要求就输出，请把这行的i全换成j
        }
    }
	return 0;//很简洁的代码
}
```
