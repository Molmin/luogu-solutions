这是一道好题。

废话不多说，上**思路**：

- 题目要求我们可以再字符串 $s$ 中任意位置插入 $0$ 或 $1$ , 以使字符串 $s$ 成为美丽的字符串，即使字符串 $s$ 中 $0$ 的个数不超过 $1$ 的个数。
  
- 那么我们只用考虑两种情况，即 $00$ 和 $010$ 。
  
- 在 $00$ 的情况下，我们有两种插法：一是 $010$ , 二是 $0110$ , 这两种插法都满足题目要求。所以此时 `ans+=2` 。
  
- 在 $010$ 的情况下，我们只有一种插法： $0110$ 。这种插法也满足题目要求。所以此时 `ans++` 。最后将 $ans$ 输出即可。
  
- 注意一些**细节**：
  
  - $010$ 的情况要注意防止越界。
    
  - $ans$ 每次记得清空。
    

你听懂了吗？

话不多说，上代码：(已 `AC`)

```cpp
#include <cstdio>
#define maxn 105
int t,n,ans;
char s[maxn];
inline void in(int &a){ //快速读入
    a=0;char ch=getchar();
    while(ch<'0'||ch>'9')
        ch=getchar();
    while(ch>='0'&&ch<='9'){
        a=(a<<1)+(a<<3)+(ch^48);
        ch=getchar();
    }
}
inline void op(int a){ //快速输出
    if(a>9) op(a/10);
    putchar(a%10+'0');
}
int main(){ 
    in(t); //输入数据组数
    while(t--){
        in(n);scanf("%s",s);ans=0; //读入并将答案初值赋值为0
        for(int i=1;i<n;++i){ //从头至尾扫一遍
            if(s[i]=='0'&&s[i-1]=='0') ans+=2; 
            else if(s[i-1]=='0'&&s[i]=='1') 
            if(i<n-1) //防越界
                if(s[i-1]=='0'&&s[i]=='1'&&s[i+1]=='0') ans++; 
        }
        op(ans),puts(""); //输出答案
    }
    return 0;
}
```
