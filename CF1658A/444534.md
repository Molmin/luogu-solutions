## 题意
给你一个 $01$ 字符串 $s$，在至少 $2$ 个字符的连续片段中字符 $0$ 的个数小于字符 $1$ 的个数，则被认为是美丽的。

为了确保队伍美观，你可以额外加入 $0$ 和 $1$ 在任何位置加入队伍。

求加入的最小数量。
## 思路
当相邻两个字符都为 $0$ 时，必须要增加至少两个 $1$ （$0101$）才行，因为求加入的最小数量，所以这种情况要加两个。

当相邻三个字符分别为 $010$ 时，需要加一个字符 $1$ 才行（$0101$）。

暴力枚举字符串 $s$ 中 $00$ 和 $010$ 的个数即可。
## CODE
```cpp
#include<bits/stdc++.h>
using namespace std;
string s;
int t, m, ans;
int main(){
	cin >> t;
	while(t--){	
	    cin >> m;
	    ans = 0;
	    cin >> s;
	    for(int i = 0; i < s.size(); i++){
		    if(s[i] == '0' && s[i + 1] == '0') {
    		    ans += 2;
	    	}else if(s[i] == '0' && s[i + 1] == '1' && s[i + 2] == '0') {
		        ans++;
		    }
        }
	    cout << ans << endl;
	}
	return 0;
}
```
