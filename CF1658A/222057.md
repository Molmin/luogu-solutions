## 题目分析

很明显，我们是要往两个 $0$ 里面加 $1$。所以，只要我们遇到 $00$，就需要往里面加 $1$。我们来想一下要加几个 $1$。

如果只在中间加 $1$ 个 $1$，那么还是能截出一段 $010$，里面 $0$ 的数量比 $1$ 的数量多。加两个的话，就会变成 $0110$，就会一样多。所以在遇到 $00$ 时，把答案 $+2$ 就对了。

和上面一样，如果我们遇到 $010$，那么就把答案 $+1$ 就对了。

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int T,n,ans;
string s;
int main(){
	cin>>T;
	while(T--){
		cin>>n>>s;
		ans=0;
		for(int i=1;i<s.size();i++)
			if(s[i]=='0')
				if(s[i-1]=='0')//找00
					ans+=2;
				else if(i>1&&s[i-2]=='0')//找010，同时防止越界
					ans++;
		cout<<ans<<endl;
	}
	return 0;
}

```
