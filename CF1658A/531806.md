# 很水的字符串题
[注：blog 食用更佳](https://www.luogu.com.cn/blog/531806/CF1658A)

------------
### 题目大意
我们称一个仅包含字符 `0` 和 `1` 的字符串是美丽的，当且仅当对于其所有长度$\geqslant 2$ 的子串，都满足当中 `0` 的个数不超过 `1` 的个数。

现在，给定一个长度为 nn 且仅包含字符 `0` 和 `1` 的字符串 $s$，你可以在该字符串的任意位置插入字符 `0` 或 `1`。求使得字符串 $s$ 是美丽的，至少要插入多少个字符。
### 思路
根据题意可知：两个 `0` 之间至少要有两个 `1`，所以就只有判断 `00` 和 `010` 的情况，所以一共就只有两种操作：
- 如果是 `00` 则中间加两个 `1`，如子串 `00` 变为 `0110`；

- 如果是 `010` 则中间加一个 `1`，子串 `010` 同样也变为 `0110`。

### AC Code:
```cpp
#include<bits/stdc++.h>//不用万能头也可以，但个人觉得万能头方便。
using namespace std;
int t,len,ans=0;
string s;
int main(){
	scanf("%d",&t);//输入"t"
	while(t--){
		scanf("%d%s",&len,s);//输入长度(len)和字符串(s)
        ans=0;//初始化
        for(int i=1;i<=len-1;++i){//循环
			if(s[i-1]=='0'&&s[i]=='0') ans+=2;//如果是情况一就加2
			if(s[i]=='1'&&s[i+1]=='0'&&s[i-1]=='0') ans++;//如果是情况二则加1
		}
		printf("%d\n",ans);//输出答案
	}
	return 0;//好习惯
}
```
