## 题目分析
很显然，**你为了让最后的代价尽可能小，肯定是跳最多次免费的**，也就是从一格跳到这格子右边的另一格，前提是**这两个格子都是土地**。但是由于你不能呆在有水的格子上，所以当这些“免费的跳”跳到一个有水的格子时，就必须中断，必须要使用金币跳到下一个是土地的格子。

问题是用金币要跳多少个格子呢？

**我们最终的答案应该是这样的：**

例如，下面这组数据：

$5$

$1 \ 0 \ 1 \ 0 \ 1$

我们先从最左边开始跳，发现只能跳一格就遇到水，于是只能使用金币。如果跳到第三个格子，那么肯定跳不到终点，因为它与终点间还隔着一个水的格子，所以我们必须直接跳到终点。

于是，我们的答案就出来了，最终的答案 $=y-x$ ,其中 $x$ 与 $y$ 分别代表从起点往右连续的 $1$ 到 $x$ 为止和从终点往左连续的 $1$ 到 $y$ 为止
## 代码
前面应该已经讲的很清楚了，所以直接上代码。
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[11000];
int main(){
	ios::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);
	int t;cin>>t;
	while (t--){
		int n,p1=0,p2=0;
		cin>>n;
		for (int i=1;i<=n;i++) cin>>a[i];
        for (int i=1;i<=n;i++)
            if (a[i]!=1){p1=i-1;break;}
        for (int i=n;i>=1;i--)
            if (a[i]!=1){p2=i+1;break;}
        cout<<p2-p1<<'\n';
	}
	return 0;
}
```
