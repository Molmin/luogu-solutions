## 下面ac100分代码；~~思路并不是很特别~~

------------

```
#!/usr/bin/perl -w
($sum,$k)=split /\s+/,<>
#k是换一支烟的烟头数，nonu记录烟头的个数
#sum是最后的答案
for($nonu=$sum;++$sum,++$nonu){$nonu-=$k}
print $sum;		
#输出烟的个数 
```


------------



## 大概解释一下

实际上就是直接模拟，烟头数目足够换一只烟时就换烟，也就是烟头数加一，到换不了烟的时候就输出烟的个数。
```
k是换一支烟的烟头数，nonu记录烟头的个数
sum是最后的答案

$nonu=$sum；  烟头初始数目;
$nonu-$k>=0； 烟头数目足够换一只烟时;
$nonu-=$k；   换烟
++$sum；      多一个烟;
++$nonu；     吸烟之后多一个烟头;

print $sum;   
输出烟的个数 
```
