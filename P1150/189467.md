有N支烟，那么就有N个烟蒂，又因为K个烟蒂可以换一支烟，所以最多可换N/K支烟，这里的余数可以看成是剩余烟蒂的数量，当然这里也可以把最多可换烟的数量也给看成是烟蒂数量，因此就可以求出下一轮的烟蒂数量，即(N/K)+(N%K)，然后判断N<K，成立则得到结果，反之则继续重复上述步骤。

PS:不要忘记把中间结果给累加起来。

再啰嗦一句，这题可以直接算，我是真没想到QWQ



------------
```c
#include <stdio.h>

int main(void)
{
	int n,k,sum;
	
	scanf("%d %d", &n,&k);
	sum = n;
	while(n >= k)
	{
		sum += n/k;
		n = (n/k)+(n%k);
	}
	printf("%d", sum);
	
	return 0;
}
```
