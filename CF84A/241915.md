### 题意简述（较原题面有化简）：

给定一个数 n。

如果 n 是 4 的倍数，那么输出 n * 3 / 2;

否则，输出 n * 2 - (n / 4 * 2 + 1)

### 方法：

先解释一下题意，由于题目要我们求的是被打死的人数，所以按照原题面的公式计算后还要加上 n * 2 ，故有此化简。另外，因为 c++ 的除法自动向下取整，所以原题面中的向下取整符可以无视。

其实要做出这道题最简单的方法，还是要先打表观察一下的↓

```cpp
#include <bits/stdc++.h>
using namespace std;
long long a,b,c,k;
int main()
{
	for (int i = 2; i <= 400; i += 2)
		{
			if (i % 4 == 0) cout << i * 3 / 2  << " ";//依照题面公式。
				else cout << i * 2 - (i / 4 * 2 + 1) << " ";
		}
	return 0;	
}
```
