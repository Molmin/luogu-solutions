这道题让我们输出一个日历。看似很简单，实际却**大有乾坤**。

1. 每月不一定从星期一开始。
2. 每行前后有 `|`。
3. 缺失的日子用 `...` 代替

通过本题需要特别注意第 $3$ 点。接下来我们来逐步解决这些难题。

通过观察发现，开头缺失的日子是 $x-1$ 天，输出即可，同时输出日历第一行的装饰。
```cpp
cout<<"+---------------------+"<<endl<<"|";
for(int i=1;i<x;i++) cout<<"...";//输出缺失的x-1天
```
第二步是输出每一天。按照要求模拟即可。**别忘了最后要加 `|` 和换行。**
```cpp
for(int i=1;i<=n;i++)
{
  if(i<10) cout<<".."<<i;//按要求输出
  else cout<<"."<<i;
  if(s%7==0&&i!=n) cout<<"|"<<endl<<"|";//是星期天就换行
  //加i!=n是为了防止在最后一天换行，多输出一个"|"。
  s++;//s表示今天是星期几
} 
```


第三步，输出最后缺失的日子。**非常重要**！
首先我们得知道这个月最后一天是星期几，才能知道离周天差多久。$7-(n-8+x)\bmod7$ 表示离星期天差多久。但是外面还要套一层 $\bmod$ $7$。因为如果这个月最后一天是星期天，那么 这个式子的值为 $7$，就会出现问题。最后输出装饰，本题就完成啦！撒花~

```cpp
for(int i=1;i<=(7-(n-8+x)%7)%7;i++) cout<<"...";//可以思考7-(n-8+x)%7是如何得出的。
cout<<"|"<<endl<<"+---------------------+";
```
