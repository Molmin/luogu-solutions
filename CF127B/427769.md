# 分析
题目会给出几个线段的长度，要你求能拼出几个矩形。而拼出矩形需要的条件是 $2$ 条长和 $2$ 条宽。矩形的长都相等，宽也都相等。那么我们就可以用数组计数来判断每根线段有几根与它相等。在用数组一个个查找。如果大于或等于 $2$  就加上这个数除以二。最后在用统计到的对数除以 $2$ 。就可以对了。顺便说一下，这个方法目前是最优解。

# 代码
```
#include<bits/stdc++.h>
using namespace std;
int a[10000],n,w,s;
int main(){
	cin>>n;
    //数组计数
	for(int i=0;i<n;i++)
	{
	cin>>w;
    a[w]++;
    }
    for(int i=1;i<=100;i++)//最大数据是100，取上限为100
	if(a[i]>=2)s+=a[i]/2;//因为2个数一对，所以要除以2
	cout<<s/2<<endl;//2对线段一个矩形，也要除以2
	return 0;
}
```
