## 题意
给你一大堆棒子，问你最多能够拼成几个方形？
## 思路
应为要想拼成方形，有以下两种情况：  
1、四边相等，也就是正方形。  
2、对边相等，也就是长方形。  
而正方形也可以看做两组对边的长度都相等的方形。  
故，我们对棒子进行桶排，在用两个循环来枚举两组对边的长度，正方形就是两个长度一样的情况，那么这个长度的棒数就要四根，才可以组成一个，所以最多可以拼总数除以四个；长方形就是两个长度不相等的情况，需要两种长度各两根，才可以拼成一个，所以最多可以拼两种长度的根数分别除以二后的较小数。当然了，用完后，小棒的数量也要相应减少。
## Code
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,i,j,f[101],x,ans,h;
int main(){
	scanf("%d",&n);
	for (i=1;i<=n;i++) scanf("%d",&x),f[x]++;//桶排
	for (i=1;i<=100;i++)
		if (f[i]>=2)//一定要大于2
		for (j=1;j<=100;j++)
			if (f[j]>=2)//一定要大于2
				if (i!=j)//如果不相同
					h=min(f[i]/2,f[j]/2),ans+=h,f[i]-=h*2,f[j]-=h*2;//最多能取几个
				else h=f[i]/4,ans+=h,f[i]-=h*4;//如果相同，又最多能取几个。
	printf("%d\n",ans);	//输出
	return 0;
}

```
