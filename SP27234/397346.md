# 思路
这题暴力显然不行，即使洛谷题面没有给出数据范围，想想就不可能是用暴力的。

那该怎么办呢？先来看个样例：

比如 $8$，要求它的答案，我们就要知道 $5$ 和 $6$ 的答案（因为最后一步可能是 $5+3$ 或 $6+2$）。再往前推，会发现只需要知道 $3$ 和 $2$ 的答案就能推出 $5$ 和 $6$，接着就能推出 $8$ 的答案。以此类推，对于任何一个数据，我们只要知道 $3$ 和 $2$ 的答案就能推出它的答案。

手动算一下，就会知道 $3$ 和 $2$ 的答案都是 $1$。然后，就能得出公式：

$ f_{i} = f_{i-3} + f_{i-2} $

然后按照这个公式来做就好了。

# 代码
如上。

```cpp
#include<bits/stdc++.h>
using namespace std;
long long f[1000010];
int main(){
	long long n;
	cin>>n;
	f[1]=0;//其实这句不加也能对，但是个人觉得看起来整齐一点就加上了。
	f[2]=1;
	f[3]=1;
	for(int i=4;i<=1000000;i++){
		f[i]=(f[i-3]+f[i-2])%1000000007;//不要忘记取余！
	}
	for(int i=1;i<=n;i++){
		long long x;
		cin>>x;
		cout<<f[x]<<endl;
	}
	return 0;  
}
```
