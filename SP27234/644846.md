## 题意
给出 $n$ 个数，求每个数用 $2$ 和 $3$ 求和的方案数。

## 思路
简单的递推题。

我们可以构造数列 $a$，$a_n$ 为 $n$ 用 $2$ 和 $3$ 求和的方案数，找递推公式。

对于 $n$，如果我们算出了 $n-2$，那么加上 $2$ 就可以算出 $n$ 了，算出 $n-3$ 时同理。也就是说，只有算出 $n-2$ 或 $n-3$ 的时候，才可以只用一步就算出 $n$，因此，递推公式为:
$$a_n=a_{n-2}+a_{n-3}$$

## 代码
```
#include<bits/stdc++.h>
using namespace std;
int n,a[1000005],x;
int main(){
	a[2]=1;
    a[3]=1;
	cin>>n;
	for(int i=4;i<=1000000;i++){
    	a[i]=(a[i-2]+a[i-3])%1000000007;                             
    }
	while(n--){
		cin>>x;
		cout<<a[x]<<endl;
	}
	return 0; 
}
```
