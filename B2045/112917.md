## Content
贝贝邀请晶晶下个星期 $x$（如果 $x=7$ 表示星期日）一起去看展览，但是晶晶下个星期 $1,3,5$ 都有课。请你判断晶晶能否同意贝贝的请求。

**数据范围：$x\in[1,7]$。**
## Solution
`if-else` 的题解想必有很多，此处就不再多说，这里介绍一下**三目运算符**。

三目运算符有三个部分组成，分别用 `?` 和 `:` 间隔，格式是：`条件1?语句1:语句2`，相当于：

```cpp
if(条件1)	语句1;
else		语句2;
```

同 `if-else` 一样，它可以嵌套，但是如果要有多个语句的话建议**不要用三目运算符**，不过对于这道题目来说是足够了。

注意处理多个条件的或关系可以用 `||` 表示，与关系可以用 `&&` 表示。
## Code
```cpp
#include <iostream>
using namespace std;

int main() {
	int n;
	cin>>n;
	cout<<((n!=1&&n!=3&&n!=5)?"YES":"NO");
	return 0;
}
```
