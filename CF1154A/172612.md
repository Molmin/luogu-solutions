## 很简单的一题思维题，就告诉你几个约束条件，求出任意一组解就可以了
### 首先我们来看一下几个约束条件：
### $ x1=a+b $ 
### $ x2=a+c $
### $ x3=b+c $
### $ x4=a+b+c $
### 观察一下这些柿子，发现$  x4=a+b+c  $一定是最大的，所以要求$c$只要$x4-x1$就可以了，很简单！最后在将解出来的$c$代入上柿，就可以得出$a,b$的解了。代码也很短只有13行。
```cpp
#include<bits/stdc++.h>
using namespace std;
int X[5],A,B,C;
int main()
{
	cin>>X[1]>>X[2]>>X[3]>>X[4];
	sort(X+1,X+4+1);//排个序
	C=X[4]-X[1];
	B=X[3]-C;
	A=X[2]-C;
	cout<<A<<" "<<B<<" "<<C<<endl;//按照上面说着做就可以了
	return 0;
}
```
