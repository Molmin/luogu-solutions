# 题意  
给你一个字符串，每次你可以选择字符串的一个子序列进行删除，删除的时候必须满足子序列的是个合法的括号序列  
字符串仅包括`()[]`这$4$个字符  

# 题解  
这很显然了，经典的括号匹配问题，思博题。  
把`()`和`[]`分开讨论即可。  
记录之前出现过的`(`次数，然后在出现`)`时如果前面有`(`就`ans++,cnt--`。  
`[]`同理  

```cpp
signed main()
{
	cin>>T;
	while (T--)
	{
		cin>>str;
		l=L=ans=0;
		for (int i=0;i<str.length();i++)
		{
			if (str[i]=='(') l++;
			if (str[i]=='[') L++;
			if (str[i]==']' && L) {L--;ans++;}
			if (str[i]==')' && l) {l--;ans++;}
		}
		cout<<ans<<endl;
	}
}
```
