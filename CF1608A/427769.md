# 分析
题目会给出一个 $N$ ，也就是知道了要构造的数列的数字个数。而构造的数列每一个数都要大于或等于 $1$ 且 小于或等于 $100000000$ 。且数列一个比一个小，同时每个数都不能被前一个数整除。也就是两数之间互质，那就可以是每一个都是前一个数加一。此题数据较小，可以这样做。我的思路是先用一个变量来装数列当前的数，用 ```while``` 语句来判断数字是否超过 $N+1$ ,超过自动退出，否则变量的数加一，一边变一边输出就可以了。

# 代码
```
#include<bits/stdc++.h>
using namespace std;
int t;
int main(){
	cin>>t;
	for(int i=0;i<t;i++)
	{
		int n,w=2;//w用来装当前数字，每回都要初始化
		cin>>n;
		while(w<=n+1)//由于w初始化为2，所以n要加一
		{
			cout<<w<<" ";
			w++;//每回都加一
		}
	}
	return 0;
}
```
