### 题目翻译

- 多组测试数据。
- 每组数据给定 $n$，构造一个长度为 $n$ 的序列 $a_1,a_2,\dots,a_n$，使得对于任意 $2\leqslant i \leqslant n$，$a_{i-1}\nmid a_i$。

------------
需要满足的条件是很弱的。

几乎不需要思考就能够想到，除了 $1,2$ 这一对之外的所有相邻的正整数都互质。

于是从 $2$ 开始之后往后输出 $n$ 个正整数即可。

```cpp
t=read();
while(t--)
{
	n=read();
	for(int i=1;i<=n;i++)printf("%d ",i+1);
	printf("\n");
}
```
