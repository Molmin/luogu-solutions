[题目传送门](https://www.luogu.com.cn/problem/UVA446)

**注意事项**

题目给出的 $16$ 进制数转换成 $2$ 进制数时需要补前导 $0$，直到有 $13$ 位为止。

**思路**

考虑到不同进制之间的加减法不是很好算，不妨先转换成 $10$ 进制再运算。至于进制转换什么的都是基本功了，不会的同学建议做一下这题 [P1143](https://www.luogu.com.cn/problem/P1143)。

这里有两种 $16$ 进制转 $2$ 进制的思路。

**法一**

先转换成 $10$ 进制，在转成 $2$ 进制。

**CODE**
```cpp
int t(string s){
	int num=0;
	for(int i=0;i<=s.size();i++){
		if(s[i]>='0'&&s[i]<='9')
			num=(num<<4)+s[i]-'0';
		if(s[i]>='A'&&s[i]<='Z')
			num=(num<<4)+s[i]-'A'+10;
	}
	return num;
}
string g(string s){
    int num=t(s);//先转成 10 进制
    string w=" ";//细节空格，纯属好玩
    while(num){
        w=char((num&1)+'0')+w;
        num>>=1;
    }
    while(w.length()<14)
        w='0'+w;
     //补足前导 0
    return w;
}
```
**法二**

打个小表。
```cpp
map<int,int> m;
m[0]=0,m[1]=1,m[2]=10...
```
