### 题目大意

给出 $n$ 个 $16$ 进制数加减算式，把运算数按 $2$ 进制输出，把结果按 $10$ 进制输出。

### 题目解法

Python 语言具有较强的进制转换功能，如下（设被转数为 $a$ ，类型为 `str` 且无前缀）。

| 进制（上至左） | $2$ | $10$ | $16$ |
| :----------: | :----------: | :----------: | :----------: |
| $2$ | / | `bin(a)[2:]` | `bin(int('0x'+a.lower(),16))[2:]` |
| $10$ | `int('0b'+a,2)` | / | `int('0x'+a.lower(),16)` |
| $16$ | `hex(int('0b'+a,2))` | `hex(a)` | / |

那么就可以直接转换并输出了。

不过，注意样例，$2$ 进制数必须是 $13$ 位！我们可以格式化，但更简单的办法是：输出 $13-(2$ 进制串长度$)$ 个 $\tt 0$，再接上原串。

### AC 代码

注意多测。

```python
n=int(input())
for i in range(n):
    s=input().split()
    a,b=int('0x'+s[0].lower(),16),int('0x'+s[2].lower(),16)#16->10
    a1,b1=bin(a)[2:],bin(b)[2:]#10->2
    print('0'*(13-len(a1))+a1,#13位
    s[1],'0'*(13-len(b1))+b1,#13位
    '=',
    eval(str(a)+s[1]+str(b)))#结果
```
