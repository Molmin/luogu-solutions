# [题目传送门](https://www.luogu.com.cn/problem/CF1181A)

## 题目大意：

输出两个人可以得到的最多的糖果。

如果需要一方给对方一些钱才可以使得到的糖果最多的话，那么输出给钱的最小值。否则，输出 $0$ 。

## 题目分析：

非常简单地分两种情况：

（1） 如果两人现在每人可以得到的糖果的个数的和等于这两个人钱的总数可以得到的糖果的和的话，那么证明当前就可以得到最多的糖果，那么输出这个糖果数，然后输出 $0$ 。 即：

```cpp
if(a / c + b / c == (a + b) / c){
	cout << (a + b) / c << " 0";
}
```

（2） 否则的话，那么证明必须得一方给另一方钱。

如果 $a$ 给 $b$ 钱的话，那么需要给 $c$ - $b$ % $c$ `RMB`，反之，则需要 $c$ - $a$ % $c$ `RMB`。说到这里，结果就出来了，最多的糖果数仍然可以用两个人钱的总和可以得到的糖果数，需要给的钱的最小值就是上述两种情况的最小值。

## 完整代码：

```cpp
#include <iostream>
#include <cstdio>

using namespace std;

long long a,b,c;

int main(){
	cin >> a >> b >> c;
	if(a / c + b / c == (a + b) / c){
		cout << (a + b) / c << " 0";
		return 0;
	}
	cout << (a + b) / c << " ";
	cout << min(c - a % c,c - b % c);
	return 0;
}
```
