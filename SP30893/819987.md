这道题是一道英文题，所以需要中文翻译。

------------

翻译：

萨比尔是一个小男孩。他喜欢玩耍。

一天，他的朋友塔斯金建议玩一个有趣的新游戏。


------------

规则如下：

游戏中有 $n$ 个关卡。玩家必须以正生命值通过游戏中所有关卡。

在一些关卡中，玩家可以通过击败游戏中的反派而增加自己的生命值，或者在无法击败反派时失去一些生命值。

问题:

萨比尔知道他在每一关中会失去或增加的点数。

确定萨比尔最初（在游戏开始时）应该拥有的最低生命点，以便以正生命点通过所有关卡。

我刚拿到这道题是第一反应是模拟,大概只有二层循环所以不会超时。 


------------

这是输入格式：

输入最多包含 $100$ 个测试用例。
第一行包含单个整数 $T(1 \le T \le 100)$ 个测试用例。
每个测试用例由两行组成。第一行包含一个整数 $n(1 \le n \le 1000)$。
第二行由 $n$ 个空格分隔的整数 $a_{n}$ 组成。

思路：

可以一个循环控制 $T$ 关，然后根据 $a_{i}$ 加减生命值，再在输出时判断是否为整数。


```cpp
#include <bits/stdc++.h>
#define long long int
using namespace std;
int n,Min=1e9,ans,T,d;//Min求最小所以定义最大
int main()
{
	cin>>T;
	for(int i=1;i<=n;i++)
	{
		cin>>T;
		for(int j=1;j<=T;j++)
		{
			cin>>d;
			ans=ans+d;//累加生命值
			Min=min(ans,Min);//求最小
		}
		if(Min<=0)//判断
		{
			cout<<abs(Min+1)<<'\n';
		 } 
		 else
		 {
		 	cout<<0<<'\n';
		 	break;
		 }
	}
	return 0;
}
```

注意 $Min$ 一定要定义的较大，以及输出时必须要求绝对值避免负数。
