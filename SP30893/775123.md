本题的大致中文意思：

萨比尔是个小男孩，他喜欢玩。一天，他的朋友塔斯金发现了一个新的有趣的游戏。游戏中有 $n$ 个关卡。一个人应该带着自己的生命值过所有的关卡。在某种程度上，一个人可以通过击败游戏中的恶棍来增加自己的生命值，或者在无法击败恶棍时失去一些生命值。萨比尔知道他在每一关将要失去或增加的点数。确定萨比尔最初(在每一关游戏开始时)应该拥有的最低生命点，以**正生命点**通过所有关卡。

一看中文意思，那本题的难度就大大降低了，主要思想为模拟，再进行完善即可。

我们将每一关分开来讨论，先将一个计数器赋值为 $0$，随后每输入一次将要失去或增加的点数，便用计数器去加，求出该关结束后萨比尔的生命值，再与之前的生命值求一个最小值。最后输出最小值的绝对值 $+1$ 便可。

具体解释见代码。

```cpp
#include<bits/stdc++.h>
using namespace std;
long long int n,k,x,ans,minn;
int main()
{
   cin>>n;
   for(int i=1;i<=n;i++)
   {
      cin>>k;
      minn=10000000;//赋值为最大值。
      ans=0;
      for(int j=1;j<=k;j++)
      {
         cin>>x;//边输边做，节约时间。
         ans+=x;//计算生命值。
         minn=min(minn,ans);//求生命值最低的时候。
      }
      if(minn>0)//特殊情况，注意不要用大于或等于，若使用则会显示错误。
      {
         cout<<0<<"\n";
         continue;
      }
      else cout<<abs(minn)+1<<"\n";//注意。
   }
   return 0;
}
```
