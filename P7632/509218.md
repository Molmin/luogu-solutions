# 思路(来自官方题解)
------------
暴力枚举从 $0$ 秒到 $2880$ 的每一秒，我们存储在该秒中得分的球队(如果有的话)。为了填充这个数组，我们必须将给定的事件从 $MM:SS$ 格式转换为从游戏开始经过的秒数 $MM*60+SS$。

现在我们遍历这个数组，跟踪每个团队的当前得分。每过一秒，比分没有平局，就增加获胜队伍的计数器。

## 补充一个关于 printf 和 scanf 的小知识

------------
输入时间可以用
```cpp
scanf("%d:%d:%d", &h, &m, &s);
```
可以自动忽略“：”。
输出时间用
```cpp
printf("%02d:%02d:%02d", hh, mm, ss);
```
可以自动用前导 0 填充至 2 位。

# AC 代码 

------------

```c++
#include <bits/stdc++.h>
using namespace std;
int n, a, b, t, m, s, d[2885];
int main() {
    scanf("%d", &n);
    for(int i=1; i<=n; i++) {
        scanf("%d%d:%d", &t, &m, &s);
        if(t == 1) d[m*60+s]=1;
        else d[m*60+s]=-1;
    }
    for(int i=0; i<2880; i++) {
        if(i != 0) d[i]+=d[i-1];
        if(d[i] > 0) a++;
        if(d[i] < 0) b++;
    }
    printf("%02d:%02d\n%02d:%02d", a/60, a%60, b/60, b%60);
    return 0;
}
```
