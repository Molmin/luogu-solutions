### 简单介绍一下博弈论
- 必败点( $P$ 点) :前一个选手将取胜的位置称为必败点。
- 必胜点( $N$ 点) :下一个选手将取胜的位置称为必胜点。

对于此题，从小数据开始分析，可通过找规律得出必败点与必胜点，再判断 $n$ 的大小即可。

 假设当前数大小为 $m$ ，由先手进行操作。显然 $n > m > = n / 9$ 时先手获胜，则以后手最优方案分析，后手需将 $m$ 控制在 $n / 9 > m > = n / 18$ 时才可获胜。问题转化成达到 $m$ 的方案。将 $n = m$ 代入到上述不等式中可得出再前一阶段需要控制的 $m$ 范围，不断代入，将原数除以 $18$ 即可实现缩小 $n$ 的取值，到此我们便分析出本题的循环节为 $18$ 。

当 $n < = 18$ 时，显然在 $1 $~$ 9$ 范围内先手获胜，反之后手获胜。**注意 $n$ 的数据类型为小数。**
## 代码如下
```cpp
#include<cstdio>
#include<iostream>
using namespace std;
double n;
int main(){
	while(cin>>n){
		while(n>18)n/=18;
		if(n>9)printf("Ollie wins.\n");
		else printf("Stan wins.\n");
	}
	return 0;
}
```
