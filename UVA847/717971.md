## [题目传送门](/problem/UVA847)
最开始看上去是不是毫无头绪，~~让我们来复制题解吧！~~

开个玩笑，现在进入正题。

做题要结合样例看，来找找规律：
+ 当 $2\le n\le9$ 时，乘 $1\sim 9$ 之间一个数，一定能胜。
+ 当 $10\le n\le18$ 时，先手无论 $1\sim9$ 之间任意一个数，后手在乘一次，就能超过 $n$。
+ 当 $19\le n\le27$ 时，先手乘 $2$，后手无论乘几，都不能超过 $n$，但此时先手随意乘一个数，就能获胜。
+ ......

从而得出结论：将 $n$ 不断除以 $18$，直到小于等于 $18$ 为止，最后判断，如果 $n\le9$，那么先手胜；否则后手胜。

**注意：**这题的数据范围是 $1<n<4294967295$，所以 `int` 装不下，还得保证 $\frac{n}{18}$ 的精度，还不如直接用 `double` 装（`double` 是双精度64位范围）。

有了结论，代码就简单了：
```cpp
#include<bits/stdc++.h>
using namespace std;
double n;
int main(){
	while(cin>>n){
		while(n>18)n/=18;
		if(n<=9)puts("Stan wins.");
		else puts("Ollie wins.");
	}
	return 0;
}
```
