这道题一看就是博弈论，先看看度娘是怎么说的：

> **博弈论是二人在平等的对局中各自利用对方的策略变换自己的对抗策略，达到取胜的目的。**

双方都想取胜，要让对方必败就得让对方，到一个必败的点（也就是自己的必胜点），对方就会必败，而必败的点可以从最后反推回来。  

设当前的数到了 $m$ ，那么先手的必胜点在 $n\gt m \ge n \div 9$ 这个区间里，反观后手，后手的必胜点在 $n\div 9 \gt m \ge n\div 18$ 这个区间里。很容易发现，若我们把 $n$ 一直除以 $18$ 直到 $n\le 18$ 时为止，可以缩小这个不等式的范围，最后若 $n\gt9	$ 则后手赢，否则先手赢。  
**注意 $n$ 一定要开成小数类型的变量，否则会向下取整，不准确。**

# AC CODE:
```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    for(double n; cin >> n;){
    	while(n > 18) n /= 18;
        if(n > 9) puts("Ollie wins.");
        else puts("Stan wins.");
    }
    return 0;
}
```
