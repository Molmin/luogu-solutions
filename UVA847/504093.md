一道经典的博弈论问题。

在说这道题目之前，我们可以先来想到和这个问题类似的取石子游戏。总共有 $n$ 个石子，每个人一次可以取的石子数量在 $[2,9]$ 的范围内。那么我们就知道，如果先手存在必胜策略，那么就要先取走 $n \bmod 11 = 0$ 个石子，接着后手每次取走 $k \in [2,9]$ 个石子。先手就取 $11 - k$ 个石子；如果 $n$ 是 $11$ 的倍数，那么后手必胜。
***

再来看这个问题，一个数字从 $1$ 开始，每次乘上一个 $k \in [2,9]$，谁让这个数字大于等于 $n$ 就谁获胜。那么不就是上面的取石子问题，只是把上面的加和，改为乘积。这样，我们就可以得到以下的必胜策略：

- Stan 和 Ollie 两个人在一个回合（也就是两个人各乘一次）内，要把这个数扩大 $2 \times 9 =18$ 倍。

- 把数字 $1$ 一直乘 $18$，直到再乘一次就会超过 $n$ 时，看一下先手能不能把这个数字乘一次 $[2,9]$ 范围中的一个数，使其超过 $n$。如果可以，那么先手获胜。如果不行，后手获胜。

具体代码如下，仅供参考，请勿抄袭。

```cpp
#include<bits/stdc++.h>
using namespace std;
double n;
main()
{
	while(scanf("%lf",&n)!=EOF)
	{
		double ji=1;
		while(ji*18<n)ji*=18;
		if(n/ji>9)puts("Ollie wins.");
		else puts("Stan wins.");
	}
	return 0;
}
```
