### 思路：从2开始枚举，直到根号n。

Q：为什么不用枚举到n呢？

A：两个数相乘为n，一个小于根号n，另一个必然大于根号n，所以，根号n到n的部分在前面就相当于枚举过了。

**精髓**：
```cpp
if(n<2){  //特判
    cout<<"NO"<<endl;
    return 0;  //结束程序
}
for(i=2;i<=sqrt(n);i++){
    if(!(n%i)){  //如果在2到sqrt(n)中有个数能整除n
        cout<<"NO";
        return 0;  //结束程序
    }
}
cout<<"YES";
```
