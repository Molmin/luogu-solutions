首先，看了眼题目，感觉很简单，再一看，不就是一个模拟吗。

那接下来要知道的有几点：

- 每一局a和b只要分别:**a=a/2+a%2**(拿a举例)(如果不理解这个，可以自己推一下)
- 然后每一局n都要除以2
- 每一局计数，最后看下n是否等于1，是说明最后只有a和b，否说明除了a和b还有其他队

知道这些后就很简单了，代码：
```cpp
#include<Bits/stdc++.h>
using namespace std;
int n,a,b,s=0;
int main() {
	cin>>n>>a>>b;//输入
	while(a!=b) {//a和b不等时循环
		a=a/2+a%2;
		b=b/2+b%2;
		s++;//累加
		n/=2;
	}
	if(n!=1)cout<<s<<endl;//输出
	else cout<<"Final!"<<endl;
}
```
