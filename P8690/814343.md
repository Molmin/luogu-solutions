# P8690 [蓝桥杯 2019 国 B] 填空问题 题解

## problem A

枚举题。

我们可以用循环对 $x$ 枚举（$2020 \sim 5 \times 10^{4}$），每一次的 $y$ 就等于 $\lfloor \sqrt{2x^{2} - 2019^{2}} \rfloor$，如果 $y^2 - x^2 = x^2 + 2019^2$，那就跳出循环，输出 $x + y$。

## problem B

dp 题。

我们可以先预处理 $2$ 至 $2019$ 的指数（线性筛），然后用 dp $01$ 背包求解，状态转移方程为 $f_j = f_j + f_{j - p_i}$，$p_i$ 为第 $i$ 个质数。

核心伪代码：

```
for i = [0...sum]//这里的 sum 是素数个数
	for j = [2019, p(i)]
		f(j) = f(j) + f(j - p(i))
```

## problem C

搜索题。

我们可以把所有 $x = y$ 的坐标全部搜索一遍，也就是 $i$ 从 $0$ 至 $7$，每次从搜索 $(i, i)$。每次把 $v_{i, i} = 1$，其他 $= 0$。每一次往四个方向搜索，搜索完回溯。

核心代码：

```
if (x < 0 || x > 7 || y <= x || y > 7 || v[x][y] == 1) continue;//条件
vis[x][y] = 1;//标记
dfs(x, y);//搜索
vis[x][y] = 0;//回溯
```
## problem D

枚举题。

枚举每个数，每一次枚举它含有多少个约数，如是 $100$ 输出即可。

核心代码（筛约数）：

```
if (!(i % j))
{
	++ ans;
	if (p2(j) != i)//p2(j) = j * j，是一个自创函数
		++ ans;
 }
```
## problem E

搜索题。

跟 C 题相似，只不过是从 $(0, 0)$ 开始搜，$v$ 数组全为 $0$。如果长度 $> 12$，直接跳出，否则向四个方向搜索，搜索完回溯。
