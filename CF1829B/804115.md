思路：遍历一遍数组，如果遍历到的元素是 $0$ 就将连续 $0$ 的数量加一，否则和目前的答案取最大值再重新记录。

注意遍历完后需要再取一遍最大值，否则就可能错过答案在数组末端的情况。

关键代码：

```cpp
int maxn = 0, len0 = 0; 
for(int i = 1; i <= n; i++)
	if(a[i] == 0) len0++; // 是 0 数量加一
	else
		maxn = max(maxn, len0), // 不是 0 记录
		len0 = 0; // 连续 0 的数量归零
ans = max(ans, len0); // 最后再取一遍最大值
cout << ans << "\n"; // 输出，用 "\n" 通常更快
```
