看了看大部分的题解，直接模拟了整个过程并计数。但是，毕竟冒泡中，该浮上去的早晚都要浮上去，不用模拟，直接计数即可。

# 接下来就是最激动人心的代码了：

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,cnt=0,a[10010];
int main(){
	cin>>n;
	for(int i=0;i<n;i++){
		cin>>a[i];
	}
	for(int i=0;i<n;i++){
		for(int j=0;j<i;j++){
			if(a[j]>a[i]){
				cnt++;
			}
		}
	}
	cout<<cnt;
	return 0;
}
```
