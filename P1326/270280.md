# **回归正题**
### **题目描述：**
在中超联赛中，有一堆~~巨星~~蒟蒻在那里踢球。采取的是三分制。已知一只~~宇宙队~~菜鸡队伍进了S个球，丢了T个球，一共打了N场比赛。问这只菜鸡队最多能那几分，最少能拿几分？
### **思路分析**
#### **首先来考虑最大得分（本来就只能得那么几分，干嘛要把国足的自信心打压没呢？）**
大体思路：尽量把失球都集中在一起，其他比赛都是1:0小胜。

具体思路：

①：如果进球数大于等于比赛数。那么这时，我们可以保证有的N-1场比赛我们能全取三分，再用剩下的进球数和丢球数相比较，来确定最后一场能得到多少分。代码如下：
```
mx+=3*(n-1);
if(s-n+1>t) mx+=3;
if(s-n+1==t) mx+=1;
```
②：如果进球数少于失比赛数。这时候就没那么复杂了。直接可以确认赢了S场，平了N-S-1场。（注意：T=0分数要加1，不然就60了，不要问我这个分数是怎么来的）：
```
mx+=3*s;
mx+=n-s-1;
if(t==0) mx+=1;
```
#### **再来考虑最低分（丑媳妇迟早得见公婆）**
大体思路：最低分不像最高分那么好考虑，不是简简单单将最高分反了过来。来举个栗子：两场比赛，一个进球，一个丢球，问最小得分是什么？答案显然不是3（我1:0敌；我0:1敌）而是2（我1:1敌；我0:0敌）。所以这题要分两类来讨论：不赢球或者一把全赢光。

具体思路：

①如果进球数大于失球数。那么，这时不可能出现不胜的状况，最佳策略必为一场全赢光。上代码：
```
mi+=3;
if(t<n-1) mi+=n-t-1;
```
②如果进球数小于等于失球数，那么此时，有可能会出现不胜的状况。所以我们需要分类讨论。

情况一：一场全赢完。分数的初始值为3分（肯定会赢一场）。接下来如果T<N-1，必定会有场N-T-1平局，加N-T-1分。

情况二：一场不胜。如果T-S<N，必定会有N-T+S场平局，加N-T+S分。代码如下。
```
long long seca=3,secb=0;
if(t<n-1) seca+=n-t-1;
if(t-s<n) secb+=n-t+s;
mi=MIN(seca,secb);
```
这样，这道题就做完了。上整体代码（一定要开long long）:
```
#include <cstdio>
#include <cstring>
#include <cmath>
#include <cstring>
using namespace std;

long long MIN(long long a,long long b){
	if(a<b) return a;
	return b;
}

int main()
{
	long long s,t,n;
	while(~scanf("%lld%lld%lld",&s,&t,&n)){
		long long mx=0,mi=0;
		if(s<n){
			mx+=3*s;
			mx+=n-s-1;
            if(t==0) mx+=1;
		}
		else{
			mx+=3*(n-1);
			if(s-n+1>t) mx+=3;
			if(s-n+1==t) mx+=1;
		}
		if(t<s){
			mi+=3;
			if(t<n-1) mi+=n-t-1;
		}
		else{
			long long seca=3,secb=0;
			if(t<n-1) seca+=n-t-1;
			if(t-s<n) secb+=n-t+s;
			mi=MIN(seca,secb);
		}
		printf("%lld %lld\n",mx,mi);
	}
    return 0;
}
```
