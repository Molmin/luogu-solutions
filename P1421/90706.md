##  **这是一道非常友好的水题**

非常适合新手们来试一试。

数据类型double或者float知道吗？

来找找下面两个程序的不同点：

### 第一篇：

```
#include<bits/stdc++.h>
using namespace std;
int main()
{
    double a,b;
    int c;
    cin>>a>>b;
    c=(a+b/10)/1.9;
    cout<<c;
    return 0; 
}
```
### 第二篇：

```
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int a,b;
    cin>>a>>b;
    cout<<(a*10+b)/19;
    return 0; 
}
```

这两篇程序都可以AC，只不过数据类型不一样。

### 仅此而已

解释一下：

第一篇用了double的数据类型，直接输出除出来的结果会变成一个小数，以至于全WA了……………………

所以才要开一个整形变量c，就算除出来是1.9999999999999999999，赋值给一个整型变量…………

## 还是1！！！

没想到吧？？？

所以开一个整形变量c相当于把商向下取整，就可以丢掉余数。

比如样例中小玉有10.3元，除法算式：10.3/1.9=5（支）余0.8（元）  
c的作用就是把那个0.8元~~扔了~~不，是找回来，就是售货员还给小玉0.8元。

### 懂了吗？

第二篇，用的全是int类型，也就是把元都换成角了。

## 咳咳，开始讲课！

10.3/1.9=10.3乘10/1.9乘10=103/19

就是103角除以19角，等于5支多8角。
