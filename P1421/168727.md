现在来让我们总结一下本题的知识点吧：

细心的兄台们或许已经发现， “(a*10+b)/19” 这里是有可能除出余数的，但俺并没有写去余的代码，原因就是：

int类型有一个**自动过滤余数 **的性质

大家可以编编代码试一试，像酱紫：
```
#include <iostream>
using namespace std;
int main(){
	int a,b;
    cin>>a>>b;
	cout<<a/b;
	return 0;
}
```
如果我输入6跟3，输出则必然是2；
 
如果我输入5跟4，5/4=1.25，计算机就会把那个“0.25”给省掉，输出1；

再如果我输入5跟2，以及7跟4，5/2=2.5，7/4=1.75，如果按四舍五入的方法，应该输出3和2，但计算机 _是**省**掉小数，而不是**舍**掉小数_ ，所以应该输出2跟1；
