

#### 这题对新人来说还是比较友善的，做起来也比较简单，因为不知道你们有没有学到双精度double和强制转换，可能比较复杂，所以我会适当地加以说明。


------------
首先来了解一下double，double是双精度浮点数，用来存储小数的值，用法跟int类似。因为我们知道，一个int的变量是没有小数这个概念的，比如下面：
```
int a=3;
cout<<a/2<<endl;
```
结果就是1，因为3里面只有一个2，所以就输出1。

来看double，同样的代码，只要把int改为double，结果就会完全不同。
```
double a=3;
cout<<a/2<<endl;
```
会输出1.5，因为这里的3其实理解为3.0,3.0/2=1.5。
——————————————————————————
强制转换其实比较好理解，结构是（变量类型）（要转换的变量）。
打个比方:
```
int a=3;
cout<<(double)(a)/2<<endl;
```
会输出1.5，尽管a原来是个int型变量，但是在输出中已经强制转换成了double类型，3.0/1.5=2。
————————————————————————————
下面，就上代码了。
```
#include <iostream>
using namespace std;
int main()
{
	int a,b;   //定义两个int变量
	cin>>a>>b;   //输入变量
    /*这里元理解为整数，角理解为小数*/
	double c=0;    //定义一个变量c，用来存储总和
	c=a+b/10.0;   //计算元与角的和
	cout<<(int)(c/1.9)<<endl; //强转最后的得数，舍去尾数，得到结果
	return 0; //别忘了结束程序哦
}
```
