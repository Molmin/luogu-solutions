首先这里是头文件，在这个文件中包括cin，cout这些基础的关键词。作为一名c++选手，不引入库是没法做题滴。

```cpp
using namespace std;
```
这是一个标准，否则的话得加“std：：”。不过现在不用在意，只需要把它当作模版来打就好。


##接下来我们进入实际运算的部分。

```cpp
int main(){
```
这是程序的**主函数**，在你学到函数与递归章节之前，都可以暂且不理它，把它当作模板打

请注意每个大括号都应包括{"与"}"两个，否则在编译时将会出错。

```cpp
    int a,b,m;
    ```
定义一些变量以进行后面的运算。请注意没有定义的字母不能随意使用

```cpp
    cin>>a>>b;
    ```
cin是c++中一种比较基础的输入方式，输入a与b

   ```cpp
   m=10*a+b;
   ```
###  我们把a元b角转换为角的形式进行计算

   ```cpp
    cout<<m/19<<endl;
    ```
    
    
cout是一种输出方式。这里19指的是1元9角一份，做除法.

这里注意一下，在“/”当中，如果直接这样写的话，将会默认是舍弃余数后的。比如10/3=3······1，直接写10/3是整型变量，因此得到结果为整数商“3”

```cpp
    return 0;
```
在正规比赛中都要这样写一表示你已经结束了整个程序，从现在编程开始养成这个习惯，是有用处的～
