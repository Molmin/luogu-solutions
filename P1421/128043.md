1. 数学解题 按照给的样例，小玉有10元9角，而一只签字笔是1元9角，所以我们用10.9/1.9=5.7几，但是这是买笔，所以要用退一法，因此答案是可以买5支笔。
1. 注意事项 
   1. 如果用float的话，就必须要强制类型装换（至少我是这么认为的），比较	麻烦，但是，把钱以角来当单位的话，就可以完美解决这个问题。在计算机上	 的一个int型的变量，如果通过计算赋的值是一个小数，那么计算机会自动把这	个变量的小数点以后的全清除。这不是四舍五入，是退一。比如这个变量是		1.987654321，那么这个数存储时就是1。
1. 代码实现：
```cpp
#include<iostream>//c++基本头文件
using namespace std;//cin输入cout输出
int main()
{
int a,b; cin>>a>>b;//a是元b是角
cout<<(10*a+b)/19;//把元转换成角再计算
return 0;//养成好习惯
}
```
