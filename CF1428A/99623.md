如果起点与终点在同一直线上（严谨来说，是如果以它们为端点的线段平行于坐标轴），那么兔兔直接拉向终点就好了，答案就是**它们间的距离**。

但是如果不是，那么兔兔就要拐一次弯。比如起点在 $(1,1)$，终点在 $(4,4)$，那么兔兔先拉着箱子向右走，直到箱子到达 $(4,1)$，此时兔兔在 $(5,1)$。为了转向，兔兔必须要先走到 $(5,2)$，然后走到 $(4,2)$，然后继续向上拉直到终点。不难看出，拐弯的时候必须要多耗费两秒。所以这种情况下，答案要在**起点与终点的曼哈顿距离下**再加上 $2$。

下面是代码（毒瘤码风）：

```
#include<bits/stdc++.h>
#define y1 brown //防止与cmath的y1重名
using namespace std;
int T,x1,y1,x2,y2,ans;

int main(){
	cin>>T;
	while(T--){
		cin>>x1>>y1>>x2>>y2;
		ans=abs(x1-x2)+abs(y1-y2);
		if(x1!=x2&&y1!=y2)ans=ans+2;
		cout<<ans<<'\n';
	}
}
```
