#### 算法分析

只需要考虑2种情况。

Case 1:
起点和终点在同一个方向上，所需的最小时间为$|x1-x2|+|y1-y2|$.

Case 2:
当起点和终点不在同一个方向时，兔子可以先向右把盒子拉$|x_1 - x_2|$个单位，而此时的兔子恰好在$(x_2 + 1, y_1)$这个位置，然后兔子先抛弃盒子向上走一个单位并且向左走一个单位，此时它位于终点处，然后拉盒子向上走$|y_1 - y_2|$个单位，于是盒子恰好被拉到终点处.所以最终所需的最小时间为$|x_1-x_2|+2+|y_1-y_2|$.


#### C++代码

```cpp
#include <bits/stdc++.h>

int main() {
	int t;
	std::cin >> t;
	
	while (t--) {
		int x1, y1, x2, y2;
		std::cin >> x1 >> y1 >> x2 >> y2;
		if (x1 == x2 or y1 == y2) {
			std::cout << abs(x1 - x2) + abs(y1 - y2) << '\n';
			continue;
		}
		std::cout << abs(x1 - x2) + 2 + abs(y1 - y2) << '\n';
	}
	
	return 0;
}
```
