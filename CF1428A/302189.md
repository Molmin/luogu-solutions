#### 题意

给你两个坐标，让你求出把箱子从一个坐标移动到另一个坐标的最小步数

#### 思路

1.可以知道的是，当有一对横坐标或者一对纵坐标相等的话，我们的兔子就可以直接拉过去，不用转弯

步数就是

```c++
abs(x2-x1)和abs(y1-y2)取个max
```

因为我懒得判断那个是横坐标相等还是纵坐标相等了

2.当不满足至少有一对横坐标或者一对纵坐标相等的话，我们的兔子就得转圈圈了，转一次的步数是2，

让步数最少，肯定就是让兔子转弯的次数最少，那么我们的兔子可以先到一个和目标位置横坐标或者纵坐标相等的一个位置

步数就是

```c++
abs(x1-x2)
```

然后拐个弯步数+2，但是拐弯并不会影响我们推的箱子的位置

所以再加上个

```c++
abs(y1-y2)
```

最后的答案就是

```c++
abs(x2-x1)+abs(y2-y1)+2
```

代码挺短的

#### 代码

```c++
#include<bits/stdc++.h>
using namespace std;
int t;
int main()
{
	cin>>t;
	for(int i=1;i<=t;i++)
	{
		long long x1,y1,x2,y2;
		cin>>x1>>y1>>x2>>y2;
		if(x1==x2||y1==y2)
		cout<<max(abs(x2-x1),abs(y1-y2))<<endl;
		else {
			cout<<abs(x2-x1)+abs(y2-y1)+2<<endl;
		}
	} 
	return 0;
 } 
```
