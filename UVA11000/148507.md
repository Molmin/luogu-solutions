这道题比较简单。其实，这只需要打一张表，每次读里面的数据就行了。
| day | 0 |  1| 2|  3|
| -----------: | -----------: | -----------: | -----------: | -----------: |
| male |0  | 1 | 2 | 4 |
| female |  1|  1| 2 | 3 |
|  total| 1 | 2 | 4 | 7 |

我们可以得出：

$$
male[i]=total[i-1]$$
$$
total[i]=total[i-1]+male[i-1]+1
$$
不错，代码就很容易了。


```c++
//main函数部分
int a[1005],b[1005];
int main()
{
	int n;
    a[0]=0;
    b[0]=1;
  	for(int i=1;i<=1000;i++)
    {
    	a[i]=b[i-1];
      b[i]=a[i-1]+b[i-1]+1;
    }
   while(cin>>n)
   {
   if(n==-1) return 0;
   cout<<a[i]<<' '<<b[i]<<endl;
   }
}
```
