## 题意
有一种蜜蜂，这种蜜蜂的一只雌蜂生育一只雄蜂，而一只雄蜂生育一只雌蜂和一只
雄蜂。科学家想知道在 $n$ 年后会有多少蜜蜂。请计算 $n$ 年后雄蜂的数量和所有的蜜蜂总数。

## 思路
题目描绘了两个数列，分别是代表雄峰数量的数列 $a$，以及代表雌峰数量的数列 $b$，$a_n$、$b_n$ 代表第 $n$ 年雄峰、雌峰的数量。根据题意可得：
$$a_i=a_{i-1} + b_{i-1}$$
$$b_i=a_{i-1}$$


然后按公式构造序列即可。

## 代码
```
#include<bits/stdc++.h>
using namespace std;
int a[100005],b[100005],n;
int main()
{
    a[0]=0;b[0]=1;
  	for(int i=1;i<=1000;i++){
    	a[i]=b[i-1];
        b[i]=a[i-1]+b[i-1]+1;
    }
    while(cin>>n){
    	if(n==-1) return 0;
   		cout<<a[n]<<" "<<b[n]<<endl;
    }
}
```
