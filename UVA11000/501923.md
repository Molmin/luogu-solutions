# 题意

**每年，一只雌蜂生育一只雄蜂，而一只雄蜂生育一只雌蜂和一只雄蜂。**

但原本只有一只雌蜂，且~~千年不老~~是不死的。

# 分析

第 0 年，一共有 1 只蜜蜂，是雌蜂。

第 1 年，一共有 2 只蜜蜂，其中有 1 只雄蜂，1 只雌蜂。

第 2 年，一共有 4 只蜜蜂，其中有 2 只雄蜂，2 只雌蜂。

第 3 年，一共有 7 只蜜蜂，其中有 4 只雄蜂，3 只雌蜂。

……

可以得出：

设 $a[n]$ 指第 $n$ 年雄蜂个数，$b[n]$ 指第 $n$ 年蜜蜂个数。

第 $n$ 年，一共有 $b[n-1]$ 个雄蜂，一共有 $a[n-1]+b[n-1]+1$ 个蜜蜂。

原因：

雌蜂和雄蜂都可以生育一只雄蜂，所以 $a[n]=b[n-1]$。

雌蜂和雄蜂都能生育一只雄蜂且雄蜂又可以生育一只雌蜂，但是还有一只千年不老的雌蜂，所以 $b[n]=a[n-1]+b[n-1]+1$。

# Code

```cpp
#include<bits/stdc++.h>
using namespace std;
//此处为快读代码
const int N=1000+10;
ll n,a[N],b[N];
int main(){
	a[0]=0,b[0]=1;
	for(int i=1;i<=1000;i++){
		a[i]=b[i-1];
		b[i]=a[i-1]+b[i-1]+1;
	}
	while(true){
		read_(n);
		if(n==-1)break;
		write_(a[n]," ",b[n],"\n");
	}
	return 0;
}
```
