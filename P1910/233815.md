知道是二维背包以后，来，找状态转移方程

背包最重要的部分是状态转移方程。

一维方程：f[j]=max(f[j],f[j-w[i]]+c[i]);

二维就是转一下：f[j][k]=max(f[j][k],f[j-w[i]][k-v[i]]+c[i]);

知道状态转移方程以后，来，上代码


------------代码分割线------------




```cpp
#include<bits/stdc++.h>//万能头文件，不多解释了
using namespace std;
int n,W,V,w[110],v[110],c[110],f[1010][1010];
int main()
{
	cin>>n>>W>>V;//W代表总侦查能量，V代表手中只有V块钱
	for(int i=1;i<=n;i++)
	{
		cin>>c[i]>>w[i]>>v[i];
      //输入价值c，条件量w，v; 
	}
	for(int i=1;i<=n;i++)
	{
		for(int j=W;j>=w[i];j--)
		  for(int k=V;k>=v[i];k--)
		    f[j][k]=max(f[j][k],f[j-w[i]][k-v[i]]+c[i]);
	}
	cout<<f[W][V];
	return 0;
}
```
