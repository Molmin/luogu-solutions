题目给出了很多概念，这里重新整理一下：

{
路径：从A到B点经过的的边组成，路径长度为各边的权值的和；

树网的直径：图中最长的路径

点到路径的距离：点v到路径F的距离=min( d（v ,u）,u∈F)；

偏心距：图中距路径F最远的节点到路径F的距离（一定要澄清这个概念）

}
概念澄清后，题目要求出一个在直径上的路径F，其长度不超过S，令到其偏心距最小。

题目要我们输出偏心距的最小值。

题目说了这是一个无圈且连通的无向图，显然这是一棵树，只不过没有规定根节点而已，因此我们想到，在树中，最短路等于最长路。

所以我们可以用floyd算法求出各点之间的距离，而且还要记录中间节点，中间节点我们可以用一个数组mid[][]来表示:

if g[I, K] + g[K, J]<g[I, J] then 

     g[I, J]=g[I,K]+g[K, J]；

     mid[I, J]=K；    

同时，我们也可以找到直径。{题目说直径不唯一，但是可以证明，我们只要求其中一条直径就够了(证明略，ms许多大牛的题解有证明)。}

找到直径后，我们用一个数组来记录直径上的节点，记录了直径上的节点之后，我们便开始枚举路径F，枚举的方法就是枚举起点和终点，然后又枚举图里面的其他节点，计算当前路径F的偏心距，若当前路径F的偏心距小于目前的最优值，就替换。

这样此题就可以圆满解决了。时间复杂度大概是O(n^3).
