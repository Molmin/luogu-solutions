### 题目大意

有 $N$ 个人围成一圈，编号 $1\sim N$，现在从 $1$ 号开始数，每数 $D$ 个人，就将那个人踢出圈子，剩下的人就是胜者，求胜者的编号。（$N,D<10^6$）

### 思路分析

这不是典型的约瑟夫问题吗？

一种暴力的写法就是按照题意模拟，但是显然会超时，这里不放代码了。

那么怎么办呢？

我们设 $f(N,D)$ 为共 $N$ 个人，每数 $D$ 个便被踢出去的胜者的编号。

例如当 $N=5,D=3$ 时，此时环为 $1,2,3,4,5$，当踢出第一个人（$3$ 号）后，该环变为 $4,5,1(6),2(7)$，其答案应当就是 $N-1$ 个人每 $D$ 个就被踢出的答案加上 $D$ 再对 $N$ 取模。写成数学语言就是，

$$f(N,D)=(f(N-1,D)+D)\bmod N$$

为了调整当答案正好为 $N$ 的情况，我们对这个式子略加修改。

$$f(N,D)=(f(N-1,D)+D-1)\bmod N+1$$

所以我们可以去掉 $D$ 这一维度。

$$f(N)=(f(N-1)+D-1)\bmod N+1$$

易得，

$$f(N)=1+\sum_{i=2}^{N}(D\bmod i)$$

### AC Code

```cpp
#include <bits/stdc++.h>
using namespace std;
#define int long long
int n, d, w;
signed main()
{
    while(cin >> n >> d) {
        w = 0;
        if(n == 0 && d == 0)
            break;
        for(int i = 2; i <= n; i++)
            w = (w + d) % i;
        cout << n << " " << d << " " << w + 1 << endl;
        // 注意本题奇怪的输出格式。
    }
    return 0;
}
```
