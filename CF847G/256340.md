题意：一共有 $\ n$ 个小组，每个小组都可以选择 $\ 0-7$ 时段上课，一个教室同时间只可以让 $\ 1$ 个小组上课，求最少需要的教室数。

思路：题目让求最少所需的教室数目，也就可以想成在同时在上课的小组数最多的一次的数目。就是输入的矩阵中的每一列的数字 $\ '1'$ 的个数的最大值。

方法：双层循环读入并统计矩阵每一列的数字 $\ 1$ 的个数，然后自用一次循环找出最大值，输出即可。
```cpp
#include<stdio.h>
int n,c[7],t,s,k,maxn;
int main()
{
	scanf("%d",&n);	//读入小组的数量。
	for(;t<n;t++)
	{
		for(s=0;s<7;)
		{
			k=getchar();	//读入一个字符
			if(k=='0'||k=='1')	//把 '\n' 、'\r' 等字符过滤掉，防止影响统计。
			{
				c[s]+=k-'0';	//把对应数组加一。
				s++;	//循环的变量加一。
			}
		}
	}
	for(t=0;t<7;t++)
	{
		maxn=maxn>c[t]?maxn:c[t];	//统计出七个和的最大值。	
	}
	printf("%d",maxn);	//输出最大值。
	return 0;	//Don't forget this.
}
```
