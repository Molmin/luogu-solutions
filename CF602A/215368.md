### 前置知识
从高到低依次给出你一个 $n$ 位十进制数各位的数字，输出这个数。

**样例输入：**
```cpp
4
2 0 1 9
```

**样例输出：**
```cpp
2019
```

**思路：**  
简单题，有多种解法，这里介绍其中一种。  
我们单独设一个变量 $x$，初值为 $0$。对于每一位数字，先将 $x\times 10$，然后加上这一位数字。

手玩一下。  
最初 $x$ 的值为 $0$。输入 $2$，$0\times 10+2=2$，$x$ 的值变为 $2$。  
输入 $0$，$2\times 10+0=20$，$x$ 的值变为 $20$。  
输入 $1$，$20\times 10+1=201$，$x$ 的值变为 $201$。  
输入 $9$，$201\times 10+9=2019$，$x$ 的值变为 $2019$。  
循环结束，$x$ 此时的值即为所求的答案，输出 $x$ 即可。

如果仍然不好理解，请结合代码实现。

**CODE：**
```cpp
scanf("%d", &n);
for (int i = 1; i <= n; ++i) {
	int num; scanf("%d", &num);
	x = x * 10 + num;
}
printf("%lld", x);
```

### 题目大意
这道题是前置知识的拓展版。

给你两个数：$n$ 位 $a$ 进制数 $x$，$m$ 位 $b$ 进制数 $y$，从高到低依次输入它们各位数字的十进制表示。要求你比较它们的大小。  
如果相等，输出 $``="$；如果 $x>y$，输出 $``>"$；如果 $x<y$，输出 $``<"$。

手玩一下样例 $\#1$：  
$6$ 位 $2$ 进制数 $(101111)_2=(47)_{10}$；  
$2$ 位 $10$ 进制数 $(47)_{10}=(47)_{10}$；  
两数相等，输出 $``="$。

### 思路
观赏一下题目大意。  
比较大小好说，把这两个数转成十进制，直接比较即可。  
问题是转成十进制的过程，怎样做较为简便。

不难发现，这道题与前置知识处的差别在于，前置知识输入的是十进制数，而这道题输入的可以是任意进制。  
所以将前置知识推而广之：当基数为 $10$ 时，对于每一位数字，先将 $x\times 10$，然后加上这一位数字；同理，当基数为 $R$ 时，每一步操作改为将 $x\times R$ 即可。

### AC CODE
```cpp
#include <iostream>
#include <cstdio>
using namespace std;

typedef long long ll;

int n, a;
int m, b;
ll x, y;

int main() {
	scanf("%d%d", &n, &a);
	for (int i = 1; i <= n; ++i) {
		int num; scanf("%d", &num);
		x = x * a + num;
	}
	scanf("%d%d", &m, &b);
	for (int i = 1; i <= m; ++i) {
		int num; scanf("%d", &num);
		y = y * b + num; 
	}
	x == y ? putchar('=') : (x > y ? putchar('>') : putchar('<'));
	return 0;
}
```
