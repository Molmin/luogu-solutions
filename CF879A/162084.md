这道题思路非常简单。

首先先模拟一下样例。
```
3
2 2
1 2
2 2
```

- $day1:$发现第$1$个医生没有工作。

- $day2:$发现第$1$个医生开始工作，赶紧去看。

- $day3:$发现第$2$个医生今天工作，赶紧去看。

- $day4:$发现第$3$个医生今天工作，赶紧去看。

然后就发现$4$天可以看完医生。

接下来就是代码了。

```
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int n;
	cin>>n;
	int a[n][2];
	int day=0;//这个是定看医生的天数。
	for(int i=0;i<n;i++)
	{
		cin>>a[i][0]>>a[i][1];//因为病人得按顺序看病，所以可以直接进行模拟。
		while(1)//死循环，因为天数是不确定的。
		{
			day++;//开始先天数+1。
			if((day-a[i][0])%a[i][1]==0&&day>=a[i][0])//如果这天那个医生已经开始工作且今天是那个医生的工作日。
			{
				break;//跳出循环。
			}
		}
	}
	cout<<day;//输出天数。
	return 0;
}
```
