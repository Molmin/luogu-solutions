## 题意：
一个人要看医生，一共要看 $n$ 个医生，题目给出每个医生第 $x$ 开始工作，每隔 $y$ 天工作一次。求看完所有医生要多少天。

**注：必须按照输入的顺序去看医生!**
## 分析：
不用开数组，边输入边计算就行了。由题可知：当天数 $k\geqq x$ 并且 $y$ 能被 $(k-x)$ 整除时，就符合题意。按照这个思路用一个循环不断输入判断就可以得出答案了。

代码如下：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int n,x,y,k=0,i;
	cin>>n;
	for(i=0;i<n;i++){
		cin>>x>>y;
		for(;;){//无限循环
			k++;//每次加一天
			if((k-x)%y==0&&k>=x)//当满足题意时，跳出循环。
				break;
		}
	}
	cout<<k;//最后输出结果。
	return 0;
}
```
