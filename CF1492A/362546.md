### 这道题是什么意思：

有 $t$ 组数据，每一组里都有 $p,a,b,c$ 四个数。在每一组数据中，三个游泳者们都会从泳池左边出发。第一个游泳者 $a$ 分钟游一个来回，第二个游泳者 $b$ 分钟游一个来回，第三个游泳者 $c$ 分钟游一个来回。你在三个游泳者开始游泳后 $p$ 分钟来到泳池的左边，问你在多少分钟后能遇到一个到达泳池左边的游泳者。

------------

### 解题思路：
1. 用 $p$ 分别除以 $a/b/c$，得到三个游泳者到左边的时间离 $p$ 最近的应该是多少 $a/b/c$（如果不整除就把这个值 $+1$）。

2. 把这三个值 $\times a/b/c$。

3. 把这三个值减去 $p$，得到你在左边分别遇到 $3$ 个游泳者的最短时间。

4. 比较一下，输出那个最小的值即可。

最后看一下数据范围，$1 \le p,a,b,c \le 1e18$，这就代表我们要开 ```long long```。

------------

### C++ Code:

```
#include<iostream>
#include<bits/stdc++.h>
using namespace std;
long long t,p,a,b,c;
int main()
{
	long long n=0,j=0,k=0,l=0,s=0;
	cin>>t;
	for(int i=1;i<=t;i++)
	{
    	cin>>p>>a>>b>>c;
        n=p/a;
        if(p%a!=0) n++;
        n=n*a-p;
        s=n; //因为没有东西给它比较所以就先存为最小值。
        n=p/b;
        if(p%b!=0) n++;
        n=n*b-p;
        if(n<s) s=n;
        n=p/c;
        if(p%c!=0) n++;
        n=n*c-p;
        if(n<s) s=n;
        cout<<s<<endl;
	}
    return 0;
}
```
