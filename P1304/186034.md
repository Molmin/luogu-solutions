### 对，就是暴力。

我比较喜欢调用函数，这里有两个函数：

1. 暴力枚举的函数

我们不需要双重循环凑数，因为太耗时。如果只有两个数的话，只要枚举一个数，判断$i$和$x-i$是否为质数即可。

题目说只要输出第一个数字典序最小的答案，所以找到后可以直接退出了。

```cpp
void f(int x) {
	for(int i=2; i<=x-1; i++) {//i为2，因为2是最小的质数
		if(pd(i) && pd(x-i)) {//调用pd函数
			cout<<x<<"="<<i<<"+"<<x-i<<endl;//输出
			return ;//退出
		}
	}
}
```


2. 质数判定的函数

这一个函数就不细讲，相信大家再熟悉不过了。

```cpp
int pd(int y) {
	for(int i=2; i<=sqrt(y); i++)//在平方根里找因数
		if(y%i==0) return 0;//找到因数
	return 1;//是个质数
}
```

解释一下为什么$if(n$%$2==1) n--;$

我自己的看法是，要求验证的是$4-n$中的偶数，如果$n$为奇数，不需要循环枚举。

```cpp
int main() {
	int n;
	cin>>n;//读入
	if(n%2==1) n--;
	for(int i=4; i<=n; i+=2) f(i);//开始暴力
	return 0;
}
```
