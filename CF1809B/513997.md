## 题意
+ 在二维平面上放置 $n$ 个整点
+ 每个点之间的欧几里得距离严格大于 $1$。
+ 放置点 $(x,y)$ 的花费为 $|x|+|y|$，求最大花费的最小值。

## 做法
该处的两点间距离为**欧几里得距离**，即直线距离。相邻的两个整点肯定是不行的，因为它们的距离刚好是 $1$。而点的花费是到原点**曼哈顿距离**，即横坐标与纵坐标之和。但是无论怎么说，肯定是放的离原点近一些要好。

为了能够让最远的的点离原点更近，我们肯定要节约空间。两个整点如果是放在一个单位正方形的对角上，那么它们的距离是 $\sqrt{2}$。

![](https://cdn.luogu.com.cn/upload/image_hosting/zfxthei9.png)

这是最节约空间的方法，可以很简单的证明没有哪两个整点的距离比 $\sqrt{2}$ 还小却比 $1$ 大的。因此我们可以把所有的点全部按照这种方式来摆放，最终得到一个如下所示的网状结构：

![](https://cdn.luogu.com.cn/upload/image_hosting/5d5b7kyo.png)

或者也可以不在原点上放置，如下所示：

![Never gonna give you up](https://cdn.luogu.com.cn/upload/image_hosting/4s65vcsb.png)

当你把你的头倾斜四十五度（或者把你的电脑倾斜四十五度），你会发现这些蓝色的点构成了一个很好看的正方形。前一幅图里的那个正方形边长为 $5$ 个点，里面有 $25$ 个点。后一幅图里的那个正方形边长为 $4$ 个点，里面有 $16$ 个点。  
显然，我们可以按照算面积的方式来算点的个数。而题目中给出了点的个数，自然也能求出边长的最小值。

已知边长的最小值了，怎么求最小花费呢？相信大家不难发现，规律就是花费比边长小 $1$。（不会找规律的话可以多画几个）这是可以严谨证明的，过程不详于此。

那么，我们已经得到了求答案的方法了，其实很简单。设答案为 $ans$，则可以得到 $ans = \lceil {\sqrt n} \rceil - 1$，经过一些不详数学推导后我们可以得到更简单的版本，即 $ans = \lfloor {\sqrt {(n-1)}} \rfloor$。

## 代码
```cpp
#include <iostream>
#include <cmath>

using namespace std;

int main() {
  int t;
  cin >> t;
  while(t--) {
    long long n; //注意数据范围
    cin >> n;
    cout << (long long)sqrt(n - 1) << '\n';
  }
}

```
