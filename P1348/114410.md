**拿例题看：
1~10判断：

1=1^2-0^2

2=没有 //（2-2）%4==0（也算）

3=2^2-1^2

4=2^2-0^2

5=3^2-2^2

6=没有 //（6-2）%4==0

7=4^2-3^2

8=3^2-1^2

9=5^2-4^2

10没有 //（10-2）%4==0

不难发现，当判断数字k时，看看（k-2）是否是4的倍数，若是4的倍数，则不是Couple number

代码如下：

```
#include<bits/stdc++.h>

using namespace std;

long long n1,n2,i,sum=0;

int main(){

	cin>>n1>>n2;
    
	sum=n2-n1+1;//共有n2-n1+1个数
    
	for(i=n1;i<=n2;i++){
    
		if((i-2)%4==0)sum--;//判断
        
	}
    
	cout<<sum;
    
}
```
