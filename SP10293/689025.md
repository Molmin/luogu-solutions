## 题目大意
这道题其实是要我们把这个字符串分割成若干字串，而且分割出的子串中的数字必须相同。

------------

## 分析


假设我们用一块隔板分割字符串，那么两个字符之间可能插了隔板也可能没插，这样就有两种情况了。而且题目要求我们分割出来的子串必须有相同的数字组成，所以我们不能不在不同的字符之间插隔板。

因此，我们可以得出结论，只需统计有多少个相同的字符就可以了。

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
string s; 
int n,ans;
int main(){
    cin>>n;//输入电话号码个数
    for(int i=1;i<=n;i++){
        ans=0; //记得归零
        cin>>s;
        for(int j=0;j<s.size()-1;j++){
            if(s[j]==s[j+1])ans++;//统计有几个相同的地方
        }
        long long t=1;//由于电话号码长度小于三十，所以最坏情况用乘方函数可能会爆，所以还是自己来打一下
        for(int j=1;j<=ans;j++){
            t*=2;
        }
        cout<<t<<endl;
    }
    return 0;
}
```
