 这道题其实用普通的数学方法就能做，因此大家只要掌握了基本的输入输出方法就没问题了。由于最后只能剩余一个选手（也就是冠军），因此需要进行选手数减一场比赛，每场比赛都有两个选手参加，因此要乘每个选手要用的水的数量的二倍，再加上每场比赛裁判要用的一瓶水就可以了。毛巾的计算方法也很简单，只要用选手数乘上每个选手要使用的毛巾数就可以了。

# AC代码如下
```cpp
#include<iostream>//输入输出流头文件
using namespace std;
int main(){
    int n,b,p,x,y;//定义变量
    cin >> n >> b >> p;
    x=2*(n-1)*b+(n-1)*1;//水
    y=n*p;//毛巾
    cout << x << ' ' << y << endl;//记得打空格
    return 0;//程序结束
}
```
