### 良心T1

## 题意

$n$ 个选手进行若干场比赛（每场比赛为两人，胜者晋级，败者淘汰）

1.每场比赛每位选手需要 $b$ 瓶水，裁判需要 $1$ 瓶水；

2.每位选手每次比赛需要 $p$ 块毛巾.

询问一共需要多少瓶水，多少块毛巾?

## 分析

**方法1：朴素算法（期望 $100$ 分）**

模拟比赛进程即可。

每次的比赛场数为小于当前 $n$ 的最大 $2$ 次幂即可。

特别地，需要的毛巾数即为 $n \times p$ (根据题意）

Code:

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,b,p,ans1,ans2;
int main()
{
	cin>>n>>b>>p;
	ans2=n*p;
	while(n!=1)
	{
		int j=1;
		while(j<=n)j*=2;
		int k=j/2,kk=n-k;
		ans1+=(k/2)*(2*b+1);
		n=k/2+kk;
	}
	cout<<ans1<<" "<<ans2;
}
```

**方法2：数学（期望 $100$ 分）**

原来有 $n$ 人，淘汰到最后成了 $1$ 人，而每次比赛只能淘汰 $1$ 人，说明比赛进行了 $n - 1 $场。

所以水瓶要$(2 \times b + 1) \times (n+1)$个。

代码与一除水瓶计算部分以外基本雷同，故代码就不贴了。
