# 回归正传

这道题我们不用管是怎么比赛，总而言之，就一共比了$n-1$场。每一轮比赛,都是两个人比,所以我们就能得出每一轮需要$2\times b+1$瓶水，一共需要$(n-1)\times(2*b+1)$瓶。每位选手每次比赛需要 $p$ 块毛巾，一共需要 $p \times n$ 块毛巾。  
AC代码如下，请勿抄袭：
```cpp
#include <iostream>
using namespace std;
int main(){
    int n,x,y,s=0;
    cin>>n;
    cin>>x>>y;
    for(int i=1;i<n;i++)
    {
        s+=2*x+1;
    }
    cout<<s<<" "<<n*y<<endl;
    return 0;
}
