### 题目分析

每三个空瓶子可以换一瓶新的饮料，可以问别人借瓶子，但需要还回去，问最后能喝到几瓶饮料。

直接找规律即可。

- $n=2$ 借一个瓶子 $\to$ 三个空瓶换一瓶饮料 $\to$ 喝完之后空瓶还回去，共三瓶。

- $n=3$ 直接三瓶换一瓶，共四瓶。

- $n=4$ 借两个瓶子 $\to$ 六个空瓶换两瓶饮料 $\to$ 喝完之后空瓶还回去，共六瓶。

- $n=5$ 借一个瓶子 $\to$ 六个空瓶换两瓶饮料 $\to$ 其中一瓶还回去，共七瓶。

之后的推导过程略，得出结果如下：

| $n$ | $ans$ |
| - | - |
| $1$ | $1$ |
| $2$ | $3$ |
| $3$ | $4$ |
| $4$ | $6$ |
| $5$ | $7$ |
| $6$ | $9$ |
| $7$ | $10$ |

简单观察得出公式 $ans=3\lfloor\frac{n}{2}\rfloor+n\%2$。

### 代码

```cpp
#include<iostream>
using namespace std;
int n;
int main(){
	cin>>n;
	cout<<n/2*3+n%2<<endl;
	return 0;
}
```
