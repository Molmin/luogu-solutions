### 题意

本题原来的题面有点问题，不过不影响阅读。

正确的题面应为：

>四个袋子里有糖果要分给两个人。
>
>一个袋子只能给一个人，可以一人得到多袋，不能自己保留或者打开。
>
>如果**可以做到**两个人分得的和相同则输出 `YES`，反之输出 `NO`。

### 思路

那么我们的思路就是想办法做到让两个人分得的和相同。要做到这个，一共有两种可能：

1. 让一方拥有极大和极小的数量，另一方拥有没那么大也没那么小的数量；

2. 让一方拥有一个极大地数量并吊打另外三个极小的数量。

要实现这种判断语句，首先需要使输入的数据有序，也就是要排序。本题虽然输入数据数量不多，但是使用 `sort()` 函数会更方便一些。

### 代码

```cpp
#include <algorithm> //sort()
#include <iostream>
using namespace std;
int main() {
  int a[4]; //小数组可以定义为局部变量
  cin >> a[0] >> a[1] >> a[2] >> a[3];
  sort(a, a + 4);
  if (a[0] + a[1] + a[2] == a[3] || a[0] + a[3] == a[1] + a[2])
    cout << "YES" << endl;
  else
    cout << "NO" << endl;
  return 0;
}
```
