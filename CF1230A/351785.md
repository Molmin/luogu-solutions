### 题意
给出 ``4`` 个数，看能否将它们分成和相等的两部分。

### 思路
将这 ``4`` 个数存入数组 ``a[4]`` ，从小到大排序， 得到 ``a[1]`` ，  ``a[2]`` ， ``a[3]`` ， ``a[4]`` 。

有两种情况：
- 最大数和最小数的和等于次大数和次小数的和。(即 ``a[1]+a[4]=a[2]+a[3]`` )
- 最大数等于其它三个数的和。( ``a[4]=a[3]+a[2]+a[1]`` )

### 代码实现

$\texttt{code:}$
```cpp
/* light_chaser */
  #include<bits/stdc++.h>
  using namespace std;
  int a[4]={0};
  int main(){
      for(int i=1; i<=4; i++){
          cin>>a[i];
      }
      sort(a+1,a+4+1);
      if(a[1]+a[4]==a[2]+a[3]){
          puts("YES");
      }
      else if(a[4]==a[3]+a[2]+a[1]){
          puts("YES");
      } 
      else{
          puts("NO");
      }
      return 0;
  }
```
