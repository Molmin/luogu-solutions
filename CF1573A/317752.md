## 题目大意
给你$t(1<=t<=1000)$个长度为$n(1<=n<=100)$的数字$a(1<=a<=10^n-1)$，你可以对这个数字进行两种操作：
- 将该数字减1
- 交换任意两位数字

请问每个数字最少经过几次操作才能使该数字变为0

------------
## 没用的分析
将该数字减一其实就是将个位减一~~（废话~~

所以我们只要把大于0的每一位数字交换到个位再减到0即可

即：$ans_i=a_i+1$

它减到0的次数（也就是他的大小）加上交换的那一次

当然，个位上的数字就不用再交换了
```cpp
#include<bits/stdc++.h>
using namespace std;
int t,n,a[110],ans;
int main()
{
    scanf("%d",&t);
    while(t--)
    {
        ans=0;//记得每次清零
        scanf("%d",&n);
        for(int i=1;i<=n;++i)
        {
            scanf("%1d",&a[i]);//利用%1d，每次读入一位数字
            if(i==n)ans+=a[i];//由于是从高位到低位读，所以个位其实是第n位
            else if(a[i]) ans+=a[i]+1;
        }
        printf("%d\n",ans);
    }
    return 0;
}
```
