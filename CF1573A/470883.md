## 题意：
给你一个数，你可以做两种操作：
1. 将它的最后一位减一。
2. 交换任意两位。

问要做多少次操作才能把它变成 $0$ 。

## 思路1：
既然要变成零，从最后一位开始减。直接把最后以为减到零就可以了。然后我们和前面一个非零位呼唤后再把最后减成零......

所以说还是不要把 $A$ 题想难了啊！

AC CODE：
```cpp
#include<bits/stdc++.h>
using namespace std;
  
string s;
int t,n,sum,p;
  
int main()
{
    cin>>t;
    while(t--)
    {
        sum=0,p=0;//初始化
        cin>>n>>s;//输入
        for(int i=0;i<n;i++)//遍历字符串
       {
            sum+=s[i]-'0';//更新sum，-‘0’就是ASCII码减48，相当于把string变回int。
            if(i!=n-1&&s[i]-'0'>0) 
  						p++;//更新p
        }
        cout<<sum+p<<endl;//输出答案
    }
    return 0;
}
```
