## 题目大意
给定一个字符串，每次操作可以将这个字符串的最后一位减一或交换任意两位，求将这个字符串的所有位都变为 $0$ 的最少操作次数。
## 思路
令 $sum$ 为所有位的和，那么每次操作最多让 $sum$ 减 $1$，当 $sum=0$ 时操作完成。可以采取以下的操作方法：
* 当最后一位不是 $0$ 时，将最后一位减一。
* 当最后一位是 $0$ 时，交换任意一个非 $0$ 的数到最后一位。
* 重复以上操作，直到 $sum$ 为 $0$。

很容易知道这是最简单的操作方法。

因此如果令 $p$ 为最初序列中大于 $0$ 且不是最后一位的位的个数，那么答案就为 $sum+p$。
## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
string s;
int t,n,sum,p;
int main(){
    cin>>t;
    while(t--){
        sum=0,p=0;//初始化
        cin>>n>>s;
        for(int i=0;i<n;i++){//遍历字符串
            sum+=s[i]-'0';//更新sum
            if(i!=n-1&&s[i]-'0'>0) p++;//更新p
        }
        cout<<sum+p<<endl;//输出答案
    }
    return 0;
}
```
