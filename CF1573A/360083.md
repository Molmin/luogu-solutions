## 题目大意

1. 你可以将这个字符串(注意有前导零)的最左边的数字减去一。

2. 你可以将这个字符串的任意两个位置的数字交换。

现在让你求把一个给定的字符串通过以上操作**变成0**的**最小**操作次数是多少。

## 分析

模拟几个字符串，比如说下面这个:

```
007
1234
```
首先是 $007$，只有个位的 $7$，要用 $7$ 次 $1$ 操作，答案也就是$7$。

再是 $1234$，先把 $4$ 用 $1$ 操作处理掉，也就是 $4$，再把$1$，$2$ ，$3$分别移到个位上(运用 $2$ 操作)，用一样处理 $4$ 的方法处理掉这些数字(运用 $1$ 操作)，可以算出答案是 :  $1+2+3+4+1+1+1=13$，不难发现，答案是字符串上所有的数字之和加上不是个位的非零数位上的数字个数，那接下来我们就看代码吧！

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int t;//次数
int main(){
	ios::sync_with_stdio(false);//优化
	cin.tie(NULL);
	cout.tie(NULL);
	cin>>t;
	while(t--){
		int n,ans=0;//记得一定要清0，多组数据！
		cin>>n;
		string x;
		cin>>x;
		for(int i=0;i<x.length();i++){//循环没一个数字
			ans+=x[i]-'0';//注意是字符串，不能直接赋值
			if(i==x.length()-1){//判断这个数字是不是个位， 如果是， 则跳过
				continue;
			}
			else if(x[i]-'0'>0){//判断2操作用的次数
				ans++;
			}
		}
		cout<<ans<<endl;//记得输出
	}
}//完美撒花！
```
