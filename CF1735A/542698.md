### Solution
题目其实就是让我们在 $n$ 天中选出不相邻的 $3$ 天，让它们两两之间的距离的最小值最大。

分类确认怎样最小：

如果第 $1$ 天和第 $2$ 天相差很近，那么结果为它们的距离，显然不为最优解。

如果第 $1$ 天和第 $2$ 天相差很远，那么第 $2$ 天和第 $3$ 天会相差很小，结果为第 $2$ 天和第 $3$ 天的距离，显然也不为最优解。

我们发现，如果休假日期两两距离太近或太远，显然都不能达成最优解。

那就需要一种两全其美的方法，休假日两两的距离要尽量相等，聪明的你肯定想到了，那就直接把 $\lfloor \frac{n-3}{3}\rfloor-1$ 就好了，$-1$ 是因为第一天不能和最后一天相邻。

### Code
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int t;
	cin>>t;
	for(int i=1;i<=t;i++)
	{
		int n;
		cin>>n;
		cout<<(n-3)/3-1<<endl;
	}
}

```
