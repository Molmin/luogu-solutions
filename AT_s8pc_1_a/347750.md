题意：一辆火车从 $5$ 时开始经过`A`点， $23$ 时是第 $n$ 次回到`A`点，问这辆火车几分钟经过`A`点一次。

**solution：**

火车每次经过`A`点所用时间是相等的，这样火车开过了 $n$ 次`A`点，也就是开过了 $n-1$ 段路程（植树问题，两端都栽），总用时为 $19$ 小时，所以答案就是总用时除以每段用时。

注意点：

题目所给出的单位是小时，但输出要是分，那要将 $19$ 小时，化成 $1080$ 分钟。

 _**CODE：**_ 
 
 ```
#include<bits/stdc++.h>
using namespace std;
double ans;
int n;
int main()
{
    cin>>n;
    ans=1080.0/(n-1);//求出每段时间间隔，也就是题目所求的量。
    cout<<fixed<<setprecision(7)<<ans;//题目要求误差小于0.000001，那我们就将ans保留7位输出即可。
    return 0;
}
```
