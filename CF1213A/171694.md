## [题目传送门](https://www.luogu.com.cn/problem/CF1213A)

这是一道挺弱的题。

### 核心思路

对于每一个正整数$x$，有$x-1$或$x+1$消耗为$1$，有$x-2$或$x+2$消耗为$0$。

问题：求使每个数都相等时的最小消耗。

解析题意不难发现，消耗1实际上就是改变一个数的奇偶，而改变一个数的大小而不改变奇偶不消耗。

所以，问题就转化为：使数列奇偶性相同的最小改变次数。

也就是**奇数个数和偶数个数的最小值。**

### AC代码

上代码->

```

#include<bits/stdc++.h>/万头可以偷懒，但ccf不让用
using namespace std;//所以要背头文件
int j,o,n;//全局变量会自动把初始值赋为0
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){//边输入边处理
		int p;
		cin>>p;
		if(p%2==0) o++;//输入的数是偶数，偶数个数加一
		else j++;//输入的数是奇数，奇数个数加一
	}
	cout<<min(j,o);
	return 0;
}

```
