### 题目 + 方法分析
这道题就是一个子串搜索问题。这是蓝题？！不应该啊，对于 Python 的朋友们来说，这也太简单了吧！

这道题用 Python 的字符串自身方法，一行代码就能搞定。代码为 
`
str.count(obj)
`
方法就能实现。这个方法就是统计字符串 $str$ 中有多少个 $obj$ 字符子串。    
### 知识拓展

### 代码部分
代码如下，非常简单，不再做解释了。
```python
print(input().count(input()))
```
