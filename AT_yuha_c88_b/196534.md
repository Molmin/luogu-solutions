对本题进行分析，因为每个人可以取1,2或者3枚，所以两人总和最大可取4枚。所以对石头总数进行分析，如果对4取余，如果有余数，，只要先手将余数的石头取走，剩下的跟着对方取就行了，如果是四的整数倍，就后手取，满足两人每次取的总数为4即可。
```
#include<bits/stdc++.h>//万能库开头 
using namespace std;
int main()
{
	long long a;//定义石头数 
	cin>>a;//输入 
	if(a%4==0)//判断除以4的余数 
	{
		cout<<"GO"<<endl;//如果是4的整数倍输出GO
	}
	else
	{
		cout<<"SEN"<<endl;//否则输出SEN 
	}
	return 0;//结束 
}
```
