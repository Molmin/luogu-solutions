
## 这道题思路其实很简单：
## 你（A）和精灵（B)方法：

    
- ### 如果硬币和为4的倍数
####  那就后手
#### 你拿4-B，这样你们每一轮就都和为4了
#### 最后非你莫属


------------

- ### 如果硬币和不为4的倍数
#### 那就先手
#### 刚开始拿B%4
#### 这样就相当于上面一种情况了


------------
## 最后，上AC代码：
```
#include<bits/stdc++.h>
using namespace std;
int a;
int main(){
	cin>>a;	//输入
	if(a%4==0)	//按上述方法判断
		cout<<"GO"<<endl;	//输出
	else
		cout<<"SEN"<<endl; //你可以试试AT题不换行
    return 0;
} 
```
