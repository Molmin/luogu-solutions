这道题不愧为$div3\,T1$，是真的水啊。

首先我们理清题意就会发现，这道题要我们求每组数据下的$a$和$b$的补数。

补数是什么？补数实际上就是余数的逆运算，余数是整除后还能剩下多少，补数是一个数加上多少才能被整除，我们如果仔细看定义的话，我们就会发现：

补数是从余数开始，终止于除数，因为每次加的是$1$，所以我们可以推出如下式子：

### 补数$=$除数$-$余数。

现在我们可以把代码交上去了，但我们会发现，是WA，这个时候我们就需要特判了，因为当余数为$0$的时候补数同样为$0$，但按我们的代码会输出除数，所以我们发现可以把最终结果再模一遍除数，因为按我们的方法算出来的最大值也就是除数了，并且模一遍就能达到特判的效果。

接下来上代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
long long a,b[10005],c[10005];
int main(){
	cin>>a;
	for(int i=1;i<=a;i++) cin>>b[i]>>c[i];
	for(int i=1;i<=a;i++) cout<<(c[i]-(b[i]%c[i]))%c[i]<<endl;
} 
```
