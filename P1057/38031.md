###小学最简单的一道数学题

###正常某思是教过的

###我这里直接使用该思路：

**画一个表格，下标1标为1**

**1，0，0**

**一次循环，自己的值变成左边和右边的值之和**

**0，1，1**

**0，1，1**

**2，0，0**

**次数一到，结束操作**

**以下是代码**


```cpp
#include<iostream>
using namespace std;
int a[31][31]={0};  //建立表格 
int main()
{
    int n,m;
    cin>>n>>m;
    a[0][1]=1;  //标1标为1 
    for(int i=1;i<=m;i++)  //m次操作 
    for(int j=1;j<=n;j++)
    {
        if(j!=1) //边界条件 
        a[i][j]+=a[i-1][j-1];
        else
        a[i][j]+=a[i-1][n];
        if(j!=n)  //边界条件  
        a[i][j]+=a[i-1][j+1];
        else
        a[i][j]+=a[i-1][1];
    }
    cout<<a[m][1];  //输出最终值
```
return 0；
}
