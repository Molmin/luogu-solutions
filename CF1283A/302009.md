题目含义：


------------
### 题目背景

新年来了，你很高兴，想知道离新年还有多少分钟。你知道当前时钟显示h小时和m分钟，其中0≤hh<24 并且 0≤mm<60。我们使用24小时的时间格式!

你的任务是计算出距离新年还有多久的时间t
（时钟显示00时00分）

### 输入格式

输入的第一行包含一个整数t  (1≤t≤1439)——测试用例的数量。

下面的t行描述了测试用例。第i行包含两个整数h和m (0≤h<24, 0≤m<60)。

保证这个时间不是午夜，即不能同时满足以下两个条件:h=0和m=0。保证h和m都没有前导零。



### 输出格式

对于每个测试用例，输出新年前的分钟数t。

------------

说明了我们要求这个时间距离新年的时间（分钟）

我们看代码：

```cpp
#include<bits/stdc++.h>  //万能头文件
using namespace std;
int main() //定义主函数
{
	int h,m,t; //按题意定义变量
	cin>>t;
	for(int i=1;i<=t;i++) //读一个输出一个
	{
		cin>>h>>m;
		cout<<60*(24-h-1)+60-m<<endl; //小时减一加上分钟（这里要倒过来算）
	}
	return 0; //结束程序
 } 
```
