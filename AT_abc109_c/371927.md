
我们先设要到达的地方为 $C$ 。

推到：从现在的位置到 $C$ 肯定是答案 $D$ 的倍数。

结论：从现在的位置到前面或后面的 $C$ ，一定是固定的点。

方法：求现在的位置到任意一点间的距离的最大公约数，再遍历取答案。

注：求距离用到的减法会出现负数，记得要取下绝对值！



这是求最大公因数的辗转相除法

```cpp
inline int gcd(int a,int b){
	if(b==0) return a;
	else return gcd(b,a%b);
}
```

这是求绝对值的函数

abs(n)//这是求n的绝对值，要加cmath库
