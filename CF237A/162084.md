这道题就是让我们找重复最多的$(x,y)$。

让我们先来模拟一下样例$1$。

```
4
8 0
8 10
8 10
8 45
```
我们记这个重复的次数为$sum$。

$(8,0)$总共出现了$1$次。

$(8,10)$总共出现了$2$次。

$(8,45)$总共出现了$1$次。

那么我们选择最大的$2$,就是输出。

不过还有一个难点，就是寻找重复的$(x,y)$会$TLE$。

所以我们需要用到一个计数器来辨别这个数组是否被数过，可以大大加快程序的运行速度。

接下来了看看代码吧。

```
#include<bits/stdc++.h>
using namespace std;
int a[10000001][2];//记录（x，y）。
bool b[10000001]={0};//上文提到的计数器。
int main()
{
	int n;
	cin>>n;
	for(int i=0;i<n;i++)
	{
		cin>>a[i][0]>>a[i][1];
	}
	int sum=0;//记录（x,y）出现的次数。
	int max=1;//记录（x,y）最大出现的次数。
	for(int i=0;i<n;i++)
	{
		if(b[i]==0)//如果b[i]为0的话，可以开始进行计数。
		{
			for(int j=0;j<n;j++)
			{
				if(b[j]==0)//如果这个为b[j]为0,表示你还没记过这个数，可以开始比较。
				{
					if(a[i][0]==a[j][0]&&a[i][1]==a[j][1])//如果(a[i][0],a[i][1])=（a[j][0],a[j][1]）。
					{
						sum++;//sum+1。
						b[j]=1;//这个数组被统计过了，那么计数器变化。
					}
				}
				
			}
			if(sum>max)//如果这次重复的次数大于之前最大重复的次数。
			{
				max=sum;//max就要改变。
			}
			sum=0;
		}
		
	}
	cout<<max;//输出。
    return 0;
}
```
