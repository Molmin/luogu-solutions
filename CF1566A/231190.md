# CF1566A

## Description

  给你两个正整数 $n$ 和 $s$ 。 找到一个由 $n$ 个非负整数（不一定不同）组成的数组的中位数最大，且其元素的总和等于 $s$ 的数组 。输出这个最大中位数。我们称长度为 $m$ 的整数数组的中位数是第 $\lceil {\frac{m}{2}}\rceil$ 个的元素。注意，中位数是排好序的数组中的第 $\lceil\frac{m}{2}\rceil$ 个。 元素从 1 开始编号。 例如，数组 $[20,40,20,50,50,30]$ 的中位数是 $30$ 。

## Solution

要想让中位数最大，我们要让中位数之前的数尽量小，这样才能分配给中位数的尽量多，那中位数之前的不妨都放上 0。

然后我们考虑怎么把 $S$ 分配给这后面的数。

显然，如果要想中位数最大，其他数就要尽量小，而其他数又不能小于中位数，于是便产生了我的做法，平均分配。这样能够保证中位数得到的答案是最大的，如果有余数，显然也不能加到中位数上，肯定就是分配到后面的数上去了，不过我们也不用管这个。

所以答案就是 `s/(n/2+1)` 。


## Code

```c++
#include <bits/stdc++.h>
using namespace std;
#define int long long 
int T,n,s;
signed main()
{
	scanf("%lld",&T);
	while(T--)
	{
		scanf("%lld%lld",&n,&s);
		printf("%lld\n",s/(n/2+1));
	}
	return 0;
}
```
