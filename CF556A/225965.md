本题解与楼上二位的方法有所不同。
## 【题意】
若0和1相邻，则删去这两位数。
## 【思路】
我没有用string，而是用char来存储。每一次输入后判断：

①当前为0，上一位为1；

②当前为1，上一位为0；

若符合①或②则将数组下标向前移动2位，总长度n减2，for循环不会受到i和n的改变的干扰（n和i的差没有变，循环次数不变）。最后的剩余长度即为n的大小。
## 【代码】
```cpp
#include <cstdio>
int n;
char c[200005];
int main()
{
	scanf("%d",&n);getchar();//消去n后面的'\n'
	for(int i=1;i<=n;i++)
	{
		c[i]=getchar();
		if(c[i]=='0'&&c[i-1]=='1'||c[i]=='1'&&c[i-1]=='0') i-=2,n-=2;  //判断①②是否成立
	}
	printf("%d",n);
	return 0;
}
```
