## 前置知识：

**一点点初一的数学知识**

----

题目要求我们将三个数字组成一个三角形，而三角形三边关系满足**两边之和大于第三边**的性质，所以我们就可以依据这个性质来判定三条线段能否组成三角形。

假设有 $3$ 个数 $a,b,c$，如果它们可以组成三角形，那么任意的两边之和都大于第三边。

所以在判断时，只需要**找最小的两边之和是否大于第三边**就可以了，因为若最小两边之和大于第三边，那么较大的两边之和也大于较小的边。

所以我们就有了暴力的思路：枚举每个边成为第三边，再计算剩余两边之和。

因为题目要求只要有一组可以组成三角形就输出 `YES`，所以可以将整个数组由小到大排序，取每个边 $a_i$ 的后两个边 $a_{i-1}$ 和 $a_{i-2}$，这样可以保证 $a_{i-1}$ 和 $a_{i-2}$ 一定是小于 $a_i$ 的最大值。

由于 $n\leqslant 10^5$，所以排序算法要用快排，总时间复杂度为 $\mathcal{O}(n\log n)$。

## 代码如下：

```cpp
#include<iostream>
#include<cstdio>
#include<algorithm>
using namespace std;
#define N 100005
int n,a[N];
int main() {
    scanf("%d",&n);
    for(int i=1;i<=n;i++) scanf("%d",&a[i]);
    sort(a+1,a+n+1);    //快排
    for(int i=3;i<=n;i++) {    //从3开始枚举，因为1和2有可能使得下标为负，导致RE
        if(a[i]<a[i-1]+a[i-2]) {
            printf("YES\n");
            return 0;    //因为只要有一组就可以，所以直接结束
        }
    }
    printf("NO\n");
    return 0;
}
```
