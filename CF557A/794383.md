## 题意
一场比赛设一二三等奖，每一个参加比赛的人都能得奖，每个奖项都有一个获奖人数的上下限，求一种方案，使得一等奖数量最多，若有多种方案的一等奖数量相同，则取二等奖最多的。若二等奖数量相同，则取三等奖最多的。

## 思路
考虑**贪心**。

由于每个奖都有一个下限，则先设定答案为每一等奖项的下限，若还有未分配奖项的名额，则按一二三等的顺序分配给每一等奖项，这样显然最优。

## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,ans[10],sum[10],a[10];//ans记录答案，sum记录每一等奖项上下限之差，a记录最大值。
int main(){
	scanf("%d",&n);
	for(int i=1;i<=3;i++){
		scanf("%d%d",&ans[i],&a[i]);//先设定ans为下限
		sum[i]=a[i]-ans[i];//计算差值
		n-=ans[i];//减去已经分配的名额
	}
	for(int i=1;i<=3;i++){
		if(n>=sum[i])ans[i]=a[i],n-=sum[i];//如果可以达到上限，则分配成上限
		else ans[i]+=n,n=0;//否则全部放到这等奖并清空名额
		printf("%d ",ans[i]);//输出
	}
}

```
