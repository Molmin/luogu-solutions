#### 这是一道简单的递推
###### [博客食用更佳](https://www.luogu.com.cn/blog/yx20070829/solution-cf171g)
- 题目中已经给出了递推式：$f_n=f_{n-1}+f_{n-2}$
- 简单的来说就是当前项为前两项之和
- 所以就只需要一个数组来进行递推：
```cpp
f[i]/*当前项*/=f[i-1]/*前一项*/+f[i-2]/*大前项*/;
```
- 接着亮一下代码：
```cpp
#include<cstdio>
int f[21],n;//数组开的刚刚好，要注意数据大小
int main()
{
	scanf("%d%d%d",&f[0],&f[1],&n);//读入（注意是第零项和第一项）
	for(int i=2;i<=n;i++)f[i]=f[i-1]+f[i-2];//for循环递推
	printf("%d",f[n]);//输出
	return 0;//拜拜
}
```
