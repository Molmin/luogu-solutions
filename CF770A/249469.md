恩，题目总结的很清楚，这个字符串满足：

1、长度必须是 $n$

2、必须是由小写字母组成

3、共有 $k$ 种不同的字符

4、任意两个相同字符不可以相连

1,2无需解释，我们这里详细讲 3,4 ，共有 $k$ 种不同的字符，为了方便，我们就用a,b,c,d,……,y,z 的顺序作为密码组成字符即可，而任意两字符不相连，则可以直接输出abcd abcd abcd …… 即可。

代码：

	#include<stdio.h>
	using namespace std;
	int main(){
	    int n,k,i;
   	    scanf("%d%d",&n,&k);//输入
  	    for(i=0;i<n;i++)printf("%c",'a'+i%k);//如上规则依次遍历，输出
        return 0;
	}
