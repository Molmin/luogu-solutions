## 思路：
为什么好多大佬都说这题很难啊？我感觉很简单的啊！

要都变成零的话，那至少得有一个变零吧，那我们让 $P_1$ 和 $P_2$ 都减去 $P_1$ , 这样， $P_1$ 一定是 $0$ 了，我们看到 $P_2$ , 如果 $P_2 < 0$,那就没有办法将$P_1$ 和 $P_2$ 同时变为 $0$ 了。就直接输出 ```NO``` 就行了。以后的也按这种方案就可以了。最后就是是 $0$ 就是 ```Yes``` ,不是的话，就输出 ```NO``` 。

总结一下。

1. $P_1$ 和 $P_2$ 都减 $P_1$ 。
2. 判断是否满足要求。

#include<bits/stdc++.h>//万能头文件，懒人专属。
using namespace std;

int main() 
{
	int t;
	cin>>t; //输入有几组样例。
	while(t--)//每一组数据，用t--,就是循环t次。
    	{
		int n,p[10003];
		cin>>n;
		for(int i=0;i<n;++i)
        	{
			cin>>p[i];//输入每一组中的数据。
		}
		for(int i=0;i<n-1;++i)//正式代码开始，将P_1 和 P_2 都减 P_1。
        	{
			if(p[i]>0)
            		{
				if(p[i]==0) //是零就break。
                		{
					p[n-1]=1;
					break;
				}
                		else//不是就减掉。
                        	{
					p[i+1]-=p[i];
				}
			}
		}
		if(p[n-1]>0) //判断是否满足要求。
        	{
			cout<<"NO"<<'\n';
		}
        	else
            	{
			cout<<"YES"<<'\n';
		}
	}
	return 0;
}
