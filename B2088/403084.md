这道题可以先分别求出每种图书需要花费的价钱，最后再输出总价就可以了。

下面放代码：
```C
#include<bits/stdc++.h>
int main(){
  int a[10] ;
  double b,c,d,e,f,g,h,z,j,l;
  for(int i=0; i<10; i++){
  scanf("%d", &a[i]);
}
  b = a[0]*28.9;//计算概论的价钱
  c = a[1]*32.7;//数据结构的价钱
  d = a[2]*45.6;//数字逻辑的价钱
  e = a[3]*78;//C++程序设计教程的价钱
  f = a[4]*35;//人工智能的价钱
  g = a[5]*86.2;//计算机体系结构的价钱
  h = a[6]*27.8;//编译原理的价钱
  z = a[7]*43;//操作系统的价钱
  j = a[8]*56;//计算机网络的价钱
  l = a[9]*65;//JAVA程序设计的价钱
  printf("%.1lf", b+c+d+e+f+g+h+z+j+l);//输出总价
return 0;
}
```
