# 【题意】
### 给出5个数值（分别代表硝酸、王水、粗汞、铅、矾的量），输出每个量除以每个量需要的用量，输出经过计算后中数组的最小值
# 【解决方案】
### 使用语言：Python3
### 解决方案：直接计算，从2开始，因为前面都是1，所以可以略去。
### 代码：
```python
s = input().split()
tot = []
s[2] = str(int(s[2])//2)
s[3] = str(int(s[3])//7)
s[4] = str(int(s[4])//4)
s = map(int,s)
print(min(s))

```
# 【坑点分析】
### 1.使用符号问题：与C++不同，Python的/是代表直接除，即含有小数位，所以我们需要使用的是 // ，这个符号的功能才是与 C++ 的/相对应
### 2.数组类型问题：在默认读入中（即input()）后跟个split()是把input()的字符串读入转成字符串数组，而在最后一行我们用的min函数就因不能识别string类型而会输出错误的答案，所以我们需要用到map函数（详见【函数讲解】）
# 【函数讲解】
### 1.map
### map函数是个可以将一个数组执行相应的命令。
### 它的使用方法是
```python
map(func,*iterbale)
```
### 其中，第一个参量 func 是代表要执行的命令，第二个参量iterable是代表可遍历的类型（如string或list）
### 使用方法：
```python
my_list = ['1','2','3']
map(int , my_list)
print(min(my_list))
```
### 输出：
### <map object at XXXXXX
### >
### 1
### 第一个输出代表map类型所在的位置
### 第二个输出代表在map映射中的最小值
