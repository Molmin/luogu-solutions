下面来看**思路**

有一道题叫[小鱼的游泳时间](https://www.luogu.org/problemnew/show/P1425)

那道题给的是从a时b分到c时d分

这道题和那道题很相似

具体思路：

他给你开始的月和日，你就算出来那天是第几天

比如给你1月9日，你就算出第九天

同理，算出结束是第几天

比如2月2日，是第33天

然后，两个时间相减，一输出，大功告成！

### 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int a[101]={0,31,29,31,30,31,30,31,31,30,31,30,31};
  //每月有几天，注意2012年是闰年，2月29天，数组从一开始用，所以得敲进去一个0
int d1,d2,m1,m2,i,a1,a2;
int main()
{
    cin>>m1>>d1>>m2>>d2;
  //读入
    for(i=1;i<=m1-1;i++)a1+=a[i];
    a1+=d1;
    //详细解释一下：这里a1是开始的第几天，这个循环就是在求他。比如8月22日，1到7月都是整的过完了的，用循环，循环1到7月，把1到7月的天数往里加（所以a1一定要事先清0），然后还有8月份的22天，最后再加上22，就行了                    
    for(i=1;i<=m2-1;i++)a2+=a[i];
    a2+=d2;
  //同理
    cout<<a1-a2<<endl;
  //注意换行，等着AC！
    return 0;
}
```

### 题目中并没有说，前面的日期更靠前一些，所以应该输出减完后的绝对值才对!

真正的**AC**代码！

```
#include <bits/stdc++.h>
using namespace std;
int a[101]={0,31,29,31,30,31,30,31,31,30,31,30,31};
int d1,d2,m1,m2,i,a1,a2;
int main()
{
    cin>>m1>>d1>>m2>>d2;
    for(i=1;i<=m1-1;i++)a1+=a[i];
    a1+=d1;
    for(i=1;i<=m2-1;i++)a2+=a[i];
    a2+=d2;
    cout<<max(a1-a2,a2-a1)<<endl;
    return 0;
}
```
输出绝对值，就对了。也可以把第12行的语句改成
```
cout<<max(a1-a2,a2-a1)<<endl;
```
因为a1-a2和a2-a1是**相反数**，所以必然一正一负，且距离原点的距离相等，所以max一下，肯定正数大呀，就把正数输出出来了
