题意：给出2012年的两个日期（月、日），输出两个日期间隔了几天

###  **注意：2012年是闰年！！！2月有29天**

可以想到一个办法，用前缀和来统计m2(后一个月份)到m1(前一个月份)有多少天。

如果用f[i]来表示前i个数数的和，第x个数到第y个数的和就是f[y]-f[x-1]

这里不计算第m1月，就是f[m2]-f[m1]

这样时间复杂度就是O(1)+制作f前缀和的时间复杂度O(12)

再加上后一个日到前一个日有多少天，就直接用后一个日减前一个日

如果后一个日比第前一个日小呢? 

比如是

 3 31
 
 5  1
 
月是f[5]-f[3]，即61天

日是1-31，即-30天

用61+(-30)正好抵消掉一部分，得到的31就是两个日期隔的天数

上代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[13]={0,0,31,29,31,30,31,30,31,31,30,31,30};  //每个月的天数
int f[13];  //前缀和
int main()
{
	ios::sync_with_stdio(false);
	for(register int i=1;i<=12;i++)   //前缀和
		f[i]=f[i-1]+a[i];
	int m1,d1,m2,d2,s;
	cin>>m1>>d1;    //输入
	cin>>m2>>d2;
	s=(f[m2]-f[m1])+(d2-d1);
        //(f[m2]-f[m1])代表m2到m1的天数
        //((d2-d1)代表d2到d1的天数
	cout<<s<<endl;  //输出
	return 0;
}
```
