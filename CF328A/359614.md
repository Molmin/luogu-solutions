# [题目传送门](https://www.luogu.com.cn/problem/CF328A)
### 第一步，咱得知道等差数列和等比数列是神马东东
### 其他的都不重要，你只需要知道等差数列的每两项之差相同，等比数列的每两项之商相同即可
### 知道这些东东之后就可以上代码了（其他注意事项见注释）：

```cpp
#include<bits/stdc++.h>//懒人专用万能头
using namespace std;//标准名字空间
int a,b,c,d;//题目输入的四个数
int main(){
	cin>>a>>b>>c>>d;//输入
	if(a-b==b-c&&b-c==c-d)//判断等差数列
	cout<<d+(d-c);//非常简单的输出，只要加上公差即可
	else if(a*c==b*b&&b*d==c*c)//非常明显可以看出来如果 ac=b^2 ，那么 a,b,c就是一个等比数列
	if(c/d==0){
		if(d*(1.0*d/c)>=1&&d*(1.0*d/c)-floor(d*(1.0*d/c))==0)//判断循环小数和0点几的数
		cout<<d*(1.0*d/c);//公比也可能是小数哦~~~
		else
		cout<<42;//别忘输出42
	}
	else{
		if(d*(1.0*d/c)>=1&&d*(1.0*d/c)-floor(d*(1.0*d/c))==0)//同样，判断循环小数和0点几的数
		cout<<d/(1.0*c/d);//公比也可能是小数哦~~~
		else
		cout<<42;
	}
	else cout<<42;//如果既不是等差也不是等比
	return 0;
}

```
