###  ****2018noip唯一自己想出来的题~~~****

这道题我想了一个多小时（原谅我是蒟蒻~），然后终于想出用不上升序列来做（时间复杂度为O（n））。~~然而后面的题都不会做了。~~

其实这道题的思路很巧妙。ans先赋值a[1]（初始化），因为对于第一组数据需要a[1]次。然后用不上升序列的思路，一直往下循环，直到后一个数比前一个数大，加上后一个数，减去前一个数（原因只可意会，不可言传，~~因为我都不知道自己是怎么想出来的~~），就得到最终的答案。

不说了，上代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,ans,a[100005];
int main()
{
	scanf("%d",&n);
	for(int i=1;i<=n;i++)
	scanf("%d",&a[i]);
	ans=a[1];
	for(int i=1;i<=n;i++)
	{
		if(a[i+1]>a[i])
		{
			ans+=a[i+1];
			ans-=a[i];
		}
	}
	printf("%d",ans);
	return 0;
}//像我这样的人估计也就坐得起水题了吧！
```
