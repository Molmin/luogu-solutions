$$
\sum_{i=1}^n\lfloor {n\over i}\rfloor=2\sum_{i=1}^{\lfloor\sqrt n\rfloor}\lfloor {n\over i}\rfloor-\lfloor\sqrt n\rfloor^2
$$
```cpp
int f(int n){
	int m=sqrt(n),ans=0;
	for(int i=1;i<=m;i++)ans+=n/i;
	return ans*2-m*m;
}
```
