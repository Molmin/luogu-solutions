我们并不需要精确算出 $ n! $ 的值，每次运算时，我们只需根据最右边的几个非零位进行运算即可，对于高位的数字，因为对答案没有贡献，我们可以忽略。

```cpp
#include <stdio.h>
int a[10005];
int main()
{
 int x;
 a[0]=1;
 for(int i=1;i<=10000;i++)
 {
  a[i]=a[i-1]*i;
  while(a[i]%10==0)a[i]/=10;//为0的位已经对运算没有帮助
  a[i]=a[i]%100000;//最前面的位也对于结果没有贡献
 }
 while(~scanf("%d",&x))
  printf("%5d -> %d\n",x,a[x]%10);
 return 0;
}
```
