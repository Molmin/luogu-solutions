**第五题解析：**

我们可以枚举每一个二极管是否亮着，从一个亮着的二极管开始一次遍历。判断一下即可。

**代码：**

```cpp
#include<bits/stdc++.h>
using namespace std;
string ans[]={ // 存储结果
    "624",
    "2481215",
    "761",
    "8879",
    "80"
};
char T;
int main() {
    cin>>T;
    cout<<ans[T-'A'];
    return 0;
}

```
