**很水**的一道**蓝题**，简单分析一下就可以解决  

先手放在**正中心**，之后放在与后手位置**中心对称**的位置，这样就不会**在后手之前**悬空

但是如果**第一次**就悬空呢 ? ( 即盘子比桌子大 )  
所以要加**特判**！！！

 _**code**~_ 
 ```cpp
#include<bits/stdc++.h>
using namespace std;
int a/*长*/,b/*宽*/,c/*半径*/;
int main()
{
    cin>>a>>b>>c;
    c*=2;//半径*2=直径 
    if(a>=c&&b>=c) cout<<"First"<<endl;//盘子比桌子小
    else cout<<"Second"<<endl;//盘子比桌子大
    return 0;
}
```
