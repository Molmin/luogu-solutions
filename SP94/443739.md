本题光看题面或许会有点糊涂，在这里给大家详细解析一下这道题。

### 题意：
简单来说就是输入 $n$，知道 $n$ 为0结束，而 $n$ 代表的是玛雅数字的行数，每一行就代表一个数字，注意 `.` 代表1，`-` 代表5，`S`一行只有一个代表0，这里举个例子：

|`S`| 0 |
| -----------: | -----------: |
|`.. ---`| 17 |
|`....`| 4 |

接下来就是进制，玛雅人与我们不一样，我们的进制是十进制，而他们的进制……也不能说算是进制吧，大家看一下下面这张表。

| 位 | 个 |  | 十 |  | 百 |  | 千 |  | 万 |
| -----------: | -----------: | -----------: | -----------: | -----------: | -----------: | -----------: | -----------: | -----------: | -----------: |
| 数 | 1 |  | 3 |  | 5 |  | 9 |  | 11 |
| 进制 | 1 |  | 20 |  | 360 |  | 7200 |  | 144000 |
| 进阶 |  |$\times$20| |$\times$18 | |$\times$20 | |$\times$20 |

可见除了百位外，其他位进阶全都是 $\times$20，而题目就是要我们求这个数。

我以上面这个表为例。

则此数为 $1\times1+3\times20+5\times360+9\times7200+11\times144000$

### 思路：
此时的思路就很简单了，就像上面我举的例子一样，将进制算出来，然后求得每个位上的数，将其一一相乘得到的结果再一一相加，便是此题目的答案了。

### Code：
```cpp
#include<bits/stdc++.h>
using namespace std;
long long a[20]{1,20,360};
int main(){
	int i,j; 
	for(i=3;i<=8;i++)//算进制
		a[i]=a[i-1]*20;
	while(1){
		int n;
		long long summ=0;
		scanf("%d\n",&n);
		if(!n) break;
		for(i=n;i>0;i--){
			char s[20000];
			gets(s);
			int sum=0;
			int len=strlen(s);
			for(j=0;j<len;j++){//求数
				if(s[j]=='.') sum++;
				if(s[j]=='-') sum+=5;
			}
			summ+=sum*a[i-1];
		}
		printf("%lld\n",summ);
	}
	return 0;
}
```
