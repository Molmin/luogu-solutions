# **正文：**
## 题意：
本题只要明白了思路代码非常简单，只需要一个循环即可搞定。首先，这个序列的最大公约数一定 $≤m \div n$ ，也就是这个序列的平均数，所以我们就可以从这里开始循环，一直循环到 $1$ 如果发现一个可以被 $m$ 整除的数，就代表可以将 $m$ 分成 $n$ 份，每份都可以整除这个数。本题答案最小为 $1$ ，不可能有不输出的情况。
## 简单代码：
```cpp
#include<iostream>
using namespace std;
int main()
{
    int n,m;
    cin>>n>>m;
    for(int i=m/n;i>=1;i--){
    	if(m%i==0){
    		cout<<i;
    		return 0;
		}
	}
}
```
