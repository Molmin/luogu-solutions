### 这道题主要考察字符串的运用。直接模拟即可 AC。

### 介绍一个函数：split()。

它可以将字符串分割并转化为列表。

举个栗子：

```python
a = "1 2 3"
a.split()

这时，变量 a 就变成了 ["1","2","3"]
```

#### 不仅如此，split 函数中还可以填内容。

再举个栗子：

```
a = "1s2s3s4s5"
a.split("s")

这时，变量 a 就变成了 ["1","2","3","4","5"]
```

#### 然后，这题就很简单了。

#### Python3 AC 代码：

```python
a = input()
s = input().split()  # 输入，再将字符串分割成列表。
d = ''.join(s)      # 拼接。
print(int(d) % 1000000007)  # 用 int 函数转化为数字类型，对 1e9+7 取余并输出。
```
