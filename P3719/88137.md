## 分析：
运用递推：

看到这道题就想到了表达式求值的问题。

表达式求值问题有两种算法：栈，或递推。

就这道题而言，递推的代码要好写一些。

具体操作如下：

- 读到了后括号就返回当前统计的值

- 读到了a就累加和+1

- 读到了前括号就加上新的统计 

-  读到|就取最大值

代码如下：

```cpp
int work(int s){//简单递推 
    char c;
    while(scanf("%c",&c)==1){//一边读一边递推 
        if(c==')')return s;//搜到后括号就返回 
        if(c=='a')s++;//搜到a的话统计和加一 
        if(c=='(')s+=work(0);//加上新的统计 
        if(c=='|'){s=max(s,work(0));break;}//重新统计答案 
    }
    return s;
}
```

在主函数中：

```cpp
int main(){
	cout<<work(0);//初始化得答案为0 
    return 0;
}
```
