## 题意简析

在蜂巢上建立一个平面坐标系，定义右下方向为 $x$ 轴正方向，下方向为 $y$ 轴正方向，一个动点从原点格开始向下做顺时针螺旋向外移动，问他经过的第 $n$ 格的坐标。

根据图示简单模拟并统计每个方向的移动次数，从向下开始到向右结束为一个周期，可以发现第 $t$ 周期的路线一定是：

1. 向下移 $t$ 格；
2. 向左下移 $t-1$ 格；
3. 向左移 $t$ 格；
4. 向上移 $t$ 格；
5. 向右上移 $t$ 格；
6. 向右移 $t$ 格。

根据这个规律预处理出所有答案即可。

## 代码/解释
```cpp
#include<bits/stdc++.h>
using namespace std;
int x[100835],y[100835],a,b;
int main()
{
	int t,n,i,j;
	for(t=1,i=2;t<=183;t++)
	{
		for(j=1;j<=t;j++,i++)
		x[i]=a,y[i]=++b;
		for(j=1;j<t;j++,i++)
		x[i]=--a,y[i]=++b;
		for(j=1;j<=t;j++,i++)
		x[i]=--a,y[i]=b;
		for(j=1;j<=t;j++,i++)
		x[i]=a,y[i]=--b;
		for(j=1;j<=t;j++,i++)
		x[i]=++a,y[i]=--b;
		for(j=1;j<=t;j++,i++)
		x[i]=++a,y[i]=b;
	}
	while(cin>>n)
	printf("%d %d\n",x[n],y[n]);
	return 0;
}
```
