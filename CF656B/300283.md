## 题意：

从 $1$ 到 $N$，读入两个数 $M_i$ 和 $R_i$，
设 $D$ 为即将到来的连续整数天数（包括当前天数，且当前天数为 $0$），如果整数 $i$ 满足 $D \bmod M_i=R_i$，则你洗盘子，反之，你的室友洗盘子。计算并输出你洗盘子的可能性。（假设之后有无数天）


------------
## 解题思路

~~暴力枚举~~

可以发现，只要不断增加天数 $D$ 然后模拟一遍，一定可以求出正确答案。



为了防止我这种垃圾看不懂，写出了以下思路：

```
i 从 0 枚举到 D ——> D mod M[i] 是否等于 R[i]？ ——> 等于，更新计数     
```


------------

那么 $D$ 应该设定为多少呢？显然，它应该是 $N$ 的**倍数**。

试了一下，开 $300000$ 也是能过的。（


------------
## 代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,s,a[50],b[50];
double ans;
int main()
{
	scanf("%d",&n);//输入
	for(int i=1;i<=n;++i) scanf("%d",&a[i]);
	for(int i=1;i<=n;++i) scanf("%d",&b[i]);
	for(int i=1;i<=2000000;++i)
	for(int j=1;j<=n;++j)
	if(i%a[j]==b[j]){s++;break;}//枚举判断
	ans=s/2000000.0;//求几率
	printf("%.6lf",ans);//输出记得取6位小数
	return 0;
}
```
