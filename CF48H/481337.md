# 题目大意
根据传说，柏林国王柏林一世以其对美丽和秩序的热爱而闻名。有一天，他下令用黑白瓷砖按照他发明的规则几何图案在宫殿大厅的地板上贴上瓷砖，以前在那里举行舞会和招待会。然而，就像之后的情况一样，由于融资低，只有 $a$ 黑和 $b$ 白瓷砖交付给皇宫。其他的 $c$ 瓷砖是黑白的(见图)。
![](https://cdn.luogu.com.cn/upload/vjudge_pic/CF48H/9c65623eb9b5bf5301cdf29083a6378da5565d98.png)
最初计划失败了！得知此事后，国王下达了一个新命令:用可用的瓷砖铺地板，这样瓷砖的黑色面就不会碰到白色面。瓷砖是 $1\times1$ 大小的正方形，每个黑白瓷砖都可以用四种方式之一旋转。

法院的程序员被赋予了制定铺砖计划的任务，他完成了任务，没有遭受不服从的后果。你能应付吗？

# 题目思路
首先输入判断这一块到底是全黑:```####```，还是全白```....```，还是半黑半白：
```
\#
.\

./
/#

\.
#\

#/
/.
```
题目说输出从左至右，如果碰到 $a$，直接输出，输出再加上个```<```，中间打完乱码，然后```>```输出。

如果碰到的是 $b$，那么就有点麻烦了：先得判断要输出几个```....```，可以判断长和宽的比是不是长多宽少，否则就输出 $2$ 个。

碰到 $c$，把乱码打一遍就完了。

再次提醒输入：

第一行：两个数，表示地面的长和宽。

第二行：三个数，表示a,b,c三种瓷砖的个数。

### 温益提示 
题目最后一行要加换行符。
