这个题和P1150有异曲同工之妙……

因为燃尽的蜡烛可以拼成新的蜡烛，同样拼成的新的蜡烛燃尽后也可以继续拼成新的蜡烛……

例如：10根蜡烛，每2根燃尽的蜡烛可以拼成一根新的蜡烛

第一轮：10根蜡烛燃尽，燃烧时间总共10小时

第二轮：拼成5根新蜡烛，燃烧时间总共15小时

第三轮：拼成2根新蜡烛，**剩下1根多余已燃尽蜡烛**，燃烧时间总共17小时

第四轮：拼成1根新蜡烛，**剩下1根多余已燃尽蜡烛**，燃烧时间总共18小时

## **第五轮注意！**

第五轮：拼成1根新蜡烛（第四轮燃尽的1根蜡烛加上剩下的1根已燃尽蜡烛）燃烧时间总共19小时

所以输出 19 。

```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int n,k,sum,a;
int main()
{
	cin>>n>>k;
	sum=n;
	while(n>=k)
	{
		sum+=n/k;
		a=n/k;
		n=n%k;
		n+=a;
	}
	cout<<sum;
	return 0;
}
```
