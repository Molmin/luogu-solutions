**这其实就是空瓶换橙汁，烟蒂换烟头的问题**

**首先抽光烟，留下a个烟蒂**

**再把a个烟蒂换成a/b根烟，剩下a%b个烟蒂**

**把a/b+a%b换成烟，以此类推**

**当烟数<b时结束**

**举例子10,3--3,1--1,1,,,抽了10+3+1=14根**

上代码
```c
#include<bits/stdc++.h>
using namespace std;
int a,b,ans;//a根，b根换一根，答案
int main()
{
	cin>>a>>b;//输入
	ans+=a;//先用光并加上数量
	while(a>=b)//a>=b说明还能换成蜡烛
	{
		ans+=a/b;//换了a/b根蜡烛并加入总数
		a=a%b+a/b;//换了a/b根剩下a%b根，总共就有a=a%b+a/b根
	}
	cout<<ans;//输出
}
```
