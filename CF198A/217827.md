下面是重点： 

既然这样，10^6的大小完全足够。只要求出第一次的，达到第二次的细菌初始量要几秒，再用原先的天数减就可以了。

接下来就不用多说了，已经非常简单了。直接上代码。
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int k,b,n,t;
	long long z=1,z2,ans=0;//ans记录几秒
	cin>>k>>b>>n>>t;
	while(z<=t && ans<=n) z=k*z+b,ans++;//主要程序，当1变到t要几秒 
	cout<<n-ans+1;//别忘记加一 
	return 0;
}
```
