这道题首先准备好4个（输入用3个，另一个表示爬的高度，清零

然后开一个特判，如果下降的长度比上升的长度大，且第一天无法爬上去，就输出-1

例如

```
if(上升<下降&&上升<井深)
{
	printf("-1");
    return 0;//一定要结束，否则程序无法结束了
}
```

然后就可以开一个循环
```
for(int i=1;1;i++)//一直循环下去，直到乌龟能出去
```
这个循环中的i表示的就是天数

那么如果i%2的值为1，就是奇数天，向上爬（爬的高度加上向上爬的高度），否则减去向下落的高度

在每次循环的最后都要写上如果爬的度大于井深就输出i（也就是天数，然后return 0;结束循环
