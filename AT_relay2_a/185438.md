这道题又是小学奥数，所以，这道题的公式只要念过小学的人应该都会找的吧

这道题大概就分为三点

1.如果第一天就上去了，那么他一生快快乐乐，直接输出1

2.如果滑下来得比爬上去的更多，那就与世界再见吧，输出-1

3.每一次都可以向上爬一点点，那就只好计算后输出了

其他的，就按公式推导吧

最最重要的一点：**岛国题要换行**

直接来代码吧

```
#include<bits/stdc++.h>
using namespace std;
long long a,b,c,n;//由于数字可能很大，要开 long long 
int main()
{
    cin>>a>>b>>c;
    if(b>=a)
    {
        cout<<1<<endl;//第一天就爬上去了 
        exit(0);//快乐一生，可以停止了 
    }
    if(c>=b)
    {
        cout<<-1<<endl;//永远上不去了 
        return 0;//和外面的世界再见吧
        //注意：一定要在这里打return 0；，不然程序结束不了
    }
    //没办法，还是要一系列计算其他结果 
    if((a-b)%(b-c)>=1) 
		n=1;//计算前判断一个特殊情况 
    cout<<((a-b)/(b-c)+n)*2+1<<endl;//其他的就只好按公式算出答案，输出
    /*这里的计算其实也不难
    首先，我们在之前已经排除了几个特殊情况
    所以，在这里，就只好一步一步计算其他的情况了
    如果井的长度可以整除两天之后爬上去的数，那么就除一下就好了
    如果不能整除，那么最后加上1
    特别注意：要先将最后一天爬上去的长度给减去
    因为最后只需要一天就出来了，不用再等掉下去的另一天了
    而且，记住，一定要*2
    因为他是分奇数偶数来算的，一定要*2
    就算你可以整除，那么也不可以不加上1
    因为你已经预留了一天的量而没有将那一天计入总天数
    */
    return 0;//结束程序 
}
```
