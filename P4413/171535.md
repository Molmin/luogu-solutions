 ### **P4413 [COCI2006-2007#2] R2**
用已知量将未知量表示出来
------------
正常思路一步步向下列就好，不需要循环判断

上代码，~~喵喵喵~~————



------------

#include<iostream>
  
using namespace std;

int main()

{

    int S,R1;//定义已知量
    cin>>R1>>S;//输入
    int R2=2*S-R1;//已知量归到一边，未知的R2归到一边
    cout<<R2<<endl;//输出
    return 0;
}
