### 这是一道模拟题

### 思路：打表

定义一个字符串$c$，然后把$c$的表打出来，然后输出对应项即可。

**Q:如何打表？**

A:每次把一个数拆分，记入$a$数组，然后每次把a数组里的数导到$c$里。

**Q:如何把整形数导到字符串$c$里？**

A：需要一个指针变量$tmp$，记录c实时的长度。每枚举一个整数，要记录一个数的长度，$a$数组记录这个数的每一位，然后通过ASCII码与int差48的特性把每个数导到$c$里。

用文字实在是说不清，还是要搁一个代码：

```cpp
#include<iostream> 
#include<cstdio>//头文件
using namespace std;
char c[100010];//一个表
int k;//要输入的k
int a[110];//记录数的拆分情况（经常变）
int weishu(int x)//这是计算位数+a数组操作的函数
{
	int cnt=0;//位数变量
	while(x)//数不为0的时候
	{
		cnt++;//位数增加
		a[cnt]=x%10;//a数组记录
		x/=10;消除记录的已被记录的末尾
	}
	return cnt;//返回位数
}
int main(){
	scanf("%d",&k);//输入一个k
	int tmp=1;定义tmp数组，因为第一项是c[1],所以初始值为1
	for(int i=1;i<=10000;i++)//这个题数据太水，打到10000肯定不超时，也保险毒瘤数据
		for(int j=weishu(i);j>=1;j--)注意：这个数在a数组里是倒着存储的，所以一定要倒着循环
			c[tmp]=char(a[j]+48),tmp++;//导入c且指针变量++
	cout<<c[k]<<endl;//表打完后，输出对应位就行了
	return 0;//结束程序
}
```
