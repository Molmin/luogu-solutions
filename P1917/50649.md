我用大号分发了一个题解，又不会修改，忽然发现不完整，赶紧用小号补充一下：

上面说到，如果一个棋盘为：

```cpp
- K -
K O K
- K -
```
K和符号的含义见 @引领天下 的题解

我来继续分析为什么uim下在K处就输了：

取一种情况：

```cpp
- X -
- O -
- - -
```
会演变为：

```cpp
- X O
- O -
- - -
```
uim会这样堵：

```cpp
- X O
- O -
X - -
```
这时，小a只需在这里下：

```cpp
- X O
- O -
X - O
```
很明显，无论uim怎么下，小a都赢了。

接下来几种情况，就像刚才的旋转，其结果都一样。

故我的方法正确。
