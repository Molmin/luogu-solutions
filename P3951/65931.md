```cpp
#include <stdio.h>
int main()
{
    long long a,m,n;                    /*因为数据较大（最高达10^9）应用long long*/
    scanf("%lld %lld",&m,&n);
    a=(m-1)*n-m;                       /*此式由不完全归纳法求得，后附求式过程*/
    printf("%lld",a);
    return 0;
}
```
----------------------------------
求式过程

设其中一个数为2

2、3=>1

2、5=>3

2、7=>5

2、11=>9

得2、n=>n-2

设其中一个数为3

3、5=>7

3、7=>11

3、11=>19

3、13=>23

3、n=>2n-3

设其中一个数为5

5、7=>23

5、11=>39

5、13=>47

5、17=>63

5、n=>4n-5

由此得

m、n=>(m-1)n-m

PS.算数方法：

已知所求在两数据最大公因数和最小公倍数之间

从两数之积开始，向前逐个代数，直到无法求得，即合题
