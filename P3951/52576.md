证明如下

由于式子ax + by = gcd（a，b）又由于a，b互质，所以这个式子显然可以化为by = gcd（a，b）- ax，等式两边同时乘上（1~a-1）也就是
（1 ~ a-1）* by = （1~a-1）* gcd（a，b） - a*（1~a-1）x
这个式子显然还可以再化，变成
（1 ~ a-1）* by = （1~a-1）* gcd（a，b）（mod a）
由于AB互质，gcd（a，b） = 1
显然我们证明了对于nb % a有着a-1个剩余系
又由于0 % a = 0，ab % a = 0
又由于是从1~a-1枚举，这种情况下、余数具有周期性，所以我们证明了1~a-1 * b有着a-1个不同的余数
而且由于这将会是在每种剩余系中最小的数，为什么，因为nb + ma与nb对a取余的余数相同，所以1~a-1*b是每个剩余系中最小的数了，那么这些数中最大的数是哪个呢？当然是（a-1） * b而最大的不能表示的数显然就会是这个数-a咯因为其他的剩余系显然是会被覆盖完全的，不存在比a-1 * b小而且属于不同剩余系还表示不出来的，可以下去好好理解一下这句话
