这题其实就是一个不定方程

ax + by = n, 其中x, y 为非负整数~~支付宝是个好东西~~

我们求的，就是使得该不定方程没有满足条件的解的n，且对于任何大于n 的 整数皆可求出满足要求的解。

### 引理： 若（a, b) = 1， 且不定方程ax + by = c有整数解（x', y'), 则该不定方程的所有整数解为
x = x' + bt, y = y' - at, t为整数

证：~~议政~~设（x, y) 是不定方程的一组整数解， 结合（x', y')为该方程的解， 可知

ax + by = c;
ax' + by' = c;
a(x - x') = b(y' - y)
所以b | a(x - x')

所以b | x - x', 设 x - x' = bt, 则 y - y' = -at, ~~怔弊~~ 证毕

### 证明 ax + by = ab - a - b 无非负整数解， 其中a, b 互质：

假设ax' + bx' = ab - a - b, 则 a(x' + 1) + b(y' + 1) = ab

容易得到a | b(y' + 1), a | y' + 1, a <= y' + 1

#### 同理可证 

b <= x' + 1, 那么左 >= 2ab = ab, 矛盾！

### 而当方程右边大于或等于ab - a - b + 1

易证可以得到一个x' 大于-1且小于b

此时by' = c - ax' > ab - a - b - ax' >= ab - a - b - a(b - 1) = -b;

所以y' > -1

所以y' >= 0
