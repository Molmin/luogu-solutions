感觉这道题比较简单吧，因为数据比较小（4 ≤ n ≤ 100），完全可以用循环直接模拟。

但是代码中会有一些判断语句，我们在 **C++** 使用 “ **？**”来替代 “ **if** ”语句，可以有效地缩短代码。

具体的用法： “  **表达式** ？ **A** : **B** ;  ”

当程序运行到这条语句时，它会检测表达式的值。如果是真，则运行 **A** 语句；如果是假，则运行 **B** 语句。

但是 A 和 B 语句都不可以为空（会编译错误）, 你可以在哪里写一个变量或常量。如 “ **b?b:b=n;** ”  ，程序在运行这条语句时，如果 b 不等于 0 ，则不会执行任何操作。

------------

这是 **C++** 的代码：
```cpp
#include<cstdio>
int a,x,b,y,n;bool tf;
int main()
{
    scanf("%d%d%d%d%d",&n,&a,&x,&b,&y);//把变量 n、a、x、b、y 读进来。
    while(a!=x&&b!=y)//当 a 不等于 x 且 b 不等于 y 时，循环继续运行。
    {
        a=a%n+1,b--;
        b?b:b=n;
        a-b?tf:tf=1;
    }
    tf?printf("YES"):printf("NO");
    return 0;//Don't forget this.
}
```
------------

亲测：**1.09s / 0B**
