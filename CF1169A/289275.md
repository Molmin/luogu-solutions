# Circle Metro ~~（圆形地铁）~~

这不就暗示着这两条地铁路线是两个圆吗！

题目大意：有两条总站数为 $n$ 的地铁：

其中第一条是以 $1,2,3,...,n-1,n,1,2...$ 的顺序行驶；

另外一条是以 $n,n-1,n-2,...,2,1,n,n-1$ 的顺序行驶。

这时候有两个人：第一个人从第一条线路编号为 $a$ 的站开始，到了编号为 $x$ 的站就下车；二个人从第二条线路编号为  $b$ 的站开始，到了编号为 $y$ 的站就下车。所以依照题意模拟就好了啊！

### 但是！！！

题目中有一句说：
## 请注意，这包括丹尼尔或弗拉德进入或离开地铁的时刻。

也就是说，在循环里的判断一定要先判断是否相遇再判断是否下车！！~~（我就在这里卡了一次）~~

于是代码就出来啦！

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n,a,x,b,y;
    cin>>n>>a>>x>>b>>y;
    while(1){//懒得写别的了于是写了个while(1)
        if(a!=n) a++;//循环模拟第一个人到达的站台编号
        else a=1;
        if(b!=1) b--;//循环模拟第二个人到达的站台编号
        else b=n;
        if(a==b){//先判断是否相遇！！！
            cout<<"Yes"<<endl;
            break;//若是不break有好看的
        }
        if(a==x){//第一个人到站了
            cout<<"No"<<endl;//下车了
            break;
        }
        if(b==y){//同理
            cout<<"No"<<endl;
            break;
        }
    }
    return 0;
}
```
