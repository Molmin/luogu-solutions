# 分析
这题思维难度不大，只是需要分类讨论：

- 当 $X$ 和 $Y$ 都等于 $0$ 时，我们只需要构造一组相等的数即可。

- 当 $X$ 和 $Y$ 相等且都不为 $0$ 时，因为一个数除以一个与其相等的数，余数必为 $0$，所以一定无解。

- 当 $X>Y$ 且 $Y=0$ 时，说明 $B>A$ 且 $B$ 是 $A$ 的倍数，所以只需要输出 $X$ 和 $X$ 的任意倍数即可。**注意：不能输出 $X$ 的 $1$ 倍。**

- 当 $X>Y$ 但 $Y$ 不等于 $0$ 时，说明 $B>A$ 且 $B$ 等于 $X$ 的任意倍数加上 $Y$，所以只需要输出 $X$ 和 $X$ 的任意倍数加上 $Y$ 即可。

- 当 $X<Y$ 且 $X=0$ 时，情况与 $X>Y$ 时大致相同，最后输出 $Y$ 的任意倍数 和 $Y$ 即可。

- 当 $X<Y$ 但 $X$ 不等于 $0$ 时，情况与 $X>Y$ 时大致相同，最后输出 $Y$ 的任意倍数加上 $X$ 和 $Y$ 即可。

# 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
    long long x,y;
    cin>>x>>y;
    if(x==0&&y==0)cout<<"2 2"<<endl;//余数为0时，构造一组相等的数即可
    else if(x==y)cout<<"-1"<<endl;//其余x和y相等的情况，无解
    else if(x>y)//x大于y
    {
        if(y==0)cout<<x<<" "<<x+x<<endl;
        else cout<<x<<" "<<x+y<<endl;
    }
    else//x小于y
    {
        if(x==0)cout<<y+y<<" "<<y<<endl;
        else cout<<x+y<<" "<<y<<endl;
    }
    return 0;
}
```
