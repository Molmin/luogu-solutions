# 打表找规律，暴力出奇迹！！！
先把表打出来再说：
```cpp
/*
   n=1   2   3   4   5   6
step=2   6   14  30  62  126

```
发现规律了吗？如果用A_n 表示2n个盘子移动所需要的 最少步数，用递推关系表示A_n则是
### A_n=2*A_(n-1)+2
这个式子虽然不好证明，但自己去试着模拟一下就会恍然大悟，也就不再赘述。（~~懒得写~~）
### 根据递推式写出代码：
```python
n=eval(input())
sum=0
for i in range(1,n+1):
  sum=sum*2+2

print(sum)
```

可能有很多眼尖的人发现，A_n=2^(n+1)-2。证明也很简单，如下：
```
A_0=0,A_1=2
A_n=2*A_(n-1)+2
设B_n=A_n+2
则B_1=4
B_n=2*A_(n-1)+4=2*(A_(n-1)+2)=2*B_(n-1)
  =2^(n+1)
又因为 B_n=A_n+2
所以 A_n=2^(n+1)-2
```
代码：
```python
n=eval(input())
def pow(u):  #计算2^n
  m=1;
  for i in range(1,u+1):
    m*=2
  return m

print(pow(n+1)-2) 
```
