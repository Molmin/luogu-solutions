作为标签党我首先选择看标签，一看到高精这东西顿感不妙，数据肯定很大，不过不需担心，就用高精的算法算就行了。

首先先看公式就是啊a[i]=a[i-1]*2+1(i>1)完全没有什么可讲的，数学好的同学都可以自己推一下。
注：这是单塔的公式，双塔只需最后的结果乘2即可。

若只用此公式可望50分。

现在开始讲高精的做法了：

直接上代码不多讲了

```
#include<iostream>
#include<cstdio>
using namespace std;
int a[1000],ans;
int main()
{
	int n;
	cin>>n;
	a[1]=1;
	n=n-1;//只需运行n-1次
	while(n--){
	for(int i=1;i<=100;i++)
	{
		a[i]=a[i]*2;
	}//高精，每一位乘2
	for(int i=1;i<=100;i++)
	if(a[i]>=10)a[i+1]+=1,a[i]-=10;
		a[1]+=1;
		if(a[1]>=10)a[2]+=1,a[1]-=10;
	}//判断当前位是否需要进位
		for(int i=1;i<=100;i++)
	{
		a[i]=a[i]*2;
	}
	for(int i=1;i<=100;i++)
	if(a[i]>=10)a[i+1]+=1,a[i]-=10;
	for(int i=100;i>=1;i--)
	{	
	if(ans==1)cout<<a[i];//输出
	if(a[i]!=0&&ans==0)cout<<a[i],ans++;//输出判定，前几位是否为零，不为零了ans=1为接下来输出的判断依据。
	}
	return 0;
}
```
