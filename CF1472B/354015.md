### 题意简述
有$n$个数，其中有$p$个数是$1$，另外$q$个数是$2$，求能不能将这$n$个数分成2个集合,使每个集合中的数字之和相等。
### 算法简析
这是一道典型的难度800的均分数字题，这道题的标签有“动规”“贪心”“数论”这三个。现在，让我们用“数论”来解决这道题。   
很明显的一点，如果这$n$个数的总和是奇数，马上判"NO"（相信大家都知道）
那么其他的情况呢？引入一个变量$mind$为这$n$个数的总和除以2。换句话说，如果可以分的话，$mind$表示为每个人分到多重的糖。那么，如果$mind$是奇数，而且没有一个重量为1，肯定分不了（见输入样例第四组数据）。
其他情况，即为"YES"。
### demo of C/C++
```c
#include<stdio.h>
#define in(x) scanf("%d",&x)
#define out(x) printf("%d\n",x)
#define rg register
int p,q,n,mind; 
int main(){
	int T,x;in(T);
	while(T--){
		in(n);
		p=q=0;/*初始化！一定不要忘记这步！*/
		for(rg int i=1;i<=n;i++){
		/*register关键字将变量存入CPU寄存器中，调用速度更快，加速程序运行*/
			in(x);
			if(x==1)p++;
			/*p为重量为1的糖果数量*/
			else q++;
			/*q为重量为2的糖果数量*/
		}
		if((p+2*q)%2!=0)puts("NO");
		/*如果这n个数的总和是奇数，马上判"NO"*/
		else{
			mind=(p+2*q)/2;/*引入变量mind为每个人分到的糖果重量*/
			if((mind%2!=0)&&(p==0))puts("NO");
			/*如果mind为奇数，且没有重量为1克的糖果，那么就分不了*/
			else puts("YES");
			/*可以证明，除上述两种情况外，其他情况均有分法*/
		}
	}
	return 0;
	/*
	至此，本解法结束。
	对于每组数据：
	时间复杂度：O(1)
	空间复杂度：O(1)
	可以通过本题。
	*/
}
```
