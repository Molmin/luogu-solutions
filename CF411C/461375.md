# 思路
思维题，只需要分类讨论就行了。
1. 当第 $1$ 队赢时，必须满足有一对组合能碾压对面的攻防手。
代码如下：
```cpp
if(a>f&&a>h&&d>e&&d>g){
	cout<<"Team 1";
	return 0;
}//第一对
if(b>e&&b>g&&c>f&&c>h){
	cout<<"Team 1";
	return 0;
}//第二对
```
2. 当第 $2$ 队赢时，必须满足对面不管选什么都要有可以碾压对面的组合。
代码如下：
```cpp
if(((e>d&&h>a)||(f>a&&g>d))&&((e>b&&h>c)||(f>c&&g>b))){
		cout<<"Team 2";
		return 0;
	}
```
3. 其余情况输出 $\texttt{Draw}$。
