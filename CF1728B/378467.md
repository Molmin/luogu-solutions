构造题。

容易发现，序列权值最大一定不会超过 $2\times n - 1$。

然后考虑构造一段单调递减序列，然后插入一个较大的数，让这一段的数列的权值为插入的较大的数。

然后插入一个单独的数 $1$，这样这一段的权值一定为 $0$。

然后就可以放心插入 $n-1$ 和 $n$ 这两个最大的数，这样数列的权值一定为 $2\times n - 1$。

注：该方法仅在 $n\ge 4$ 的时候有效。

```cpp
void solve()
{
  int n;
  cin >> n;
  vector <int> a(n + 3);
  a[n] = n;
  a[n - 1] = n - 1;
  a[n - 2] = 1;
  a[n - 3] = n - 2;
  for (int i = 1; i < n - 3; i ++)
    a[i] = i + 1;
  reverse(a.begin() + 1, a.begin() + n - 3);
  for (int i = 1; i <= n; i ++)
    cout << a[i] << ' ';
  cout << '\n';
}
```
