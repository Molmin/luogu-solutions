思路：根据题意，排列 $p$ 的最大权值肯定是 $n+n-1$，因为要求 $x$ 严格小于 $p_i$。可以通过 $n$ 的奇偶性进行分讨：先初始排列 $p_i=i$，如果 $n$ 是偶数，那么只需要交换相邻奇数下标和偶数下标的元素，注意，$p_{n-1}$ 和 $p_{n}$ 不用交换，因为要求 $x$ 严格小于 $p_i$；如果 $n$ 是奇数，可以将数字 $1$ 移到排列的开头，其他元素的下标依次递增，然后将 $p_2$ 到 $p_{n-2}$ 的数字翻转，即可以构造出答案，因为 $x$ 到 $p_{n-2}$ 时必将被初始化为 $0$。

代码：
```cpp
inline void solve(){
	int n;
	cin>>n;
	for(int i=1;i<=n;++i) a[i]=i;
	if(!(n&1)){
		for(int i=1;i<n;i+=2) swap(a[i],a[i+1]);
		swap(a[n-1],a[n]);
	}
	else{
		for(int i=2;i<=n-2;++i) a[i]=(n-2)-i+2;
	}
	for(int i=1;i<=n;++i) cout<<a[i]<<' ';
	cout<<'\n';
}
```
