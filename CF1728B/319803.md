### 题目大意

（部分来自原题目翻译）

给定 $t$ 次询问。对于每次询问，给定正整数 $n$，设序列 $p$ 是 $1\sim n$ 的全排列。定义一个变量 $X$，初始时 $X=0$。遍历 $p$，每遍历到一个 $p_i$，就将其和 $X$ 作比较。若 $X<p_i$，就将 $X$ 加上 $p_i$，否则将 $X$ 改为 $0$。请你构造序列 $p$ 使得遍历结束后 $X$ 最大，输出构造出的序列。

数据范围：$1 \le t \le 97$，$4 \le n \le 100$。

------------

### 解法分析

~~这题咋在 CF 评了个 *800？对得起我比赛的时候在这题嗑掉的半小时吗？？~~

讲一下我想到解法的过程。

首先思考一个问题：$X$ 的最大值可以是多少？

不难发现是 $2n-1$。只要使得 $X$ 在遍历完 $p_{n-2}$ 的时候为 $0$，而 $p_{n-1}=n-1,p_n=n$ 就可以了。~~我才不会告诉你我赛时想到这一点花了二十多分钟……~~

诶，那我们是不是让 $p_i=i+1(1 \le i \le n-3)$，并且 $p_{n-2}=1,p_{n-1}=n-1,p_n=n$ 就可以了？

没那么简单，如果 $X$ 在 $p_{n-3}$ 的时候被归零，那它到 $p_{n-2}$ 的时候就是 $1$ 了，最终会在 $p_n$ 被归零，不满足条件。可以试一下 $n=6$。

那正着排不行，反着排呢？

不完全可以。如果 $p_i=n-i-1(1 \le i\le n-2),p_{n-1}=n-1,p_n=n$，我们会发现，在 $i \le n-2$ 的条件下，当 $i$ 为偶数时，$p_i=0$，满足条件；而 $i$ 为奇数时则并不。

那我们在 $n$ 为奇数的时候做个特殊处理，把 $p_{n-2}$ 的 $1$ 移到序列最前面，可以吗？

没有问题。

于是我们的构造思路就是：$n$ 为奇数时，$p_1=1,p_i=n-i(2 \le i \le n-2),p_{n-1}=n-1,p_n=n$；$n$ 为偶数时，$p_i=n-i-1(1 \le i\le n-2),p_{n-1}=n-1,p_n=n$。

这题还存在其他构造方法，可以想一下。

### 代码

```cpp
#include <bits/stdc++.h>
#define ll long long
#define pb push_back
using namespace std;
ll n;
void solve() {
	scanf("%lld",&n);
	if (n&1) {
		printf("1 ");
		for (ll i=n-2;i>1;i--) printf("%lld ",i);
	}
	else for (ll i=n-2;i;i--) printf("%lld ",i);
	printf("%lld %lld\n",n-1,n);
}
int main() {
	ll t; scanf("%lld",&t); while (t--) solve();
	return 0;
}
```
