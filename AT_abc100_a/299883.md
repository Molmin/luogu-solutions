# **正文：**
## 题意：
切成16片的蛋糕，使同一个人拿取的所有块都不相邻，我们可以先考虑最多能够有几片蛋糕是不相邻的。如果我们隔一片、取一片的话，最多能够取到 $16/2=8$ 片，如果超过此数则必定有两片蛋糕相邻的情况。所以两方不得有任意一方取得超过八片，否则就拿不到相应的片数。

## 程序说明：
我们使用if语句判断两数是否都 $≤8$ ，若是则输出 _Yay!_ 否则输出 _:(_ 。
```cpp
#include<iostream>
using namespace std;
int a,b;                   //一行定义
int main(){
    cin>>a>>b;              //一行输入
    if(a<=8&&b<=8)cout<<"Yay!";  //两行判断+输出
    else cout<<":(";
    return 0;
}
```
