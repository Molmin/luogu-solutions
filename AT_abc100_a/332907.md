既然题目说了一共16块蛋糕，不能取相邻两块，那么我们就让他们一人一块，一块一块轮流取。也就是2个为1组，16÷2=8组，而每个人1组是只能取1块，每个人最大限制是8块，否则就无法满足题目要求.

代码如下
------------
```
#include<bits/stdc++.h>
using namespace std;
int main()
{
  	int a,b;
  	cin>>a>>b;			//输入
  	if(a<=8&&b<=8)			//判断
    	{
    	     cout<<"Yay!";		//输出Yay！
    	}
    	else				//否则
    	{
    	     cout<<":(";		//输出:(
    	}
  	return 0;
}
```
