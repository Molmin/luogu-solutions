## 题意

给你一个字符串，求在它当中所有回文子序列中字典序最大的那个串。

## 思路

一上来我想，直接暴力呗，然后~~不会写~~突然想到如下证明：

1. 如果 $a$ 串字典序大于 $b$ 串，那么 $a$ 的首字母和尾字母一定大于 $b$ 串的首字母和尾字母。

2. 字典序和长度无关，那么我们可以让这个回文串只有同一个字母，就是在字符串中出现的 ```ASCII``` 最大的字母，这样就可以最大了。

听起来有些拗口，咱们举个栗子康康：

一个字符串为 ```abxxyyzz``` 。

那么根据第二条我们得到字典序最大的是 ```zz``` 。

如果 ```zz``` 中间可以加入其它字母并变得更大，我们可以添加成为：```zzazz```，```zzbzz```，```zzxzz``` 等等，我们发现，因为字典序是按位对照的，而后续添加的字母没有原本的字母大，所以字典序对照到这一位时会认为原先的大。

那么我们就得到结论了。实现也很简单，输入串，然后用一个桶记录，再从后向前扫一遍，找到第一个出现的字母，按出现的次数一一输出就行了~

## 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
char x;
int h[30]; //h记录字符数量（其实就是是否出现过） 
int main()
{
	while(cin>>x) h[x-96]++; //统计 
	for(int i=26;i>=1;i--)
	{
		if(h[i]) //如果出现过 
		{
			while(h[i]--) cout<<(char)(i+96); //输出 
			return 0;
		}
	}
}
```
