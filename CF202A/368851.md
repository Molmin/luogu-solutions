

**算法:字符串模拟**

## 思路

*  分析

通过样例和题目，我们不难得知，题目是要我们输出字典序最大的回文字符串，而我们也可以输出只由一个字母构成的回文串，这样一分析，不就简单了吗

* 做法

我们只需要用数组对字符串的字母数进行统计，再用循环暴力倒序遍历。

若有一个字母数不为0，直接输出即可

因为题目想让我们输出字母序最大的回文串，所以要倒序找，
第一个就是字母序最大的回文串。

## CODE


```cpp
#include<bits/stdc++.h>
using namespace std;
string s;
int p[27];//统计字符串里各个字母的数目 
int main(){
	cin>>s;//输入 
	for(int i=0;i<s.size();++i){//遍历字符串
		p[s[i]-'a']++;//统计 
	}
	for(int i=25;i>=0;--i){
		if(p[i]){
			for(int j=1;j<=p[i];++j){//输出字典序最大的回文字符串
				cout<<char(i+97);//输出 
			}
			break;//跳出循环 
		}
	}
    return 0;
}
```
