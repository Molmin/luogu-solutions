这道题的大意是给你一个矩阵，有一些地方是黑色，有一些地方是白色，每次可以选择一个黑色的点，之后把这个黑色点所在行或所在列全部涂成黑色。

给定坐标 $(r,c)$，求需要几步把矩阵上坐标为 $(r,c)$ 的那个点涂成黑色。

**思路：**

简单分类讨论解决。

我们可以把给定的矩阵分作四大类：

1. 给定的矩阵全是白色，这样的矩阵无法操作，所以答案是 $-1$。
2. 给定点 $(r,c)$ 这个点在原矩阵上就已经是黑色，这样我们不用进行任何操作，所以答案是 $0$。
3. 给定点 $(r,c)$ 这个点所在行或所在列上有一个点为黑色，这样在这个为黑色的点上进行操作即可将点 $(r,c)$ 染成黑色，所以答案是 $1$。
4. 其他情况：这时我们知道给定矩阵上有黑色的点，但是给定点 $(r,c)$ 所在行或列都没有黑色的点，这时我们可以选择任意一个点做任意染色。容易看出染色之后给定点 $(r,c)$ 所在行或一定会有一个点被染色，这时重复第三条操作即可，答案为 $2$。

```cpp
#include<iostream>
using namespace std;
char a[100][100];
int main(){
	int t;
	cin>>t;
	while(t--){
		int n,m,r,c;
		cin>>n>>m>>r>>c;
		bool flag=0;
		for(int i=1;i<=n;i++){
			for(int j=1;j<=m;j++){
				cin>>a[i][j];
				if(a[i][j]=='B') flag=1;
			}
		}
		if(a[r][c]=='B'){
			cout<<0<<endl;
			continue;
		}
		if(!flag){
			cout<<-1<<endl;
			continue;
		}
		flag=0;
		for(int i=1;i<=n;i++){
			if(a[i][c]=='B') flag=1;
		}
		for(int i=1;i<=m;i++){
			if(a[r][i]=='B') flag=1;
		}
		if(!flag) cout<<2<<endl;
		else cout<<1<<endl;
	}
}
```
