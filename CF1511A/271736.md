为了从推荐票最多的角度出发，三种票可以这样解释：

- 第一种：推荐票；

- 第二种：反对票；

- 第三种：若第一种的的个数大于等于第二种，推荐票。

由此可见，推荐票最多的情况应该是第一种和第三种全部都是推荐票（因为第二种票都是反对的性质已经无法改变）。

考虑如何使第三种票都是推荐票（因为第一种票都是推荐的性质已经无法改变）：

构造一个盒子全是第一和第三种，另一个盒子全是第二种即可。

则答案为：第一种票和第三种票的个数。

代码如下：
```cpp
#include<bits/stdc++.h>
using namespace std;
int t,n,a[55],ans;
int main(){
	cin>>t;
	while(t--){
		cin>>n;ans=0;
		for(int i=1;i<=n;++i){
			cin>>a[i];
			if(a[i]!=2)ans++;
		}cout<<ans<<endl;		
	}
}
```
