### 题意
简单明了，有一个投票的网站，这个网站有2个服务器，现有$t$ 组数据，每组数据有$n$ 个人投票，现把这$n$ 个人随机分到这2个服务器中，求赞成票最多的那个服务器中的赞成票数量。

投$1$ 为赞成，$2$ 为反对，$3$ 则是查看这个服务器中的赞成和反对票数量，如果反对票多，就投反对票，反之投赞成票。
### 做法
这道题的题眼在于，随机分配到两个服务器中（不限人数），这样我们就可以把投赞成和中和（下文均把$3$ 称为中和）的人分到第一个服务器中，投反对的人分到第二个服务器中，这样我们就只需查看第一个服务器中投赞成和中和的人数之和（因为第一个服务器中无反对票，所以我们无需判断中和），然后就是代码框架，首先输入$t$ ，然后写循环输入$n$ ，再循环输入$r$ ，进行分配服务器，将结果存入答案，最后循环输出。
### Code
```cpp
#include<iostream>

using namespace std;
int t,n,ans[10010];//数据，人数，和答案

int te(int a[],int n) {
	int b=0;//要初始化为零
	for (int i=1;i<=n;i++) {
		if (a[i]==1 or a[i]==3) {//不用考虑2
			b++;
		}
	}
	return b;
}

int main() {
	cin>>t;//数据
	for (int i=1;i<=t;i++) {
		cin>>n;//人数
		int r[51]={};//初始化
		for (int j=1;j<=n;j++) {
			cin>>r[j];
		}
		ans[i]=te(r,n);//判断并加入答案
	}
	for (int i=1;i<=t;i++) {
		cout<<ans[i]<<endl;//输出
	}
	return 0;//完美结束
}
```
### 总结
主要难点在于判断赞成人数和分服务器。
