一道简单的小贪心。

考虑到第一类用户只会投推荐，第三类用户只会投反对，所以为了获得更多的推荐，我们需要尽量争取到第二类用户的推荐。

由于有两个服务器，为了让第二类用户投推荐，我们可以让它看见的票全都是推荐票，这样我们就可以获得所有第二类用户的推荐。

所以我们可以极端地将第一、二类用户放一个服务器，第三类用户放一个服务器。

答案就为第一类用户与第二类用户的总个数。

```cpp
int main(){
	int t=read();
	while (t--){
		int n=read(),ans=0;
		for (int i=1;i<=n;i++){
			int a=read();
			if (a==1||a==3) ans++;
		}
		printf("%d\n",ans);
	}
	
	return 0;
}
```
