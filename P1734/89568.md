## 主要思路：
## 1.先预处理，处理出每个数的约数和（a数组）
## 2.然后显然易见是个01背包，每个数仅可以取一次（dp数组）
## 如何处理约数和？


------------
```c
for(i=1;i<=n/2;i++)//循环每个数（小优化：因为i*j<=n j>=2 所以i<=n/2）
		for(j=2;i*j<=n;j++)//循环倍数（排除本身，从2开始）
			a[i*j]+=i;//目标数加上约数
```





------------
## 这些问题都解决后，代码就能轻松实现了
------------


```c
#include<bits/stdc++.h>
using namespace std;
int i,j,n,a[1001],dp[1001];
int main()
{	cin>>n;//输入
	for(i=1;i<=n/2;i++)
		for(j=2;i*j<=n;j++)
			a[i*j]+=i;//预处理
	for(i=1;i<=n;i++)//循环n个数
		for(j=i;j<=n;j++)
			dp[j]=max(dp[j],dp[j-i]+a[i]);//取或不取
	cout<<dp[n];//输出
  	return 0;
}
```
