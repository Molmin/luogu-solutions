### Description

给定 $a_1, a_2, a_3$，可以做无限次以下操作：

选择不同的两个 $i,j$，给 $a_i+1,a_j-1$.

目标是最小化 $|a_1+a_3-2\times a_2|$.

多测。

**数据范围**

$1\leq a_1,a_2,a_3 \leq 10^8$

$1\le T\le 5\times 10^3$

### Solution

容易发现，选 $a_1,a_3$ 是没有区别的，所以我们考虑将问题转化成：

有两个整数 $A=a_1+a_3,B=a_2\times 2$，每次可以将 $A-1,B+2$，或是将 $A+1,B-2$，最小化 $|A-B|$.

这个时候我们发现每次会将差值减小 $3$，所以最后的答案一定不会大于 $1$，接着就会发现，如果 $A-B$ 能被 $3$ 整除，则答案就是 $0$，反之为 $1$.

### Code

```cpp
inline void solve() {
	in(a, b, c);
	if ((a + c - b * 2) % 3 == 0) {
		out('\n', 0);
	} else {
		out('\n', 1);
	}
}
```
