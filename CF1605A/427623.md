## 前言
一道数论题。
## 题意概述
给定三个整数 $a_1,a_2,a_3$，允许你无限次的将其中一个数增加一并将另一个数减一，问 $|a_1+a_3-2\times a_2|$ 最小是多少。
## 解题思路
1. 首先，很明显，我们无论怎么对 $a_1$ 与 $a_3$ 操作，答案总不会改变。因为 $a_1 +a_3$ 的值总不会改变。
2. 接下来，若我们增加 $a_1$ 或 $a_3$，减少 $a_2$，会对答案产生什么贡献呢？显然，答案会增加 $3$，因为 $a_2\times 2$ 减少了 $2$，$a_1+a_3$ 增加了 $1$。
3. 最后，若减少 $a_1$ 或 $a_3$，增加 $a_2$ 呢？显然，答案会减少 $3$。

由上述可得，我们只需要考虑 $a_1+a_3-2\times a_2 \bmod 3$ 的值就好了。若上式 $\bmod 3$ 为 $1$ 或 $2$，则答案为 $1$，否则就为 $0$。
## 代码
```cpp
#include<cstdio>
int main(void){
    int t,a1,a2,a3;
    scanf("%d",&t);
    while(t--){
        scanf("%d%d%d",&a1,&a2,&a3);
        printf("%d\n",((a1+a2+a3)%3!=0));
    }
    return 0;
}
```
