#  CF62A 最棒的题解。
## 这道题其实只需要您做一个判断。
### 判断1：男孩用左手握女孩的右手。
- 女孩的两根手指中间必须有男孩的一根手指：br+1>=al。
- 男孩的三根手指之间必须有女孩的一根手指：al*2+3>br。
### 判断2：男孩用右手握女孩的左手。
- 女孩的两根手指中间必须有男孩的一根手指：bl+1>=ar。
- 男孩的三根手指之间必须有女孩的一根手指：ar*2+3>bl。
### 如果数据符合判断1或判断2
- 输出YES。
- 否则输出NO。
# 最终实现代码
```
#include<bits/stdc++.h>
using namespace std;
int al,ar,bl,br;//定义。
int main(){
	cin>>al>>ar>>bl>>br;//输入。
    if(al*2+3>br&&br+1>=al||ar*2+3>bl&&bl+1>=ar)
    //如果符合判断1或判断2。
    	cout<<"YES"<<endl;//输出YES。
    else
    	cout<<"NO"<<endl;//否则输出NO
    return 0;//完美结束(ง •_•)ง
}
```
