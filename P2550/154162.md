这题输入完以后只要将小明买的每一张彩票中的每一个数跟中奖号码的每一个数比较一遍，记录相同数字的个数，再判断是几等奖就可以了。

下面是代码。

------------
```cpp
#include<iostream>
using namespace std;
int main()
{
	int n,x[7],ss[7]={0,0,0,0,0,0,0};    //x用来存储中奖号码，ss用来存储中奖情况 
	cin>>n;
	const int nn=n;           //数组大小不能是变量，所以要定义一个常量 
	int a[nn][7];             //一个二维数组用来存储小明买的彩票 
	for(int i=0;i<7;i++)      //输入中奖号码 
	{
		cin>>x[i];
	}
	for(int i=0;i<n;i++)      //输入小明买的彩票 
	{
		for(int j=0;j<7;j++)
		{
			cin>>a[i][j];
		}
	}
	int sum=-1;               //sum用来存储每一张彩票中与中奖号码中同样的数字个数
	                          //注意要设置为-1，因为数组的下标是从0开始的 
	for(int i=0;i<n;i++)
	{
		for(int j=0;j<7;j++)  //i,j用来循环小明的彩票 
		{
			for(int k=0;k<7;k++) //k用来循环中奖号码 
			{
				if(a[i][j]==x[k])  
				{
					sum++;
				}
			}
		}
		if(sum!=-1)          //只要sum不等于-1，就意味着相同数字的个数不为0 
		{
			ss[6-sum]++;     //6-sum即为n等奖的下标（将特等奖的下标视为0） 
			sum=-1;          //进行下一轮判断前记得重置sum 
		}
	}
	for(int i=0;i<7;i++)     //输出 
	{
		cout<<ss[i]<<' ';
	}
	return 0;
} 
```
