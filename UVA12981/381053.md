# 题意
这道题就是给出两个 $2\times 2$ 的矩阵，问你是否可以通过翻转从第一个矩阵变为第二个矩阵？
# 思路
~~这道题其实很简单~~，我们可以先固定一个矩阵然后去翻转一个矩阵 $4$ 次，并在每次翻转完后与第二个矩阵相比较即可。那我们的翻转过程可以用暴力并按规律来写（毕竟才 $2\times 2$）。
# 代码
翻转代码：
```cpp
	int t=arr[1][1];
	arr[1][1]=arr[2][1];
	arr[2][1]=arr[2][2];
	arr[2][2]=arr[1][2];
	arr[1][2]=t;
```
比较代码：
```
	for (re int i = 1;i <= 2;i++){ //逐行比较
		for (re int j = 1;j <= 2;j++){ 
			if (arr[i][j] != brr[i][j]) return false;
		}
	}
	return true;
```
