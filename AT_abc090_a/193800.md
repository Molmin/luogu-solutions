### 这道题挺简单的，正好适合我这种萌新。

#### 主要思路：
由于这个矩阵的大小已经固定了，所以不妨暴力一点。定义九个字符，依次输入，然后输出对角线上那三个字符。
##### 上代码：
```cpp
#include<bits/stdc++.h>//万能头
using namespace std;
int main()
{
	char a,s,d,f,g,h,j,k,l;//定义九个字符
	cin>>a>>s>>d;//读入第一行
	cin>>f>>g>>h;//读入第二行
	cin>>j>>k>>l;//读入第三行
	cout<<a<<g<<l<<endl;//输出对角线上的字符。注意换行。
	return 0;
 } 
```
这道题就能过了。
