对于新手来说一道基础练习

题目已经明确说明了输入一个$3*3$的一个正方形

~~这样会让这道题简单很多~~

然后我们只需要输出这个正方形从左上到右下方的对角线就可以啦

也就是说，我们用三个字符串数组$a_i,b_i,c_i$ ，输出$a_1,b_2,c_3$即可。

~~应该很好理解。~~

接下来是$Solution$:
```cpp
#include<bits/stdc++.h>//万能头大法好
using namespace std;
int main()
{
    char a[4],b[4],c[4];//个人习惯
    for(int i=1;i<=3;i++)//循环输入
    {
        cin>>a[i]>>b[i]>>c[i];
    }
    cout<<a[1]<<b[2]<<c[3];
    return 0;//完结撒花
}
```
