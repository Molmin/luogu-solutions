	三角波(Triangular Wave)也称锯齿波
    是指主要用在CRT作显示器件的扫描电路中的波形
    因其显示图像像一个一个三角形而得名
    
其实这里就是让你**打印像三角波一样的图像**啦……

**振幅**就是指你所需要打印的三角波的峰值

**频率**就是指你在一组数据中这组三角波会重复几次

## 另外要注意的是：UVA的I/O格式贼恶心

再看一眼题目就会看到一句话：

	两个连续案例的输出将用空行包围
    
又看到：

	除最后一个波形外，每个单独的波形后面都有一个空行。
    
也就是说：每个波形中间都要空出一行，即换行两次

### 但是最后一个波形后面不能空行！只能换行一次！！！

总的来说就是这样啊

我的做法里把每一组三角波拆分成上半波和下半波，分开算更简单

上代码：

```
#include<bits/stdc++.h>
using namespace std;
int a,f;  //a是振幅，f是频率
void Upwave( int n ){  //打印上半波
	int i,j;
	for ( i = 1 ; i <= n ; i++ )
	{
		for ( j = 1 ; j <= i ; j++ )
			cout << i;
		cout << endl;  //注意在哪里换行
	}
	return;  //这可以不打，个人习惯
}
void Downwave( int n ){  //计算下半波
	int i,j;
	for ( i = n - 1 ; i >= 1 ; i-- )  //这里就反过来打印了
	{
		for ( j = 1 ; j <= i ; j++ )
			cout << i;
		cout << endl;
	}
	return;
}
int main(){
	int n;
	cin >> n;  //先读入有几组数据
	for ( int i = 1 ; i <= n ; i++ )
	{
		cin >> a >> f;
		while ( f-- )  //一直执行到 f = 0
		{
			Upwave( a );
			Downwave( a );
            //打印上半波和下半波
			if ( i != n || f != 0 )  //做换行判断，为啥上面都讲过了
				cout << endl;
		}
	}
	return 0;
}
```
