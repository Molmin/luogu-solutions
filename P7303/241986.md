高斯求等差数列的和的公式：

和=（首项+末项）×项数/2

~~就这么简单~~

```
这题应该是入门题QWQ，就是枚举K个互不相同的正整数，使它们的和为N,连数据范围……都是那么友好

```

等等，满分怎么是50分QWQ

```cpp
#include<stdio.h>
int k,n;
int main(){
	scanf("%d%d",&k,&n);
	for(int i = 1;i < n;i ++) printf("%d\n",i);//就是1、2、3、4、5……QWQ
	//这些数的和为（1+n-1）*（n-1）/2，高斯公式
	//即n*（n-1）/2 
	printf("%d\n",k-n*(n-1)/2);//别忘了换行 
} 
```
