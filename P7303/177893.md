### 题意简述
给定两个整数 $n,k$ ，要求构造出一个长度为 $n$ 的序列 $a$ ，满足

- $\forall 1 \leq i \leq n ,a_i \geq 1$
- $a_i$ 两两互不相等。

数据保证有解。
### Solution
首先~~根据直觉~~，我们可以贪心：

1. 对于 $i \in [1,n-1]$ ，令 $a_i=i$
1. 让 $a_n=k - \sum_{i=1}^{n-1} a_i$

那为什么这样是对的呢？

首先，因为下标 $i \geq 1$ 且每两个下标互不相同，所以对于 $a_1\sim a_{n-1}$ 满足条件。

同时，这样构造的话， $a_1\sim a_{n-1}$ 的和最小，那么 $a_n$ 就会是最大的。

题目保证有解，那么 $a_n$ 一定 $> n - 1$ 。

如果 $a_n \leq n - 1$ ，那么 $a_n$ 就会和前面某个 $a_i$ 相等。于是我们只能修改前面的 $a_i$ 来满足题意，但是无论怎么修改，都一定会有两个相同的值。

所以在这样的情况下无解，与题目矛盾。

所以这样的贪心是正确的。

代码如下：
```cpp
#include <cstdio>
int n ,k;
signed main() {
    scanf("%d%d" ,&k ,&n);
    for (int i = 1;i <= n - 1; i++) printf("%d\n" ,i) ,k -= i; //直接在循环里让 k -= i ，就可以避免等差数列求和公式
    printf("%d\n" ,k); //k 已经减完了，剩下的就是 a_n
    return 0;
}
```
