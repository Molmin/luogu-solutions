我们可以这么想：把 $n$ 个各不相同的玩家分成 $k$ 个名次组，比如甲乙丙丁可以分成两个名次组的方案为:
1. {甲, 乙}{丙, 丁}
2. {甲, 丙}{乙,丁}
3. {甲, 丁}{乙, 丙}
4. {甲}{乙, 丙, 丁}
5. {乙}{甲, 丙, 丁}
6. {丙}{甲, 乙, 丁}
7. {丁}{甲, 乙, 丙}

每个名次组之间可以随意排列，比如 {甲, 乙} 第一，{丙, 丁} 第二，或者 {丙, 丁} 第一，{甲, 乙} 第二。

那么把$n$个各不相同的玩家分成 $k$ 个名次组的方案数其实就是第二类斯特林数 $S(n,k)$，$k$ 个名次组排列的的方案数显然是 $k!$，于是对于 $n$ 个玩家的答案就是
$$
\sum_{i=1}^{n} i!\times S(n,i)
$$

```cpp
    S[0][0] = 1;
    fact[0] = 1;
    for (int i = 1; i <= 1000; i++) {
        fact[i] = fact[i - 1] * i % MOD;
        for (int j = 1; j <= i; j++) {
            // 第二类斯特林数的递推公式
            S[i][j] = ((S[i - 1][j] * j % MOD) + S[i - 1][j - 1]) % MOD;
        }
    }
    for (int i = 1; i <= 1000; i++) {
        for (int k = 1; k <= i; k++) {
            // ans 就是最终答案
            ans[i] += (S[i][k] * fact[k]) % MOD;
            ans[i] %= MOD;
        }
    }
```
