## 主要的思想是转化为二叉树

首先我们读入是可以很自然的计算出每个玩具的deep[x],即深度

然后题目里这样说道:

**"所有的玩具都在同一层(也就是说，每个玩具到天花板之间的杆的个数是一样的)或至多相差一层。"**

也就是说读入时计算出最深与最浅，计算出之差判断即可

如下
```
if(maxdeep-mindeep>1)
{
 cout<<-1<<endl; 
 return 0;
}
```
