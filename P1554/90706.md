## 正文部分：

### 1.除号（“/”）
这个东西就是用来做除法的，却和我们小学学的不一样。

小学除法：11/4=2.75。

c++：11/4=2。（？？）

某人问c++:“还有3呢？”

c++曰：“~~被我私吞了！~~”

就是这个区别：c++会向下取整。

那么/10会发生什么呢？

#### 1234/10=123
#### 123/10=12
#### 12/10=1
#### 最后1/10=0

运行次数刚好是它的位数。

所以：
```cpp
int x=1234，ans=0; //x是要分离的数,这里取1234;ans是位数。
while(x!=0)
{
    x=x/10;//位数减一
    ans++;//答案加一
}
```
这样就求出位数了。

可是题目没叫我们求位数啊！求的是每个数字出现的次数。

### 2.模（“%”）

这个比较简单：

小学算数：11/4=2余3。

c++：10%4=3。

对，就是余数。

那么%10呢？

#### 1234%10=4
#### 123%10=3
#### 12%10=2
#### 最后1%10=1

诶？求出了尾数！

结合除法：
#### 1234%10=4
#### 1234/10=123
#### 123%10=3
#### 123/10=12
#### 12%10=2
#### 12/10=1
#### 1%10=1
#### 1/10=0
#### 然后结束。

于是乎~~
```
#include<bits/stdc++.h>//万能头文件
using namespace std;
int n,m;//m为开始，n为结束。
int a[10];//a[0]到a[9]代表每个数码出现了几遍。
int main()
{
    cin>>m>>n;//输入
    for(int i=m;i<=n;i++)//循环
    {
        int x=i;//防止i改变。
        while(x!=0)//小循环
        {
            a[x%10]++;//最后一位的数码加一
            x/=10;//下一位
        }
    }
    for(int i=0;i<=9;i++)
    	printf("%d ",a[i]);//输出
    return 0;
}
```
