实际上这道题可以只用4个变量解决

——————————~~我是华丽的分割线~~———————————

因为第i次累加的不高兴值（设为x[i]）总是满足这样的式子：

x[i] = x[i - 1] + a[i] + b[i] - 8;

研究后可以发现：

a[i] 和 b[i] 都加了 (n - i + 1) 次

8 一共减了 (n * (n - 1) / 2) 次

而且由于a[i] 和 b[i] 的加的次数一样，就可以愉快的合起来加了！(虽然我自己都懒得合起来。。)

下面，上代码！

```cpp
#include <bits/stdc++.h>

using namespace std;

int n, a, sum;

int main() {
    scanf("%d", &n);
    for(int i = 1; i <= n << 1; i++)
        scanf("%d", &a), sum += a * (n - (i - 1 >> 1));
    printf("%d\n", sum - 8 * n * (n + 1) / 2);
    return 0;
}
```
