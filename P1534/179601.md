#### 第13课：不高兴的津津（升级版）


------------

## 注意事项：
1. 共n天，一天一组数据
1. 昨天的不高兴程度加上2组时间的和减去8，便是今天不高兴程度（可正和负）
1. 题目要求的是n天不高兴程度的和而不是第n天的不高兴程度。
## 现在献上AC代码
#### 梦雪小课堂提醒您，注释很重要
```
#include<iostream>	//或万能头
using namespace std;
int a,b,c,s,v;
int main(){
	cin>>a;	//及题目中的n
	while(a--){	//共a组数据，a次循环
		cin>>b>>c;	//输入本次循环的两个时间
		s=s+b+c-8;	//s+b+c-8及今日的不高兴程度(s及昨天的)
        //补充知识点，因为第一天不需要加上昨天的不高兴程度，所以s需初始化0，方法是把s的定义放在主函数上方或在定义时加上=0
		v=v+s;	//用v累加（详情请见注意事项3）
	}
	cout<<v;	//输出
}
```
