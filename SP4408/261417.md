题目大意：有一堵无限大的墙，用长度为n的绳子与这堵墙所能围成的区域**最大面积**是多少？

所以，这题要求周长相等时的最大面积。

大家首先想到的可能是半个正方形（为什么是半个呢？因为要把那堵墙利用上），但大家很快又想到：题目没有说围成的图形不能是曲线图形，那为什么不围成圆形呢？

这里要引入一个概念：在同周长的情况下，圆形的面积最大。

举个例子：周长为120.

正方形面积：120$\div$4=30。

30*30=900

圆形面积：120$\div$2$\div$π=60$\div$π。

（60$\div$π)$^2$ $\times$π$\approx$1146

显然，正方形面积<圆形面积（同周长时）
 
但这题并不是求圆形面积而是**半圆形面积**，因为你要把墙利用上。
 
所以，这题转换成了：有一堵无限大的墙，用长度为n的绳子与这堵墙所能围成的**半圆形**是多少？
 
圆的周长公式为：2πr

圆的面积公式为：πr$^2$。

代码：
```cpp 
#include <bits/stdc++.h>
using namespace std;
double pi=3.1415926;//π的定义
int main(){
	int n;
	while(1){
		cin >> n;
		if (n==0) break;//读到0就退出
		double z=n*1.0/2.0/pi;//圆的半径，注意要把int型的n转化为double型的
		printf("%.2lf\n",z*z*2*pi);//按公式输出，注意是半圆面积！！！
	}
}



```
