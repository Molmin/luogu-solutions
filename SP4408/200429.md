### 题意：
有一堵无限大的墙，用长度为n的绳子与这堵墙所能围成的区域最大面积是多少？
### 思路：
我们都知道，周长相同，圆形的面积最大。面积要最大，而且这里要与墙围成的区域，就要围成半圆形。所以这一题实际上就是给出一个半圆的弧的长度，让我们求这个半圆的面积。这里我们可以把这个半圆看成是一个圆，弧的长度就是圆周长的一半，这样我们就能求出这个圆的半径，也是这个半圆的半径，然后计算出这个圆的面积，再除二就是这个半圆的面积，也就是围成的最大面积。
### 代码:
```cpp
#include<bits/stdc++.h> //头文件
using namespace std;
int main() //主函数
{
	ios::sync_with_stdio(false); //输入输出优化流
	double n,r; //定义，n表示面积，r表示半径
	while(cin>>n) //输入
	{
		if(n==0) break; //返回
		n=n*2; //圆的周长
		r=n/3.14159/2; //圆的半径，也是这个半圆的半径
		n=r*r*3.14159; //圆的面积
		n=n/2; //半圆的面积
		cout<<fixed<<setprecision(2)<<n<<endl; //输出
	}
	return 0; //结束
}

```
