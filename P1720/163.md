代码如下：（别抄！发代码不是为了让你抄，而是为了让你理解啊）

No.1 模拟：

```cpp
        int n（输入数据）;
        double a=1（公式左上角）,b=1（公式右上角）;
        scanf("%d",&n);
        for(int i=1;i<=n;++i){a*=0.5+sqrt(5)/2;b*=0.5-sqrt(5)/2;}
        printf("%.2f",(a-b)/sqrt(5));
```
No.2 递推：
        int n,a=0,b=1,t;

        scanf("%d",&n);  

        while(n--){t=a;a=b;b+=t;}
