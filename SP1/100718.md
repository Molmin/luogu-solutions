各大佬大多用while，我用用for好像也OK吧  
思路：  
1.边读入边判断是否为42  
2.是输出原数，不是则停止循环

代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,i;
int main()
{
    for(i=1; ;i++)
    {
        cin>>n;
        if(n==42)
          break;
        cout<<n<<endl;  
    }
    return 0;
}
```
