 这题的确很水，主要考察的是循环输入。
 
 在这里，我来介绍一种表达式：逗号表达式。
 
 逗号表达式是几个信息用逗号连接起来的表达式。
 
 这种表达式也有返回值。是什么？是它的最后一个信息。
 
 例如， $1,2,3,4$的返回值就是4，$1<2,2<1$的返回值就是$false$。
 
 那么，它怎么运用到循环输入上面呢？
 
 例如知道题，一般的循环输入都是$while(cin>>n)$
 
 而简便方法就是$while(cin>>n,n!=42)$
 
 这样，你既能输入，又能判断$n$的值，如果$n$是42，就跳出循环。

附9行AC代码：
```cpp
#include<iostream>
using namespace std;
int main()
{
    int a;
    while(cin>>a,a!=42)
        cout<<a<<endl;
    return 0;
}
```
