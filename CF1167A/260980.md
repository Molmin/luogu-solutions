### 分析

感觉题目描述已经很到位了就讲一下思路吧。

思路就是先找到第一个 8 的位置然后判断一下是否 $\ge$ 11 位，如果大于那么可以从后面开始删除。

总之如果满足这个条件，那么就可以的，输出 ```YES``` 否则输出 ```NO``` 注意大小写。

```cpp
#include <iostream>
#include <cstdio>
using namespace std;
int n,t;
char a[100];
//变量定义和头文件。
int main()
{
    cin>>t;
    //输入数据组数。
    for(int i=0;i<t;i++)
    {
        for(int j=0;j<100;j++)
           a[j]='0';//清 0 否则会有玄学的错误出现。
        cin>>n;
        //输入位数，其实这个位数不要也可以，但是题目要求。
        for(int j=0;j<n;j++)
            cin>>a[j];
        //输入字符串。
        int sum=n;
        //记录 n 
        for(int j=0;j<n;j++)
        {
            if(a[j]!='8')
                sum--;
                //如果不等于那么就 -1 
            else
                break;
                //如果找到了那么就跳出循环
        }
        if(sum<11)
            cout<<"NO\n";
        else
            cout<<"YES\n";
        //反过来，如果没有到达 11 那么就 输出 NO 否则 输出 YES。
    }
}
```
