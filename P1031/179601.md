#### 第14课：均分纸牌

- ## 方法讲解：
1. 定义两个变量:s,pj,并初始化0。补充知识点：因为s,pj是用来累加的，所以s，pj需初始化0，方法是把s/pj的定义放在主函数上方或在定义时加上=0
1. 用pj将所有数累加后除以a(堆数），得到均值。
1. 从0到a-1,多的给后面，少的找后面
- ## 现在献上AC代码
#### 梦雪小课堂提醒您：注释很重要
```
#include"iostream"	//或万能头<bits/stdc++.h>
using namespace std;
int a,s,pj,b[11111];	//自动初始化0
int main(){
	cin>>a;	//堆数
	for(int i=0;i<a;i++){
		cin>>b[i];
        pj=pj+b[i];	//累加（方法讲解2.1）
	}
    pj=pj/a;	//求均值（方法讲解2.2）
	for(int i=0;i<a;i++)
		if(b[i]!=pj){
			b[i+1]=b[i+1]-pj+b[i];	//下一堆的变化
			s++;	//移动次数加1
		}
	cout<<s;	//输出结果
    return 0;	//؏☝ᖗ乛◡乛ᖘ☝؏完美结束
}
```
