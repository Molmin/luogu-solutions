发现没锁定词条便发个题解，第一次发，大犇见谅。。

我们可以把平均值设为0，牌数大于平均值的排堆牌数为正数，反之则为负数。

我们把牌堆数向后挪，就像这样：

9 8 17 6 化简为 -1 -2 7 -4

1、 0 -3 7 4 //a[1]的牌数向后退与a[2]累加（其实是相减，但就是加上负数）

2、 0 0 4 0 //重复

3、0 0 0 0 //此时所有牌堆牌数为平均值，停止循环。

共3次，输出为3。

如果牌堆数已经为平均数，便可以不再移动，以此省略移动次数。

下附AC代码：

```cpp
#include<iostream>
using namespace std;
int a[101],mean=0,t=0,n;
int main()
{
    cin>>n;  
    for (int i=1;i<=n;++i)
    {
        cin>>a[i];
        mean+=a[i]; 
    }
    mean/=n; //求平均值
    for (int i=1;i<=n;++i)
    a[i]-=mean; //简化
    for (int i=1;i<=n;++i)
    {
    if (!a[i]) continue; //如果牌堆数已经为平均数，便可以不再移动，以此省略移动次数
    a[i+1]+=a[i]; //把牌数向后挪
    t++;  //累加次数
    }
    cout<<t;
    return 0;
}
```
