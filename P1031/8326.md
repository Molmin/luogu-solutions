这道题采用分堆的方法，假若某个连续区间的总数量恰能自给自足(即为区间内总牌数为平均值乘以该区间长度)，并且该区间不能分为更小的符合条件的区间，这时对于这段区间的移动步数恰为区间长度减去1，就这样有技巧的分割出来符合条件的区间。（加油理解吧！）

```cpp
#include<stdio.h>
int n,a[110],t=0,m=0,s=0;
int main()
{
    scanf("%d",&n);
    for(int i=1;i<=n;i++)
    {
        scanf("%d",&a[i]);
        t+=a[i];
    }
    t/=n;//t为平均值
    for(int i=1;i<=n;i++)
    {
        a[i]-=t;//计算差多少
        s+=a[i];//求到目前为止的差值和
        if(s!=0) m++;//如果不能恰好补完，就加上
    }
    printf("%d\n",m);
    return 0;
}
```
