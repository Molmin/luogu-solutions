# 题目大意

你有一个 $n\times n$ 的棋盘。给出如下定义：

- 定义棋子的**一步**为从该棋子当前位置开始向上下左右四个方向走**一格**。
- 两个棋子互相侵犯，当且仅当一个棋子走一步可以走到另外一个棋子的位置。

你希望在上面摆放 $k$ 个棋子，使得所有棋子在允许走最多一步的情况下不存在有两个棋子互相侵犯的情况。你想知道这个目标是否能够达成，如果能的话请给出一个方案，否则输出 `-1`。

# 题目分析

要想满足条件，即没有任意一行一列车辆数大于等于 $2$，我们于是可以发现——直接放对角线即可满足条件（当然需要隔一行一列咯）：

```plain
5 3:
R....
.....
..R..
.....
....R
```

通过计算和简单归纳可知：一个 $n\times n$ 的矩形对角线长度为 $n$，对角线可以放 $\left\lfloor\dfrac{n+1}{2}\right\rfloor$ 个车。

然后这道题就做完了。

# 代码

```cpp
inline void solve()
{
	n=read(),k=read();

	if((n+1)/2<k)
	{
		puts("-1");
	}
	
	else
	{
		for(register int i=1;i<=n;i++,puts(""))
		{
			for(register int j=1;j<=n;j++)
			{
				if(i==j && i%2==1 && k>=1)
				{
					printf("R");

					k--;
				}

				else
				{
					printf(".");
				}
			}
		}
	}
}
```
