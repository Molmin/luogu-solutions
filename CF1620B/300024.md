这道题目考虑贪心求解。

面积最大的三角形可能有四种情况，这时三角形的底边分别在正方形的四条版上。

对于底边在四边形的某一条边的三角形来说，我们想让它的底边和高都尽可能的长。那么高明显应该取在底边所在的边的对边上的任意一点，底边应为相距最远的两个给定点中间的距离。

如果我们把底边取在四边形的某一条水平的边上，那么由于题目保证每条边上至少有一个给定点，那么这个三角形的高就是一个确定的 $h$，反之，如果三角形的底边在四边形的一条纤直边上，那么这个三角形的高就是 $w$。

更多的注释将在代码中给出。

```
#include<iostream>
#define int long long
using namespace std;
int p[10][200005]; 
signed main(){
	int t;
	cin>>t;
	while(t--){
		int w,h;
		cin>>w>>h;
		for(int i=1;i<=4;i++){
			cin>>p[i][0];
			for(int j=1;j<=p[i][0];j++) cin>>p[i][j];//输入每个点的坐标 
		}
		int x=(p[1][p[1][0]]-p[1][1])*h;//如果将底边取在y=0时的最大值 
		int y=(p[3][p[3][0]]-p[3][1])*w;//如果将底边取在y=h时的最大值 
		int z=(p[2][p[2][0]]-p[2][1])*h;//如果将底边取在x=0时的最大值 
		int q=(p[4][p[4][0]]-p[4][1])*w;//如果将底边取在x=w时的最大值 
		cout<<max(x,max(y,max(z,q)))<<endl;//综合输出 
	}
}
```
