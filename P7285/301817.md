# 1.题意
给定一个由$0$或$1$组成的数组。

每次可以使$0$变为$1$。

问要多少次使为数组中最长连续$1$子段的长度-修改的元素的个数最大。
# 2.思路
既然要让我们将$0$变为$1$。那么每一次由$0$变为$1$会增加几个连续的$1$呢？
## 1.最坏情况：
 $01100$
 
如，将最后的$0$变为$1$,增加$0$个，但可以让其他$2$个$0$变为$1$故不存在此情况

## 2.最好情况
 $11011$
 
将最后剩余的$0$变为$1$,增加2个

所以说不论如何，每次至少增加$1$,用$1$次操作增加$1$对于数组中最长连续$1$子段的长度-修改的元素的个数是没有影响的，所以直接输出全$1$的情况即可

# 3.代码
```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
	int x;
	cin>>x;
	while(x--){
		int ans=0;
		int y;
		cin>>y;
		int z=y;
		while(y--){
			int z;
			cin>>z;
			if(z==1)
			ans++;
		}
		cout<<ans<<endl;
		while(z--){
			cout<<1<<" ";
		}
		cout<<endl;
	}
	return 0;
}
```
