### - [题目传送门](https://www.luogu.com.cn/problem/CF1691A)

------------

### 题意

$T$ 组数据中，给你一个长 $n$ 的数列，可任意删除若干个数，让每两个相邻的数的和皆为偶数，问最少删去几个数。

### 思路

可将此数列皆变成一个全为奇数或偶数的数列，为什么呢？

因为两数相加，有三种可能。

- 偶 + 偶 = 偶
- 偶 + 奇 = 奇
- 奇 + 奇 = 偶

这样将全部数变成奇偶性全相同时，才能满足题目要求。之后再判断变奇数删除的少还是偶数的少，哪个少输出哪个。

### 代码
```cpp
//ji 为奇数个数
//ou 为偶数个数
while(T--){
	n=read();
   ji=0;ou=0;
   //注意清0
	for(int i=1;i<=n;i++){
		long long a;
		a=read();
		if(a%2==1)
			++ji;
		else ++ou;
	}
	printf("%lld\n",min(ji,ou));
}
```
