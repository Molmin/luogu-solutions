~~很水~~的一道题。
首先，我们观察每种摆放方式的不同之处，可以发现每个摆放方式的不同只有中间的一个菱形。所以，$n$的长度有多少，相对应的摆放方式就有几种不同。
```
 __________
/\/\/\/\/\/\
------------
\/\/\/\/\/\/
 -----------
```
如上，$n=6$，摆放方式就有$6$种。
所以，话不多说，上代码。
```
#include<bits/stdc++.h>//万能头
using namespace std;
int main()
{
  int a,b;
  cin>>a;
  for(int i=0;i<a;i++)
  {
     cin>>b;//输入
     cout<<b<<endl;//输出
  }return 0;
}
```
