首先，这个题只要会解分段计价应用题的都会解。


------------

我们分析一下题目：


------------

###### 原题（月用电量在150千瓦时及以下部分按每千瓦时0.4463元执行，在151~400千瓦时的部分按每千瓦时0.4663元执行，在401千瓦时及以上部分按每千瓦时0.5663元执行;要求整形输入，保留一位小数输出）


------------

意味着电价分三段：1-150，151-400，401-。三段价格不等，需进行分段计算(先把代码摆出来，C++）：
```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int main()
{
	int a;
	double b,c;
	cin>>a;
	if (a<151)
	{
		b=a*0.4463;
		printf("%.1lf",b);
	}
	if ((a>150)&&(a<401))
	{
		b=(a-150)*0.4663;
		c=150*0.4463+b;
		printf("%.1lf",c);
	}
	if (a>400)
	{
		b=(a-400)*0.5663;
		c=150*0.4463+(400-150)*0.4663+b;
		printf("%.1lf",c);
	}
	return 0;
 }
```
这里就是对价格进行了分段。
