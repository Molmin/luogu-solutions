看了一圈题解，再一次发现没有和我的做法一样的，就又来惹一发题解惹awa。

我们可以把电费的单价看作是一个分段的函数，把0~150千瓦时的单价0.4463设为基准价格，那么超过150千瓦时的就可以写成如下等式：

151~400千瓦时：W单价=W基准单价+W差价=0.4663=0.4463+0.02；

401千瓦时以上：W单价=W基准单价+W差价=0.5663=0.4663+0.1=0.4463+0.02+0.1

所以，后面计算我们就不必麻烦地去计算什么(a-150)和(a-400)的东东awa。

判断分段函数的等式如下：

```cpp
if (a<=150) ans=0.4463*a;
else if (a>150 && a<=400) ans=a*0.4463+0.02*(a-150);
else if (a>400) ans=a*0.4463+0.02*(a-150)+0.1*(a-400);
```

就这样，计算~~十分简单~~（我是这么认为的），最后再一个快乐的输出，但是输出时要注意保留一位小数！注意完了所有的细节，我们就快乐地AC了awa。

AC代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int a;
	scanf("%d",&a);
	double ans=0.0;
	if (a<=150) ans=0.4463*a;
	else if (a>150 && a<=400) ans=a*0.4463+0.02*(a-150);
	else if (a>400) ans=a*0.4463+0.02*(a-150)+0.1*(a-400);
	printf("%.1f\n",ans);
}
```
