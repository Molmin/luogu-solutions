最近在学Python，然而并没有AFO，所以想到了一个顽固Python语法的好方法：在luogu刷题

这题可以先判是否超过401，超过就把那部分计算掉，然后再判剩下的是否超过151，超过也把那部分计算掉，然后剩下的直接计算即可

```cpp
a=int(input()) #读入用电量
b=0.0		#储存答案的
if a>=401:		#判断是否超过400
	b+=(a-400)*0.5663	#计算超过部分钱数
	a=400				#减掉超过部分
if a>=151:
	b+=(a-150)*0.4663
	a=150
b+=a*0.4463
print('%.1f' %b)		#保留一位输出答案

```
