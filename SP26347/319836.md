**所有的结论题的正确做法，都从找规律开始，这题也不例外。**

我们将前几秒机器人的移动路线画出来，不难发现机器人在做周期为 $4$ 的螺旋移动。

我们设在第 $n$ 秒时，机器人位于 $(x_n, y_n)$ 处。

**下面对于 $n$ 模 $4$ 的余数分类讨论。**

若 $4|m$, 则有：

$$x_{n+4} = x_n+(n+1)-(n+3) = x_n-2$$

$$y_{n+4} = y_n+n-(n+2) = y_n-2$$

对于其他的余数也有类似式子。

于是我们根据模 4 的余数分类讨论，很容易就做出了这道题。

喜闻乐见的代码：

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
    int t,a;
    int x[4]={0,1,1,-2};
    int y[4]={0,0,2,2};
    int dx[4]={-2,2,2,-2};
    int dy[4]={-2,-2,2,2};
    cin>>t;
    while(t--){
        cin>>a;
        int q=a%4,r=a/4;
        cout<<x[q]+r*dx[q]<<" "<<y[q]+r*dy[q]<<endl;
    }
    return 0;
}
```
