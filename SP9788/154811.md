题目大意：

**Bob几乎一直使用社交网站。 他想知道那个社交网站里他有多少个朋友的朋友。 如果“X”是他的朋友，而“Y”是X的朋友而“Y”不是他的朋友，那么“Y”被称为他朋友的朋友。 你必须找到Bob有多少朋友的朋友。 （该社交网站中的每个用户都有一个唯一的4位ID号码）**

输入：

**第一行包含整数“n”（1 <= n <= 100）表示Bob的朋友数。 然后是行，每行中的第一个整数是Bob的朋友的ID号，然后输入整数“m”（1 <= m <= 100），代表他朋友列表中的朋友人数。 然后后面输入m个整数，表示他朋友列表中的朋友的ID号（不包括Bob）。**

输出：

**输出表示Bob朋友的朋友数量。**

------------
这道题只要逻辑思维稍微好一点就可以AC，详解见代码。

------------
```cpp
#include<bits/stdc++.h>
using namespace std;
struct px{//定义结构体，方便读入
	int a;//代表Bob的朋友的ID
	int b;//代表Bob这个朋友的朋友数
	int c[1001];//这个数组代表Bob这个朋友的b个朋友的ID
}x[1001];
int s=0,t[1000001];//s计数,t数组用来表示这个这个朋友有没有记录在内，因为Bob的n个好友中可能会有相同的朋友的朋友，避免重复计数
int main(){
	int n;//代表Bob的朋友的总数
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>x[i].a>>x[i].b;//输入Bob的i个朋友的ID和这个朋友的朋友总数
		t[x[i].a]=1;//因为是要求朋友的朋友，所以Bob的朋友不能计算在内，所以将其标记，避免计入总数
		for(int j=1;j<=x[i].b;j++)//循环读入Bob这个朋友的朋友的ID
		cin>>x[i].c[j];	
	}
	for(int i=1;i<=n;i++){
		for(int j=1;j<=x[i].b;j++){
				if(t[x[i].c[j]]==0)//如果Bob这个朋友的朋友没有被标记，也就是这个人并不是Bob的朋友也没有被计入过
				++s,t[x[i].c[j]]=1;//找到一个朋友的朋友，计入总数，同时将这个人标记，避免在接下来几个人中重复
		}
	}
	cout<<s;//输出
	return 0;//养成好习惯
}
```
