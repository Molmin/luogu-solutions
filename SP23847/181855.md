# 题目：两个新手和一个高手 或 两个高手和一个新手 可以组成一队，最多组成多少对？

额，这道题可以分为三种情况讨论


------------
### 1 高手*2 < 新手

因为每个高手最多和两个新手组队，所以这种情况下，
一定有新手不能与高手组队，且三个新手无法组队

#### 因此最大组队数等于高手数

可以证明没有更优的组队方法

------------
### 2 新手*2 < 高手

与第一种情况类似，新手分完时，多出高手无法组队

#### 最大组队数等于新手数

------------
### 3 其它情况

新手*2 >= 高手 且 高手 *2  >= 新手

这种情况下高手和新手都能刚好组完

#### 最大组队数即为(新手数+高手数)/3

------------

上代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int n,x,y;
	scanf("%d",&n);//输入样例组数
	for(int i=0;i<n;i++)
	{
		scanf("%d%d",&x,&y);//输入
		if(y>2*x) printf("%d\n",x);
        //情况一 输出高手数
        
		else if(x>2*y) printf("%d\n",y);
        //情况二 输出新手数
        
		else printf("%d\n",(x+y)/3);
        //情况三 输出(新手数+高手数)/3

	}
	return 0;
}

```
