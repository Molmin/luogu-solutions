这道题靠的就是数学和找规律，可以分为三种情况：

解：设高手数为 $x$ ，新手数为 $y$ 。

第一种，$x$ > $y$：

$$x \times 2 < y$$

所以是两个新手和一个高手配对。

高手，高手。

新手，新手，新手，新手，新手。

然后，由两个新手和一个高手配对，会发现多出来一个孤单的新手，于是此时最大配对数为 $x$ 。

第二种，$y$ > $x$：

$$y \times 2 < x$$

所以是两个高手和一个新手。

高手，高手，高手，高手，高手。

新手，新手。

然后，由一个新手和两个高手配对，会发现多出来一个孤单的高手，于是此时最大配对数为 $y$ 。

第三种，$x$ > $y$ 或者 $y$ < $x$：

$$x \times 2 \geq y$$ 
或者

$$y \times 2 \geq x$$

所以是两个高手带一个新手或者两个新手带一个高手。

高手，高手。

新手，新手，新手，新手。

或者

高手，高手，高手，高手。

新手，新手。

然后，由两个新手和一个高手配对，或者两个高手和一个新手配对，会发现刚好分完，于是此时最大配对数为:
$$\frac{x+y}{3}$$

总结出以上公式以后，便有了 _AC Code_ ：
```cpp
#include<bits/stdc++.h>
using namespace std;
long long x,g,n;          //x代表新手个数，g代表高手个数，n代表数据组数。
int main()
{
    scanf("%lld",&n); 
    for(int i = 0; i < n; i++) //for循环模拟
    {
        scanf("%lld%lld",&g,&x);  
        if(g * 2 < x)
        {
            printf("%lld\n", g);//情况一。
        }
        else if(g > x * 2)
        {
            printf("%lld\n", x);//情况二。
        }
        else
        {
            printf("%lld\n",(g + x) / 3);//情况三。
        }
    }
    return 0;
}
```
