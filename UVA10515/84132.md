**题意简译**

输入 $m$ 和 $n$，$1 \le m,n \le 10^{101}$，求 $m^n$ 的个位。

有多组数据，当 $m=n=0$ 时结束读入。

**题目分析**

“简单，用 `python` 直接求幂，再 $\bmod {10}$ 就好了。”

但是题目有多组数据，且时限只有 $3$ 秒，用 `python` 死算会 TLE。

我们知道，无论底数是什么，当底数相同指数不断增加时，幂的个位是以 $4$ 为周期的。

比如，底数为 $2$，指数由 $0$ 开始递增。

| 幂|$2^0$ | $2^1$| $2^2$ | $2^3$ | $2^4$ | $2^5$| $2^6$ | $2^7$ |…|
|:-:|:-:|:-:|:-:|:-:|:-:|:-:|:-:|:-:|:--:|
| **幂的个位** | $1$ | $2$ | $4$ | $8$ | $6$ | $2$ | $4$ | $8$ |**…**|

因为是求个位，所以只用考虑底数的个位（$m \bmod 10$），并按照周期计算（$(n-1) \bmod 4+1$），再进行幂运算并对 $10$ 取模就是答案。

当指数为 $0$ 时，需要特判输出 $1$。

不知道怎么高精取模？因为 `python` 自带高精，所以 `python` 题解参考：

```python
while True:
    s=input().split() #读入一整行
    if int(s[0])==0 and int(s[1])==0: #当输入为"0 0"时应跳出循环
        break
    elif int(s[1])==0:
        print("1",end="\n")
    else:
        s[0]=int(s[0])%10
        s[1]=(int(s[1])-1)%4+1
        print(int(s[0])**int(s[1])%10,end="\n")
```

---

我们再来考虑如何对底数和指数取模。

底数 $m \rightarrow m \bmod 10$，指数 $n \rightarrow (n-1) \bmod 4+1$。

模 $10$ 的话，取个位即可。

模 $4$ 可以看它的最后两位，也就是 **十位** 和 **个位** 组成的两位数对 $4$ 取模的结果就是这个数对 $4$ 取模的结果。

需要注意的是，算指数模 $4$ 的时候一定要 $+4$ 再模，避免出现十位和个位都是 $0$，也就是 $(0-1) \bmod 4+1=0$ 的情况。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int mod=10; //取个位，所以模数是10
string m,n;
int qpow(int a,int b) { //快速幂模板
    int t=1,y=a;
    while(b) {
        if(b&1) t=t*y%mod;
        y=y*y%mod;
        b>>=1;
    }
    return t%mod;
}
int main() {
    while (cin >>m>>n&&!(m=="0"&&n=="0"))
        printf("%d\n",n=="0"?1:qpow(m[m.size()-1]-'0',((n.size()>=2?(n[n.size()-2]-'0')*10:0)+(n[n.size()-1]-'0')+4-1)%4+1));
    //如果n=="0"，则直接输出1（特判m的0次方）
    //否则，则计算 (m%10)^[(十位+个位+4-1)%4+1]
    return 0;
}
```
