## 思路
一个区间中的按位或（$a|b$）的和大于等于区间长度即可。在最坏情况下，也是珂以等于滴。毕竟这只是CF Div.2的A题，还为了引起大家打比赛的兴趣，出题人特意设置了 SPJ，不需要考虑字典序之类的繁杂问题，所以任意输出一个排列就行。为了简单考虑就直接输出 $1$ 到 $n$ 好了。

## CODE
```cpp
#include <stdio.h>
int main(void) {
    int t;
    for (scanf("%d", &t); t--; ) {
        int n;
        scanf("%d", &n);
        for (int i = 1; i <= n; ++i)
            printf("%d ", i);
        puts("");
    }
    return 0;
}
```
