# 有一点点小小的心思
首先这道题其实不难，最重要的是要把控好几个细节，先说一下思路：

我们先输入，然后**从尾到头**查找，如果发现相同，输出YES，没有则输出NO

那么我在这里用了两个小心思：

1. **从尾到头**：理论上来说肯定是如果前面有出现过，一般情况下从尾到头的速度会快一点，当然不排除特殊情况，不过倒着搜的好处是如果出现过三次及以上有好处。

1. **如何跳出循环？？？**

我们先看一下这一段循环的代码：
```cpp
for(j=i-1;j>=1;j--){
	if(st[i]==st[j]){
		cout<<"YES"<<endl;
		j=-1;
	}
}
if(j==0)cout<<"NO"<<endl;
```
这个j的取值很有讲究：首先我们知道**for语句的精髓就在于变量j的变化**，如果就j<=0那么就结束，那么此时**j必定为0**，**不可能再出现-1的情况**。那么如果j==0就一定查找不到，输出NO，但如果j==-1，既可以直接跳出循环，又可以不再重复输出NO，一举两得！！！

那么完整代码就出来了：
```cpp
#include <iostream>
#include <string>
using namespace std;
string st[10001];
int main(){
	int i,j,k,n;
	cin>>n;
	for(i=1;i<=n;i++){
		cin>>st[i];
		for(j=i-1;j>=1;j--){//小心思1
			if(st[i]==st[j]){
				cout<<"YES"<<endl;
				j=-1;//小心思2
			}
		}
		if(j==0)cout<<"NO"<<endl;//小心思2
	}
	return 0;
}
```
