# 题目大意

首先，给你一个 $N\times N$ 的矩阵 $a$。

接下来，有 $q$ 次操作，有两种。

第一种是从 $(a,b)$ 到 $(x,y)$ 这个矩阵里每个元素都加上一个 $v$。

第二种是再给你一个矩阵 $f$，判断这个矩阵 $f$ 是否是原矩阵 $a$ 的矩阵乘法。

我先来解释一下矩阵乘法是什么，$a$ 的矩阵乘法用符号表示就是下面这样。

$\mathit{d}_{i,j}=\sum\limits_{k=1}^n\mathit{a}_{i,k}\times\mathit{a}_{k,j}$

注：$\mathit{d}_{i,j}$ 为矩阵 $a$ 的矩阵乘法。

而用代码则是以下这样的。

```cpp
for(int i=1;i<=n;i++)
	for(int j=1;j<=n;j++){
    	long long ans=0;
		for(int k=1;k<=n;k++)
			ans+=a[i][k]*a[k][j];
        d[i][j]=ans;
    }
```


# 思路

首先，我们先将矩阵 $a$ 输入进来。

再读入 $q$ 次操作。

如果为第一次操作，则按正常直接暴力累加。

若为第二次操作，则就用矩阵乘法来得到 $d$，也就是 $a$ 的矩阵乘法。然后，每得到一个 $\mathit{d}_{i,j}$，我们就判断一下矩阵 $f$ 的值是否相同。如果相同，则继续算 $d$。如果不同，则输出 ```no```，退出循环。

当全都遍历了一遍后，都还没有不等于 $f$ 的，就输出 ```yes```。
