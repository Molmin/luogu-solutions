### 审题+思考
题目要求选**一段连续的区间**，所以如果同一个高度，又在相连的区间，那么一次操作就可以搞定

如果高度相同的位置出现缺口，那么就要增加一次操作

### 答案就在眼前
于是我们要怎么实现这个操作呢？

从左往右读入数据，模拟从左往右增高的过程

如果当前位置的高度大于之前位置的高度，那么增加高度之差次操作

如果等于或低于则不用操作

OK，我们看代码
```cpp
#include<iostream>
using namespace std;
int main(){
	int n,s=0;//s为计数器
	int h0=0,h1;//h0前次的高度，h1为当前高度
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>h1;
		if(h1>h0)//如果大于则增加操作
			s+=h1-h0;
		h0=h1;	//更新当前高度
	}
	cout<<s;//输出
	return 0;
}
```
