依照题意输出即可：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main()
{
    double f;
    scanf("%lf", &f);
    printf("%f\n%.5f\n%e\n%g", f, f, f, f);
    return 0;
}
```
```double``` 类型数读入站位符是 ```%lf```、```%le``` 或 ```%lg```（效果完全相同），而输出占位符是 ```%f```、```%e``` 或 ```%g```，和 ```float``` 一样。

在 C++ 或 C99 中 ```double``` 占位符可以是 ```%lf```、```%le``` 或 ```%lg```，和不加 ```l``` 没有任何区别。
- ```%f```：直接输出（不是指数形式），一般保留小数点后 $6$ 位，要保留小数点后 $5$ 位就用 ```%.5f```。
- ```%e```：以指数形式（科学计数法）输出，见样例。
- ```%g```：根据情况输出简单的，**有可能**以指数形式输出。



这题其实挺玄学的——语言标准中并没有规定 ```%e``` 具体的指数位数，所以同一份代码评测结果可能不同（话说没有 SPJ 的吗……）。代码多交几次就过了。
