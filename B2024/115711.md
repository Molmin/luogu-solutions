这道题本质上是 `printf` 中输出浮点数时各种格式符的比较。

`%f` 是最常见的浮点数格式符，使用时可以在 "f" 前添加 ".x" 进行修饰，其中 x 代表保留小数点后的位数。`%f` 一般用于 `float` 型的输出，而 `double` 型通常使用 `%lf`。

`%e` 以指数形式输出，具体可以参照样例，输出格式大体为 "a e±x",其中 a 的绝对值大于 1 且小于 10，x 是科学计数法中 10 的幂次的绝对值。

`%g` 则在输出时自动选择前两种格式输出时宽度小（即输出字符数小）的那种格式输出，在输出时不输出无意义的 0。

```cpp
#include<stdio.h>
int main()
{
    double a;
    scanf("%lf",&a);
    printf("%f\n%.5f\n%e\n%g",a,a,a,a);
    return 0;
}
```
