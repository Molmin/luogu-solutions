我们观察一下题目可以发现：2由2^1个1组成；3由2个2，也就是2^2个1组成；4由2个3，也就是2^3个1组成……我们发现2^i个小史莱姆可以组合成一个权值为i+1的大史莱姆。比如输入n=4，4=2^2，那这个史莱姆的权值就是2+1=3。

那算法也就很清晰了：输入n以后，每次找到一个不大于n的，形如2^i的数，就输出i+1即可,再从n中减去这一部分即可。

```cpp
#include<cstdio>
using namespace std;
int n;
int main()
{
	scanf("%d",&n);
	while(n)//只要还有史莱姆
	{
		int i=0,x=1;//i表示次数
		while(x<=n) x=x<<1,i++;//找到比n小的最大的2^i
		printf("%d ",i);//输出
		n-=x>>1;//减去这一部分
	}
	return 0;
}
```
