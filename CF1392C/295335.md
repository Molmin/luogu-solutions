三倍经验（P5019 P1969）

贪心： 
$$ans=\sum_{i=2}^n max(0,a_{i-1}-a_i)$$

证明：

* $a_{i-1}<=a_i$ 不用管，增加反而会答案增加

* $a_{i-1}>a_i$ 直接拔高 $i$ 及 $i$ 以后的所有数，使此后每次被拔高的代价一定不大于相邻两数之差

```
		cin>>n;ans=0;
		for(int i=1;i<=n;i++){
			cin>>a[i];
			if(i>1&&a[i]<a[i-1])ans+=a[i-1]-a[i];
		}
		cout<<ans<<endl;
```
