## 题目
已知三个正整数 $a,b,c$，现有一个大于 $1$ 的整数 $x$，将其作为除数分别除 $a,b,c$，得到的余数相同。

## 解决方案
解决方案：将 $x$ 作为除数分别除 $a,b,c$，得到的余数相同，其实就是这句代码的意思：
```
a%x==b%x&&c%x==b%x
```
注意不能连等！

现在我们考虑数据范围，$a,b,c$ 均不大于 $1000000$.如果跑一个 $2$ 到 $1000000$ 的循环，完全没问题。但会不会担心跑出 $1000000$ 炸掉呢，完全不用！我们假设 $x$ 大于 $a,b,c$ 三个数。那么 $a,b,c$ 除以 $x$ 余数分别为它们三个自己，而题目要求这三个余数相等，所以 $a,b,c$ 就相等了。既然 $a,b,c$ 相等，那么 $a,b,c$ 除以任何大于等于 $2$ 且小于等于 $a,b,c$ 三个数的数余数皆相等，这些数中最小的就是题目中的 $x$ ，所以 $x$ 大于 $a,b,c$ 三个数的假设不可能成立！因此，放心大胆地枚举吧！

## 代码
```
#include<bits/stdc++.h>
using namespace std;
int main(){
	int a,b,c;
	cin>>a>>b>>c;
	for(int x=2;;x++){//两个分号中间是循环进行的条件，既然无条件进行，那就空着不写，但注意，必须是两个分号。
		if(a%x==b%x&&c%x==b%x){
			cout<<x;
			return 0;//找到了满足条件的x，一定要立即结束程序或跳出循环！
		}
	}
	return 0;
}
```
