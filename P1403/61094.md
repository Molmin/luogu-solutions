正经：这个题，乍一看   直接求因数有一点点麻烦.
So：我们倒~着瞅———对于每个数i，在1到n的范围内有多少个数是它的倍数？~~~~答案显然是n/i。于是最终的答案就是∑(n/i)。（嘿，划呀划）附上代码：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main()
{
    int n;
    while(cin>>n)
    {
        int sum=0;
        for(int i=1;i<=n;i++)
            sum+=n/i;
        cout<<sum<<endl;
    }
    return 0;
}
```
