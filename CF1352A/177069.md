# **1.题意理解**

所谓将一个正整数用圆数和表示，就是将此正整数进行10进制分解表示。

举例：对于2061，它可以用3个圆数表示（三个位数上的数字不为0），依次是（从小到大来）1*10^0+6*10^1+2*10^3，也就是1,60,2000.


------------


# **2.算法部分**

**首先**，对于每一个题目输入的正整数，我们要获取这个正整数有多少位不为0，我们可以采用从小到大扫一遍的方式：

判断当前数的个位数是否为0，如果不为0，就进行答案累加；

将当前数除以10，相当于自动抹去最后一位；

如此重复，直到获取所有位的情况为止。“当前数”的初始值为数据所给的数。


------------

**然后**是以任意顺序输出圆数。既然顺序没有要求，我们可以利用上面的方法，从小到大获取当前位上数的大小，然后乘以10^n即可。10^n可以随着每一次当前数除以10时乘以10。

------------


# code：

```cpp
#include<cstdio>
using namespace std;
int read()//本人祖传快读
{
	int ans=0;
	char ch=getchar(),last=' ';
	while(ch>'9'||ch<'0')last=ch,ch=getchar();
	while(ch>='0'&&ch<='9')ans=(ans<<3)+(ans<<1)+ch-'0',ch=getchar();
	return last=='-'?-ans:ans;
}
int t,n,num,now,k;//t为数据组数，n为当前数，num为圆数个数，now为10^n累乘器，k是当前数的个位数
int main(){
	t=read();
	while(t--)
	{
		now=1;
		num=0;
		n=read();
		int n1=n;
		while(n1)//n在下一个函数进行操作，就不劳烦n了，copy了个n1
		{
			k=n1%10;//获取个位数
			if(k)num++;//如果非0，圆数个数++
			n1/=10;//考虑下一位
		}
		printf("%d\n",num);
		while(n)
		{
			k=n%10;//同上
			if(k)//不为0
			{
				printf("%d ",k*now);//当前数个位数乘10^n，还有个小空格
			}
			now=(now<<3)+(now<<1);//相当于now*10，但是位运算更快我就采用了
			n/=10;
		}
		printf("\n");
	}
}
```
