## 1. 思路

### 2.找规律

发现了没有？这是不是跟刚学循环时把数字分离下来很像？

我们总结一下：找到 $n$ 的非零数位，将它在后面用 $0$ 补足。

## 2.编代码
### 1.思路
|变量|方法|
|:----:|:----------:|
|$k$|找到不是 $0$ 的数字的个数|
|$k$ 个圆数|找到不是 $0$ 的数字，用 $0$ 补足后输出|
那么要补多少个 $0$ 呢？

它后面有多少个数字，就补几个 $0$

对于第 $i$ 位，它后面有：字符串的长度-它前面的数字数量-1（还有它本身）=`s.length()-i-1`（C++字符串是从 $0$ 开始的，所以它前面有 $i$ 个数字）个数字

这题就这么解决了。
### 2.代码
AC代码：
```
#include<bits/stdc++.h>
using namespace std;
long long T,sum;
string s;
int main(){//前面不必解释了
	scanf("%lld",&T);
	getchar();//读入行末换行符
	while (T--){//相当于for (int k=0;k<T;k++)
		sum=0;//归零
		getline(cin,s); //读入字符串
		for (int i=0;i<s.length();i++)
			if (s[i]!='0') sum++;//找到不是0的数字个数
		printf("%lld\n",sum);//输出
		for (int i=0;i<s.length();i++)
			if (s[i]!='0'){//找到不是0的数字
				putchar(s[i]);
				for (int j=0;j<s.length()-i-1;j++) putchar('0');//补s.length()-i-1个0
				putchar(' ');
			}
		putchar('\n');
	}
	return 0;//功德圆满
}
```
