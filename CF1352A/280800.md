来句闲话：虽然是入门题，但是翻了翻题解，这个代码还算最简短的(/w\ )

题目可以理解为：给你一个数，将这个数表示为：
$$A_1\times10^{k_1}+A_2\times10^{k_2}+A_3\times10^{k_3}...$$
且要使得这个多项式的项数尽可能少。

---

因为这个数的每一位至少要一项，且**最优情况下**仅需要一项就能表示，举个例子：

$1234$可表示为
$$
1234=1000+200+30+4
$$
又或者
$$
1234=1000+100+100+10+10+10+1+1+1+1
$$
很明显可以看到，第一种表示方法肯定是最优的，接下来的难点，怎么输出。

---

因为不用考虑输出的顺序，所以可以考虑从末尾开始输出，每次对$10$取模，取出这个数的个位，然后再把这位数字对应的$0$输出。
```cpp
int cnt=0;
while(n){
	int x=n%10;
	n/=10;
	if(x){
		printf("%d",x);
		for(int i=1;i<=cnt;++i)
		printf("1");
		printf(" ");
	}
	++cnt;
}
```

---

这里的数字如果当做字符串处理会更快速且适用范围更广（可以处理高精度数字），这里放出伪代码：
```cpp
读入字符串
for(从个位到最高位枚举字符串位置){
	如果这位数字不为0，输出，然后把0加上
}
```
