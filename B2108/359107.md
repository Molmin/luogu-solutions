### 题意：

输入一个 $n \times m$ 的矩阵，对它进行如下操作：

* 对矩阵四周最外侧的数不作任何特殊处理；

* 将矩阵中间的数取上、下、左、右及自己的平均数，取四舍五入的结果。

---


### 分析：

我们定义了两个数组，一个用来存放原始矩阵，一个用来存放操作后的矩阵。

拿样例举个例子：

| i\j | 1 | 2 | 3 | 4 | 5 |
| :----------: | :----------: | :----------: | :----------: | :----------: | :----------: |
| 1 | 100 | 0 | 100 | 0 | 50 |
| 2 | 50 | 100 | 200 | 0 | 0 |
| 3 | 50 | 50 | 100 | 100 | 200 |
| 4 | 100 | 100 | 50 | 50 | 100 |


分析矩阵，我们可以发现，第一行的 $i$ 都为 $1$ ，第一列的 $j$ 都为 $1$ ，最后一行的 $i$ 都为 $n$ ，左后一列的 $j$ 都为 $m$ 。

那么根据以上结论进行判断是否在四周即可。如果是，就直接将其赋值到另一个数组中，如果不是就将其进行操作。

题中还说了要四舍五入，那么我们就要用到一个函数 ```round()```
 ，它需要调用 ```cmath``` 库。

---

是不是很简单，代码实现它吧。

```cpp
#include<iostream>
#include<cmath>
using namespace std;
double a[101][101],b[101][101];  //b 数组为处理后的数组
int main(){
	int n,m;
	cin>>n>>m;
	for(int i=1;i<=n;i++){
		for(int j=1;j<=m;j++){
			cin>>a[i][j];
		}
	}
	for(int i=1;i<=n;i++){
		for(int j=1;j<=m;j++){
			if(i==1||j==1||i==n||j==m){  // 如果在边缘
				b[i][j]=a[i][j];  //直接赋值
			}
			else{
				b[i][j]=round((a[i-1][j]+a[i+1][j]+a[i][j-1]+a[i][j+1]+a[i][j])/5.0);  // 否则求平均值
			}
		}
	}
	for(int i=1;i<=n;i++){
		for(int j=1;j<=m;j++){
			cout<<b[i][j]<<' ';
		}
		cout<<endl;
	}
	return 0;
}
```
