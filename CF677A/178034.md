**一、输入部分，输入n和k（~~废话，各位大佬可以无视~~）**

```
cin>>n>>k;//输入
```
**二、主体部分，从零到n-1遍历输入，判断**
```
	for(int i=0;i<n;i++)//从零开始输入 
	{
		cin>>a;//输入
```

最后贴上完整代码

```
#include<iostream>
using namespace std;
int a,n,ans,k;
int main()
{
	cin>>n>>k;//输入 
	for(int i=0;i<n;i++)//从零开始输入 
	{
		cin>>a;//输入 
		if(a>k)ans++;//如果大于了那个数，就额外加一，因为无论多高，最少加一，少加的在后面那条语句补全 
		ans++;//每个人都加1 
	}
	cout<<ans;//输出 
	return 0;//完美结束 
} 
```
