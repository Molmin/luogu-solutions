# 题解 CF1140A 【Detective Book】

###### ~~大家好，本奆蒟蒻又来写题解了~~~~~


------------


我来把题干明了地翻译给大家，比题目中翻译的会更详细。

**伊万在读一本书，每一页上都有一个题，题的答案在第 $ai$ 页，伊万每看到一个谜题，必须读到它的答案那一页才会停止，当然，答案的那一页也会有谜题，他还要继续看到它的答案，直到读完迷题的答案且不会再有新的谜题才会睡觉，进入下一天，继续这样读书。**


------------


怎么样，是不是明了了许多？

我这一用了一个变量 $zdy$ ，即读得最多一页，又即读到的那一页。

如果 $ai$ 大于了这个 $zdy$ ，说明了还没有读到答案的那一页

将 $ai$ 赋给 $zdy$ ，也就是读到了那一页。

当 $i$ 大于 $zdy$ 时，说明了伊万读到了谜底的那一页，且不再读新的一页了，就可以让新的一天来到了。

话不多说，上代码

## $Ac$ $Code$

```cpp
#include<iostream>
using namespace std; 
int main()
{
    int n,zdy=-1,day=0,y;  //zdy要比较小，要比ai小
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        cin>>y;  //y即ai
        if(y>zdy) zdy=y;
        if(i>=zdy) day++;  //读完书了，增加一天
    }
    cout<<day;
    return 0;
}
```
