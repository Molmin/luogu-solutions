## 题目简述：

你有三堆石子，分别有a,b,c个，每次你可以有两个操作： 1:从第一堆取1个，第二堆取2个 2：从第二堆取1个，第三堆取2个。

最后最多可以得到多少个石子。

## 分析
1：从第一堆取1个，****第二堆取2个****。

2：****从第二堆取1个****，第三堆取2个。

通过看题我们可以发现，无论是操作一还是二，都是要用到第二堆里的石头的，并且操作一和操作二每次都共取三块石头。

所以我们要尽可能地少取第二堆里的石头，这样可以使取石头的次数最多，最终取到的数量也最多。

∵2>1

∴要尽可能多地进行操作二，直到无法进行操作二，再进行操作一。

## 代码如下

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int a,b,c,t,ans;
    cin>>t;//多组数据
    while(t--){
    ans=0;
        cin>>a>>b>>c;//分别输入三堆石头的个数
        while(b>=1&&c>=2)//判断能否进行操作二
        { 
            ans+=3;b-=1;c-=2;
        }//如果能进行操作二，优先进行操作二
        //不能就进行操作一
        while(b>=2&&a>=1)//判断能否进行操作一
        {
            ans+=3;b-=2;a-=1;
        }
        cout<<ans<<endl;//不要忘了换行
    }
  return 0;
}
```
