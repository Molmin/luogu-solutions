[题目传送门](https://www.luogu.com.cn/problem/CF1236A)

### 分析：
1. 首先仔细审题。不管是选择方案一还是方案二，都会取走第二堆里面的石头。
1. 如果要得到更多的石头，那么取的次数就要尽可能的多。

------------

### 思路以及做法：
1. 通过上面的分析，不难看出，第二堆是最核心的。
1. 必须得尽量选择第二种方案，直到第二种方案没得选择。
1. 如果第二种方案没得选择的话，那么就去选择第一种方案，直到第一种方案也没得选择，就输出。

我们一步一步来实现：

1. 选择第二种方案，直到第二种方案没得选择。

这一步并不难，只要用 ```while``` 语句来实现就可以了。
```
while(b-1>=0&&c-2>=0){
    sum+=3;
    b-=1;
    c-=2;
}
```
2. 第二种方案都选择完后就选择第一种方案。如果第一种方案也没得选择的话，就输出累加器里的值。

其实代码跟上一步没什么区别，只需要注意累加器要清零。
```
while(a-1>=0&&b-2>=0){
    sum+=3;
    a-=1;
    b-=2;
}
cout << sum << endl;
sum=0;
```

------------

### 完整 ```AC``` 代码:
```
#include<iostream>
using namespace std;
int n;
int a,b,c;
int sum;
int main()
{
    cin >> n;
    for(int i=1;i<=n;i++){
        cin >> a >> b >> c;
        while(b-1>=0&&c-2>=0){
            sum+=3;
            b-=1;
            c-=2;
        }
        while(a-1>=0&&b-2>=0){
            sum+=3;
            a-=1;
            b-=2;
        }
        cout << sum << endl;
        sum=0;
    }
    return 0;
}
```
