 这道题的思路是：有三种可能（兔子和猫头鹰，兔子和驴,先去驴/猫头鹰再从驴和猫头鹰徘徊）下面说明：兔子=a；猫头鹰=b；驴=c；
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a,b,c;
int main()
{
	cin>>n>>a>>b>>c;
	cout<<max(min(min(a,b)*(n-1),min(a,b)+c*(n-2)),0)<<endl;//把这几句拆开，min(a,b)*(n-1)是判断出小熊在第一条与第二条路那个近，就在那个徘徊，min（a,b)+c*(n-2)是第三条路（见上），因为n会-1，所以有可能会出现负数，所以要以0作最低结果
	return 0;
}
```
