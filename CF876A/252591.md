这道题有一个小坑点：

因为小熊开始是在兔子家吃蜂蜜，所以吃蜂蜜的数量要减去1.

这个坑点找到之后，我们就可以写代码了

我先输入吃蜂蜜个数，再输入三个数，三个数为数组。接下来排序，找到前两个最短路径。然后循环蜂蜜个数减1，最后输出。

代码如下：
```
#include <bits/stdc++.h>
using namespace std; 
int hyf[10];
int main() 
{
    long long n,n1,m1,m=0,j,k,i,t=0;
    cin>>n>>hyf[1]>>hyf[2]>>hyf[3];//输入
	sort(hyf+1,hyf+1+3);//排序，使hyf[1]和hyf[2]为最短的两条路径
	for(i=1;i<=n-1;i++)//循环，注意要减1
	{
		if(i%2==1)//考虑是奇数的情况
		t+=hyf[1];
		else//考虑是偶数的情况
		t+=hyf[2];
	} 
	cout<<t;//输出
}
```
