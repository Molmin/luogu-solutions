## 题意：
1. 有两条道路$h_i$和$v_i$， 一共铺了$n^2$天,

2. 如果这一天有一条道路被铺过了，就直接不铺，偷懒

## 思路：

根据题意，一共有两条道路，使用两个数组$a_i$和$b_i$来标记这一段路有没有铺过

模拟每一天， 如果计划铺的路面没铺过就输出$i$，并将该路段标记为$1$

铺了的话啥也不干，偷懒


## AC Code:
```cpp
#include <iostream>
#include <algorithm>
using namespace std;
bool a[100000], b[100000];//创建两个数组标记该路段有没有铺过
int main() {
	int n;
	cin >> n;
	for (int i = 1; i <= n * n; i++) {
		int x1, x2;
		cin >> x1 >> x2;//输入计划铺的路段
		if (!a[x1] && !b[x2]) {//如果计划铺的路段都没有被铺过
			a[x1] = 1;//标记为1
			b[x2] = 1;//标记为1
			cout << i << " ";//输出这是第几天
		}
	}
	return 0;
}
```
