其实就是P1001嘛，输出a+b的值。

没什么好说的，就是提醒各位新手要注意细节（但我觉得应该没什么新手会翻到这道题。。。）

我的代码没什么特殊的：

```cpp
#include<bits/stdc++.h>//万能头文件！ 
using namespace std;
int a,b;//定义变量，10的3次方，int完全够！ 
int main()
{
	ios::sync_with_stdio(false);//此题没啥用的输入输出加速器。 
	cin>>a>>b;//输入 
	cout<<a+b<<endl;//输出，endl换行。 
	return 0;//程序结束返回0，养成好习惯。 
 }
```
基础简单的题目，但不知为何还有那么多人错。。。
