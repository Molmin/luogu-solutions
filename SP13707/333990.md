## Part1 冷静分析

分析一下它的运作过程：

输入一个点 -> 左右都有人站着/左右都没人站着 -> 与其合并到一个队伍中/形成一个新的队伍。

那么我们就可以用数组解决这个问题。

- 都有人站着时会把两队连成一个队伍，队伍数量会减 $1$；

- 都没人站着时，就会形成一个新的队伍，队伍数量会加 $1$。

- 左右只有一边有队伍的话，将会新产生一个队伍，同时减少一个队伍，跟原来的队伍数量一样

注意一个点：**题目末尾要换行！！！** 我因为这一点被卡了四次……

## Part2 吊打红题

```cpp
#include<bits/stdc++.h>
using namespace std;
int n; //这个是队伍总人数
int q[20005]; //这个就是队伍的数组
int que; //这个是目前队伍的数量
void print(){printf("Justice\n");} //防作弊函数，普通人基本不会这么写
int main(){
	scanf("%d", &n);
	for(int i = 1;i <= n;i++){
		int now;
		scanf("%d", &now);
		q[now] = 1; //这个位置有1个人站着了
		int qnum = q[now - 1] + q[now + 1]; //计算该人左右的人数
		if(!qnum) //如果他左右都没有人，就自己构成新队伍
			que++;
		else if(qnum == 2) //如果左右都有人，他就把这两个队伍连起来了，会新产生一个队伍，同时减少两个队伍，共减少一个队伍
			que--;
		else;
		printf("%d\n", que); //记得换行
	}
	print(); //这里也是要换行的
	return 0;
}
```
