# 题目大意

给定若干段文章（每段文章以一行以 `#` 为开头字母结束，这一行不算在文章内），让你统计文章中所有单词出现的次数（每个单词长度不超过 $30$）。

单词的定义：

- 任意两个单词之间由若干个分隔符（包括空格、问号、感叹号、逗号和句号）分隔；

- 带撇号的单词，如 `I'm` 和 `You've`，以及带有连字符的单词被视为一个单词，并且撇号、连字符不算在单词长度里。例如 `you've` 是一个5个字母的单词；

- 连字符可以出现在一行的末尾（保证一定不是文章的最后一行），表示连接该行的最后一个字符和下一行的第一个字符（例如样例中的第 $1$ 至 $2$ 行），并且其他情况下换行符都有着分隔符的作用。

在每组数据输出完后输出一个空行。

# 做法

搞一个桶 $a$ 记录单词长度出现的次数和一个计数器 $num$ 记录当前单词长度。

如果遇到了一个普通字符，那么直接让 $num=num+1$；

如果遇到了分隔符，那么把 $num$ 统计进 $a$ 里面，即 $a_{i}=a_{i}+1$；

如果遇到了 `#` 号，那么就遍历一遍 $[1,30]$，判断 $a_{i}$ 是否为 $0$ 并输出。

# 难点

对应 单词的定义 里面的第三条，怎么特判连字符出现在末尾的情况呢？

观察发现，若连字符出现在最后，那么下一个字符必然是换行符，所以可以搞一个字符 $lc$ 表示上一个字符，每输入进来一个字符 $c$ ，在判断分隔符时特判一下就行了，而且需要注意在判断普通字符时也要特判不等于换行符。

# AC Code

```c++
#include<bits/stdc++.h>
using namespace std;
int a[35], num;
char c, lc;
signed main()
{
#ifndef ONLINE_JUDGE
	freopen("data1.in", "r", stdin);
	//freopen("data1.out", "w", stdout);
#endif
	while(~scanf("%c", &c))
	{
		if(c == '#')
		{
			a[num]++;
			num = 0;
			for(int i = 1; i <= 30; i++)
				if(a[i])
					printf("%d %d\n", i, a[i]);
			puts("");
			while((c = getchar()) != '\n');
			memset(a, 0, sizeof a);
		}
		else if(c == ',' || c == '.' || c == '?' || c == '!' || c == ' ' || (c == '\n' && lc != '-'))
			a[num]++, num = 0;
		else if(c != '\n' && c != '-' && c != '\'')
			num++;
		lc = c;
	}
	return 0;
}
```
