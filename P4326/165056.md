### 同志们，有样例啊！

- 第二组是输入21输出882.000000，恩，没错就是n方乘2。
- 关系有了就可以敲代码了。
#### 代码如下
```cpp
#include<bits/stdc++.h>
using namespace std;
const double Pi=3.1415926535;
double n,nn,s1,s2;
int main()
{
	cin>>n;
	nn=pow(n,2);//蒟蒻装大佬，用个函数方便。 
	s1=nn*Pi;//欧氏圆的面积
	s2=nn*2.0;//出租车圆的面积
	printf("%0.6lf\n",s1);//cout也能实现小数点数位输出，不过比较麻烦，还是用标准化吧。
	printf("%0.6lf\n",s2);
	return 0;//结束
}
```
