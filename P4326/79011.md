## 一篇比较注重讲解的题解

这一篇题解采用**解析几何**的思路。


无论是欧氏几何还是什么鬼的出租车几何，关于圆的定义都是到定点的**距离**等于定长的点的集合。但是，两个几何体系不一样的地方在于，这个定义中的“距离”不同。

在欧氏几何下，两点间的距离为$\sqrt{(x_1-x_2)^2+(y_1-y_2)^2}$

设一个点$(x,y)$,它到原点$(0,0)$的距离为$\sqrt{(x-0)^2+(y-0)^2}$

即$\sqrt{x^2+y^2}$,根据圆的定义，我们令上式等于定长$R$，两边平方

可得出圆的方程为$x^2+y^2=R^2$。

![](https://cdn.luogu.com.cn/upload/pic/52863.png)

所以欧氏几何下圆的图像是这样的。

欧氏几何下，圆的面积是$S=πr^2$。它实际上是用了微积分的思想。把组成圆的曲线无限分割，分割成极其微小的段，那么圆面积就近似于无数个弧长为圆周长曲线的小扇形的拼接。而弧长的长度之和就是圆的周长$2πr$。扇形的面积公式是$S=\frac{1}{2}lr$，用周长换掉扇形面积公式里的$l$，可得公式：

$S=πr^2$

然而出租车几何下圆并不是这样的。虽然都是到定点的距离等于定长，但是距离的定义不一样了，圆的形状也要发生改变。

出租车几何下，距离公式是$|x_1-x_2|+|y_1-y_2|$。同样，我们可以求出它的标准方程。

设一个点$(x,y)$，在出租车几何下，它到原点$(0,0)$的距离是$|x-0|+|y-0|$，即$|x|+|y|$。我们令上式等于定长$R$，可得公式$|x|+|y|=R$。这就是出租车几何下圆的标准方程。

那么我们把它画出来吧。

在四个象限中，分别设：

x为正，y为正，得$x+y=R$

x为正，y为负，得$x-y=R$

x为负，y为负，得$-x-y=R$

x为负，y为正，得$-x+y=R$

随便设一个$R$，然后描点连线，这时我们发现，这个“圆”的样子居然是······

![](https://cdn.luogu.com.cn/upload/pic/52865.png)

### 正！方！形？！

没错，根据我们逻辑严密的完美推断（雾），出租车几何下的圆，就是这个鬼畜的样子。

那么这个“圆”的面积公式很简单就能推导出来了。这个正方形的面积可以看做上下两个三角形面积的和。我们知道的是对角线的一半长度$r$。一个三角形的面积是$\frac{1}{2}( 2r*r)=r^2$，两个三角形的面积当然就是$2r^2$了。

那么我们现在就可以华丽地打出完美的代码咯~

### ちょっと待って !

还有最后一个小问题，就是精度的问题。方法再正确，$π$的精度如果不对，这题也要WA。

其实，math库里已经给我们定义好了常量M_PI作为$π$。如果不想用，还可以直接define，但是要注意精确度一定要高。

下面是代码。


```
#include<cstdio>
#include<string>
#include<iostream>
#include<algorithm>
#include<cmath>
#include<vector>
using namespace std;
int r;
double s1,s2;
int main()
{
	cin>>r;
	s1=M_PI*r*r;
	s2=2*r*r;
	printf("%.6lf\n%.6lf",s1,s2);//取小数点后6位用printf比较方便 
	return 0;
}
```
