其实呢，这个题首先要把π定义好，~~千万不要定义~~常知的3.1415926，我试过，误差很大，应该只过第一个点，别的WA，所以我定义的是

```cpp
pai=3.14159265358979323;//一定定义大一点，计算更接近正确答案
```
 说实话，比起其他几位大佬(其实各位大佬都解释的很详细了（懒~） )，我发现规律的方法真的好玩，我就用计算器把结果除以r，没想到出来了r*2（暂且认为我运气好！！）。
#### code：
```cpp
#include<iostream>//不建议万能头
#include<cstdio>
using namespace std;
int main()
{
	double r,pai;//一定定义成double
	cin>>r;
	pai=3.14159265358979323;一定定义大一点，计算更接近正确答案
	double S1=r*r*pai;//欧几里得计算面积
	double S2=r*r*2.000000;//非欧几里得
	printf("%.6lf\n",S1);//完美输出
	printf("%.6lf",S2);//输出！
	return 0;//完美の结束
}
```
