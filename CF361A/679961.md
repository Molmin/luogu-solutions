# 题意简述

给定 $n$ 和 $k$，你需要构造出一个长宽都是 $n$ 且每行每列的和都是 $k$ 的矩阵。

# 解题思路

题目要求每行每列的和要等于 $k$，最简单的思路就是每行每列**恰有一个** $k$，其余位置**均为** $0$。

换句话说，我们输出的矩阵要是这样的：

```
k 0 ··· 0 0
0 k ··· 0 0
·····k·····
0 0 ··· k 0
0 0 ··· 0 k 
```

观察矩阵，不难发现第 $(i,j)$ 个位置如果满足 $i = j$ 则值为 $k$，否则为 $0$。

这样，这道题的代码就呼之欲出了。

# AC 代码

```cpp
#include<cstdio>
int main(){
	int n, k;
	scanf("%d %d", &n, &k);
	for(int i = 1; i <= n; i++){
		for(int j = 1; j <= n; j++){
			if(i == j){
				printf("%d ", k);
			}else{
				printf("0 ");
			}
		}
		printf("\n");
	}
	return 0;
}
```
