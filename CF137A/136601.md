### 题目翻译（虽然是机翻看看就好）
```
Polycarpus把明信片和照片一排挂在墙上。他决定把它们放到壁橱里，把一幅著名画家的画挂在墙上。波利喀普斯是这样做的：他从左到右，连续地移动物体。由于Polycarpus不希望发生任何混淆，他不会把两种不同类型的物品拿在手中。换句话说，Polycarpus不能同时携带明信片和照片。有时他会去壁橱，把东西放在那里，这样他的手就自由了。Polycarpus必须把所有的明信片和照片放在壁橱里。他不能跳过物体。如果他不能携带超过5件物品，他至少应该去壁橱几次？
```
### 大体分析
```
此题过水没有什么技术含量，大概就是用top代表取物品的数量，先读入字符串n，从头搜到尾，如果n[i]!=n[i-1]，代表必须要再去一次壁橱（ans++，此时因为换了物品所以重置top=1），否则若top==5，代表不能再去了，同样（ans++，top=1），否则top++
```
### 代码（有注释，有防作弊）
```
#include<bits/stdc++.h>//万能头文件
using namespace std;
string n;
int top,ans;
int main(){
	cin>>n;
	n=' '+n+' ';//因为要判断n[i]与n[i-1]，如前边不加‘ ’，会越界；同样，如果最后不加‘ ’，最后一组就判不到
	for(int i=1;i<n.size();i++){
		if(n[i]!=n[i-1]) ans++,top=1;
		else if(top==5) top=1,ans++;
		else top++;
	}//见上述分析所说
	cout<<ans;//输出最终结果
   //return 0；
} 
```
