观察图片,得方块=

1+3+5+...+n×2-1+...+5+3+1

=2n²-(n×2-1) //1+3+5+...+2n-1=n²,两个这么多,再减去多算的那个2n-1

=(n*2<<1)-(n<<1)+1 //n<<1等价于n *2;

接下来展示入门难度的ac代码

```cpp
#include<bits/stdc++.h> //偷懒头文件。
using namespace std;
int main(){
    int n;
    cin>>n;
    cout<<(n*n<<1)-(n<<1)+1; //位运算的优先级较低,最好加上括号。
    return 0;
}
```
