第一篇 `py` 的题解。

首先我们知道，如果一个数为偶数，那么他的末位数字一定是偶数。对于这道题，如果字符串的第 $i$ 位对应的数字是偶数，那么所有前面的连续数字都可以与他组成偶数，总共有 $i$ 中情况。

那么代码就很好写了，直接扫描字符串并判断即可。

```python
n = int(input())
s = input() # 输入
ans = 0
for i in range(len(s)): # 扫描字符串
	if (int(s[i]) & 1) == 0: # 如果某位字符对应数字是奇数
		ans += i + 1 # python字符串下标从 0 开始，所以要 + 1
print(ans) # 输出答案即可
```
