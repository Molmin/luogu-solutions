## 思路

给我们的信息是k个人。

如果这k个人全是排名<=25，根据已知信息，我们不知道此时有没有人拒绝，所以输出0（没有人拒绝）。

如果任意一个人排名>25 ，由于`排名前25的选手参加决赛` 所以要使拒绝的人最少，那么所有晋级选手的排名都应在已知排名最靠后的人之前（注意晋级名单包括但不限于这k个人），所以此时，我们只需要找到排名最后的人的排名减去25，即是最少的拒绝人数。

***

    #include<bits/stdc++.h>
    using namespace std;
    int n,x,m;
    int main()
    {
        scanf("%d",&n);
        for(int i=1;i<=n;i++)
          scanf("%d",&x),m=max(m,x);  //找出排名最靠后的
        printf("%d",max(0,m-25));     //判断是情况1还是2
        return 0;                     //好习惯
    }
