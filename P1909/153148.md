在顺序与分支这一栏目里出现了循环语句For的 题，好像有点儿不寻常。所以正解应该是把For语句赶出门的。

这题**好像并没有什么难点**（虽然说我提交了四次才全部AC）。

一共三种包装，老师只统一买一种包装。因为“需要的铅笔数刚好是该包装内含的铅笔数的倍数”这种情况不会百分百出现，所以，如果n%a不等于0，我们需要购买的包装数就是n/a+1。（本段话定义n为需要购买的铅笔数，a为该包装内含有的铅笔数。）

废话不多说，来看我的过程。

------------

    #include<iostream>
    using namespace std;
    int main(){
       int n,a1,a2,b1,b2,c1,c2,da,db,dc,ans;
	   /*需要铅笔数(n)，最少要花的钱(ans)
       a~c包装单包铅笔数(a1,b1,c1)，a~c包装单价    (a2,b2,c2)
       每个包装的总价(da,db,dc)*/
       cin>>n>>a1>>a2>>b1>>b2>>c1>>c2;
       //a包装 
       if(n%a1!=0) da=(n/a1+1)*a2;
       else da=(n/a1)*a2;
       //b包装
       if(n%b1!=0) db=(n/b1+1)*b2;
       else db=(n/b1)*b2;
       //c包装
       if(n%c1!=0) dc=(n/c1+1)*c2;
       else dc=(n/c1)*c2;
       //比较
       if(da<db) ans=da;
       else ans=db;
       if(dc<ans) ans=dc;
       cout<<ans;
       return 0;
    }
------------
再说一下比较那里。

若da<db且db<dc，则输出da；若db<da且da<dc，则输出db……然后你会发现——哎呀！好多种情况呀！不写了不写了！

然而我们可以拆分开来，先比较da与db。

比较完了呢，就把较小的一个设为ans(即将输出的数)。

别急，还没完！你漏掉了dc！

再比较ans和dc。

然后你发现，哎呀，dc要比ans小！幸亏没有急着输出。

然后我们重新给ans赋值。ans=dc。

然后输出ans，完毕！
