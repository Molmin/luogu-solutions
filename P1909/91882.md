先来理清题目思路
1.循环m次执行，计算m种商品，每一次都一样内容
2.编写循环执行内容：先计算每一样要购买的盒数，因为盒数不为小数，所以有小数部分进1，否则会少。然后由单价乘数量得总价。
3.最后比较，因为n为无限大，所以第一次n为第一种商品的价钱，接下来如果有比n小的，就将自己赋值给n，这样到最后只会留下小的，n为输出结果
```
#include<iostream>
#include<cmath>//ceil调用cmath库
using namespace std;
int main(){
	long long a,b,c,s,h;//long long可放的量较大，所以比较保险
	long long m,n=0x7ffffff,v;//设n为无限大，以便接下来比较   
	cin>>m;
	for(long long i=0;i<3;i++){//循环三次
		cin>>a>>b;
		s=ceil(m/a);//输入后同时执行，ceil是向上取整函数，如果有余数就商加1
		c=s*b;//c会重新赋值，不是定值，所以无需担心
		if(c[i]<n){
			n=c[i];
		}//n为无穷大，所以n的值改为c[i]，这样下面也可与上面的数进行比较，小的代替大的
	}
	cout<<n<<endl;
	return 0; 
}
```
PS：如有基础结构不懂的，看我第一篇题解。接下来普及一下循环结构。
循环结构是一个基本结构，其作用巨大。循环结构循环次数就是大括号里所写的内容需要执行的次数，也可做数组下标（这个以后再讲），i表示执行的次数，其实可以用以枚举，如：for（long long i=1；i<3；i++）表示枚举一到二
基本格式：for（开始的值；结束条件；每次加的值）i++就是i+=1（编程时表示i+1的单独表达式。
