## 梦雪的思路

 **思路一**.是30的倍数的数的性质：
 
 1. 结尾为0。（即为10的倍数的数的性质）
 
 1. 每位数字的和为3的倍数。（即为3的倍数的数的性质）
 
 **思路二**.符合题目描述的数的性质：
 
 1. 含0。
 
 1. 每位数字的和为3的倍数。
 
 **思路三**.用符合题目描述的数的每位数字所组成的数,只要末尾为0就都是30的倍数。
 
 **思路四**.其中最大的一个便是将每位数字从大到小排序后的结果（0最小，所以可以保证0在末尾）。
 
##  你们的最爱

```
#include<bits/stdc++.h>
using namespace std;
char n[100004];
int k,sum;
bool qwq(char x,char y){
	return x>y;//将每位数字从大到小排序
}
int main(){
	cin>>n;
	for(int i=0;i<strlen(n);i++){
		if(n[i]=='0')
			k=1;//判断该数是否含0
		else
			sum+=n[i]-'0'; //计算每位数字的和
	}
	if(k==0||sum%3!=0){
		cout<<-1;
		return 0;
	}//判断是否符合思路一
	sort(n,n+strlen(n),qwq);
	cout<<n;//输出
	return 0;
}
```
