这题可以说是一道数学题，先说一下十进制小数怎么转三进制小数：

十进制转任何进制都是采用整数除n取余倒序排列，小数乘n取整顺序排列的方法。

比如　32.12　转　三进制

整数部分：

32除以3商10余2

10除以3商3余1

3除以3商1余0

1除以3商0余1

所以整数部分是　1012

小数部分：

0.12×3＝0.39　整数部分拿出　0

0.39×3＝1.17　整数部分拿出　1

0.17×3＝0.51　整数部分拿出　0

0.51×3＝1.53　整数部分拿出　1

0.53×3＝1.59　整数部分拿出　1

......

依次类推直到余数为0或者达到要求的精度，比如到小数点后5位就为：0．01011

上代码：
```cpp
#include <bits/stdc++.h>

using namespace std;

bool c = false;

int main(){
	double n = 0;
	int s[10086];
	while(scanf("%lf" , &n) == 1){
		for(int i = 0;i < 10086;i++){
			n *= 3;
			s[i] = n;
			n -= s[i];
			if(s[i] == 1){
				cout<<"NON-MEMBER"<<endl;
				c = true;
				break;
			}
		}
		if(c == false)
		cout<<"MEMBER"<<endl;
	}
	
	return 0;
}
```
