这个题很容易超时，时间限制500ms，所以我就不用for循环。
使用数学方法->
放code：
```cpp
	(n/k+1)*k
```
#### 解释：
一个数 **n** 除以另一个数 **k** ，假设得到 **t** ，那么一定满足 **t * k <=n** ，不过等不等于**n**，那么加上**k**，得到的结果一定大于**n**，但是小于等于**n+k**，
该值即为结果。

------------

### code:
```cpp
#include<iostream>		//这辈子也不用万能头（划掉）
#include<cstdio>
#include<cmath>
using namespace std;
long long n,k;
int main()
{
	scanf("%d%d",&n,&k);	//输入
	cout<<(n/k+1)*k;		//输出
	return 0;				//结束
}
```
