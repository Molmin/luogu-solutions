------------

首先，一个比较麻烦的条件：$x>n$

那么我们先不管这个条件

考虑一个简化的版本：求**不大于** $n$ 的，能整除 $k$ 的**最大的** $x$

很明显，是 $n - n\mod k$

考虑怎么把这个式子改成符合原题条件的式子

注意到 $n \mod k$ **一定**小于 $k$

即：$n-n \mod k + k > n$

而且显然，原式加上 $k$ 后，依然可以被 $k$ 整除

所以答案为：$n-n \mod k + k$

------------

程序就非常非常非常非常非常非常短了：

```cpp
#include<cstdio>

int main(void)
{
	int n,k;
	scanf("%d%d",&n,&k);
	printf("%d",n - n%k + k);
	return 0;
}
```
