# 0x00 思路
先看题。
>有一个质数 $P$（$5\leq P\leq 10^9$），请你找到两个数使 $P$ 除以这两个数余数相同，有多组解输出任意一组。

也就是余数相同，那……随便输出一组不就好啦？

# 0x01
~~实在想不到用什么名字了……~~

就一句话：基数 $x,y$ 千千万，何苦找不到另一半？

既然如此，直接指定一个 $x$ 或者 $y$，然后去找 $y$ 或 $x$ 就好了。

但是由于本着简单容易的原则，我们决定指定 $x$ 去找 $y$。

又由于 $2\leq x<y\leq P$，所以我们直接指定 $x=2$，让 $y$ 去满足 $x$ 即可。否则会出问题。不相信的谷友们可以自己尝试一下 $x=3,4,5$ 的情况。

# 0x02 论证
这里解释一下为什么 $x=2$ 是完全可行的。因为 $5\leq P\leq 10^9$ 且 $P$ 是质数。所以这里的 $P$ 一定是奇数。那么 $P\bmod 2$ 一定等于 $1$，由此，$y=P-1$。

当我们解释完了之后，这件事就变得简单了。
# code:
```c
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int i,P,t;
	scanf("%d",&t);
	while(t--)

		scanf("%d",&P);
		printf("2 %d\n",P-1);
	}
	return 0;
}
```
