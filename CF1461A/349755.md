# 分析：
要求的回文子串长度只要$≤k$即可，而$k≥1$。这里不用看$k$的上限，因为题目要求只要写出一种字符串即可。所以不妨设$k=1$  _(~~其实是本蒟蒻太笨了，不会算~~）_ 。因为单个字符也可以被视为回文字符串，所以$k=1$是不会出问题的。因此回文子串这个题目要求就可以暂时不用管了。

那么最后题目就变成了输出一个长度为$n$的字符串，且字符串的构成元素为 **$'a','b','c'$**. 那么我们就不妨以"$abc$"的顺序为一个周期构造字符串（这样也可以防止不小心“**踩雷**”）此处“**踩雷**”指的是在某种构造中会出现一个（或多个）更长的回文子串，且长度$>k$。比如 $n=6,k=2$ 时, $"abccba"$，$a$确实为一个回文子串，但是整个字符串$abccba$也是这个字符串的回文子串（一个字符串本身就是它的一个字串），而这个回文子串的长度$=6$,  $6>2$了，要舍弃。 而按照$abcabc......$这个顺序是可以证明不会出现上述不符合题意的情况的。

#include<bits/stdc++.h>
using namespace std;
int main(){
    int t;
    cin>>t;
    while(t--){
        int n,k;
        cin>>n>>k;
        for(int i=1;i<=n;i++){
            if(i%3==1) cout<<'a';
            if(i%3==2) cout<<'b';
            if(i%3==0) cout<<'c';
        } 
        cout<<endl;
    }
    return 0;
}
