下面证明打表的正确性

本题疑似出现恶意评分嫌疑


对$4k+1(k∈Z)$ $1+(2-3)+(5-4)+...=1$

对$4k+2(k∈Z)$ $(2-1)+(3-4)+...=1$

对$4k+3(k∈Z)$ $1+(2-3)+(5-4)+(6-7)+...=0$

对$4k(k∈Z)$ $(2-1)+(3-4)+(6-5)+(7-8)+...=0$

易得正解
```cpp
#include <stdio.h>
int main()
{
	int n;
	scanf("%d",&n);
	if(n%4==1||n%4==2)
	{
		printf("1\n");
	}
	else
	{
		printf("0\n");
	}
	return 0;
}
```
