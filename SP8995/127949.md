题目大意：你有一块 $2^m$ 长的巧克力，要给别人 $n$ 的长度，每次只能将它对半分开，求 $m$ 的最小值和要对半分几次。

先看第一问。很明显要找到一个最小的 $m$ 满足 $2^m\ge n$，即 $m\ge\log_2n$。要让整数 $m$ 最小，很明显 $m=\lceil\log_2n\rceil$。

再看第二问。我们把 $n$ 转成二进制。容易发现，它从低往高的第 $i$ 位就表示了要不要第 $2^i$ 这一小块。例如，当 $n=22=(10110)_2$ 时，它需要 $2^1,2^2,2^4$ 这三个小块。

而事实上，我们只需要关心最小的块就可以了。因为分的时候，一定是从大往小分的，所以大的块一定会在分小的块的路上就分出来了。所以分成最小的块所需的步骤，就是第二问的答案。

最小的块对应着二进制的最低位，可以使用 $\operatorname{lowbit}$ 运算取出最低位，设为 $2^x$。则要从 $2^m$ 分到 $2^x$，很明显需要分 $m-x$ 次，答案也就出来了。

时间复杂度 $O(\log n)$。核心代码如下：
```cpp
int t, n;
inline int lg(int x)    //求log2(x)的上取整
{ int z = 0; for ( unsigned int y = 1 ; y < x ; y <<= 1 ) z++; return z; }
signed main()
{
	read(t);
	For(i, 1, t)
		read(n), printf("%lld %lld\n", 1ll << lg(n), lg(n) - lg(n & -n));
	return 0;
}
```
