## 思路：
建一个判断函数来判断报的数是不是 $7$ 的倍数或者里面是否含有 $7$。

```cpp
inline bool pd_7(int x){
	if(x%7==0){
    	return 1;//7 的倍数 
    }
    while(x){//把 x 拆开判断 
    	if(x%10==7){
          return 1;
       	}
		x/=10;
    }
	return 0;
}
```
