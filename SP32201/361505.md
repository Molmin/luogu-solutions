位运算基础题，建议评红。

# 题意

给定 $T$个整数 $n$，求 $n$ 二进制右移 $1$ 位后的结果。

# 思路

这题如果知道位运算是什么，那么就很好做，否则没法做。

首先我们先了解什么是位运算。

假设有一个十进制整数 $11$，那么我们先把它化成二进制（整数如何化二进制就不再赘述了），结果是 $1011$。

“右移”$k$ 位的定义就是将原数的二进制形式的最后 $k$ 位去掉，与之对应的“左移”$k$ 位的定义就是在最后添加 $k$ 位。

十进制整数 $11$ 左移一位的结果是 $(10110)_2$，即十进制的 $22$；右移一位的结果是 $(101)_2$，即十进制的 $5$。

因为左移和右移的基础都是二进制，所以左移 $k$ 位和右移 $k$ 位的结果**分别是原数乘 $2^k$ 和原数除以 $2^k$（下取整）**。讲解到这里你可以尝试做一下了，这样做的思路就是每次把数除以 $2$。

但是，``C++`` 中有两个非常好用的运算符 ``>>`` 和 ``<<``，它们分别是右移运算符和左移运算符。之所以我们不直接除以 $2$ 是因为它们速度快而且它们不止处理一位。程序详细写法在程序里。

**数据范围警告**

因为 $n<2^{64}$，而 ``int`` 的范围是 $-2^{31}$ 至 $2^{31}$。所以我们必须使用 ``unsigned long long``，否则会炸。

# 代码

```cpp
#include<bits/stdc++.h>
using namespace std;

int t; 
int main(){
	cin>>t; // t组数据
	for (int i = 1;i <= t;i++){
		unsigned long long temp;
		cin>>temp;
		cout<<(temp >> 1)<<endl;
		//一定要加括号！否则程序会识别为输入输出流运算符，尤其是用 iostream 库的同学
	}
	return 0;
}
```
