题目大意：

有一个1....n的全排列。现在让你从值为1的那一个位置走到值为2的节点的位置，从2走到3，....从n-1走到n。每一步的距离是两个位置的下标之差的绝对值。问你这总路径是多少。

题解：

记录每一个数所在的位置。循环从1.....n，累加即可。

```cpp
int wz[200005];//记录的位置
int n,x;
int main(){
	rdi(n);
	int i,j;
	for(i=1;i<=n;++i) rdi(x),wz[x]=i;
	ll ans=0;
	for(i=1;i<n;++i) ans=(ll)(ans+abs(wz[i]-wz[i+1]));//累加
	print(ans);
}
```
