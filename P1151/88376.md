### 这个是Python题解

### 题目思路：
简单的来说就是暴力解决问题

枚举所有的10000~30000

寻找符合条件的数字输出

很适合Python练手

### 代码+注释：
```python
a=int(input())
# 这里是数字的输入（好麻烦）
d=False
# 判定是否有符合条件的记录变量
for c in range(10000,30001):
	# 暴力的枚举
    b1=int(c/100)
    b2=int((c/10)%1000)
    b3=int(c%1000)
    # 取得三个子数，以下是三个连续的判定
    if b1%a==0:
        if b2%a==0:
            if b3%a==0:
                print(c)
                d=True 
                # 记录是否有解
if d==False:
    print("No")
# 如果没解，退出输出No
```
