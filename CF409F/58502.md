# 题解 CF409F 【000001】

------------

第一次写CF的题解，激动


这其实是`April Fools Day Contest 2014`的一篇题目，顾名思义，这是一道愚人节题目，它的题目（**000001**）其实是指[OIES](https://baike.baidu.com/item/OEIS/1149139?fr=aladdin)上的序列[**A000001**](http://oeis.org/A000001)，对照表找出规律求解即可。


```cpp
#include<cstdio>
const int a[]={0, 1, 1, 1, 2, 1, 2, 1, 5, 2, 2, 1, 5, 1, 2, 1, 14, 1, 5, 1, 5, 2, 2, 1, 15, 2, 2, 5, 4, 1, 4, 1, 51, 1, 2, 1, 14, 1, 2, 2, 14, 1, 6, 1, 4, 2, 2, 1, 52, 2, 5, 1, 5, 1, 15, 2, 13, 2, 2, 1, 13, 1, 2, 4, 267, 1, 4, 1, 5, 1, 4, 1, 50, 1, 2, 3, 4, 1, 6, 1, 52, 15, 2, 1, 15, 1, 2, 1, 12, 1, 10, 1, 4, 2};//常量数组可以自动测算大小
int main(){
    int n;
    scanf("%d", &n);//输入序号
    printf("%d", a[n]);//输出对应的值
    return 0;
}
```
