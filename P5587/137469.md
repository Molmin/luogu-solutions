## $100\%$做法
存在换行符。  
我们发现很明显的性质：  
对于一个字符串，越晚打进来的字符越早被删掉。  

很好发现是栈的性质，所以考虑用栈来维护所有**没有被删去**的字符。  
扫一遍字符串  
①如果遇到了非`'<'`的符号，就压入栈；  
②如果遇到了`'<'`：  
- 1栈是空的（没有可以删的），啥都不做；
- 2栈不是空的（可以删），弹出栈顶元素（删掉最后打进来的字符）；  

核心代码：  
```cpp
string s;//需要处理的字符串
cin>>s;
stack<char> st;//用栈来维护
for(int i = 0;i < s.size();i++)//扫一遍字符串
{
	if(s[i] != '<')//如果不是退格符
	{
		st.push(s[i]);//压入
	}else if(s[i] == '<')//如果是退格符
	{
		if(st.empty()) continue;//空的别管（会出错）
		st.pop();//删掉最后打进来的字符
	}
}
```  
这段代码的功能就是去掉$s$里面的所有的退格符。  
然后用$40\%$的算法搞一遍处理后的字符串即可
