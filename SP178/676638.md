## 题目大意

给定一张图的邻接矩阵。

如果两个点之间的最短路就是两点之间的直接连边且再无别的最短路，这两个点就是“相邻的”。我们要找出所有“相邻的”点对，并按照字典序输出。

------------
## 思路

考虑到要求所有点之间的最短路，而且注意到点数只有 $200$，我们考虑改装弗洛伊德算法。我们发现，如果两个点是“相邻的”，那么它们在循环中必然不会被更新，同时不会存在一个点 $u$ 使两点之间有一条最短路经过 $u$。所以我们直接在弗洛伊德中判断即可。

具体来讲，如果在处理点 $i,j$ 时出现了 $u$ 使得 $\text{dis}(i,j)\ge \text{dis}(i,u)+\text{dis}(u,j)$，就说明这两个点不是“相邻的”，标记并退出循环；到了最后还没有标记，就输出。

我们从 $1$ 开始枚举保证字典序。同时，这道题虽然有多组数据，但是每一次数据都是一个完整的矩阵，所以不清空也没问题。主函数如下：

```cpp
signed main(){
	int t=read();
	while(t--){
		int n=read();
		for(int i=1;i<=n;i++)
			for(int j=1;j<=n;j++) mp[i][j]=read();
		for(int i=1;i<=n;i++)
			for(int j=i+1;j<=n;j++){
				bool flag=1;
				for(int k=1;k<=n;k++)
					if(mp[i][k]+mp[k][j]<=mp[i][j]&&k!=i&&k!=j){flag=0;break;}//判断条件
				if(flag) printf("%lld %lld\n",i,j);//直接输出
			}
	}
	return 0;
}
```
