本题其实很简单，只需要暴力枚举每个点，判断该点以及周围四个点是不是 $X$ 就行了。因为数据很弱，就算最多也只需要${2*500^2}=500000$次，因此枚举完全没问题。

代码应该很好理解，就不做过多赘述了，贴上代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int n,i,j,ans=0;
	char c[510][510];
	cin>>n;
	for(i=1;i<=n;i++)
		for(j=1;j<=n;j++)
			cin>>c[i][j];
	for(i=1;i<=n;i++)
		for(j=1;j<=n;j++)
			if(c[i][j]=='X'&&
            			c[i-1][j-1]=='X'&&
            				c[i-1][j+1]=='X'&&
            					c[i+1][j-1]=='X'&&
            						c[i+1][j+1]=='X')//看起来美观一点
			ans++;
	cout<<ans;
}
```
