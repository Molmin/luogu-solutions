题目描述（翻译后）：
输入：第一个整数是a[0]。之后a[0]个整数a[1]，a[2]，......，a[a[0]]， 输出：一个整数=a[1] + 2 x a[2] + 3 x a[3] + ...... + a[0] x a[a[0]]， a[0]<=100，a[i]<=1000

解题思路：
首先我们看到第一个整数a[0]，已知最后一个数是a[a[0]]，所以这里一共有a[0]+1个数，同时后面的运算也没有用到a[0]，所以综上所述：a[0]只是用来告诉你参与运算的一共有多少个数。知道了这些这个题就很容易了。
------------
```
#include<iostream>
using namespace std;
int main(){
	int a,s=0;	//a及上面的a[0]，s用来存最后的值
	cin>>a;
	int b[110];
	for(int i=1;i<=a;i++){
		cin>>b[i];
		s=s+i*b[i];		//一边输入,一边计算
	}
	cout<<s;	//输出最后的值
}
```
