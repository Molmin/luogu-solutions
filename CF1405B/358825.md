题意概括：给出一些序列，让它满足序列顺序下和为零。可以操作两个数使它满足要求，记录操作花费最后输出。

思路：最少性问题，考虑贪心算法。

推导贪心方法：为了最简，尽可能让操作免费，所以我们要用前面大于零的去填平小于零的，当前面的数小于零数，才不得不花代价去填平。

深化思考：题目给出条件序列和为零，所以说直接让顺序下序列和为零，记录代价，不会让序列发生改变，也可以得出代价。

注意：题目的意思是加一花费代价是一，并不是一次操作代价为一。

```cpp
#include<iostream>
using namespace std;
int a[1000001];
int main()
{
	int x,n;
	cin>>x;
	for(int i=1;i<=x;i++){
		cin>>n;
		for(int j=1;j<=n;j++){
			cin>>a[j];
		}
		long long js=0,h=0;//统计时要用长整型。
		for(int j=1;j<=n;j++){
			h+=a[j];
			if(h<0){
				js-=h;//记录代价。
				h=0;//使它符合条件。
			}
		}
		cout<<js<<endl;
	}
	return 0;
}
```
