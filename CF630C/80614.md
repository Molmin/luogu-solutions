我们发现，其实就是从 $1$ 到 $n$ 枚举每多少位的排列情况之和，因为这个数每一位只能由 $7$ 或 $8$ 组成，意味着每位数你有两种选择。那么两位数有几种选择呢？？？我可以先选第一位选第一种选择，第二位选第一种选择，然后第一位选第二种选择，第二位选第一种选择，第一位选第一种选择，第二位选第二种选择。最后第一位选第二种选择，第二位选第二种选择。一共四种选择，从枚举过程中，我们发现，第一位选出特定的一种选择时，第二位可以选两种，所以一共有 $2 * 2 = 4 $  种选择。那么显然，我们就可以发现，当第 $n$ 位选择一种选择时，前 $n-1$ 位每位可以选两种选择，组合起来就是 $2^{n-1}$ 次选择，再加上第 $n$ 位可以选 $2$ 种，所以一共可以选 $2^n$ 次选择，那么我们直接求和即可，你可以使用快速幂，可以暴力算，也可以用数学方法，也可以使用位运算，这些方法均不会超时。可放心食用（如果数据再强点就可以卡掉暴力之类的算法）

# code:

```
#include<cstdio>
using namespace std;
int main(){
	long long a,b=2,c=1;
	scanf("%lld",&a);
	a++;
	while(a){
		if(a%2){
			c*=b;
		}
		b*=b;
		a/=2;
	}
	printf("%lld",c-2);
	return 0;
}
```
