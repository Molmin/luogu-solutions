一眼博弈论题，首先，我们先观察数据，五花八门的，~~看不出~~什么规律，只能看题目内容,下面给出算数和几何两种思路。

### 算数做法：

从以下这个式子入手： $(w+2)\times(w+2)$

变型得： $(w+2)\times(w+2)=w^2+4\times w+4$

容易发现，这个式子的结果总是奇偶性不变的。
所以，大胆猜测这道题的答案与**奇偶性**有关。

我们先刨去除 $n$ 以外的数据，发现当 $n$ 为 $2$、$8$ 这样的偶数时小B胜，当 $n$ 为 $3$ 这样的奇数时小A胜。

### 几何做法

当 $n$ 为偶数时只要小B选择下在小A的对称点初时即为最优，小B必胜。但是当 $n$ 为奇数会**多出**一行一列，此时小A必胜。

ACcode：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int t, n, q, seed;
	cin >> t;
	while (t--)
	{
		cin >> n >> q >> seed;
		if (n % 2 == 0)
			cout << "B won" << endl;
		else
			cout << "A won" << endl;
	}
}
```
