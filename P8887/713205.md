## 思路

式子 $(w+2)\times(w+2)$ 可以变换成另外一条式子 $w^2+4(w+1)$，具体变换过程在 [Link](https://www.luogu.com.cn/paste/9irxs2ss)。

假设 $w$ 为奇数，$w^2$ 也是奇数（这个应该没人不知道吧），$4(w+1)$ 是偶数，奇数 $+$ 偶数 $=$ 奇数，所以边长 $n$ 一开始是奇数，边长无论扩大几次，面积都永远是奇数（其实边长也是，因为奇数 $+$ 偶数 $2$）。

假设 $w$ 为偶数，$w^2$ 也是偶数，$4(w+1)$ 是偶数，偶数 $+$ 偶数 $=$ 偶数，所以边长 $n$ 一开始是偶数，边长无论扩大几次，面积都永远是偶数。

以上推论均奇偶性有关，可以大胆猜测此题也与奇偶性有关。

根据以上推论，可得：$q$ 无论是多少，都不影响 $n$ 的奇偶性，而 $seed$ 其实是一个无用的数据，生成了 $x_i$ 一点用没有 $T\times n$ 最坏情况已经是 $10^8$ 了，可能会导致时间超限（$\operatorname{TLE}$）。思路理清下来，好像只有数据 $n$ 是有用的，那接下来就是推结论了：

假设 $n$ 为偶数，那么后手的最优策略是放在先手落子的位置的对称点上，这样是必胜的（不信约上你的好朋友一起试一下）；但当 $n$ 为奇数时，会多出一行一列（这是显然的），这时后手必输。

以上推论证明：
- $n$ 为奇数先手赢，反之后手赢。

## 代码
```cpp
#include <bits/stdc++.h>
using namespace std;
int T,n,q,seed;
int main(){
    cin>>T;
    while(T--)
    {
    	cin>>n>>q>>seed;
    	if(n%2==1) cout<<"A won"<<endl;
    	else cout<<"B won"<<endl;
      //蒟蒻专用条件表达式
	}
    return 0;
}
```
