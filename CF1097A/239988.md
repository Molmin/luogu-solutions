# CF1097A C++题解
### 题目简述
- 有六张牌，每张是一种花色和一个点数
- 如果后五张牌中，有一张的点数或花色是头一张点数或花色的话，请输出 YES
- 没有的话输出 NO
### 题目分析
在我书写本题解时，似乎其他题解做得略复杂，对于一道红题，不一定需要数组甚至结构体上场吧。

分析后，可以知道，本题在输入后面牌的花色、点数时，就可以进行判断是否相同，一旦相同，立刻输出 YES，并且结束程序。结束后，就不再判断。如果输入完了，程序还没结束，就意味着没有相同的牌，于是输出 NO。

分析完毕，上代码
### 代码
```cpp
#include<iostream>
using namespace std;
char a,b,c;
int main()
{
    cin>>a>>b;
    for(int i=0;i<5;i++)
    {
        cin>>c;//输入其中一个数据，并判断
        if(c==a)
        {
            cout<<"YES";
            return 0;
        }
        cin>>c;//输入另一个数据，并判断
        if(c==b)
        {
            cout<<"YES";
            return 0;
        }
    }
    cout<<"NO";
    return 0;
}
```
### 注意点
1. 看清楚输出的格式
1. 输出之后马上 `return 0;`
1. 数据都开成 `char` 类型
1. 不要犯其他低级错误
