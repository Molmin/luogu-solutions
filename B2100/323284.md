# 题解 B2100 【同行列对角线的格】
### 题目理解
就是在一个 $N \times N$ 的棋盘里有一个坐标为 $(i,j)$（这里以 $i$ 为行，$j$ 为列）的点，按题意输出一些点。
### 题目分析
我们不妨设一个点为 $(x,y)$。

- **同一行上格子的位置**

那么 $x$ 肯定等于 $i$，而 $y$ 随便取都行。

那么按照样例给的顺序 $y$ 是按照 $1,2,3...n$ 的输出。

- **同一列上格子的位置**

其实没啥区别，就是反过来。$y$ 肯定等于 $j$，而 $x$ 随便取都行。

而输出 $x$ 顺序就是 $1,2,3...n$。

- **从左上到右下输出同一对角线格子位置**

我们~~用脚趾头~~想一想~~都~~可以得到 $i-x=j-y$ 这一个等式。

由于存在顺序问题，所以我们可以先从 $(i,j)$ 这一个位置到 $(i-1,j-1)...$

最后 $x$ 与 $y$ 一定有一个会等于 $1$。

然后再不停地输出 $(x,y) (x+1,y+1)...$ 直到有一个等于 $N$。

- **从左下到右上输出同一对角线格子位置**

把等式进行变化。$i-x=y-j$.

剩下的自己想。

### 参考代码

```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int n,i,j;
int main(){
	scanf("%d%d%d",&n,&i,&j);
	for(int y=1;y<=n;y++)
		printf("(%d,%d) ",i,y);//小坑点' '。
	printf("\n");
	for(int x=1;x<=n;x++)
		printf("(%d,%d) ",x,j);
	printf("\n");
	int x,y;
	x=i;y=j;
	while(x>1&&y>1){x--;y--;}//为什么不能取等于？因为当 x 或 y 等于 1 时 x，y 依然会减 1，结果就变成了 0。
	while(x<=n&&y<=n){
		printf("(%d,%d) ",x,y);
		x++;
		y++;
	}
	printf("\n");
	x=i;y=j;
	while(x<n&&y>1){x++;y--;}//同理。
	while(x>=1&&y<=n){
		printf("(%d,%d) ",x,y);
		x--;
		y++;
	}
	printf("\n");
	return 0;
}
```
