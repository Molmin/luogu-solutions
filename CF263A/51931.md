# 我还是搞不懂为啥要开数组啊

## 本蒟蒻表示能不开二维就不开二维啊

因为已经固定了是5*5了，那么**只要一层一层读，读到1算距离就好了**

另外算距离也很简单：由于不是直线距离（这牵涉到勾股），只要算两点之间的最短路线即可（参见小学奥数）

所以，设x，y为数字“1”的横纵坐标这里最短路线的公式即为：**|x-3|+|y-3|**

## 不要忘了加绝对值！

|x-3|求出两点之间横坐标的差，加绝对值是因为数字“1”可能在中心点左边

同样的，|y-3|算出两点之间纵坐标的差

在C++中，绝对值可以用函数**abs**求出

当然，你用函数求我也不反对，就是当心重名【滑稽】

总的来说就是这样，上代码：

```
#include<bits/stdc++.h>
using namespace std;
int main(){
	int t,i,j,ans;  //t存每行每列的数据，i、j做循环，ans记答案
	for ( i = 1 ; i <= 5 ; i++ )
		for ( j = 1 ; j <= 5 ; j++ )
		{
			cin >> t;  //读入【i，j】的数据
			if ( t == 1 )  //读到1了
			{
				ans = abs( i - 3 ) + abs( j - 3 );  //套公式了啊……
			}
		}
	cout << ans;  //输出答案
	return 0;  //勿忘勿忘勿忘……
}
```
