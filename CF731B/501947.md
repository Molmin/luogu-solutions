在简单理解了题意之后，我们发现这是一道 **有关奇偶数** 的问题！

先标一下号：

- 套餐 $A$ 在同一天购买正好两个披萨

- 套餐 $B$ 在连续的两天各购买一个披萨

首先，最后一天只能购买套餐 $A$ ，因为只能 $n$ 天内购买， **结束后不可购买** 。

一个问题 —— 如果最后一天买的是奇数个怎么办？

显然倒数第二天应该购买套餐 $B$ ，让最后一天满足要求。

那此时我们在讨论倒数第二天，以此类推。

到了第一天，我们看看第一天是否有办法满足即可

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a[100005];
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        cin>>a[i];
    }
    for(int i=n;i>=2;i--)
    {
        if(a[i]<0)
        {
            cout<<"NO";
            return 0;
        }
        if(a[i]%2)
        {
            a[i-1]--;//前一项
        }
    }
    if(a[1]%2)cout<<"NO";
    else cout<<"YES";
    return 0;
}
```
