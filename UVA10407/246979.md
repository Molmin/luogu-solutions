### 题目大意

- 每行输入一串数字，记为 $a_i$ 。求出最大的k，使得对于每一个 $x_i(x_i=a_i\bmod k)$ 都相等。

- **多组测试数据**。每组数据以0结尾。一行只有一个0，表示多组数据的输入结束。

------------
### 思路

应该还算很简单的吧……

因为所有的数除以这个k所得的余数都相同，所以它们减去余数之后都是k的倍数。

但是我们没必要把余数求出来。因为减去之后都是k的倍数，假设 $a_i$ 减去余数之后为k的 $c_i$ 倍。显然 $c_i$ 是整数。于是我们发现，这些数两两间的差都是k的倍数。

所以就很显然了，都是k的倍数，并且k最大，言下之意k就是最大公约数。

因此求出a的差分数组b，然后用欧几里得算法求出b数组的最大公约数即可。

------------
```cpp
int a[100005],b[100005],n,tot,ans;//a,b数组的含义见上
int gcd(int a,int b){//欧几里得算法
	return b?gcd(b,a%b):a;
}
int main(){
	while(1){//一直循环，中间再判断输入
		tot=0,ans=99999999;//每组数据初始化
		cin>>n;
		if(n==0)return 0;//输入结束
		a[0]=n;//原数组第一个数
		while(1){//一直循环，中间再判断输入
			cin>>n;
			if(n==0)break;//本组数据输入结束
			tot++;
			a[tot]=n;
			b[tot]=abs(a[tot]-a[tot-1]);//求差分数组
		}
		ans=gcd(b[1],b[2]);
		for(int i=2;i<tot;i++)ans=gcd(ans,b[i+1]);//求最大公约数
		cout<<ans<<endl;
	}
	return 0;
}
```
