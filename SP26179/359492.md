### 关于题目

[题面传送门](https://www.luogu.com.cn/problem/SP26179)

这道题还是比较简单的，大概评红。

### 思路分析

结果读题，我们不难得出，如果需要时间最短的话，我们直接将大炮开向目标点的方向，这样可以充分利用他的初速度。

下一步我们来考虑风的方向和速度。

经过思考，我们可以发现，风的方向和我们开炮的方向只有同向和异向两种情况。设大炮与目标点间的距离为 $s$， 大炮的速度为 $v_1$， 风的速度为 $v_2$， 时间为 $t$。 当两者同方向时， $t=s\div(v_1+v_2)$。 当两者异向时， $t=s\div(v_1-v_2)$。 不难发现，若此时 $v_2 > v_1$， $t$的值会为负数，那么此时就输出 ```Impossible```。


其他细节请阅读代码。
### 代码实现
```
#include <bits/stdc++.h>
using namespace std;

double x1, x2, v1, v2, t;
char d;

int main()
{
    scanf("%lf%lf%lf%lf %c", &x1, &x2, &v1, &v2, &d);//注意，这你是浮点数读入
    if ((x1 > x2 && d == 'R') || (x1 < x2 && d == 'L'))//如果开炮方向与风的方向不一致的话，就把风的速度去相反数，相当于加上他的负数，这样就可以与同向一起处理。
        v2 *= -1;
    t = (abs(x2 - x1)) / (v1 + v2);
    if (t < 0)//按照之前的分析进行输出。
        puts("Impossible");
    else
        printf("%.6lf\n", t);

    return 0;
}
```
