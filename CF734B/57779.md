这题的题目意思**比较难理解**，这里讲一下：

现在有$k_{2}$个2，$k_{3}$个3，$k_{5}$个5，$k_{6}$个6。要你选其中的数将它们拼成256或32.求所有拼成的数的最大和。

建议**多读几遍**就理解了。

现在讲讲**思路**：

其实思路很简单，就是**尽量多凑256，剩下的再凑32就行了**。

那256的个数怎么求呢，多加思索一下会发现其实就是min($k_{2}$,min($k_{5}$,$k_{6}$))。 接下来就是32的个数，先用$k_{2}$减去刚刚用的数量，然后32的个数就是min($k_{3}$,$k_{2}$)。最后加起来就行。

上代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int k2,k3,k5,k6,s=0;
int main(){
    cin>>k2>>k3>>k5>>k6;
    s+=256*min(k2,min(k5,k6));
    k2-=min(k2,min(k5,k6));
    s+=32*min(k2,k3);
    cout<<s<<endl;
}
```
