## 题意
搭建一个颁奖台，使 $h1$ 高于 $h2$ 高于 $h3$，且 $h1$ 尽量小，并且给的方块全用上。
# 分析
经过思考，我们发现三个台子，最理想的状态是 $n$ 为 $3$ 的倍数，那么 $h2$ 是 $\dfrac{n}{3}$，而 $h1$ 为 $\dfrac{n}{3}+1$，$h3$为 $\dfrac{n}{3}-1$，此时恰好将方块用完。

那么当 $n$ 不是 $3$ 的倍数的时候呢？如果 $h1$ 还是 $\dfrac{n}{3}+1$，那无论如何也无法用完方块了。那么可以让 $h1$ 为 $\dfrac{n}{3}+2$，即再加 $1$，$h2$ 为 $\dfrac{n}{3}+1$，也跟着加 $1$，那么 $h3$ 就应该是 $n-h1-h2$。

**需要注意的是，此时会出现一个问题，如果 $n-h1-h2$ 为 $0$，$h3$ 就是 $0$，但是题目要求每个颁奖台至少要有一个方块，就需要将 $h2$ 减 $1$，对于 $h3$ 直接输出 $1$ 即可。**

### 代码如下：
```cpp
#include <bits/stdc++.h>//万能头文件 
using namespace std;
int t, n, sec;//sec为第二座台的高度 
int main () {
	std::ios::sync_with_stdio(0);//cin输入加速 
	cin >> t;
	while (t--) {
		cin >> n;
		if (n % 3 == 0) //当 n 为 3 的倍数时 
			sec = n / 3; // h2 为 n / 3; 
		else
			sec = n / 3 + 1; //否则 h2 + 1 
		if (sec * 2 + 1 < n) //当 n - h1 - h2 > 0 时 
			cout << sec << ' ' << sec + 1 << ' ' << n - sec * 2 - 1 << "\n";//按照上面的分析，输出 h2, h1, h3 
		else //当 n - h1 - h2 <= 0 时 
			cout << sec - 1/*或写成 n - sec - 2*/ << ' ' << sec + 1 << ' ' << 1 << "\n";
	}
	return 0;
}
```
