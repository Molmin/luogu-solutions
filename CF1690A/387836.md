### 一、思路

显然，要使 $h_1$ 最小，应该让它尽可能靠近 $n\div3$。

分 $3$ 种情况考虑。

#### 1、$n$ 取余 $3$ 为零。

此时，$h_1,h_2,h_3$ 分别是 $n\div3+1,n\div3,n\div3-1$。

可以发现，此时 $h_1$ 最小。因为，根据题意，$h_1$ 是三个数中最大的那个。如果它比 $n\div3+1$ 还要小，则其它两个数都要减小，这样子三个数的总和就小于 $n$ 了。

注：这里和以下的除法均为向下取整的除法。

#### 2、$n$ 取余 $3$ 为一。

此时，$h_1,h_2,h_3$ 分别是 $n\div3+2,n\div3,n\div3-1$。因为，此时 $n-1$ 取余 $3$ 为零，就是在前一种情况的基础上，给 $h_1$ 加上 $1$。注意不是给其它的数加上 $1$， 因为加不了（否则一定会有一个数等于另一个数）。

#### 3、$n$ 取余 $3$ 为二。

此时，$h_1,h_2,h_3$ 分别是 $n\div3+2,n\div3+1,n\div3-1$。这也很好理解，就是第 $2$ 种情况的延伸。

### 二、代码

```cpp
#include<cstdio>
using namespace std;
int t;
int n;
int main()
{
	scanf("%d",&t);
	while(t--)
	{
		scanf("%d",&n);
		int a=n/3;
		if(n%3==0)
		{
			printf("%d %d %d\n",a,a+1,a-1);
		}
		else if(n%3==1)
		{
			printf("%d %d %d\n",a,a+2,a-1);
		}
		else
		{
			printf("%d %d %d\n",a+1,a+2,a-1);
		}
	}
	return 0;
}

```
