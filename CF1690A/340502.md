~~小学数学题~~

---

**题意**

>给你 $n$ 个正方体，问你怎样分配可以拼出一个领奖台（只要求大小关系符合即可），要求第一名（最高）的台要尽量矮。

---

**解题**

首先分析题意，最大的数最小，和不变，即是要尽量平均。

我们先以最理想的情况再微调。

假如 $n$ 刚好是 $3$ 的倍数，那么领奖台高度分别是：

$$n\div 3\quad n \div 3+1 \quad n\div 3-1$$

要是多一点（$n\geq1$），我们当然是先分配到最高的领奖台，即中间的数（因为不这样就会有相等高度）。再多点（$n\geq2$），那么就放在第 $2$ 高的领奖台。

所以，现在的领奖台高度就可以有一个统一的答案：

`n/3+(n%3>=2)` 

`n/3+1+(n%3>=1)` 

`n/3-1`

---

**代码**

```cpp
#include<bits/stdc++.h>
using namespace std;

int t,n;
int main()
{
	cin>>t;
	while(t--)
	{
		cin>>n;
		cout<<n/3+(n%3>=2)<<' '<<n/3+1+(n%3>=1)<<' '<<n/3-1<<endl;
	}
	
	return 0;
}
```
