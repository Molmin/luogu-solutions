**首先**，我们可以发现$-10$和$10$配对,$-9$和$9$配对......$-10$到$0$里每一个数都有相对应个配对数，而这个可以配对的数正好是那个数的绝对值。

**然后**，我们不妨考虑用同桶存储，但输入的数有负数，所以我们得把它变成正整数在存到桶里。    
  
**最后发现**存到桶里后把除了$0$以外所有的配对的数两两相乘(乘法原理)。其中$0$的数很烦人(不能直接乘)，所以我们得特判。首先因为不能跟自己加，所以要先减去$0$的个数。然后我们发现每一个$0$都可以跟后面所有的$0$配对，不能跟自己和前面的配对，而前面和后面是对称的，所以我们可以在减去$0$的个数后直接$/2$。
   
```
#include<iostream>
using namespace std;
unsigned long long Zhoumy,Bank_Count[25],n;
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){
	    int in;
	    cin>>in;
		Bank_Count[in+11]++;//如果是-10就会变成1 
	} 
	for(int i=1;i<=10;i++)Zhoumy+=Bank_Count[i]*Bank_Count[22-i];//配对(无0)
	Zhoumy+=Bank_Count[11/*0的ID*/]*(Bank_Count[11]-1)/2;//加上0
	cout<<Zhoumy;
	return 0;
}
```
