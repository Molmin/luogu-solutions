对于开关状态相同的一段，他都可以在一秒之内走完

所以只要统计有几个开关状态不同的段即可

特别地，有以下两点要注意

1.如果第一个开关的开始状态为关的话，TA必须在原地等一秒，所以若a[1]=1,t++;

2.我算的是开关状态变化了几次，所以要+1才是实际次数

上代码
```cpp
#include<bits/stdc++.h>
using namespace std;
bool a[1000001];//保存开关状态
int t=0,n;
int main()
{
	cin>>n;
	cin>>a[1];
	if(a[1]==1)//第一个为关，必须等一秒
	t++;
	for(int i=2;i<=n;i++)
	{
		cin>>a[i];
		if(a[i]!=a[i-1])//如果状态与上一个不同，t++
		t++;
	}
	cout<<t+1;//记得+1
        return 0;
}
```
