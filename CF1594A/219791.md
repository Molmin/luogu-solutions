### Solution

因为 $l, r\ (l < r)$ 可以取负数所以非常水（

我们想可以取 $[1, n]$，但是他会多出来 $[1, n - 1]$，把他们去掉就行了，对于 $k$，我们取他的相反数 $-k$ 相加即为 $0$，这样就可以将 $[1, n - 1]$ 这个部分给抵消掉。

也就是说，我们取 $l = -n + 1,\ r = n$，那么这个区间的和就是：


$$(-n+1) + (-n + 2) + \cdots + (n - 1) + n = n$$
