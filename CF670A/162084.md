先用我的方法模拟一下样例。

| 这年开始的星期 | 一 | 二 | 三 | 四 | 五 | 六 | 七 |
| -----------: | -----------: | -----------: | -----------: | -----------: | -----------: | -----------: | -----------: |
| 假期天数 | $4$ | $4$ | $4$ | $4$ | $4$ | $4$ | $4$ |

发现为最多为$4$天，最少也为$4$天。

代码思路也很简单，就是用一个双层循环来做。

第一层循环就是看今这年的第一天是星期几。

第二层循环就是来计算这年的假期天数。

接下来就是代码了。

```
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int n;
	cin>>n;
	int max=-1;
	int nin=INT_MAX;
	int jq=0;//表示假期数。
	int t=1;//表示星期。
	for(int i=1;i<=7;i++)//计算今年开始是星期几。
	{
		t=i;
		for(int j=1;j<=n;j++)//模拟1年的的假期为多少。
		{
			if(t==7)//如果今天是星期日。
			{
				t=1;//那么t就等于1。
				jq++;//假期数就要增加。
			}
			else if(t==6)//如果今天是星期6。
			{
				t++;//t还是照样增加。
				jq++;//假期也要增加。
			}
			else//如果不是休息日的话。
			{
				t++;//只有t要增加。
			}
		}
		//一年过去了，要开始与最多假期和最小假期进行对比。
		if(max<jq)
		{
			max=jq;
		}
		if(nin>jq)
		{
			nin=jq;
		}
		jq=0;//假期数要重置为0。
	}
	cout<<nin<<" "<<max;//输出。
    return 0;
}
```
