这题是一个典型的简单贪心题，值得注意的是，这一年的第一天只能从星期一来算，因为如果从星期二-星期日来算的话，会少一些工作日，这样对于算最小假期有不利影响。
```cpp
对于每个7天（即一周），我们的最短假期和最长假期都会多两天，
所以我们可以把周数先处理出来，再去处理剩下的n%7天，
所谓的贪心思想来求最大值，就是能休息就休息，只不过剩下的n%7天不足一周，所以休息天数不能超过两天，所以我们把最大假期加上n%7和2这两个数中的最小值
求最小值，即能工作就工作，只不过一周最多工作5天，所以当n%7大于5时，我们把最小假期加上n%7-5
```
源代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,mi,ma,m,k;
int main(){
ios::sync_with_stdio(0);cin.tie(0);//cin优化，用cin的小伙伴可以在代码前加上，可以提升读入速度
cin>>n;
m=n/7,k=n%7;//处理周数和余下需要特殊处理的天数
ma=min(2,k);//min函数自带，可以返回两个数中的最小值，这里是取剩下几天中的最大休息天数
if(k>5)//特判
mi=k-5;//必须休息的天数
cout<<mi+m*2<<" "<<ma+m*2;//记得加上整周的休息数
return 0;
}
```
