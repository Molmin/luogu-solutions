### 题目大意：
有 $n$ 个商人和 $v$ 元钱，每个商人有 $x$ 个古董，分别价值 $s1,s2...sx$ 元。

如果 $v$ 大于这个商人**其中一个**古董的价钱，便可与这商人达成协议。

问可以跟多少个商人达成协议，并输出商人的编号（升序）

### 思路：

在读入商人的古董价钱时，如果 $v$ 大于此古董的价钱，就将 $ans$ 加 1。

注意：如果该商人已经加入答案，就不可重复加，要判定。

### 代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n,v,ans=0,a[100];
    cin>>n>>v;
    for(int i=1;i<=n;i++){
    	int x,jq,p=0;    //p判定此商人有没有加入答案。
    	cin>>x;
    	for(int j=1;j<=x;j++){
    		cin>>jq;
    		if(v>jq&&p==0){    //如果v大于该古董的价格，且该商人没有加入，则加入答案。
    			ans++;
				a[ans]=i;
				p=1;
			}
		}
	}
	cout<<ans<<endl;
	for(int i=1;i<=ans;i++) cout<<a[i]<<" ";     //保证升序输出。
	return 0;
}
```
