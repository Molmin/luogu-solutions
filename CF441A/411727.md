### 题意

有 $n$ 个商家，你有 $m$ 元钱，如果你的钱大于某一个商家的最便宜的古董，那么你就可以达成协议，输出你可以跟多少个商家达成协议，分别是哪几个？

### 分析

首先输入商家的个数（假设为 $N$），输入 $V$（你有 $V$ 元钱），再输入商家的古董价格。

我们现在要找到最便宜的古董，那么就用 sort 排序（我很懒）, $a_1$ 就是最便宜古董的价格。

如果你的钱大于 $a_1$，那么就统计个数，并且把商家编号用一个数组记录下来。

最后，输出个数以及输出达成协议的商家编号，但是如果你跟某一个商家没有达成协议吗，那么对应的数组的元素值就为 $0$（因为没有达成协议就把他跳过了），所以要特判为 $0$ 的情况。

### 代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,v;
int nn,a[10005];
int b[10005];
int s;
int k=0;
int main(){
	cin>>n>>v;
	for(int i=1;i<=n;i++){
		cin>>nn;
		for(int j=1;j<=nn;j++){
			cin>>a[j];
		}
		sort(a+1,a+nn+1);
		k++;
		if(v>a[1]){
			s++;
			b[k]=i;
		} 
	}
	cout<<s<<endl;
	for(int i=1;i<=n;i++){
		if(b[i]!=0)
		cout<<b[i]<<' ';
	}
}
```
