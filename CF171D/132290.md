题目大意：给你一个1~5之间的数，让你经过一番特定的操作后输出1~3的一个数。

思路：看楼下的题解就给了一个例子，我在这里讲讲本人的思考过程。首先我们发现这里最后要输出的是1~3的数，当然就想到了%3，但是这里是0~2的，所以我们要+1。这样做后会发现交上去了后WA on test5 于是乎我大胆猜测第五个点是5，又想了一想刚刚有一个%的操作，所以我又加了一个%5（这里加特判也可以）。

```cpp
#include<bits/stdc++.h>
using namespace std;
int a;

int main()
{	cin>>a;
	cout<<a%5%3+1<<endl;
	return 0;
}
```
