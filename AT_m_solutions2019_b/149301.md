## 2.进入正题
题意转换：给定一个 $xo$ 串，长度为15(珂以想象后一部分隐藏了)，求字符串中 $x$ 和 $o$ 的数量，若有可能 $o$ 的数量超过 $x$ 的数量，输出 $YES$ 否则输出 $NO$。

### 思路
按照上述题意模拟即可，珂以用贪心思想，认为隐藏部分全为 $o$，所以只需现在 $x$ 的个数不超过 $[15/2]=7$ 就输出 $YES$ 否则输出 $NO$。

## 3.上代码


```cpp
#include<bits/stdc++.h>//万能头，包含string和char的大部分库函数。
using namespace std;
char s[15];//这里用char来写，由题意长度不能超过15。
int cnt;//计数器，计算x的数量。
int main()
{
    cin>>s;//输入
    for(int i=0;i<strlen(s);i++)//i从0开始，初学者极容易错
    {
        if(s[i]=='x')//核心部分：计算x的数量
        {
            cnt++;        
        }
    }
    if(cnt>7)//按题意输出即可
    {
        cout<<"NO";
        return 0;
    }
    cout<<"YES";
    return 0;
}
```
