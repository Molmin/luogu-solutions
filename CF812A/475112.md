## 翻译：

有一个十字路口，四个方向依次标为 $1,2,3,4$ ，现在给出这 $4$ 个方向的 $16$ 个方向，当有车经过绿灯的人行横道时会发生事故，问会不会发生事故。

## 题解：

这题并不是很难 ~~毕竟是A题~~ ，但需要考虑到所有情况。

如图：

![](https://cdn.luogu.com.cn/upload/vjudge_pic/CF812A/a08cde5c54e404c978bae7b092917e8befa50c5b.png)

因为四个方向是对称的，所以我们只考虑 $1$ 号路口。

当 $p_1$ 为绿灯时，可以看到：

- 显然，当 $l_1,s_1,r_1$ 中任何一个为绿灯时，会发生事故。

- 另外，当 $l_2,s_3,r_4$ 中任何一个为绿灯时，也会发生事故。

其他三个方向也是类似的。

所以我们可以很容易得到代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int a[5][5];//这里我用了一个二维数组 
int main(){
	ios::sync_with_stdio(0);
	cin.tie(0);
	cout.tie(0);
	for(int i=1;i<=4;i++)
		for(int j=1;j<=4;j++)
			cin>>a[i][j];//输入 
	if(a[1][4]&&(a[1][1]||a[1][2]||a[1][3]||a[2][1]||a[3][2]||a[4][3]))
		cout<<"YES";//一号方向 
	else if(a[2][4]&&(a[2][1]||a[2][2]||a[2][3]||a[3][1]||a[4][2]||a[1][3]))
		cout<<"YES";//二号方向 
	else if(a[3][4]&&(a[3][1]||a[3][2]||a[3][3]||a[4][1]||a[1][2]||a[2][3]))
		cout<<"YES";//三号方向 
	else if(a[4][4]&&(a[4][1]||a[4][2]||a[4][3]||a[1][1]||a[2][2]||a[3][3]))
		cout<<"YES";//四号方向 
	else
		cout<<"NO";//没有车祸 
	return 0;
}
```
