大家好，我就是可爱的Telon，今天我们来讲一道很有趣的题解——接水问题。这是一道非常简单的题，主要讲述的是一群蒟蒻在一些水龙头前~~接客~~的故事。

由于每个~~客~~人需要的时间不尽相同，所以接客窗口前总有m条不同长度的队伍，我们的任务是让总时间最长的队伍的总时间最短，所以我们可以在每个客人到来时就让他去排队。
    
愉快的我们可以得出排队时可以这样处理
~~~
for(int i=1;i<=n;i++){
    cin>>x;ans=1;
    for(int j=1;j<=m;j++)
        if(a[j]+x<a[ans]+x)
            ans=j;
    a[ans]+=x;
}
~~~
这里a[ i ]就是水龙头 i 号，ans就是总时最短的那条队伍。

接着，我们就可以愉快的得出下面的标程了，是不是很简单？
```
#include<bits/stdc++.h>
using namespace std;
int a[10010];//设为水龙头前接水点
int main()
{
    int x,n,m,ans;
    cin>>n>>m;//输入n和m
    for(int i=1;i<=n;i++){
        cin>>x;ans=1;
        for(int j=1;j<=m;j++)
            if(a[j]+x<a[ans]+x)//寻找到总时最短的队伍
                ans=j;
        a[ans]+=x;//将客户放至总时最短的这条队伍
    }
    sort(a+1,a+1+m);//找出时间最长的队伍
    cout<<a[m];
    return 0;
}
```
