## 思路
可以看到 $1 \le n \le 101$ 和 $2.00s$ 的时限，便想到 $O(n^2)$ 可过。

根据题意，我们只需要在输入时判断这个数是不是在矩阵对角线，中间行和中间列的元素之间，如果是就加上，否则跳过。

**注意**：在做加法中，最中间的位置最后要去掉 $3$ 遍。

### 1. 中间行或列：

如果横坐标或纵坐标是中间，就是中间行或列。

code:
```cpp
if(x==n/2+1||y==n/2+1){
	ans+=k;
}
```

### 2. 矩阵对角线：
1. 如果横纵坐标相等，就是向右的对角线。
2. 如果横纵坐标和为 $n+1$，就是向左的对角线。

code:
```cpp
if(x==y||x+y==n+1){
	ans+=k;
}
```

### 3.去重：
如果 $x$ 和 $y$ 都在中间，就需要先去重。

code:
```cpp
if(x==y||y==n/2+1){
	ans-=k*3;
}
```
