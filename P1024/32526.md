我提供一个不一样的思路，就是我们可以思维数学化一点，我们可以对这个三次函数求导，f(x)' = 3ax^2 + 2bx + c

可以看出这是一个二次函数，我们可以通过求根公式计算出f(x)'的两个根，

xx = 4b^2 - 12ac;                           

x1 = (-2b)-sqrt(xx) / 6a;

x2 = (-2b)+sqrt(xx) / 6a;

其实这两个根就是这个三次函数的最高点与最低点，所以这三个根的范围分别是（-100，x1)  (x1,x2)  (x2,100)

有了这个范围我们就可以直接写一个二分函数，然后分别输出这三个区间的跟就行，对于（-100，100）的范围，二分是很快的。

```cpp
double cul(double l,double r){
    double m;
    while(r-l >= 0.001){            //误差范围内 
        m = (l+r)/2;
        double mid = f(m),fl = f(l),fr = f(r);
        if(mid * fl > 0)l = m;          //（l,m）中有零点 
        else if(mid * fr > 0)r = m;        //（m,r）中有零点 
        if(mid == 0)return m;
    }
    return m;                  //当没找到确切的零点时，返回近似值 
}
```
