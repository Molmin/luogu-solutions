思路：因为数据范围中$n<=50$，所以输入的数字最大可能为$10^{50}$，因此采用字符串输入，之后在枚举累加的同时判断字符串的每一位数字是否为'4'或'7'，若不是则输出"NO"，最后判断累加器的值是否相等，相等则输出"YES"，否则输出"NO"。

上代码：
```cpp
#include<cstdio>
int a,b,n;
char s[51];
main(){
	scanf("%d%s",&n,s+1);	//输入
	for(int i=1;(i<<1)<=n;++i)	//枚举
		if((s[i]==52||s[i]==55)&&(s[n-i+1]==52||s[n-i+1]==55))	//判断
			a+=s[i],b+=s[n-i+1];	//累加
		else return puts("NO"),0;	//依题意输出
	puts(a==b?"YES":"NO");	//依题意输出
}
```
