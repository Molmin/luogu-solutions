## 入门经典

此题考查基本的排序，下面提供两种方法，视自己所学选择。

### 方法一：顺序结构-条件语句

我们知道，三个数字当中只会有一个最大的，如果有两个一样的都是最大，那么只需要输出任意一个即可。而且，如果其中一个是最大的，那么肯定会大于另外两个，而第二大的肯定是从除了这一个数当中选出来的，所以我们只需要写条件语句：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
   int a,b,c;
   cin>>a>>b>>c;
   if(a>b&&a>c){//"&&"表示“并且”，此情况是a最大。
     cout<<a<<" ";//先把最大的输出。
     if(b>c)//当b是第二大，那么c就是第三大。
        cout<<b<<" "<<c;
     else cout<<c<<" "<<b;//如果b不是第二，那么c肯定是第二，顺序输出即可。
   }
   else if(b>c){//a不是最大的，那么要么是b要么是c
      cout<<b<<" ";
      if(a>c)cout<<a<<" "<<c;
      else cout<<c<<" "<<a;
   }
   else {//如果a不是最大，也不是第二大，那么肯定是最小的了。
      cout<<c<<" "<<b<<" "<<c;
   }
	return 0;
}
```
### 方法二：数组+快速排序

定义一个数组包含>=3个元素，使用sort排序即可。

关于sort排序函数，其他博客里有详细介绍，这里不再赘述，只需要强调**区间左闭右开**。

代码如下：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int a[4];
    cin>>a[0]>>a[1]>>a[2];
    sort(a,a+3);//左闭右开。
    cout<<a[0]<<" "<<a[1]<<" "<<a[2];
}
    return 0;
```
