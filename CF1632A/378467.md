思维题。

当字符串 $s$ 的长度等于 $1$ 时，只有 $0$ 和 $1$ 这两种组合方式，都没有一个子串是回文字符串，全部输出 `YES`。

当字符串 $s$ 的长度等于 $2$ 时，有如下的几种组合方式：

$$
\begin{matrix}
& 00\\
& 01\\
& 10\\
& 11\\
\end{matrix}
$$

在上面的 $4$ 种组合方式中，$00$ 和 $11$ 是不满足没有一个子串是回文的条件的，输出 `NO`，另外两种都是满足上述条件的，输出 `YES`。

当字符串 $s$ 的长度等于 $3$ 时，有如下的几种组合方式：

$$
\begin{matrix}
& 000\\
& 001\\
& 010\\
& 011\\
& 100\\
& 101\\
& 110\\
& 111\\
\end{matrix}
$$

从这些组合方法可以看出，$|s|= 3$ 时，应该输出 `NO`。

当字符串 $s$ 的长度大于 $3$ 的时候，可以通过字符串长度等于 $3$ 不满足来推出这些都是不满足的，因为这些长度大于 $3$ 的字符串都有长度为 $3$ 的字符串。

然后判断即可。时间复杂度 $\mathcal {O(T)}$。

**Code**

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int t;
    cin >> t;
    while (t--) {
        int n;
        scanf ("%d", &n);
        string s;
        cin >> s;
        if (n > 2 || s == "00" || s == "11") puts("NO");
        else puts("YES");
    }
    return 0;
}
```
