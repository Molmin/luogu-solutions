# A.ABC

## 001 思路

考虑如果字符串长度 $\ge3$，就一定存在至少 $1$ 个长度为 $3$ 的子串，不妨设为 $XYZ$，

因为 $XY$ 是长度为 $2$ 的子串，所以 $X\ne Y$。

因为 $YZ$ 是长度为 $2$ 的子串，所以 $Y\ne Z$。

因为 $XYZ$ 是长度为 $3$ 的子串，所以 $X\ne Z$。

因为这是 $01$ 串，所以这是不合法的！

所以只有 `0`、`1`、`01`、`10` 这四个串应输出 `YES` ，其他都输出 `NO`。

## 002 代码

```c++
void run(){
	int n;
	cin>>n;
	string s;
	cin>>s;
	if(s=="0"||s=="1"||s=="01"||s=="10")
		cout<<"YES\n";
	else
		cout<<"NO\n"; 
}
```
