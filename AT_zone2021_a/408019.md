#  AT_zone2021_a UFO襲来 題解

### Part 1 题意

给定一个字符串 $S$，求 $S$ 有多少个字串 `ZONe`（大小写敏感）

### Part 2 解析

一道字符串入门题。

我们可以从这个字符串的开头开始遍历到字符串的第 $S-3$ 位，每次检查该字符及其后面的 3 个字符是否为 `ZONe`，如果是，则将答案加上 1。

核心代码如下：

```cpp
int len = s.size();
for (int i = 0; i < len - 3; i++){
	if (s[i] == 'Z' && s[i+1] == 'O' && s[i+2] == 'N' && s[i+3] =='e') ans++;
}
```

**注意事项**：

- 需要从 0 开始，因为 `std::string` 是从第 0 位存储的；
- `if` 中每个条件之间要用 `&&`，因为要同时满足。
