## 题目大意
给定 $n$ 个字符串，对于每个字符串，若其开头字符为小写字母，则转变为大写字母，对于其他字母，若为大写，则转变为小写，字母以外不做变动。
## 思路
### I 前置小知识
#### 1.
在 c++ 中有一个 ```islower( )``` 函数，会判断一个字符是否为小写字母，并返回一个整型值，还有一个函数 ```isupper( )``` ，会判断一个字符是否为大写字母，同样返回一个整型的值。
#### 2.
在 c++ 中，小写的字母的 ASCII 码与大写字母的 ASCII 码相差 32 ，因此**大写字母加 32 可转化为小写字母**，同理，**小写字母减 32 可转化为大写字母**。
### II 解法
用字符型数组存储字符串，循环判断每一位字符是否为大/小写字母，转变大小写即可。
## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,lena;
char a[30];//char型数组存字符串
int main()
{
    cin>>n;
    for(int i=0;i<n;i++)
    {
        cin>>a;
        lena=strlen(a);;//strlen返回char型数组的长度
        if(islower(a[0])) a[0]-=32;
        //判断开头字符是否是小写字母，若是，则转化为大写
        for(int j=1;j<lena;j++) if(isupper(a[j])) a[j]+=32;
        //判断其他字符是否为大写字母，若是，则转化为小写
        cout<<a<<endl;
    }
    return 0;
}
```
