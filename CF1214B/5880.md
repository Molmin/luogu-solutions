$QAQ$ $B$这么简单，早知道就先做$B$题了，害得$A$题看半天

题目给定有$N$个人参加，总共$b$个男孩，$g$个女孩。

如果$N>b$，那么编号$i$满足$i>b$的组合就不需要选

如果$N>g$，那么编号$i$满足$(N-i)>g$的组合就不需要选

题目已经给定$1\leq N \leq b+g$，所以不需要担心重复的问题。
```cpp
#include<cstdio>
int B,G,N,ans;
int main()
{
	scanf("%d%d%d",&B,&G,&N);
	ans=N+1;
	//总共有N+1个组合
	if(N>B)ans-=(N-B);
	//去掉为男孩多准备的组合
	if(N>G)ans-=(N-G);
	//去掉为女孩多准备的组合
	//注意：N≤b+g，所以不存在蓝徽章>男孩数且红徽章>女孩数的组合，即两个多准备的集合无交集
	printf("%d\n",ans);
}
```
