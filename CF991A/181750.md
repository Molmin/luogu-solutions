第一版：

容斥原理：如果被计数的事物有A、B、C三类，那么，A类和B类和C类元素个数总和= A类元素个数+ B类元素个数+C类元素个数—既是A类又是B类的元素个数—既是A类又是C类的元素个数—既是B类又是C类的元素个数+既是A类又是B类而且是C类的元素个数。（A∪B∪C = A+B+C - A∩B - B∩C - C∩A + A∩B∩C）
                                              ————摘自百度
                                              
上面这一大段话可能不太好理解，编者友情为初中的小伙伴们提供注释和图片：

![](https://cdn.luogu.com.cn/upload/image_hosting/lbaipvq7.png)
如图所示，满足条件1的写在圆圈A中，满足条件2的写在圆圈B中，两个圆圈中有重合的部分，既满足条件1，也满足条件2，用A∩B表示。而A∪B表示满足1和2均可。

例如圆圈A表示10<x<100的数，B表示2<x<20的数，那么A∩B表示10<x<100和2<x<20的公共部分，即10<x<20。而A∪B表示2<x<100。

因此，在原题中，最终答案是n-(a+b-c),即n-(A∪B)。

第二版：

逻辑运算：当我们看到∩和∪这两个符号时，一定会有一丝熟悉。他们分别代表&和|。
接下来，我们就来介绍一下逻辑运算。

&：两个数中，只要有一个数是0，结果就为0.只有在两个数都为1时才为1.

|：两个数中，只要有一个数是1，结果就为1.只有在两个数都为0时才为0.

!: 取反运算。为1时取0，为0时取1.

^: 两个数中，同数得0，异数得1.

例如25|5，严格按照法则计算，先把它们转成2进制：
```
          25：       1 1 0 0 1
          5：  |     0 0 1 0 1
              ——————————————————
                     1 1 1 0 1
      
```
再将11101转成十进制，即为29.
再例如7^5
```
       7:         1 1 1
       5:   ^     1 0 1
           ——————————————
                  0 1 0
```
所以结果为010，转成10进制是2.
