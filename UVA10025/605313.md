# UVA10025 Solution

先证明任何一个正整数都可以表示成几个不同的正整数相加的和：（废话命题）

对于正整数
$a$
，必然可以找到数字
$\Sigma _{i=1}^{n} i$
与
$\Sigma _{i=1}^{n+1} i$
，使得
$\Sigma _{i=1}^{n} i < a \le \Sigma _{i=1}^{n+1} i$

$a$ 就可以表示成：
$\Sigma _{i=1}^{n+1} i-b$ 

其中
$0\le b \le n$
，构造完毕，原命题得证。

回到原题，考虑这样一个式子：

$\Sigma _{i=1}^{n} i$

它和我们想要的式子只差负数项，那怎么办呢？我们令所有负数项之和为
$m$
，那么可以得出原题中的
$k = \Sigma _{i=1}^{n} i-2m$
，注意要减两次（先减到没有，再减到负数）

结合先证明的那个命题，我们可以得知，只要有使得等号成立的
$n,m$
就都有正整数
$k$

注意：当且仅当
$\Sigma _{i=1}^{n} i$
与
$k$
同奇偶时，有使得等号成立的正整数
$n,m$

如果
$k$
是负数，那么符号全部取相反（对于绝对值下的
$k$
）即可。。
