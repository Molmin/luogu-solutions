### 题意

求出最小的自然数$Q$，使得$Q$的各位数字乘积为$N$，如果无解，就输出`-1`。

### 题解

暴力枚举。

从小到大枚举$2$~$9$之间的数字。遇到一个可以整除的，就记录。

可是，你们有没有发现一个问题，我们的$9$就会被分成$33$，可是$ans=9$。

所以，我们考虑从大到小枚举。即从$9$~$2$枚举。

**举个例子**

比如$10$，从大到小枚举$9$~$2$，我们找到了$5$，把$10÷5$得到$2$。

之后是$2$，从大到小枚举$9$~$2$，我们找到了$2$，把$10÷2$得到$1$，这时，结束。

~~懂了吧~~。

$Code$:

```cpp
#include <cstdio>
using namespace std;

const int N=100+7;

int n,a[N];

signed main()
{
    scanf("%d", &n); // read

    for(int i=1; i<=n; i++)
    {
        int t, cnt=0; // cnt表示当前a数组中数的个数.
        scanf("%d", &t); // read.

        while(t > 1)
        {
            int flag = 0;
            for(int i=9; i>=2; i--)
            {
                if(t%i == 0)
                {
                    a[++cnt]=i;
                    t /= i;
                    flag = 1;
                    break;
                }
            } // 暴力枚举
            if(!flag) break;
        }

        if(t > 1) puts("-1"); // 如果无解，输出-1.
        else 
        {
            for(int i=cnt; i>=1; i--) printf("%d", a[i]);
            puts(""); // 记得倒叙输出
        }
    }
    return 0;
}
```
