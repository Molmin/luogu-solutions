### 题目翻译

给你一个信用卡号 $s$，判断其是否合法。

判断步骤如下：

1. 从第一位开始每次向后移动两位，把经过的数翻倍，并把它们的**每一位数**分别加起来。
2. 把没被翻倍的数字加起来，将两个结果相加。
3. 如果总数以 $0$ 结尾，则信用卡号有效，否则无效。

多组数据。

### 思路

使用字符串存储卡号，然后模拟整个过程即可。

总数以 $0$ 结尾这个条件可以用模运算解决。

即，当一个数除以 $10$ 的余数是 $0$ 时，它的结尾一定是 $0$。

### 坑点

1. 注意题目翻译中的粗体字。因为一个一位数翻倍之后位数必定小于等于两位，所以不需要写循环拆位。
2. 因为读入字符串包含空格，所以可以选择循环读入然后再拼起来，也可以使用 `getline`（不推荐，因为会在后续操作中增加代码难度）。
3. 字符转数字要减字符 `0`（的 ASCII 码，也可以直接减字符常量）。
4. 输出字符串开头字母要大写。
5. 多测记得清空（或者变量定义在多测循环内）。

### 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int q;
signed main()
{
	cin>>q;
	while(q--)
	{
		string s,t;
		for(int i=1;i<=4;i++) cin>>t,s+=t;//循环读入
		int ans=0,x;
		for(int i=0;i<s.length();i+=2) //处理偶数位（翻倍）
			x=(s[i]-'0')*2,ans+=x%10+x/10;//算出翻倍后的结果（注意减字符0）然后拆位。
		for(int i=1;i<s.length();i+=2)//处理奇数位（不翻倍）
			x=s[i]-'0',ans+=x;
		
		if(ans%10==0) puts("Valid");//判断是否合法
		else puts("Invalid");
	}
    return 0;
}

```
