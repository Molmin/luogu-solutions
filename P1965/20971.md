然后这个题实际上就是快速幂的一个模板题，不理解快速幂的同学硬背的话快速幂模板也不是很难背，下面就是代码：



```cpp
#include<iostream> 
using namespace std;
long long n,m,k,x;
long long fast(long long a,long long n,long long p)//快速幂，注意加long long，不加好像也可以，不会写的话背下来也可以
{
    long long s=1;
    while(n)
    {
        if(n&1)
            s=(s*a)%p;
        a=(a*a)%p;
        n=n>>1;
    }
    return s;
}
int main()
{
    cin>>n>>m>>k>>x;
    cout<<(x+(m*fast(10,k,n)%n)%n)%n;//注意多模几次没有坏处，好像对大佬们也是适用的 
    return 0;
}
推荐萌新们多背一些模板，比如说并查集，Kruskal，spfa，线性筛，LCA，快速幂，RMQ，总之会有用的（大佬们可以无视）
```
