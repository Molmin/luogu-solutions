## 题目大意：
给定一个区间 $[ l,r   ]$，求区间内的数的因子有奇数个的个数。

其中有一个小问题：
为什么是奇数就可以呢？

- 因为 $1$ 的倍数相当于把所有的灯都关上了，按了按钮了奇数次，灯就是亮的！


##  是不是非常 的简洁明了？有没有豁然开朗？

### 解题思路
- 只有完全平方数的因数个数为奇数个！

为什么呢？因为完全平方数有一个单一的因子，而不是完全平方数的数的因子都是成对出现的！

- $\ \sqrt{n} $ 可以求出从 $1$ 到 $n$ 中的平方数个数！

所以我们只需要求出 $[ l,r   ]$ 区间内的元素个数，再减去其中完全平方数的个数就好了！

代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
long long n,l,r; //不要忘记开long long
int main(){
	cin>>n>>l>>r;
	long long s;
	s=r-l+1; //[l,r]区间内的元素个数
	long long sl,sr;
	sl=sqrt(l-1); //1到l-1中的完全平方数个数
	sr=sqrt(r); //1到r中的完全平方数个数
	//有点像前缀和思想
	cout<<s-(sr-sl); //总个数减去完全平方数的个数
  	return 0;
}
```
