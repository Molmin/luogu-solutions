本题考主要小学数学。

首先要知道对于这道题来说唯一的分解定理:n只能分解为两个质数的积。可以推导出，若输入的数满足题目条件，n就只能分解为两个质数的乘积,所以在比n小且大于3的自然数中，只有两个质数能除n，之间不可能再有任何合数或质数能整除它了，所以，既然n=MAX*MIN的话，我们只要从大到小枚举，找出最大的那个质数就行了。并且，当n≡0(modi)时，因为只有两个质数能除n，所以当从大到小搜到i时，这道题不需要判断，直接输出就行了。
所以代码就很容易实现了！
```
#include<bits/stdc++.h>//万能头文件
using namespace std;
int main()
{
	int n;//要分解的那个数
	cin>>n;
	for(int i=n-1;i>=2;--i)//从大比从小搜更快，因为必定分成两个质数，所以i>=2
		if(n%i==0)//当符合条件时
		{
			cout<<n<<endl;//输出答案，最好加一个回车换行
			return 0;//求过qwq
		}
}
