打这道题的时候有人会超时

主要是因为一个数学的小问题

一个数除了1之外最小的因数肯定是质数

[证明过程略]

而且题目说了，给出的数字是两个质数的乘积

那就保证了另一个因数肯定是质数

自己拿数据测的时候也要注意符合这个条件

输个50就有趣了

冒出来个25然后纠结自己哪错

[对就是我]

其实找到除了1之外的最小因数

再用n除以这个数就好了

送一发C++程序

```cpp
#include<iostream>
using namespace std;
int i,n;
int main()
{
    cin>>n;
    for (i=2;i<=n;i++)//注意从2开始。但是循环到多少无所谓，反正找到就break，不过要大于sqrt(n) 
    {
        if(n%i==0)
        {
            cout<<n/i<<endl;//要找大的那个而不是小的 
            break;//找到就退出，因为只要求最大的因数 
        }
    }
}
```
