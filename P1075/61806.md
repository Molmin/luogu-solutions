##### 这道题目非常的简单
楼下的很多大佬都是直接的把最小的因数找出来

由于本题中说明了这个数一定不是质数，所以这个数的最小的因数既然不是1，就肯定是质数了

但是本蒟蒻为了保险起见，还是打了一个判断质数的函数（虽然确实没必要），给新手程序猿们参考

下面是本蒟蒻的代码：

```
#include <bits/stdc++.h>
using namespace std;
int n;
int zs (int x)  //判断是否为质数
{
	for (int i=2;i*i<=x;i++)
		if (x%i==0) return 0;  //有1以外的因数就肯定不是质数了，返回0
	return 1;   //返回1
}
int main()
{
	cin>>n;  //输入n
	for (int i=2;i<=n;i++)  //注意要从2开始循环，1能被任何整数整除
		if (zs(i) && n%i==0) { //如果i是质数，而且能被n整除
			cout<<n/i;  //i是n的最小质因数，又因为题意说明是两个质数之积，那么n/i就一定是最大的质因数了
			return 0;   //输出完后就直接中断程序
		}
}
```
