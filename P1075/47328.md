# 首先，你要了解唯一分解定律
就是一个数只能分成两个质数的积。若输入的数满足题目条件，就只有两个数相乘等于n，之间没有任何数相乘为n。

所以我们可以从最小的因数开始找，之后用n/它就可以了！

代码还是很简短的！
```cpp
#include <iostream>
using namespace std;
int main()
{
	int n;
	cin>>n;
	for(int i=2;i<n/2;i++)//从最小的开始找，n/2是为了优化，大于n/2就说明没有 
		if(n%i==0)
		{
			cout<<n/i; 
			return 0;
		}
}
```
