#讲真这道题开始没读懂题，数据没想清楚，我觉得下面的题解都没说清楚（个人感觉）首先任意一个合数都可以被分解成数个素数的乘积（不一定是两个），比如说20分解质因子为2\*2\*5，显然无法分解成两个质数之积（三个），所以数据一定是可以分解成两个质数之积的我们不必担心（我开始打出一个大表从头搜TLE,RE一堆），如果一个数可以被一个合数整除那它必然无法之分解成两个质数之积（多个），因此我们只需要直接去找能整除n的最小数（除1外），取平方根加速即可

    
```cpp
#include <iostream>
#include <cstdio>
#include <cmath>
using namespace std;
int main(){
    int n;
    int i;
    cin >> n;
    for(i = 2 ; i <= sqrt(n) ; i++){
        if(n % i == 0)//找能整除n的最小数
            break;
    }
    cout << n / i;//即为能整出n的最大质因子
    return 0;
}
```
