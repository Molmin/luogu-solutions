题目描述

		一般的文本编辑器都有查找单词的功能，该功能可以快速定位特定单词在文章中的位置，有的还能统计出特定单词在文章中出现的次数。
		现在，请你编程实现这一功能，具体要求是：给定一个单词，请你输出它在给定的文章中出现的次数和第一次出现的位置。注意：匹配单词时，不区分大小写，但要求完全匹配，即给定单词必须与文章中的某一独立单词在不区分大小写的情况下完全相同（参见样例1 ），如果给定单词仅是文章中某一单词的一部分则不算匹配（参见样例2 ）。
输入输出格式

		输入格式：
			共2行。
			第1行为一个字符串，其中只含字母，表示给定单词；
			第2行为一个字符串，其中只可能包含字母和空格，表示给定的文章。
		输出格式：
			一行，如果在文章中找到给定单词则输出两个整数，两个整数之间用一个空格隔开，分别是单词在文章中出现的次数和第一次出现的位置（即在文章中第一次出现时，单词首字母在文章中的位置，位置从0开始）；如果单词在文章中没有出现，则直接输出一个整数-1。

输入输出样例

		输入样例#1： 
	To
	to be or not to be is a question
		输出样例#1： 
	2 0
    
		输入样例#2： 
	to
	Did the Ottoman Empire lose its power at that time
		输出样例#2： 
	-1
说明

		数据范围
		1≤单词长度≤10。
		1≤文章长度≤1,000,000。
		noip2011普及组第2题
思路

		1.因为不区分大小写，所以把输入全转换为小写或大写。
        2.因为给定单词仅是文章中某一单词的一部分则不算匹配，所以我的思路是定义一个结构体，用一个存从文章分解出来的单词，一个存该单词在文章中的位置。
        3.用一个循环来分解文章，然后再用一个循环来配对指定单词，记下次数和第一次出现的位置。
        4.如果次数为零，那就输出-1；如果不为零，就输出次数和第一次出现的位置。
        5.别忘了最后加上return 0;
程序(仅供参考)

	#include<iostream>
	#include<string>
	#include<cstring>
	using namespace std;
	struct node//定义结构体
	{
		string dc;//存从文章分解出来的单词
		int wz;//存该单词在文章中的位置
	}
	shu[1000001];//由于不知道单词数，所以用单词的总长度，以确保不会溢出
	int main()
	{
		string dc;//需配对的单词
		getline(cin,dc);//输入一整行，遇回车结束
		for(int i=0;i<dc.size();i++)//转小写
			if(dc[i]>='A'&&dc[i]<='Z')
				dc[i]=char(dc[i]-'A'+97);
		string wz;
		getline(cin,wz);//输入一整行，遇回车结束
		for(int i=0;i<wz.size();i++)//转小写
			if(wz[i]>='A'&&wz[i]<='Z')
				wz[i]=char(wz[i]-'A'+97);
		int shul=1;//记分解的单词数
		int book=1;//为了避免文章中单词与单词中不只一个空格，加个条件
		for(int i=0;i<wz.size();i++)//分解
		{
			if(wz[i]==' '&&book==1)//如果有空格，表示已经分解了一个单词，单词数+1，但为了避免文章中单词与单词中不只一个空格，所以要判断是否真分解了一个单词
			{
				shul++;//单词数+1，表示该分解下一个单词了
				book=0;//单词未分解
			}
			if(wz[i]>='a'&&wz[i]<='z')//如果是字母，表示该分解了
			{
				book=1;//单词已分解
				shu[shul].wz=i;//记下该单词在文章中的位置
				while(wz[i]>='a'&&wz[i]<='z')//从文章把该单词分解出来
				{
					shu[shul].dc+=wz[i];//加上分解字符，从而组成单词
					i++;//i+1，下一个
				}
				i--;//因为该循环结束后，i会+1，可是我们要它不变（好好理解）
			}
		}
		int ans=0;//用来记次数
		int ii;//用来记第一次出现的位置
		book=1;//因为只记第一次出现的位置，所以加个条件
		for(int i=1;i<=shul;i++)//枚举分解的单词
			if(shu[i].dc==dc)//配对
			{
				ans++;//次数+1
				if(book==1)//只记第一次出现的位置
				{
					ii=shu[i].wz;
					book=0;
				}
			}
		if(ans==0)//是否在文章有这个单词
			cout<<"-1"<<endl;
		else
			cout<<ans<<" "<<ii<<endl;
		return 0;
	}
