## 题目大意

一条街上从左到右有 $n$ 座房子，相邻之间间距为 $1$。每天每座房子都会有一个葡萄酒交易量，负数表示卖出，正数表示买入。将 $1$ 单位葡萄酒搬运 $1$ 距离需要 $1$ 代价。给定一天所有房子的交易情况，求完成交易的最小代价。

## 思路

首先，一个很明显的事实是：没有被买入或者被卖出的葡萄酒会一直累积，直到他被买入。注意到这个累积的性质，我们考虑用前缀和维护。

我们用 $sum_i$ 表示到第 $i$ 座房子时累积的葡萄酒有多少，那么从 $i-1$ 座房子运送到到 $i$ 就需要 $sum_i$ 的代价。我们从前往后求前缀和，同时累计答案。

另外，累积的答案应该是 $sum_i$ 的绝对值。假如当前被卖出的量小于要买入的量（也就是 $sum_i<0$），仍然需要之后的卖出来补充。也就是说，不管当前 $sum_i$ 的正负，运输的代价是一直存在的。

原题中数据范围没了，但为了保险我们开 `long long`。代码如下：

```cpp
signed main(){
	while(1){
		n=read(),ans=0;if(!n) return Strelitzia_H;
		memset(sum,0,sizeof(sum));
		for(int i=1;i<=n;i++) sum[i]=read();
		for(int i=1;i<=n;i++){
			sum[i]+=sum[i-1];
			ans+=abs(sum[i]);
		}
		printf("%lld\n",ans);
	}
	return 0;
}
```
