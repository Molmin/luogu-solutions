Vladimir发明了一种广义俄罗斯套娃，把套娃的容积增大了，一个里面可以装多个。为了方便记录，我们用负数表示套娃的上半身，正数表示下半身。现给出一个套娃序列，请判断他是不是一个合法的广义俄罗斯套娃。

举个栗子：
```
-1 -2 2 -3 3 -4 4 -5 5 1
```
就表示在一个大的 $1$ 号套娃里套着四个小套娃，分别是 $2$,$3$,$4$ 和 $5$，是一个合法的广义俄罗斯套娃。

我们可以用栈来存每一次输入的套娃序列，这里就直接使用C++里的`stack`库了，不过记得写`#include<stack>`。

每次我们输入到一个数，都需要分情况，如果是负数就入栈，正数就跟栈顶配对。计算部分代码：
```cpp
stack<int> s;//注意格式
bool sta=1;//表示是否成功 
for(int i=1;i<=n;i++){//检查每一个数，假设n是数的个数，a是广义套娃数组 
	if(a[i]<0) s.push(a[i]);//负数
	else{//正数 
		if(s.empty()||s.top()!=-a[i]){//如果栈是空的或者第一个跟他不匹配 
			cout<<":-( Try again."<<endl;//符号表情好评
			sta=0;//不是广义套娃 
			break;//既然不行了，就退出 
		}
	}
}
if(sta) cout<<":-) Matrioshka!"<<endl;//是广义套娃 
```
