注意循环时第一重循环的是 $f(x)$，第二重循环的是 $x$。因为画图是从上到下，而坐标是从下往上的，所以在循环 $f(x)$ 的时候要从 $m-1$ 开始循环到 $0$。

当循环到一个点 $(x,y)$ 的时候，求出 $f(x)$ 的值，如果 $y=f(x)$，则在这点输出 `*`，否则输出 `.`。

核心代码：

```cpp
for(int y=m-1;y>=0;y--)
{
    for(int x=0;x<n;x++)
    {
        int fx=0;
        for(int i=0;i<=k;i++)
            fx+=a[i]*pow(x,i);
        if(y==fx)
            cout<<'*';
        else
            cout<<'.';
    }
    cout<<'\n';
}
```
