本题题意的翻译给的十分清楚，不过多解释。

首先完全平方数是指：对一个数$x$，如果存在一个数$y$，使其满足$x=y^{2}$，则称$x$为完全平方数。

接下来说下本蒟蒻的思路。题目要求输出最大的那个非完全平方数，那么我们可以在排序时将完全平方数去掉，然后由大到小排序非完全平方数即可。

具体的实现方法是，利用C++中algorithm库中的sort函数，sort函数支持自定义比较器，所以我们可以手写一个cmp比较函数来达到我们的排序目的。

cmp函数写法是，对于要比较的两个数，先取它们各自的平方根再对其取整，如果此数的平方与原数相等，则将此数去掉，最后按降序排序。写完的cmp函数如下：
```cpp
bool cmp(int a,int b)
{
	int sa=int(sqrt(a));//取平方根再对其取整。
	int sb=int(sqrt(b));
	if(sa*sa==a)a=-2147483647;//如果此数的平方与原数相等，则将此数去掉，去掉的方式是将其设为最小值。
	if(sb*sb==b)b=-2147483647;
	return a>b;//按由大到小降序排序。
    /*这里的意思是：如果a>b，则返回1；a<=b，则返回0；
   	而在sort的自定义比较函数中，返回1代表不将两数交换，返回0代表将两数交换，故这样写。*/
}
```
