#### 题目翻译：
你的一个朋友告诉你，你所在城市附近的一家剧院有新的演出机会。它将放映超级热门的施瓦辛格电影《终结者 $2$》。在售票时，他们会记录买主的生日。如果某人的生日和在他之前买票的人的生日相匹配，他（在后买票的人）将有幸与施瓦辛格共进晚餐。为了给排在队伍前面的人一些机会，他的生日将与卖票人的生日进行比较。如果给你机会，你想排到哪个位置?（我知道你们很多人都想要其他名字作为晚餐伙伴。不要犹豫，把施瓦辛格换成这个名字吧，因为你可能需要所有这些灵感来解决这个问题）。

输入文件包含几行输入。每行包含一个整数 $N$ （$30 < N < 100001$），表示一年有多少天。输入以文件结尾结束。

对于每一行输入，在一行中输出一个浮点数和一个整数，用一个空格分隔。浮点数是你在队列中的最佳浮点位置（小数点后有两位数字），整数是你在队列中的最佳整数位置(实际位置必须是一个整数）。

### 题目分析：

本质上给你一个数 $n$，求关于 $a$ 的方程 $a^2+a=n$ 的正根（保留两位小数）与其向上取整的整数。

使用一元二次方程求根公式即可。

### 参考代码：

```cpp
#include<bits/stdc++.h>
#define ll long long
using namespace std;
int main()
{
	ll n;
	double x;
	while(cin>>n)
	{
	x=(sqrt(1+4*n)-1)/2;
	n=x+1.0;
	printf("%.2lf %d\n",x,n);
	}
}
```
