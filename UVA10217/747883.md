你的一个朋友告诉你，你所在城市附近的一家剧院有新的演出机会。它将放映超级热门的施瓦辛格电影《终结者 
2》。在售票时，他们会记录买主的生日。如果某人的生日和在他之前买票的人的生日相匹配，他（在后买票的人）将有幸与施瓦辛格共进晚餐。为了给排在队伍前面的人一些机会，他的生日将与卖票人的生日进行比较。如果给你机会，你想排到哪个位置?（我知道你们很多人都想要其他名字作为晚餐伙伴。不要犹豫，把施瓦辛格换成这个名字吧，因为你可能需要所有这些灵感来解决这个问题）。
## 说~~人话~~数学语言
已知 $n$。

有一个无限长的整数序列，对于序列中的每个整数 $\texttt{a[i]}$ 都有 $0<\texttt{a[i]}≤n$。

你需要找到一个 $m$，使得 $\texttt{a[1]}$～$\texttt{a[m-1]}$ 各不相同，且 $\texttt{a[m]}$ 与某个 $\texttt{a[k]}(0<k<m)$ 相等的概率最大。

## 分析
我们首先考虑 $\texttt{a[1]}$～$\texttt{a[m-1]}$ 各不相同的概率。

显然，如果没有任何限制条件，这 $m-1$ 个数共有 $n^{m-1}$ 中不同的搭配，其中各个数互不相同的情况数为 $\dfrac{n!}{[n-(m-1)]!}$ 种，因此根据公式可以得到：
$\texttt{a[1]}$～$\texttt{a[m-1]}$ 各不相同的概率为 $\dfrac{n!}{[n-(m-1)]! \cdot n^{m-1}}$

再结合第 $m$ 个数与前面 $m-1$ 个数相同的概率 $\dfrac{m-1}{n}$，得到：

记第 $m$ 个数满足条件的概率为 $P(m)$，则$P(m)=\dfrac{n! \cdot (m-1)}{[n-(m-1)]! \cdot n^{m-1} \cdot n}$。

要使 $P(m)$ 最大，则必有 $P(m)>P(m+1)$，故 $\dfrac{P(m)}{P(m+1)}>1$，代入并化简得：

$m^2+m>n$

这一题中只用取等就可以了。

（题外话：除了我之外的另两篇题解也不知道为什么可以直接取等，望大佬详解，如果我知道了答案，我会马上更新此篇题解）

写成标准形式：
$m^2+m-n=0$

套用求根公式得：$m=\dfrac{-1±\sqrt{1+4\times n}}{2}$

因为$m>0,n>0$，所以取 $m=\dfrac{-1+\sqrt{1+4\times n}}{2}$

## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
	int n;
	double m;
	while(cin>>n)//本题有多组数据
	{
		m=(sqrt(1+4*n)-1)/2;//计算 m
		printf("%.2lf ",m);//先输出一个两位小数浮点数
        printf("%.0lf\n",m);//因为一个序号只能是整数，再输出一个整数
	}
    return 0;
}
```
