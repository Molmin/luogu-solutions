作为这道题在洛谷的第 $4$ 个 AC 者，我来讲一下这道题。

## 题意
高桥发布了用 $15$ 位数字制作的代码 $S$。$S$ 是只由 $0\ldots9$ 组成的长度为 $15$ 的字符串。

这个 $15$ 位代码的最右边的位数为校验码，可以从剩下的 $14$ 位开始计算。

首先，把左边 $14$ 位中从左边开始奇数的数（把最左边的数计为第 $1$ 位）全部相加，把那个值乘以 $3$。然后，将从左边开始的所有偶数位数字加到这个相乘的值上。该值除以 $10$ 后的余数与第 $15$ 位的数一致是正确的代码，不一致是不正确的代码。

请代替高桥编写判定代码 $S$是否正确的程序。

## 分析
- 用 $sum$ 表示前 $14$ 位中奇数位数字的总和，加完后再乘 $3$；
- 用 $sum1$ 表示前 $14$ 位中偶数位数字的总和；
- 将 $sum$ 与 $sum1$ 相加再除以 $10$，判断余数是否等于第 $15$ 个数即可。

## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	string s;
	cin>>s;
	int sum=0,sum1=0;
	for(int i=0;i<s.size()-1;i++)//i<s.size()-1是为了不算第15位
	{
		if((i+1)%2==1) sum+=s[i]-'0';//奇数位
		else sum1+=s[i]-'0';//偶数位
	}
	sum=sum*3+sum1;//乘三再相加
	if(sum%10==(s[14]-'0')) cout<<"Yes"<<endl;//余数为第15个数
	else cout<<"No"<<endl;//余数不为第15个数
    	return 0;
}
```
