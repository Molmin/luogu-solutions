## 题目大意

给定 $2$ 个数 $n$ 和 $k$。问能否在 $1-n$ 这 $n$ 个数（包括 $n$ ）中选择 $k$ 个不同的整数使它们的差不等于 $1$。

## 题目分析
由于要使 $2$ 个数相减 $≠1$，所以我们选数的时候，不能选连续自然数，不然的话两个连续自然数相减必定是 $1$（如：$5-4=1$）。所以，我们只要判断 $k$ 是否小于等于 $n \div 2+1$ 就可以了。

## AC Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int n,k;

int main()
{
	cin>>n>>k;
	
	if(k<=(n+1)/2)
		cout<<"YES";
	else
		cout<<"NO";
}
```
