------------
这道题就是叫你求x,y,z，满足以下条件

```
X+Y+Z=S

0<=X，Y
      
Z<=k
```

求一共有几组x,y,z



------------
让人想到的大概是枚举三个数，但是如果k卡到3k，s卡到2500，我们的o(n^3）程序就会爆掉，我们就要减掉一重循环



------------
上代码

```c
#include<stdio.h>
int main()
{
	int k,s,ans=0,x,y;
	scanf("%d%d",&k,&s);
	for(x=0;x<=k;x++)
	{
		for(y=0;y<=k;y++)
		{
			if(s-x-y>=0&&s-x-y<=k)ans++;//可以在这里加一层判断，直接通过题目的X+Y+Z=S求出z，就可以少一重，将n^3变为n^2
		}
	}
	printf("%d\n",ans);
	return 0;
}
```
