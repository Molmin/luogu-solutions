# CF177A C++题解
## 题目大意
- 有 $1$ 个边长为 $n$ 的矩阵。
- 每个元素包含 $1$ 个整数。
- 求出该矩阵的两条对角线、中间列、中间行所有元素之和。
- $n$ 为奇数且不超过100

## 题意分析
这是一道模拟题。

我发现这道题主要有两种做法，时间复杂度均为 $O(n^2)$。
### 第一种
解题思路：将矩阵存入二维数组，然后用循环找出每一个指定的数，存储到 sum 变量并输出。

优点：这种做法很好看出来，实现起来也很容易。

缺点：

当 $n$ 过大时，

- 假设空间限制很严格，很可能无法开出可以完整存储矩阵的二维数组，那么这种做法就会被淘汰。

- 如果空间限制很严格，可能还没有用循环找出所有指定的元素，就超时了，那么这种做法也会被淘汰。

### 第二种
解题思路：在读入每一个元素时判断是否为指定的元素，如果是，存储至 sum 变量，最后输出。

优点：这种方法更为简便，代码量较小，而且不需要用到二维数组，可以完美解决第一种方法所述的缺点。

这种做法应该是最优解了。
## 代码
这是第二种做法的核心代码。
```cpp
if(i==n/2||j==n/2||i==j||i+j==n-1)
    sum+=a;
```
解释：

$i,j$是坐标，$a$ 是元素的值，$n$ 是矩阵的边长，`sum`是指定元素的和。

`i==n/2`和`j==n/2`判断的是中间行、中间列的元素。

`i==j`判断的是由左上至右下的对角线。

`i+j==n-1`判断的是由右上至左下的对角线。

完整代码
```cpp
#include<iostream>
using namespace std;
int a,sum,n;
int main()
{
    cin>>n;
    for(int i=0;i<n;i++)
        for(int j=0;j<n;j++)
        {
            cin>>a;	//读入元素
            if(i==n/2||j==n/2||i==j||i+j==n-1)	//上述的核心代码
                sum+=a;
        }
    cout<<sum;	//输出结果
    return 0;
}
```
