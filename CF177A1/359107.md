### 题意：

给出一个 $n×n$ 的矩阵，求它的对角线、中间行、中间列之和。

坑点：所有的数只能算一遍，不能重复！！！

---

### 分析：

为了不把重复的的数再算一遍，我们需要再开一个数组把每个算过的数标记一下，以后再加和的时候就不再加上标记过的。

1. 中间行就是纵坐标都为 $\frac{n}{2}+1$ 的数；

2. 中间列就是横坐标都为 $\frac{n}{2}+1$ 的数；

3. 左上-右下对角线的横坐标等于纵坐标；

4. 右上-左下对角线的横坐标加纵坐标等于 $n+1$ ；

依据以上四条在输入时进行判断、标记、和加和即可。

---

### $AC$ $Code$

```cpp
#include<iostream>
using namespace std;
bool flag[110][110];  //用布尔型可以减小内存占用
int main()
{
	int x,n,sum=0;
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		for(int j=1;j<=n;j++)
		{
			cin>>x;
			if(i==j&&flag[i][j]==0)  //判断左上-右下
			{
				sum+=x;
				flag[i][j]=1;  //标记
			}
			if(i+j==n+1&&flag[i][j]==0)  //判断右上-左下
			{
				sum+=x;
				flag[i][j]==1;  //标记
			}
			if(i==n/2+1&&flag[i][j]==0)  //判断中间行
			{
				sum+=x;
				flag[i][j]=1;  //标记
			}
			if(j==n/2+1&&flag[i][j]==0)  //判断中间列
			{
				sum+=x;
				flag[i][j]=1;  //标记
			}
		}
	}
	cout<<sum;
	return 0;
}
```
