翻译什么的柯怀俊2333已经说过了，此题简单易懂，一个模拟搞定，我的方法在一般情况下跑的比柯怀俊2333的题解快。
```cpp
for(i=1;i<=n;i++)
    {
     ans=ans+a[i][(n+1)/2];
     a[i][(n+1)/2]=0;
     ans=ans+a[(n+1)/2][i];
     a[(n+1)/2][i]=0;
     ans=ans+a[i][i];
     a[i][i]=0;
     ans=ans+a[i][n-i+1];
     a[i][n-i+1]=0;
    }
```
a数组是地图，ans代表答案。
