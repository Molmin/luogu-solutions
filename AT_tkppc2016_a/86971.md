此题大概的题意，根据本蒟蒻的理解，就是：输入两个字符串，把它们连接起来。那么此题就是考察字符串的输入输出   
下面给出我们的蒟蒻代码  
```
#include <stdio.h>//C党万岁！
int main()
{
	char a[51],b[51];
	gets(a);gets(b);//gets字符串读入，也可以换成scanf("%s",a);scanf("%s",b);
	printf("%s",a);//注意输出不能用puts，要不然自动换行！
	printf("%s",b);
}
```
