题目意思：

有n个王子和n个女孩，每个王子可能喜欢多个女孩。
先给出一个初始的完备匹配，问每个王子可以选择哪些女孩，即无论王子选择这些女孩中的哪一个，使得剩下的王子仍能够选择喜欢的女孩。

for (int u=1; u<=n; u++)
{
   int tail=0;
   for (int i=head[u]; i; i=edge[i].next)
   {
   		int v=edge[i].to;
      if (iss[u]==iss[v])  
         ans[tail++]=v-n;
   }
   sort(ans,ans+tail);
   printf("%d",tail);

      printf(" %d",ans[i]);

}
