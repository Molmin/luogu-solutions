### 题目大意：
给你 n 张 $2$ $\times$ $2$ 的牌，问你经过旋转（不能翻转） $90^\circ$ ,  $180^\circ$ , $270^\circ$ 后有几张不相同的牌。
## 大体思路：
对于**任意**一个卡牌：
| **a** | **b** |
| :----------: | :----------: |
| **c** | **d** |

经过一次向左旋转 $90^\circ$，都**绝对**会变成这个样子： 

| **c** | **a** |
| :----------: | :----------: |
| **d** | **b** |

可以发现，$a$ 走到了 $b$ 的位置 ; $ \quad  $ $b$ 走到了 $d$ 的位置 ; $ \quad  $ $d$ 走到了 $c$ 的位置 ; $ \quad  $ $c$ 走到了 $a$的位置 。

而每一个没出现过的卡牌要进行 $4$ 次这样的操作。所以我写了一个和别的大佬不一样的代码，用 $for$ 代替了手推。

### 核心代码


```cpp
char A,B,C,D,e,f;
cin>>A>>B>>C>>D;
int a=A-'0',b=B-'0',c=C-'0',d=D-'0';
if(vis[a][b][c][d]==0)

	{
		int tmp=a;//拿个东西存下a
		a=c;
		c=d;
		d=b;
		b=tmp;
		vis[a][b][c][d]=1;//当前卡牌
	}
}
```
