理解了就懂了，题目要求的数组去重后是一个首项为$1$，公比为$2$的等比数列。所以尽可能小，就是先都放$1$，再放完一组L项的等比数列；尽可能大，就是先放完一组$L$项的等比数列，再都放$2^{r-1}$，可以保证一定是是极端数据。

于是一行输出，一行输入，超简洁代码来了：

```cpp
#include <cstdio>
#include <cmath>
int main(){
    int n,l,r;
    scanf("%d%d%d",&n,&l,&r);
    printf("%d %d",(int)(pow(2,l)-1)+n-l,(int)(pow(2,r)-1)+(n-r)*(int)pow(2,r-1));
    return 0;
}
```
