### 题意解析
这是一个倒水问题。现在知道有两个桶，容积分别为 $x$ 升和 $y$ 升。现在需要 $z$ 升水，问是否能通过两个桶，倒出 $z$ 升水。   
一共有 $T$ 组数据，每组数据输入 $x$，$y$，$z$，如题意。请判断是否能通过容积为 $x$，$y$ 的桶，倒出 $z$ 升水。$x,y,z \le 10^8$。
### 样例分析
样例输入：
```python
5
2 4 3
2 5 1
9 3 6
3 8 7
6 1 10
```
解析：    
* 第一组数据，无法用偶数升的水的桶倒出奇数升水，不可以。
- 第二、三、四组数据，都可以实现。
- 第五组数据，两个桶的总容积未达到需要水量，不可以。  

样例输出：
```python
NO
YES
YES
YES
NO
```
### 解题过程
第一步，输入数据组数。
```python
dt=input()
dt=int(dt)
```
第二步，定义最小公倍数，具体定义方法借用 C++ 的最小公倍数函数。如果有 0 存在，返回那个非 0 的数，否则返回第二个数和第一个数取余第二个数的最小公倍数。
```python
def gcd(e,i):
	if i==0:
    	return e
    return gcd(i,e%i)
```
第三步，循环进行数据判断。下面我逐行解释一下代码含义。
* 第一步，题目中说一行输入三个整数，我们用字符串 $a$ 来存储他们。
* 第二步，重新定义 $a$，使得 $a$ 变成三个分开的数。
* 第三步，将这三个字符串型的裂变中的每一项转换成整形的。
*

while dt:
