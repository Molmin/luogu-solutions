**开垦无人区题解$……$**

这是一道很鬼畜的题目，第一眼可能会认为这题很难，然而仔细看一下题目给的算式，就会发现这题实在$……$

我们来看一下题目给的算式$:$

$$00000000\ 00000000\ 00000000\ 00000100$$

$$+$$

$$00000000\ 00000000\ 00000000\ 00000110$$

$$\Downarrow$$

$$00000000\ 00000000\ 00000000\ 00000010$$

我们来试着提取一下有效信息，即它的运算法则，然后我们发现$:$

- $0+0=0$

- $0+1=1$

- $1+1=0$

- $1+0=1$

然后我们发现，这~~TMD~~不就是异或吗$?!$

于是，我们就愉快的解决了这道题$QwQ$，上代码$:$

**code:**

```cpp
#include <iostream>
#include <cstdio>
#include <cstring>

int a, b;

int main() {
    while (scanf("%d%d", &a, &b) != EOF) {
        printf("%d\n", a ^ b);
    }
    return 0;
}
```
