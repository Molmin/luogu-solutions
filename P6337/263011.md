其实这道题是可以打表的

我们来找找规律,先算前面几个:$2,4,6,9,12,16,20,25,30,36,42,49,56...$可以发现从第四个数开始,每过两个数,这个数与前面那个数的差就加$1$。由此我们可以推导出规律

那怎么证明呢?

首先,与长平行的刀数必须和与宽平行的刀数尽量接近(和定差小积大),每切一刀与长边平行的,就必须切一刀与宽边平行的。假设现在与宽平行的比与长平行的多切$1$刀,则此时该切与长平行的,增加的面数为与宽平行的刀数$+1$。然后再切与宽平行的,增加的面数为此时与长平行的刀数$+1$(也就是切上一刀时与宽平行的刀数$+1$),两次增加的面数一样。然而切下一刀时增加的面数就比原来多$1$了(现在与宽平行的刀数$+1$)。由此可证明

所以打表程序应从$4$(从$2$也行)开始循环,每当循环变量$i$ $\bmod$ $2=1$,每次增加的面数$+1$

打表程序:

```cpp
#include<bits/stdc++.h>
using namespace std;
int a[105]={0,2,4,6},x=3;//初始2,4,6,开始每次加3
int main(){
	for(int i=4;i<=100;i++)
	{
		a[i]=a[i-1]+x;
		if(i%2==1)
			x++;
	}
	cout<<0;
	for(int i=1;i<=100;i++)
		cout<<","<<a[i];
	return 0;
}

```
