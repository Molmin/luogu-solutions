## 题目大意
本题很好理解，就是在一个矩形棋盘上，切与矩形的边平行的 $n$ 次，问最多棋盘能被切成多少块。
## 大体思路
我们可以来看：如果只是横着切或竖着切，显然块数不是最多。因此切法应该是横竖交替。这就意味着本题要分奇偶讨论。先来上规律：
|刀数  |块数  |
| -----------: | -----------: |
| 1 |2  |
|2 | 4 |
|3 | 6 |
|4  |9  |
|5  |12  |
|6  |16  |
|7  | 20 |
|8  | 25 |
也就是说，本题的第一种方法已经浮出水面：~~打表大法qaq~~

正经的做法是：可以发现当刀数 $n$ 为偶数时，块数是一个完全平方数 $(n/2+1)^2$。而当刀数为奇数时，块数为 $(n+1)×(n+3)/4$。这样代码就显而易见了：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n;
    cin>>n;//输入
    if(n&1) //判断奇偶，相当于n%2==1
    	cout<<(n+1)*(n+3)/4;
    else cout<<(n/2+1)*(n/2+1)//代入公式
    return 0;//完美
}
```
