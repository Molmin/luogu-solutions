来发一篇题解吧。

首先，有一个非常重要的结论便是，如果想要切的块更多的话，横着切的刀数要与竖着切的刀数**尽可能接近**。

所以为了让他们更接近，我们必须让它们的差为$1$或$0$。

所以我们得出如下结论：

$n$为偶数：横着切$n/2$刀，竖着切$n/2$刀。

$n$为奇数：横着切$n/2$刀，竖着切$n/2+1$刀。

最后还要注意是$n$刀$n+1$段，所以：

$ans=($横着切的刀数$+1)\times ($竖着切的刀数$+1)$

然后就可以上代码了：

```cpp
#include<bits/stdc++.h>
using namespace std;
long long a;
int main(){
	cin>>a;
	a%2==1?cout<<(1+a/2)*(2+a/2):cout<<(1+a/2)*(1+a/2);
}
```
