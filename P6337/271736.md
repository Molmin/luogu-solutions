# STEP1
因为这道题是矩形切割的，而矩形平行的边只有横着或者竖着的，我们可以发现，当两条线相交的时候，被分割的个数最多。所以我们要尽可能的使他们相交，也就是二的倍数，当然也可能剩下那就继续添上。

------------

这样的话，我们就发现了一个神奇的规律，当有两条线相交的时候，分割的个数为二的平方，当有两对相交线的时候，被分割为三的平方，当有三对相交线的时候，被分割为四的平方，而每少一条线呢，就是少了它是往上取平方数减一。当然，假如少两条线的话，就回到前一个平方了。也就是说，只要它是二的倍数，那它肯定和平方数有关系，而这个关系呢，就是，这个数是第几次减一的平方。也可以说成减去它的二分之一减一的差的平方。而假如是奇数的话呢，就往下减一，的那个数的二分之一减一的差，再加上他最多被切成的块数这样题目的结果和代码就显而易见了。**假如您现在还没有看懂的话，请看看代码。**
# STEP2
代码：
```
#include<bits/stdc++.h>
using namespace std;
int main(void){
	int n;
	cin>>n;
	if(n%2==0){
	    cout<<(n-(n/2-1))*(n-(n/2-1));
	}else{
		cout<<((n-1)-((n-1)/2-1))*((n-1)-((n-1)/2-1))+((n-1)-((n-1)/2-1));
	}
	return 0;
}
```
这个方法可能相对其他方法来说比较晦涩难懂一些，但是确实是比较基础的。
