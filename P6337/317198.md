$\boxed{\text{思路区}}$

首先，我们要明确一点：

$\text{横着切的步数要和竖着切的尽可能相似。}$

明确了这一点，我们又推出来了下一步：

$\text{横着切的步数和竖着切的要差0或1。}$

毕竟就两个对边，要么奇数要么偶数嘛。

还有一个要注意的点：

$\text{总的块数，是 (横着的边切的刀数 + 1) * (竖着的边切的刀数 + 1)}$

这个就没什么好证的了，自己画个图：

|  |  |  |
| -----------: | -----------: | -----------: |
|  |  |  |
|  |  |  |

切了两刀横，两刀竖，是 $(2 + 1) * (2 + 1)$

然后，分两种情况讨论：

1. 如果 $n$ 是偶数

	横着切 $n / 2$ 刀
    
    竖着切 $n / 2$ 刀
2. 如果 $n$ 是奇数

	横着切 $n / 2 + 1$ 刀
    
    竖着切 $n / 2$ 刀
