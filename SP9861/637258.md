# 题意如下

我们需要在这亚得里亚海沿岸的 $N$ 家酒店中，找到一串价值总和尽可能大的酒店，但是不能大于 $M$ ，因此呢，我们找的就是加起来尽量接近 $M$ 的一段数。

# 思路

我们用前缀和，轻松求出部分和，再搭配双指针，枚举 $i$ 和 $j$，如果 $a_i$ 到 $a_j$ 的所有数和大于 $M$ ，再缩短区间，继续枚举。

# 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,m,a[1000005],zd;//定义变量
int main(){
	cin>>n>>m;//输入不用多说
	for(int i=1;i<=n;i++){
		cin>>a[i];//输入不用多说
		a[i]+=a[i-1];//a数组进行前缀和
	}
	for(int i=0,j=2;j<=n;j++){//尺取，双指针循环找答案
		if(a[j]-a[i]<=m)
        		zd=max(a[j]-a[i],zd);//如果他小于M欧元的彩票，就进行打擂台，找最大值
		else i++,j--;//缩短区间
	}
	cout<<zd;
	return 0;
}
```
