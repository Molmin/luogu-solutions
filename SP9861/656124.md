## **题目意思：**
有 $ n $ 个数字，要你求出一个连续子序列，使其和要小于 $ m $ 且最大，最后输出最大值。


------------

## **分析：**
由于是要你求出连续一段所以脑海里会有两种办法：

1.我会暴力

两层循环枚举长度和起点，但是复杂度 $ O\left(n^2\right) $ 显然是没法接受的。

2.尺取法

因为题目中的“连续”一次，会自然而然想到尺取。

用双指针。右端点从 $ 1 $ 枚举到 $ n $，让总和加上当前酒店的价值。

如果总和大于 $ m $，总和减去左端点的酒店价值，让左端点向右移。

最后求出最大的总价值。~~十分简单~~

------------


## **AC代码：**
```c
#include<bits/stdc++.h>
using namespace std;
int n,m,s,zd=-1,a[1000010];
int main(){
    cin>>n>>m;
    for(int i=1;i<=n;i++)
        cin>>a[i];
	int l=1,r=1;
    while(r<=m)
    {
        s+=a[r++];
        while(s>m)s-=a[l++];
        zd=max(zd,s);
    }
    cout<<zd;
    return 0;
}
```
