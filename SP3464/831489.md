- 这道题其实并不用过多的复杂算法去解决，可以套公式。
## 一，了解公式计算
- 本题是为了计算星期，所以我们可以使用**蔡勒公式**或者**基姆拉尔森公式**，本题解只说明蔡勒公式的写法。
- 蔡勒公式：
$$
w=(d+2\times m+\frac{3 \times (m+1)}{5}+y +\frac{y}{4} - \frac{y}{100} + \frac{y}{400} + 1)\bmod 7
$$
- 其中，$w$ 表示星期，$d$ 表示具体的天，$m$ 表示月份，$y$ 表示年份,$\mod 7$ 是为了求出具体的天。
- 特别的，在蔡勒公式中 $0 \sim 6$ 分别表示星期日到星期六 (注意，星期日在本公式中并不是 $7$ )，而且，蔡勒公式会把 $n+1$ 年的第 $1$ 月和 $2$ 月看成第 $n$ 年的第 $13$ 月和 $14$ 月，千万记得特判。
## 二，源代码及解析
- 对于直到输入 `EOF` 我们可以使用 `while` 语句来解决。
- 因此得出如下的代码：
```cpp
#include<bits/stdc++.h>//万能头文件
using namespace std;
int y,cnt;// y 表示输入的年份，cnt 表示星期六的计数
int Ziller(int year,int month,int day){
    //定义一个新函数用来套用蔡勒公式
    if(month==2&&((year%4!=0||year%100==0)&&year%400!=0)) return 0;//判断闰年，只有闰年才可以计算二月，平年不可以
    if(month==1||month==2) {
        //蔡勒公式中将次年一月和二月看成这一年的十三月和十四月
         month+=12;
        year--;
    }
    return (day+2*month+3*(month+1)/5+year+year/4-year/100+year/400+1)%7;//蔡勒公式
}
int main(){
    while(cin>>y){//输入 EOF 结束，每个年份循环计算一次
        cnt=0;//重置计数
        for(int i=1;i<=12;i++){//遍历月份
            if(Ziller(y,i,29)==6) cnt++;//套蔡勒公式的函数，如果算出来是星期六，那么计数++
        }
        cout<<cnt<<endl;//每次记完输出，然后循环开头再重置
    }
    return 0;
}
```
## 三，题目的特殊要求
- **本题要求代码最多 $256$ 字节**
- 因此我们需要对代码进行压行
- [代码压行机](https://mivik.gitee.io/compress)
- 当然，我们如果直接拿上面的 `C++` 代码压行后交上去肯定还是超过了限定范围，跟 `C++` 有一个很相似的语言，就是 `C` 语言，`C` 语言是可以省略 `int` 这个变量类型，所以，我们可以用 `C` 语言来进行压行处理。
- 压行后的代码：
```c
#include<stdio.h>
y,c;Z(y,m,d){if(m==2&&((y%4||y%100==0)&&y%400!=0))return 0;if(m==1||m==2){m+=12,y--;}return(d+2*m+3*(m+1)/5+y+y/4-y/100+y/400+1)%7;}int main(){while(scanf("%d",&y)!=EOF){c=0;for(int i=1;i<=12;i++){if(Z(y,i,29)==6)c++;}printf("%d\n",c);}}
```
