首先我们注意一点，本题代码长度必须 $\leq 256$，因此一些毒瘤方法就被卡掉了。  
那么，我们如何快速地求出几年几月几日是星期几呢？

# 蔡勒公式
$W = y + [y / 4] + [c / 4] -2c + [13(m + 1) / 5] + d - 1$,  
其中 $W$ 是星期数，对 $7$ 取模，若整除则是星期天。(注意对负数的取模。)  
$c$ 是世纪数 $-1,$ 即年份前两位。  
$y$ 是年份后两位。  
$m$ 是月份，注意只能是 $3$ 到 $14,$ 即 $2000$ 年 $1$ 月 $29$ 日可看作是 $1999$ 年 $13$ 月 $29$ 日。  
$d$ 是日期。

那么，我们就有了一个笼统的想法，可以把这个公式写成一个函数。
```
int cella(int y, int m, int d) {
	int a = y % 100, b = y / 100;
	if ((a + a / 4 + c / 4 - 2 * c + 13 * (m + 1) / 5 + d + 90) % 7 == 6){ //注意对负数的取模，这里把-1替换成了+90，就是防止出现负数。
		return 1;
	}
	else{
		return 0;
	}
}
```
