### 题目大意

给定三个正整数$x,y,z$，求三个正整数$a,b,c$，使得$x=max(a,b),y=max(b,c),z=max(c,a)$。

**本题有多组数据**。

如果存在这样的解，输出两行：

- 第一行一个字符串`YES`；
- 第二行三个用空格隔开的整数$a,b,c$，这三个整数以任意顺序输出均可。如果有多组解，输出任意一组。

如果不存在这样的解，仅输出一行一个字符串`NO`。

### 思路

我们不妨设$a>=b>=c$来便于分析。

那么我们可以把$x,y,z$表示出来：

$x=a,y=b,z=a$。

有关系：

$x=z>=y$

这个时候就发现：

如果存在这样的解，$x,y,z$必须满足**其中有两个数相等，且这两个数都不小于第三个**。

这样利用排序能够快速解决这道题。看代码吧！

```cpp
#include <bits/stdc++.h>
using namespace std;
int main()
{
    int a[4],t;
    cin>>t; //t组数据
    while (t--)
    {
        cin>>a[1]>>a[2]>>a[3]; //输入三个数（相当于x,y,z）
        sort(a+1,a+4); //对其排序，保证更大的两个数在后面，这样只要写一个if就可以了
        if (a[2]==a[3]) cout<<"YES"<<endl<<a[2]<<" "<<a[1]<<" "<<a[1]<<endl; //较大的两个数相等，依次按要求输出
        else cout<<"NO"<<endl; //不是上面这种情况就无法找到
    }
    return 0;
}
```
注：在YES输出的那一行前面必须输出$a[2]$或$a[3]$，以及$a[1]$；而第三个数原则上只要不大于$a[1]$都可以，因为最小数不会影响最大值。
