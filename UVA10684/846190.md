[传送门](https://www.luogu.com.cn/problem/UVA10684)


------------


翻译

- 某某某投注是一笔钱，要么赢钱了（这被记录为正数），要么输钱（这被记录为负数）。

输入

- 集由一个正数 $N$ 组成，后跟 $N$ 个整数。每个投注都是一个大于 $0$ 且小于 $1000$ 的整数。**输入以 $N = 0$ 终止。**

输出

1. 如果有赢钱，输出**连赢**的最大值（赢一次也算）。

2. 如果全是负数，输出  ```Losing streak.```

## 思路
~~看到最大值，肯定想到打擂台。~~

 - 先判断赢钱了吗
 1. 如果赢钱了，就相加以前也赢过的钱，然后打擂台。
 2. 如果没赢，输的次数加一，然后累计连赢的钱清空。 

- 输出

  1.如果输的次数等于总次数，那么，说明他没赢过，输出```Losing streak.```
  
  2.如果赢过，输出最大值。
	 

# AC Code
```cpp
#include <bits/stdc++.h>
using namespace std;
int N,a[10000],b=0,b2,c;
int main(){
	while(cin>>N){ 
		if(!N) break; //结束标志
		for(int i=1;i<=N;i++){
			cin>>a[i]; //输入
			if(a[i]>=0){ //如果赢钱了
				b=b+a[i]; //赢的钱数加以前赢的钱数
				b2=max(b2,b); //判断最大值（钱数）
			}
			else{ //如果输钱了
				c++; //输的次数加一
				b=0; //以前连赢的钱一起清空
			}
		}
		if(c==N) cout<<"Losing streak."; //如果没赢过，输出
		else cout<<"The maximum winning streak is "<<b2<<"."; //不然输出最大值
		c=0; //输的次数清空
	}
	return 0; 
}

```
