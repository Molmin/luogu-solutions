这道题通过观察样例可以得知：输入 $4$ 个不同的数，当一个数出现的次数大于 $2$ 时，就不符合要求。

正确性证明：

输入的每一行有编号不同的 $2$ 个点，这两个点之间有一条**双向**道路，形成一个图。如果**每条路只通过一次就可以到达所有点**，就满足题目要求。为了方便理解，可以记录每一个编号的点的数量。

一共输入 $6$ 个点，所以至少有 $2$ 个点的编号出现 $2$ 次。

1. 当每一个点的编号**最多**出现 $2$ 次时（样例一)，形成的图(箭头表示行走路径）：
    
    ![](https://cdn.luogu.com.cn/upload/image_hosting/u58zivso.png)
    
  这种情况满足要求，路径为 $4-2-1-3$。
 
 . 当一个点的编号出现次数**大于** $2$ 次时（样例二），形成的图：
 
 ![](https://cdn.luogu.com.cn/upload/image_hosting/kg75f0eo.png)
 
    
    
     
    
  

   第二种情况可以发现，无论怎样遍历都至少有一条路会被重复走，因为其他点都需要通过编号出现次数最大的点来连通。所以不满足题目要求。
   
结论：只有当每个点的编号的出现次数**不大于** $2$ 时，才能保证每个点都能遍历到且每条路只通过一次。

可以用一个桶来记录这些数字的出现次数。一个数出现一次，这个数对应的桶就加 $1$。最后循环遍历这些桶来确定每个数字出现的次数是否大于 $2$。

```cpp
#include <bits/stdc++.h>
using namespace std;
int t[10];
int main(){
    int a;
    for(int i=1;i<=6;i++){
       cin >> a;
       t[a]++;
    }
    for(int i=1;i<=4;i++){
	   if(t[i]>2){puts("$");return 0;} 
    }
    puts("YES");
    return 0;
}
```
