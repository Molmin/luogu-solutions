## 题意
通过 $n+1$ 个长椅，长椅可以补充能量，每经过其中一条道路都要损失能量，初始有一个能量值，求最少还需要补充多少能量。
# 题目分析
显而易见，两条长椅间的距离即为所消耗的能量，我们不必去想每次需要补充多少能量，因为能量可以无限大，并且可以一直用，所以我们大可以只在开头一次补充完所有的能量，这样，需要补充的能量 $sum$ 就是**每条路所需能量和（即边权和）再减去初始能量 $m$**，即：
$$sum=\displaystyle\sum_{i=1}^nx_i-m$$
另外，如果所需总能量小于初始能量，那么显然不需要补充，输出 $0$ 即可。

#### 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int n, m, sum, t;// sum为所需总能量，m为初始能量 
int main() {
	cin >> t;
	while (t--) {
		sum = 0;/*每组数据 sum 一定注意要清零！！！*/ 
    	cin >> n >> m;
    	for (int i = 0, a; i < n; i++)
    		cin >> a, sum += a;//输入 求和 
    	if (sum >= m)//写成 > 也可以，毕竟都是 0  
    		cout << sum - m << "\n";
    	else
    		cout << 0 << "\n";
    }
    return 0;
}
```
