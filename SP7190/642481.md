[题目传送门](https://www.luogu.com.cn/problem/SP7190)

这就是最小公倍数模板。

那怎么求最小公倍数呢？

我们知道，最小公倍数 = 两数的乘积 / 最大公因数。

那最大公因数怎么做呢？

辗转相除法就行。

那怎样写呢？

```
int gcd(int a,int b){
    if(b==0) return a;
   	 return gcd(b,a%b);
}
```

这样就可以了。

但是这一道题要算很多个数的该怎么办呢？

我们可以两两配对算，最后会剩下一个数，就是它们的最小公倍数了。
