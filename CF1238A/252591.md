这道题代码难度不难，考的是思维。

很多大佬都提到了“**唯一分解定理**”，但如果不懂这个我们也可以解决这道题。

首先我们写一个超级暴力的写法，就是枚举每个数（注：因为可能结果太多，所以找1000内的），每次找到比这个数小的质数然后有没有可能成立。

因为1本生就不可以，所以不考虑。

结果什么也没输出！

仔细检查代码，发现没问题，说明只要两数之差不是1就可以了。

代码如下：
```
#include <bits/stdc++.h>
using namespace std; 
int main() 
{
	long long n,m,t=2,i,j,k,x,y;
    cin>>n;
    for(i=1;i<=n;i++)
    {
    	cin>>x>>y;
    	if(x-y!=1)//如果不是1，可以成立
    	cout<<"YES"<<endl;
    	else//不可以成立
    	cout<<"NO"<<endl;
	}
    return 0;
}
```
