# 题意
输入一个整数 $n$，输出一个整数 $x$，使得 $x$ 能被 $3$ 整除且 $n-x$ 能被 $5$ 整除，并且要让 $5$ 最大。如果没有这样的数，输出 $-1$。本题有多组数据(借一楼大佬翻译)。

# 思路
对于每一个 $n$：

1、若 $3\mid n$，$x$ 需要从 $n-3$ 开始枚举，因为当 $x=n$ 时 $5\mid n-x$，但是这不符合题意。

2、若 $3\nmid n$，$x$ 只需从离 $n$ 最近的整除 $3$ 的数开始枚举，即：$n-(n\bmod 3)$。

找到开始枚举的数后，接下来就是枚举过程了，由于迭代器是从
第一个 $<n$ 的 $3$ 的倍数开始枚举，所以我们只需把它 $-3$，$x$ 就必定整除 $3$，只需判断 $5\mid n-x$ 即可。
# code
```c
#include<bits/stdc++.h>
using namespace std;
int main(){
    int t,n,ans;
    scanf("%d",&t);
    for(int i=1;i<=t;i++){
        ans=-1; //每次将ans赋初值很重要，要是没有找到答案ans没有更新过仍然是-1。
        scanf("%d",&n);
        printf("Case %d: ",i);
        for(j=n%3?n-n%3:n-3;j>=3;j-=3){ //同思路
            if((n-j)%5==0){ //因为j%3==0，所以只需判断是否整除5。 
                ans=j; 
                break; //因为我们是倒叙查找的，找到的第一个答案就是最大值，直接退出
            }
        }
        printf("%d\n",ans);
    }
    return 0;
}
```
