[题目传送门](https://www.luogu.com.cn/problem/P7742)

简单构造题

~~根据生活经验~~，换频道的时候都是先找到频道再将频道置顶。

转换到这题中就是先将光标移到目标频道再一点点往上交换，于是得出主要代码：

```
for(int i=1;i<=n;i++){
	if(str[i]!="BLJTV1"/*或BLJTV2，两者代码几乎一样*/){
		cout<<1;//光标下移
		continue;
	}else{//找到了
		for(int j=i;j>1;j--){//从当前位置往上移一直到正数第一个位置
		//注意 BLJTV2 的时候只有这里不一样，变成 for(int j=i;j>2;j--) 因为 BLJTV2 是调到正数第二个位置
			swap(str[j-1],str[j]);//交换
			cout<<4;//往上交换
		}
		break;//找到了就不用再找了
	}
}
```
