## 题意
要求一些操作，将 `BLJTV1` 放到第一个位置，将 `BLJTV2` 放到第二个位置，输出任意一种可行方案的操作序号。
## 思路
我们先看一下有哪些操作：

操作 $1$：将箭头向下移动一个位置（从频道 $i$ 到频道 $i+1$）。

操作 $2$：将箭头向上移动一个位置（从频道 $i$ 到频道 $i-1$）。

操作 $3$：将箭头和选定的频道向下移动一个位置（频道 $i$ 和频道 $i+1$ 交换位置，箭头在频道 $i+1$ 的位置结束）。

操作 $4$：将箭头和选定的通道向上移动一个位置（频道 $i$ 和频道 $i-1$ 交换位置，箭头在频道 $i-1$ 的位置结束）。

所以，怎么写程序呢？

我们很容易想出一种程序好写的方法：**先将箭头指向要换的频道，然后向上换位，直到指定的位置。**

然后直接模拟操作过程即可。

~~再然后，你就会发现，因为题目的特殊性，按照此思路，在此题中，只需要操作 $1$ 和操作 $4$。~~
## 代码
因为两次操作类似，所以写成一个函数更简便。
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,i;
string a[105];
void put(string s,int w){
	while(a[i]!=s)i++,putchar('1');//将箭头指向要换的频道
	while(i>w)swap(a[i],a[i-1]),i--,putchar('4');//向上换位，直到指定的位置
}
int main(){
    scanf("%d",&n);
	for(i=1;i<=n;i++){
		cin>>a[i];
	}
	i=1;//初始箭头指向位置1 
    put("BLJTV1",1);//将BLJTV1放到第一个位置 
    put("BLJTV2",2);//将BLJTV2放到第二个位置
	return 0;
}
```
