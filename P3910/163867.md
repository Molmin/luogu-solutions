开始写：

a是第一个数，b是最后一个数，len为长度

（a+b）* len / 2 = m     (公式）

然鹅介里有两个变量，就意味着要两重循环

所以开始启动我们的 数 学 模 式

(咳咳，敲黑板）

```
因为len=b-a+1
所以b=len+a-1
那么式子就变成了（a+a+len-1)*len/2=m
然后移项 2a+len-1=2m/len
2a=2m/len-len+1
a=(2m/len-len+1)/2
```
那么，我们只要枚举len就好啦ヾ(◍°∇°◍)ﾉﾞ

代码：

```
#include<bits/stdc++.h>
using namespace std;
int m,n;
int main()
{
	cin>>n>>m;
	for(int i=n;i>=1;i--)
	{
    	        //当式子中的数不是整数或超出范围，就continue掉
		if((2*m)%i!=0)continue;
		if(((2*m/i)-i+1)%2!=0)continue;//不是整数
        
		int a=((2*m/i)-i+1)/2;
        
		if(a<1)continue;
		if(a+i-1>n)continue;//超出范围
		cout<<"["<<a<<","<<a+i-1<<"]"<<endl;
	}	
	return 0;	
}
```
