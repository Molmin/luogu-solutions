### 主要思路：
1.判断是否有多余的
2.计算
### 要点：
1.虽然本题是入门级，但还是有坑的。如果输入是样例2这种的（比如7），你会发现需要一个特判，也就是判定最后一组是否满一组（貌似有点绕口，具体自己体会吧），这就需要一个while循环和一个if判定。
2.在做while的时候要注意，有些不能用原变量来做，需要一些寄存器（直白点就是需要给一些变量在其他地方再存一个）
### 剪枝：
既然已经有一个while了，那就可以顺便把下面for循环的范围也确定了。
### 代码如下：
```
#include<bits/stdc++.h>
using namespace std;
int k,i,s,a,b,c;
int main()
{
    cin>>k;
    i=1;a=k;//while一定要记得定义初值
    while(a-i>=0)
    {
        a-=i;b=i;i++;//a是寄存器，用于特判，下面的c也是;b是顺便做的for的取值范围
    }
    for(i=1;i<=b;i++) s+=i*i;
    c=b;
    if(a!=0)//如果不满一组，再加相应的值
    {
        b=(1+b)*b/2;
        s=s+(k-b)*(c+1);
    }
    cout<<s;
    return 0;
}
```
