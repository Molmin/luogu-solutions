这道题，我默默的说循环都不要

先普及下公式1^2+2^2+3^2+...+n^2=n(n+1)(2n+1)/6

1+2+3+..+n=n(n+1)/2

先求一下n在每天拿多少钱的阶段设这个数的d

d(d+1)/2=n -> d^2+d-2n=0 -> d=(sqrt(1+8\*n)-1)/2   [(-sqrt(1+8\*n)-1)/2   直接舍掉]

然后 显然这个d(d+1)/2<=n 不要问我为什么，精度问题

所以求1^2+2^2+3^2+...+d^2就可以了 然后我们就愉快 的用公式吧 d(d+1)(2d+1)/6

当然还有一部分钱就是n-d(d+1)/2这些天里每天拿的钱是d+1  所以

ans=d\*(d+1)\*(2\*d+1)/6+(n-d\*(d+1)/2)\*(d+1)

完结撒花


```cpp
#include<cstdio>
#include<cmath>
int n,d,ans;
int main()
{
    scanf("%d",&n);
    d=(sqrt(8*n+1)-1)/2;
    ans=d*(d+1)*(2*d+1)/6+(n-d*(d+1)/2)*(d+1);
    printf("%d",ans);
return 0;
}
```
