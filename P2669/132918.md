# CORE IDEA:模拟+三角形数
#### 三角形数
不懂的人看这里：[三角形数](https://baike.baidu.com/item/三角形数/5836794?fr=aladdin)

其实啊，三角形数就是1，2，3……各数的阶加之和。而题目中工资变更日期正好是$1+2+3$……

所以用一个数组来储放工资的变更日期就好啦。

三角形数公式：$T_n=\frac{n(n+{1})}{2}$

#### 代码实现
拥有了三角形数加成，我们完全可以一天一天地模拟，每次循环判断一下有没有越界，再变更工资和下一次变更时间就好了。

可能上面的解释有点绕，大家看代码理解吧。

打表机：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	freopen("generator.out","w",stdout);
	for (int i=1;i<=1000;++i)
	  cout<<i*(i+1)/2<<',';
	return 0;
}//这个会打到第1000项（500500），其实不用打那么多，但是我太懒了……
```
打完拷贝时记得去掉最后一个逗号欧。

AC代码：
```
#include<bits/stdc++.h>
using namespace std;
int k,p=1,m=1;
long long ans;
const int a[]={};//防抄袭+太长，不予显示。
int main()
{
	cin>>k;
	for (int i=1;i<=k;++i)
	{
	  if (i>a[p])//判断越界
	  {
	  	++p;//进入新的工资周期
	  	m=p;//更新工资
	  }
	  ans+=m;//发工资		
	}
	cout<<ans;
	return 0;
}
```
