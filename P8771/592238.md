[传送门](https://www.luogu.com.cn/problem/P8771)     
本题其实就是解答两个问题。     
第一题很简单，我们手动模拟，得出结果为 $1478$。   
第二题我们手动枚举：    
一月的月份区域是 ```01```，所以只要天数的十位是 $2$ 就组成了顺子，$1$ 月的天数十位是 $2$ 的日期为 $20~29$，共 $10$ 个。   
二月的月份区域是 ```02```，后面只能接上 ```34```，二月是没有 $34$ 号的，所以二月不行。
三月到九月都不行，同二月。    
十月的月份区域是 ```10```，可以接 ```12```。   
十一月的月份区域是 ```11```，可以接 ```23```。   
十二月的月份区域是 ```12```，后面天数的十位需要是 $3$，$12$ 月有 $31$ 天，所以有 $2$ 种方法。    
结果就是 $10+1+1+2=14$ 种。     
CODE：   
```cpp
#include <iostream>
using namespace std;
char a;
int main(void){
	cin>>a;
	if(a=='A'){
		cout<<"1478";
	}else{
		cout<<"14";
	}
}
```
