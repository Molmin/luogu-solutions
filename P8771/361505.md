蓝桥杯的题目还是简单一点。

## 问题A

将 $(2022)_9$ 转为十进制。

直接手算即可。

具体方法：

$2\times9^0+2\times9^1+2\times9^3$

$= 2 + 18 + 1458$

$=1478$。

## 问题B

将 $2022$ 年中的年、月、日按照 ``yyyymmdd`` 排序，问有几个日期中有三位是 $3$ 个连续的。

**我们认为所谓顺子是单调递增的。**

考虑形成顺子的结构，无论三位中有三位或者两两位是属于年的部分都不可能。

因此考虑 ``ymm`` 的情况，``y`` 是 $2$，又没有 $34$ 月，舍去。

一月的 ``mm`` 为 $01$，所以 ``d`` 是 $2$ 就行了，也就是 $20$ 号到 $29$ 号。

二月的 ``mm`` 为 $02$，但是没有 $34$ 号。所以不存在。同理从二月到九月都不存在。

十月的 ``mm`` 为 $10$，日是 $12$ 就行。共一个。

十一月的 ``mm`` 为 $11$，日是 $23$ 就行。共一个。

十二月的 ``mm`` 为 $12$，只要是三十几号就行。就是$20221230$ 和 $20221231$，共两个。

因为按照月份来算，所以不存在重复，无需去重。

因此加起来一共有 $10 + 1 + 1 + 2 = 14$ 个。

# 代码

按照题目给定的答题模板就行，把答案填进去就是：

```cpp
#include<iostream>
using namespace std;
int main() {
    string ans [] = {
        "1478",
        "14",
    };
    char T;
    cin >> T;
    cout << ans[T - 'A'] << endl;
    return 0;
}
```
