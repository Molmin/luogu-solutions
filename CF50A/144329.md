## 这真是一道水题啊!!
刚刚看到的时候,我还以为是一道铺砖问题,像[P1212](https://www.luogu.org/problem/P1212)和[P1003](https://www.luogu.org/problem/P1003)一样.

### 结果......

这明明就是一道红题,居然假装成了橙题.

总的翻译过来就是
```
a*b/2//就是这么简单
```
所以我们可以得到下面的代码
```
#include<bits/stdc++.h>//万能头文件
using namespace std;
int main(){//主函数
   int a,b;//定义
   cin>>a>>b;//输入
   cout<<a*b/2<<endl;//按上面的输出
   return 0;//好习惯
}
```
