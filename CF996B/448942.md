# [题目传送门](https://www.luogu.com.cn/problem/CF996B)

## 题目大意：

一个序列的数在你没向后走一位是所有数组的大小减 $1$，问你什么时候第一次走到的这个数组的数是 $0$。注：

当你走向 $a_2$ 时数组大小才开始减。

如果你走到这个下标时数组恰好减为 $0$，那么游戏也将结束。

当你到最后一格向后走时，你会到达第一格。

## 题目分析：

如果你当前这个数组的数小于等于 $i-1$ 的话，那么你走到这个格时数组已经等于 $0$ 或者恰好等于 $0$，游戏结束，输出 $i$，并且将这个数组的数直接减去 $n$，因为等下次再到此下标时数的大小减了 $n$。如此循环往复，最终可以找到答案。

## 代码：

```cpp
#include <iostream>
#include <cstdio>

using namespace std;

int a[100005];

int main(){
	int n;
	cin >> n;
	for(int i = 1;i <= n;i++){
		cin >> a[i];
	}
	while(1){
		for(int i = 1;i <= n;i++){
			if(a[i] <= i - 1){
				cout << i;
				return 0;
			}
			a[i] -= n;
		}
	}
}
```
