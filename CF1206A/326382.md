本题如果打大暴力是过不掉的，不妨来看另一种算法。

首先，输入 $n$（表示第一个集合）

接下来，输入的 $a$ 不需要**数组**，因为在每一次计算一个最大值 $Maxa$ 即可。

其次，输入 $m$（表示第一个集合）

接下来，输入的 $b$ 不需要**数组**，因为在每一次计算一个最大值 $Maxb$ 即可。

**综上，即可得出 _一种情况_ ，输出 $Maxa,Maxb$即可。**

推理&证明如下：

不妨设两个最大值为 $a$ 和 $b$，即可得出以下情况：

当 $a$ 和 $b$ 相加时（$a + b$），此时两个集合（$[n],[m]$）没有任何两个比最大值相加**更大**的数。

---

一组简单数据：

集合 $[n]$：

$10,20,30$

其中最大值为 $30$。

集合 $[m]$:

$10,25,50$

其中最大值为 $50$。

综上，当 $30+50$ 时，无数可以超过 $80$。

得证。

---

综上所述，即可得出结论（代码）：

```cpp
#include<bits/stdc++.h>//万能头文件
using namespace std;
int main(){
    int tmpa,tmpb,n,m,Maxa=INT_MIN,Maxb=INT_MIN;//定义，注意不用定义数组
    cin>>n;//输入
    for(int i=1;i<=n;i++){
        cin>>tmpa;//输入集合中的数
        Maxa=max(Maxa,tmpa);//找出最大值
    }
    cin>>m;//同上
    for(int i=1;i<=m;i++){
        cin>>tmpb;
        Maxb=max(Maxb,tmpb);
    }
    cout<<Maxa<<' '<<Maxb;//输出最大值即可
    return 0;    //华丽结束
}
```
