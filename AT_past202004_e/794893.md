题目思路：
-
根据题意模拟：这道题让我们将所有的 $a_i$ 全部变成对应的 $i$，所以我们可以开个循环，创建一个变量，让它等于 $a_i$，那怎样才能算出要按几次按钮呢？我们可以弄个内循环，跟题目所说的一样，循环到 $b$ 等于 $i$ 时就可以停下，翻译讲的很明白，要将此时的变量 $b$ 等于 $a_b$，由于每变一次，就等于我们按了一次按钮，所以让 $ans$ 加 $1$，这些操作完成后，就可以将此时的 $ans$ 输出了，记得空格（之所以让 $ans$ 的初始值为 $1$，是因为题目中说要将答案加 $1$）。

有 $1$ 个坑：由于只是**假设**，我们不能直接动用 $a_i$ 的值。

题目代码：
-
```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;
signed main()
{
	int n , b , ans = 1;
	int a[10000];
	cin >> n;
	for(int i = 1;i <= n;i++) cin >> a[i];
	for(int i = 1;i <= n;i++)
	{
		b = a[i];
		while(b != i)//循环到b等于i
		{
			b = a[b];
			ans++;//增加按按钮的次数
		}
		cout << ans << " ";//输出按按钮的次数
		ans = 1;//记得将ans重新赋值为1
	} 
	return 0;
}
```
