### 这题是纯模拟的字符串题
照着题意写就行了。  
题目要求根据输入的命令来计算出流量大小（即消息长度 * 人数）  
那么我们先判断命令的三种种类：  

	1.判断命令开头是否有'+';  
	2.判断命令开头是否有'-';  
	3.循环查找命令中是否有‘:'  
于是就可以得出各个消息长度了。  
当前总人数可以再定义一个变量sum来表示 

------------

## AC代码
```cpp
#include <bits/stdc++.h>
using namespace std;
char a[10001];
int ans = 0,sum = 0;
int main(){
	while(gets(a)){//用gets函数输入字符串 ，当没有东西可输入时，gets会返回0，跳出循环 
		if(a[0] == '+'){//判断命令开头是否有'+'
			sum++;//总人数 + 1 
		}else if(a[0] == '-'){//判断命令开头是否有'-'
			sum--;//总人数 - 1 
		}else{
			for(int i = 0;i < strlen(a);i++){
				if(a[i] == ':'){
					ans += (strlen(a) - i - 1) * sum;//计算答案 
				}
			}
		}
	}
	cout<<ans;
	return 0;
}

```
