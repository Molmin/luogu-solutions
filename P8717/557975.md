一道非常**经典**的题，[传送门](https://www.luogu.com.cn/problem/P8717)。

## Solution

### 题目简意

- 输入 $n$ 个数。
- 输出这些数的最大值、最小值、平均数。
- 注意，最大值与最小值都是整数，而平均数是**实数**，需要保留 $2$ 位小数。

### 核心分析过程

首先，是求最大、最小数。我们首先来看看最大数的定义是“一个数列中最大的数”（最小数亦然），那么想要求出最大数，我们只需将当前数列中的每个数与当前的最大数进行比较：

- 若当前数更大，就说明目前的“最大数”不是数列中最大的，我们就需要将最大数更新为目前遍历的数。
- 若最大数更大，就说明目前的“最大数”没有问题（暂时），不需要进行任何操作。

最小数亦然。

所以，我们可以写出下面代码（`maxn` 表示最大数）：
```cpp
if (maxn < a[i]) {
    maxn = a[i];
}
```

但是，我们还可以简化这行代码，不需要 `if`，怎么做呢？

我们知道，STL库中有 $2$ 个函数：`max()` 和 `min()`。他们的作用分别是：返回 $2$ 个参数的最大数、最小数。那么，我们如何利用这 $2$ 个函求最大最小值？

答案很简单，只要一行代码：
```cpp
maxn = max(maxn, a[i]);
```

看懂了吗？这段话翻译过来就是：“更新最大值 = 求最大(当前最大值, 遍历到的数);”。

好了，现在解决了最大数和最小数，只剩一个平均数了。小学知识：$平均数=总和\div 个数$，所以，我们要求出平均数，必须知道 $2$ 个值：数列总和与数列中数的个数，个数就是 $n$，总和呢？

很简单！总和就是把数列中的每个数加起来，所以，我们只需遍历每个数，并且加到一个变量中，最后再除以 $n$。

最后，我们需要注意一些细节：
- 首先，是这些变量的初值，`sum`（也就是上文提到的“总和”）的初值不用说，肯定是0。但是 `maxn` 与 `minn` 就不一样了，他们的初值是 $2$ 个极端—— $-1$ 与 $2\times 10^9$，因为我们不知道在做第一次比较时的数是多少，只好让第一次变更的成功率为 $100\%$。
- 第二，就是题目中的“第三行包含一个**实数**，四舍五入保留正好**两位小数**”。怎么保证保留 $2$ 位小数呢？我们要了解这段代码：`cout << fixed << setprecision(2) << n << endl;`，这段代码的意思大致就是：将一个 `double` 类型的变量 `n` 四舍五入致 $2$ 为小数，再输出。
- 第三，我们没必要再开一个循环遍历每一个数，只需在输入时操作就可以了。

## Code

**AC代码，仅供参考，切勿抄袭！**

```cpp
# include <iostream>
# include <cmath>
# include <cstring>
# include <algorithm>
# include <bits/stdc++.h>
# define ll long long
using namespace std;

int a[10005], n, minn = 2e9, maxn = -1, sum = 0; // 注意初值

int main() {
    cin >> n;
    for(int i = 1; i <= n; ++i) {
        cin >> a[i];
        minn = min(minn, a[i]); // 最小值
        maxn = max(maxn, a[i]); // 最大值
        sum += a[i]; // 总和
    }
    cout << maxn << endl;
    cout << minn << endl;
    cout << fixed << setprecision(2) << 1.0 * sum / n << endl; // 注意输出格式
    
    return 0;
}
```
