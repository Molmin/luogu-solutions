这个题的关键在于 条件语句的使用

当我们明白题意后 首先把1放到二维数组中第一行的正中间

然后看下面四句话

1.若(K−1)在第一行但不在最后一列，则将K填在最后一行，(K−1)所在列的右一列；

2.若(K−1)在最后一列但不在第一行，则将K填在第一列，(K−1)所在行的上一行；

3.若(K−1)在第一行最后一列，则将K填在(K−1)的正下方；

4.若(K−1)既不在第一行，也不在最后一列，如果(K−1)的右上方还未填数，则将K填在(K−1)的右上方，否则将K填在(K−1)的正下方。

这便是本题的四个条件语句 我们只需要将这四句话用 代码 “翻译”一下 就好了  （当成文言文翻译做哈哈哈

```cpp
1. if(x==1&&y!=n)xx=n;yy=y+1;
2.else if(y==n&&x!=1) yy=1;xx=x-1;
3.else if(x==1&&y==n) xx=x+1;yy=y;
4.  else if(x!=1&&y!=n){ if(!a[x-1][y+1])xx=x-1,yy=y+1;   else xx=x+1,yy=y; }       
```
