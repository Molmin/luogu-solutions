给你一个很大的球，将这个球切 $n$ 刀，让你求表面积增加多少百分之多少。

首先，我们可以在百度上查到，球的表面积为 $4\pi r^2$；根据小学五年级知识，圆的面积为 $\pi r^2$。

所以当我们切一刀的时候，表面积变成原来的 $\dfrac{5}{4}$，也就是说，切 $1$ 刀表面积增加 $\dfrac{1}{4}$。

切了 $n$ 刀，表面积就会增加 $\dfrac{1}{4}n$，即 $25\% \cdot n$。

- 当您使用 C++ 时，请注意开 `long long`。

**Code**

```python
n = int(input())
while n > 0 :
	if n == 1:
		print("0%")
	else :
		print(str(n * 25) + "%")
	n = int(input())
```
