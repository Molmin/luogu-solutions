### AT_joi2022_yo1a_a 余り (Remainder) 题解

### 题意：

输入一个整数 $x\ (1 \leq x \leq 100)$，输出 $x$ 除以 $21$ 的余数即可。

### Code:

```cpp
#include <bits/stdc++.h>

using namespace std;

int x;

int main() {
  cin >> x;
  cout << x % 21 << '\n';
  return 0;
}

```
