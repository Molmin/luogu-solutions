### 思路：
数据范围 $1\le h_i \le 10^6$，那么，答案的范围在 $1$ 到 $10^6$ 之间。 

我们很容易联想到**二分法**。

二分的左边界就是 $0$，右边界为 $h_i$ 的最大值。

枚举所有的答案，判断是否能满足条件，找最大值即可。

### 核心代码：
```
inline bool check (int x) {
	int tot = 0;
	
	for (int i = 1; i <= n; i ++) {
		if (x < a[i]) {
			tot += a[i] - x;
		}
	}
	
	return tot >= m;
}

```
