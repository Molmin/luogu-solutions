题意是说有 $n$ 个身高不同的人排队，从前面看可以看到 $p$ 个人，从后面看

可以看到 $r$ 个人，问有多少种排队方式。

考虑动归，我们另 $f(i)(a)(b)$ 表示 i 个人排队，从前可以看到 a 个，从后可以看到 b 个的方案数。

考虑转移，首先，如果从前往后放一个高的，那 $f(i)(a+1)(b) += f(i)(a)(b) $

从后往前放高的同理。

如果放一个矮的，因为开头和结尾不能插入，所以中间有 $i-2$ 个空可以插入，所以 $f(i)(a+1)(b) += f(i)(a)(b) * (i-2)$。

```cpp
#include <bits/stdc++.h> 
int f[15][15][15], n, N, P, R;
int main()
{
	f[1][1][1] = 1;
	for (int i = 2; i <= 13; i++) 
	{
		for (int a = 1; a <= i; a++) 
		{
			for (int b = 1; b <= i; b++) 
			{
				f[i][a+1][b] += f[i-1][a][b];
				f[i][a][b+1] += f[i-1][a][b];
				f[i][a][b] += f[i-1][a][b]*(i-2);
			}
		}
	}
	while (~scanf("%d",&n)) 
	{
	   while (n --) 
	   {
		   scanf("%d%d%d",&N,&P,&R);
		   printf("%d\n",f[N][P][R]);
	   }
    }
	return 0;
}
```
