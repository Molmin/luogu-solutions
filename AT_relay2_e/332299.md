暴力会超时，所以尝试一种贪心做法。

------------

假设这几张票对应了一定的分值，则可以把一张赞同票的分值看作 $100-P$，把一张反对票的分值看作 $-P$，然后设总分值为 $t$，易得当 $t$ 大于等于 $0$ 时选举通过。

先假设这 $n$ 个人全部投了反对票，那么此时的 $t$ 即为所有 $b_i$ 之和。然后，第 $i$ 个人投完他的赞同时， $t$ 就会增加 $w_i \cdot (100-P)+b_i \cdot p$ ，把这个数记作 $a_i$，表示的是第 $i$ 个人从投反对票到投赞同票时 $t$ 的变化量。

而所有 $a_i$ 都是正的，因为人数尽可能少，所以要使 $t$ 大于等于 $0$，必须要使每次加上的 $a_i$ 尽可能大。

然后，把每个 $a_i$ 从大到小排个序，再循环一下做个累加器比较就可以了。

**注意：岛国题要换行！！！！！！！！！！！！**

------------

代码：

```cpp

#include<bits/stdc++.h>
using namespace std;
int n;
long long p,t,a[100010],x,y;
int cmp(long long a,long long b)
{
	return a>b;
}
int main()
{
	cin>>n>>p;
	for(int i=1;i<=n;i++)
	{
		scanf("%lld%lld",&x,&y);
		t-=p*y;
		a[i]=x*(100-p)+y*p;
	}
	sort(a+1,a+n+1,cmp);
	for(int i=1;i<=n;i++)
	{
		t+=a[i];
		if(t>=0) {cout<<i;return 0;}
	}
	return 0;
}

```
