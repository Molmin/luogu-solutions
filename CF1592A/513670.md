## 题意

这道题的题意是，一个特工拿着 n 个武器，每个武器有一个伤害值，他要拿这些武器去攻击敌人，直到敌人死亡。但是有一个条件，他不能用一个武器连续攻击两次。求他至少要攻击多少次。

---
## 这道题用贪心算法
根据贪心原则，我们每次使用能用的、攻击力最大的武器进行攻击，能使攻击次数最小而最强的武器使用一次后就暂时不能用，所以这时就换用伤害第二大的武器。之后攻击力最大的武器就变为可用了，再使用它。如此循环。

然而暴力模拟不行，敌人的生命值 H 的范围是十的九次方，会超时。所以简单地用除法计算一下。

用 l 变量存下他用最强武器和第二强武器攻击各攻击一次的伤害，用敌人的生命值除以 l 。在计算出余数，余数为 0 说明敌人已死亡。余数大于最强武器的伤害说明他还需要在攻击两次，否则一次。
```cpp
#include<iostream>
using namespace std;
int main(){
    int t;cin>>t;
    while(t--){//处理多组输入
      int mx=0,mx2=0;
      int n,m;//n: 武器数，m: 敌人生命值
      cin>>n>>m;//读入
      int x;
      for(int i=0;i<n;i++){
        cin>>x;//读入
        if(x>mx){
           mx2=mx;mx=x;//当前武器伤害比最大值大，更新最大值，并把最大值放到第二大值
        }
			else if(x>mx2){
	   			mx2=x;//否则判断是否更新第二大值
			}
      }
      int sum=0;
      int l=mx+mx2;
      sum=(m/l)*2;//用最强、次强武器轮流攻击
      if(m%l!=0){//处理余数
          if(m%l>mx)sum+=2;//敌人剩余生命值大于最强武器伤害，需要再攻击2次
          else sum++;//否则只需再攻击一次
      }
      cout<<sum<<endl;//输出
   }
}
```
# 注意：
1. 第 12 到 16 行部分易错，最大值被替换时要把它更新为第二大值。
2. 第 17 行，更新第二大值时一定要保证该值比最大值小。
