### 题意简述：

给定一个字符串为 s，问在这个字符串中最少删去几个字符才能使字符串中不含有 “xxx”，注意，保证输入为小写。

### 方法：

就是我们得先把这个字符串从左到右遍历一遍，如果发现了‘x’，那就从当前位置 i + 1 的地方继续遍历直到字符不为‘x’时，找到连续的 x 的个数 t，然后将这串 x 变成 两个 x，即使答案增加 max(t - 2,0) 就可以了，同时也能避免负数。

所以接下来就看代码吧：

```cpp
#include <bits/stdc++.h>
using namespace std;
string s;
int n,t,ans; 
int main()
{
	cin >> n >> s;
	for (int i = 0; i < n; i++)
		{
			if (s[i] == 'x')//如果找到了x
				{
					for (int j = i + 1; j <= n; j++)
						{
							if (s[j] != s[i]) {i = j,ans += max(t - 1,0),t = 0; break;}//因为最开始的x没算上，所以只要t - 1就行了；
                            				//i = j相当于交替，从当前遍历到的非x字符继续遍历，当然，改成j加一也行。
								else t++;//更新x的个数。
						}
				}
		}
	cout << ans;//输出答案。
	return 0;//AC！
}
```
