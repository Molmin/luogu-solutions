### 题目大意：
一个公园有 $n$ 种鹅卵石，有一个人有 $2$ 个口袋，容量都为 $k$，每天他都可以装小于 $k$ 个鹅卵石到每个兜里，但是一个兜只能装一种鹅卵石，需要几天才能全部装完？

### 题目分析：
我们可以把 $2$ 个兜看成一个兜，但是这一个兜里可以装 $2$ 种，如果剩余的鹅卵石个数装不满一个兜，那也算 $1$ 天装走的量。

### 具体代码：
```cpp
#include <iostream>
using namespace std;
int n, k, a, ans;
int main(){
	cin>>n>>k;
	//n种，口袋的容量k 
	for(int i=1;i<=n;i++){ 
		cin>>a;
		//第i个鹅卵石有a个 
		ans+=a/k;
		//把能装的都装了 
		if(a%k!=0){
			//没有装完也按1天拿的量来算 
		    ans++;
		}
	}
	cout<<(ans+1)/2;
    //要除以2，因为我们是把两个都合并成了一个兜
	return 0;
}
```
