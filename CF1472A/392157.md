## 1. 题意简述

多组数据，给定纸的大小$w×h$，问有没有只使用这两个办法的一种分法，把纸张剪出 $n$ 张或更多。（办法一：如果 $w$ 是偶数，可以将该纸片对半剪开。办法二：如果 $h$ 是偶数，可以将该纸片对半剪开。）

能 输出“YES”（不输出引号），不能 输出“NO”（不输出引号）。$1≤t≤10^4,1≤w,h≤10^4,1≤n≤10^9$

## 2. 题目简析

考点：数论

这道题相对较简单，但是要思考一会儿才可以做出来。

根据题意，总个数起始为一。

如果$w,h$除以二都整除，那么总个数乘四，$w,h$除以二；

如果$w$除以二整除，那么总个数乘二，$w$除以二；

如果$h$除以二整除，那么总个数乘二，$h$除以二；

如果$w,h$除以二都不整除，那么退出循环。

最后让总个数与$n$比较即可。

## 3. 主代码 和 简单讲解

```cpp
int w,h,n,ge=1;
cin>>w>>h>>n;
while(1){//一直判断，在循环内部设置条件（退出循环）
    if(w%2==0&&h%2==0){//w,h除以二都整除
        w=w/2;
        h=h/2;
        ge=ge*2*2;
    }
    else if(w%2==0){//w除以二整除
        w=w/2;
        ge=ge*2;
    }
    else if(h%2==0){//h除以二整除
        h=h/2;
        ge=ge*2;
    }
    else{
        break;//退出循环
    }
}
if(ge>=n){
    cout<<"YES\n";
}
else{
    cout<<"NO\n";
}
```
