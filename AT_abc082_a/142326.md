这道岛国水题说了:x の小数点以下を切り上げて得られる整数を出力してください

所以它要求两个数的平均值，也就是只会/2，那么这样的话直接+1就可以向上取整了

## code:
```
#include<挑战最短代码>//我猜可以哦，故意压行的不算
#include<bits/stdc++.h>//竞赛不可以用的懒人专属表头
int main()
{
	int a,b;//定义变量不说了
	std::cin>>a>>b;//输入
	std::cout<<(a+b+1)/2<<"\n";//输出，不打using···不给打endl
}
```
本蒟蒻提醒一下：

岛国题都喜欢坑人。有时不输换行能AC，但是，它有时不输换行有爆零，坑得好多人把自己的代码查了又查，最后发现要输出换行

岛国题可以不写return 0，但是必须输出换行(^ -- ^)
