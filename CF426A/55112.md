# 题意

给定$n$个数和$s$，求其中最小的$n-1$个数加起来是否超过$s$

## 输入格式

第一行两个正整数$n$和$k$。

接下来一行$n$个整数表示这$n$个数字。

## 输出格式

如果不超过$s$，则输出```YES```，反之输出```NO```。

---

# 题解

- 方法1：

	直接输入$n$个数，然后从小到大排序，取出前$n-1$个数求和然后判断。
	
	但是时间复杂度是$O(nlogn)$的，太慢了（虽然好像$O(n^3)$也不会超时）！

- 方法2：

	把题意转化一下，变成求$n$个数的和减去他们的最大值是否超过$s$，就简单了，$O(n)$过。

---

# 代码

> 注：本代码使用的是方法2

```cpp
#include<stdio.h>
int n,s,x,mx,sum;
int main()
{
	scanf("%d%d",&n,&s);
	for(register int i=0;i<n;++i)
	{
		scanf("%d",&x);
		sum+=x; //求和
		if(x>mx) mx=x; //找最大值
	}
	if(sum-mx>s) printf("NO\n"); //判断和减去最大值是否超过s
	else printf("YES\n");
	return 0;
}
```
