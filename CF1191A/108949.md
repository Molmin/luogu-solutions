设血量为$H$，则有：

$$(4n+x)=H(0\leq x \leq 3)$$ 

且

$$x=H \mod 4$$

于是题意变成了：将$H$加上一个不超过$2$的自然数，使$H  \mod 4$尽量为$1$（$A$类），如果无法为$1$则尽量为$3$（$B$类），如果无法为$3$则尽量为$2$（$C$类），否则为$0$（$D$类）。

$H \mod 4$有$4$种余数：

- $1$。这种情况下根本不用加，它本身就是最好的$A$类。

- $2$。不加为$C$类，加$1$为$B$类，加$2$为$D$类。选择加$1$成为$B$类。

- $3$。不加为$B$类，加$1$为$D$类，加$2$为$A$类。选择加$2$成为$A$类。

- $0$。不加为$D$类，加$1$为$A$类，加$2$为$C$类。选择加$1$成为$A$类。

至此，这个问题就解决了。只需要$4$个判断。不过要提醒大家的是这种题目在比赛中一定要快速解决，争取拿到更多的分（我就手太慢了只拿了$484$分）。

# include <bits/stdc++.h>

	scanf("%d",&n);
	n%=4;
	if(n==3){
		printf("2 A");
		return 0;
	}
	if(n==2){
		printf("1 B");
	}
	if(n==1){
		printf("0 A");
	}
	if(n==0){
		printf("1 A");
	}
	return 0;
}
