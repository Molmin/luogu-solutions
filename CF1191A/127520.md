## 首先的共识
因为%4的余数有四种，可以选到3种，所以加完后的等级要么A级要么B级

我们只要考虑可不可以达到A，不可以就输出B

思路：弄出理想的A级血量，再判断可不可以实现，不可以实现就输出B级

根据以上思路，贴出AC代码：

```
#include<bits/stdc++.h>//万能头
using namespace std;
int a,b;
int main(){
	cin>>a;
	b=a/4*4;
	if(a%4!=0) b+=4;
	b++;//这三行就弄出了理想的血量值，但是当a%4为1时会出问题，所以下面的%4就是属于特判
	if((b-a)%4<=2) printf("%d A",(b-a)%4);//当血量可以实现时，输出A级与增加HP
	else{
		b-=2;
	printf("%d B",(b-a)%4);//如果A级达不到，b就变为理想的B级血量
}
	return 0;
}
```
