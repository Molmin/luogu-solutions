## [题目](https://www.luogu.org/problemnew/show/P1077)

*	个人认为这是一道比较裸的dp题目，我们看看我们的摆花方案数与什么有关，与摆花的花数量和摆花的花种数有关，所以我们的动态规划数字f[i][j]的意思是i种花放j盆的摆花方案数。因为花是按顺序排的，这就大大减小了我们动态规划的麻烦。

*	我们可以知道i种花放j盆的摆花方案数其实为放i-1种花，放 j-我们新放入的花的可放入盆数 到 j 的方案总数和，即 f[i][j]=f[i][j]+f[i-1][j-k] （k为0~a[i] a为存放一开始第二行读入的数据）

*	初始化的化我们不能初始化f[1][1]因为f[1][1]不一定为一种方案(~~我一开始在这错了好多次~~)，我们要赋值的是f[0][0],这个我们可以保证为1种方案。


------------
    

```cpp
#include<bits/stdc++.h>
using namespace std;
int f[1001][1001],a[10001],n,m;
//f[i][j]表示i种花放j盆的摆花方案数
int main()
{
    cin>>n>>m;//读入
    for (int i=1;i<=n;i++)
    {
    	cin>>a[i];//读入
    }
    f[0][0]=1;//赋初始值
    for (int i=1;i<=n;i++)//枚举放花的种数
    {
        for (int j=0;j<=m;j++)//枚举放花的盆数
        {
            for (int k=0;k<=a[i];k++)//枚举新放入放花的盆数
            {
            	if (j-k>=0)
                {
                	f[i][j]=(f[i][j]+f[i-1][j-k])%1000007;
                }
            }
        }
    }
    cout<<f[n][m];//输出
    return 0;//结尾
}
```
