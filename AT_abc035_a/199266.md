题意就是说：输入电视的长和宽，然后能化简得要化简，问化简之后的结果是4：3还是16：9  
题意也没有说有其他比例或者如果是其他比例怎么办，那我们就不用管了  
主要就是化简过程比较重要吧，求出两个数的最大公约数，然后都除以下就好了  
STL里面有现成的求最大公约数函数  
```cpp
__gcd(x, y);
```  
当然要手写也是最好的，勤劳改变人生（说明我很懒），陪一陪Pascal的同学  

---
代码  
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
	int a, b;
	scanf("%d %d", &a, &b);
	int GCD = __gcd(a, b);
	a /= GCD, b /= GCD;
	if (a == 4 && b == 3) puts("4:3");
	else puts("16:9");
	return 0;
}
```
