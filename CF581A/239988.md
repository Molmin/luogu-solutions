# CF581A C++题解
tip: 我坚信，这是最简做法。
### 题目描述
- 有 $a$ 只红袜子， $b$ 只蓝袜子。
- 优先选择不同颜色的穿，即 $1$ 红 $1$ 蓝。
- 当无法组成不同色时，才穿同色的，即 $2$ 红或 $2$ 蓝。
- 直到袜子数不足两只为止。
### 题意分析
根据题意，可以发现，一开始总是穿不同颜色袜子的。等到什么时候呢？正好就是其中一种颜色用完（也有可能同时用完）的时候。不难，得出第一个数据为 `min(a,b)`。

之后，便是同色的，根据刚才分析的，也就是最多存在 $1$ 种颜色，剩下的数量除以 $2$ 就是第二个数据，也就是`(max(a,b)-min(a,b))/2`,根据数学定理，化简可以发现就是`abs(a,b)/2`。

OK，题目分析完毕，开始写代码！
### 代码
```cpp
#include<bits/stdc++.h>//万能头包含了math.h库
using namespace std;
int a,b;
int main()
{
    cin>>a>>b;//输入
    cout<<min(a,b)<<" "<<abs(a-b)/2;//处理+输出
    return 0;
}
```
