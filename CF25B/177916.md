实际上这道题很简单。

>题目描述：给一串长度为n的数字串，在两个或三个数字之间加'-'，使得这个数字串便于记忆。 如果有多解，随意输出其中一个就好

既然随意输出一个就好，那我们就输出我们认为最简单的那个。

#### 最简单？

必须分奇偶数讨论了。

## 偶数 123456  ->  12-34-56

## 奇数 1234567 -> 123-45-67

就是这样，偶数两两一段，奇数前面先输出三个，然后两两一段。

题目中标明了长度范围$n(2<=n<=100)$，那么奇数情况下就要特判一下了。

奉上AC代码
```
#include<bits/stdc++.h>
using namespace std;
void ev(string a,int la)//偶数情况
{
    for(int i=0;i<la;i++)
    {
        cout<<a[i];//每输出两个，后面加一个减号，注意最后的那个可不能输出！
        if(i%2==1 && i!=la-1)cout<<'-';
    }
    cout<<endl;
    return ;//好习惯
}
void od(string a,int la)//奇数情况
{
    cout<<a[0]<<a[1]<<a[2];//先输出前三个
    if(la!=3)cout<<'-';//如果长度不只就三个，那就输出减号
    else return ;//否则(长度=3)，输出完事后直接拜拜。
    for(int i=3;i<la;i++)//依旧是这样
    {
        cout<<a[i];//每输出两个，输出一个减号，我们发现，只有数组下标达到4 6 8等数时输出减号，所以就是i是偶数的时候。
        if(i%2==0 && i!=la-1)cout<<'-';
    }
    cout<<endl;//别忘了换行
    return ;
}
int main()
{
    int la;string a;
    cin>>la>>a;//直接把长度给你了，如果没有给还要用la=a.size();去求字符串长度
    if(la%2==0)ev(a,la);//如果这些是全局的都不用传参。
    else od(a,la);//even是偶数，odd是奇数，这里函数名用缩写
    return 0;
}
```
