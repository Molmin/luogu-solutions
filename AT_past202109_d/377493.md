本题为一道较简单的数学题及模拟题。
- 首先，先阅读题目。题目大意就是输入两个数，然后比较两个数因数个数。
- 然后，要弄清楚什么是因数。题目最后也简单地解释了一下：比如两数 $a$ 和 $b$，如果 $a$ 能整除 $b$ 或 $b$ 能整除 $a$，则 $b$ 是 $a$ 的因数或 $a$ 是 $b$ 的因数。
- 最后，理清思路。先编写一个计算一个整数因数个数的函数，并将其命名为 $cul$；主函数中输入 $x$ 和 $y$，建立 $2$ 个变量来存储 $x$ 和 $y$ 两数各自的因数个数；条件语句来判断哪个数因数个数多，如果 $x$ 因数个数更多，则输出大写字母 $X$；如果 $y$ 因数个数更多，则输出大写字母 $Y$；如果两数因数个数一样多，则输出大写字母 $Z$。
- 那么怎么计算一个数有多少个因数呢？用 for 循环以及计数变量来解决。从 $1$ 循环到此数，以此判断如果此数除以正在循环的这个数余数为 $0$，则将计数变量加 $1$ 即可。
- 下面为我的 AC 代码：
```cpp
#include <iostream>
using namespace std;
int cul(int x) {
		int cnt = 0;
		for (int i = 1;i <= x;i++) {
			if (x % i == 0) cnt++;
		}
		return cnt;
}
int main() {
		int x,y;cin >> x >> y;
		int ans1 = cul(x), ans2 = cul(y);
		if (ans1 > ans2) cout << 'X';
		else if (ans1 < ans2) cout << 'Y';
		else cout << 'Z';
		return 0; 
}
```
- 当然，$cul$ 函数中的内容完全可以放在主函数中写，但是我认为用一个函数打包起来看着更简约一些，这些都是个人喜好问题，不过多阐述。
