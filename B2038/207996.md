这是一篇 **Python 题解**。

我们可以将题目要求分解为三步：

1. 将字符转化为 ASCII 码。
1. 求出该 ASCII 码对 $2$ 的余数。
1. 根据余数输出 `YES` 或者 `NO`。

首先来看第一步，Python 中的 `ord(s)` 函数接受一个**长度为 $1$** 的字符串 `s`，返回它对应的 ASCII 或 unicode 数值。这样就完成了第一步。

第二步可以通过位运算来解决：

对于一个整数，如果它的最后一个二进制位为 $1$ 则它是奇数，否则是偶数。

第三步可以使用 Python 的「内联 `if` 语句」（类似于 C 等语言中的三目运算符）来解决。

```python
print('YES' if ord(input().strip())&1 else 'NO')
```
