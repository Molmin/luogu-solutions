### 题意

将一个数列中的一些子区间进行反转，求数列中任意相邻两数在之和是多少。

### 解题思路

很多新手一看到题目就被吓跑了，以为要用 DP 或一些其他的算法，其实这题非常简单。

先说结论：答案就是数列中两个最大数之和。

为什么是这样的呢？

举个例子：$[8,9,2,5,4,11,3]$

我们先找到最大数和次大数 $11$ 和 $9$，反转区间 $[2,5]$，即 $[9,2,5,4]$ 反转为 $[4,5,2,9]$，原数列变为 $[8,4,5,2,9,11,3]$，这时两个最大数就相邻了。

另一种反转方法：反转区间 $[3,6]$，即 $[2,5,4,11]$ 反转为 $[11,4,5,2]$，原数列变为 $[8,9,11,4,5,2,3]$

不难发现，对于任意一个数列，我们都可以采取相似操作：设两个最大数的下标分别为 $x,y$，反转区间 $[x,y-1]$ 或区间 $[x+1, y]$ 都可以使这两个数相邻。

### 代码实现


```cpp
int t;
cin >> t;
while (t--)
{
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++)
    {
        cin >> a[i];
    }
    sort(a + 1, a + 1 + n);
    cout << a[n] + a[n - 1] << endl;
}

```
