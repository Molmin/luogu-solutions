# 题目大意
有多组数据，对于每组数据，有多个数字，对其中的一段进行一次翻转，求反转后相邻两个数字之和的最大值。
# 思路
本题要找相邻两个数字和的最大值，那能否把最大的两个数弄在一起，这样保证答案是最大的？答案是可以的。

1. 当最大的两个数相邻，将这两个数反转，反转后二者仍在一起。

2. 当最大的两个数不相邻，设这两个数的编号分别是 $r$ 和 $q$ ,且 $r<q$ ,则交换 $r$ 到 $q-1$ 的数，这样最大的两个数就相邻了。

由此本题就变成了找每组数据中最大的两个数的和。

# 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int t;
    cin>>t;
    while(t--){
        int n,m1=0,m2=0,ll;//用m1和m2存储最大的两个数
        cin>>n;
        while(n--){
            cin>>ll;
            if(ll>m1){
                m2=m1;
                m1=ll;    
            }
            else if(ll>m2) m2=ll;
        }
        cout<<m1+m2<<endl;
    }
    return 0;
}
```
