## 题意翻译
### 题目描述
给出了一定时间的雪深H1和1小时前的雪深H2。此时，请计算并输出雪深度差H1-H2。

输入输出格式
输入格式：

标准输入格式如下

第一行：给出某一时刻的雪深h1

第二行：一小时前的雪深h2

输出格式：输出h1-h2 结尾需加空格
### 输入输出样例
输入 #1
```
15
10
```
输出 #1
```
5
```
输入 #2
```
0
0
```
输出 #2
```

```

```

```
输出 #3

