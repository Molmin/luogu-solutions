## 桶

由样例可知，这是让我们判断唯一奇偶性与其他数不同的数的位置。

所以直接开桶，记奇/偶数出现次数，可以省码量。
```cpp
#include<cstdio>
int n,a,x[6],ans[6];//x[0]表示偶数个数，x[1]奇数个数。
//ans[0/1]表示偶数/奇数最后一次出现位置。
//因为目标位置唯一，所以目标ans只会有一次更新。
int main(){
	scanf("%d",&n);
	for(int i=1;i<=n;++i){
		scanf("%d",&a);
		++x[a&1];
		ans[a&1]=i;
        //a&1=a%2，作用如上
	}
	if(x[0]==1)printf("%d\n",ans[0]);//偶数只出现一次时
	else printf("%d\n",ans[1]);//奇数只出现一次时。
	return 0;
}

```
