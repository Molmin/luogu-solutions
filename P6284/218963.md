## 模拟题
模拟，按照题意分析，__题目让我们干什么我们就干什么__。  
***
\[思路\]：问Pero第n+1个月能用多少流量，可以把Pero的n+1个月可用流量的总和算出来，再用循环把前面月份用的流量减掉，就是第n+1个月剩余的流量了  
***  
我的代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int x,n;//x是每月可使用的流量，n是n个月中每月用的流量 
	int a[105];//a数组用来存储n个月中每月用的流量，例如a[0]就是第一个月中用的流量 
	cin>>x>>n;
	int sum=x*(n+1);//这里的sum是到n+1个月的可用流量总和 
	for (int i=0;i<n;i++) cin>>a[i];//输入n行数据，注意这个程序中是从a[0]开始存储，存到a[n-1] 
	for (int i=0;i<n;i++) sum-=a[i];//按照刚才的顺序，循环n次，每次都用sum减去这个月用掉的流量(sum-=a[i]与sum=sum-a[i]是同一个意思），最后剩下的就是这个月可以用的流量了 
	cout<<sum;//最后输出这个月可用的流量 
	return 0;
} 

```
