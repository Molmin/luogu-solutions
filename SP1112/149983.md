这种题找规律呗，先看题目中给的图，然后列个表：

| 数 | 坐标 |
| :-----------: | :-----------: |
| $0$ | $(0,0)$ |
| $1$ | $(1,1)$ |
| $2$ | $(2,0)$ |
| $3$ | $(3,1)$ |
| $4$ | $(2,2)$ |
| $5$ | $(3,3)$ |
| $6$ | $(4,2)$ |
| $7$ | $(5,3)$ |
| $8$ | $(4,4)$ |
| $9$ | $(5,5)$ |
| $10$ | $(6,4)$ |

列到这感觉规律就很明显了吧，不难发现**有数的坐标**有两种：$x=y$（如 $0,1,4,5...$）和 $x-2=y$（如 $2,3,6,7...$），然后就解决了是否有数的判断。接下来先处理 $x$ 为偶数的情况，再观察一下表格，发现可以用 $x+y$ 来解决，然后 $y$ 为奇数的情况就可以用 $x+y-1$ 解决，接着这个题就结束了。

代码：

```
#include<bits/stdc++.h>//Rp++
using namespace std;
inline int read()//快读 
{
	int s=0,w=1;
	char ch=getchar();
	while(ch<'0' || ch>'9')
		if(ch=='-')
			w=-1,ch=getchar();
	while(ch>='0' && ch<='9')
		s=s*10+ch-'0',ch=getchar();
	return s*w;
}
int t,x,y;
int main()
{
	int t=read();
	while(t--)//对于每组数据，三目判断是否有数，然后判断奇偶即可 
		x=read(),y=read(),!(x==y||x-2==y)?puts("No Number"):(x&1?printf("%d\n",x+y-1):printf("%d\n",x+y));
    return 0;
}
```
