# 	我的第一篇题解
今天必须要给大家讲一种~~鬼畜~~实用的作法，不知道算不算桶排序，大概就是每输入一个数据，就把数组名[数据]+1，让表示数据的变量数组名[数据]不为空，~~顺带~~起到了查重的作用。
## 再看一下题目
好吧，n<=100,数据<=1000，范围很小，
## 最适合这种做法了！
好了，不多说，上代码...（经测试，可以AC）
```cpp
#include<iostream>
using namespace std;
int a[101];
int b[1001]={0};//所有桶清零
int main()
{
    int n,cnt=0;
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        cin>>a[i];
        b[a[i]]++;//使表示数据的桶不再为空
    }
    for(int i=1;i<=1000;i++)
    {
        if(b[i]!=0)
        {
            cnt++;//统计查重，把出现了1次或1次以上的数据统计
        }
    }
    cout<<cnt<<endl;
    for(int i=1;i<=1000;i++)
    {
        if(b[i]!=0)
        {
            cout<<i<<" ";//桶的编号，即是表示的数据
        }
    }
    return 0;
}
```
