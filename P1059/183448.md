## 奉上AC代码，18行搞定
```cpp
#include<iostream>
using namespace std;
int i,n,x,tot=0,a[101];
bool t[1001];
int main()
{
	cin>>n;
	for(i=1;i<=n;i++)
	{
		cin>>x;
		t[x]=true;
	}
	for(i=1;i<=1000;i++)
	 if(t[i]) a[++tot]=i;
	cout<<tot<<endl;
	for(i=1;i<=tot;i++) cout<<a[i]<<' ';
	return 0;
}
```
为什么越是这种简单题大家的代码越长……
不知道大家是想麻烦了还是写麻烦了……

在这里蒟蒻提供一个超短代码……

哪怕是背代码的新手也可以轻松记住

虽然是道红题，但我还是稍讲解下吧：

读题，题目中非常明显的提示我们：“去重”和“排序”——这就是桶排！

再看一眼数据范围，最大的数只有1000，我们只需要开个1000的桶就可以了。

不会桶排的可以看一下第一页大佬的讲解，我只负责提供精简代码就不详细讲解了。
