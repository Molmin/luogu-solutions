这道题需要我们完成**排序**以及**去重**两项工作。

### 看到排序，肯定要请我们**STL**中的sort大法了！

 _ps:sort的时间复杂度为（nlogn）_ 
 
 在所有的排序中，sort的代码量是最短的！而且它可以加自定义函数！
 
  _ps:简化后的的用法为sort(数组名,数组名+数量,~~自定义函数（可有可无）~~），当然这是从0到n-1的情况，如果是1到n的情况，我们需要在数组名后+1。_ 

示例：sort（a,a+n)和sort(a+1,a+n+1)

但是，在这如星河般茫茫的一堆数中，我们如何实现**去重**呢？

在众多方法中，我使用的是与前者进行比较。

与前面的数进行比较，如果它与前面的数不同，我们就将num（计数君）++；

如果要这样比较的话，我们就需要从第一个位置读入，而不是第0个位置。

 _ps：第零个位置，我插入了一个0，因为题目里的范围是1~1000。_ 

要知道数量，我们可以使用两个分开来使用的for循环.

虽说这样的时间复杂度会高一些(即乘以2），但由于题目的数据范围比较小，也就无妨。

最后，附上我的14行AC代码：
```c
#include<bits/stdc++.h>//万能头文件 
using namespace std;
int main(){
	int n,i,num=0;
	cin>>n;//n指的是数据个数 
	int a[10000];
	a[0]=0;//这一句是为了方便a[1]与前面的数进行比较 
	for(i=1;i<=n;i++) cin>>a[i];//标准的读入 
	sort(a+1,a+n+1);//重点：STL的排序 
	for(i=1;i<=n;i++) if(a[i]!=a[i-1]) num++;//计数 
	cout<<num<<endl;//输出个数 
	for(i=1;i<=n;i++) if(a[i]!=a[i-1]) cout<<a[i]<<" ";//输出排序好的数 
	return 0;//结束 
}
```
