其实重点题目都描述清楚了：**排序和去重**。
### 排序（sort大法好）
如果不知道sort的童鞋，可以去百度了解一下。毕竟是一个万能排序函数。sort函数在头文件algorithm（算法）里面。处理此问题的调用方法是:
```
sort(数组名，数组名+数据个数);
```
排序完成后输出即可。

当然sort还有很多用法，这只是其一。
### 去重（bool判断）
怎么去重？输入一个数，把前面的数全部搜索一遍看有没有重复？

不不不。这种方法耗时又长（虽然这么写也不可能TLE），代码量也长（重点QWQ）。我只需要开一个bool数组，假设输入了10,我就把bool[10]打上flag（赋值为true），后面输入的10因为bool[10]为true,就会被残酷的拒之门外QWQ（读不进数组）。其他数据同理。

只要把握好这两个重点，用什么方法都可以啦。
### 上19行代码！
```
#include<bits/stdc++.h>
using namespace std;
int n[105],g,tmp,gg;//这里要单独开一个计数器。因为过滤过了数字，所以实际存储的数据个数是不等于输入的数据个数的。
bool pn[1005];//bool数组判断
int main()
{
    cin>>g;
    for(int i=0;i<g;i++)
    {
    	cin>>tmp;
    	if(pn[tmp]==1)continue;
    	pn[tmp]=1;
    	n[gg++]=tmp;
    }
    sort(n,n+gg);
    cout<<gg<<endl;
    for(int i=0;i<gg;i++)cout<<n[i]<<" ";
}
```
