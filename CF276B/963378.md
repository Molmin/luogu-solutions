### 理解题意

一道考验思路的好题。

给定一个字符串 $s$，字符串内容全为小写，要求 First 和 Second（先手与后手）轮流取出其中的小写字母。如果当前字符串通过排序可以获得回文字符串，则当前轮到的玩家获胜。在双方都使用最优策略的情况下，判断哪个玩家会赢。

### 思路分析

如何判断字符串是否可以通过排序得到回文字符串？

给字符串内每个字符种类一个计数器，若每个计数器计数后为偶数，则此字符串可以通过排序回文。

特殊情况：若字符串中有且仅有一种字符数量为奇数，则也可以通过排序回文。

在此之前，我们需要把“阻碍”回文的字符让两名玩家轮流删掉。所以只需要判断需要删除的字符总个数是奇数还是偶数就好了。若为奇数，或没有需要删除的，则先手胜；否则，后手胜。

还有不懂的看注释。

### 代码实现

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,m;
string s;
int a[27],sum;
int main()
{
   cin>>s;
   for (int i=0;i<=s.size()-1;i++)
   {
    a[s[i]-'a'+1]++;
	}
	for (int i=1;i<=26;i++)
	{
		if (a[i]%2==1) sum++;//将需要删除的字符总个数进行统计
	}
	if (sum%2==1||sum==0)//先手胜利条件
	{
		cout<<"First"<<endl;
		return 0;
	}
	else cout<<"Second"<<endl;
}
```
