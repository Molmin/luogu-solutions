## 一道“数论”题目  
#### 题目大意： 在走廊中有n个灯泡都是关上的，第i趟拔动编号能被i整除的灯的开关，问走过n趟之后，第n盏灯是否是亮的，如果亮输出yes，否则输出no
#### ~~那我们来纯模拟来做一遍吧，然后信心满满地看向数据范围~~……  
### 于是乎就来找一找规律： 
**第n盏灯**，怎么确定他是开的关的——决定于它的**因数**。  
很显然，如果它的因数是**偶数**个，那么这盏灯最后的状态便是**关**；如果它的因数是**奇数**个，那么这盏灯最后的状态便是**开**  
### 那么有什么样的性质的数的因数的奇数个呢？  
那就是**完全平方数**  
**25**是**完全平方数**，除了**1**和**25**（**它本身**）以外，25还有一个因数**5**，所以25的因数是**奇数个**的  
### 所以我们现在的目标很明确，就是判断n是不是完全平方数，如果是，输出yes，否则输出no  
我们将**n开平方**，放进一个**int变量m**里面，如果根号n为**整数**，那么**m=根号n**；否则**m=根号n的整数部分**  
然后再**m * m**，看看是否等于n；等于n那就说明**n是完全平方数**，否则**就不是**  
最后我们就可以**AC**这道题了！

------------

### 代码如下：

```cpp
#include <cstdio>
#include <iostream>
#include <cmath>
using namespace std;//头文件

long long n,m;//为了保险，开long long

int main(void)
{
    while(scanf("%lld",&n)&&n!=0)//输入，由于不知道有多少个数要读进去，所以用while（记得要判断n≠0，虽然不判断可以过，但是实际上是不正确的，会额外输出一个“yes”）
    {
        m=sqrt(n);//sqrt函数开平方
        
        if(n==m*m) printf("yes\n");//如果等于n，那么输出yes
        else printf("no\n");//否则输出no
    }
    
    return 0;//完美的结束！
}


```
