## 翻译
有 $N$ 个广告牌，第 $i$ 个广告牌设置的时间为 $S_i$，且设置地点相同，为京都大学。你想看到所有广告牌，并且你可以在喜欢的实数时刻从家一瞬间来到京都大学，看到广告牌。

现在，给出一个时间 $T$，每当时间到达 $T$ 的倍数时刻，广告牌便会被拆下，你也会在一瞬间回家！

现在请求出你需要至少去几次京都大学才能看遍所有广告牌。

## 思路
如果你的设想是一个一个遍历，一个一个判断，那来看这篇题解就是一个明智的选择！

一开始，我的思路也是一个一个遍历，一个一个判断，但未免有些许差错。

我们不妨这样理解：每次在 $T$ 秒的时间段里，只要有广告牌摆放，你都要去看；也就是说，我们可以把所有时间点分成堆，统计堆的个数即可。

遍历 $N$ 次，如果 $S_i \div T$ 的值和其他的一样，就把它们放在一个堆里，最后统计个数。

值得一提的是，$S_i$ 不是按从小到大的顺序排列，所以可以有选择性的排一下序。

## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a[200010],t,js=0,ans=1;
int main()
{
	cin>>n;
	for(int i = 1;i<=n;i++)
	{
		cin>>a[i];
	}
	cin>>t;
	sort(a+1,a+n+1);//排序 
	for(int i = 1;i<=n;i++)
	{
		a[i]=a[i]/t;//分组前的准备 
	}
	for(int i = 2;i<=n;i++)
	{
		if(a[i]!=a[i-1])ans++;//分组，统计，合为一体！ 
	}
	cout<<ans<<endl;//撒花 
	return 0;
}
```
