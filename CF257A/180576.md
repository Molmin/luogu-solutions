### 题目大意：
Vasya 有 $m$ 个电器需要用电，
他有 $n$ 个插线板，
第 $i$ 个插线板有 $a_i$ 个插口，
每个插口可以插入另一个插线板或一个电器
他的公寓一共有 $k$ 个插座，
每个插座可以插入一个插线板或一个电器，
求他最少需要多少个插线板，
才可以让所有电器都通电。
若无论如何都无法让所有电器通电，
就输出 $-1$ 。

### 方法：贪心
每次选取插口最多的且没有使用过的一个插线板，
来替换掉任意一个通了电的电器，
并把这个电器转移到这个插线板上。
若所有的插线板都用上了，
还是无法达到目标，
就输出 $-1$ 。

### 代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,m,k,a[51];
int main()
{
	scanf("%d%d%d",&n,&m,&k);
	for(int i=1;i<=n;i++)
	{
		scanf("%d",a+i);
	}
	m-=k;//借m来表示当前未通电的电器
	//初始时不使用插线板，直接把电器尽可能插进插头 
	sort(a+1,a+n+1);//从小到大排序 
	int now=n,ans=0;
	while(m>0&&now>0)//注意now也不能>0 
	{
		m+=1;//替换 
		m-=a[now];
		now--;
		ans++;
	}
	if(m<=0)//等于0即为刚好，小于0即为还有空余，但是是最优方法 
	printf("%d\n",ans);
	else
	printf("-1\n");//用完了所有的插线板，还是有没有通电的电器 
	return 0;
}
```
