言归正传，这一题考的就是最基本的贪心。

先假设只有一个插座。那么先插孔最多的插线板。先把它插满。

如果能插完或者都插不满就没有用电器了。那么最好了，就不用其他插线板了。

但是如果插不完，那就拔去最后一个，再接孔第二多的插线板。以此类推。

一个插座的情况说完了。多个插座其实是一样的。

唯一需要想清楚的就是任意一个插线板（及上面的所有子插线板和用电器）都可以和一个单独的用电器互换。因为它们都是一个孔，还不用担心跳闸。

所以我们可以只留一个插座插插线板。其它的全部插上用电器就好了。或者要是用电器的的数量小于等于插座的数量就不用插线板了。

代码实现时注意一下，先把插线板（或插座）插满做判断后再删去不要写错了。

```
#include<bits/stdc++.h>
using namespace std;
long long n,m,k;
long long a[55]={};
int main()
{
	cin>>n>>m>>k;
	m-=k;
	if(m<=0)
	{
		cout<<0<<endl;
		return 0;
	}
	m++;
	for(int i=1;i<=n;i++)
	{
		cin>>a[i];
	}
	sort(a+1,a+n+1,greater<int>());
	for(int i=1;i<=n;i++)
	{
		m-=a[i];
		m++;
		if(m<=1)
		{
			cout<<i<<endl;
			return 0;
		}
	}
	cout<<"-1"<<endl;
	return 0;
}//应该不难理解
```
