这是一道贪心题。

## **题目思路：**

这一道题的基本思路是：我们先将插线板上插口的数量从大到小进行排序。然后模拟插插线板。当我们发现能插的插口数量已经满足电器的需求时，输出答案即可。

这道题有两种特殊情况，我们需要注意：

1. 当插座能够直接满足电器的需求时，我们需要在程序开头特判。如果插座的数量大于或等于电器的数量时，直接输出 0 即可。
2. 怎么插都无法满足电器的需求。当我们循环一遍过发现没有任何一种情况能够满足电器需求，即循环结束未能输出时，直接输出 -1 即可。

另外，我们需要注意：插线板本身就是需要一个插口的。所以我们在计算能够插电器的数量时应当减去它本身。

## **Code:**

```cpp
#include<iostream>
#include<algorithm>
using namespace std;
int n,m,a[55],k,ans;
int main(){
	cin>>n>>m>>k;
	if(m<=k){cout<<0;return 0;}//特判插座能够直接满足电器需求时，直接输出 0
	ans=k;//初始时能够插电器的数量即为插座的数量。
	for(int i=1;i<=n;i++)cin>>a[i];
	sort(a+1,a+n+1,greater<int>());//从大到小排序
	for(int i=1;i<=n;i++){
		ans=ans-1+a[i];//计算加入该插线板时，能够插电器的数量
		if(ans>=m){cout<<i;return 0;}//一旦满足要求，立即结束程序
	}cout<<-1;return 0;//如果不行，输出 -1
}
```
