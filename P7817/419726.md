## 最后
~~暴力打完一发上去 2 分到手~~

发现了一个显而易见的规律发现你输进去的 n 在 $<=2$ 时只会输出 $1$ ， 在 $>2$ 时只会输出 $0$

好，规律明显直接上代码
```c
#include <bits/stdc++.h>
#define int long long
using namespace std;
int n, x;
signed main()
{
    scanf("%lld", &n);
    for (int i = 1; i <= n; i++)
    {
        scanf("%lld", &x);
        puts(x <= 2 ? "1" : "0");
    }
    return 0;
}

```
