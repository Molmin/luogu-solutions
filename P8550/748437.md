## 思考

1. 当 $x$ 与任意 $i$ 使得 $a_i$ 同号一定能完成 $10^{100}$ 次操作。

例子：

```
x=10,a[1]=-1,a[2]=10,a[3]=-114514

```
$x$ 不断加 $a_2$ 即可。

2. 当第一个条件发现不能判断一定完成时，若有任意 $i$ 使得 $x\not\equiv 0\pmod {a_i}$ 不是 $x$ 的因数一定能完成 $10^{100}$ 次操作。

例子：

```
x=10,a[1]=-1,a[2]=-2,a[3]=-3

```
$x$ 不断加 $a_3$ 即可。

3. 当第一二个条件发现不行能判断一定完成时，若有任意 $i,j$ 使得 $a_i \neq  a_j$ 一定能完成 $10^{100}$ 次操作。

例子：

```
x=10,a[1]=-1,a[2]=-2,a[3]=-5

```
$x$ 先加 $4$ 个 $a_3$，再不断加 $a_3$即可。

（假设 $a_i \neq  a_j$，且$a_i <  a_j$，$x$ 先加上 ```x/a[i]-1``` 个 $a_i$，再不断加 $a_j$ 即可）

4. 只剩所有数都相等且与 $x$ 异号且 $x\not\equiv 0\pmod {a_i}$，此时一定执行不完（$1 \le \left| x \right|,\left| a_i \right| \le 100$）。

例子：

```
x=10,a[1]=-2,a[2]=-2,a[3]=-2

```

## AcCode

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int t;
    cin>>t;
    while(t--){
        int n,x,a[6]={0},f=0;
        cin>>n>>x;
        for(int i=1;i<=n;i++){
            cin>>a[i];
            if(i>1&&a[i]!=a[i-1])f=1;//不需要与前面的每一项判断。
            if(x>0&&a[i]>0)f=1;
            if(x<0&&a[i]<0)f=1;//异号
            if(x%a[1]!=0)f=1;//不是整倍数关系，即使一正一负也可以。
        }
        if(f) cout<<"Yes\n";
        else cout<<"No\n";
    }
    return 0;
}
```
