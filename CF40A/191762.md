**前言：**

觉得各位神犇们讲的都不是特别细致（说白了就是我语文不好听不懂），我来讲解一下最基本的方法。

**思路：**

一句话：先判断点在第几个圈上，再判断颜色。

要判断一个点在不在圆里，首先要看点离圆心的距离：

- 如果离圆心的距离小于半径，则该点在圆内；

- 如果离圆心的距离等于半径，则该点在圆上；

- 如果离圆心的距离大于半径，则该点在圆外。

把点到圆心的连线看作直角三角形的斜边，根据勾股定理，斜边的长度 $=$ 该点到圆心的距离 $= sqrt( (x1-x2)^2 + (y1-y2)^2 )$，其中$(x1,y1)$为点的坐标，$(x2,y2)$为圆心的坐标。

此题所有的圆心都在$(0,0)$，故公式变为$sqrt( (x1-0)^2 + (y1-0)^2 )$，最后为$sqrt(x1^2 + y1^2 )$

圆找完了接下来就该确定颜色了。不难看出，x轴与y轴把图案分成了四个部分，其中右上和左下是从内到外先黑后白，右下和左上是从内到外先白后黑。于是我们就想到了先判断点是在左上，左下，右上，右下的那个部分，再判断原点的距离向下取整的奇偶性来决定是黑还是白。

右上和左下的x坐标和y坐标均大于或小于零，右下和左上的x坐标和y坐标一个大于零，一个小于零。通过这个就能判断在哪个部分了。

最后一个细节：如果x或y坐标等于0，或者距离是整数，那么该点在边界上，所以是黑色。

**代码：**

```cpp
#include <cmath>
#include <cstdio>
#include <iostream>
using namespace std;

int x, y;
double dis;// 距离

int main()
{
	scanf("%d%d", &x, &y);// 读入坐标
	dis = 1.0 * sqrt(x * x + y * y);// 距离计算公式
	if (x == 0 || y == 0 || floor(dis) == dis) printf("black");// 如果x或y坐标等于0，或者距离是整数就是黑
	else if (x >= 0 && y >= 0 || x <= 0 && y <= 0)// 如果在右上或左下
	{
		//    ↓这里的(int)是强制转换，因为double不能%int（不太确定，反正Dev-C++会报错）
		if ((int)floor(dis) % 2 == 0) printf("black");// 如果能被2整除
		else printf("white");// 如果不能
	}
	else// 如果不在右上或左下，那就只能是右下或左上
	{
		if ((int)floor(dis) % 2 == 0) printf("white");// 如果能被2整除
		else printf("black");// 如果不能
	}
	printf("\n");// 闲的没事换个行
	return 0;// 完结撒花
}

```
