####  思路
这题就是要看判断输入的正整数能否分成两个正偶数。
众所周知，偶数只能成偶数+偶数。
所以只要他不是2并且是偶数即可
##### 代码部分，AC代码
```
#include<bits/stdc++.h>//万能头
using namespace std;
int main(){
	int a;
	cin>>a;
	if(a!=2&&a%2==0){//判断
		cout<<"YES";//正确输出YES
	}else{
		cout<<"NO";//否则NO
	}
	return 0;
}
```
