正偶数=正偶数+正偶数

所以先要判断是否为正偶数，而由于最小正偶数为2，因此最小为4。所以要同时满足。（**0不是正数！！！！**）
  

好了，上代码。
```
#include<bits/stdc++.h>//万能头文件
using namespace std;//命名空间
int main(){//主函数
	int a;//定义
	cin>>a;//输入
	if(a<=0){//判断正负
		cout<<"NO";//如为负直接输出
		return 0;//输出后立即退出（节约时间）
	} 
	else if(a%2==0 && a>=4){//判断奇偶且至少为4（两个2至少为四）
		cout<<"YES";
		return 0; //同上
	} 
	else{
		cout<<"NO";
		return 0;
	}//如均不满足，输出“NO”
	return 0;//结束程序
}
```
