# 坑多啊！

主要有这几个：

1. 第一个数字之前不用空格
2. 每两个数字之间空一格就可以了
3. 1是右对齐输出的

其他的就比较简单了。这一题我的思路是酱紫的：

首先，一个数字（我们以“8”为例），可以分为五个部分：

```
 -    ---1
| |   ---2
 -    ---3
| |   ---4
 -    ---5
```

1. 最上面的一横（可能是空的）
2. 中间的1/2竖
3. 中间的一横（也可能是空的）
4. 下面的1/2竖
5. 最下面的一横（也可能是空的）

那么这样的话，可以分成大概两大类：横和竖。

横线的话要么为空，我们就输出k+2个空格，要么为横，输出一个空格，接着输出k个横，再输出一个空。

竖线的话有：左竖，右竖，全竖三种情况。

左竖：先输出一个竖，接着k+1个空

右竖：先k+1个空，接着输出一个竖

全竖：一个竖，k个空，一个竖

那么如果是大一点的
```
 --
|  |
|  |
 --
|  |
|  |
 --
```

计算一下，总共$2k+3$行（代码里面是$4+2k-1$，懒得化简了），最中间一行是$k+2$，最下面一行是$4+2k-1$，最上面一行是1。

竖线的话就没有那么多限制，上面那一条（2号）就是大于1，小于$k+2$的情况，下面那一条（4号）就是大于$k+2$，小于$4+2k-1$的情况。

当k>1的话我们就用一个i变量来循环，表示输出的行数。

上面的横竖五种情况，我们写一个draw函数，传入一个参数c表示输出的五种情况。
