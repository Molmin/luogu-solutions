####  2019CSP-J T1数字游戏（~~名字真长~~）
首先，写题前要先看清题意。

给一个长度为8的01字符串，求
其中1的个数。

思路其实很明显。定义一个字符数组，再暴力记录1的个数。

理清思路后，就开始快乐的打代码啦！

 ```cpp
#include <bits/stdc++.h>//万能头
using namespace std;

int main()
{
	char x[8];//定义字符数组
	cin>>x;//输入
	int ans=0;
	for(int i=0;i<8;i++)
		if(x[i]=='1')ans++;//记录结果
	cout<<ans<<endl;//输出
	return 0;//别忘
}

```
