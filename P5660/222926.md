## 思路
首先，题目规定 _对于100%地点数据，长度为八_

所以可以确定答案<9

所以我们只要让ans加上getchar()-'0'八次就可以完美AC
```cpp
#include<cstdio>
int main()
{
    int ans=0,t=0;
    for(;t<8;t++)ans+=getchar()-'0';
    putchar(ans+'0');
    return 0;
}
```
