考虑n的奇偶性

如果n是偶数 答案是相邻两个数之和的最大值

为什么呢？

首先相邻两数之和的最大值显然是答案的下界，同时又能根据相邻两组最大值间隔取得到解

例如a = {1,2,3,4} 最大值为7
先取最大值两边的数
b[3] = {1,2,3} b[4] = {4,5,6,7}
那么b[2] = {4,5} ({6,7})b[1] = {1}({2,3})

显然，如果答案冲突那么就有另外一个相邻两数最大值
再考虑n是奇数
问题就有些棘手，因为环的存在我们要考虑这些数怎么取最优。
根据**杨颙大定理** 我们有一个自然的想法：
```
相邻两数尽量分散的取
例如b[1] 取 1，2
	b[2] 取 8，9  
```
那么我们又自然的想到二分这个最大值（想一想，为什么可以二分）
然后相邻两数尽量向极端取，答案的合理性就只需判断b[n] 和 b[1]

这道题就这么做完了，可见朴素算法也有它的优点
**对于取数，我们贪心的取**
**对于答案具有单调性，我们可以二分**
**这就是这道题给我们的启发**
