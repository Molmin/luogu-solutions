# 题意

给你一个 $a$，$b$，每次可以让 $a$ 加或减去 $k$,$k=[1,10]$。求 $a$ 变成 $b$ 的最小次数。

# 分析

我们记录一个 $c=|a-b|$。

在记录下 $s$ 为 $c$ 整除 $10$ 的商，

就是说，一开始我们就一直 加或减 $10$,直到 $a,b$ 之间相差 $10$ 以内。

如果 $c$ 不是 $10$ 的倍数，那么我们还要将 次数加一，

因为我们还需要加或减来让 $a$ 变成 $b$。

# 代码

**核心部分**

```cpp
void solve()
{
    while(m--)
	{
		int a,b;
		cin>>a>>b;
		long long zc=abs(a-b);//zc就是我说的c
		long long  zs=zc/10;//zs就是我说的s
		if(zc%10!=0)
		zs++;
		cout<<zs<<endl;
	}
}
```
