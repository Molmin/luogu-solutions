### 题意简述
输入男人的数量 $b$ 和女人的数量 $s$。如果 $b$ 和 $s$ 都为 $0$，结束程序。否则输入男人的年龄和女人的年龄，输出数据组数。如果每个男人都能与女人结婚，输出 $0$，否则输出有多少个男人无法与女人结婚和最小的男人的年龄。
### 题目分析
循环无限次，输入 $b$ 和 $s$。如果 $b$ 和 $s$ 都等于 $0$，结束程序。否则把表示男人年龄的最小值的变量初始化为一个很大的数。

循环 $b$ 次，输入男人的年龄后更新当前男人年龄的最小值，然后输入女人的年龄，输出数据组数。如果 $b \leq s$ 即男人的数量不超过女人的数量，输出 $0$。否则输出 $b - s$ 和男人年龄的最小值。

时间复杂度为 $O(n)$。
### 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    for(int i=1;;i++){//i表示数据组数
        int b,s;
        int minn=2147483647;//男人年龄的最小值，注意要初始化为很大的数
        cin>>b>>s;
        if(b==0&&s==0){
            exit(0);//结束程序
        }
        for(int j=1;j<=b;j++){
            int a;
            cin>>a;
            minn=min(minn,a);
        }
        for(int j=1;j<=s;j++){//求出当前男人年龄的最小值
            int c;
            cin>>c;
        }
        cout<<"Case "<<i<<": ";
        if(b<=s){//如果男人的数量不超过女人的数量
            cout<<0<<endl;
        }
        else{
            cout<<b-s<<" "<<minn<<endl;
        }
    }
}
```
