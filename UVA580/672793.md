# 思路：打表

## 方法
先打递推，再打表。我们可以先找安全组合，我们惊奇的发现这是一个斐波拉契数列（不同的是本题为前三项之和）。那么我们就可以用数组 $f$ 储存安全组合个数，$f_i$ 表示当 $i$ 个盒子放在一起时，安全组合的个数。

初始化 ：$f_0=0,f_1=2,f_2=4,f_3=7$。

循环递推 ：$f_i=f_{i-1}+f_{i-2}+f_{i-3}$。

最后只需要用总组合数减去安全组合数就可以得到危险组和个数了。

总组合个数为 $2^n$。

下面附上 AC 代码！！！

## Code
```cpp
#include<bits/stdc++.h>
#define int long long
const int N=50;
using namespace std;
int n,ans,f[N]={0,2,4,7,13,24,44,81,149,274,504,927,1705,3136,5768,10609,19513,35890,66012,121415,223317,410744,755476,1389537,2555757,4700770,8646064,15902591,29249425,53798080,98950096};//打表 
signed main()
{
	while(1)//循环输入 
	{
		cin>>n;
		if(n==0) return 0;
		ans=pow(2,n)-f[n];//答案 
		cout<<ans<<endl;//输出 
	}
	return 0;//收场 
}
```
