### 算法简析
既然本题是道难度800的题，就不难想出解决方案。  
由于$n$是偶数，我们就简化模型：给定$a$,$b$,求出一组$p$,$q$，使得$ap+bq=0$。很容易发现：当$p=-b$,$q=a$时，原方程可化为$-ab+ab=0$，此方程恒成立。因此，这道题的思路就很清晰了：每输入一对数，依次输出第二个数的相反数、第一个数。
### demo of c/cpp
```c
#include<stdio.h>
#define in(x) scanf("%d",&x)/*输入*/
#define outs(x) printf("%d ",x)/*带空格输出*/
int main(void){
    register int a,b,t,n,i;
    /*使用register关键字将需要反复修改的变量存入CPU寄存器中加快运行速度*/
    in(t);
    while(t--){
        in(n);
        for(i=1;i<=n/2;i++){/*n为偶数*/
            in(a);in(b);
            /*输入一对数*/
            outs(-b);outs(a);
            /*依次输出第二个数的相反数、第一个数*/
        }
        putchar(10);/*换行，ASCII(10)即为'\n'*/
    }
    return 0;
    /*至此，本解法结束。
    对于每个数据点：
    时间复杂度：O(n)
    空间复杂度：O(1)
    可以通过本题。
    */
}
```
