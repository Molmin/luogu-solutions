## 题意
给出三个点，求过这三个点的圆的方程的标准式和一般式。
## 分析
其实就是求圆心坐标和半径，可以先列方程。
$$(x_1-x_0)^2+(y_1-y_0)^2=R^2$$
$$(x_2-x_0)^2+(y_2-y_0)^2=R^2$$
$$(x_3-x_0)^2+(y_3-y_0)^2=R^2$$
发现这几个值是没法高斯消元的，继续变形，我们的目的是消去平方项。所以考虑，一式减去二式，二式减去三式这种方法：
$$(x_1^2-x_2^2+ 2x_0\times (x_2-x_1) )+(y_1^2-y_2^2+ 2y_0\times (y_2-y_1) )=0$$
$$(x_2^2-x_3^2+ 2x_0\times (x_3-x_2) )+(y_2^2-y_3^2+ 2y_0\times (y_3-y_2) )=0$$
再移项可得：
$$
2(x_2-x_1)\times x_0+2(y_2-y_1)\times y_0=x_2^2-x_1^2+y_2^2-y_1^2
$$
$$
2(x_3-x_2)\times x_0+2(y_3-y_2)\times y_0=x_3^2-x_2^2+y_3^2-y_2^2
$$
这就是个二元一次方程，就可以高斯消元了。
## 代码
```cpp
for(int i = 1;i <= 2;i++)
a[i][1] = 2*(s[i+1].x-s[i].x),
a[i][2] = 2*(s[i+1].y-s[i].y),
a[i][3] = s[i+1].x*s[i+1].x-s[i].x*s[i].x+s[i+1].y*s[i+1].y-s[i].y*s[i].y;
for(int i = 1;i <= 2;i++)
{
	for(int j = 1;j <= 2;j++)
	{
		if(i!=j)
		{
			double temp = a[j][i]/a[i][i];
			for(int k = 1;k <= 3;k++)
			{
				a[j][k] -= a[i][k]*temp;
			}
		}
	}
}
if(fabs(a[1][1]) <= 1e-10) s[4].x = 0;
s[4].x = a[1][3]/a[1][1];
s[4].y = a[2][3]/a[2][2];
```
