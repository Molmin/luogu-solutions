把 $\frac{k}{100}$ 约分，然后输出**分母**就好了。

为什么呢？是因为我们其实就是想找到一个比值和 $k:100-k$ 相等的一个比 $a:b$，使 $a+b$ 最小，很明显 $a:b$ 为 $k:100-k$ 的**最简整数比** 时 $a+b$ 最小
然后就变成了求 $\frac{k}{100-k}$的最简分数的分母和分子之和，但我们发现这样要先减一次再加一次，实际上可以让分母先加上分子，也就是变成 $\frac{k}{100}$，再约分，输出分母就好了。为了追求更快的速度，我打了一个表。
```cpp
#include<cstdio>
const int ans[]={100,50,100,25,20,50,100,25,100,10,100,25,100,50,20,25,100,50,100,5,100,50,100,25,4,50,100,25,100,10,100,25,100,50,20,25,100,50,100,5,100,50,100,25,20,50,100,25,100,2,100,25,100,50,20,25,100,50,100,5,100,50,100,25,20,50,100,25,100,10,100,25,100,50,4,25,100,50,100,5,100,50,100,25,20,50,100,25,100,10,100,25,100,50,20,25,100,50,100,1}; 
int main() {
	int t,k;std::scanf("%d",&t);
	for(;t--;){std::scanf("%d",&k);std::printf("%d\n",ans[k-1]);}
	return 0;
}
```
