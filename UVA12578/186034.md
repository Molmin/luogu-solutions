>孟加拉国国旗是绿色长方形，长度为$L$ ，长宽比为 $10:6$。绿色的背景上有一个红色的圆圈。长度 $L$ 与半径比为 $5:1$。

求红色部分和绿色部分的面积。

我记得圆的面积推导过程好像小学六年级的数学课本上有，让我们回顾一下吧：

把圆平均分成若干份，可以拼成一个近似的长方形。长方形的宽就等于圆的半径（$r$），长方形的长就是圆周长（$C$）的一半。长方形的面积是$ab$，那圆的面积就是：圆的半径（$r$）的平方乘以$π$。

所以得出，圆的面积公式为：$πr^2$。

根据题意，得出：

长方形的宽$=L/10*6$

圆的半径$=L/5$

圆的面积$=$圆的半径$^2*π$。

有人说$π$取$3.14$精度不够？我会被圆周率前$100$位，前$10$位应该就够了吧。

$π=3.1415926535897932384626433832795028841971...$

```cpp
#include<bits/stdc++.h>
using namespace std;
double x,y,r,pai=3.14159265358979323846,green,red;
int n;
int main() {
	scanf("%d",&n);
	for(int i=1; i<=n; ++i) {
		cin>>x;
		y=1.0*x/10*6;
		r=1.0*x/5;
		red=pai*r*r;
		green=(x*y)-red;
		printf("%0.2lf %0.2lf\n",red,green);
	}
	return 0;
}
```
