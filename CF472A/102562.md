题目要输出合数，我们来看一下各种情况  
- $12=6+6$
- $13=4+9$
- $14=6+8$  

各位有没有发现  
当$N$是$4$的倍数时，直接$/2$.  
当$N$是$2$的倍数时，一个数是$N/2-1$，一个是$N/2+1$.  
否则，就需要一个试验了，最小合数$4$与最小奇数合数$9$之和为$13$,也就是$N$所可能是的最小奇数。  
如果是$15$,则合数$6$与最小奇数合数$9$相加就是$15$  
这样，程序就很简单了。

```pascal
var
 n:longint;
begin
  readln(n);//输入
  if n mod 4=0 then writeln(n div 2,' ',n div 2)
    else if n mod 2=0 then writeln(n div 2-1,' ',n div 2+1)
      else writeln('9 ',n-9);//照上面的做
end.
```
这就结束了
