这题其实是一个赤裸裸的抽屉原理（或者鸽笼原理），偶的蒟蒻方法如下：

- 先将每门学科都假设成2分
- 若剩下的分数大等于学科，则直接输出0
- 否则输出剩下分数与学科数的差

需要注意的是，我们需要特判$n=1$的情况，不然就会听取$WA$声一片。。。

$Code:$
```
#include <bits/stdc++.h>//万能头文件
using namespace std;
int main()
{
	int n,m;
	cin>>n>>m;//输入
	int p=n*2;//将每个学科都假设成2分
	if(n==1) cout<<(m==2);//特判n=1
	else if(m-p>=n) cout<<0;//若剩余分数大等于学科数，输出0
	else cout<<n-m+p;//对于其它情况，输出剩余分数与学科数的差
	return 0;//完美地结束
}
```
