####  我们可以用一个代数式来确定最终的答案：$3n-k$

$1.$  当这个代数式的值不是正数时，说明 $k ≥ 3n$，即 $k/n ≥ 3$ ，这就意味着 $k$ 能够被拆成 $n$ 个不小于 $3$ 的整数，所以在这种情况下答案为 $0$

$2.$  当这个代数式的值为正数时， $k < 3n$ ， $k$ 不能够被拆成 $n$ 个不小于 $3$ 的整数，又因为得分的范围在 $2-5$ 之间，所以至少有一个科目分数为 $2$ 。

$3.$  在 $(2.)$ 的情况下，最终的答案是多少呢 ？ 如果要使所有的科目得分不为$2$ ，就至少需要 $3n$ 的分数，但现在少了 $(3n-k)$ 分。对于 $3n$分来说，每一个 $2$ 分的科目都会使总分少 $3-2=1$ 分，所以 $2$ 分的科目一共有 $(3n-k)/(3-2)=3n-k$ 分。

#### 综上所述，当 $3n-k ≤ 0$ 时，答案为 $0$ ，否则答案为 $3n-k$ .可以简写成 $max( 0 , 3n-k )$

代码（很短，只有 $8$ 行）：
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,k;
int main()
{
	cin>>n>>k,cout<<max(n*3-k,0);
	return 0;
}
```
