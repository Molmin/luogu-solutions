这里采用了和上一篇题解不一样的思路：

由题意得每科的得分至少为2，那么就先假设这个人每科都得了2分，还剩下 $k-2n$ 分可以分配到这 $n$ 科里。

因为题目要求使得2分的学科最少，所以要尽可能地使这 $k-2n$ 分 分配到较多的科目。显然，每个学科都分配1分是最好的办法。

所以可以使 $\min(k-2n,n)$ 个学科的得分超过2分，最终的答案就是 $n-\min(k-2n,n)$

其实如果把这个式子整理一下，就得到了和上一篇题解一样的结果了。。


```
var
    n,k,t:longint;
begin
    readln(n,k);
    t:=k-n*2;
    if t>n then writeln(0) //答案为n-n=0
    else writeln(n-t);
end.
```
