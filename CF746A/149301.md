### 1. 思路1--最值思想（贪心策略）
来用数学证明方法证明
#### 求证：柠檬最多时水果总数最大
证明：

依题意，可设柠檬有a个

则 苹果2a个，梨4a个

∵苹果+柠檬+梨=a+2a+4a=7a

∴a取最大时，7a取最大

### ∴柠檬最多时水果总数最大

zuo死完毕。

所以，我们可以循环柠檬个数（0≤i≤a)，从大到小（因为输出最大值），如果苹果和梨也满足，则输出（1+2+4)×a=7a。

### 注意：a可以为0.
上代码c++(蒟蒻代码，仅供参考)
```
#include<bits/stdc++.h>//万能脑袋
using namespace std;
int a,b,c;//有a个柠檬，b个苹果，c个梨
int main()//主晗二叉树
{
	cin>>a>>b>>c;//输入
	for(int i=a;i>=0;i--)//倒叙循环
	{
		if(i*2<=b&&i*4<=c)//判断苹果和梨是否满足
		{
			cout<<i*7;//输出
			break;
		}
	}
	return 0;//结束了！结束了！结束了！
}
```

### 2.思路——最优思想（好像也是贪心）

#### step1:求出总数
#### step2:总数--,来查看是否满足情况。

上代码c++(蒟蒻代码，仅供参考，思路比较详细，不多打注释)
```cpp
#include<bits/stdc++.h>
using namespace std;
int a,b,c,k;
int main()
{
	cin>>a>>b>>c;
	k=(a+b+c)/7;
	for(int i=k;i>=0;i--)
	{
		if(i<=a&&i*2<=b&&i*4<=c)
		{
			cout<<i*7;
		}
	}
	return 0;
}
```
