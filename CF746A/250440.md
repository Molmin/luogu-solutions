# 温馨提示：此题有$84$个测试点

一道英文题，先翻译一下：

Nikolay 要做一份沙拉，他有a个柠檬，b个苹果，c个梨。沙拉中柠檬，苹果，梨的数量比必须是1:2:4 他不能把水果切开——沙拉中的水果数量一定是整数。 求沙拉中放的水果（柠檬，苹果，梨）的总数的最大值

首先让我们先设循环量为 $i$ ，我们应该以 $a$ ， $b$ ， $c$ 中哪个变量为基础循环呢？我们已知：$a:b:c=1:2:4$，为避免出现除不尽的状况，而且这道题的范围较小，我们最好以a为基础循环量。不过以 $b$ ,$c$ 为循环量也是可以的，但本文以 $a$ 循环量。

那么 $a$ ， $b$ ， $c$ 都可以用 $i$ 表示出来：

$a$ : $i$ 

$b$ : $2*i$

$c$ : $4*i$

当满足 ```2*i<=b&&4*i<=c``` 时，即我们有循环量那么多的水果，那就找到答案了。不过我们需要最优解，所以我们应该从后往前循环。

然后代码就水到渠成了：

```
#include<bits/stdc++.h>
using namespace std;
int main(){
	int a,b,c;
	int i;
	cin>>a>>b>>c;
	for(i=a;i>=0;i--){
		if(2*i<=b&&4*i<=c){
			cout<<7*i<<endl;
			return 0;
		}
	}
}
```
