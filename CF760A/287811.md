我看了一下本题的题解都是循环或者大段的if语句做的。

其实完全没必要，本题的规律非常简单。

#### 原理：

一、首先减去第一行的天数 7-m+1。

二、再看剩下的天数能排列多少行 除以 7。

三、再看模 7 看有没有余数。

#### 那么代码就有了：

```cpp
#include<iostream>
using namespace std;
int main()
{
    int n,m,a[13]={0,31,28,31,30,31,30,31,31,30,31,30,31};
    cin>>n>>m;
    int s=(a[n]-7+m-1)/7+1;//+1是因为第一行
    if((a[n]-7+m+-1)%7)
        s++;//有没有多出一行
    cout<<s;
    return 0;
}
```

当然还有一个小技巧：加上 6 就会自动向上取整。

因为除以 7 之后余数肯定是 0-6 中的一个数如果余数为 0 加上 6 除以 7 结果不会变但如果有余数 1-6 中的任何一个数加上 6 除以 7 结果都只会加 1 当然这个思路可以扩展到任意除数。

化简以后代码如下：

```cpp
#include<iostream>
using namespace std;
int main()
{
    int n,m,a[13]={0,31,28,31,30,31,30,31,31,30,31,30,31};
    cin>>n>>m;
    cout<<(a[n]+m-2)/7+1;
    return 0;
}
```
