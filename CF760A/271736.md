- 首先这一道题，如果我们想找到一种比较简便的方法来解除的话，我们需要判断它输入数字，有没有什么规律？

- 而输入数字只有两个，一个是月份，一个是星期几，因为星期几可能性是1/7，而月份它的多少最多六最少为四，所以我们可以从少的也就是月份的可能性来判断。

- 假如是四列的话，那么这个月份一定是二月，一号必须是星期一。

- 假如是5列的话，那么这个月份可能是二月份，也可能是拥有30天且这一月的一号一定是星期六及之前的。假如是31天的话，那么这一个月他的第一天一定是星期五及之前的。

- 那么剩下的话就是六列。

$OK$，代码就这样出来啦。

```
#include<bits/stdc++.h>//期末考试没考好，我TM！

int main(){
	int m,n;
	cin>>m>>n;
	if(m==2&&n==1){
		cout<<"4"<<endl;
		return 0;
	}
	if((m==1||m==3||m==5||m==7||m==8||m==10||m==12)&&n<=5){
		cout<<"5"<<endl;
		return 0;
	}
	if((m==4||m==6||m==9||m==11)&&n<=6){
		cout<<"5"<<endl;
		return 0;
	}
	if(m==2&&n<=7){
		cout<<"5"<<endl;
		return 0;
	}
	cout<<"6"<<endl;
	return 0;
}
```
