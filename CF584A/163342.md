先看数据范围：$1 \leq n \leq 100$ , $2 \leq t \leq 10$

什么？有可能要输出一个100位数？超出了long long的范围！那么肯定是按一定方法来输出。既然 $2 \leq t \leq 10$，那么符合条件的数肯定很多，这里提供一类。

$t \times 10$ 是 $t$ 的倍数，$t \times 100$ 也是 $t$ 的倍数...... $t$ 后面不管加个0都是 $t$ 的倍数。那么我们只要在 $t$ 后面加0知道补全到 $n$ 位数就行了。

题目又说：如果没有解，输出“-1”。考虑无解的情况，当 $n$ 为1，$t$ 为10时无解。当 $t$ 为个位数时，需要补上 $n-1$ 个0；当 $t$ 为10时，需要补上 $n$ 个0，那我们不妨把10改为1，和个位数一样补上 $n-1$ 个0。

代码：
```cpp
#include<bits/stdc++.h>//万能头文件
using namespace std;
int n,t;
int main()
{
	cin>>n>>t;//输入
	if(t==10&&n==1)//特判无解的情况
		cout<<-1<<endl;
	else
	{
		
		if(t==10)
			t=1;//如果是10改为1
		cout<<t;//先输出原本的数
		for(int i=1;i<n;++i)
			cout<<0;//补上n-1个0
		cout<<endl;//换行
	}
	return 0;//程序结束
 } 
```

这是我的方法。当然，你也可以用别的方法，只要满足题意就行。这应该就是最简单的方法了吧。
