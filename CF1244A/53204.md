CF的题很考验脑子~~虽说这题很水~~

题目大概翻译:有人需要写a篇报告与画b张蓝图,一支钢笔可以写c篇报告，一支铅笔可以画d张蓝图，他的笔盒里最多只能放下k支笔，请输出任意一种方案使他能完成这两项工作，如果无法完成请输出-1.

很显然，由于是spj，所以我们只需要计算出最少用笔的方案即可满足条件，如果连最少的方法也无法放下，那么就直接输出-1

最少用笔的方案即为 上取整(报告数/每支笔能写的报告数) 上取整(蓝图数/每支笔能画的蓝图数)

这里我用了一个小技巧(a/c)+(a%c>0)后面的表达式在非整除时值会为1，相当于上取整 ~~当然用double+ceil当我没说~~

```cpp
#include <bits/stdc++.h>
using namespace std;
int T;
int a,b,c,d,k;
int main ()
{
	cin>>T;
	while(T--)
	{
		cin>>a>>b>>c>>d>>k;
		if(((a/c)+(a%c>0))+((b/d)+(b%d>0))<=k)
		{
			cout<<(a/c)+(a%c>0);
			putchar(' ');
			cout<<(b/d)+(b%d>0);
			putchar('\n');
		}
		else
			puts("-1");
	}
	return 0;
}
```
