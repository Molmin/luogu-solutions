#### [题 目 传 送 门](https://www.luogu.com.cn/problem/CF1468N)

#### [题意解析]

有三个桶，五种垃圾，每个桶有固定的容量。

$1$ 垃圾放入 $1$ 号桶中，$2$ 垃圾放入 $2$ 号桶中，$3$ 垃圾放入 $3$ 号桶中，$4$ 垃圾可以放入 $1$、$3$ 号桶中，$5$ 垃圾可以放入 $2$、$3$ 号桶中。

对于给定的桶容量 $c$ 和垃圾量 $a$，请问垃圾是否可以全部放入桶中？

#### [思路解析]

首先不难想到，由于复杂的垃圾装桶系统，我们需要考虑将垃圾们放进哪个垃圾桶。首先，$1$、$2$ 和 $3$ 号垃圾一定会装进对应的垃圾桶，它们也只能装进对应的垃圾桶。现在考虑 $4$ 号垃圾，它与 $5$ 号垃圾都是可以装入垃圾桶 $3$ 的，所以，$3$ 号垃圾桶显然是比较珍贵的，$4$、$5$ 号垃圾都可以投入。然而对于垃圾来说，进入哪个垃圾桶都是一样的，那我们就先把 $4$ 垃圾放入 $1$ 号垃圾桶中，把 $5$ 垃圾放入 $2$ 号垃圾桶中，放不下的再放到 $3$ 号垃圾桶中。

#### [贴上码码]

```cpp
#include<bits/stdc++.h>
using namespace std;
long long t;
int c[10],a[10];
int main()
{
	cin>>t;
	while(t--)
	{
		bool p=1;
		cin>>c[1]>>c[2]>>c[3];//容器 
		cin>>a[1]>>a[2]>>a[3]>>a[4]>>a[5];//垃圾 
		c[1]-=a[1];
		c[2]-=a[2];
		c[3]-=a[3];
		if(c[1]<0)p=0;
		if(c[2]<0)p=0;
		if(c[3]<0)p=0;
		a[4]-=c[1];
		c[1]=0;
		a[5]-=c[2];
		c[2]=0;
		if(c[3]<a[4]+a[5])p=0;
		if(p)cout<<"YES\n";
		else cout<<"NO\n";
	}
	return 0;
}
//1放1 
//2放2 
//3放3 
//4放1 3 
//5放2 3 
```
