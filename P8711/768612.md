这是一道简单的**模拟**题。

***

### 题意简述

给出一个整数 $n$，不断用 $2$ 整除 $n$，直到结果为非正，按顺序输出整除过程中得到的所有正数。

***

### 解法

数据范围为 $10^{18}$，显然要开 `long long`，而且不能使用数组存储运算过程中的数，于是考虑算出来一个就输出一个。

显然，计算过程中不可能出现负数，只会有 $0$。而当算出一个 $0$ 之后，接下来所有数就都是 $0$，于是出现 $0$ 就可以结束程序。

***

### 代码

```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;
int n;
signed main(){
    scanf("%lld",&n);//输入
    while(n!=0){//一直循环，直到 n 非正（变为 0）
        printf("%lld ",n);//输出
        n/=2;//用 2 整除
    }
	return 0;
}
```
