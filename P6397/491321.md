显然，如果要让所有人知道消息，时间肯定是以最后一个人为准的，也就是说，要让最后一个人知道消息的时间最短。看看，像不像“最大值最小”，二分解决。

由于这次的二分涉及小数，所以可能和普通二分不一样,要开大一位精度。
```cpp
while(r-l>0.0001){//因为要输出三位小数，所以要四位
    double mid=l+(r-l)/2.0;//这里也要注意精度
    if(ef(mid)) r=mid;
    else l=mid;
}
//记得 l 附最小值，r 附最大值
```
**接下来 check 函数:**

可以发现，第一个人可以传达到的最大距离是 $d[i]+k+dis$，这里 $d[i]$ 指的是第 $i$ 个人的位置，$k$ 如题意所示，$dis$ 为二分的答案。那么如果此时第 $i+1$ 个人的位置与第 $i$ 个人的位置的距离大于 $k$，那此时的答案就不符合要求，退出 check 函数。
```cpp
bool ef(double dis){
    double left=d[1]+dis;//初始化位置
    for(int i=2;i<=n;++i){
        if(left+k<d[i]-dis) return 0;//不符合要求
        left=min(left+k,d[i]+dis);//以小的值为准，更新 left 的值
    }
    return 1;
}
```
