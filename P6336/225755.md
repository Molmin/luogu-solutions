**分析：**

本题给定每种棋子**应该有**的数量与**现在**有的棋子数量，直接输出**差值**就好。

**代码如下：**
```cpp
#include<iostream>
using namespace std;
int main()
{
	int qizi[6]={1,1,2,2,2,8};//生动形象地用拼音数组记录每种棋子的个数
	int k;
	for(int i=0;i<6;i++)//有六种棋子
	{
		cin>>k;//输入当前棋子个数 
		cout<<qizi[i]-k<<' ';//输出棋子需要个数与现有棋字个数的差值（记住输出空格）
	 } 
	return 0;//完结撒花 
 } 
```
