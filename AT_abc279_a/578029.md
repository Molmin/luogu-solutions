# 题意：

输入一个只包含 `v` 和 `w` 的字符串，每读入一个 `v` 加 $1$ 分，一个 `w` 加 $2$ 分，问最后总共多少分？

# 思路：

字符串也可以用 `char` 读入。

一个一个字符读入，是 `v` 加 $1$ 分，是 `w` 加 $2$ 分，最后输出即可。

# 代码：

截至 `2022-11-28 8:16` 是 C++ 最短解（111B）。

```cpp
#include<iostream>
int x;
char s;
int main(){
    while(std::cin>>s) x+=(s=='w'?2:1);
    std::cout<<x;
}
```
