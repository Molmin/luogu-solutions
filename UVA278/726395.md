## 题意
给定一个棋盘尺寸，给定你需要放入的棋子种类，问最多放几个。
## 思路（分类讨论）
### 后
因为后能走对角和横竖，所以一行一列一斜只能有一个后，因为斜排的可以避免，所以最多是 $\min(m,n)$。
### 车
与后差不多，因为至少一个不影响结果的斜排，所以同样为 $\min(m,n)$。
### 马
因为马走的格子颜色必然不同，所以让它们都站在同颜色的格子（国际象棋上白格子黑格子相间）上即可，也就是 $(m\times n+1)\div2$。
### 王
因为王等于是后的一格版，能攻击相邻的八个格子，所以在斜排上隔一个，在横竖上也都隔一个，即 $((n+1)\div 2)\times((m+1)\div 2)$ 。
### 代码
核心代码：
```cpp
	if ( s == 'Q' || s == 'r' )

	if ( s == 'k' )
```
