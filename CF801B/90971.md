## 题解 CF801B 【Valued Keys】

### 1、题意

这道题的题目意思很简单，就是说，找到一个字符串s3，可以用它与s1的每一位找最小值之后组合起来可以形成s2，输出s3。

列个表会明白些：

----

### 2、分析

这道题，我们找题目意思去想，在枚举s2中的每一位时就会得出以下结论:

- 如果 s1[i] 大于 s3[i] ,那么我们的答案字符串s2[i] 就必须是 s3[i]

- 如果s1[i] 等于 s3[i] ，那么s2[i]是什么都行

- 如果s1[i] 小于 s3[i] ，那么就直接输出-1，结束程序。因为最小值已经小于 s3[i] ，肯定无法满足要求。

----

### 3、找到详细思路

根据上面的分析部分，我们可以总结出 s2[i] 的求法：

**无论如何 s2[i] = s3[i] 都可以满足，但如果出现 s1[i] < s3[i] 就得在模拟中途输出-1 并结束程序。**

----

### 5、代码

由于样例是可以多种可能性的，所以大家可以放心做。

```cpp
#include<iostream>
#include<algorithm>
#include<string>
using namespace std;
string a,b,ans;
int main()
{
  cin>>a>>b;
  for(int i=0;i<a.size();i++)
  {
    if(int(a[i])<int(b[i])){cout<<-1;return 0;}
    ans+=b[i];
  }
  cout<<ans<<endl;
  return 0;
}

```
