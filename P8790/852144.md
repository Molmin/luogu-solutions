## 试题 A: 斐波那契与 $7$
这一题一开始我用的模拟斐波那契数列，不过由于一共要跑 $202202011200$ 次，我的电脑愣是跑了半小时都没出结果，具体暴力代码如下：

```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
	long long cnt = 0,a = 1,b = 1,c = 2;
	for(long long i = 4; i <= 202202011200; i++){
		a=b;b=c;c=a+b;
		c %=10;
		if(c==7)cnt++; 
	}
	printf("%lld", cnt);
}
```

当然，这肯定不是最优解，由于时间复杂度飘上天去了，正式比赛中谁会来给你这么长时间，那么我们就来找规律，首先先输出斐波那契数列前 $500$ 项找一下：

代码：


```cpp
#include <bits/stdc++.h>
using namespace std;
int a[1001];
int main(){
	a[1] = 1;
	a[2] = 1;
	for(int i = 1; i <= 500; i++){
		if(a[i] == 0){
			a[i] = a[i - 1] + a[i - 2];
		}
		a[i] %= 10;
		cout << a[i] <<" ";
	}
}

```

## 试题B 小蓝做实验
虽然这题没有 `prime.txt`，但是判定质数还是可以的，提供一种暴力求解的方法（别问我为什么不写筛法，问就是不会）……

```cpp
#include <bits/stdc++.h>
using namespace std;
#define ll long long
bool isprime(ll n){//判定质数
	if(n < 2) return 0;
	for(ll i = 2; i * i <= n; i++){
		if(n % i == 0) return 0;
	}
	return 1;
}
int main(){
	ll cnt = 0;
	ll x;//不开long long见祖宗
	for(ll i = 1; i <= 2000000; i++){//共200万个数
		cin >> x;
		if(isprime(x)){
			cnt++;
		}
	}
	cout << cnt;
}
```

答案 $342693$。
