这道题不难，只要理清关系就没问题。

邮资计算规则：
+ 重量在 1000 以内（包括），基本费 8 元。

+ 超过 1000 克的部分，每 500 克加收超重费 4 元，不足 500 克部分按 500 克计算。

+ 如果用户选择加急，多收 5 元。

输入的字符用 char 类型储存。算超过一千克的部分时，可以先用这部分除以 4，再看是否能整除。
代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int a;
char b;
int sum;
int main(){
    cin>>a>>b;
    if(a<=1000){//一千以内
        sum+=8;
    }
    else{
        sum+=8;
        a-=1000;
        sum+=a/500*4;//加上超过的部分
        if(a%500!=0){//不足按500算
            sum+=4;
        }
    }
    if(b=='y'){//判断是否加急
        sum+=5;
    }
    cout<<sum;
    return 0;
}
```
