### 题目大意

有两条球队编号为 $1,2$。

题目会给你 $n$ 条信息，每条信息包含 $m,s,w$，代表在 $m$ 分 $s$ 秒 $w$ 队进了一球。

求两支球队分别领先了几秒。

### 思路分析

一看数据范围 $m < 48$，最多才 $2880$ 秒，直接暴力大法即可。

在这题中可以将时间统一转换为离开始已经过了几秒，如 $m$ 分 $s$ 秒，即可转换为 $m \times 60 +s$ 秒。我们可以考虑用一个数组来记下每一秒有有没有进球：$0$ 没进球，$1$ 表示 $1$ 队进，$2$ 表示 $2$ 队进。

如果 $1$ 队进球，那么 $1$ 队得分加一；若 $2$ 队进球，则 $2$ 队得分加一，然后再在每一秒比较比分，得出哪一队领先来统计。最后输出解。

在输出过程中由于要前导 $0$，我们可以用 C++ 的 printf 函数解决。

```cpp
printf("%02d:%02d\n%02d:%02d",x/60,x%60,y/60,y%60);
```

然后这题就愉快的解决啦~

### 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
short t[3001];
int main()
{
	int n,m,s,w;
	char ch;//吃掉冒号 
	cin>>n;
	while(n--)
	{
		cin>>w>>m>>ch>>s;
		t[m*60+s]=w;
	}
	int cntx=0,cnty=0,x=0,y=0;
	for(int i=0;i<2880;i++)
	{
		if(t[i]==1) cntx++;
		if(t[i]==2) cnty++;
		if(cntx>cnty) x++;
		if(cntx<cnty) y++;
	}
	printf("%02d:%02d\n%02d:%02d",x/60,x%60,y/60,y%60);
	return 0;
}

```
