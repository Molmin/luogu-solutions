## 输入输出样例
输入：
```
3
1 01:10
2 21:10
2 31:30

```
输出：
```
20:00
16:30
```
# 解题思路
本题直接暴力求解并不会超时。  
一秒一秒枚举，发现进球就加分。  
如果甲队领先，把甲队领先时间增加一秒。  
如果乙队领先，把乙队领先时间增加一秒。  
不过输出时需要输出前导零，这是个很烦的东西。  
  
假设要输出的两位数为 $n$：
- 如果 $n\le 9$，即 $n$ 为一位数，则**先输出前导零**，后输出 $n$。
- 否则，直接输出即可。

而且，分和秒有进率。为更方便地处理进率，时间**一律用秒表示**。
# AC代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,x,y,x_,y_;
//x表示甲队得分，y表示乙队得分； 
//x_表示甲队领先时间，y_表示乙队领先时间。 
int b,c;//方便输入，b表示分，c表示秒 
struct in{//其实不用拆成两个独立数组也可以，只是不好理解 
	int a,sec;
} s[102];
char t;
//输入整数时遇到文件末尾或者其它字符就会停止输入，t用来吃掉冒号 
void prtime(int a){
	if(a<10) cout<<'0'<<a;//前导零 
	else cout<<a;
}
int main(){
	cin>>n;
	for(int i=0;i<n;++i) cin>>s[i].a>>b>>t>>c,s[i].sec=b*60+c;//转换 
	for(int i=0;i<2880;++i){//48分钟有2880秒
		//如果是 i<=2880 就会多一秒 
		for(int j=0;j<n;++j){
			if(s[j].sec==i){//这一秒进球了，处理得分 
				if(s[j].a==1) ++x;
				else ++y;
			}
		}
		//处理领先时间 
		if(x>y) ++x_;
		if(x<y) ++y_;
	}
	prtime(x_/60),cout<<':',prtime(x_%60),cout<<'\n';
	prtime(y_/60),cout<<':',prtime(y_%60);
	return 0; 
}
```
