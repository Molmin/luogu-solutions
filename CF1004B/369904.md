# Section 1 题目分析

## 题意

构造一个 0−1 串，使得给定的几个区间内 1 和 0个数的乘积最大。

## 算法分析

数学题

我们不妨想一下，在确定0和1的个数的情况下怎么让1和0的个数的乘积最大？

记得小学学过一个口诀：和同近积大。

也就是说，在0和1个数确定的个数的情况下只需要让1和0的个数尽量相近就可以了。

那怎么做到呢？

交替着输出呗。

# Section 2 代码

```c++
#include<bits/stdc++.h>//万能头
using namespace std;
//管理大大求过
//管理大大辛苦啦
int n;
int main()
{
    cin>>n;//输入
    for(int i=1;i<=n;i++) 
    {
        cout<<i%2;//因为%2的结果只能是0或者1，因此只要输出i%2就可以了。
    }
    return 0;
}
```
