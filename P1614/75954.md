### *什么是前缀和
举个栗子

现在有一个数组……

A[]={1,2,3,4,4,5,5,4,3,4,3,2}

如果B为A的前缀和数组，那么：

B[0]=A[0]

B[1]=A[0]+A[1]

B[2]=A[0]+A[1]+A[2]……

B[n]=A[0]+A[1]+…+A[n]

观察一下，我们发现：

**B[n]=B[n-1]+A[n]**

### *区间和与前缀和

这道题里显然我们求的是一段长度为m的区间中的和。

（也就是A[i+1]+A[i+2]+…+A[i+m]）

那么我们可以推导出：

B[i+m]-B[i]=A[i+1]+A[i+2]+…+A[i+m]

也就是说，**区间(i,i+m]的和可以用O(1)的时间求！**

那就完事辽！

# 代码

```cpp
#include<iostream>
using namespace std;
int a[3001];//前缀和数组
int main()
{
	int n,m;
	cin>>n>>m;
	for(int i=1;i<=n;i++)
	{
		int x;//因为原数组并没有什么用，所以不用存
		cin>>x;
		a[i]=a[i-1]+x;//前缀和
	}
	int ans=100000;
	for(int i=m;i<=n;i++)
		ans=min(ans,a[i]-a[i-m]);//用更小的更新ans的值
	cout<<ans;
	return 0;
}
```
