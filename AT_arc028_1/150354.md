                      **又是一道愚人节的题目**

## 题目大意：

#### 有一堆石子，两个人轮流取，一个人取a个，一个人取b个，谁先取完这一堆石子。

## 解题思路：

### 看了看题目，发现可以通过枚举来写出这一题的代码，所以我就懒得想公式直接用题意枚举就可以了。

## 程序如下：
```cpp
#include<iostream>
using namespace std;
int n,a,b;
main(){
    cin>>n>>a>>b; //输入
    while(1)  //作死循环来求数
    {
    	n-=a; //一个人取a个
    	if(n<=0)  //如果取完，就输出。
    	{
    		cout<<"Ant"<<endl; //记得换行
    		return 0; //结束
    	}
    	n-=b;  //另一个人取b个
    	if(n<=0) //如果取完，就输出。
        {
        	cout<<"Bug"<<endl; //记得换行
        	return 0; //结束
        }
    }
}
```
