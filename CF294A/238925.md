这么水的模拟题呀！可居然是一道紫题！不可思议。

思路：按照题目一行一行模拟，把打到的左侧的鸟都移动到左边的电线上去，右侧的鸟都移动到右边的电线上去即可，很水的。

模拟样例：

$5$

$10\ 10\ 10\ 10\ 10$

$5$

$2\ 5$

$3\ 13$

$2\ 12$

$1\ 13$

$4\ 6$

1、$14\ 0\ 15\ 10\ 10$

2、$14\ 12\ 0\ 12\ 10$

3、$25\ 0\ 0\ 12\ 10$

4、$0\ 12\ 0\ 12\ 10$

5、$0\ 12\ 5\ 0\ 16$

$Code:$

```cpp
#include <bits/stdc++.h>
using namespace std;
int n,m,x,y,a[105];
int main() {
	scanf("%d",&n);
	for(int i=1;i<=n;i++)scanf("%d",&a[i]);
	scanf("%d",&m);
	for(int i=1;i<=m;i++){
		scanf("%d%d",&x,&y);
		a[x-1]+=y-1;	//左侧的鸟飞到了左边的电线上
		a[x+1]+=a[x]-y; //右侧的鸟飞到了右边的电线上
		a[x]=0;	//这根电线上没有鸟了
	} 
	for(int i=1;i<=n;i++)printf("%d\n",a[i]);
	return 0;
}
```
