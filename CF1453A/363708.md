# 题解
题目要求：输出一个整数，为了防止所有碰撞，应该取消的最小列车数。

这道题其实是查重，即给你两个数组，找出它们有几组数据是相同的。我这里提供一个简单粗暴的方法：**二重循环枚举查重，利用计数器数组，数据小，爆不掉。**

具体就是用一个for循环看第一个数组，再用一个看第二个数组，发现一样的计数器加一。


 一样的，注意格式，**换行**！ ！

直接上有注释代码了（简洁版）


```CPP
#include<bits/stdc++.h>
using namespace std;
int t,n,m,x[109],y[109],cnt[109];
int main() {
cin>>t;  //t代表数据组数 
    for(int z=1;z<=t;z++) { //其实可以写为：while(z--)利用括号里非零 
        cin>>n>>m;   //n,m重复输入 
        for(int i=1;i<=n;i++) cin>>x[i];
        for(int j=1;j<=m;j++) cin>>y[j];  //输入环节不多说 
        for(int i=1;i<=n;i++) 
            for (int j=1; j<=m; j++)  
                if (x[i]==y[j])   //双重循环枚举，发现相同，计数器数组增加 
                   cnt[z]++;
        cout<<cnt[z]<<endl; //输出计数器数组，别忘了换行 
    }
    return 0;
}
```
