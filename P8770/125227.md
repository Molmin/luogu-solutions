本题是提交答案题，有两问。

第一题：

注意观察图片和题目，不要漏了裁掉**边缘的$4$次**，接下来就是小学数学题：先裁行再裁列（其实是先裁缝隙少的，后面看到公式就懂了），很容易发现剩下裁的刀数为 **横向缝隙数$+$纵向缝隙数$\times$横向份数**

综上，答案为：$4+19+21\times20=443$

第二题：

这一小问其实是博弈论题，经过模拟不难发现（自己可以画一下）只要把两行中的其中一行**全部**占满，而另一行剩下四个空时谁是当前谁是**后手**谁必胜。

综上，第一、二、三种情况小蓝必胜，而第四种情况小乔必胜，答案为: $VVVL$

AC代码如下：
```cpp
#include<iostream>
using namespace std;
int main() {
    string ans [] = {
        "443",
        "VVVL",
    };
    char T;
    cin >> T;
    cout << ans[T - 'A'] << endl;
    return 0;
}

```
