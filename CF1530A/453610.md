题目传送门： [CF1530A](https://www.luogu.com.cn/problem/CF1530A)

一道自认为思维难度较高的简单题。

### 题意：

给你 $t$ 个数，求出每个数最少可由多少个只由 $0$ 或 $1$ 组成的数组成。

### 思路：

此题必定有解，易证。

每个个位数都可以由多个 $1$ 组成，这是必然的。而每个个位数后面加上几个 $0$ 就会成为新的位。所以我们只需要枚举统计这个数每个位上的每个数由几个 $1$ 组成（实际上就是这个位上的那个数），然后取每个位上的数的最大值。

#### 再次简化：就是求组成这个数的多个个位数中最大的那一个。

### 怎样实现？
我们不妨定义一个数，初始值为0，表示答案。然后一点一点往后推，一边循环一遍判断，不断与这个位上的数取 $max$ 的值就行了。

思路很明确了，直接模拟即可。下面上代码：

### Code：

```cpp
#include<iostream>
#include<algorithm>
using namespace std;
int t,n,maxans;
int main(){
	cin>>t;
	for(int i=1;i<=t;i++){
		cin>>n;
		maxans=0;//答案
		while(n){
			maxans=max(maxans,n%10);//取这一位，然后取max
			n/=10;//往后推一位
		}
		cout<<maxans<<endl;//输出
	}
	return 0;
}
```
