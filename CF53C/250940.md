既然只需要找到一种解法，那么我们找一个通解就可以了。

我们发现（自动省略其中的推理过程），从 $1$ 号开始，每次跳到对面的端点，那么每一步的距离正好是 $n-1,~n-2,~...,~1$ ，满足要求。此时跳土堆的顺序是 $1,~n,~2,~n-1,~...$ 以此类推。

因此代码如下：
```cpp
#include<cstdio>
int n; 
int main()
{
	scanf("%d",&n);
	for(int i=1;i<=n/2;i++) printf("%d %d ",i,n+1-i);//通式，请同学们自己推
	if(n%2) printf("%d",n/2+1);//特判，奇数的话中间的那个数搜不到
	return 0;
}
```
