这道题还是比较简单的。

要使字符串尽量长，那么删的肯定不是a。题目说a的个数要是字符串的一半以上，就先算出a有这么多时，字符串最多有多长。但是不能加字符，就和原字符串的长度比一下。

上代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a;
char x;
int main()
{
	while(cin>>x)//输入最后别忘c^z。
	{
		n++;//统计原字符串的长度。
		if(x=='a')
			a++;//统计a的个数。
	}
	cout<<min(a*2-1,n)<<endl;//输出答案。
	return 0;
}
```
