# 一道水题

首先应使用数组和$for$语句输入总共可以住的人数和现在的人数。

输入后，用总共可以住的人数减去现在的人数。因为人数不可能是负数，所以我们不用考虑负数的情况。

若其值大于等于2，就说明满足条件，可以空出至少2个位置，计数器 $s$ 加一。

循环完毕后，输出空出至少2个位置的房间数 $s$ ，就ok了。最后应养成写`return 0`的好习惯。

温馨提示：此题有27个测试点

双手奉上代码：


```
#include<bits/stdc++.h>
using namespace std;
int main(){
	int n;
	cin>>n;
	int a[n+1],b[n+1],s=0;
	for(int i=1;i<=n;i++){
		cin>>a[i]>>b[i];
		if(b[i]-a[i]>=2)s++;
	}
	cout<<s<<endl;
}
```
