### 二、代码部分（附注释版）
```python
a = int(input())#输入询问次数
#以下为每一次的询问过程，可以使用函数，我这里使用的是循环
for k in range(0,a):#进行a次询问
    n = int(input())#获得问题
    s = ""#声明并初始化一个空的字符串
    for i in range(1,n+1):
        s += str(i)#从1开始将每一个数字加入s中
    for i in range(0,9):
        print(s.count(str(i)),end = " ")#数字0~8采用循环count方法输出带有空格的答案
    print(s.count(str("9")))#单独使用count方法输出带有换行符的答案
```
**（所有py代码请注意缩进）**

### 三、重点方法 count 说明
用于统计字符串里某个字符出现的次数。可选参数为在字符串搜索的开始与结束位置。


使用模板：


```str.count(sub, start= 0,end=len(string))```




1. sub -- 搜索的子字符串。
2. start -- 字符串开始搜索的位置。默认为第一个字符,第一个字符索引值为0。
3. end -- 字符串中结束搜索的位置。字符中第一个字符的索引为 0。默认为字符串的最后一个位置。

（其中start 和 end是可选参数，就像我并没有使用这两个参数。）
