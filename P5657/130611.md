考场上找规律找了挺久的。\
第一眼就觉得是**格雷的疑惑**。\
解法很简单：
我们看到二进制，首先想到位运算。那么&、|和^三种运算，我们应该试哪一个呢？
我们这样想，如果我们有两个数a、b，我们a&b得到结果c，或者a|b得到另一个结果c，我们都是无法通过c和b推出a或者通过c和a推出b的。
但是如果我们用^运算就可以。
考场上，只要转换成十进制推了规律的同学应该都能很快发现，两个分一组，每组间的数都是+1或者-1的，相邻两个数^一下正好对应这个关系。
这种关系叫**成对变换**。
我们利用这个关系，做这样一个运算：
```cpp
int f[11]={0,1,3,2,6,7,5,4};//手算一组格雷码
for(int i=0;i<8;i++){
	cout<<(i^f[i])<<' ';	
}
```
可以得到0 0 1 1 2 2 3 3这样一个结果。
于是我们得到了$f[i] \oplus i=i/2$的关系式。
移项得到$f[i]=(i/2)\oplus i$，我们求第k个，就是
$ans=(k/2)\oplus k$。
然后我们再计算一下位数，转换成二进制补零后输出即可。
