卡特兰数递推式： $f[n]= f[0]*f[n-1]+f[1]*f[n-2] + ... + f[n-1]*f[0] (n>=2)$

当然这种算法的时间复杂度略高，比不上O(1)，但还是可以0ms过的

最重要的一点：

##要开long long！

###千万别忘了！

具体见代码



```cpp
#include<cstdio>
long long n,f[45];
int main()
{
    scanf("%d",&n);
    f[0]=1;
    f[1]=1;//卡特兰数第0、1项 
    for(int i=2;i<=n;i++)
    {
        int x=0,y=i-1;
        while(x<i && y>=0)
        {
            f[i]+=f[x]*f[y];
            x++;
            y--;//递推式的表现 
        }
    }
    printf("%d",f[n]);//输出，结束 
    return 0;
}
```
