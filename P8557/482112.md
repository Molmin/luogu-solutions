## Solution

**首先扔出结论**，答案是 $(2^k-1)^n$。

## Proof

熔炉选金属不好做，变成金属选熔炉。

每个金属都选至少一个熔炉，也就是选 $\{1,2,3,\dots,k\}$ 的非空子集，$2^k-1$ 种，没啥好解释的。

对于每个金属都是这样，乘法原理 $(2^k-1)^n$ 就好了。

## Code

快速幂，$\mathcal O(\log (n+k))$ 的。

```cpp
int solve() {
  int n=getInt(), k=getInt();
  putInt(pw(pw(2, k, 998244353) - 1, n, 998244353), '\n');
  return 0;
}
```
