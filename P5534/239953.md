等差数列：
首先，等差数列是常见数列的一种，如果一个数列从第二项起，每一项与它的前一项的差等于同一个常数，这个数列就叫做等差数列，而这个常数叫做等差数列的公差，公差常用字母d表示。
通项公式为：an=a1+(n-1)d。 
前n项和公式为：Sn=a1n+[n*(n-1)d]/2（首项加末项乘以项数除以二），另外需要注意数组范围，之前用的int直接一片红。
题解代码
```
ll a,b,d,sn,n,endd;
int main()
{
    cin>>a>>b>>n;
    d=b-a;
    endd=a+(n-1)*d;
    sn=(a+endd)*n/2;
    cout<<sn<<endl;
    return 0;
}
