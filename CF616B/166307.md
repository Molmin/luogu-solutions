### 样例分析

这道题呢，大概通读一遍题目，就来看样例：
> 3 4   
4 1 3 5   
2 2 2 2   
5 4 5 1   

既然样例解释没有给出，那就大概看一下。

按题目要求说的话艾玛一定会选择第 $1$ 或第 $3$ 条街（因为输入中最大的数字为 $5$），但如果艾玛选择第 $1$ 或第 $3$ 条街，那么我们 ~~吝啬~~ 节俭的杰克就会选择费用为 $1$ 的那个巷子。如果艾玛选择第 $2$ 条街的话，杰克就只能选择费用为 $2$ 的那个巷子了。所以艾玛为了吃贵点，就会选择第 $2$ 条街，答案也就为 $2$ 了。

### 解法分析

这道题怎么说呢，也就模拟吧。

艾玛为了多花钱（~~大家一定不要学习她~~），一定会看每条街里最便宜的巷子，并选择所有最便宜的巷子中最贵的那个（如果选最便宜的那个，即使那条街里有所有街里最贵的巷子，但杰克还是会选最便宜那个）。所以输入 $a_i$ 时，顺带求出 $a_i$ 数组里最便宜的（即最小的数字） $minn$ 后，再用一个变量 $maxn$ 求出 $minn$ 中最大的，答案就是 $maxn$。

### 代码实现

```
#include<bits/stdc++.h>
using namespace std;
int n,m,maxn=0;
int main(){
	cin>>n>>m;
	for(int i=1;i<=n;i++){
		int minn=INT_MAX;//一条街中最便宜的费用
		for(int j=1;j<=m;j++){
			int x;
			cin>>x;
			minn=min(minn,x);
		}
		maxn=max(maxn,minn);//所有街中最小值最大的那个
	}
	cout<<maxn<<endl;
	return 0;
}
```
