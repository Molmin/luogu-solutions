这道题既然说每个长度为 $a$ 的子串都只由 $b$ 种字母构成，不妨我们就直接让整个字符串只有 $b$ 种字母就可以，那我们就要让每种字母都出现，因为 $a \ge b$，所以肯定能保证每种字母都出现一次，当然子串匹配得话为了方便我们要先从开头开始匹配，这样的话我们就从开头的前 $b$ 个位置中依次填上不同的字符，当这个子串匹配成功的话，开头肯定得往右移一格，这样的话之前的第 $0$ 个位置的字符（在这里我们把最左边的字符设为第 $0$ 个字符）就不再这个子串上，所以我们要在第 $b$ 个位置上补上和第 $0$ 个字符一样的字符，以此类推，这样我们就可以得到一个可行的规律，从开头开始，依次循环地填入字母，循环节长度为 $b$，至于字符是什么可以自己定（**只要是小写字母就行**），但是为了方便，直接用 $a,b,c……$ 顺次填上就可以了。

**注意多组数据！！！**

时间复杂度：$O(\sum{n})$

# code:
```cpp
#include<cstdio>
using namespace std;
int main(){
    int t;
    scanf("%d",&t);
    while(t--){
        int n,a,b,i;
        scanf("%d%d%d",&n,&a,&b);
        for(i=0;i<n;i++){
            printf("%c",'a'+i%b);
        }
        printf("\n");
    }
    return 0;
}
```
