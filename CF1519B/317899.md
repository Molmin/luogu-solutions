为了简化思路，我们假设目前我们站在 $(x,y)$ 上。

然后我们为了到达 $(x+1,y+1)$ 会有 $2$ 种走法：

1. $(x,y)\to(x+1,y)\to(x+1,y+1)$
2. $(x,y)\to(x,y+1)\to(x+1,y+1)$

接下来，让我们分析这两种走法的代价：

1. 
- $(x,y)\to(x+1,y)$ 花费 $y$。
- $(x+1,y)\to(x+1,y+1)$ 花费 $x+1$。
- 共花费 $x+y+1$.

2. 
- $(x,y)\to(x,y+1)$ 花费 $x$。
- $(x,y+1)\to(x+1,y+1)$ 花费 $y+1$。
- 共花费 $x+y+1$.

所以我们得知，从 $(x,y)$ 到 $(x+1,y+1)$ 所需要的钱数是一定的。

接下来，我们再算算从 $(x,y)$ 到 $(x+2,y+1)$ 的钱数。有几种走法：

1. $(x,y)\to(x+1,y)\to(x+1,y+1)\to(x+2,y+1)$
2. $(x,y)\to(x,y+1)\to(x+1,y+1)\to(x+2,y+1)$
3. $(x,y)\to(x+1,y)\to(x+2,y)\to(x+2,y+1)$

这三种方法分别花费：

1. $y+(x+1)+(y+1)=2y+x+2$
2. $x+(y+1)+(y+1)=2y+x+2$
3. $y+y+(x+2)=2y+x+2$

三种方法消费的值是相同的！

那么我们就可以猜出结论，从 $(x,y)$ 到任何一个点所花的价钱是一样的。

那么我们就找一下规律：发现从 $(1,1)$ 到 $(n,m)$ 所花的钱正好是 $nm-1$。

代码都会写了吧。

```cpp
/*
火车头已过滤
*/
int t;
signed main(){
	cin>>t;
	while(t--){
		int n,m,k;
		cin>>n>>m>>k;
		if(k==n*m-1)
			cout<<"YES"<<endl;
		else
			cout<<"NO"<<endl;
	}
	return 0;
}
```
