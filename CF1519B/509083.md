## 思路分析

结论题，先上代码：
```cpp
#include<iostream>
using namespace std;
int main()
{
	int n, x, y, k;
	cin >> n;
	for(int i = 1; i <= n; i ++)
	{
		cin >> x >> y >> k;
		if(x * y - 1 == k) printf("YES\n");
		else printf("NO\n");
	}
	return 0;
}
```

从代码中可以看出结论：

- 不论行走方案如何，走到终点所获得的分数都相同，且均为 $n\times m-1$。

**接下来是证明**；

若我们想从 $(x,y)$ 行走到 $(x+1,y+1)$，则有以下两种行走方案：

- $(x,y)\to(x+1,y)\to(x+1,y+1)$
- $(x,y)\to(x,y+1)\to(x+1,y+1)$

两种方案的分数分别为：

- $y+(x+1)$
- $x+(y+1)$

可以看出：

- 不管如何行走，从 $(x,y)$ 行走到 $(x+1,y+1)$ 所获得的分数均相同。

因此，在此基础上做一些延伸，便可以得出：

- 不管如何行走，从 $(x,y)$ 行走到 $(x_1,y_1)$ 所获得的分数均相同。
