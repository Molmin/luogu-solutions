**题意简述**

Rita 从 $4$ 岁起开始收集每年生日的蜡烛，而她的弟弟 Theo 从 $3$ 岁开始。

有天弟弟把自己盒子里的若干根蜡烛放到姐姐盒子里了，给出年龄差 $D$，姐姐现有蜡烛 $R$ 和弟弟现有蜡烛 $T$，求弟弟放了几根蜡烛。

有多组数据。

---

**样例解释**

姐姐比弟弟大 $2$ 岁，姐姐盒子里有 $26$ 根蜡烛，弟弟盒子里有 $8$ 根。

求出姐姐原有 $22$ 根蜡烛（$4+5+6+7$），弟弟原有 $12$ 根蜡烛（$3+4+5$），即弟弟把自己盒子里的 $4$ 根蜡烛放在姐姐盒子里了，所以输出 $4$。

---

**做法1：枚举**

发现网上有的做法特别麻烦，提供一个简单一点的做法。

一个一个试弟弟现在多少岁。

如果他有 $x$ 岁，则姐姐有 $x+d$ 岁。

**弟弟应有蜡烛**：

$$S=3+4+...+x$$

$$=\dfrac{\left(3+x\right) \times \left[ \left(x-3\right)/1+1 \right]}{2}$$

$$=\dfrac{1}{2}(x+3)(x-2) $$

**姐姐应有蜡烛**：

$$S=4+5+...+(x+d-1)+(x+d)$$

$$=\dfrac{ \left[ 3+(x+d) \right] \times \left[(x+d)-3+1 \right]}{2}$$

$$=\dfrac{1}{2}(x+d+4)(x+d-3)$$

---

因为弟弟只是把自己盒子里的蜡烛**移**到姐姐盒子里的，总量不变，所以直接求总和判断是否一致就行了。

理论来说可以用二分查找，但是懒得写了。

理论上也可以用一元二次方程，但是还不会解/kk，留给好朋友做吧。

#include <bits/stdc++.h>
using namespace std;

int main() {
	int d,r,t,sum;
	while (cin >>d>>r>>t) {
		for (int x=3;;x++) { //弟弟从3岁起收集
			sum=(x+3)*(x-2)/2+(x+d+4)*(x+d-3)/2;
			if (sum==r+t) { //如果当前计算蜡烛总量和 现姐+弟的蜡烛总量 一样
				printf("%d\n",(x+3)*(x-2)/2-t);
				break;
			}
		}
	}
	return 0;
}
