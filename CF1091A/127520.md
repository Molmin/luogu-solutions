### 瞟一眼数据100，很养眼，简单暴力就能AC

本蒟蒻用的纯暴力，楼上的公式我都看不懂

只要不断地让三种装饰数量++，加到不够就停下就好了

不说了，贴上AC代码（感谢数据小）

```
#include <bits/stdc++.h>
using namespace std;
int a,b,c,i,j,k,l;//a,b,c分别为黄、蓝、红的饰品上限，j,k,l为黄蓝红的实际使用量
int main(){
cin>>a>>b>>c;//输入就不多说
j=0;
k=1;
l=2;//给实际使用量赋初值（题目说了至少会有6个饰品）
do{
	j++;
	k++;
	l++;//循环把三种饰品数量都加一
}while(j<a&&k<b&&l<c);//停止循环的条件，写等于答案会多3
cout<<j+k+l;//输出
return 0;
}
```
