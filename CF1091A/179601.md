#### 详情请见注释
```
#include<bits/stdc++.h>
using namespace std;
int a,b,c;//a=y,b=b,c=r
int main(){
	cin>>a>>b>>c;//输入
	cout<<3*min(b,min(a+1,c-1));
	/*
		由题我们可得：
		黄+1=蓝
		红-1=蓝
		所以：总和=黄+红+蓝= 蓝-1+蓝+1+蓝=3*蓝
		然后题目又要求我们求最大值
		所以：我们需要寻找最大且能满足3个蓝的式子
		所以：每个式子最大值中最小的即为所求 
	*/ 
	return 0;
}
```
