# 温馨提示：此题有42个测试点。

以下为代码核心：

     for(int i=a;i>=1;i--)
	    if(i+1<=b&&i+2<=c){
			cout<<3*i+3;
			return 0;
		}


楼下的代码虽然短，但是复杂度较高，这个题解有更好的方法。我们可以直接从$a$到$1$循环，且设循环的量为$i$。那么$a$的值为$i$，b的值为$i+1$，$c$的值为$i+2$

//$a$为黄装饰，$b$为蓝装饰，$c$为红装饰

若$i+1$小于等于$b$且$i+2$小于等于$c$（必须要有那么多装饰拿得了）循环就可以结束了，输出$3*i+3$
