总体来说，这题还是一道不错的入门字符串模拟题。但是翻译貌似有点问题，完全没把拆分子串的部分表现出来，导致我过不了。

这题其实就是输入一个字符串，让找出它的拆分完再拼接后的字符串。

举个例子，例如 `abbc`两个字符的字串 `ab`, `bb` 和 `bc`,把这些子串拼在一起，成了`abbbbc`,现在是把这个步骤倒过来，让你求出原来的字符串。

操作步骤：先直接输出第一位，然后两位一输出，但是要先对字符串的长度进行一个小处理，详见代码。

代码贴上：

```cpp
#include <cstring>
#include <iostream>
using namespace std;
int main()
{
	char str[101];
	int i,j,k,t,len;
	cin>>t;
	for(i=1;i<=t;i++)//循环读入
	{
		cin>>str;
		len=strlen(str);//字符串取长度
		if(len%2==1)//如果字符串长度为奇数，减一
			len--;
		cout<<str[0];
		for(j=1;j<len;j=j+2)   cout<<str[j];
        //从第二位起，两位一输出
		cout<<endl;
	}
    return 0;
}
```
