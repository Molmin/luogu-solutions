~~正解是换一个店~~

首先，要知道一点：对于买的商品没有限制，而要花最少的钱，所以选最便宜的

另外，因为每天只能买一个，而每一天就要涨一块钱，如果买的个数已经定下来了，那么商品总共涨的价钱和也是一定的

根据以上两点，就可以写出程序啦！

另：我一开始看到这道题的时候还以为是个堆的题，后来才发现不需要，但是已经写好的$pq$就懒得改了

写$pq$的时候还是有一些小技巧的，程序中具体讲

```cpp
#include<iostream>
#include<queue>
using namespace std;
priority_queue<long long>Q;
long long n,a,k;    //long long
int main()
{
    cin>>n>>k;
    for(int i=1;i<=n;i++)
    {
        cin>>a;
        Q.push(-a);    //压入的时候的小技巧：它要的是最小，然而pq提供的是最大值，所以我们把每一个数取一个负，这样提供的就是最小值的相反数了，最后输出的时候要再倒过来
    }
    n=0;     //n已经没用了，就让他来当我们的求和了，所以要开long long
    for(int i=0;i<k;i++)
    {
        n-=i;   //一定是-不是+！！！！如果要用前面pq的小技巧，一定要时时刻刻想着正负！
        n+=Q.top();
        Q.pop();
    }
    cout<<-n<<endl;    //输出相反数
    return 0;
}
```
