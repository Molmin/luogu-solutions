题意：高桥喜欢在拍卖网站购物。现在，这个拍卖网站上展出了N个商品，第i个商品的价格是Ai日元。这个拍卖网站一天只能买一个商品，每过一天哪个商品都会增加1日元。当然，同样的商品只能买到一次。
高桥君想买K个商品。请高桥君购买K个商品所需的金额的最小值。


------------

思路：先读入后将所有商品价格排序，最后循环k次加上这些物品的价格和增加的价格。


------------

代码：
```cpp
#include<iostream>
#include<algorithm>//sort的头文件 
using namespace std;
int n,k,a[100005];
long long s;//防炸 
int main()
{
int n,k;
cin>>n>>k;
for(int i=1;i<=n;i++)cin>>a[i];
sort(a+1,a+n+1);//快排（~~懒懒~~） 默认从小到大 
for(int i=1;i<=k;i++)s+=a[i]+i-1;
//加上商品原价和涨的价格（减一是因为实际上第一天不涨价但程序加了）
cout<<s<<endl;//在AT必须换行（~~保平安~~） 
return 0;
} 
```
