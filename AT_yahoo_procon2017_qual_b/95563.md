## 题意翻译：
 
 一个人（懒得说是谁），要买k的东东（不知道是啥），但他的时间太少，每天只能买一件，这时，商家不乐意了，于是从 **第二天** 开始每个商品都会上涨1元（~~猥琐~~），问最少花多少元才可以买完。
 
## 思路：
#### 1.定义
```
int n,k;
int a[100002];
long long h;//不用会爆炸
```
#### 2.输入
```
cin>>n>>k;
for(int i=1;i<=n;i++)
    scanf("%d",&a[i]);//用scanf保险
```
#### 3.排序（从小到大）

快排:
```
sort(a+1,a+1+n);//懒人专用（默认从小到大）
```
or

选择排序（选排）
```
for(int i=1;i<n;i++)//枚举1~n-1（最后一个不需要和后面的数比较）
		for(int j=i+1;j<=n;j++)//枚举第i位后面的数
			if(a[i]>a[j])swap(a[i],a[j]);//只要前面的大于后面的，就交换
```
......（以下省略多种高级方法）

#### 4.累加
```
for(int i=1;i<=k;i++)
		h+=a[i]+i-1;//第i个数应该加i-1，在累加
```
#### 5.输出
```
cout<<h;//孤单的我，QAQ
```

好了，做完了。这篇题解是很难Ctrl+C的。
