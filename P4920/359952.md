# program1

等价于 $a\times b\mod c$ 直接 int128 即可。

# program2

就是每次对 $a,b,c$ 做线性变换，可以矩阵快速幂。

# program3

需要求 $\sum i^k,k\in{0,1,2,3,4}$，这个先把公式写出来然后直接打开计算器！！！

# program4

以每个 $0$ 为源跑最短路，直接加和即可。

# program5

这个真的完全是传统题了，考虑枚举矩阵上面那个行，接下来设每个点往下长度为 $h_i$。

问题转化为给定一个数列求任意两点之间最小值的和，单调栈计算出以一个点为最小值的左右端点即可。

# program6

$F_0=0,F_i=(aF_{i-1}^2+b)\mod c$ 直接求通项就算能求也不行，因为前面溢出了。

那么只能判环了，使用 Brent 判环即可。

# program7

数独。直接搜不太能过的去，每次枚举的改为 可填的数最少的格子就能过了。。

# program8

求满足特定限制的七元组个数。

仔细思考发现答案应该是关于 $n$ 的 $7$ 次多项式，直接插值。。。

# program9

这题给定了答案的 MD5.

前三个问题会的都会。实在不会枚举答案试一试。

第四个问题直接枚举三个符号带进去测。

第 $5$ 到 $9$ 个问题用 program2 的词典带进去即可。

第 $10$ 个问题可以把这句名言在 program10 里找到。

答案为 $\textsf{we\ hold these truths to be selfevident}$。

# program10

这个问题在不停的函数调用，但是有一个简单的方法可以快速优化：把 $A$ 到 $Z$ 的函数简化为一次加法。

时间复杂度是 $\mathcal{O}(\sum 3^{|word|})$

然后因为单词不可能太长，就飞快的跑出来了？？
