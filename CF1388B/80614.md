因为这道题要去去掉后面 $k$ 位后数最大，所以我们首先得保证这个数的位数最多，所以每一位都得是 $8$ 或 $9$ $(8$ 是 $1000$，$9$ 是 $1001$，两个数化成二进制后都是 $4$ 位$)$。因为只有后面 $n$ 位被覆盖，前面 $3n$ 位未被覆盖，所以前面的 $3n$ 位必须尽可能的大。后面 $n$ 位必须尽可能的小。我们知道，$8$ 和 $9$ 化成二进制后唯一的差别是个位不同，所以我们就要数这 $n$ 位覆盖掉了多少个每个数化成二进制后的个位。我们知道，所有每个数化成二进制后的个位都在从右往左数第 $a$ 位上 $(a \equiv 1(\mod 4))$。那么我们只需要求出有多少个这样的 $a$ 满足 $a\le k$ 就行了。

时间复杂度 $O(\sum\limits_{i=1}^tn)$

# code:
```cpp
#include<cstdio>
using namespace std;
int main(){
	int t;
	scanf("%d",&t);
	while(t--){
		int n,ans,i;
		scanf("%d",&n);
		ans=(n+3)/4;
		for(i=1;i<=n-ans;i++){
			printf("9");
		}
		for(i=1;i<=ans;i++){
			printf("8");
		}
		printf("\n");
	}
	return 0;
}
```
