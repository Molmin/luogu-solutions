# 思路

首先，本题的输出只有 $4$，$7$，$-1$ 三种。

由于题目要求输出字典序最小，所以 $4$ 和 $7$ 的各种组合的字典序比 $4$ 和 $7$ 的字典序大。

统计一下输入字符串内 $4$ 和 $7$ 的个数。

- 若没有 $4$ 或 $7$，输出 $-1$。

- 若 $4$ 的个数**大于等于** $7$ 的个数，输出 $4$。注意等于的情况也要输出 $4$，因为 $4$ 的字典序比 $7$ 小。

- 若 $4$ 的个数**小于** $7$ 的个数，输出 $7$。

# 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
string s;
int x,y;
int main()
{
	cin>>s;
	for(int i=0;i<s.size();i++)
		x+=(s[i]=='4'),y+=(s[i]=='7');
	if(!x&&!y) cout<<-1;
	else if(x>=y) cout<<4;
	else cout<<7;
	return 0;	
} 
```
