## 思路 & 解法

- 如果要求人数最多：好办，把 $m$ 个人捆成一团塞在人数最多的椅子上。输出 $\max (a_i) + m$。

- 如果要求人数最少：**理论的**最好情况是每个长椅上尽可能平均坐： $ave = \frac{S + m}{n}$ （上取整）个人（这里的 $S = \sum^{n}_{i=1} a_i$）。但是，~~理想总是很丰满，现实总是很骨感~~，我们发现有一种情况会很恶心：

```
5 2
1 1 10 1 1
```
这样的话理论最好是每个椅子坐 3~4 个人，但是由于有 10 个人坐在一把椅子上，我们又没法把他们赶走，所以无论如何，最多的那把椅子上至少都要坐 10 个人。

所以我们用一个 $maxn$ 记录最大的 $a_i$，最后再和我们求出的平均数 $ave$ 取一个 $\max$ 即可。

然后我们就快快乐乐地做完了。

---

## 代码


放上极简代码：

```cpp
#include <cstdio>
#include <algorithm>
int main()
{
	int n,m,maxn=0,sum=0;
	scanf("%d%d",&n,&m);
	for(int i=1;i<=n;i++)
	{
		int x;
		scanf("%d",&x);
		maxn=std::max(maxn,x);
		sum+=x;
	}
	printf("%d %d",std::max(maxn,(sum+m+n-1)/n),maxn+m);
	return 0;
}
```
