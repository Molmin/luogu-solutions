###题解 P1887 【乘积最大3】

//代码要短QWQ

设一个数为x，另一个数为a-x。

则x(a-x)=ax-x²=-(x²-ax+a²/4)+a²/4=a²/4-(x-a/2)²

所以x与(a-x)之积越大则要求x-a/2越小。

即x越接近a/2。

因此两个数越接近,乘积越大。

将以上结论推广。

所以为使若干个数之积最大，这些数应该尽可能相近。

及各个数字相差不超过1。

所以直接输出b-a%b个a/b，a%b个a/b+1即可。

```cpp
#include <cstdio>
int a,b,i;
int main()
{
    scanf("%d%d",&a,&b);
    for (i=1;i<=b-a%b;++i) printf("%d ",a/b);
    //为使乘积最大，各个数字之差不超过1
    //输出b-a%b个a/b
    for (;i<=b;++i) printf("%d ",a/b+1);
    //把余数尽可能平均分配给各个数，输出a%b个a/b+1
}
```
