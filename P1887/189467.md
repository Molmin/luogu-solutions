

------------
#### 这个题算是找规律的吧。


------------

#### 规律：凑成N的每一项都是当前N与M的商。

#### 由于凑成N的每项都是当前N与M的商，这样就能在剩下的M步中用尽量大

#### 的数给它走完，这样也就保证了最终的乘积是尽量大的。

------------

#### 下面附代码：


------------
```c
#include <stdio.h>

int main(void)
{
	int n,m;
	
	scanf("%d %d", &n,&m);
	while(m)
	{
		printf("%d ", n/m);
		n -= n/m;
		m--;
	}
	
	return 0;
}
```
