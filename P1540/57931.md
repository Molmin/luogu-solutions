方法分析：

1.读入m与n的值；

2.利用for循环语句，一边读入一边做；

3.用一个bool数组当作内存，桶排思想，O(n)时间复杂度；

4.结束；

AC代码，仅供参考，谢绝ctrl+c，ctrl+v：

```cpp
#include<iostream>
using namespace std;
int m,n,k,t,c;
bool b[10005];                   //开大一点，以免爆掉
int a[10005];                    //a数组存放查字典的先后顺序，b数组是判断内存里有没有这个数
int main()
{
    ios::sync_with_stdio;       //小小的优化，让cin与scanf一样快
    cin>>m>>n;                  //读入m与n的值
    for(int i=0;i<n;i++)
    {
        cin>>k;                 //读入当前单词
        if(b[k]==0)             //如果内存中没有，则——
        {
            a[t]=k;             //用于记录是第几个，方便后面删除最早的一个
            b[k]=1;             //把第k桶变成true
            t++;                //累加查字典次数
            if(t%m>=0&&t>m)     //判断，如果当前内存中已存入的单词数超过m-1，则——
            {
                b[a[c]]=0;      //把最早读入的桶变成0
                c++;            //将c的值加一
            }
        }
    }
    cout<<t;                    //输出t的值
}
```
回顾：
要注意bool类型在外面定义默认值是0，还有数组不能开太小了。大家也可以用队列做。
