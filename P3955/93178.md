 废话笔者也不多说，我们首先看这道题目，有编码、有长度，首
    先笔者自己就会想到结构体的综合运用（不知道各位有没有和笔
    者达成共鸣呢），但是本题的一个难点在于怎么将数字的一部分
    截取下来呢，那就需要用到函数了。
    
   我们来观察一下：
    len=1 如果截取数字来匹配的话应该要%（取余的符号，下文皆用
    此符号代替）10，取到数字的个位。
    len=2 应该取到百位
    len=3 应该取到千位
    ......
    笔者在此不一一举例，请大家自己去尝试。
   
我们会发现len等于多少，那截取的长度就要乘上几个十，也就是
    说截取的长度等于10的len次方。
    
   那么我们可以写一个while循环，每次截取的长度先乘10，然后	
    len--，如果len==0，那么说明可以退出循环了，所以while循环
    的退出条件就是len是不是为0,得出数字的截取长度后，我们通过
    取膜（也就是%）运算截取数字的一部分与题目给的需求码符不符
    合，符合就返回1，不符合就返回0。
   
   最后再自己完善一下输入输出，用for循环存入各个图书码，输入
    一个需求码，就查看整个数组有没有符合要求的图书码，记住，
    在输入完图书码后要用sort排序，因为它要输出最小的。如果有
    符合要求的图书码就输出那个图书码，如果没有，就输出-1。

	#include<bits/stdc++.h>
      using namespace std;
      struct book{
          int w;
          int num;
      }a[1005];
      int i,j,k,l,n,m;
      int b[1005];
      int ll(int x,int a,int y){
          int len=a;
          int w=1;
          do{
              w*=10;
              len--;
          }while(len!=0);
          if((x%=w)==y)return 1;
          return 0;
      }
      int main(){
          scanf("%d%d",&n,&m);
          for(i=1;i<=n;i++){
              scanf("%d",&b[i]); 
          }
          sort(b+1,b+n+1);
          for(i=1;i<=m;i++){
              scanf("%d%d",&a[i].w,&a[i].num);
          }
          for(i=1;i<=m;i++){
              for(j=1;j<=n;j++){
                  k+=ll(b[j],a[i].w,a[i].num);
                  if(ll(b[j],a[i].w,a[i].num)==1)break;
              }
              if(k==0)cout<<"-1"<<endl;
              else cout<<b[j]<<endl;
              k=0;
          }
          return 0;
      }
