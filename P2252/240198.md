# 威佐夫博弈
    威佐夫博弈（Wythoff's game）：有两堆各若干个物品，两个人轮流从任一堆取至少一个或同时从两堆中取同样多的物品，规定每次至少取一个，多者不限，最后取光者得胜
#
   	两个人如果都采用正确操作，那么面对非奇异局势，先拿者必胜；反之，则后拿者取胜。
	那么任给一个局势（a，b），怎样判断它是不是奇异局势呢？我们有如下公式：
	ak =[k（1+√5）/2]，bk= ak + k （k=0，1，2，...n 方括号表示取整函数)
	奇妙的是其中出现了黄金分割数（1+√5）/2 = 1.618...因此，由ak，bk组成的矩形近似为黄金矩形，
    由于2/（1+√5）=（√5-1）/2，可以先求出j=[a（√5-1）/2]，若a=[j（1+√5）/2]，
    那么a = aj，bj = aj + j，若不等于，那么a = aj+1，b = aj + j + 1，若都不是，那么就不是奇异局势。
    然后再按照上述法则进行，一定会遇到奇异局势
## 所以。。。
```
#include <bits/stdc++.h>
using namespace std;
const double L=(sqrt(5.0)+1.0)/2.0;
int main() {
	long long a,b,d;
	cin>>a>>b;
	if(a>b) swap(a,b);
	d=b-a;
	if(a==int(d*L)) printf("0");
	else printf("1");
	return 0;
}
```
