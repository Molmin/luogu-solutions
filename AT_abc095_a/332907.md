既然o是加配料，x是不加配料，那么啥也不加的拉面的价钱是900-2*100=700元。

得出了光面的价钱，那么接下来就很好算了：

一共也就3个字符，如果是o，那么价钱就加100元，如果是x，那就不变。

# **代码如下**
```
#include<bits/stdc++.h>
using namespace std;
int a=700;//初始化一下光面的价格
int main()
{
    char x,y,z;
    cin>>x>>y>>z;//输入
    if(x=='o')//开始三个字符一个一个判断
    {
        a=a+100;//如果是o，拉面价格就+100
    }
    if(y=='o')
    {
    	a=a+100;
    }
    if(z=='o')
    {
        a=a+100;
    }
    cout<<a<<endl;//输出，别忘了换行
    return 0;
    
}
    
```
