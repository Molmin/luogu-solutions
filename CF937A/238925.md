看到题解中的许多大佬都用了排序或STL等高级算法，但其实这道题并没有那么麻烦。

我们可以发现输入的$a[i]$的范围只有$0$到$600$，那么我们为什么不计数呢？

$F[x]$表示$x$在输入中出现的次数。
这样最后我们只用扫描一下，看$F[x]$大于$0$的数有几个即可。

注意：如果输入中含$0$，是不算入答案中的(见样例$3$)。所以最后扫描的时候应该从$1$到$600$哦！

$Code:$

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,x,ans,f[605];
int main() {
	scanf("%d",&n);
	for(int i=1;i<=n;i++){
	    scanf("%d",&x);
	    f[x]++;   
            //记录x在输入中出现的次数	
	}
	for(int i=1;i<=600;i++)//  注意：从1到600 而不是0哦
	    if(f[i])ans++;
	printf("%d\n",ans);
	return 0;
}
```
