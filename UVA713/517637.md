### 题目描述
这题很绕。给你两个数（根据样例，题目翻译十有八九错了），将这两个数反转求和，意思就是 例如给你 $12,24$，你需要求 $21+42=63$，再把和反过来输出 $36$。
### 思路分析
首先我们需要输入两个整数 $a,b$。用来表示两个数。
```python
dt=input().split(' ')
a=dt[0]
b=dt[1]
```
这里无门先暂时将格式设定为字符串存储。接下来是反转函数。其实对于列表，有一个 `list.reverse(start,end)`。由于是内置的，所以对于字符串不可用。

但是不需要新开函数，由于字符串有一个内置的切片方法，格式为 `string[start:end:step]`。这里需要反转，只需要声明 $step$ 为 $-1$ 就可以实现。
```python
a=a[::-1]
b=b[::-1]
```
然后现在就可以把他们转换成整数，并且求和。不用考虑前导0，因为这个 `int()` 对象十分强大，会自删前导零。
```python
a=int(a)
b=int(b)
sm=a+b
```
下一步，老套路把 $sm$ 转换为字符串，反转一下，再洗转换成整型，这一步也是消除前导零。
```python
sm=str(sm)
sm=sm[::-1]
sm=int(sm)
```
最后一步，输出就行。
```python
print(sm)
```
别忘了加一个主循环。
