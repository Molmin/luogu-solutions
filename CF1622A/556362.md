### 题意
有 $3$ 根木棍，问是否可以将其中的一根木棍分成两段，使这 $4$ 段可以构成矩形。

首先，我们可以考虑矩形的性质：对边相等。我们就可以发现，在拆分一根木棍时有两种方案。

- 当一根正好是另外两根的长度之和时，把这根拆成另外两根。

此时，我们可以发现，因为三个数都是正整数，所以被拆的那一根一定是最长的。

- 当一根可以平分且另外两根相等时，把这根平分。

对于这种情况，另外两根做长，分成的两根做宽，对于任意一根都有可能满足。

这两种情况都可以，输出 `YES`；如果两种情况都不满足，输出 `NO`。
### AC Code:
```cpp
#include<bits/stdc++.h>
using namespace std;
int t,l[3];
int main(){
	scanf("%d",&t);
	for(int i=1;i<=t;++i){
		scanf("%d%d%d",&l[0],&l[1],&l[2]);
		stable_sort(l,l+3);//排序，方便查询最大值
		if(l[2]==l[1]+l[0]||(l[1]==l[0]&&l[2]%2==0)||(l[1]==l[2]&&l[0]%2==0))
			puts("YES");
		else
			puts("NO");
	}
	return 0;
}
```
