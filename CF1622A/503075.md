# CF1622A题解
### 题目描述
给定三根长度分别为 $l_1$，$l_2$，$l_3$ 的木棍，仅能将其中一根木棍劈成两根较短的木棍。问这四根木棍能否围成矩形。

### 思路

分类讨论。

首先可以想到，当其中两根木棍长度之和等于第三根木棍时，可将第三根木棍劈成两段使得每段的长度分别等于另外两根木棍的长度。

例如样例 1，输入 $6$，$1$，$5$，因为 $6=1+5$，所以输出 "YES" 。

再往下分析样例，样例 2 和样例 3 都符合这个规律，但样例 4 不然。输入 $5$，$5$，$4$，显然不符合刚才我们的结论，然而输出仍然是 "YES" 。不难发现，长度为 $4$ 的木棍可以劈成两根长度为 $2$ 的木棍，仍可以围成矩形。

那么我们可以推测，三根木棍中任意两根长度相等，且另一根的长度为偶数时，可以围成矩形。

再考虑其他情况，发现都不能围成矩形。所以仅在满足以上两种情况时输出 "YES"，其他情况下输出 "NO" 即可。

### CODE
```cpp
#include<bits/stdc++.h>
using namespace std;
int t;

void solve(){
	int l[3];
	cin>>l[0]>>l[1]>>l[2];
	sort(l,l+3);
	if(l[0]==l[1]&&l[2]%2==0||l[1]==l[2]&&l[0]%2==0||l[0]==l[2]&&l[1]%2==0){
		cout<<"YES"<<endl;
		return;
	}
	if(l[0]+l[1]==l[2]){
		cout<<"YES"<<endl;
		return;
	}
	cout<<"NO"<<endl;
}

int main(){
	cin>>t;
	while(t--){
		solve();
	}
	return 0;
}
```
