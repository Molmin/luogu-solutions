## 题意

把题面翻译成人话就是：有三个数 $A$、$B$、$C$，把它们组成一个**两位数**和一个**一位数**，使这两个数的和最大。
## 分析
Q1：如何使这两个数的和最大呢？

A1：使两位数尽量的大。

Q2：如何使两位数尽量的大呢？

A2：使十位上的数尽量的大。

那么思路就出来了。

## 思路
将 $A$、$B$、$C$ 从大到小排序(**假设**排序后 $A\ge B\ge C$)

用 $A$、$B$ 组成两位数($A*10+B$)，$C$ 是一位数

则答案为 $A*10+B+C$。
##  Code
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int a[5];
	cin>>a[1]>>a[2]>>a[3];
	sort(a+1,a+3+1);
	cout<<a[3]*10+a[2]+a[1];
	return 0;
}

```
