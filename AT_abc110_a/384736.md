要想让 $x + y$ 最大，就让最大的数作为十位，两个小的不论哪个作为十位的个位，都是最大的情况。所以可以先用 $sort$ 函数进行排序。排序默认是从小到大排，所以最后一个数就是最大的，就让那个数作为十位，乘 $10$ ，其余的两位直接加上就可以了。

$AC$ 代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int a[3];
    cin >> a[0] >> a[1] >> a[2];
    sort(a,a + 3); // 排序
    cout << a[2] * 10 + a[1] + a[0] << endl;//最大做十位 其余加上就行
    return 0;
}
```
