## 题意
读入一个字符，一个整数，一个单精度浮点数，一个双精度浮点数，然后按顺序输出它们，输出时每个变量之间用一个空格分隔。浮点数时保留 6 位小数。
## 思路
分别使用一个整形变量，一个字符型变量，一个单浮点型变量，一个双浮点变量，依次输入，并输出，浮点型变量使用 ```printf("%.af",)``` 来输出保留a位小数的结果。
## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
char a;
int b;
float c;
double d;
int main()
{
    cin>>a>>b>>c>>d;
    cout<<a<<" "<<b<<" ";
    printf("%.6f %.6lf",c,d);
    return 0;
} 
```
