## 题目大意

高桥君想做很多花束，每束花束由 $A$ 朵红花和 $B$ 朵蓝花组成。高桥君手里有 $X$ 朵红花和 $Y$ 朵蓝花。

问：高桥君最多能做多少花束？

## 思路

- 如果只看每束花束由红花组成，那么高桥君可以做 $X \div A$ 束花束。

- 如果只看每束花束由蓝花组成，那么高桥君可以做 $Y \div B$ 束花束。

注：以上均为整型除法。

需要以上两点都达成才能做花束，所以我们将以上两点可做的花束数取较小值及可。

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int A,B,X,Y;
int main(){
    cin>>A>>B>>X>>Y;
    cout<<min(X/A,Y/B);
    return 0;
}
```
