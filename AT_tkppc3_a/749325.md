## 题意

在地球上，经度每相差 $15$ 度就有 $1$ 小时的时差。时差是为了不跨越国际日期变更线而取的差。 输入有两行，每行给出一个字符 $C$（西经表示为 `W` ，东经表示为 `E` ）和一个地点的经度 $A$，请求出其间的时差。

- $C$ 是 `W` 或 `E`。
- $A$ 是 $0$ 以上 $180$ 以下 $15$ 的倍数。

## 解析

首先，我们在初中的地理课已经学过了经度差如何去计算了，因为经度每相差 $15$ 度就有 $1$ 小时的时差，所以 经度差 $\div  15$ 就可以得到时差。



- 即，由于不跨越国际日期变更线（约为 $180^\circ$ 经线），在同侧的（同为东经或者同为西经）两点经度差为它们经度的差，而异侧的两点，要经过 $0^\circ$ 经线，所以它们的经度差就是它们经度的和。最后将经度差直接除以 $15$ 就可以得到时差。

## AC Code

```cpp
#include <iostream>
using namespace std;
char c1,c2;//字符C，表示东经或西经 
int a1,a2;//A来表示这个地点的经度 
int main()
{
	cin>>c1>>a1>>c2>>a2;//输入 
	if(a1<a2) swap(a1,a2);//swap函数可以交换两数值，保证大减小，结果非负（也可以直接用绝对值函数abs()）
	if(c1==c2) cout<<(a1-a2)/15;//是否在同侧 
	else cout<<(a1+a2)/15;
	cout<<"\n";//岛国题记得换行 
	return 0;//完结撒花（好习惯） 
}

```
