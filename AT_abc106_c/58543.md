这题其实特别简单，因为当经过了$10^{15}$

$S=12$时,新数列就会变成$S=12222222.....$

$S=123$时,新数列就会变成$S=12222222.....33333....$

但是，你算算$2$有多少个？

$2$经过$10^{15}$天后，会有几乎无穷大多个$2(2^{(10^{15})})$

这时除了$K=1$,在$1<K<10^{18}$时，必定为$2$

___

所以，答案其实很简单，当$S$前面有$I$个$1$

如果$I<=K$输出$1$

否则输出第一个不为$1$的数字
```cpp
#include<bits/stdc++.h>
using namespace std;
string n;
long long a;
int main()
{
	cin>>n>>a;
	for(int s=0;s<a;s++)
	{
		if(n[s]!='1')
		{
			cout<<n[s]<<endl;
			return 0;
		}
	}
	cout<<1<<endl;
}
```
