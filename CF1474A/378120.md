## 01. 题目大意
对于两个长度为 $n$ 的二进制整数 $a$，$b$，通过以下方式构造出整数 $d$：
- 将 $a$ 和 $b$ 按位相加但不进行进位运算。
- 将 $c$ 中相同且连续的数字替换为一位。

给定一个 $b$，求一个 $a$ 使得 $d$ 最大。

 ## 02. 题目分析
 这道题可以使用贪心的方法，因为根据题意可以发现位数越长的时候，$d$ 的大小越大。

先输出一个 $1$，因为第一位肯定为 $1$，然后就可以从最高位往后枚举，与上一位的数值比较，如果说加上 $1$ 之后不相等就在字符串后加一个 $1$，否则加一个 $0$。

时间复杂度 $O(n)$，肯定不会超时。

## 03. AC Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
	int T;
	cin >> T;
	for(int i = 1; i <= T; i++)
	{
		int n;
		cin >> n;
		string s1, ans = "1"; //注意要初始化到循环里面
		cin >> s1;
		for (int i = 1; i < n; i++)
		{
			if ('1' + s1[i] != ans[i - 1] + s1[i - 1]) //前面已经说过了，进行判断
			{
				ans += "1";
			}
			else
			{
				ans += "0";
			}
		}
		cout << ans << endl;
	}
	return 0; //完结撒花
}
```
