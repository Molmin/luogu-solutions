###【切入正题】

任意非0自然数的0次方=1（不会？）

然后我发现了一个规律：每到只有k^i（任意自然数）的时候，也就是没有加任何数的时候，我们就可以做一个队列：

把从队列的第一个（我们的循环从2开始）~第i-1个（此处的i与前面无任何关系，仅表示队列的当前项依次加上这个数，然后每加上一个数，就放在队列里，然后就能AC了。。。

###【不懂看代码】

（Ps:减少代码抄袭，共建美好洛谷）

```cpp
#include <iostream>
#include <cstdio>
using namespace std;
long long k,n,a[100001],head,p,ii; //这里的变量开longlong，不然只有90分
int main()
{
    cin>>k>>n; //输入
    a[1]=1;p=1;//队列的第一项一定是1
    for(int i=2;i<=n;i++)
    {
        a[i]=1;//注意每次大循环都是只有k^i（任意自然数）的时候，也就是没有加任何数的时候，所以我们为了求k的p次方，先赋初值为1
        for(int j=1;j<=p;j++)
        a[i]*=k;ii=i; //求k的p次方，顺便把i暂时存在ii里
        for(int j=1;j<ii;j++)
        {
            i++;
            a[i]=a[ii]+a[j];
        } //如前面所说的，把从第一项到第i-1项+第i项依次存进队列
        p++;ii=i; //其实这个ii=i没必要，因为后面已经有了
    }
    cout<<a[n]<<endl; //输出
    return 0;
}
```
