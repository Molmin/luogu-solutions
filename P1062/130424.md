这就是一道二进制的题目，先把k的0-15（应该够了）次方算出来，储存到d[]里面。

接着对n找，如果n&1这个是真的，就说明要取k的i次方。

举个栗子：

1. 输入3 5
1. 5的二进制是101
1. 所以取第一位和第三位，即3的0次方和3的2次方，和为10
1. 不懂的可以自己手推一下

上代码


------------

```cpp
#include<iostream>
using namespace std;
long long k,n,d[20],t=1,ans=0;
int main(){
	cin>>k>>n;
	d[0]=1;//初始化，任何正数的0次方都是1
    for(int i=1;i<=15;i++)d[i]=d[i-1]*k;//累计
    for(int i=0;n;i++){
        if(n&1)ans+=d[i];//看看这一位有没有数
        n>>=1;//n右移1，相当于/2
    }
    cout<<ans<<endl;
}
```
