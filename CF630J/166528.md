## C党请注意
这道题是非常简单的。

题目中说的n能被2到10每个数整除意思就要求2到10的最小公倍数，即n能被这个最小公倍数整除就一定能被2到10每个数整除。

2到10的最小公倍数是多少呢？

因为这个范围并不大，我们很容易就能将2到10中的所有质数找出来：有2，3，5，7；

在2-10中

2最多有3次方

3最多有2次方

5最多有1次方

7最多有1次方

直接算出 
$$2^3*3^2*5*7$$
就可以了！
答案就是2520！

求1-n中有多找个数能被2520整除就直接n/2520就可以了（放心，系统会帮你自动向下取整的）

**注意！这题的范围到了$ 10^{18} $!必须用long long!**

但是在C99以下的版本，是不支持long long的，那么你只能用高精来算了，我就不细说了

于是code就非常简单了
```c
#include <stdio.h>//基本输入输出流
long long x;//注意一定要开long long
int main()
{
	scanf("%lld",&x);//输入题目里的n，（我在这里是x）注意输入格式一定不要搞错了
	printf("%lld\n",x/2520);//直接输出x/2520的结果，也要注意格式
	return 0;//养成好习惯，竞赛中没有直接爆零
}
```
