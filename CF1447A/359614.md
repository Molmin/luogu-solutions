**题目大意**

有 $n$ 袋糖果，第 $i$ 袋有 $i$ 个糖，要你进行 $m$ 次(就是要求的东西)操作，每次操作选择一袋糖果。

第 $j$ 次操作除了选中的糖果袋，其他每袋糖果放入 $j$ 个糖。

让你输出操作次数和每次操作选中的糖袋子序号（输出任意一种情况即可）

分析一下，实际上就是把选中的袋子里减去 $j$ 个糖就行了。

稍微思考一下，第 $j$ 次修改把第 $j+1$ 个袋子选中就变成 $1$ 了。 $(j+1)-j=1$ ,所以这种情况下的 $m$ 就是 $n-1$ 啦，只要输出一种情况嘛，完工~~

每组数据中第二行上面也说了，直接把从 $2$ 到 $n$ 都输出一遍就行了，顺序无所谓，当然直接输出会更快。

std code:
```cpp
#include<bits/stdc++.h>
using namespace std;
int t,n;
int main(){
	cin>>t;
	while(t--){//多组数据
		cin>>n;
		cout<<n-1<<'\n';//输出题目中m的值
		for(int i=2;i<=n;i++)cout<<i<<' ';//依次输出
		putchar('\n');//换行符，用puts("");或cout<<endl;也行，个人喜好。
	}
	return 0;
} 
```
