第 $ i $ 次操作可以选择下标为 $ j $ 的数，将除了这个数以外的其它数加 $ i $ 。

将题意稍加转换一下，其实就是将下标为 $ j $ 的数减去了 $ i $ ,因为这样其实所有数之间的相对差是不变的，可以认为这是等价的。

这样题目一下子就变简单了，就是一个从 $ 1 $ 到 $ n $ 的序列先对一个数减 $ 1 $ ，再对一个数减 $ 2 $ ，以此类推一直到序列里面所有数都相等。

很容易想到对 $ 2 $ 减 $ 1 $ ，对 $ 3 $ 减 $ 2 $ ，$ \cdots $ ，到对 $ n $ 减 $ n-1 $ 为止，这样序列里所有数最后都是 $ 1 $ 。

一共 $ n-1 $ 次操作。（应该是最少的，因为不可能用 $ n-2 $ 次操作是序列所有数相等）

对于每个 $ n $ ,输出 $ n-1 $ 和 $ 2 $ 到 $ n $ 即可。

~~样例那个 $ 3 $ 完全是障眼法，求最少操作次数反而可能更好想~~

做题时，简化题意，不被题目迷惑是很重要的一步！

```cpp
#include<iostream>
#include<cstring>
#include<algorithm>
#include<cstdio>
using namespace std;
int t,n;
int main(){
	scanf("%d",&t);
	while(t--){
		scanf("%d",&n);
		printf("%d\n",n-1);
		for(int i=2;i<=n;++i)
			printf("%d ",i);
		if(n-1)puts("");
	}
	return 0;
}
```
