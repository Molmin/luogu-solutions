## 题目描述

给你$n$袋糖果，第$i$袋有$i$个糖，现在你可以进行若干次操作，每次操作选择一袋糖果


第$j$次操作定义如下：除了选中的糖果袋，其他每袋糖果放入$j$个糖

让你输出操作次数和每次操作选中的糖袋子序号（输出任意一种情况即可）

## 题目思路

~~生草翻译刚开始看的我一脸懵~~

我们先来化简操作，我们在给除了其中一袋，其他袋都加上同一个数时，相当于给这一袋减去了相应数量的糖果，这一点是很明显的

而且因为本质上还是加，所以不用考虑负数

现在我们来看原来的序列

例如

$1,2,3,4,5,6……$

这样的序列，显而易见的，相邻的数相差1，这个序列就是一个公差为1的等差序列

于是我们可以将这个序列转化为：


$a,a+1,a+2,a+3,a+4,a+5……$

其中$a$为$1$

那么运用我们上面的思路，第$j$次操作相当于给一袋糖果减去$j$

$j$是从$1$开始的，那我们就从比$a$大$1$的第二袋开始，每袋减去j就行了

举个例子，例如：

$n=3$

那么原序列为：$1,2,3$

第一次操作后为：$2,2,4$

第二次操作后为：$4,4,4$

此时结束操作

**总结**

操作次数为$n-1$

操作目标序列为从$2$到$n$，依次输出即可

## 代码
```c++
#include<bits/stdc++.h>
#define re register int
using namespace std;
int main(){
	int t;
	cin>>t;
	while(t--){
		int n;
		cin>>n;
		cout<<n-1<<endl;
		for(re i=2;i<=n;i++){
			cout<<i<<" ";
		}
	}
	return 0;
}
```
