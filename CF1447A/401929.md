### 题意：
给一个长度为 $n$ 的数组，第 $i$ 个数为 $i$ 从 $1$ 开始，我们进行 $m$ 次操作，第 $j$ 次操作时，选择一个数，除它以外每一个数都加上 $j$ 后，问如何操作可以使所有数都相等。如题 $m$ 没有限制，且不要求 $m$ 最小。

### 思路：
每一次，可以看成将选择的数减去 $j$ 的操作（与：每次选一个数，除它以外所有数都加 $j$ 的效果相同）。

因为要操作的数是 $1$ 到 $n$，所以我们按顺序选，第一次操作选 $1$，让第 $1$ 个减去 $1$ 个，第二次操作选第 $2$ 个，以此类推。

所以代码先输出 $n$ 再输出 $i$ 了。

### CODE
水题，无注释代码易懂（自认为）。题解区没有更简单的代码了。

```cpp
#include <bits/stdc++.h>
using namespace std;
int n, T;
int main() {
    cin >> T;
    while(T--) {
        cin >> n;
        cout << n << endl;
        for(int i = 1; i <= n; i++)
            cout << i << " ";
        cout << endl;
    }
    return 0;
}
```
