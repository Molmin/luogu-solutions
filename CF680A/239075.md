- 首先设总和为 $sum$ ，选出来要删除的数的**总和**为 $ans$
	- 例如 $1$ $1$ $3$ $3$ $5$ 五个数 $sum$ 就是 $13$ ， $ans$ 就是 $3*2=6$
- 最后答案就是 $sum-ans$
- 如何达到目的呢？
- 首先用一个数组记录所有出现的数，不重复，再用桶排计数每一个数字出现次数，因为只能减去重复的数，所以要更新 $ans$ 的条件一定要满足几条：
	1. 这个数出现次数大于1（桶排判断）
	2. 这个数的出现次数乘以本身大于 $ans$ 注意这里如果出现次数大于 $3$ 就算成出现了三次
- 最后看代码

```cpp
#include<bits/stdc++.h>//懒人万能库
using namespace std;
int ans,sum,s[6],d[101],tot;//分别为之前设的ans、sum以及记录出现的数的数组、桶排数组，所有不重复出现的数的个数
int main()
{
	for(int i=1;i<=5;i++)//循环五次不用讲
	{
		int x;
		scanf("%d",&x);//读入一个x
		if(!d[x])//如果没有出现过
		{
			s[++tot]=x;//就加到数组中
			d[x]++;//桶排++
		}
		else
			d[x]++;//否则桶排++
		sum+=x;
		if(d[x]>1)ans=max(ans,x*min(3,d[x]));
  		//如果满足条件就更新值取max，min的作用是保证不超过3
	}
	printf("%d",sum-ans);//输出
	return 0;//程序拜拜
}
```
