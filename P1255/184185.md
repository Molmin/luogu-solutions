这道题就是**高精度+斐波那契数列**

首先是斐波那契（兔子）数列，除了0,1两项为0,1，剩下的每一项都是前两项之和。

即
```cpp
f[0]=0;
f[1]=1;
f[n]=f[n-1]+f[n-2];
```


然而高精度可以用Python，这就是我为啥不用c++，~~我懒~~，瞬间少了一堆代码。

上代码: （**python2**）
```python
n=int(raw_input())          	#输入
f=[0,1]                	        #初始化，将f数组中放入0,1
for i in range(2,n+2):      	#循环2到n+1
    f.append(f[i-1]+f[i-2]) 	#公式
if n==0:			#特判
    print 0					
else:
    print f[n+1]
    #最短代码（应该）
```
