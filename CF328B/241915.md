### 题意简述：

给定两个字符串 x ，y，在 y 中 5 能当 2 用，6 能当 9 用，反之亦可。

求用 y 中的字符能拼成几个 x ？

### 方法：

因为 2 和 5、6 和 9 通用，所以我们可以把这两个数字看成一个，如 2 和 5 都看成 2。

遵循这个规则，我们可以将字符串 x 和 y 中出现的各种数字的个数分别用 a 数组和 b 数组表示，唯一需要注意的就是上面这行字。

当我们分别用a 、 b数组存储好之后，只需要算出在 x 中出现过的数（除 5 和 9 外）取b[ i ] / a[ i ] 的最小值输出就行了。

拿样例解释（懂了的直接去看代码吧）：

输入：

```
169
12118999
```
这时数组a 、b中的数（0 ~ 9）分别是：

```
0 1 0 0 0 0 2 0 0 1 
0 3 1 0 0 0 3 0 1 3 
```
对于 1 来说 b[ i ] / a[ i ] 为 3，对于 6 来说 b[ i ] / a[ i ] 为1（c++自动向下取整），9 自动屏蔽，因为已加到 6 中，最后取最小值为 1 输出。

AC代码：

```cpp
#include <bits/stdc++.h>
using namespace std;
int a[11],b[11],ans = 1000000;//因为ans要取min，所以尽可能设大。
string s;//为了少设变量我把x和y都变成了s，对代码没影响。
int main()
{
	cin >> s;
	for (int i = 0; i < s.length(); i++) a[s[i] - 48]++;//累加a数组
	cin >> s;
	for (int i = 0; i < s.length(); i++) b[s[i] - 48]++;//累加b数组
	a[2] += a[5],b[2] += b[5],a[6] += a[9],b[6] += b[9];//将（2,5）（6,9）合并，当然在累加的时候合并也行。
	for (int i = 0; i <= 9; i++) if (a[i] > 0 && i != 5 && i != 9) ans = min(ans,b[i] / a[i]);//注意要屏蔽掉5和9的情况，原因上面已讲。
	cout << ans << endl;
	return 0;//AC！！！
}
```
