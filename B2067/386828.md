## 题意分析

一道入门语法题。

将 $m$ 的意义修改为药房当前还剩的库存量，设 $t$ 为当前病人希望取走的药品数量，则：

- 若 $m \geq t$，则病人从 $m$ 的库存量中取走 $t$ 份药；

- 否则，病人无法取药，答案加 $1$。



## 代码

```cpp
#include <cstdio>
int m,n,t;
int ans;
int main(){
	scanf("%d%d",&m,&n); //输入 n, m
	while (n--){
		scanf("%d",&t);
		m >= t ? m-=t : ++ans; //重点语句
	}
	printf("%d\n",ans); //输出
	return 0;
}
```
### 三目运算符介绍

本人代码中有以下语句：

```cpp
m >= t ? m-=t : ++ans;
```

该表达式即使用了三目运算符。

表达式中问号前方为条件，

若条件为真，则运行冒号前方的表达式，

若条件为假，则运行冒号后方的表达式。

等价于以下语句：

```cpp
if (m>=t)
    m-=t;
else 
    ++ans;
```
