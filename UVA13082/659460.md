## 题意翻译：

给定一个长度为 $n$ 的序列，序列中的数互不相同。

你需要进行以下操作，使得序列单调不减。
- 将一个数移到序列末尾。

## 解法：
这是一道贪心题。

通过“序列中的数互不相同”可以发现，给出的数列是 $n$ 的一个排列。

发现不管怎么移，先要将最小的数 $1$ 移到序列的最开头，然后将 $2$  移到序列的次开头等等。以此类推，直到序列有序。

而我们在移动数字时，就可以使移动的那些数字变为有序状态。

所以设以 $1$ 开头的最长上升子序列为 $m$，则我们的最终答案就是 $n-m$。

code：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int T,n,a[10001] = {0};
	cin>>T;
	for(int j = 1;j<=T;j++){
		cin>>n;
		for(int i = 1;i<=n;i++){
			cin>>a[i];
		}
		int c = 0;
		for(int i = 1;i<=n;i++){
			if(a[i]==c+1)c++;
		}
		cout<<"Case "<<j<<": "<<n-c<<endl;
	}
	return 0;
}
```
