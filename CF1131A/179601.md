### 所以，我们可以知道：
- 周围方格的数量=周长+4（4个顶点）。
- 根据上图我们又可以知道周长=（下方矩形的底+2矩形高）* 2。
- 所以周围方格的数量=（下方矩形的底+2矩形高）* 2+4
- 然后程序就出来了。
## AC代码：
```
#include<bits/stdc++.h>
using namespace std;
int x,y,z,k;//定义，与题目中的w1,h1,w2,h2相对应。
int main(){
	cin>>x>>y>>z>>k;//输入。
	cout<<(x+y+k)*2+4<<endl;//按上述公式计算。
	return 0; //完美结束
}
```
- **如果您想简化一点，可以把**
```
(x+y+k)*2+4
```

- **变成**
```
(x+y+k+2)*2
```
