问题：
给定$w_1,h_1,w_2,h_2$,将一个长为$ w_2 $，宽为$ h_2 $叠在一个长为$ w_1 $，宽为$ h_1 $上。$1<= w_1,w_2,h_1,h_2 <= 10^8 ,w_1>w_2 $  
求这两个方框组成的图形周围的方格（$ 8 $个方向）的个数。  
根据观察，可以发现平移后，周围的方框就是一个长为$ max(w_1,w_2) $，宽为$ h_1+h_2 $的长方形的周围方格数。    
而$ w_1>w_2 $，所以$ max(w_1,w_2) $ = $ w_1 $。  
长为$ a $宽为$ b $的长方形的周围方格数是$(a+b)×2+4$   
所以长为$ w_1 $，宽为$ h_1+h_2 $的长方形的周围方格数就是$(w_1+h_1+h_2)×2 +4 $   
于是上代码   
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int w1,h1,w2,h2;
	int ans=0;
	cin>>w1>>h1>>w2>>h2;
	ans=2*(h1+h2+w1)+4;
	cout<<ans<<endl;
}
```
