## 题意

实现一个简单计算器，即给出一个或多个算式后计算其结果。

其中要注意的是计算时不按照运算法则运算，而是要按照算式从左往右依次计算。

## 思路

理解了题意后就是计算的问题了，上面也提到过要按从左往右的顺序计算，例如 `1 + 1 * 2 =` 这个算式就要先计算 $1+1=2$ 再计算 $2\times2=4$。 

接下来返回如何写代码。读入一个算式的过程应该是这样的：先读入第一个数字，接着读入运算符号，然后再读入一个数字，读入一个运算符号，依次类推，直到读入等于号结束。

变量定义上需要一个 $n$ 表示算式个数，$sum$ 表示累计计算的结果，$ch$ 表示读入的运算符号，$num$ 表示读入的数字。

读入的过程中可以写一个循环，重复读入数字与运算符，直到出现等号结束循环。

## 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int n, sum, num;
    char ch;
    cin >> n;
    while(n--){
        cin >> num;
        sum = num; //第一个数字
        while(cin >> ch && ch != '='){
            cin >> num;
            if(ch == '+'){
                sum += num;
            }
            else if(ch == '-'){
                sum -= num;
            }
            else if(ch == '*'){
                sum *= num;
            }
            else if(ch == '/'){
                sum /= num;
            } //运算符号
        }
        cout << sum << endl; //输出结果
    }
    return 0;
}
```
