这道题是一道打表找规律的题，比较简单。

这道题要我们求的是：

>有一块饼，你要切 $a$ 刀，问最多能切成几块？有多组输入数据，读到负数时停止。

我们可以用打表找规律的方法来解，开一个数组为 $f$，存储总共的块数，一个变量 $a$，代表切了几刀。

$$a = 0, f_{a} = 1$$
$$a = 1, f_{a} = 2$$
$$a = 2, f_{a} = 4$$
$$a = 3, f_{a} = 7$$
$$a = 4, f_{a} = 11$$
$$a = 5, f_{a} = 19$$
$$\dots$$


所以，我们可以发现：

- $f_{0} = 1$
- $f_{1} = 2$
- $f_{i} = f_{i - 1} \times 2 + 1$
