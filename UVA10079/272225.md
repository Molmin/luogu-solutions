题意
------------
求:
> $n$ 条直线最多能将一个平面分成几个区域


分析
------------

| 直线数 | 0 | 1 | 2 | 3 | 4 |
| :----------: | :----------: | :----------: | :----------: | :----------: | :----------: |
|原有直线数| 0 |  0| 1 | 2 | 3 |
|新增交点数|  0|0  |  1| 2 | 3 |
|新增区域数| 0 | 1 | 2 | 3 | 4 |
| 总区域数 | 1 |2  | 4 | 7 | 11 |




观察可得：每增加 $1$ 条直线，最多能与原有的 $n-1$ 条直线 ，产生 $n-1$ 个交点数， 新增 $n$ 个区域 。

也就是 每增加 $1$ 条直线，总区域数分别增加 $1,2,3$  。

可得出公式：$ans=n×(n+1)/2+1$

代码
------------
```
#include<bits/stdc++.h>
using namespace std;
int main()
{
	long long n;//注意 
	while(cin>>n&&n>=0)//输入，读到负数，结束 
	{
		cout<<n*(n+1)/2+1<<endl;//公式 
	}
	return 0; 
}
```
