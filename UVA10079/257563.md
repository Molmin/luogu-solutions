## 题面：
将一个平面用$n$条直线分割，最多能得多少块  
输入还是uva的风格，每行一个$n$,一直读到负数
## 公式证明
我们来看看公式：$ans=n×(n+1)/2+1$   
- 如果只有一条直线  
平面被分成两块
- 如果有两条直线  
平面被分成四块
- 如果有$n$条直线        
每增加一条直线，该直线与前面的每条直线相交，所以得$n*(n+1)$，但是想想看，每两条直线相交都会在两条直线中算两次，那么我们需要除以2，得$n*(n+1)/2$，但是，一开始就有一个平面，所以我们还要加上一开始那个平面，得$ans=n*(n+1)/2+1$  
## 代码
```
#include<bits/stdc++.h>
using namespace std;
int main(){
  	long long a;//开long long  
	while(scanf("%lld",&a)&&a>=0)//循环读入
  		printf("%lld\n",a*(a+1)/2+1);//套公式输出
	return 0;
}
```
