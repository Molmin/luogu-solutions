有数 $a$,$b$ 轮流执行，以下操作

如果是奇数就减去一，再把自己的一半给去另外一个数字，

即 $a=a+(b-1)/2$;

因为计算机执行除法操作时自动向下取整，直接除以 2 即可，

即 $a=a+b/2$;

如果是偶数，就直接把自己的一半给去另外一个数字，

即 $a=a+b/2$;

由于数据小直接用 for 循环枚举

第 $2n+1$ 次循环 $b=b+a/2$;

第 $2n$ 次循环 $a=a+b/2$;

代码如下

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int a,b,k;//定义 
	cin>>a>>b>>k;
	for(int i=1;i<=k;i++){//循环k遍 
		if(i%2==1){
			b+=a/2;//第2n+1次循环 b+=b/2;
			a/=2;
		}else{
			a+=b/2;//第2n次循环 b+=a/2;
			b/=2;
		}
	}
	cout<<a<<" "<<b;
	return 0;
}
```
