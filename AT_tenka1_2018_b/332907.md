这是一道简单的操作问题。输入a、b、k（k代表操作次数），然后从a开始：如果是奇数就减去一，再把自己的一半给去另外一个数字如果是偶数，就直接把自己的一半给去另外一个数字

我们知道：C++里，不管M是奇数还是偶数，都符合：

## $\frac{M}{2}$=$\frac{M-1}{2}$
   

所以这道题可以转换为：从a开始，a=$\frac{a}{2}$，然后，b=b+$\frac{a}{2}$，然后轮到b，b=$\frac{b}{2}$，a=a+$\frac{b}{2}$……（一直循环k次）

循环里面还要有判断，当循环次数为奇数（第1、3、5、7次……）时，轮到a来操作；为偶数（第2、4、6、8次……）时，轮到b来操作

代码如下
------------
```
#include<bits/stdc++.h>
using namespace std;
int main()
{
 int a,b,k;
 cin>>a>>b>>k;//输入
 for(int i=1;i<=k;i++)//循环
 {
     if(i%2==1)//当i为奇数，轮到a来操作
     {
  	a=a/2;
        a=a+b;//b就把自己的一半给a
     }
     else//反之，i为偶数时，轮到b来操作
     {
  	a=a/2;
  	b=b+a;
     }
 }
 cout<<a<<" "<<b<<endl;//输出
 return 0;
}
```
