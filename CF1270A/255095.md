再来看一看这道题：

> 每次两人**各出一牌**，牌号大的一方会得到**这两张牌**，直到其中一方无牌， 最后有牌的一方获胜。

也就是说，每次两人只出 1 张牌，牌大的人会**返还**这两张牌。

那么，我们就有了个奇思妙想：

假如，一个神秘人给了你一张一千亿元的银行卡（~~在想屁吃~~），你完全可以拿这些钱买房买车。总之就一句话：**有限任性！**

对于这道题，最大牌在谁的手里，谁就必胜。事实上，对于这个游戏来说，你肯定每次都出手里最大牌。如果还是被压下去了，直接认输就得了嘛！

所以，我们只要知道最大的一张牌到底是不是在 hs 的手里就好了！

而在这道题中，因为每张牌都不一样，所以最大牌一定是$n$

```cpp
#include <bits/stdc++.h>
using namespace std;
int T, n, l, r, maxl, maxr, card;
int main()
{
    cin >> T;
    for (int i = 1; i <= T; i++)
    { //CF标准毒瘤输入
        cin >> n >> l >> r;
        for (int j = 1; j <= l; j++)
        {
            cin >> card;
            if (card == n)
                cout << "YES" << endl; //如果最大牌在hs手上，hs就赢了！(ﾉﾟ∀ﾟ)ﾉ
        }
        for (int j = 1; j <= r; j++)
        {
            cin >> card;
            if (card == n)
                cout << "NO" << endl; //zhy赢了>_<
        }
    }
    return 0;
}
```
