***两个主要思想：

1.使得k个值相加得整数m（题中的整数n我用m来表示，而n我用来表示有几个符合题意的解，不要弄混哦）

2.使判断出的解不出现重复，这里我利用使得出的解全部按照升序排列来避免重复，3=1+2可以，但3=2+1不会取到，因为 2比1大，在这个程序中，后面的数只能比前面的数大。


代码如下：



```cpp
#include<iostream>
using namespace std;
int m,n,k;//n个解，将m划分为k个数； 
void chai(int k,int i)//划分为k个数，为了使每个数都比前一个数大，将i设置为前一个数所取的值；
{
    for(int j=i;j<=m;j++)//从前一个所取的值开始判断，保证了所取的值大于等于前一个值，避免重复 ； 
    {
        if(k==2)//若k等于2了 证明此时已取到了倒数第二个数，进行判断倒数第一个数； 
        {
            if(m-j>=j)//若倒数第一个数也比前一个数（倒数第二个数）的值大，就满足了将数升序排列，避免了重复 ； 
            {
            n++;
            }
            }
        else//如果好没有到倒数第二个数，进行递归选取下一个数； 
        {
            m-=j;//将m值减去此时取得的数 ； 
            chai(k-1,j);
            m+=j;//为保证m的值在这一层循环时减去的数可以加回来； 
            }
        }
}
int main()
{
    cin>>m>>k;
    chai(k,1);//由题意，k个值都不能为空，所以第一个值最小要从1开始； 
    cout<<n;
    return 0;
}
```
