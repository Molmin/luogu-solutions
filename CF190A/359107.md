### 题意：

公交车上有 $n$ 个大人 和 $m$ 个小孩，一个大人只可以免费带一个小孩，小孩不能单独乘车，一张票 $1$ 块钱，求赚钱的最大值和最小值。

---

### 分析：

一共分为 $3$ 种情况。

1. 车上只有大人。那么不用考虑小孩的免费问题，直接输出两次大人们所花的钱，即大人的人数。

2. 车上只有小孩。这样是不合法的，直接输出 "Impossible" 。

3. 车上既有大人也有小孩。考虑两种最极端的情况。      
   （1）一位大人带了所有的小孩，那么其中只有一个小孩不能付钱，其他所有人都要付钱，那么结果是 $n+m-1$ 。

   （2）每位大人都带了一个小孩，那么所付的钱数即为大人人数或小孩人数的最大值。因为如果大人较多，那么所有的小孩都不用付钱；如果小孩较多，那先付多出来的小孩的钱，再付接下来大人的钱，其余的小孩不用付钱，即小孩的人数。
   
### 代码：

```cpp
#include<iostream>
using namespace std;
int main()
{
	int n,m;
	cin>>n>>m;
	int maxn;
	if(m>=n)maxn=m; //求出第三种中的最大值
	else maxn=n;
	if(m==0)cout<<n<<" "<<n;  //没有小孩
	else if(n==0)cout<<"Impossible";  //没有大人
	else cout<<maxn<<" "<<m+n-1;  //大人小孩都有
	return 0;
}
```
