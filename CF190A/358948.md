# 题意：
一辆~~宝宝巴士~~上有n位大人和m位小孩，每人的车票为一卢布，而一个大人可以免费带一名小孩，但是小孩不能自己坐车（雾），也就是说车上不能没有大人，然后求赚钱的最大值和最小值。
# 本题思路：
由题意得，我们可以分为三种情况：

1.车上没有大人，只有小孩，输出“Impossible”；
       
2.车上有大人有小孩，并且一个大人带一个小孩；

3.车上有大人有小孩，但是是一个大人带所有小孩，剩下的人支付自己的车票（大雾）；

分析完毕，那么我们可以写出代码去实现它。
## 代码如下：
```c
#include<iostream>
#include<cstdio>
#include<algorithm>//max所对应的头文件 
using namespace std;
int main() {
	int n,m,ans;
	cin>>n>>m;//简单的输出
	ans=max(m,n);//求出m和n的最大值赋给ans
	if(n==0&&m!=0) {//车上没有大人时必须要有小孩，车上不能没有人！！！ 
		cout<<"Impossible"; 
		return 0;//直接结束 
	} //当没有大人，只有小孩的时候 乘车是不合法的，所以输出Impossible
	else if(m==0) {
		cout<<n<<" "<<n;
		return 0;
	} //当没有小孩，只有大人的时候 就输出大人每人一张票需要的钱数就可以了 
	else {
		cout<<ans<<" "<<m+n-1;
	}//当大人和小孩都有的时候，最小值为一个大人带一个小孩 最大值为一个大人带了所有的孩子
	return 0;//完美结束 
}
```
