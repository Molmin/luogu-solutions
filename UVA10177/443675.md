# 翻译
给定 n，求分别在二维（$n^{2}$）的正方形，三维（$n^{3}$）的正方体中和四维（$n^{4}$）的超立方体中，分别有几个小正方形（体）和几个小长方形（体）。  
为了方便表示，在下面的过程中，我们用 k 来表示其维度，即 n 的指数。另外指出，下文的长方形是不包括正方形在内的，与课本定义稍有不同。
# 分析
首先，分析小正方形（体）的数量。
简单模拟可知：  
二维正方形中小正方形的数量 $=1^{2}+2^{2}+3^{2}+...+n^{2}$。  
三维正方体中小正方体的数量 $=1^{3}+2^{3}+3^{3}+...+n^{3}$。  
猜想：推广到 k 维就是 $1^{k}+2^{k}+3^{k}+...+n^{k}$。  
接下来想知道小长方形（体）的个数，只需要知道所有小长方形（体）和小正方形（体）的个数之和即可。  
从 $k=2$ 开始，简单模拟可知总数为 $(1+2+3+...+n)^{k}=(\frac{n(n+1)}{2}) ^{k}$。  
那么小长方形（体）的个数就是总数减去小正方形（体）的个数了。
# 代码
我把长方体的部分展开到输出中了，省去几个变量。
```cpp
#include <bits/stdc++.h>
using namespace std;
long long n,temp,s2,s3,s4;
int main(){
	while (scanf("%lld",&n)==1) {
		for (int i = 1;i <= n;i++) s2 +=  i * i,s3 += i * i *i,s4 += i * i *i * i;
		temp = (n+1)*n/2;
		printf("%lld %lld %lld %lld %lld %lld\n",s2,temp * temp - s2,s3,temp * temp *temp -s3,s4,temp * temp *temp * temp - s4);
		s2=s3=s4=0;
	}
}
```
