### 分析与思路

这题看着是一个复杂的图片和三行密密麻麻的文字，其实只需要仔细阅读就可以简单 ```AC``` 。我给大家分步整理了一下题目意思:

1. 给定一个字符串 ```s``` 和一个整数 ```k``` 。

1. 遍历 ```s``` 的每一个字符。

1. 将遍历到的字符转为小写。

1. 每次遍历时需判断该它的字母表序是否小于等于先前输入的那个数字所代表的字母的字母表序，若是的话就将其转为大写。

1. 整个字符串遍历完以后输出新的字符串。

观察一下第三和第四步，那我们是不是除手写之外没有其他的办法将一个字符转大写和转小写？

当然不是。我们可以用 ```toupper``` 和 ```tolower``` 函数来完成此操作。



------------
### ```toupper``` 与 ```tolower``` 函数

这是两个非常好用的函数。其中， ```toupper``` 是用来转大写的， ```tolower``` 是用来转小写。一定要记牢，区分开。这两个函数的用法为：

```int toupper(int c);```

```int tolower(int c);```

用这两个函数可以更加方便的将字符转大小写。



------------
### 完整 ```AC``` 代码

```
#include<iostream>
#include<string>
using namespace std;
string s;
int k;
int main(){
    cin >> s >> k;
    int num=k+'a'-1;//数字k所代表的字母就是k+'a'-1
    for(int i=0;i<s.size();i++){
        s[i]=tolower(s[i]);
        if(s[i]<=num)s[i]=toupper(s[i]);
    }
    cout << s << endl;
    return 0;
}
```
