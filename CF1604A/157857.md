思考一下如何插数才是最优的。

我们可以找到可以找到一种数让它合法，然后就一直插吗？

观察可以得到，无论如何，插 $1$ 永远是合法的，那么这题就很简单了。

每次考虑贪心的将不合法的 $a_i$ 前面插 $1$，直到合法为止。

正确性显然：因为你要使 $a_i$ 合法，你就需要在 $a_i$ 前面插入数使得 $a_i\le i$，这样子因为 $a_i$ 并没有变，但是它的位置在增大，所以它一定能合法。我们又知道，无论插入什么数，只要是合法的，那它对位置的贡献都是 $1$，所以插入什么合法的数对答案其实是没有影响的。所以我们可以使用前面说到的万能数 $1$ 作为那个合法的数插入。

~~当然肯定不能模拟一个一个插数的过程~~。

将插数的过程快速计数即可。

**code**

```cpp
#include<cstdio>
using namespace std;
int T,a[110],ans,ret,n;
int main()
{
	scanf("%d",&T);
	while (T--)
	{
		scanf("%d",&n);ans=0;
		for (int i=1;i<=n;i++)
			scanf("%d",&a[i]);
		ret=0;
		for (int i=1;i<=n;i++)
		{
			ret++;
			if (a[i]>ret)
			{
				ans+=a[i]-ret;
				ret=a[i];	
			}
		}
		printf("%d\n",ans);
	}
	return 0;
}
```
