### 第一步 题目解析

这个题我们可以发现（没看好的[回去看题](https://www.luogu.com.cn/problem/CF1081A)）：

1、$n$（$3 \le n$）一定和 $n-1$ 互质，我们 $x$ 就可以选择 $n-1$（这里就是这个坑，因为他的说明样例1没有选择 $x = 7$）。那么我们就可以得到结论：如果 $3 \le n$，就输出1。

2、 $n = 2$ 时我们无法选择任何一个比2小且与2互质的数字（1是2的因数谁都知道）。所以当 $n = 2$时就输出2。

3、 $n = 1$ 时我们也都看到样例2了，输出1（因为没有比他小的互质的数字）。

下面给出一个表格看的更清楚：

| n | 输出 |
| :----------: | :----------: |
| 2 | 2 |
| $\not= 2$ | 1 |

### 第二步 知识拓展

1、三目运算符：

格式为
```
判断条件？执行1：执行2
```
当判断条件成立时会执行 执行一，否则执行 执行2。

最后给一个核心代码：

```
printf("%d\n",(read()==2?2:1));//read()函数是我写的一个快速读入函数，如果你不会写可以写一个cin然后返回。
```
