拿到这道题目，很明显的可以看到是要求 $n$ 的最小值，我们自然而然的就能想到贪心。

如何贪心？其实就是模拟。从大到小（因为 $n$ 要最小）枚举 $x$ 的值，若 $x$ 不是 $n$ 的因子则 $n - x$ ，直到不能再变小，最后输出 $n$ 就能 AC 了。

但是明显的，这题有更优算法。

还是顺着贪心的思路走，第一个要枚举的 $x$ 是 $n - 1$，根据小学数学我们可以知道两个相邻的正整数数必定互质，然后我们可以惊奇地发现 $n$ 直接就变成 1 了。但是题目在这里埋了一个坑，题目中说 $x$ 不是 $n$ 的因子时才可进行操作，所以因为 1 是任何数的因子，却与任何数又都是互质数，我们需要特判和 1 相邻的正整数 2，在碰到 2 时输出 2，其余输出 1 即可 AC。

代码：

```cpp
#include <bits/stdc++.h>
inline int read(){
   register int s=0,w=1;char ch=getchar();
   while(ch<'0'||ch>'9'){if(ch=='-')w=-1;ch=getchar();}
   while(ch>='0'&&ch<='9')s=s*10+ch-'0',ch=getchar();
   return s*w;
}
int main(){	return printf("%d",(read() == 2 ? 2 : 1)),0;	}
```
