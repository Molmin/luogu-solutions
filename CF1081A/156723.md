First，如果n是个奇数，那么就一直减2减2……直到减到1。所以就丢给最后一步去特判吧！

偶数嘛，先找一个比他小的奇数，且不是他的因数，减成一个奇数，然后，你知道的~~。

我们在最后一步还是要特判一下~~

程序如下↓↓↓
```
#include <bits/stdc++.h>
using namespace std;
int n; 
int main()
{
	cin>>n;
	if(n%2==0)//偶数就开始寻找
		for(int i=3;i<n;i+=2)//从3开始找 
			if(n%i!=0)//如果i不是n的因数 
			{
				n-=i;//n减去i 
				break;//跳出循环，免得找到两个又变成偶数 
			} 
	if(n%2==1)//如果是奇数 
	cout<<1;//输出1 
	else
	cout<<n;//输出n 
} 
```
