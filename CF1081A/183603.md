**首先读懂题目：对于一个数n，选择一个数x且满足n % x!= 0 && x < n，然后n = n - x。最后要使得n最小，输出这个最小的答案。**     

**要使这个差值最小，差值最小为1（x < n）。而我们又发现当两数差值为1时两数互质，因而对于一个n（n ！= 2），这个最小的答案1一定是由n与n - 1作差得到的。【当然，n = 1时本身就是最小，取0次即可】**

**对于2这个特殊的数 n - 1与n成因数的关系（1能被2整除，~~废话~~），所以此时最小为2（就是操作0次），特判即可。**

**代码：（10行）**
```
#include <iostream>
using namespace std;
int main()
{
	long long n;
	cin>>n;
	if(n >= 3 || n == 1) cout<<1<<endl;//n与n - 1互质情况与n == 1 
	else cout<<2<<endl;//n == 2特判 
	return 0;
}
```
