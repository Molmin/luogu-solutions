```cpp
if(minn>500)
    {
        for(re int i=1;i<=n;i++)
        {
            jia+=w[i]-minn;
            w[i]=500+w[i]-minn;
        }
        if(m%minn==0)
        m=(m/minn)*500;
        else
        m=(m/minn)*500+jia;
    }
```
其中minn是最便宜的那种价钱。
