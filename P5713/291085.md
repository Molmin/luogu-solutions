**题目思路**：  
首先，在本地上传一道题目要$5$分钟的时间，一共有$n$道题目，那么容易得出在本地上传需要$5n$分钟，在$\text{C++}$中表示为$5*n$；而在洛谷上传一道题目则需要$3$分钟的时间，一共有$n$道题目，$\color{red}\text{注意}$，这个$11$分钟是指所有题目上传之前需要额外的$11$分钟，不是指上传每一道题目都需要这额外的$11$分钟，所以，在洛谷上传一道题目需要$3n+11$分钟，$\text{C++}$中表示为$3*n+11$。  

我们知道了在本地上传一道题目要$5×n$分钟的时间，在洛谷上传一道题目需要$3×n+11$分钟，那么题目就变成了：给定$n$，比较$5×n$和$3×n+11$的大小，如果$5×n$大，输出$Local$，否则$3×n+11$大，输出$Luogu$。例如，有$5$道题目，在洛谷上传要$5×3+11=26$分钟，而在本地上传则要$5×5=25$分钟，因为在本地上传时间比洛谷时间少，所以输出$Local$；再比如，有$7$道题目，在洛谷上传要$7×3+11=31$分钟，而在本地上传则要$5×7=35$分钟，因为在洛谷上传时间比本地时间少，所以输出$Luogu$。做这一道题目需要使用到$if$语句，代码如下：
```
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int a;
    scanf("%d",&a);
    if(a*5>a*3+11)printf("Luogu");//如果在洛谷上传的时间短，就输出“Luogu”。
    else printf("Local");//否则输出“Local”。
    return 0;
}
```
时间复杂度为$\text{O(1)}$。

------------
当然还有另外一种做法，想一想，在洛谷每上传一道题目比在本地上传一道题目要短$(5-3)=2$分钟，但是洛谷总时间还有额外的那$11$分钟，所以只用比较洛谷单纯题目上传速度比在本地多的与$11$分钟做比较，如果洛谷单纯题目上传速度比本地多的的部分比$11$分钟少则输出$Local$，否则输出$Luogu$，由于洛谷每上传一道题目比在本地上传一道题目要短$2$分钟，$11$不是$2$的倍数，所以不存在本地和洛谷上传时间一样的可能性，再往深处想，每上传一题抵$2$分钟，抵到$11$分钟以上洛谷的速度就比本地快，那么题目就又变成了：每上传一道题目积$2$分钟，问最少上传多少道题目可以积到$11$分钟及以上。显而易见是上传$6$道题目，于是得到以下代码：
```
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int a;
    scanf("%d",&a);
    if(a>=6)printf("Luogu");
    else printf("Local");
    return 0;
}
```
时间复杂度为$\text{O(1)}$。
