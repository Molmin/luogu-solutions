**part1 枚举出规律**

枚举几个数
```
1 1   2*1-1
2 3   (3*3-1)/2-1
3 7   4*2-1
4 11  (5*5-1)/2-1
5 17  6*3-1
6 23  (7*7-1)/2-1
7 31  8*4-1
8 39  (9*9-1)/2-1
9 49  10*5-1
10 59  (11*11-1)/2-1
```

不难看出规律：

1.当 n 为奇数时，答案为：  
 
 $(n+1) *  \tfrac{ (n+1) }{2}-1=\tfrac{ (n+1)^2-2}{2}$ ;

2.当 n 为偶数时，答案为：             

$\tfrac{ (n+1)^2-1 }{2}-1=\tfrac{ (n+1)^2-3}{2}$ .

**part 2 代码实现**

规律都写出来了，那代码就是手到擒来。

```
#include<bits/stdc++.h>
using namespace std;
long long a;//记得开long long
int main(){
	while(cin>>a){
		if(a%2==1){
			cout<<((a+1)*(a+1)-2)/2<<endl;
		}else{
			cout<<((a+1)*(a+1)-3)/2<<endl;
		}
	}
	return 0;
} 
```
