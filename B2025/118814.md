本人在向大家提供多种方法

方法一：$\texttt{C++}$

$\texttt{C++}$中的`printf`函数有一种功能，可以多行打印字符串:

```cpp
#include<bits/stdc++.h>
using namespace std;
long long t,a,b,i;
int main()
{
    printf(
  	"  *\n"
    " ***\n"
    "*****\n"
    " ***\n"
    "  *\n");
    return 0;
}
```

方法二：$\texttt{Pascal}$

用`writeln`直接输出即可:

```pascal
begin
    writeln('  *');
    writeln(' ***');
    writeln('*****');
    writeln(' ***');
    writeln('  *');
end.
```

方法三：$\texttt{Python}$

用`print`函数输出:

```python
print("  *")
print(" ***")
print("*****")
print(" ***")
print("  *")
```

方法四：$\texttt{PHP}$

$\texttt{PHP}$ 有一个特性，即碰到字符回原样输出，直接复制样例即可：


```php
  *
 ***
*****
 ***
  *
```
