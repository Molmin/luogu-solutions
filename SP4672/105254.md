## 解析

挺巧妙的一道题

首先题意可以理解为：求**任何前缀**中，持有 $10$ 元钞票人数不超过持有 $5$ 元钞票人数，的排列占比；其中持有 $10$ 元钞票的共有 $n$ 人，持有 $5$ 元钞票的共有 $m$ 人

我们发现，如果不考虑持有同样面额钞票的人相对位置关系，在求出任意一种排列后，再考虑持有同样面额钞票的人相对位置关系，其情况数**都是相同的**。因此我们可以不考虑同面额的排列情况，只考虑不同面额的排列

接着可以把这个问题转化为，求所有从格点 $(0, 0)$ 到 $(m, n)$ 的格点路径，且该路径和直线 $y=x+1$ 没有交，的占比

（这还是挺显然的，想想这条直线的实际含义就行）

不过这还是不太好做，我们考虑把问题再化简下：对于每条**不合法**（和直线有交）的路径，我们将它从起点 $(0, 0)$ 到和直线**第一个**交点的这段路径，以直线为对称轴 "反射"

![反射](https://cdn.luogu.com.cn/upload/image_hosting/ixbx806u.png)

我们发现，经过这样的处理后，所有不合法的路径与所有从 $(-1, 1)$ 出发到 $(m, n)$ 的路径**一一对应**（双射）

&nbsp;

首先我们仅考虑原路径到反射后路径的映射关系，因为它们的证明是类似的。我们采用反证法：

考虑两条不同的反射前路径在反射后一致

首先它们和直线的交点得一致

接着交点到终点 $(m, n)$ 的路径也得一致

起点到交点的路径因为反射，也得一致

可以发现，这两条反射前的路径其实是完全一致，由此得证

（话说我初看到这个方法是总感觉有问题，但细读后发现正确性还是挺显然的。因为这个方法**唯一确定**了反射和不反射的断点以及反射的部分；而如果仅仅是将路径 "在直线下方" 的点反射到直线上面，就错了（也是我一开始理解的））

&nbsp;

于是就可以知道答案为 $\frac {\binom{n+m}{n}-\binom{n+m}{n-1}} {\binom{n+m}{n}}=\frac{m+1-n}{m+1}$

注意这个式子在 $n>m$ 时不再使用（此时其几何图形中，直线在 $(0, 0)$ 下方）

## CODE

```cpp
#include <cstdio>

int main(){
	int n, m;
	scanf("%d %d", &n, &m);
	while(n != 0 || m != 0){
		if(n > m)
			puts("0.000000");
		else
			printf("%.6lf\n", (m+1-n)*(double)1.0/(m+1));
		scanf("%d %d", &n, &m);
	}
}
```
