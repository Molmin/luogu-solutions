蒟蒻的第二篇题解-不知道有没有dalao已经发过了（逃）


蒟蒻的暴力枚举做法，就是输入后得到长度，然后逐个扫过去；当发现s[i]有字母出现而且后面跟的是：的时候就去赋值；

如何判断要赋什么值呢？这里用的是判断s[i+3]是否是字母，是的话就赋相应的字母值，否则就赋数字值；

因为用a，b，c比较麻烦，所以就用数组a来代替；

下面附上代码，学习各位dalao的做法在里面埋下了坑，如果直接复制大概会爆0咯


```cpp
//防伪标识
#include<bits/stdc++.h>//万能头文件不用说的
using namespace std;
char s[256];//长度的范围要注意
int a[4] = {0}; //a[1]是a的值，a[2]是b，a[3]是c
int main(){
    gets(s);//因为空格要一起读进去所以gets会比较方便
    int len = strlen(s);//长度计算
    for(int i = 0;i<len;i++){
        if(s[i] >='a'&&s[i]<='c'&&s[i+1] == ':'){ //判断s[i]是不是要赋值的变量
            if(s[i+3] >= 'a'&&s[i+3]<='c') // 判断所要赋的是某个字母变量的值还是数字
            a[s[i]-'a'] = a[s[i+3]-'a'+1]; // 这里就是转化的过程，将字母a，b，c变成数组的下标，此行有坑请注意
            else a[s[i]-'a'+1] = s[i+3]-'0';
        }
    }
    printf("%d %d %d",a[1],a[2],a[3]); //最后输出不要忘记使用标准输出格式
    return 0;
}
```
