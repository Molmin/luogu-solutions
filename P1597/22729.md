看到楼下这么多好长吧长的题解，我惊呆了

这个题目其实并不需要楼下大牛用的筛选":=",";"。

如果硬是要用筛选，也并不需要一个个判断，用scanf("%[abc]%[:=]%[0-9abc]%[;]")**扫描集**就可以了。

为什么不要用上筛选呢？看题啊！

里面说了，数字只占一位！

于是数字只占一位，字母也只占一位，中间":="两位，后面";"一位，我们就可以通过提取每5个字符中的第1个和第4个来读取！

附上简洁代码

```cpp
#include <stdio.h>
int main() {
    int a[5] = {0}, i, f;  
    char ch[300]; 
    scanf("%s", ch); 
    for(i = 0; ch[i] != 0; i++) {
        if(i%5 == 0) {
            if(ch[i] == 'a') f = 1; 
            else if(ch[i] == 'b') f = 2; 
            else f = 3; 
        } //f:用于存储被赋值字母是哪一个 
        if(i%5 == 3) {
            if(ch[i] == 'a') a[f] = a[1]; 
            else if(ch[i] == 'b') a[f] = a[2]; 
            else if(ch[i] == 'c') a[f] = a[3]; 
            else a[f] = ch[i]-'0'; 
        } //赋值操作 
    }
    printf("%d %d %d", a[1], a[2], a[3]); 
}
```
