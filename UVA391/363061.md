# 思路：

1. `\b` 与 `\i` 没有区别，遇到直接跳过即可；
2. `\s` 与 `\b`,`\i` 唯一的区别是遇到 `\s` 需要找到 $s$ 后面的第一个既不是数字也不是小数点的字符；
3. `\*` ~~个人认为~~是这个里面最难的一部分了。因为它决定着后面`\b`,`\i`,`\s`,`\\` 是否按关键字处理与输出，所以需要添一个 bool 变量来判断；
4. `\\` 与 `\b`,`\i` 一样，只是跳过以后需要输出一个`\`。

### 敲黑板
## 这个题有四大坑点：

1. `\*` 部分一定要首先判断，因为它决定着 Mark-up 模式是否打开；
2. C++ 中 `\` 不能被直接判断，需用他的 ASCLL 码表示；
3. 如果 `\` 后面不是关键字， `\` 不应保留(~~翻译的锅~~)；
4. 各行之间互相影响。例如：

输入：
```
\*
\\
```
输出 ：

```

\\
```

而不是 

```

\
```

剩下的按照题意模拟即可，思路也在代码里。

# Code:

```cpp
#include<bits/stdc++.h>
using namespace std;
string a;
bool ox=true;//看Mark-up模式是否打开（默认为开）。 
int main()
{
	while(getline(cin,a))
	{
		int len=a.size();
		int i;
		for(i=0;i<len;i++)
		{
			if(a[i]==92&&a[i+1]=='*')//ASCLL码里 \ 是92。 
			{
				ox=!ox;//Mark-up模式的开关。 
				i++;//跳过'*' 
				continue;//一定要从头开始，因为i已经改过了。 
			}//判断是否打开。 
			if(a[i]==92&&ox==true)//如果当前字符是\并且Mark-up模式开着。 
			{
				if(a[i+1]=='s')
				{
					for(int j=i+2;j<len;j++)
					{
						if((a[j]<'0'||a[j]>'9')&&a[j]!='.')
						{//找第一个不是数字或小数点的字符。 
							i=j-1;//这里要注意，因为返回时要i++，所以这里的i要存j-1。 
							break;
						}
					}
					continue;
				}
				if(a[i+1]=='b'||a[i+1]=='i')
				{
					i++;//碰到\b和\i就跳过，为什么i++而不是i=i+2,同前文的i=j-1。 
					continue;
				}
				if(a[i+1]==92)
				{
					i++;
					cout<<char(92);
					continue;
				}
			}
			else
			  cout<<a[i];//如果不是关键字或者Mark-up模式关闭了，就原样输出。 
		}
		cout<<endl;
	}
	return 0; 
}
```
