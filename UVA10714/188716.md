### 题意

一根长为 $l$ 的木棍，上面给出 $n$ 只蚂蚁的初始位置，你可以任意指定这些蚂蚁的朝向。蚂蚁会以每秒 $1$ 单位的速度爬行，两只蚂蚁相遇后会转向，问所有蚂蚁掉下木棍的最短/最长时间。

### 题解

只需要想通一个点：我们只关心时间，不关心是哪只蚂蚁，因此所有蚂蚁都是没有区别的。

考虑两只蚂蚁相遇时会发生什么：

假设蚂蚁A在位置 $1$ 向右爬行，蚂蚁B在位置 $5$ 向左爬行，那么把这两只蚂蚁接下来数秒的位置记录下来。

| 时刻  | $0$  |  $1$   |  $2$   |  $3$   |  $4$   |
| :---: | :--: | :--: | :--: | :--: | :--: |
| 蚂蚁A |  $1$   |  $2$   |  $3$   |  $2$   |  $1$   |
| 蚂蚁B |  $5$   |  $4$   |  $3$   |  $4$   |  $5$   |

假设没有对方，那么两只蚂蚁的位置如下：

| 时刻  |  $0$   |  $1$   |  $2$   |  $3$   |  $4$   |
| :---: | :--: | :--: | :--: | :--: | :--: |
| 蚂蚁A |  $1$   |  $2$   |  $3$   |  $4$   |  $5$   |
| 蚂蚁B |  $5$   |  $4$   |  $3$   |  $2$   |  $1$   |

前面说过，所有蚂蚁没有区别，因此这两种运动轨迹对于答案是没有影响的。

也就意味着，**蚂蚁相遇等价于互相穿过**。

想到这里，这道题就解决了。

### Code

```cpp
#include<cstdio>
#include<algorithm>
#include<math.h>
#include<cstring>
using namespace std;

int n,m,maxn,minn,x,y;

int main(){
	int k;
	scanf("%d",&k);
	while(k--){
		maxn=-99999999;minn=-99999999;
		scanf("%d%d",&n,&m);
		for(int p=1;p<=m;p++){
			scanf("%d",&x);
			x=max(x,n-x);//向左的运动时间与向右的运动时间比较
			y=min(x,n-x);
			maxn=max(x,maxn);minn=max(y,minn);
		}
		printf("%d %d\n",minn,maxn);
	}
	return 0;
}
```
