## AT_nikkei2019ex_h 8^kゲーム 题解

### 题目思路：
通过凑数思想可以知道找必胜策略时要与对方凑一个数，对于题目中取 $8^k$ 个的情况，我们可以找距离 $8$ 最近的数字 $9$ 作为凑必胜策略的值，通俗点讲，就是与对手每回合一直凑 $9\times k$ 个，一直取到最后剩下的不及 $9$ 个时，也就是 $n$ 对 $9$ 取余得到的结果来分析。

- 当 $n$ 对 $9$ 取余为 $0$ 时，刚好凑完，从题目中得知青木君为后手，则他刚好取完最后一个，高桥君没有必胜策略，输出 `Lose`。

- 当 $n$ 对 $9$ 取余为 $2$、$4$、$6$ 时，最后的那 $2$、$4$、$6$ 个都是对手先取，并且最后一轮对手只能取 $1$ 个，最后一个为青木君所取，高桥君没有必胜策略，输出 `Lose`。

- 当 $n$ 对 $9$ 取余为  $1$、$3$、$5$、$7$、$8$ 时，对手也可以使用一样的方法让青木君最后一轮取不到最后一个，高桥君有必胜策略，输出 `Win`。

### 代码：
```cpp
#include<bits/stdc++.h>
#define int long long //十年OI一场空，不开 long long 见祖宗
using namespace std;
signed main()
{
	int n;
	cin>>n;
	if(n%9==0||n%9==2||n%9==4||n%9==6)
	{
		cout<<"Lose";
	}
	else
	{
		cout<<"Win";
	}
	cout<<endl; //岛国题记得换行！！！
	return 0; //好习惯
}
