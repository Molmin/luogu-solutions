### 思路：
我们都知道，在必胜策略中，要想必胜，必须与对方一直凑一个数，只要控制好这个数的值，就可以是必胜策略。

所以对于一次取 $n = 8 ^ k$ 个的情况，我们很容易将数 $p = n\bmod (8 + 1)$ 即 $p = n\bmod 9$ 作为一个判断参数。然后进行枚举即可：
 
- 若 $p = 0,2,4,6$，显然青木君必胜。因为他可以每一次都和对手凑 $9 \times k$ 一组，而最后的 $0,2,4,6$ 必须有对手先取，且只能每次取 $1$ 个，显然后手青木君会获胜。

- 同理，若 $p = 1,3,5,7,8$，先手也可使用相同的方法，使得青木君必须陷入只能去选 $1$ 的僵局，所以青木君会输。

#### 综上所述：

- 若 $p = 1,3,5,7,8$，高桥君胜。
- 若 $p = 0,2,4,8$，高桥君输。

### 代码：
```cpp
#include<iostream>
using namespace std;
int main(){
	long long a;//要开long long
	cin >> a;
	int p = a % 9;
	if(p == 0 || p == 2 || p == 4 ||p == 6)
		cout << "Lose" << endl;
	else
		cout << "Win" << endl;
	return 0;
}
```
