### 1 结论

如果 $n$ 是奇数，那么黑棋可以赢白棋。最优方案一定是所有的移动都与中心线对称。

如果 $n$ 不是偶数，白棋可以把他的皇后放在 $(1,2)$ 上（因为 $(1,1)$ 不行，所以 $(1,2)$ 是字典序最小的位置），并且对称地去下棋。 

### 2 证明

#### 2-1 为什么奇数时黑赢

因为 $n$ 是奇数的时候，白怎么走，黑的走法就关于 $x=\dfrac{n+1}{2}$ 这条中轴线对称。

白向右 $x$ 格，黑就向左 $x$ 格，白向下 $x$ 格，黑就向下 $x$ 格，白向左下 $x$ 格，黑就向右下 $x$ 格。

这个时候，一旦白要吃 $x=\dfrac{n+1}{2}$ 上的兵，就会立马被黑吃掉，因为当白走到 $(x,\dfrac{n+1}2)$ 的时候，镜像地，黑也正好到 $(x,\dfrac{n+1}2)$。

然后白皇后就被吃掉了。

#### 2-2 为什么偶数时白赢

当 $n$ 是偶数的时候，总共就有 $n^2-2$ 个棋子。

接下来，白下到 $(1,2)$，这个时候，黑怎么走，白的走法就关于 $x=\dfrac{n+1}{2}$ 这条中轴线对称。

因为 $2\,|\,n$，所以 $2\,|\,(n^2-2)$。所以最后黑就没有子吃了，白就赢了。

没有比 $(1,2)$ 字典序更小的，已经提及过。

### 3 代码

```cpp
#include <bits/stdc++.h>
void YN(bool x) { puts(x ? "black" : "white\n1 2"); }
signed main() { int n; std::cin >> n; YN(n % 2); }
```
