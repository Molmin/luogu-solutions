
公式：$n*(n-1)/2*(n-2)/3*(n-3)/4$

思路楼下的都讲过了，但是P党必须要注意场宽。

在pascal中，除法是$/$，在保证整除的情况下，建议大家用div，因为返回值为整数，但是$/$返回的是实数，在输出的时候必须要注意场宽了，如果想要将一个实数输出，有两种方法，设s为实数，先介绍第一种：

```pas
writeln(s:0:0);
```
第二种

```pas
writeln(trunc(s));
```
下面的代码展示的是第一种

```pas
var
  n:longint;
begin
  readln(n);
  writeln(n*(n-1)/2*(n-2)/3*(n-3)/4:0:0);
end.
```
