### 找规律

通过画图可发现：

```cpp
3:0；
4:1；    ->1；
5:5；    ->4；    ->3；
6:15；  ->10；  ->6；    ->3；
7:35；  ->20；  ->10；  ->4；  ->1；
8:70；  ->35；  ->15；  ->5；  ->1；
```
然后，根据初中所学的找规律的相关内容，可以解出一个一元四次方程：(n^4-6\*n^3+11\*n^2-6\*n)/24，然后，进行化简得：

n\*（n-1）/2\*(n-2)/3\*(n-3)/4,然后就完成了。

代码：

```cpp
#include<cstdio>
#include<iostream>
using namespace std;
unsigned long long n;
int main(){
    cin>>n;
    cout<<(n*(n-1)/2*(n-2)/3*(n-3)/4)<<endl;
    return 0;
}
```
