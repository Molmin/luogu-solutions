此题很简单

通过打表，前4项分别是：

$$a[1]=1;$$


$$a[2]=2;$$

$$a[3]=4;$$

$$a[4]=6;$$


我们得出一个公式




 $$a[i]=2a[i-1]-2a[i-3]+a[i-4];$$

~~嗯，得出这个结论应该不会还做不出来了吧~~

## AC CODE：


```cpp
#include<iostream>
int n,cnt,m;
int a[1000000]{0,1,2,4,6};//打表
int main(){
	scanf("%d",&n);//输入
	for(int i=5;i<=n;i++){//循环算出数值
		a[i]=2*a[i-1]-2*a[i-3]+a[i-4];//套公式
   	}
	printf("%d",a[n]);//输出a[n]的值
   	return 0;//华丽的结束
}

```
