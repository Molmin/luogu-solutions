这道题是一道**找规律**题,不需要打表呀。

我们先列出前$10$项的值:$1\ \ 2\ \ 4\ \ 6\ \ 9\ \ 12\ \ 16\ \ 20\ \ 25\ \ 30$ 

我们可以发现:

$9=6*2-2*2+1$

$16=12*2-6*2+4$

$30=25*2-16*2+12$

因此我们可以总结出：$a[i]=a[i-1]*2-a[i-3]*2+a[i-4]$

$Code:$

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a[105];
int main(){
    scanf("%d",&n);
    a[1]=1;a[2]=2;a[3]=4;a[4]=6;
    for(int i=5;i<=n;i++)
        a[i]=2*a[i-1]-2*a[i-3]+a[i-4];
    printf("%d\n",a[n]);
    return 0;
}
```
