这道题直接算一算，就可以找到如下的公式：
$$A_1=0$$
$$A_2=1$$
$$A_3=2$$
$$A_4=4$$
$$A_4=6$$
$$A_i=2*A_{i-1}-2*A_{i-3}+A_{i-4}(i\geq5)$$

所以，直接递推就行了。

## CODE
```cpp
#include<stdio.h>
int main(void) {
	int n;
	scanf("%d", &n);
	int a[101] = {0, 1, 2, 4, 6};
	for(int i = 5; i <= n; ++i)
		a[i] = (a[i-1] << 1/*左移更快*/) - (a[i-3] << 1) + a[i-4];
	printf("%d\n", a[n]);
	return 0;
}
```
