# 题目大意
输入 $t$ 组数据，

每组数据有四个整数 $n, x, a, b$。

$n$ 代表队列的长度，

$x$ 表示能与相邻同学交换的次数，

$a$ 和 $b$ 分别表示两个竞争者所在的位置。

求交换后 $|b - a|$ 的最大值。

# 题意分析
不难发现，若想 $a$ 与 $b$ 距离最长，

只要使 $a$ 尽量靠前， $b$ 尽量靠后就行了。

所以他们的最远距离就是 $(b - a) + x$ （默认 $a$ 比 $b$ 靠前）

通过观察，我们还发现 $a$ 与 $b$ 的最远距离最远是 $n - 1$。

也就是说， $a$ 与 $b$ 的最远距离也不会远于 $n - 1$。

# Code
```cpp
#include <cstdio>
int main(){
  int t;
  scanf("%d",&t);
  for (int i = 1; i <= t; i++){
    int n, x, a, b, t1;
    scanf("%d%d%d%d",&n,&x,&a,&b);
    if (a > b){//如果 b 比 a 靠前，那么就交换它们的位置
      t1 = a;
      a = b;
      b = t1;
    }
    if (b - a + x > n - 1){//如果 b - a + x 远于 n - 1
      printf("%d\n", n - 1);
    }
    else {
      printf("%d\n", b - a + x);
    }
  }
  return 0;
}
```
