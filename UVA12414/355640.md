## 题目大意

你需要完成一个算缘分的程序，具体步骤如下。

### STEP 1:
 
将两人的姓名改成缩写，例如 “张三” 为 $zs$，然后再将两个缩写连接起来。
 
当然，在这道题里已经给定好了。

### STEP 2：

确定一个 $ST$ 值，将字符串转换为一个数字串。

如果 $ST$ 为 $1$ ，则字母 `A` 为 $1$ ,字母 `B` 为 $2$ ，以此类推。

### STEP 3：

每次将相邻的两个数字相加，然后取个位，直到剩下一个三位数。

如果此时这个三位数如果为 $100$ ，则停止计算。

否则最后再进行一次相加的操作，最后剩下的数就是两人的缘分。



------------

这道题有多组数据。

每组数据会给定一个字符串，如果 $ST$ 为 $X$ 时计算出来的缘分为 $100$ （注意 $X$ 要尽可能的小），则输出 $X$ 。

如果 $ST$ 在 $10000$ （ $ST$ 不能小于等于 0 ） 以内但缘分都无法为 $100$ 那就输出 `:(` 。

## 解题思路

因为数据很小，所以可以直接枚举 $ST$ ，然后模拟一下，但是这道题涉及到字符串的一些操作。

提到字符串，就得提 ASCIL 码。

ASCIL 相当于把每个字符都给了一个编号，例如字符 `0` 的编号就是 48 ，`A` 的编号是 65 。

当然 ASCIL 码里头还有其他的符号，有兴趣的可以去百度一下。

以下是字符串的一些基本操作。

```cpp
string a,b,c; //注意string类型下标默认从0开始

int k=a.size(); //表示求出a的大小，赋值给k。

a.append(x,c); //表示把x个c字符插入到a的末尾（注意此时c只能是字符）。

c=a+b; //表示把a和b拼接起来，赋值给c。

reverse(a.begin(),a.end()); //表示把a翻转。

int k=a[0]-'0'; //这里就用到了ASCIL码，假设a[0]为 '6' ，'6'的编码是54,

                //'0'的编码是48，54-48=6，这样就从字符的'6'转换为了数字6。

```

除了以上操作，还有一个问题，在第二步时需要将数字转换为字符串。

此时可以每次将个位的数插入到字符串里，然后再将数除以 10 。

最后将字符串翻转，就可以将数字转换为字符串了。

## 代码

以下是AC代码

```cpp
#include<iostream>
#include<string>
using namespace std;
string a;
string fenjie(int now)
{
	string tmp="";
	while (now)
	{
		tmp.append(1, now % 10 + '0');
		now /= 10;
	}
	reverse(tmp.begin(), tmp.end());
	return tmp;
}
bool p(string n, int now)
{
	string k = "";
	for (int i = 0; i < n.size(); i++)//题目中的第二步，fenjie函数用于将数字转换为字符串。
	{
		k = k + fenjie(now + (n[i] - 'A'));
	}
	for (int i = 1; k.size() != 3; i++)//第三步，只要当字符串的大小不到3就可以一直操作下去。
	{
		string tmp = "";
		for (int j = 0; j < k.size() - 1; j++)
		{
			tmp.append(1, ((k[j] - '0') + (k[j + 1] - '0')) % 10 + '0');
		}
		k = tmp;
	}
	int ans = (k[0] - '0') * 100 + (k[1] - '0') * 10 + (k[2] - '0');
	if (ans == 100)//这里有个小技巧，最后其实不用确定缘分值，只要保留三位然后看是不是100就行了。
		return true;
	else
		return false;
}
int main()
{
	while (cin >> a)
	{
		bool tmp = true;//这个变量用于记录是否找到了答案。
		for (int i = 1; i <= 10000; i++)
		{
			if (p(a, i))//这里的函数用于判断当前ST值算出来的缘分是否为100。
			{
				cout << i << endl;
				tmp = false;
				break;
			}
		}
		if (tmp)
			cout << ":(" << endl;
	}
	return 0;
}
```
