题意：简单来说，重新排列数列，使每个数不等于相邻两数的平均数。


------------
首先，显然是需要先排个序。要想每个数都不等于相邻两数的平均数，那么可以是这样的排列：

小 大 小 大 小……（小代表较小的数，大代表较大的数）

那么就很显然了，只需把数列**按从小到大分成两半，然后交叉在一起就行了**。因为这样要么相邻的数都大于这个数，要么都小于这个数，平均数一定是不等于这个数的。


------------
代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int t,n,a[55];
int main() {
	cin>>t;
	while(t--) {
		cin>>n;
		for(int i=1;i<=2*n;i++)cin>>a[i];
		sort(a+1,a+2*n+1);
		for(int i=1;i<=n;i++)cout<<a[i]<<" "<<a[i+n]<<" ";
		cout<<endl;
	}
}
```
