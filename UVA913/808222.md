看到之前的题解里没有用我这种方法的，来为大家介绍一下我的思路

首先分析题目，观察到 $1 < n < 10^9$ ,如果暴力的话肯定超时

既然没法用暴力，那就只能试着找规律

大家一定会想到先找出数据对应第几行，那么我们先来看几组例子：

设行数为 $ s $ 

当 $ n = 3,s = 2 $

当 $ n = 5,s = 3 $

当 $ n = 7,s = 4 $

通过几组例子，不难看出，$s=\frac{n+1}{2}$

既然题目要求的是最后三个数的和，那么我们可以算出最后一个，其他两个自然求出

我们还是以举例的方式来找规律：

设第 $s$ 行的最后一个数为 $t$ ：

当 $ n = 3,s = 2,t = 7 $

当 $ n = 5,s = 3,t = 17 $

当 $ n = 7,s = 4,t = 31 $

可以得出，$t = \frac{n*(n+1)}{2}+(s-1)$

既然最右边的数求出来了，那么它左边的两个数就是 $t-2$ 和 $t-4$

设最终答案为 $ans$

$ans=[\frac{n*(n+1)}{2}+(\frac{n+1}{2}-1)]*3-6$

由于数据太大，记得开 **long long**

```cpp
#include<iostream>
#define int long long
using namespace std;
signed main(){
	int n;
	while(cin>>n){
		int s=(n+1)/2;
		int t=n*(n+1)/2+(s-1);
		cout<<(t-4)+(t-2)+t<<endl;
	}	
	return 0;
}
```
