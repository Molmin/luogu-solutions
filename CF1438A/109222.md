### 题目大意

 输入 $t$ 个数，对于每次输入的数 $n$ ，求一个序列满足 **对于它的每个子序列各个元素之和能被其长度整除**。
 
### 解决办法
 
 仔细思考一下，题目中各个元素之和是**被除数**，子序列长度是**除数**。
 
 而构造序列时，其长度是**已知的**，所以我们要使子序列中各个元素之和等于其长度的**倍数**。
 
 于是乎，每行只要输出 $n$ 个相同的数即可...
 
### Code

```cpp
#include <iostream>
using namespace std;
int t,n;
int main()
{
	cin>>t;
	while(t--)
	{
		cin>>n;
		while(n--) cout<<1<<" ";
		cout<<endl;
	}
	return 0;
}
```
