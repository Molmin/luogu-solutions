## 题意

输出 $1$ 至 $n$ 中，所有满足数字中含有 $2$、$0$、$1$、$9$ 的数之和。

## 思路

按题意模拟即可。

``to_string(k)`` 表示将整型 $k$ 转为字符串，$\texttt{c++14}$ 新语法。自动转为字符串后枚举位数模拟。

## 代码

仅放出部分代码。

```cpp
for(int i=1;i<=n;i++){
    s=to_string(i);
    for(int j=0;j<s.length();j++){
        if(s[j]=='2'||s[j]=='0'||s[j]=='1'||s[j]=='9'){
            res+=i;
            break;
        }
    }
}
```
