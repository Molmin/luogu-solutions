### 思路：
既然下的棋子只会影响直线中间的棋子，那么我们需要讨论的就是两端的棋子。

这样，我们根据两端的棋子就有 $3$ 种情况。

情况 $1$：当两端都是```o```时，由于```x```方无法改变中间的棋子，所以```o```方必胜。

情况 $2$：当两端都是```x```时，由于```o```方无法改变中间的棋子，所以```x```方必胜。

情况 $3$：当一端为```o```，一端为```x```时，由于```o```方为先手，所以```o```方必胜。

### 代码：
```
#include <cstring>
#include <iostream>

using namespace std;

int main() {
	string s;
	cin >> s;
	if (s[0] == 'x') {
		if (s[s.length() - 1] == 'o') {
			puts ("o");//情况3 
		}
		
		else {//情况2 
			puts ("x");
		}
	}
	
	if (s[0] == 'o') {
		if (s[s.length() - 1] == 'o') {
			puts ("o");//情况1 
		}
		
		else {
			puts ("o");//情况3 
		}
	}
	
	return 0;
}
```
