这是一道入门的题目，需要我们耐心的分析。

首先，`o`  这一方先手，所以说只要刚开始的局面收尾有一个是 `o` ，那么因为它先手，它可以从另一边围堵上来，于是  `o` 赢了。

否则的话，那么两边都是 `x`，两边直接由 `x` 进行围堵，整个棋盘便都是 `x`，那样 `x` 肯定会赢。

所有的情况不可能在以上两种条件之外，所以说我们的分类讨论一定是正确的。有了理论的基础，代码实现便不难了。

```cpp
#include <bits/stdc++.h>
using namespace std;
string s;
int main() {
	cin >> s;
	if (s[0] == 'o' || s[s.size() - 1] == 'o') puts("o");
	else puts("x");
}
```
