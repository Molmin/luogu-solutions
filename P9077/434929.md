答案不超过 $2$，因为你可以删到只剩两个点然后把它们都改成 $1$。

有两个 $1$ 显然可以按照上面的策略来，答案是 $0$。

只有一个 $1$，必定需要改一个，因为合法的树至少有两个度为 $1$ 的节点。反正：如果假设只有一个度为 $0$，那只能说是叶子，则根必须有两个以上的子树，此时叶子个数超过一个。

一个也没有是显然不合法的，要改两个。

```cpp
void solve() {
	cin >> n;
	for (int i = 1; i <= n; i++)
		cin >> x, tot += (x == 1);
	cout << 2 - min(2, tot) << "\n2\n1 2\n";
}
```
