## 文字题解

### 题意简述

给出 $n,k$ 和一个 $n$ 个元素的数组 $a$，将 $a_k$ 改为 $-a_k$ 后，求 $\sum a_i$。

### 问题分析

作为考察循环结构的题目，可以在输入过程中，依次判断输入的是否为第 $k$ 个数，如果是，减去输入的数，否则加上输入的数。

```cpp
cin >> n >> k;
for(int i = 1, x; i <= n; i++) {
    cin >> x;
    if(i == k) sum -= x;
    else sum += x;
}
cout << sum << "\n";
```

当然，也可以先用数组储存所有的数据，最后处理 $a_k$。

注意：sum 需要使用 long long 类型。
