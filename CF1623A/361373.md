## 题目解读
一个扫地机器人先往右下角走，碰墙后会反弹，问何时扫地机器人会和脏物在同一行或者同一列。
## 思路
本题可以分为两个小问题，分别是二者何时再同一行和二者何时在同一列，输出最小值即可。

当扫地机器人的横坐标小于或等于脏物的横坐标，则二者在同一列所需的时间为脏物的横坐标减去机器人的横坐标；当机器人的横坐标大于脏物的横坐标，所需的时间为房间长度的二倍减去机器人的横坐标再减去脏物的横坐标。

机器人到脏物再同一行的时间得到的原理与此相同。

## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
    int t;
    cin>>t;
    while(t--){
        int n,m,x1,x2,y1,y2,ans=10000;
        //x1,y1,x2,y2表示机器人和脏物的横纵坐标。
        cin>>n>>m>>x1>>y1>>x2>>y2;
        if(x1==x2||y1==y2) ans=0;//特判，不这么做也可以
        else{
            if(x1<x2) ans=x2-x1;
            else ans=2*n-x2-x1;
            //得到二者同一列所需的时间
            if(y1<y2) ans=min(ans,y2-y1);
            else ans=min(ans,2*m-y1-y2);//取最小值
        }
        cout<<ans<<endl;
    }
    return 0;
}
```
