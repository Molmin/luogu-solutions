很水的一道题。
## 题意：
有 $N$ 条蛇，平均重 $W$。
问最多可以有多少条蛇大于平均重量。

## 分析：
平均每条蛇重 $W$，总数为 $N$，那么就是要使尽量多的蛇的重量大于 $\frac{W}{N}$。我们可以看出，那么不管 $N$ 为多少，最大数量始终为 $N-1$。

为什么呢？我们可以把一条蛇的一部分重量分给另外 $N-1$ 条，这样就可以使在总重量不变的情况下，使数量达到最大，这时，就有一条蛇小于平均重量，$N-1$ 条蛇大于平均重量。

所以不管有多少条蛇，最多可以有 $N-1$ 条蛇大于平均重量。


## 代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int a,b;
	cin>>a>>b;
	cout<<a-1<<endl;
	return 0;
}
```
