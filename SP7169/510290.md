## 这道题目是一个简单的贪心算法问题
### 下面是 ~~萌新写的第一篇~~ 题解：
首先，我们可以将每个人想要的披萨饼转化为分数的形式。对于 $\frac{1}{4}$ 的披萨饼，我们可以表示为 $\frac{1}{4}$，对于 $\frac{1}{2}$ 的披萨饼，我们可以表示为 $\frac{2}{4}$，对于 $\frac{3}{4}$ 的披萨饼，我们可以表示为 $\frac{3}{4}$。

然后，我们可以统计每种披萨饼的需求个数。假设一共有 $n$ 个人，需要 $\frac{1}{4}$ 的披萨饼的人数为 $x$，需要 $\frac{1}{2}$ 的披萨饼的人数为 $y$，需要 $\frac{3}{4}$ 的披萨饼的人数为 $z$。

接下来，我们需要计算最少需要多少个完整的披萨饼。对于 $\frac{1}{4}$ 的披萨饼，每个人需要 $\frac{1}{4}$ 个披萨饼，所以总共需要的披萨饼个数为 $\frac{1}{4} x$。对于 $\frac{1}{2}$ 的披萨饼，每个人需要 $\frac{1}{2}$ 个披萨饼，所以总共需要的披萨饼个数为 $\frac{1}{2} y$。对于 $\frac{3}{4}$ 的披萨饼，每个人需要 $\frac{3}{4}$ 个披萨饼，所以总共需要的披萨饼个数为 $\frac{3}{4} z$。

最后，我们将上述三种披萨饼的个数相加，再加上一个完整的披萨饼（给 Abotrika 吃），就是最终需要订购的最小数量的披萨饼。

下面是相应的 C++ **AC代码** 实现：

```cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
	int n;
	cin >> n;

	int x = 0, y = 0, z = 0;

	for (int i = 0; i < n; i++)
	{
		string pizza;
		cin >> pizza;

		if (pizza == "1/4")
		{
			x++;
		}
		else if (pizza == "1/2")
		{
			y++;
		}
		else if (pizza == "3/4")
		{
			z++;
		}
	}
	int count = 0;
	// 计算每种披萨饼的个数
	count += z;
	x -= z;
	count += y / 2;
	y %= 2;
	count += y;
	x -= 2 * y;
	if (x > 0)
	{
		count += (x + 3) / 4;
	}
	cout << count + 1 << endl;
	return 0;
}
```

这个代码根据题目描述，使用变量 $x$、$y$、$z$ 分别记录需要 $\frac{1}{4}$、$\frac{1}{2}$ 和 $\frac{3}{4}$ 披萨饼的人数。然后根据需求个数计算最小数量的披萨饼。

请注意，在输出结果时，我们需要加上一个完整的披萨饼（给 Abotrika 吃），所以最终结果为 $count + 1$。
