**来一个没有打表的题解。**

首先，如果是周：

同余分析（就是小学奥数经常干的那种事）可知2016第一天周五最后一天周六。

想象把2016去掉这两天。

那么剩下周六->周五正好是52个整周。

因此星期的话周五或周六输出53否则52.

接着，如果是月：

前29天每个月都有（注意2016是闰年），所以小于等于29的直接输出12.

30号只有2月没有，输出11.

31号······数一数吧：1,3,5,7,8,10,12七个月！

所以31是输出7.

好了分析完毕直接上代码吧，应该很简单了。

Python3.
```py
s=input().split()#哈哈，自动过滤of
k=int(s[0])
if s[2]=='week':
    if k==5 or k==6:print(53)
    else:print(52)
else:
    if k<=29:print(12)
    if k==30:print(11)
    if k==31:print(7)
```
