# 一个循环就可以AC！！！

解法也很简单。

首先，矩阵是n*m个数，用一个循环输入就好了，a就是输入的数。

接着，用一个条件判断a是否为1，是则计数器s累加。

再接着，判断一行是否结束（i%m==0）。若果结束则判断是否为最优值。

最后，输出最优值x就可以了，只要弄懂了就不难。

# 所有日本题一定要换行！！！

好了，接下来是代码，谢谢大家。

```cpp
#include<bits/stdc++.h>
using namespace std;
int n,m,a,s,x;
int main(){
	cin>>n>>m;
	for(int i=1;i<=n*m;i++){
		cin>>a;
		if(a)++s;
		if(i%m==0)x=x>s?x:s,s=0;
	}
	cout<<x<<endl;
	return 0;
}
```
