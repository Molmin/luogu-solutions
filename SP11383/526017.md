一道数学题。

### 结论：

$2^0+2^1+...+2^{n-1}=2^n-1$。

### 证明：

设 $2^0+2^1+...+2^{n-1}=S$，

则 $2S=2^1+2^2+...+2^n$。

$S=2S-S$

$=(2^1+2^2+...+2^n)-(2^0+2^1+...+2^{n-1})$

$=2^n-2^0$

$=2^n-1$


------------
由于数据范围大到 $2^{500}$，c++ 或 pascal 代码量太大，所以我们使用 python 或 ruby 存。

k=int(input())

    x=int(input())
    print((2**(x+1)-1)%mod)
