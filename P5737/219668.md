## 这是一道循环和分支的好题

这道题最重要的是闰年的定义，闰年分普通闰年和世纪闰年：

普通闰年:能被4整除但不能被100整除的年份为普通闰年。（如2004年就是闰年，1900年不是闰年）；

世纪闰年:能被400整除的为世纪闰年。（如2000年是世纪闰年，1900年不是世纪闰年）

于是这道题的解法就是循环一遍，判断这年是不是普通闰年或者世纪闰年。答案要先输出次数，再输出年数，于是我们开一个$ans$数组记录一下。时间复杂度为$\mathcal O(n)$,$\mathcal n<1500$，没有任何超时的风险。

AC代码如下：

```cpp
#include<iostream>
using namespace std;
int ans[1500];
int main(){
	int begin,end;
	cin>>begin>>end;//输入不解释
	int cnt=0;
	for(int i=begin;i<=end;i++){
		if((i%4==0&&i%100!=0)||i%400==0){//判断这年是不是普通闰年或者世纪闰年。
			ans[cnt++]=i;//记答案,次数加一
		}
	}
	cout<<cnt<<endl;//先输出次数
	for(int i=0;i<cnt;i++){
		cout<<ans[i]<<" ";//输出答案
	}
	cout<<endl;//换行是个好习惯☺
	return 0;//华丽的结束
}
```
