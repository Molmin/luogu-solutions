### 题目大意：
你有 $1!-15!$ 的面值的钞票，你拿它们去买东西，请问最少能用几张钞票买到那个东西。（每种钞票有无数张且有多组数据）

---
### 思路：
我们可以先把 $1!-15!$ 放到一个数组里

```cpp
int a[16]={1,1,2,6,24,120,720,5040,40320,362880,3628800,39916800,479001600,1932053504,1278945280,2004310016};
```

然后每次操作都将一个标志表到比 $n$ 小的钞票种类上

```cpp
while(n<a[num])
	num--;
```

然后再将 $n$ 减到这比这种钞票的面值还小并计数，然后计数并循环至 $n=0$ 。

---
### 代码：

```cpp
#include<bits/stdc++.h>
int a[16]={1,1,2,6,24,120,720,5040,40320,362880,3628800,39916800,479001600,1932053504,1278945280,2004310016};
using namespace std;
int main(){
	int n,num,ans;
	while(cin>>n){
		num=15,ans=0;
		while(n){
			while(n<a[num])
				num--;
			ans+=n/a[num];
			n=n%a[num];
		}
		printf("%d\n",ans);
	}
	return 0;
}
```
