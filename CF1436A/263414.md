数论结论题

根据题意，我们可以发现题目让我们求的就是这个东西
$$\sum\limits_{i = 1}^n {\sum\limits_{j = i}^n {\frac{{{a_j}}}{j}} } $$

化简得到
$$= \frac{a_{[1]}}{1} + \frac{{{a_{[2]}}}}{2} + \cdots + \frac{{{a_{[n]}}}}{n} + \frac{{{a_{[2]}}}}{2} + \frac{{{a_{[3]}}}}{3} + \cdots + \frac{{{a_{[n]}}}}{n} + \frac{a_{[3]}}{3}+ \frac{a_{[4]}}{4}+ \cdots + \frac{{{a_{[n]}}}}{n}$$
$$ = {a_{[1]}} + {a_{[2]}} + \cdots + {a_{[n]}}$$

于是题目就变得非常简单，我们只要将 $a$ 数组加起来统计和，在判断是否等于 $m$ ，输出 $\texttt{yes}$ 或 $\texttt{no}$ 即可
