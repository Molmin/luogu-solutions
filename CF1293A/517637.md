### 样例分析
样例输入：
```
5
5 2 3
1 2 3
4 3 3
4 1 2
10 2 6
1 2 3 4 5 7
2 1 1
2
100 76 8
76 75 36 67 41 74 10 77
```
样例输出：
```
2
0
4
0
2
```
分析：数据有五组
* 第一组数据，一共 5 层，主角的办公室在 2 层，1，2，3 层没有餐厅，可知，最近的餐厅在第 4 层，离现在的办公室有 2 个楼层，所以输出 2。

* 第二组数据，一共 4 层，主角的办公室在 3 层，1，2，4 层没有餐厅，可知，最近的餐厅在第 3 层，离现在的办公室有 0 个楼层，所以输出 0。

* 第三组数据，一共 10 层，主角的办公室在 2 层，1，2，3，4, 5, 7 层没有餐厅，可知，最近的餐厅在第 4 层，离现在的办公室有 4 个楼层，所以输出 4。

* 第四组数据，一共 2 层，主角的办公室在 1 层，2  层没有餐厅，可知，最近的餐厅在第 1 层，离现在的办公室有 0 个楼层，所以输出 0。

* 第五组数据，一共 100 层，主角的办公室在 76 层，76，75，36，67，41，74，10，77 层没有餐厅，可知，最近的餐厅在第 78 层，离现在的办公室有 2 个楼层，所以输出 2。
### 思路分析
首先输入 $t$，是肯定的没有问题的。
```python
t=int(input())
```
下一步，是循环输入，也没有问题。    
```python
while t:
		t=t-1
        dt=input().split(' ')
        n,s,k=int(dt[0]),int(dt[1]),int(dt[2])
        ra=[i for i in range(1,dt[0])]
```
大家唯一可能有问题的是最后一行代码。这行代码以后有用，先说一下逻辑。    
这行代码等同于：
```python
ra=[]
for i in ragne(1,n):
		ra.append(i)
```
下面是输入没有餐厅的楼层。
```python
tk=input().split(' ')
for i in tk:
		ra.remove(int(i))
```
这里就用到了刚才的列表。这样经过筛选，只保留了有饭店的楼层，这也是 Python 的一大优势。   

下面就是比较时刻。这里遍历列表，比较哪个楼层离办公室最近，打擂台形式。
```python
mx=2147483647#初始化
for i in ra:
		if abs(s-i)<mx:
        mx=abs(s-i)
        if mx==0:
        	break
```
这里加入了一个优化：如果最小值为 0，那么一定是最优解，直接跳出循环。    
最后输出，不用解释。
```python
print(mx)
```
这里不用担心 $mx = 2147483647 $ 的情况，因为最大值未达到 2147483647。    
### 完整代码
根据以上思路，无注释代码如下。
```python
t=int(input())
while t:
		t=t-1
        dt=input().split(' ')
        n,s,k=int(dt[0]),int(dt[1]),int(dt[2])
        ra=[i for i in range(1,dt[0])]
		tk=input().split(' ')
		for i in tk:
			ra.remove(int(i))
            
		mx=2147483647
		for i in ra:
			if abs(s-i)<mx:
        	mx=abs(s-i)
        	if mx==0:
        		break
		print(mx)
```
