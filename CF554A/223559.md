### 解题思路：

结论是总可能的的方案数为串的长度乘以 $25$ 加上 $26$。也就是 $|s|\times 25+26$。

------------
第一篇题解给出了一个好长的证明，这里给一个非常简单的。

考虑将所有的位置分成两个部分，非最后一个和最后一个位置。

对于所有的非最后一位都放不同于后一个字符的所有字母，最后一个位置所有的字母随便放。由于每一个每一个位置放的都和后一个不一样，但可以和之前的一个重复，所以不重不漏。

这样前面的一共有 $|s|\times 25$ 个方案，最后一个额外提供 $26$ 个方案数。相加得到方案总数是 $|s|\times 25+26$。

------------
### 代码：

```cpp
#include<iostream>
#include<cstdio>
using namespace std;
string a;
int main(){
	cin>>a;
	cout<<(a.length())*25+26<<endl;
	return 0;
}
```
