## STEP 1 题目大意

给出一个小写字母字符串，在其中任意位置添加任意小写字母，问有多少种放置方法。放完相同的字符串算一种。

## STEP 2 浅谈解法

我们先来看看只要一个字母的情况。

假设只有一个字母，则可以在它前面或后面放置任意一个小写字母，共 $26\times 2=52$ 种。但是不够，因为前面放 $a$ 和后面放 $a$ 字符串是一样的，所以只算一种。即共有 $52-1=51$ 种。

找到规律了吗？每个字符串我们都可以放置在任意一个位置，也就是 $n+1$ 个位置，然后每一个位置都可以放置 $26$ 个字母，即 $26\times(n+1)$ 个情况。又因为在一个字母两侧放置跟其相同的字母字符串一样，所以需要再减去 $n$ 种（一共有 $n$ 个字母）。所以答案为 $26\times(n+1)-n$。

## STEP 3 AC 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
char s[25];
int main(){
    scanf("%s",s);
    int l=strlen(s);//获取字符串长度
    printf("%d\n",(l+1)*26-l);//套公式
    return 0;
}
```
