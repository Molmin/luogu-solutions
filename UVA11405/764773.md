# 题意简述
这道题目给定了一个 $8×8$ 的棋盘和若干个棋子的位置，其中字母 `P`$ 表示黑色兵、`K` 表示黑色车、`p` 表示白色兵、`k` 表示白色车、`.` 表示空位置。

然后它指定了如下规则：

- 黑色兵只能前进一步，到达对面时可以走成 `K`。
- 黑色车遵常象棋规则移动，但不能吃自己的棋子（即不能直接跨过白色兵）。
- 白色兵只能前进一步，到达对面时可以走成 `k`。
- 白色车遵常象棋规则移动，但不能吃自己的棋子（即不能直接跨过黑色兵）。
- 黑方先手。

要求你计算出第一个黑色兵是否能在剩下的步数中捉住白色车，如果可以则输出 `Yes`，否则输出 `No`。

请注意：
- 棋子位于棋盘的某个位置以 $\text{.}$ 表示时两边之间存在某些未被占用的格子；
- 本题并不保证棋盘输入的合法性，即：可能出现同颜色的兵堵住同侧车等奇怪的情况，此时按照题意能合法按此摆放。

# 题目思路
这是一道基础的模拟题目，主要需要进行如下操作：

1. 解析棋盘和棋子位置：我们可以将输入的棋盘看做一个字符矩阵，然后记录黑白方兵和车的位置。

2. 进行黑兵的移动并判断是否能够吃掉白车：根据题目指定的规则，不断更新黑兵的位置信息，并检查黑兵是否位于白车的同一行或同一列上，如果是的话就说明黑兵成功抓住了白车，直接返回 `Yes` 即可。

3. 进行白车的移动：在黑兵没有获胜之前，按照题目指定的规则进行白车的移动，直到达到行动次数的限制或者黑兵已经被吃掉。

4. 根据黑兵的状态输出答案：如果黑兵最后没有成功捕捉到白车，则输出 `No`。
