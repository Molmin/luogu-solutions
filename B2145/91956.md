提供一种循环做法。

以样例为例：

```
31859 3
```
 
对于数 $n=31859$ 执行以下操作：

```
n/=10;

```
$n$ 变成了 $3185$。

再来一次。

$n$ 变成了 $318$。

此时 $n$ 的个位数即为答案。

所以,```n/=10;```这个操作相当于从右边起擦掉这个数的最后一位，我们只需要擦掉 $n$ 的后 $k-1$ 位，此时 $n$ 的个位数即为答案。

# 代码：
```cpp
#include<bits/stdc++.h>//万能头文件
using namespace std;
#define Orz cout<<"stO %王队% Orz"<<'\n';
int n,k;
int main()
{
	cin>>n>>k;
	for(int i=1;i<k;++i)	n/=10;
	cout<<n%10;  //输出个位数
	return 0; //好习惯
}

```
