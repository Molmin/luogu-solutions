## 1、基本的快速幂公式

#### $a$^$n$有两种情况，一种是$n$为偶数，一种是$n$为奇数。
#### 我们都知道：$a^n$+$a^m$=$a^{m+n}$。
#### 因此，$n$为偶数时$a^n$=$a^{n/2}*a^{n/2}$。
#### 而$n$为奇数时只需要再乘一个$a$就可以了。
### 下面奉上代码：
```cpp
if(n%2==0)    return (c*c)%k;
if(n%2==1)    return (c*c*(a%k))%k;
```

## 2、快速幂函数代码展示
```cpp
int Pow(int a,int n)
{
    if(n==0)    return 1;
    if(n==1)    return a;
    int c=Pow(a,n/2);
    if(n%2==0)  return c*c;
    return c*c*a;
}
```
