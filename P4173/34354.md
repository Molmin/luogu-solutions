具体来说，设文本串对应的多项式为 $f$，模板串对应的多项式为 $g$ 。

每次选出 $1$ 种字符 $k$，将文本串对应字符 $k$ 的位置与 $*$ 的位置均设为 $1$，将模板串对应字符 $k$ 的位置设为 $1$，再将 $g$ 反转，与 $f$ 做卷积。

每次卷积之后把每个位置匹配的数量累加（下面是累加到 `o[]` 里），然后搞一通即可。

```cpp
for(i = s = 0;i < m;i ++)
    s += a[i] == '*';

for(i = t = 0;i < n;i ++)
    t += o[i] + s == m;
printf("%d\n", t);
for(i = 0;i < n;i ++)
    if(o[i] + s == m)
        printf("%d ", i + 1);
        ```
