完全没有必要15个if-else嘛=-=

讲真、

```cpp
8 = 0b1000
4 = 0b0100
2 = 0b0010
1 = 0b0001
```
于是原来的超级难受的if-else就变成了这样的代码：
```cpp
int s = 0b1010;//s: 1 0 1 0 
if(s & 1)      //         ^
    goWest();  //         off!
s >>= 1;       //s: 1 0 1 # 
if(s & 1)      //       ^
    goNorth(); //       on!
s >>= 1;       //s: 1 0 # #
if(s & 1)      //     ^ 
    goEast();  //     off!
s >>= 1;       //s: 1 # # #
if(s & 1)      //   ^
    goSouth(); //   on!
```
