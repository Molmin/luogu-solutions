这一题的主要坑点就在:

> "元旦快到了，校学生会让乐乐负责新年晚会的纪念品发放工作。为使得参加晚会的同学所获得的纪念品价值相对均衡，他要把购来的纪念品根据价格进行分组，但每组最多只能包括**两件纪念品**， 并且每组纪念品的价格之和不能超过一个给定的整数。为了保证在尽量短的时间内发完所有纪念品，乐乐希望分组的数目最少。"

只有两件，所以不用设数组来存已经给了多少价值的物品（~~我就这么干过~~）

## 下面祭上代码：

```cpp
#include<iostream>
#include<cstdio>
#include <algorithm>
using namespace std;
int a[30010];
int main()
{
	int n,mx,ans=0,x,y;//ans总组数，x，y为下标，n为个数，mx为最大值 
	cin>>mx>>n;//读入不解释 
	for (int k=0;k<n;k++)scanf("%d",&a[k]);
	sort (a,a+n);//从小到大排序
	x=0;y=n-1;//x从价值最小的物品开始，y从价值最大的物品开始 
	while (x<=y)//只要物品没有分完就循环 
	{
		if (a[x]+a[y]<=mx)//如果可以分给这个组而不引起其它组想打你的欲望(小于mx)
		{
			x++;y--;ans++;//就给这个组 
		}
		else
		{
			ans++;y--;//放不下就只放一个 
		}
	}
	cout << ans;//输出 
	return 0;//停止程序 
}
```
