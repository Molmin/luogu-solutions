### 第一题：小蓝与钥匙

A说白了就是错排，要 $28$ 个孩子中选 $14$ 个孩给他们自己房间的钥匙，满足 $14$ 个孩子分到的是自己房间钥匙的条件。但是要求剩下 $14$ 个孩子分到的不是自己房间的钥匙，直接 $2$ 部分一乘就得到结果，得到结果为$1286583532342313400$。
```
#include<iostream>
using namespace std;
int main() {
    string ans [] = {
        "1286583532342313400", // 双引号中替换为 A 题的答案
        "The answer of task B", // 双引号中替换为 B 题的答案
    };
    char T;
    cin >> T;
    cout << ans[T - 'A'] << endl;
    return 0;
}
```

### 第二问 排列距离
B就是按康拓展开，分别求得给定的两个排列是第几个排列，设为第 $b$ 和第 $c$ 个排列，并设最后一个排列  `rqponlkjihgfedcba ` 是第 $e$ 个排列，则取 $c-b$ 和 $e-c+b$ 的最小值就是所求的答案。所以，输出结果为： $106148357572143$。

由题目给的模板可得代码为：

```
#include<iostream>
using namespace std;
int main() {
    string ans [] = {
        "1286583532342313400", //对应第一题结果
        "106148357572143" ,//对应第二题结果
    };
    char T;
    cin >> T;
    cout << ans[T - 'A'] << endl;
    return 0;
}
```
