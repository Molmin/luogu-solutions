月赛T1，主要是画图题

简化题意：在一个正方形中，若一点与四个顶点的连线将正方形划分成了 $4$ 个面积比为 $a:b:c:d$ 的区域则为合法点，求此类合法点的个数。

由图可知：四个三角形等底，所以面积之比实际就等于高之比。而高，上下两个三角形的高相加 $=$ 左右两个三角形高相加。所以，判断存不存在合法点的方法就是判断四个比的参数中，是否存在两个相加 $=$ 另两个相加。

对于左右两个三角形的高之比，因为比是没有顺序的，所以$a:b$ 等价于 $b:a$ ，也就是说除了让左右两个三角形高之比为a:b的点 $E$ 外，还会有一点 $F$ 关于正方形的对称轴对称。

$Tips:$ 如果 $a:b=b:a$ ，此时也有 $c:d=d:c$ 那么 $E$ 和 $F$ 重合，即：只有一个点。

上下两个三角形的情况同理。

然后观察样例，进行细分。

（ 因为比例无序，所以我们先排序，$a≤b≤c≤d$ ）

如果 $a:d=b:c$ ，那么上下左右颠倒后等价，此时总共有左上，左下，右上，右下，共 $4$ 个点。（ $a:d$ 横向两个，$b:c$ 纵向两个，总共：$2×2=4$ ）

如果 $a:d≠b:c$ ，那么上下左右可以颠倒，点数 $×2$ .共 $8$ 个点。

$Code:$
```cpp
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
ll s[10];
int main()
{
	int n;
	cin>>n;
	while(n--)
	{
		cin>>s[1]>>s[2]>>s[3]>>s[4];
		sort(s+1,s+5);
		if(s[1]+s[4]!=s[2]+s[3])  
		cout<<0<<endl;
		else if(s[1]==s[2]&&s[2]==s[3]&&s[3]==s[4])
		cout<<1<<endl;
		else if(s[1]==s[2]&&s[3]==s[4])
		cout<<4<<endl;
		else cout<<8<<endl;
	}

 	return 0;
}

```
