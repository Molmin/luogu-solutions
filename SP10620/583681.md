## 题目大意：
输入一个 $n$，表示有 $n$ 本书，在输入 $n$ 个数，表示每本书的编号，输出最少需要几次能把编号按照从小到大的顺序排好序，每次只能将任意一本书移到顶层。

## 题目分析：
我们用一个 `for` 循环从 $1$ 到 $n-1$ 遍历一下，如果前面的元素 $i$ 大于后面的元素 $i+1$ 就将他们交换，将答案加 $1$，最后输出答案即可。

## 注意：
* 因为我们需要用到 $i+1$ 所以循环只能到 $n-1$。

## 具体代码：


~~~cpp
#include <iostream>
using namespace std;
int n, a[100001], ans;
// 分别表示 n 本书、每本书的编号和答案
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>a[i];
	}
// 读入 n 和每本书的编号
	for(int i=1;i<=n-1;i++){
		// 因为处理下标时需要用到 i+1 如果到 n 就会出现下标越界的情况，所以到 n-1 就行了 
		if(a[i]>a[i+1]){
			// 如果上面的书比下面的书编号要大，说明这两本书的顺序是乱的，需要交换一下 
			int temp=a[i];
			a[i]=a[i+1];
			a[i+1]=temp;
			// 交换的过程想必就不用解释了吧
			// 交换的时候也可以用 swap(a[i], a[i+1]) ,这个看个人习惯 
			ans++;
			// 题目问的是需要几次才能按编号排好序，所以我们每交换一次就要将答案加1 
		}
	}
	cout<<ans;
	// 最后输出答案就行了 
	return 0;
}
~~~
