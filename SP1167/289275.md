简单的橙题 ~~，居然被恶意评紫？~~

对于这种题，我们最重要的就是——打表找规律！

| 高度 $h$ | 需要移动的数量 |
| :----------: | :----------: |
| 1 | 0 |
| 2 | 1 |
| 3 | 2 |
| 4 | 3 |
| 5 | 5 |
| 6 | 7 |
|...|...|

看不出来？将需要移动的数量 $\times6$：

| 高度 $h$ | 需要移动的数量 |
| :----------: | :----------: |
| 1 | 0 |
| 2 | 6 |
| 3 | 12 |
| 4 | 18 |
| 5 | 30 |
| 6 | 42 |
|...|...|

再用需要移动的数量除以高度 $h$（上取整）得：

| 高度 $h$ | 需要移动的数量 |相除后的结果|
| :----------: | :----------: | :----------: |
| 1 | 0 |0|
| 2 | 6 |3
| 3 | 12 |4
| 4 | 18 |5
| 5 | 30 |6
| 6 | 42 |7
|...|...|...

再看一眼所对应的高度和相除后的结果，会发现——

## **除了高度为 $1$ 的以外，相除后的结果=高度+1**

所以，需要移动的数量就是——

$\left \lfloor \dfrac{n \times(n+1)}{6} \right \rfloor$。

公式都出来了，这道题还过不了吗？

## Code：

```cpp
#include<bits/stdc++.h>
using namespace std; 
long long n;
int t;
int main(){
    cin>>t;
    while(t--){
    	cin>>n;
    	cout<<n*(n+1)/6<<endl;
	}
    return 0;  
}
```
