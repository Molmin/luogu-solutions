这道题非常的简单，只需稍作自考应该就可以写出代码。

我们首先要特判 $n=1,2,3$ 时的答案。

- 当 $n=1$ 时， $n$ 不用变化，输出 $0$ 。
- 当 $n=2$ 时， $n$ 不能 $/2$ ,要 $-1$ ，输出 $1$ 。
- 当 $n=3$ 时， $n$ 不能 $-1$ 之后在 $/2$ , 要 $-1$ 之后在 $-1$ ，输出 $1$ 。

其他的在分为 $2$ 类。

- 当 $n$ 为偶数时， $n$ 只要 $/(n/2)$ 就可以得到 $2$ 在 $-1$，输出 $2$ 。
- 当 $n$ 为奇数时， $n$ 只要 $-1$ 变成偶数在进行上面偶数的步骤,输出 $3$ 。

接下来就是代码了。

```
#include<bits/stdc++.h>
using namespace std;
int main(){
	int T;
	scanf("%d",&T);
	while(T--){
		int a;
		scanf("%d",&a);
		if(a==1)
			printf("%d\n",0);//特判n=1时的情况。
		else if(a==2)
			printf("%d\n",1);//特判n=2时的情况。
		else if(a==3)
			printf("%d\n",2);//特判n=3时的情况。
		else if(a%2==0)
			printf("%d\n",2);//特判n为偶数时的情况。
		else
			printf("%d\n",3);//特判n为奇数时的情况。
	}
	return 0;
}
```
