## 文字题解

### 简要题意

若 $a \le b$，输出 `YE5`（审题），若 $a>b$，输出 `N0`（审题）

### 关键代码

考察 if...else 结构。

```cpp
if(a <= b) puts("YE5");
else puts("N0");
```

当然，也可以用两个 if 代替。

```cpp
if(a <= b) puts("YE5");
if(a > b) puts("N0");
```
