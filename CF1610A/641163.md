## 思路&解法：

在数学中，有一种技巧叫分类讨论。

我们当然也可以把这个思路用在编程上。

## 分类讨论

&#9733; 分类1：当行数 = 1，列数 = 1 时，也就意味着网格只有一个格子。

也就换句话说，我们一次就绝对可以猜对，所以猜测答案次数为 0。

&#9733; 分类2：当行数 = 1，列数 > 1 时，或行数 > 1，列数 = 1 时，也就意味着网格只有一行格子。

也就换句话说，我们一次就绝对可以猜对，所以猜测答案次数为 1。

&#9733; 分类3：当行数 > 1，列数 > 1 时，也就意味着网格只有一行格子。

也就换句话说，我们一次就绝对可以猜对，所以猜测答案次数为 2。

这道题便愉快的解决了！

## 代码（我是大蒟蒻）：

```cpp
#include<bits/stdc++.h>
using namespace std;
int a,n,m;
int main(){ 
    cin>>a;
    for(int i=0;i<a;++i){
        scanf("%d%d",&n,&m);
        if(n==1&&m==1) {
        	printf("0\n");
        	continue;//分类1 
		}
        if(n==1||m==1) {
        	printf("1\n");
        	continue;//分类2 
		}
        printf("2\n");//分类3 
    }
    return 0;
} 
```
