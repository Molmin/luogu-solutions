### 解题思路：

题目描述其实就是一个曼哈顿距离下的圆的问题。

你可以选出几个点，然后要求用这些点到未知点的距离确定未知点的位置。其实就是在曼哈顿距离下一起中的点为圆心，然后做一个圆，要求最后的范围内的交点唯一。

考虑一般情况，可以选择左上角和左上角右边的一个点（可以有其他构造方法），这样在任意情况下都有这两个圆只有一个交点（还有一个不在范围内）。

特殊的，若这个地图是一维的，只需要找一个端点就行了；若地图只有一个点，那么这个点一定是要求点。

### 代码：
```cpp
#include<cstdio>
using namespace std;
int n,m,T;
int main(){
	scanf("%d",&T);
	while(T--){
		scanf("%d%d",&n,&m);
		if(n==1&&m==1)printf("0\n");
		else
		if(n==1||m==1)printf("1\n");
		else printf("2\n");
	}
	return 0;
}
```
