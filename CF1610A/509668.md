# CF1610A Anti Light's Cell Guessing 题解

[题目传送门](https://www.luogu.com.cn/problem/CF1610A)

[更好的阅读体验](https://www.luogu.com.cn/blog/chengjiayi666/solution-cf1610a)

### 题目大意：

现有一个大小为 $n \times m$ 网格中的坐标 $(x,y)$。进行 $k$ 次猜测，可以知道一个长度为 $k$ 的数组 $b$，并且 $b_i=|x-x_i|+|y-y_i|$。求要想猜出 $(x,y)$ 的具体位置，$k$ 最小为多少。

### 正文：

红题自然不会有什么难度，我们可以尝试**分类讨论**：

1. 可以先从 $n=1$ 且 $m=1$ 的情况开始考虑。很显然，这样的 $k$ 的最小值是 $0$。故当 $n=1$ 且 $m=1$ 的情况下，$k$ 的最小值为 $0$。

1. 再来考虑 $n=1$ 且 $m=2$ 的情况，显然这样的 $k$ 最小是 $1$。而 $n=1$ 且 $m=3$ 的情况呢？显然，同样是 $1$。$m=4$ 亦是如此。这是因为如果我们选择点 $(1,1)$，那么每个点到该点的距离均不相同，也就是说可以直接确定点 $(x,y)$ 的位置。故当 $n$ 和 $m$ 有且仅有一个 $1$ 的时候，$k$ 的最小值为 $1$。

1. 我们继续往后推，当 $n=2$ 且 $m=2$ 的情况下，我们还是先选定点 $(1,1)$，那么我们就可以将点 $(x,y)$ 的位置锁定在一条每个点距离点 $(1,1)$ 的距离都相等的斜线上。接着，我们可以再选定一点，比如点 $(n,1)$。同样的，我们也可以将点 $(x,y)$ 的位置锁定在一条每个点距离点 $(n,1)$ 的距离都相等的斜线上。而这两条斜线的交点，就是我们要找的点。故当 $n$ 与 $m$ 均大于 $1$ 的情况下，$k$ 的最小值为 $2$。

### 总结：

- 当 $n=1$ 且 $m=1$ 时，$k$ 的最小值为 $0$。

- 当 $n$ 与 $m$ 有且仅有一个 $1$ 时，$k$ 的最小值为 $1$。

- 当 $n$ 与 $m$ 均大于 $1$ 时，$k$ 的最小值为 $2$。

#include<bits/stdc++.h>
using namespace std;
int main(){
	int T,n,m;
	cin>>T;
	while(T--){
		cin>>n>>m;
		if(n==1&&m==1) cout<<"0\n";
		else if(n==1||m==1) cout<<"1\n";
		else cout<<"2\n"; 
	}
	return 0;
}
