[题目传送门](https://www.luogu.com.cn/problem/CF1610A)

## 题目大意

蓝邀请你和她玩一个游戏：

有一个 $n \times m$ 的网格。她在心里想了网格中的一个坐标 $(x,y)$。

为了猜出坐标 $(x,y)$，你可以选出 $k$ 个坐标 $(x_1,y_1),(x_2,y_2),\dots,(x_k,y_k)$，然后蓝会算出并告诉你 $k$ 个点与 $(x,y)$ 的哈曼顿距离。

现在你对于 $k$ 的最小值很感兴趣。对于给出的 $n,m$，试着求出最小的 $k$，使得存在一种选出 $k$ 个坐标的方案，使得无论蓝心中想的 $(x,y)$ 是什么样的，你都能将其猜出。

## 思路

这道题可以分成三种情况：

- $n,m$ 都是 $1$，输出 $0$。
- $n,m$ 中有且仅有一个 $1$，输出 $1$。
- $n,m$ 都不为 $1$，输出 $2$。

为什么呢？

如果 $n,m$ 都是 $1$，那么只有一个点，我们不需要猜就可以知道了，所以输出 $0$。

如果 $n,m$ 中有且仅有一个 $1$，那么我们只要猜 $(1,1)$ 这个点就可以了，所以输出 $1$。

如果 $n,m$ 都不为 $1$，那么我们只需要选择 $(1,1)$ 和 $(n,1)$ 就可以算出 $(x,y)$ 的位置了。

## 代码

```cpp
#include <iostream>
using namespace std;
int main(){
	int t;
	cin>>t;
	while(t--){
		int n,m;
		cin>>n>>m;
		if(n==1||m==1){
			if(n==m){//都等于一
				cout<<"0";
			}
			else{//有且仅有一个一
				cout<<"1";
			}
		}
		else{//一个一都没有
			cout<<"2";
		}
		cout<<endl;
	}
	return 0;
}
```
