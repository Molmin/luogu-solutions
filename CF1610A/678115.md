**题目大意**

蓝邀请你和她玩一个游戏：

有一个 $n \times m$ 的网格。她在心里想了网格中的一个坐标 $(x,y)$。

为了猜出坐标 $(x,y)$，你可以选出 $k$ 个坐标 $(x_1,y_1),(x_2,y_2),\dots,(x_k,y_k)$，然后蓝会算出并告诉你一个长度为 $k$ 的数组 $b$，满足 $b_i=|x-x_i|+|y-y_i|$。

现在你对于 $k$ 的最小值很感兴趣。对于给出的 $n,m$，试着求出最小的 $k$，使得存在一种选出 $k$ 个坐标的方案，使得无论蓝心中想的 $(x,y)$ 是什么样的，你都能将其猜出。

总之，这道题需要一个基本的思想——分类讨论。

**何为分类讨论**

简单来说，分类讨论就是把一个问题分成几种不同的情况，分别分析。

**题目分析**

现在，来让我们看看这道题

1. 如果 $n=m=1$，则只有一种情况 $(1,1)$，输出 $0$。

2. 如果  $\text{min}(n,m)=1$ ，我们可以找到点 $(1,1)$，这样任意一个点与其距离并不相同，输出 $1$。

3. 对于其他情况，我们可以找到点 $(1,1)$ 与点 $(n,1)$，输出 $2$。

**AC Code**

```c++
#include<bits/stdc++.h>//好习惯
using namespace std;
int t,n,m;
int main(){ 
    cin>>t;//输入
    for(int i=1;i<=t;i++){
        cin>>n>>m;
        if(n==1&&m==1){//第一种情况
        	cout<<0<<endl;
        }
        else if(n==1||m==1){//第二种情况
		cout<<1<<endl;
        }
        else{//第三种情况（也可以说是其他情况）
        	cout<<2<<endl;
        }
  			
    }
    return 0;//完结撒花
} 
```
