首先我们要分析一下题目：

1.当它所给的 $m$ 和 $n$ 都为 $1$ 的时候（也就是说在 $n×m$ 的矩阵中只有一个点）不需要查找，就输出 $0$。

2.当它所给的 $m$ 或者 $n$ 为 $1$ 时，就是一条线，只需给出一个点就可以确定坐标。（$x=1$ 或者 $y=1$）

3.如果上述情况都不满足，我们就可以找出 $(1,n)$ 和 $(1,1)$ （或者 $(n,1)$）就输出 $2$。（两点确定一个坐标）

大概就是这样。

所以可以给出代码了：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main(){
	int n;
	cin>>n;
	int x,y;
	for(int i=1;i<=n;i++){
		cin>>x>>y;
		if(x==1 && y==1) cout<<"0";//同 1 
		else if(x==1 || y==1) cout<<"1";//同 2
		else cout<<"2";//同 3
		cout<<endl;
	}
	return 0;
}
```
