快速幂板子，双倍经验 P1226。

计算 $x^y \!\!\mod n$。

#### 快速幂原理及实现

快速幂能在 $\log_2(y)$ 的时间内算出 $x^y \!\!\mod n$。

思路：二进制，每次通过计算 $n \And 1$ 知道答案需不需要乘 $x^{2^i}\,(0 \leq i)$，随后 $x$ 自乘，$n$ 右移一位。

举例：

```
x = 2, y = 13, ans = 1
//初始 y 的二进制表示为 1101, x = 2^1

第一次，当前 y & 1 = 1 (y 的最低位 110->1<-), ans *= x (ans = 2)
x *= x (x = 4 = 2^2), y 右移一位 y >>= 1, y = (110)_2 = 6.

第二次，y & 1 = 0, ans 不进行操作，x *= x (x = 16 = 2^4), y >>= 1, y = (11)_2 = 3

第三次，y & 1 = 1, ans *= x (ans = 32), x *= x (x = 256 = 2^8)
y >>= 1, y = (1)_2 = 1

最后一次，y & 1 = 1, ans *= x (ans = 8192), x *= x (x = 65536 = 2^16)

y >>= 1, y = (0)_2 = 0, y < 1, 结束操作
```

于是我们计算出了 $2^{13}=8192$。

代码实现：

```cpp
ll qpow(ll n, ll base, ll mod) {
	ll ret = 1;
	while(n) {
		if(n & 1) ret = ret * base % mod;
		base = base * base % mod;
		n >>= 1;
	}
	return ret % mod;
}
```
