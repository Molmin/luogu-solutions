### 题意：
把这个数列中最大的数放在最前面，最小的数放后面。

求的是逐步交换的步数,也就是规定一次只能交换相邻位置的数。


------------

### 思路：
输入的时候不断比较，但在出现等于最小值时更新位置（让定位更接近最后一位）

当最大值在最小值后时，总移动次数 - 1


------------
```cpp
#include<bits/stdc++.h>
using namespace std;
int n, i, p, q = 99, x, a, b;
int main() {
    for (cin >> n; i++ < n;)
        cin >> x, x > p ? p = x, b = i : 0, x <= q ? q = x, a = i : 0; //更新位置。
    cout << b + n - a - 1 - (b > a);//判断最大值是否在最小值后
  return 0;
}
```
