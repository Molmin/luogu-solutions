这道题……小学数学题，考点：**等差数列**

等差数列求和公式： $\dfrac{(Item_1+Item _n)\times total}{2}$

### 翻译成人话就是：$ ( 首项 + 末项 ) × 项数 ÷ 2$

~~爪巴LaTeX~~

那么现在，我们只要把上面三个条件找全即可。

## 首项：1

## 末项：$n$

## 项数：$ ( 1 + n ) / 2$

#### 注：因为只有奇数才加进去，所以在$1$~$n$中的偶数是不算的，所以要除以二。

好啦现在找全了，但是还要注意，题目要求是从 $1$ 加到 $n$ 再加到 $1$ , 这好办，把我们刚刚算好的结果乘个$2$，然后在减掉一个重复计算的$n$即可。

# Code：

```cpp
#include<iostream>
using namespace std;
int main()
{
	int m;
	cin>>m;
	for(int i=1;i<=m;i++)
	{
		int n;
		cin>>n;
		cout<<2*(((1+n)*((1+n)/2))/2)-n<<endl;//等差数列求和乘以二再减n
	}
	return 0;
}
```
