0.题目翻译

本题大意为:给定你一个数 $N$,然后要求 $n$ 边形的对角线总数不小于 $N$,问 $n$ 的最小可能值是多少？

题目数据中 $N$ 最大可以为 $10$ 的 $15$ 次方(故而要开 $long long$ )。

有不定组询问,以 $0$ 结束,需求输出格式为 $Case$  $x:$ $ans$,其中 $x$ 为组数, $ans$ 为答案。

1.题目解析以及知识扩展

本题所用到核心知识为：

   >解一元二次方程以及组合数概念。

首先我们对一个 $n$ 边形进行研究，它的对角线总条数应该怎么计算？

我们可以先选定一个点，然后我们知道这个 $n$ 边形一共是有 $n$ 个点的，我们选取了一个，还有 $n-1$ 个点，而还有两个点（刚刚选取点的左边和右边）构成了原多边形的边，所以连接这三个点不能构成对角线，那么还剩 $n-3$ 条边，我们是都可以选取的，同时要注意一点，假设我们选择了$A$ 和 $D$ 点，是和我们选取 $D$ 和 $A$ 点所连出来的对角线是同一条线，那么我们最后的答案还要除以 $2$,也就是最终答案为 $(n-3*n)/2$ .

然后就是一元二次方程的求解，我们在上面已经知道了对于 $n$ 边形而言，对角线条数为 $(n-3*n)/2$ ，那么就可以列出来如下方程式

>$n^2-3n=2N$

那么呢，我们便尝试求解该方程，不难解出其答案为：
>```
>ans=((3+sqrt(8*N+9))/2;
>```

二次方程是有两个根的，我们只要正数根，所以直接舍去了另一个根。

2.附代码
```
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
ll k,amt;
int main()
{
    while(cin>>k)
    {
        if(!k)
            return 0;
        else
        {
            ll delta=8*k+9;
            ll ans=3+sqrt(delta);
            ans=ans/2;
            if(ans*(ans-3)/2<k)
                ans++;
            printf("Case %d: %d\n", ++amt, ans);//一定要写成++amt
        }
    }
    return 0;
}

```
