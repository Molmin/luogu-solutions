------------

开始分析。

    1.有些人总是想着用一个dp来完成所有事，其实我们可以考虑把dp分开。
    2.那么我们来罗列状态，也就是我们可以干什么。
        一，不动并且加4滴能量。
        二，跑17米。
        三，用技能，跑60米，减10滴能量。（一三一类，二一类）
    3.接下来，我们可以根据是否有能量的变化，来进行分类。
    4.我们用贪心的思想看，如果可以用技能，就用技能，直到不能用为止。
      然后开始  休息——放技能 的循环。
      （只要循环T次即可）。
    5.最后，进行第二个循环，加入只跑步，不技能的情况。就可以得出答案。

打上代码

```
#include<bits/stdc++.h>
using namespace std;
int m,s,t;
int dp[300005];
int main()
{
	scanf("%d%d%d",&m,&s,&t);
	for(int i=1;i<=t;i++)
	{
		if(m>=10)
        	{
			m-=10;
			dp[i]=dp[i-1]+60;
		}//使用技能
		else
        	{
			dp[i]=dp[i-1];
			m+=4;//回能量 
		}
	}
	for(int i=1;i<=t;i++)
	{
		dp[i]=max(dp[i],dp[i-1]+17);//考虑只跑步的状态 
		if(dp[i]>=s)
          	{
			printf("Yes\n%d",i);return 0;
		}
	}
	printf("No\n%d",dp[t]);
    return 0;
 }
```
