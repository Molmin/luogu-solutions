### 这道题其实是一道 数学题（误

代码也很短，只要把公式推出来就好了。

有几个坑点就是  
**$1$：一定要按$n$下  
$2$: $\text{ }0$楼不能作为最后的答案**

因为一定要按$n$下，所以不妨设其中$k$下是按$U$,则$(n-k)$下是按$D$。因为答案问的是最后的楼层，所以**只要关心一共按几下就行了**，跟按得顺序没关系。

则最后的楼层为 $u*k-d*(n-k)$  
由于电梯不能到达比底楼低的楼层, 所以得到不等式$u*k-d*(n-k)>0$   
整理一下  
$u*k-d*n+d*k>0$  
$(u+d)*k>d*n$  
$k>\frac{d*n}{u+d}$

电梯上升$k$次，要使楼层最低也就是要使$k$最小。  
所以$k=(d[i]*n)/(u[i]+d[i])+1; $

所以枚举一下所有的电梯，每一部电梯的最低楼层取最小值就好了

```
#include <bits/stdc++.h>
using namespace std;
int n,m,u[2005],d[2005],k,ans=1<<30;
int main()
{
	int i,j;
	cin>>n>>m;
	for(i=1;i<=m;i++)
		cin>>u[i]>>d[i]; //输入
	for(i=1;i<=m;i++) //枚举电梯
	{
		k=(d[i]*n)/(u[i]+d[i])+1; //k取最小值
		ans=min(ans,u[i]*k-d[i]*(n-k)); //所有答案里面去最小的
	}
	cout<<ans<<endl; //输出
	return 0;  
}  
```
