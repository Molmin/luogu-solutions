### 题意

有多组数据，对于每一组数据，读入 $5$ 个字符。如果前两个数字之和是奇数，则输出 ```left```；如果前两个数字之和为非零偶数，输出 ```right```；如果前两个之和为零，输出 ```left``` 或 ```right``` 取决于上组数据，然后输出一个空格与后三个数字。

### 解法

其实是一个模拟题，但由于难度较低，所以讲解的稍微详细一些。

每一组数据可以使用 string 读入，后面字符串处理，也可以使用 int 读入，后面拆位处理。

### 亿些细节

1. 对于读入到 $99999$ 停止读入，可以使用
```cpp
while (true) if (/**/) break;
```
的方式。

2. 对于一组数据，如果前两个数字都是 $0$，需要依靠前一次结果。所以，我们需要加入一个 string 类型的变量 s，记录每一次的输出是 ```left``` 还是 ```right```。
