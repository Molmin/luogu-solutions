### 题目大意

[题目直通](https://www.luogu.com.cn/problem/P9022)

对于每组数据读入 $5$ 个数字字符。

- 如果前两个之和为奇数 left；

- 如果前两个之和为非零偶数，输出 right；

- 如果前两个之和为零，输出 left 或 right 取决于上组数据。

然后输出一个空格与后三个数字字符，直到读入 $99999$ 为止。

### 大体思路

由于题目中说每组数据读入 $5$ 个数字字符，所以用字符串读入会很方便。

```cpp
string a;
cin >> a;
cout << a;
```

字符串可以直接访问单个字符。

```cpp
cout << a[1] ;
```

但是其中为单个字符参与数字运算要减去字符 $0$。

```cpp
int x = a[0] - '0';
```


再看这道题，我们只要算出前两位的和，分类处理，还要记得保留上一组数据的情况。最后还要输出后三位，直接按字符串的字符输出即可。这里我们可以看出字符串的方便。


### 完整代码

```cpp
#include<iostream>
#include<cstring>//字符串头文件 
using namespace std;
string a, f;
signed main() {
	cin >> a;
	while (a != "99999") {//是否为 99999，注意是字符串类型  
		int x = a[0] + a[1] - '0' - '0';//计算前两位的和  
		if (x == 0) {//注意先判断和为 0 的情况 
			cout << f << ' ';//输出上一组数据
		}
		else if (x % 2 == 0) {
			//用取余的方式判断奇偶性，0则为偶数 
			cout << "right ";
			f = "right ";//保留这组数据
		}
		else {
			//奇数 
			cout << "left ";
			f = "left ";
		}
		cout << a[2] << a[3] << a[4] << endl;
		//输出后三位 
		cin >> a;//再次读入下一组数据  
	}
	return 0;
}
```
