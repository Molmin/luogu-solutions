这个题目其实就是P1150 peter的烟的升级版，
只是要求多了几组数据而已。。。

对于这道题目，我比较赞成直接编程做，因为很简单，就不必在纸上推导公式了。

因为是入门难度，所以直接上代码

```cpp
#include<iostream>
using namespace std;
int main(){
	int n,k,outt,els,w;  //outt代表吸得烟的数量，els代表剩余的烟蒂的数量
	while(cin>>n>>k){  //这个代表的是反复输入n,k，直到没有数据输入为止
	outt=n;   
	els=n;  //把outt与els初始值都赋为n
	w=100;  //其实w随便给一个值，只要不等于0就可以了
	while(w!=0){
		w=els/k;  //w代表这一轮可以换到的烟
		outt+=w;
		els=els-w*k+w;  //注意还得加上换到的烟吸完后变成的烟蒂
	}  //反复计算吸完后的烟蒂能换的烟
	cout<<outt<<endl;  //输出后腰换行
	}
	return 0;
}
```
毕竟是入门难度，没有什么太高深的要解释
