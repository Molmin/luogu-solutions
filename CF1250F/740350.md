## 题目

你正在建造一个新的数据中心。数据中心是一个矩形，面积正好为 $n$ 平方米。数据中心的每一侧都是整数。

你需要尽量减少外部环境对数据中心的影响。因此，数据中心的周长需要是最小的。

数据中心的最小周长是多少？

## 题目大意

有一个矩形，面积为 $n$ 平方米。这个矩形的每一侧都是整数。

要保证这个矩形周长是最小的。求这个矩形的最小周长是多少？

## 题目解答

这题要用到一个定理，就是：当积一定时，如果差小，那么和就小。

就是要保证这两个乘数没有很大，那么和才能小。

## 代码

```cpp
#include<bits/stdc++.h>//很简单 
using namespace std;
int n,i;
int main(){
    cin>>n;
    for(i=sqrt(n);i>=1;i--)//判断 
        if(n%i==0)cout<<(i+n/i)*2<<endl; 
    return 0;//完结
}
```
