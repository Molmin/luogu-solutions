[题目传送门](https://www.luogu.com.cn/problem/CF1250F)

### 题意

有一个面积为 $n$ 平方米的矩形，它的长宽都是整数。现在你要使其的**周长最小**。

### 分析

大家在小学都学过一个知识：**差小积大**。那么因此就能推出另外一个规律：**当积相等，差小则和越小**。

举个例子：

$16=4\times4$，$4+4=8$，$4-4=0$。

$16=8\times2$，$8+2=10$。$8-2=6$。

$8<10$，$0<6$。

因为长宽都是整数，所以就可以用类似筛选质数的方法，从 $1\sim\sqrt{n}$ 里找到一个 $n$ 最大的因数 $i$，把它作为宽，那长就是 $n\div i$。所以最后的周长计算公式就是 $(i+n\div i)\times2$。找到后继续往后找，直到长和宽的差最小。$n$ 的数据范围是 $10^5$，所以无需加上一些优化。最后输出答案（公式）即可。

当然也可以从后往前枚举，最先找到的一定是差最小的。正解就是这个。

### 代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int n;
	cin>>n;
	for(int i=sqrt(n);i>=1;i--)
	{
		if(n%i==0)
		{
			cout<<(i+n/i)*2<<endl;
			return 0;
		}
	}
	return 0;
}
```
