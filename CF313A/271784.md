题目传送>>[CF313A](https://www.luogu.com.cn/problem/CF313A)  
### 题意简述：  
- 给定一个银行账户余额 $n$，现有一次机会可消除其最后一位或倒数第二位数字,求最终其最大值。  
- $10\le|n|\le10^9$。  

### 题目分析:  
据题意可知账户余额可以为正，也可以为负（此时表示欠钱）。为正的时候不管消除其最后一位还是倒数第二位账户余额都会变小，所以我们此时就放弃消除，初始余额即最大。而为负时，当然欠的钱越少越好，我们就可以使用消除的机会对其进行改变，我们只需要分别计算出消除其最后一位和倒数第二位得到的最终余额，再进行比较，越大的即表示欠钱越少，账户余额也就最大。  
### 问题解决:  
#### 判断:  
我们可以使用 if...else... 进行判断，当然我们也可以使用一个小东西 —— 三目运算符 “?:”。  
例：a?b:c 的含义是：如果 a 为真，则表达式的值是 b，否则是 c。  
### Code:  
```cpp
#include<iostream>
using namespace std;
int main()
{
	ios::sync_with_stdio(0);   //关同步流，加速cin输入和cout输出
	int n;   //定义账户余额
	int xl,xs,r;   //定义消除其最后一位的结果，消除其倒数第二位的结果，最终结果
	cin>>n;
	xl=n/10;    //计算消除其最后一位的结果
	xs=(n/100)*10+n%10;   //计算消除其倒数第二位的结果
	r=(n>0)? n:max(xl,xs);   //利用三目运算符判断其最终结果，如果为正即为初始值，为负则比较两个经消除后的数值取其最大
	cout<<r;
	return 0;   //结束整个程序
}
```
