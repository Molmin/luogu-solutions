这个算是一道比较基础的数论题，这一题的题意是：

给你一个数$n$，你可以对他进行以下几种操作
1. 不变
2. 删除它的倒数第一位	
3. 删除它的倒数第二位，并把最后一位向前与倒数第三位连接

那么，我们怎么用代码实现上面的三种情况呢。首先第一种十分的简单，就是$n$本身；第二种要删除最后一位，就是$n/10$；第三种是删除倒数第二位，也就是倒数第二位以前的位数（n/100）+倒数第一位（n%10），也就是$ ((n/100)* 10+n%10) $

所以代码十分的简单，我们只要在三种情况中取最大值就好了，对于取最大值，我们可以用$max$，但是$max$只能比较两个数的掉线，这里告诉大家一种方法可以用：max(a,max(b,c))来获取三个数的最大值。

其他细节上的问题我会在代码的注释中写出，这里就不一一解释，代码如下：

[![](https://cdn.luogu.com.cn/upload/image_hosting/bhfc7ota.png)证明](https://www.luogu.com.cn/record/32443326)

```cpp
#include<bits/stdc++.h>//多余的头文件最好要删掉 
using namespace std;
int main() {
	int n;
	cin>>n;//输入n 
	cout<<max(n,max(((n/100)*10+n%10),n/10))<<endl;//三个中取最大值 
	return 0; 
}
```
