看没人发题解，我在AC此题数天之后发一篇：

- 根据我们的数据，$1\le n\le125000$

- 然后看到 **五个立方数** 几个字

- 我们可以轻松地知道在数据范围一共有50个立方数，因为$50^3=125000$

- 因此，我们的第一想法很可能就是 **打暴力** 

- 思路就是枚举每一组立方数，看有没有符合要求的，因为不是正解并且十分简单，我就不贴代码了

- 然而，如果我们自己算一下时间复杂度,最高是$O(50^5)$,几乎一定会被卡掉

- 所以，这就要求我们想一些其他的方法，毫无疑问，这个方法大家应该都能想到——DP

- 由于本人的DP很差，所以在复习并且找了好几篇题目类似的简单DP题解学习了一下之后，我打出了一份应该是很基础的DP(~~由此可见我的DP差到了何等地步~~)

- DP核心动态转移方程：dp[j+i\*i\*i][k]+=dp[j][k-1];

CODE:
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,dp[201010][7];//DP数组至少也要开到150000和5吧
int main(){
    cin>>n;
    dp[0][0]=1;//别忘了这句
    for(int i=0;i<=500;i++){//DP开始
        for(int j=0;j+i*i*i<=n;j++){
            for(int k=1;k<=5;k++){
                dp[j+i*i*i][k]+=dp[j][k-1];//转移方程
            }
        }
    }
    cout<<dp[n][5]<<"\n";//输出就好
    return 0;
}
```
