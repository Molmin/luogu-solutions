## 思路

注意，第 $i$ 层房间的亮度最多为 $i$ ，这是因为在房间 $(i,1)$ 中，只有 $i$ 个房间，即 $(1,1),(2,1), \cdots (i,1)$ 可以通过一定数量的楼梯到达(i,1)。

也有可能在金字塔中找到一种火炬的配置，使得第 $i$ 层的房间的亮度恰好是 $i$ ，即它达到了上限。

配置如下:当且仅当房间 $(i,j)$ 是第 $i$ 层最左边的房间 $(i=1)$ 或最右边的房间 $(i=j)$ 时，房间 $(i,j)$ 包含一个火炬。

这是有效的,因为所有房间 $(i, j)$ ,它可以从 $(1,1),(2,1),(3,1), \cdots ,(i-j+1,1)$ 和 $(2,2),(3,3), \cdots (j,j)$ 。换句话说，房间 $(i,j)$ 的亮度 $(i-j+1) + j - 1 = i$ ，所以金字塔是好的。

Code:
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int t;
    cin >> t;
    while (t--) {
        int n;
        cin >> n;
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= i; j++) {
                cout << (j == 1 || j == i) << ' ';
            }
            cout << '\n';
        }
    }
}
```
