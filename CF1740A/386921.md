### 题意简述

给定一个整数 $ n $，请输出一个质数 $ m $ 使得 $ n + m $ 为合数。

### 思路

**$ n $ 为奇数**

由于一个大于等于 $ 2 $ 的偶数必定是合数，而一个奇数加另一个奇数必定为偶数，所以我们只需要加 $ 3 $ 就可以了。

**$ n $ 为偶数**

一个大于等于 $ 2 $ 的偶数必定是合数，所以我们只需要加 $ 2 $ 也只能加 $ 2 $，因为一个大于等于 $ 2 $ 的偶数必定是合数。

#### 综上所述：

先输入一个数，如果它是奇数，那么输出 $ 3 $，否则输出 $ 2 $。

### 代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int T;
	cin>>T;//组数
	while(T--){//循环T次
		int n;
		cin>>n;
		if(n%2==0)cout<<2<<endl;//偶数输出2
		else cout<<3<<endl;//奇数输出3
	}
	return 0;
}
```
