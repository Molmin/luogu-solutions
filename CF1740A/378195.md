# 题意简述
给你一个质数 $n$，你输出一个数 $m$ 使得 $n+m$ 为合数。

# 思路
对于每一个 $n$，有 $2$ 种情况：
1. **是 $2$**，因为 $2+2$ 为合数，所以**输出 $2$** 就行了。
2. **是奇数**，那么**输出 $3$** 就行了。因为任意一个奇数加上 $3$ 都会是一个偶数。除了 $2$ 别的偶数都是合数，而只有 $-1$ 加上 $3$ 才是 $2$。

# 代码
```cpp
#include <iostream>
using namespace std;

int main(){
	int t;
	cin >> t; // 输入 t 
	int n;
	for (int i=0;i<t;i++){
		cin >> n; // 输入 n 
		if (n==2) cout << 2 << endl; // 是 2 输出 2 
		else cout << 3 << endl; // 否则输出 3 
	}
}
```
