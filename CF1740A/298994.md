# 解题思路

不难想到分奇偶考虑

## $n$ 为偶数

题目保证 $n$ 为素数，若其为偶数显然只能为 2。不难发现 4=2+2 不是质数且 2 为质数。所以直接输出 2 即可。

## $n$ 为奇数

不难发现全部奇质数再加上任意一个奇质数都不是质数。因为奇质数显然 $\ge 3$，且两个奇数之和为偶数，大于等于 6 的偶数显然不是质数。那么我们随便选个质数 3 即可（或者直接输出 $n$）。

综上所述，我们可以直接输出 $n$。

# 代码
```
#include<bits/stdc++.h>
using namespace std;

#define LL long long

LL t,n;

int main(){
	scanf("%lld",&t);
	while(t--){
		scanf("%lld",&n);
		printf("%lld\n",n);
	}
}
```
