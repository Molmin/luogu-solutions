# 思路
我看到这道题，本来以为会很难，但显然，我错了。

一道绿题，是一道结论题，自己做了一遍，发现代码行数极短。

我们以初中三元一次方程来看，把三个大数字：2019201913，2019201949，2019201997往x，y，z里套，然后找到最小的这个等价方程，将输入数据往里带，然后输出就完了。

这里我发现的最小z，它对应的x，y分别是x=组数-1和y=牛数。
# 代码
这里用代码实现非常简单，大家观赏一下。
```cpp
#include<bits/stdc++.h>
using namespace std;
int main() 
{
	long long N,K;//不开longlong见祖宗（WA） 
	cin>>N>>K;
	printf("%d\n",(2019201913*(K-1)+2019201949*N)%2019201997);//虽然思路里是三元一次方程，但为了直观我还是直接用数字 
	return 0;
}
```
