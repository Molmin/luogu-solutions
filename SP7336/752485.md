# 题意简述
[洛谷题目传送门](/problem/sp7336)

[更好的阅读体验](/blog/tianbiandeshenghuo11/solution-sp7336)

**这是一道 IO 交互题。**

在本题中，你需要找到一个长度为 $4$ 的**可有重复元素**的数列。

这个数列中的每个元素在 $1$ 到 $6$ 之间。

**你可以进行最多 $10$ 次询问。**

每次询问，你需要直接输出你的序列。交互库会相应的返回 $4$ 个数字。其返回值和对应的含义如下：

|返回值|含义|
|:---|:---|
|$1$|对应位置的数字正确。|
|$0$|你给出的数字在数列中出现过，但位置不对。|
|$-1$|你给出的数字在数列中未出现过。|


你应该在得到 `1 1 1 1` 返回后立刻结束程序。

在每次输出完后，请换行并使用 `fflush(stdout);` 刷新缓冲区或直接使用 `std::cout << std::endl;`换行。

# 分析
注意到本题值域较小，所以我们考虑每个数字依次枚举。

初始时，让每个数字都为 $1$。

然后读入交互库的判定。如果对应判断**不为** $1$，则将当前数字 $+1$。

同时用变量 $f$ 记录当前记录的四个判定结果是否都为 $1$，判断何时结束程序。

时间复杂度：$O(1)$。

空间复杂度：$O(1)$。

# Code
```cpp
#include<bits/stdc++.h>

using namespace std;
int ans[5] = {0, 1, 1, 1, 1};
int f = 1;
int main(){
  do{
    for (int i = 1; i <= 4; i++){
      cout << ans[i] << ' ';
    }
    cout << endl;
    f = 1;
    for (int i = 1, x; i <= 4; i++){
      cin >> x;
      ans[i] += x != 1;
      f = min(f, int(x == 1));
    }
  }while (!f);
  return 0;
}
```
