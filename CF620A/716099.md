本题考察了一个知识点，只要知道就是下位红题，要是不知道......

这个知识点就是大名鼎鼎的[切比雪夫距离](https://baike.baidu.com/item/%E5%88%87%E6%AF%94%E9%9B%AA%E5%A4%AB%E8%B7%9D%E7%A6%BB/8955729)，求两点坐标距离的公式。

手动解释切比雪夫距离：这类问题只能向自己点的上、下、左、右、左上、左下、右上、右下方向走。因为棋盘的性质，走一个斜线等于走一个直线的距离，所以只需要求出两边距离的最大值就可以了。

好，那么了解之后，代码就好写了。

首先，输入起点坐标 $(x1,y1)$，和终点坐标 $(x2,y2)$，然后代入公式求解即可。

答案呢是 $\max(|x1-x2|,|y1-y2|)$。

证明：因为 $x_1$ 每次只能改变 $1$， 所以至少要改变 $|x1-x2|$ 次。同理，至少要改变 $|y1-y2|$ 次。所以，一共至少要改变 $\max(|x1-x2|,|y1-y2|)$。

$O(1)$ 搞定。

代码

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int x1,y1,x2,y2;
	cin>>x1>>y1;
	cin>>x2>>y2;
	cout<<max(abs(x1-x2),abs(y1-y2));
	return 0;
}
```
