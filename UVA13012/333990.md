## Thinking

我们来看看[这一题](https://www.luogu.com.cn/problem/UVA13012)。看不懂嘤语没关系，我来翻译：


------------


给定若干个数据，一个数据包含两行。

第一行， 输入 $t$ 的值；第二行，输入 $a$、$b$、$c$、$d$、$e$ 的值。

求五个数中与 $t$ 相等的数有几个。

------------


这一题主要是考循环输入这个点。在循环里要用条件分支语句，最后输出要换行。注意：数据的输入有若干组，所以要持续输入，直到 $\texttt{EOF}$ 后停止。

补充：$\texttt{EOF}$ 是一个计算机的术语，为 $\texttt{End Of File}$ 的缩写，中文为“文件结束符”，通常在输入完毕后就会出现。

另外，建议各位在这种结尾换行的代码最后一次输入时尽量不换行，因为有一些题目结尾是不会忽略换行符的，可能会因为多换了行而造成错误。（但是因为这里不需要，所以窝没有这么做了。）

## Coding

```cpp
#include<bits/stdc++.h>
using namespace std;
int t, a, b, c, d, e; //因为循环输入时会自动将上一次循环的这个数的值覆盖，所以可以在这里定义
int main(){
	while(scanf("%d%d%d%d%d%d", &t, &a, &b, &c, &d, &e) != EOF){ //没有结束输入
		int ans = 0; //因为这个统计数在下一个循环时不会被覆盖，所以每次循环都要重新定义一遍，否则会在上一次循环的基础上再加
		if(a == t)ans++; 
		if(b == t)ans++;
		if(c == t)ans++;
		if(d == t)ans++;
		if(e == t)ans++;
		printf("%d\n", ans); //输出ans，注意换行
	}
	return 0;
}
```
