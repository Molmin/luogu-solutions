### **题目**
给定一个**长度为n**的串，对这个串进行**若干次操作**：**擦去**当前序列**第i个数**。

求：操作过后，**第x个数的值**。

### **分析**
直接分析数据：

| 次数 | 第一个 | 第二个 | 第三个 | 第四个 | 第五个 |
| :----------- | :----------- | :----------- | :----------- | :----------- | :----------- |
| 初始 | 1 | 2 | 3 | 4 | 5 |
| 第一次 | 2 | 3 | 4 | 5 |  |
| 第二次 | 2 | 4 | 5 |  |  |
| 第三次 | 2 | 4 |  |  |  |
会发现每次删去的数都是**奇数**

因此只要输出**x×2**即可
### **代码**
```pascal
var
 t,i,x,n:longint;//定义
begin
 read(t);//读入t
 //t表示有t行输入的数据
 for i:=1 to t do//t行数据
  begin
   read(n,x);//每行两个数n和x
   writeln(x*2);//输出x×2就行了
  end;
end.//完美结尾
```
