其他两位奆佬的题解都没有详细证明他们的结论：

>*留下来的全部都是偶数*

本蒟蒻就来证明一下吧。

题目中说明了每次操作去掉一个数。那么，对于第$i$次操作，之前已经去掉了$i-1$个数。

而对于当前的这次操作而言，要遍历到当前的第$i$个数才能取到我们要去掉的数。

设第$i$次去掉的是$f_i$，那么，$f_i=i-1+i=2i-1$，那么去掉的就是奇数。

反之，留下来的就是偶数了

```cpp
#include<bits/stdc++.h>

{
    int t;
    scanf("%d",&t);
    while(t--)
    {

    }
    return 0;
}
```
