Div.3 A

------------

**题意翻译**

给你一个大于$1$的整数$n$，现在想让你构造一个从$1$到$n$的序列$p_i$，使$p_i \ne i$，输出$p_i$。

------------

赛时做的，但没想到你谷这么晚了才爬题

可能是CF体验最好的一场了吧

对于这道题，我的做法是将 $p_i(1≤i≤n-1)$ 赋值为 $i+1$ ,然后对于最后一位 $p_n$ ，我们可以将它赋值为$1$，因为题目中的序列是从$1$到$n$的序列。

赛时刚开始想到了将序列倒序排列后直接输出，然后发现当$n=5$时序列初始为$1 2 3 4 5$，倒序排列后为$5 4 3 2 1$，此时$p_3$仍为$3$，所以是假做法。

```cpp
//代码
#include<iostream>
#include<cstdio>
using namespace std;
int main(){
	int n;
	int t;
	int a[1001];
	cin>>t;
	while(t--){//从(1->n)变为从(2->n,1); 
		cin>>n;
		a[n]=1;
		for(int i=1;i<=n-1;i++){
			a[i]=i+1;//从n->1倒序输出是不可以的，当a[]={1,2,3,4,5}时,a[3]仍为3 
		}
		for(int i=1;i<=n;++i){
			printf("%d ",a[i]);//别忘了空格
		}
		cout<<endl;
	}
	return 0;
} 

```
