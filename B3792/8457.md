> 小 A 的爸爸今年 $a$ 岁，他比小 A 年长 $b$ 岁，又比小 A 的奶奶年轻 $c$ 岁。  
> 请问小 A 和他的奶奶今年年龄多少？

我们首先来分析这一个问题：

因为小 A 的爸爸比小 A 年长 $b$ 岁，因此小 A 的年龄是 $a-b$ 岁。

因为小 A 的爸爸比奶奶年轻 $c$ 岁，因此小 A 的奶奶的年龄是 $a+c$ 岁。

接下来我们需要使用程序处理这个问题：

首先根据题意，我们需要读入这三个正整数 $a,b,c$。我们可以使用 `cin >> a >> b >> c;` 进行读入；

接着我们需要计算 $a-b$ 和 $a+c$，并且将结果输出。我们可以使用 `cout` 语句完成：`cout << a-b << " " << a+c;`

因此，这道题的代码如下：

```cpp
#include <iostream>
using namespace std;
int main() {
	int a,b,c;
	cin >> a >> b >> c;
	cout << a-b << " " << a+c << endl;
	return 0;
}
```
