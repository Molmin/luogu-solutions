#### 接着就是这题的解题思路：

其实我们不难发现，每个房间里暖气装得越多，所需花费就少。

以下是（伪）证明过程：

我们按照题目来，设此房间最多装 c 个空调，温暖度至少为 sum 。

则可以得出所需最小的花费 $ans=(\frac{\mathtt{sum}}{\mathtt{c}})^2 \times \mathtt{c}$ ，展开得 $ans=\frac{\mathtt{sum}^2}{\mathtt{c}}$ 。

相比起来，如果只装一个的话，花费 $ans=\mathtt{sum}^2$ 明显大了很多是吧？

但是我们也知道答案 ans 是个整数，所以在 $\frac{\mathtt{sum}}{\mathtt{c}}$ 中，是不能出现小数的，否则可能会导致 WA 。

嗯，就是保证每份都是整数的情况下尽量平分嘛，相信各位都是没有问题的吧？

### **Code：**
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,c,sum,ans;                              //再重复一遍，不要再浪费空间啦！ 
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>c>>sum;
		int x=sum/c;                          //此处是向下取整的平均价值 
		int y=sum-x*c;                        //判断是否有没算进去的温暖度从而做到将剩下的温暖度再平分 
		ans=0;                                //记得要归零！ 
		for(int j=1;j<=c-y;j++)ans+=x*x;      //计算小的一部分，如果直接乘起来也是一样的，这个样子可能更容易看懂一点吧 
		for(int j=1;j<=y;j++)ans+=(x+1)*(x+1);//计算剩下大的一部分，记得要将 x+1 ，因为 x 存的是向下取整的平均值 
		cout<<ans<<endl;
	}
	return 0;
}
```
