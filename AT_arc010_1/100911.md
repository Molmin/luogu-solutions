### 题意：

高桥君要去找工作了，他有 _N_ 张名片，并且他要找M天的工作，每天的工作都需要耗费一定的名片。高桥君为了保证名片充足想出了一种补充方法：如果每天的名片数小于或等于 _A_ ，那么他就会回家补充 _B_ 张名片。如果按照这种方法能坚持 _M_ 天，输出complete。否则请输出高桥君坚持的天数。

### 输入格式：
2行，第一行读入 _N，M，A，B_ ，第二行读入每天需要的名片数。

### 输出格式：
1行，complete或高桥君坚持的天数。

### 思路：
相信大家看见我的翻译都有一定思路了吧，这是一道简单的模拟题。我们只需要按照题意一天一天模拟即可，这里提供两种做法：

1. 读入后使用一个循环，一天一天处理，如果没名片了直接输出循环变量i，结束程序。然后在最后输出complete。

1. 思路和第一条一样，但是可以边读入边处理，这样只需要一个循环即可。

### 注意at的题一定要输出换行，否则爆零。
### code：
```
#include<bits/stdc++.h>
using namespace std;
int a,b,n,m,c[1001];	//c数组存放每天所需的名片数，如果边读入边处理使用一个变量就可以了。 
int main()
{
	cin>>n>>m>>a>>b;
	for(int i=1;i<=m;i++)
	{
		cin>>c[i];		
		if(n<=a)	n+=b;		//如果小于等于A，加上B 
		if(n>=c[i])	n-=c[i];	//如果名片够，直接减。 
		else					//不够输出i结束。 
		{
			cout<<i<<endl;
			return 0;
		}
	}
	cout<<"complete"<<endl;
}

```
