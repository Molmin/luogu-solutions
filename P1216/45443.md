这是道递推题，刚开始想的是贪心，可是如以下

1
1 2
1 2 3
1000000 1 1 1

贪心就不适用了

那递推怎么推呢。。。。

从下面向上推，算出各个结果，然后就知道最大结果了，我们设a[i][j]为在第i行最大的结果

就得到以下代码（为了防抄袭，仅有重要部分）

```cpp
  for (i=n-1;i>=1;i--)
   for (j=1;j<=i;j++)
     {
       if (a[i+1][j]>=a[i+1][j+1])  a[i][j]+=a[i+1][j];     //选择最大路径
       else  a[i][j]+=a[i+1][j+1];
     }
```
