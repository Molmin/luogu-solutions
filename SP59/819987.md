题意：

给出一张[有向图](https://baike.baidu.com/item/%E6%9C%89%E5%90%91%E5%9B%BE/1852743?fr=aladdin)，保证从结点 $1$ 可达其他任何节点。问删除那些节点会导致从结点 $1$ 到达不了别的节点。


------------

时间复杂度：

大概是 $O((n+m)\log n)$。


------------

算法：

先从节点 $1$ 开始 $dfs$ 出一个树。可以证明，删除节点 $i$ 会导致从结点 $1$ 到达不了别的节点，删除节点 $i$ 会导致从结点 $1$ 到它的某个子节点。

然后，沿着 $dfs$ 顺序从后向前处理每一个节点：

可以用 $pos_{i}$ 记录节点在 $dfs$ 顺序中的位置。再用 $low_{i}$ 记录能否通过父节点而到达节点 $i$ 的 $pos$ 最小的点。

考虑所有非树边的 $j>i$，有三种情况：向下的边，向上的边和横向的边。


------------
对于向下的边：

$low_{i}= \min(low_{i},pos_{j})$ 这种情况是比较明显的。


------------
对于向上的边：

$low_{i}= \min(low_{i},low_{j},low_{j>father},...)$，即从 $j$ 一直上溯到 $i$ 的 某个儿子，如果有某个 $pos$ 比 $i$ 小的节点能到达它们中的任意一个，就能使他到达 $i$。


------------
对于横向的边：

找出 $i$ 和 $j$ 的 $LCA$，记作 $k$。这个时候从 $j$ 一直上溯到 $i$ 都已经被处理过了。

因此：

$low_{i}= \min(low_{i},low_{}j,low_{j>father},...,pos_{k})$。


------------
最后，对于每个节点 $i$，如果存在某个儿子 $j$ 满足 $low_{j} \ge pos_{i}$，节点 $i$ 就是关键的。

最后在输出时排序即可。
