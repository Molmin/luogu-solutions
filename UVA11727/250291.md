### 题目大意

给定 $3$ 个整数，求中间数。

### 解题思路

循环输入 $3$ 个数（**循环值从 $1$ 开始**），再存入列表（数组），输出排序后的列表的第 $2$ 项值（即中间数）。

### 代码

Python 代码简洁又明了。

```python
t = int(input())#注意默认输入为字符串，要强制类型转换
for i in range(1, t + 1):#由 1 开始
    a, b, c = map(int, input().split())#Python 无法一行直接输入多个值
    l = [a, b, c]#存入
    l.sort()#排序
    print("Case " + str(i) + ": " + str(l[1]))#非字符串无法合并，所以要强制类型转换
```
