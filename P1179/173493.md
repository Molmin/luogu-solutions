这道题其实也**没有什么难度**，主要的思路就是把L-R之间的数给**枚举一遍再分离数位**就可以了。
首先是头文件和定义：
```
#include <cstdio>
int main(){
……
}
```
然后就是输入：
```
int r,l;//哪里需要就定义在哪里，这样读起来方便一些
scanf("%d %d",&l,&r);
```
好了，前面的部分还是很简单的，但是后面就有些麻烦了，但是我们可以大致分为这几步：
1.定义一个循环，**枚举**L-R内的数;
2.对于每一个数**进行数位分离**，并判断是否含有数字“2”，若有，则将计数器“tot”增加1;
3.输出tot;
关于第一步，这个就比较简单了，可以定义一个“i”，初始值为L，结束值为R（题目说了L<=R),代码如下：
```
int tot = 0;
for(int i = l; i <= r; i++){
……//下一步的代码
}
```
注意，运算符前面**最好打空格**，养成良好的习惯，因为在Python里面不这样做会报错的呀！
我们再来看第二步，分离数位，我们又可以分为几步：
1.使用一个变量**k代替i**（防止i被破坏）
2.循环条件为k != 0,每次使k变成k / 10,但是在这之前要判断k % 10是否等于2，如果是，则将tot + 1;
这一步似乎有些难以理解，那我举个栗子：（当k = 123）
1.k % 10 == 3;3 != 2,不改变tot,k /= 10;
2.k变成12,k % 2 == 2,2 == 2,tot + 1,k /= 10；
3.k变成1,k % 10 == 1,1 != 2,tot不变,k /= 10;
4.k变成0，循环结束;
相信这样应该能理解了吧，**不懂就再看几遍~**
代码如下：
```
int k = i;
while(k != 0){
if(k % 10 == 2)++tot;
k /= 10;
}
```
没错，这样就会从L-R找出数字2出现的次数，最后输出tot即可：
```
printf("%d",tot);
return 0;
```
**完整代码**如下：
```
#include<cstdio>
int a[100005];
int main(){
	int l,r;
    scanf("%d %d",&l,&r);
    int tot = 0;
    for(int i = l; i <= r; i++){
        int k=i;
        while(k != 0){
            if(k % 10 == 2)++tot;
            k /= 10;
        }
    }
    printf("%d",tot);
    return 0;
}
```
