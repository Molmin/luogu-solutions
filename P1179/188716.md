 由于指定了数字2，省下了不少功夫，所以只需要把每一位数字拆出来。
## 先看下拆数+判断片段
```cpp
for(int k=l;k<=r;k++)
{
    int m=k;
    while(m>=1)
    {
        k1=m%10;
        m=m/10;
        if(k1==2)
        {
            temp++;
        }
    }
}
```
原理：

利用cpp的int整除性质（例如5÷2=2.5，cpp里面还要做高斯取整，得2。类似的，20÷16=1.25，cpp得1）

先用模运算取每一位的值（k1=m%10），然后每一次把原数÷10，最后一位就~~被无情地扔飞~~没有啦（m=m/10），最后判断拆出来的数是不是2（if(k1==2)）
,是的话就加（temp++），不是的话就跳过~~~

当然，还要有个循环执行条件--

被除数不可以小于1！！！（while（m>=1））

要不然取到小数就惨了......

顺便提一句，之所以把k的值存到m里面，是~~因为爱情，在那个地方~~为了防止k的值被破坏。~~为了守护世界的和平~~

#include<iostream>
using namespace std;

int l,r,temp;

int main()
{
    int k1=0;//嫌麻烦可以放在括号外面写
    cin>>l>>r;
    for(int k=l;k<=r;k++)
    {
        int m=k;
        while(m>=1)
        {
            k1=m%10;
            m=m/10;
            if(k1==2)
            {
                temp++;
            }
        }
    }
    cout<<temp<<endl;
    return 0;
}
