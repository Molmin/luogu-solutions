其实这题思路很简单，就是从L~R中，将每个数数位分离，然后判断每个数位是否为2，是的话就sum++，最后输出sum。

数位分离小口诀：取个位，缩十倍。

话不多说，上代码。
    
    #include<iostream>  //文件头
    using namespace std;
    int l,r,sum;       //定义变量
    int w(int x)   //w是一个函数名，用来统计一个数字2的出现次数的     
    {
        int g=0;  //定义统计2出现次数的变量
        while(x)  //在x>=1时执行以下操作
        {
            g+=(x%10==2);  //如果x取个位==2，那么g++
            x/=10;  //将x去掉个位
        }
        return g;  //返回当前2的个数
    }   
    //数字分离+个数统计
    int main()
    {
        cin>>l>>r; //输入
        for(int i=l; i<=r; i++) //枚举l~r
        sum+=w(i); //调用函数，统计2的出现次数
        cout<<sum; //输出2的出现次数
        return 0;
    }
    
    
现在来解释程序中的函数部分，如分离的数字是123：

 次数　取出的个位是否为2   　2的出现次数　缩10倍后的数

 ２　　　　　取出2=2　　　　　  　１　　　　　　１
 
 ３　　　　　取出1!=2　　　　　　 1　　　　　　０
