# P1179 【数字统计】
### 题目描述
请统计某个给定范围[L, R][L,R]的所有整数中，数字 22 出现的次数。

比如给定范围[2, 22][2,22]，数字22 在数 22中出现了 11 次，在数1212 中出现 11 次，在数 2020 中出现 11次，在数 21 中出现 11 次，在数 2222 中出现 22次，所以数字22 在该范围内一共出现了 66次。
## 题解：

洛谷为什么会有这么水的NOIP的题，令我这个小蒟蒻感到很遗憾。
这是核心代码：
```
int x=i;//不可以直接用i，要找一个替代品
while(x!=0)
        {
            if(x%10==2)ans++;//如果是二ans就+1
            x/=10;
        }
```

不说了，上代码：
```c
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int n,m,ans=0;
    cin>>n>>m;//输入
    for(int i=n;i<=m;i++)
    {
        int x=i;
        while(x!=0)
        {
            if(x%10==2)ans++;
            x/=10;
        }
    }
    cout<<ans;//输出答案
    return 0;
}
```
