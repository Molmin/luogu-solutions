## 题目翻译
给你四个数$a\le b\le c\le d$，让你取三个数$a\le x\le b$，$b\le y\le c$，$c\le z\le d$，使得$x,y,z$能组成一个三角形。保证数据一定有解
## 题目分析
这个世界上，有一种三角形，叫做**等腰三角形**。等腰三角形的特殊性质：**无论底边多短，无论腰多长，它始终都是三角形**（但是必须满足二倍的腰比底边长）。

从题目中的取值范围可以看出：$x\le y$，$y\le z$。也就是说我们可以组成一个以$x,y$为腰的等腰三角形，或者一个以$y,z$为腰的等腰三角形。但是为了满足前面提到的**二倍的腰比底边长**，所以腰要尽量选长的那条。那么$y,z$就是我们的不二之选。

当$y,z$取等时，唯一的可能就是$y=c=z$，而$x$就可以随便在$[a,b]$里取值了。
## AC代码
```cpp
#include <iostream>
#include <cstdio>
using namespace std;
int t,a,b,c,d;
int main(){
	cin>>t;
	while(t--){
		a=b=c=d=0;					//先清空，不清空也罢
		cin>>a>>b>>c>>d;
		printf("%d %d %d\n",b,c,c);	//按上面所说，y,z取c，x这边随便选了个取值（也就是b，只要在a~b之间都行）
	}
	return 0;
}
```
