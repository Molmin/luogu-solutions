## 题意

共有 $t$ 组数据，对于每一组数据，给定一个 $n$ 和一个元素个数为 $n$ 的数列 $A$。其中 $n$ 为糖的种类数，$A_i$ 为第 $i$ 个种类糖的个数。有一个人一次只吃一颗糖，这颗糖是当前最多的那个种类里的一颗，知道吃完之前，他不希望有连续两次同时吃同一种类里的糖，请问他能否做到，能就输出 YES，否则输出 NO。

## 思路

非常简单！我们只需要对 $A$ 数组进行从小到大排序，然后判断，如果 $A_n-A_{n-1}\ge2$，输出 NO,否则输出 YES。

很容易证明这个算法的正确性：如果 $A_n-A_{n-1}\ge2$，那么一开始吃的两颗就是相同种类的，输出 NO。否则，一定可以满足那个人吃糖的要求。因为先吃到两种糖颗数相同，那么这两种糖交替吃；然后吃到三种糖颗数相同，那么这三种糖交替吃……因此，就避免了连续两次吃同一种糖，因此输出 YES。

## 代码

下面是本题代码，十分简洁。

```cpp
#include<bits/stdc++.h>
using namespace std;
int t,n,a[200005];
int main(){
    cin>>t;
    while(t--){
        cin>>n;
        for(int i=0;i<n;i++)cin>>a[i];
        sort(a,a+n);
        if(a[n-1]-a[n-2]>=2)cout<<"NO"<<endl;
        else cout<<"YES"<<endl;
    }
    return 0;
}
```
