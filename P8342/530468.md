首先，我们想这样一个问题：如果你有一次改变考试成绩的机会，那你会让这次的成绩尽可能高还是低？为了提高你的排名，你应该让其他人的分数尽可能高还是低？

探究完这个，其实答案就出来了：

对于这个问题，你会使自己的分数尽可能的高，而他人的分数尽可能低。放到题目里就是最好情况为你最后一轮拿了 500 分。而他人拿了零蛋。最坏排名同理。

我用的是结构体排序。条件如下：
```cpp
int cmp(abc p,abc q)//abc是选手信息结构体
{
	if(p.scr == q.scr)
	{
		return p.name < q.name ;//name表示选手姓名
	}
	return p.scr > q.scr ;//scr表示总成绩
}
```
接下来就可以直接做了。
