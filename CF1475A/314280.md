## 题目大意

判断 $n$ 是否有大于 $1$ 的奇因数，该操作重复 $t$ 次。

## 思路分析

这道题的重点在：**判断一个数是否有大于 $1$ 的奇因数** 。

## 分类讨论：

### $n=1$
肉眼观察可得，$1$ 没有大于 $1$ 的奇因数。

### $n>1$且 $n$ 是奇数

肉眼观察可得，**它本身** 就是大于 $1$ 的奇因数。

### $n>1$且 $n$ 是偶数

若 $b$ 是一个正整数，$2a$ 一定是自然数中的一个偶数。使 $n=2a$ 现在求 $2a$ 是否有大于 $1$ 的奇因数。再次肉眼观察，可得：**现在求 $2a$ 是否有大于 $1$ 的奇因数其实就是求 $a$ 是否有大于 $1$ 的奇因数** ，理由很简单： $2$ 是偶数。

然后大家就懂了，**递归** ，再次判断 $a$ 。
（~~虽然我也不是很直接这么就写的）~~。
如果$n$ 是奇数，执行以下代码后：
```python
while n%2==0:
	n/=2

```
$n$ 一定是 $1$。那么就可以把第一种和第二种情况合并到一起写。如果 $n$ 是 $1$ ，就说明没有大于 $1$ 的奇因数，否则就是有。

## $\texttt{AC Code}$：

```python
t=int(input())
n=0
while t!=0:
    t-=1
    n=int(input())
    while n%2==0:
        n/=2
    if n==1:
        print('NO')
    else:
        print('YES')

```
