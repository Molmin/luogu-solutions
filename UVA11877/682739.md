### 题意

$3$ 个空瓶换一瓶可乐，问 $n$ 个空瓶最多能换多少瓶可乐，多测以 $0$ 结束。

注意：**若有两个空瓶，可以借一瓶可乐，但最后必须还一瓶。**

### 解析

可以先模拟有 $1\sim 6$ 个空瓶的情况。

| 空瓶数 | 最多能喝到的可乐数 | 备注 |
| :-----------: | :-----------: | :-----------: |
| $1$ | $0$ | 一个空瓶，既不能借，也不能换 |
| $2$ | $1$ | 先借一瓶可乐，喝完再换一瓶还回去 |
| $3$ | $1$ | 直接换一瓶 |
| $4$ | $2$ | 分成两份 $\{2,2\}$，按 $2$ 的方式分别进行 |
| $5$ | $2$ | 分成两份 $\{2,3\}$，一份按 $2$ 的方式进行，另一份直接换一瓶 |
| $6$ | $3$ | 直接换两瓶**不是**最优解，应分成三份 $\{2,2,2\}$，然后按 $2$ 的方式分别进行 |

假设有 $i$ 个空瓶时最多能喝到 $f(i)$ 瓶可乐。通过表格，我们可以发现：

$$\forall i\geq 1,f(i) = \begin{cases}i\bmod2 = 1 & \frac{i-1}{2}\\\text{otherwise} & \frac{i}{2}\end{cases}$$

则写出代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
  int n;
  while(cin>>n)
  {
    if(n==0) break; //以0结束，退出循环
    else //若不为0
    {
      if(n%2==1) //n mod 2 = 1 时，结果为 (n-1)÷2
        cout<<(n-1)/2<<endl; //多测不换行，爆零两行泪
      else cout<<n/2<<endl; //否则，结果为 n÷2
    }
  }
  return 0;
}
```
