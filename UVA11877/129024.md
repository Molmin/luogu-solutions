**原题：[$\text{UVA11877 The Coco-Cola Store}$](https://www.luogu.org/problem/UVA11877)**
### 题意：在可乐商店里你可以用$3$个空瓶换一瓶可乐。现在给你$n$个空瓶，求你最多能喝多少瓶可乐。

### 输入多组$n$值，以$n=0$结束；对每个$n$单独一行输出答案。

**难度：普及$-$**

看完题意就知道直接模拟就可以了。

看代码注释吧。

```cpp
#include<cstdio>
int main()
{
    int n,ans;//n代表拥有的空瓶个数，sum代表答案
    scanf("%d",&n);
    while(n){//注意输入以一行0结束
        ans=0;//清零上一轮得到的答案
        while(n>=3){//空瓶个数大于3的时候可以换可乐
            ans+=n/3;//每3个瓶子换一瓶可乐
            n=n%3+n/3;//n/3是得到的可乐数(最后变成空瓶)，n%3是换可乐的时候剩下的空瓶
        }
        if(n==2)ans++;//这是题目中给出的提示：如果你有2个空瓶，可以向商店借1个空瓶换一瓶可乐，喝完把空瓶还给商家
        printf("%d\n",ans);
        scanf("%d",&n);
    }
    return 0;
}
```
