首先判断$n$是否无解，很明显$n$为奇数时必分解出$2^0$，而$n$为偶数时不会有$2^0$，因为$2|$$2^k$$(k∈N^+)$。则如果$n$为奇数则无解。

数据范围$n\le1e7$我们可以专门开一个数组来表示$n$的二进制表达式，30就够了。$n$每次都除以$2$

最后再扫一遍就完成了，上代码（自（fan）写（zheng）的（wo）函（jiu）数（shi）难（lan）看（de）死（yi）了（pi）代码中用$pow$函数）

#include<bits/stdc++.h>
using namespace std;
int n, a[35] = {0};
int main(){
	cin>>n;
	if(n & 1){
		cout<<-1;
		return 0;
	}
	int i = 0;
	while(n){
		if(n & 1)
			a[i]++;
		n /= 2;
		i++;
	}
	for(int i = 30;i >= 1;i--){
		if(a[i]){
			n = pow(2, i);//循环利用保护环境（雾
				cout<<n<<' ';
		}
	}
     return 0;
}
