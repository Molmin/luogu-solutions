## 题意理解
#### - 若干个到底包不包括一个？
注意到题中有这样一句话：


------------


一般来说，一个正整数可以拆分成若干个正整数的和。

例如，1=1，10=1+2+3+410=1+2+3+4 等。


------------
这里题目在写明“若干个”的情况下，列举出了“1=1”的拆分。

所以在本题中，“若干个”包括一个。


------------

## 题目分析
1. 当n为奇数时：

显然2的正整数次幂都是2的倍数。

所以奇数一定不存在“优秀的拆分”。

输出“-1”。

2.当n为偶数时：


------------


找到不超过n的最大的2的次幂pw；

由最大性，pw<=n<pw*2;

所以  0<=n-pw<pw;

因此从n中拆出一个pw后，

对n-pw的分拆不会与pw重复；

将n-pw作为新的n进行分拆（此时n依然为偶数）；



------------
如此进行若干次；

显然pw是递减的；

所以  若干次后，总存在一个n<2;

又因为n为偶数，

所以  此时n=0；

n=0时停止操作即完成分拆。



------------
代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int n;
int pw=2;
int main(){
  cin>>n;
  if(n%2==1){
     cout<<"-1";//当n为奇数时
  }else{
     while(pw*2<=n){
        pw*=2;
     }
     n-=pw;
     cout<<pw<<" ";
     while(n>0){
        while(pw>n){
           pw/=2;
        }
        n-=pw;
        cout<<pw<<" ";
     }//重复拆出最大的不超过n的2的次幂并输出
  }
  return 0;
} 
```
