华丽的分割线：
------------
#### 以下为解释：

因为该数可以被拆分为2的整数次幂，所以先特判是否为奇数，若是，直接输出-1，如果不是，其实就可以确定一定有解。（~~不知道为什么，考场手推的~~）

然后就从大往小找第一个比原数小的2的整数次幂，然后输出，再用原数减去该2的整数次幂，作为新的数循环，直到该数为0循环结束。

下面上代码：

```
#include<bits/stdc++.h>
using namespace std;
int n; 
int sou(int x){
	int sum=2;
	for(int i=2;i<=30;i++){
		if(sum>x) return sum/2;
		sum=sum*2; 
	}
}
int main(){
	cin>>n;
    if(n%2==1){
    	cout<<"-1"<<endl;
    	return 0;
	} 
	while(n!=0){
		cout<<sou(n)<<" ";
		n-=sou(n);
	}
    return 0;
}

```
