**题目大意：求 $N$ 能被多少个素数整除。**

--------------------------------------------

**注意事项：**

**1.** 循环到 $N$ 为 $0$ 为止。

**2.** 本题考查对素数的运用。

**3.** 可用**暴力**与**筛法**。

---------------------------------------------

**代码实现**

如果枚举的数为素数，则进行一遍筛法，并进行累加：

```cpp
for(int i=2;i<=1000000;i++)
        if(!a[i])
            for(int j=i+i;j<=1000000;j+=i)
                a[j]++;
```
--------------------------------------------

最终代码如下：

```cpp
#include<bits/stdc++.h>
using namespace std;
int a[1000005],n;
int main()
{
    for(int i=2;i<=1000000;i++)
        if(!a[i])
            for(int j=i+i;j<=1000000;j+=i)
                a[j]++;
    cin >> n;
    while(n)
    {
        cout << n << " : " << a[n] << "\n";
        cin >> n;
    }
    return 0;
}
```
