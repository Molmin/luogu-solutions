本题其实是让你求输入的数是不是某个数的六次幂。式子：

$a(n)=n$$^6$

如果是，就输出Special

不然就输出Ordinary

其实本题是可以打表的，不用手算。

扔到OEIS上~~榨汁~~

发现这个是OEIS上的第A001014数列

把这个数列复制出来：

```
0,1,64,729,4096,15625,46656,117649,262144,531441，

1000000,1771561,2985984,4826809,7529536,11390625，

16777216,24137569,34012224,47045881,64000000，85766121
 
```

注意：如果输入的数为0，那就要直接return 0

然后写程序（easy！）：

```cpp
#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>
#include<string>
#include<cstdlib>
using namespace std;
int n,a[22]={0,1,64,729,4096,15625,46656,117649,262144,531441,1000000,1771561,2985984,4826809,7529536,11390625,16777216,24137569,34012224,47045881,64000000,85766121};
int main(){
	while(cin>>n&&n){
		int e=0;
		for(int i=1;i<=21;i++)
			if(n==a[i]){
				cout<<"Special\n";
				e=1;
				break;
			}
		if(!e) cout<<"Ordinary\n";
	}
 	return 0;
}
```
