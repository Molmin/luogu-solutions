首先来看一下题。

题意大致就是讲这条鱼过得比人都好，早上干劲十足游150公里，下午累了也能游100公里，晚上休息，周末也休息，不加班不迟到不早退，~~xiongdei这可是会被开除的。~~
好的我们的输入是两个数字x、n，一个代表现在是周x，另一个代表小鱼想知道n天之后自己能游过多少路程。所以这道题的重点在于周末——如何在遇到周末的时候进行转换。其他的由于没有早晨或者下午要睡午觉或者早上没起得来，直接加250就是的（100+150），所以这道题的确不是一道难题。


虽然这不是一道难题，但用代码是如何实现的呢？先贴上AC代码：
```c
#include<bits/stdc++.h>//这个是万能头文件！有的网站是不能用的比如vjudge似乎就不能用，小伙伴们注意了！
using namespace std;//其实在这道题里面可要可不要
int main()
{
    int n,k,ans=0;//ans用来输出数据。
    scanf("%d %d",&n,&k);//输入
    for(int i=1;i<=k;i++)
    {
        if(n!=6 && n!=7) ans+=250;//ans+=250就是ans=ans+250
        if(n==7) n=1;//如果过完了周末，就没有周八了，所以从周一开始
        else n++;//如果不是周末就跳转到第二天。
    }
    printf("%d",ans);//输出
    return 0;
}

```
其实看上去还是很简单的。

思路我在之前已经说过了，现在再总结一遍：
如果并非双休日，总路程上就加上250（100+150），如果是双休日就不加，如果是周末则跳到下一周。
其实还是很容易来着xxx，小可爱们自行体会吧！
虽然知道小伙伴们应该学过循环或者判断了，依然给大家补一个for循环和if的格式：
```c
for(一个变量，通常是i,j;循环终止条件;循环进行中的变量变化，通常为i++或者j++);{}
if(进行子语句条件){}

```
如果循环/判定后面需要进行多个子语句则需要加{}，如果只有一个则不需要。（注意：有多个子语句的循环也只算一个子语句哦！）
