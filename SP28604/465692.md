# 0x00 思路
这是一道数论题。它想让我们去判断它给我们的数的因数个数是不是奇数个。

那么到底有哪些数的因数是奇数个呢？

这里我可以直接跟大家说：是平方数。

那么我们可以直接判断它是不是平方数即可。

# 0x01 论证
为什么只有平方数的因数个数是奇数个呢？

是这样的：我们可以将一个数分成两个数相乘的形式。

假若此数不是平方数，那么你将这个数分成的所有两两相乘的数一定不一样，所以这个数的因数一定是偶数个。

而当此数是平方数的时候，它必定会分出一组两个相同的数，而这两个相同的数只增加了一个因子，而其他数对都增加了两个因子，也即：此数的因子数为 $2 \times k - 1$ 或 $2 \times (k-1)+1$。

由于 $2 \times k,2 \times (k-1)$ 都为偶数，$1$ 为奇数，所以平方数的因子个数为奇数。

# code
```c
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int t,n,x;
    scanf("%d",&t);
    while(t--)
    {
        scanf("%d",&n);
        x=sqrt(n);
        if(x*x==n)
            printf("YES\n");
        else
            printf("NO\n");
    }
    return 0;
}
```

# 0x02 解释
由于应该有不少人在看我的代码时有一些疑惑，让我来解答一下。

1. `sqrt` 是什么？
	
    `sqrt` 是 `c++ cmath` 头文件中自带的开方运算函数。类型为浮点数。
1. 为什么用 `int` 类型的变量存储？

	因为 `sqrt(n)*sqrt(n)` 一定等于 `n` ，而我们这么做就是为了剔除小数点后的数只剩下整数，而平方数开方后仍是整数，所以相乘仍然等于 `n`，而其它则不然。
