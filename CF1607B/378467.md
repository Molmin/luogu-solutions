**题解**

由于 $n$ 的范围很大，暴力枚举肯定不可行。所以这一道题一定是一道思维题。

我们现在定义初始位置为 $5$。

第 $1$ 次移动：$6$。

第 $2$ 次移动：$4$。

第 $3$ 次移动：$1$。

第 $4$ 次移动：$5$。

第 $5$ 次移动：$10$。

第 $6$ 次移动：$4$。

第 $7$ 次移动：$-3$。

第 $8$ 次移动：$5$。

第 $9$ 次移动：$14$。

第 $10$ 次移动：$4$。

第 $11$ 次移动：$-7$。

第 $12$ 次移动：$5$。

以此类推。

我们可以发现：每移动 $4$ 次，就会回到初始的原点。

那么我们只用枚举当前循环的 $4$ 次，就可以求出答案了。


```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int T;
    cin >> T;
    while (T --)
    {
        long long x0, n;
        scanf ("%lld%lld", &x0, &n);
        long long a = n / 4 * 4; // 求出现在已经移动过的次数，由于C++是向下取整的，所以我们可以使用这一种方式来求值。
        long long now = x0; // 当前的位置
        a ++; // 移动了一步
        while (a <= n) // 时间复杂度 O(1)
        {
            long long dist = a; // 向右移动 dist 步
            // 其中向右移动 -1 步就代表向左移动 1 步，向右移动 -2 步就代表向左移动 2 步
            if ((now & 1LL) == 0)
                dist = -dist;
            now += dist;
            a ++;
        }
        printf ("%lld\n", now);
    }
    return 0;
}

```
