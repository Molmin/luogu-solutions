# 双向判断！！！
其实我们仔细观察一下可以发现，这些房子的标价可以转换成0或1来判断！！！**如果标价大于0并且不大于男主手中的钱，则认为可以购买，记录为1，否则认为不可购买，记录为0。**

然后我们定义l和r，**分别从左右两边查找是否出现房子记录为1的值**，如果有，就输出。

此处我们定义l和r的起点都是女主家，**所以发现可购买的房子必定是最短距离！！！**

那么就有AC代码：
```cpp
#include <iostream>
using namespace std;
int a[100001];
int main(){
	int i,j,k,n,place,money;
	cin>>n>>place>>money;
	for(i=1;i<=n;i++){
		cin>>k;
		if(k>0&&k<=money)a[i]=1;//判断：如果满足条件，认为可购买，a[i]=1
	}
	int l=place,r=place;//设置起点，l和r都是在女主家
	while(1){
		l--;//查找左边
		r++;//查找右边
		if(l>0&&a[l]==1){//注意：此处还要给l设定范围，否则会超出！！！
			cout<<(place-l)*10<<endl;//别忘了*10
			return 0;
		}
		if(r<=n&&a[r]==1){//注意：此处还要给r设定范围，否则会超出！！！
			cout<<(r-place)*10<<endl;//同样
			return 0;
		}
		if(l<=0&&r>n)return 0;
	}
}
```

大家想想为什么不把while语句改变成：
```cpp
while(l>0&&r<=n){
```
其实是女主家不一定在中间，这样很有可能l出界但r不出界；但如果改成||的话，再判断它们单独是否出界，比较麻烦。
