经典的广搜思想，不过不用广搜

我们可以从菇凉的房子开始，向左和右扩散，如果有一个房子达到要求，也就是价格小于等于```k```，那么这间房子Zane就要定了

所以我们定义一个```c```表示Zane最多里菇凉多少个10米，每次循环```c++```，直到```c + m > n```或者```m - c < 1```的时候就退出；如果```a[c + m]```或者```a[m - c]```这个位置没有人而且价格小于```k```元的话这间房子就买了，也就是输出```10 * c```，退出循环

所以代码就出来了，注释也在里面了：

```
#include <iostream>                                                             //毒瘤头文件
using namespace std;                                                            //毒瘤命名空间

int n, m, k, c, a[100001];                                                      //定义几个变态变量

int main() {
    cin >> n >> m >> k;                                                         //输入三个毒瘤变量
    for (int i = 1; i <= n; i++) cin >> a[i];                                   //循环到n输入数组
    while(1) {                                                                  //没达条件一直循环
        c++;                                                                    //C一定一定要加加
        if(m - c < 1 && m + c > n) break;                                       //如果越界退出循环
        if(m - c >= 1 && a[m - c] != 0 && a[m - c] <= k) {cout << 10 * c;break;}//左边达到条件输出
        if(m + c <= n && a[m + c] != 0 && a[m + c] <= k) {cout << 10 * c;break;}//右边达到条件输出
    }
}
```
