容斥原理典型题目。

1.最大值，就是构造一种情况，使回答一个问题的人**尽量都**回答了另一个问题。

求出 $A$ 和 $B$ 的最小值即可。、

2.最小值，就是构造一种情况，使回答一个问题的人**尽量都不**回答另一个问题。

求出 $A+B-N$ 即可。

3.但要注意，最小值还可能出现这样的情况，就是**不管怎样构造，都会有问题没人回答**。

这时最小值不为负数，为零，所以要进行特判输出。

------------

AC代码：

```cpp
#include<iostream>
using namespace std;
int main(){
    int N,a,b;
    cin>>N>>a>>b;
    cout<<min(a,b)<<" ";
    if(a+b<=N){
        cout<<0;//注意这里必须特判！否则样例都炸，更别说测试点了
    }
    else{
        cout<<a+b-N;
    }
}
```
