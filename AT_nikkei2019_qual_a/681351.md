一道简单**数学题**

**思路**

首先，求最大值，我们要尽可能多的把做上来 $x$ 题或 $y$ 题的都算上，但不可能都互相包含，所以就是两个之中被包含的那一个，也就是两个之中的最小值，因为最小的那个一定回答了另一个问题。求最小值，也就是让所有人都只回答某一个问题，可是，如果 $a+b$ 超过了人数总数，就多了一些人那些人就是回答了两个问题的人，也就是 $a+b-n$但是，如果没有超过人数总数，也就是负数,就要输出 $0$,我们就要再判断一下是否小于 $0$。

**贴代码**

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
    int n,a,b;
    cin>>n>>a>>b;
    cout<<min(a,b)<<" "<<max(a+b-n,0)<<endl;
    return 0;
}
```
