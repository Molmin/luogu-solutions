这是一道容斥原理的简单题。

先考虑最大化两个问题都回答的人数：

由于我们要让两个问题都回答的人数最大化，必然考虑让回答某种问题的人全部去回答另一个问题，显然的，我们只能让 $A$，$B$ 两批的人中取人较小的一组全部去补另一个问题的空白，因此答案为 `min(A,B)`。

再考虑最小化：

与前一种思路相反的，我们让回答 $X$ 问题的人全部不要去回答 $Y$ 问题，回答 $Y$ 问题的人全部不要去回答 $X$ 问题。显然这种思路在 $A + B \leq N$ 时显然成立，在这种情况下答案为 $0$。但如果 $A + B > N$，我们就只能让多出来的那一部分人（可能是回答 $X$ 问题的也可能是回答 $Y$ 问题的）去做原本不属于他们的问题，答案即 $A + B - N$。用 `if` 判一下就可以了。

注意事项：

- 就像刚才所说的那样，**不要在输出最小化的时候统一输出 $A + B - N$**，注意有的时候回答问题的人数小于总人数 $N$，所以总不会有负数个人同时回答两个问题吧。

考虑了这些，代码便好写了。

```cpp
#include <iostream>
using namespace std;
int main(){
    int n,a,b;
    cin>>n>>a>>b;
    cout<<min(a,b)<<" ";//这里是统一的，不要考虑特判，毕竟判了也没用
    if(a+b<=n) cout<<0;//注意这里的特判，不然样例都会炸
    else cout<<a+b-n;
    return 0;
}
```
