# 一道**~~(水)~~**题
## 思路：判断有多少相邻的相同字母
## 接下来上代码:
```cpp
#include<cstdio>
int a,d;//a就是题目中的n，d为计数器
char b,c;//这俩字符代替字符串s，b为当前字符，c为b的前一个字母
main(){
	scanf("%d",&a);//输入字符串长度
	while((b=getchar())=='\n');//过滤掉换行
	c=b;
    //由于过滤完换行后已经输入了字符串s的第一个字母，所以b从第二个字母开始，c赋值为b
	while(--a){	//因为已输入一个字母，所以只剩n-1个字母需要输入
		d+=((b=getchar())==c);//判断b是否等于c，这里用表达式省略了if
		c=b;//将c赋值为b
	}
	printf("%d",d);//输出
}
```
