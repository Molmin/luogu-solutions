这道题很简单，只需要将这其中的奥妙，或者说是一个谜解出来就行了。

我们先看题，A，B，C 三个家庭都打扫一个花园，由题可知他们的效率都是一样的，因为 A 花了 5 个小时，B 花了 4 个小时。所以一个家庭打扫整个花园需要 9 个小时，每个家庭的效率为 $\frac{1}{9}$，既然如此按照表面来看，A 应该拿到 $\frac{1}{9}\times5\times90$ 元，但是大家要想原来 A，B，C 家庭都在的时候，每个人的工作量都是 $\frac{1}{3}$，所以说有些部分是他们本应该做的，不能加入工钱分配，所以要减去，那么 A 应该拿到的真正工资是如下公式：

$C\times(A-(A+B)/3)/((A+B)/3)$。 

化简为 $C\times(2\times A-B)/(A+B)$;

既然都已经有了公式，那我们就直接上代码了：
```cpp
#include<bits/stdc++.h>
const int MAX=1000;
using namespace std; 
int a1[MAX];//用来保存A家应该拿到的钱。 
int main(){
	int i,n,a,b,x; 
	scanf("%d",&n);
	for(i=1;i<=n;i++){
		scanf("%d %d %d",&a,&b,&x);
		a1[i]=x*(2*a-b)/(a+b);
	}
	for(i=1;i<=n;i++)
		printf("%d\n",a1[i]);
	return 0;
}
```
