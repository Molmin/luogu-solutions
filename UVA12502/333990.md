## Thinking

[原题](https://www.luogu.com.cn/problem/UVA12502)题意简化：


------------


三户人家 $A$、$B$、$C$ 有一个大花园，平日三户人家一起打理花园。

今天 $C$ 度假去了，没有打理花园，所以 $A$ 和 $B$ 两家人帮 $C$ 做完了。

已知 $A$ 花费了 $x$ 个小时，$B$ 花费了 $y$ 个小时。

$C$ 为了感谢他们，给他们共 $z$ 元钱。问 $A$ 应得多少钱。



------------


这个问题可以理解为**工程问题**。$A$、$B$两家不仅要做完自己的任务，还要平摊 $C$ 的任务。

因为 $A$ 花费了 $x$ 个小时，$B$ 花费了 $y$ 个小时，所以总耗时为 $x + y$ 个小时。

故：$A$ 与 $B$ 的工作总量为 $\frac{x + y}{3}$，$A$ 帮助 $C$ 做了 $x - \frac{x + y}{3} = \frac{2x - y}{3}$ 个小时，$B$ 帮助 $C$ 做了 $y - \frac{x + y}{3} = \frac{2y - x}{3}$ 个小时。

那么一份就是 $\frac{z}{\frac{2x - y}{3} + \frac{2y - x}{3}}$。因为问的是给 $A$ 多少钱，所以乘上 $A$ 帮 $C$ 做的时间 $\frac{2x - y}{3}$，得到 $\frac{2x - y}{3} * \frac{z}{\frac{2x - y}{3} + \frac{2y - x}{3}}$，简化后为 $\frac{2xz - yz}{x + y}$。

坑点：

- 有**多组数据**。

- 输出多组数据要**换行**。

## Coding

代码驾到！

```cpp
#include<bits/stdc++.h>
using namespace std;
int t, x, y, z;
int main(){
	scanf("%d", &t); //C语言版输入，输入组数
	while(t--){ //循环每一组输入
		scanf("%d%d%d", &x, &y, &z);
		printf("%d\n", ((2 * x * z - y * z) / (x +  y))); //直接输出刚才推导出的公式
	}
	return 0;
}
```
