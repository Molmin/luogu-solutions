如何判断一个点是否会积水呢？我们暂且假设这个点左右两个点和这个点高度不相同，那么只要满足这个点左右两个点的高度都大于这个点就可以了，但是这么做考虑不到平地的问题，但是我们发现，一个积水的盆地才满足刚才那个条件，于是我们就加一句代码，使任意一个点的左右两点都和该点高度不同，详见代码:
```cpp
#include<bits/stdc++.h>
using namespace std;
int ans,n,a[100001],aa,cnt;
int main()
{
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		cin>>aa;
		if(aa!=a[cnt]) //判重
		a[++cnt]=aa;
	}
	for(int i=2;i<cnt;i++)
	{
		if(a[i]<a[i-1]&&a[i]<a[i+1])
		ans++;
	}
	cout<<ans;
 } 
```
