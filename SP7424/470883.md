## 题目翻译：
一个即将毕业的班级里有 $G$个女生和 $B$个男生。你需要在毕业典礼上把它们排成一排。为了给人一种多样化的印象，你应该避免连续坐太多女孩或太多男孩。你决定安排学生是为了减少男女比例。排列的性别规律性是指连续出现的同性别学生(全是女生或全是男生)的最大数量。已知 $G$ 和 $B$ ，计算所有可能安排中最小的性别规律性。

## 思路：
其实就是把**男生的个数平均分成女生的个数份，再向上取整**。可以更简化得出数学公式 $\lceil \frac{B}{G+1} \rceil$ (B表示boy,G表示girl)。

有了公式我们只需要套下公式就可以了。



## AC CODE：
```
#include<bits/stdc++.h> //万能头文件，懒人专属。
using namespace std;
  
int main() 
{
	int b,g;
	while(cin>>b>>g)//输入数量。
   	{
		if(b<g)
      		{
			swap(b,g);//男生小于女生就用swap函数交换。
		}
		if(b==-1)
        	{
			break; //-1就不管它了。
		}
		else if(g==0)
        	{
			cout<<b<<endl; //是0直接输出。
		}
		else
        	{
			cout<<ceil((double)(b)/(double)(g+1))<<endl; // ceil函数，向上取整（ceil在英语里就是天花板的意思），(double)是强制类型转换将b,g（int)转换为double型的b,g(因为有除法)。
		}
	}
	return 0;
}
```
