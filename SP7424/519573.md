## 题目大意：

一个即将毕业的班级里有 $G$ 个女生和 $B$ 个男生。你需要在毕业典礼上把它们排成一排。为了给人一种多样化的印象，你应该避免连续坐太多女孩或太多男孩。你决定安排学生是为了减少男女比例。排列的性别规律性是指连续出现的同性别学生(全是女生或全是男生)的最大数量。已知 $G$ 和 $B$ ，计算所有可能安排中最小的性别规律性。

### 具体思路

处理边界条件，假设两个女生紧挨在一起，那么我们就会分为两种情况讨论：第一，把两个女生拆开，把一个男生掺进去；第二：把一个女生拿走，再用一个男生替换另一个女生的位置。通过这两步操作，使得男女生可以被平均分配。

### 解决方案

1. 小学奥数抽屉原理的应用（作除法，平均分配，多余的再分配）；

2. 找出通项公式；

3. 不符合题目要求的直接特判输出。

### 上代码：

```cpp
#include<iostream>
#include<cmath>
#include<math.h>
#include<stdio.h>
#include<ctype.h> 
#include<float.h>

void do_(int x, int y){
	if(x == -1){ //方案无法进行，结束。
		return ;
	}
	if(x < y){//如果男生多于女生，交换男女生的位置，方便计算。
		std::swap(x, y);
	}
	else if(y == 0){//除数不能为"0"，进行特判。
		std::cout << x << std::endl;//直接输出。
	}
	else{
		std::cout << ceil(x / (y + 1)) << std::endl; //鸽巢原理。
	}
}

int main() {
	double boy, girl;
	while(std::cin >> boy >> girl){
		do_(boy, girl);
	}
	return 0;
}
```
本题时间复杂度为 $O(1)$ ，直接切掉。
