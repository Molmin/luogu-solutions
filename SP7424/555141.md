# 题目翻译
一个即将毕业的班级里有 $G$ 个女生和 $B$ 个男生。你需要在毕业典礼上把它们排成一排。为了给人一种多样化的印象，你应该避免连续坐太多女孩或太多男孩。你决定安排学生是为了减少男女比例。排列的性别规律性是指连续出现的同性别学生(全是女生或全是男生)的最大数量。已知 $G$ 和 $B$ ，计算所有可能安排中最小的性别规律性。

# 思路
题目本蒟蒻一开始看了很多遍都难以理解，其实就是求相邻的几个人性别相同的连续排列的**最小值**。想要求最小值，不妨把人多的性别平均分成人少的性别份，最后向上取整，即是答案。

# 代码实现

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int B,G,temp;
	while(cin>>B>>G)
	{
		if(B<G) //如果女生大于男生，交换
		{
			temp=B;
			B=G;
			G=temp;
		}
		if(B==-1)
		{
			break;
		}
		else if(B==0)//如果没有男生直接输出女生
		{
			cout<<G;
		}
		else if(G==0)//如果没有女生直接输出男生
		{
			cout<<B;
		}
		else
		{
			cout<<ceil((double)(B)/(double)(G+1));//计算人多的性别平均分成人少的性别份，并向上取整
			cout<<endl;//换行
		}
	}
	return 0;//完美结束
}
```
