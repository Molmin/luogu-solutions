题意很清晰。

给出 $n$ 个女生和 $m$ 个男生，将他们排列。排完后会出现若干个连续性别相同的段，最小化这些段中长度的最大值。

为了方便考虑，我们认为女生比男生多。毕竟此题女多还是男多都无所谓。

如果出现两个男生挨在一起，拆开他俩，拿出一个男生塞到连续的一段女生里，答案不会更劣。接下来我们可以看作是把男生放入一列女生当中，使得女生被划分的尽量均匀。


~~自己 yy 一下~~自己举几个栗子就会发现，如果 $n$ 是 $m+1$ 的整数倍，那么女生就能被划分的再均匀不过了，如果男生放得够优秀，也就是到了一个“不论在当前哪个位置放一个女生，答案一定变大”的状态。如果 $n$ 不是 $m+1$ 的整数倍，我们先让**尽可能少的女生**出队，使得 $n$ 是 $m+1$ 的整数倍。再请出队的女生回来，依次加进当前的连续的女生中，显然答案可以做到比她们进来之前只多 $1$。

$0$ 要特判。

代码：

```
#include<bits/stdc++.h>
using namespace std;
int main(){
	int n,m;
	while(cin>>n>>m&&n!=-1){
		if(n<m)
			swap(n,m);
		if(m==0){
			cout<<n<<endl;
			continue;
		}
		if(n%(m+1)==0)
			cout<<n/(m+1)<<endl;
		else
			cout<<n/(m+1)+1<<endl;
	}
	return 0;
}
```
