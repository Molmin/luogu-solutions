题意：男女生站在一排，求相邻几个人性别相同的连续排列最小值。

不妨把这个问题看成**鸽巢问题**。

先设性别较少的是女生。男生比女生多，把女生想象成巢，两女生夹一个男生。但是显然的是，最靠边的男生没必要夹，因为他们左边/右边已经没有人了。

根据鸽巢原理，我们需要尽可能把鸽子平均化，然后才能取到最小值。由于最终取的是最多的一组（鸽子最多的巢），**所以我们需要向上取整**。

但是这里要用到除法，显然不能用整除，我们可以将它**变为浮点类型**再除。

我们只需要按照思路模拟即可。


```cpp
#include<bits/stdc++.h>
using namespace std;
int G,B;
int main(){
	while(1){
		scanf("%d%d",&G,&B);
		if(B<G)swap(B,G);//因为我们设男生大于女生，所以如果与情况相背我们可以交换性别
		if(G==-1&&B==-1)break;//如果是-1退出
		else if(G==0&&B==0)printf("0\n");
		else if(G==0)printf("%d\n",B);	//特判除数为0的情况
		else printf("%d\n",ceil(double(G,B)/(double(G+1))));//ceil是c++的自带向上取整函数
	}
	return 0;
}
```
