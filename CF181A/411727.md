### $\text{Solution 1}$：思路
首先，要组成矩形的话，那么这 $4$ 个点的坐标要有一定的要求。

事实上，这些点的坐标有一个恒古不变的定律:

想象一下自己面前有个长方形，那么假设第一个点的坐标为 $(x1,y1)$ ,第二个点是和 第一个点同一行的点，坐标设置为 $(x1,y2)$ ,那么第三个点是和 第一个点同一列的那个点，坐标为 $(x2,y1)$ ,第四个点是和第三个点同一行的那个点,坐标为 $(x2,y2)$。

下面给出一个不怎么样的图：
```c++
*1**2*
******
******
*3**4*
```

其中，$1,2,3,4$ 分别代表了第一，二，三，四个点。

大家会发现:上面4个点的坐标 $x1,x2,y1,y2$ 每个出现了两次，也就是每个都出现了偶数个。

所以在另一个点没有给的情况下，$x1,x2,y1,y2$ 就有两个出现偶数个，另外两个出现奇数个。

这么说来，给定的那三个点（总共有 $6$ 个坐标）只出现一次的就是第四个点的坐标.

说了这么多，来总结一个结论:**在前三个点中，只出现一次的横坐标是第四个点的横坐标，只出现一次的纵坐标是第四个点的纵坐标**。
### $\text{Solution 2}$：代码
```c++
#include<bits/stdc++.h>
using namespace std;
int n,m;
char a[105][105];
int x[505],y[505];
int l;
int main(){
	cin>>n>>m;
	for(int i=1;i<=n;i++){
		for(int j=1;j<=m;j++){
			cin>>a[i][j];
			if(a[i][j]=='*'){
				x[i]++;//统计当前的点的横坐标出现了多少次
				y[j]++;//统计当前的点的纵坐标出现了多少次
			}
		}
	}
//	x[1] y[1]
//	x[2] y[2]
//	x[3] y[3]
//  
	for(int i=1;i<=n;i++){
		if(x[i]==1){//如果只出现1次输出
			cout<<i<<' ';
		}
	}
	for(int i=1;i<=m;i++){
		if(y[i]==1){//如果只出现1次输出
			cout<<i<<' ';
		}
	}
}
```
