# CF765C 题解
**思路**

本题要想比赛场数多，只需要让双方轮流获胜就可以了，但是在本题中可能出现胜利分 $a, b$ 未用完的情况。

- 第一种是双方分数都小于要求分。
- 第二种是一方赢不了整数轮而另一方不够胜一轮。

# AC Code
```c
#include <bits/stdc++.h>
using namespace std;
int main()
{
	int k, a, b;//要求分与双方得分
	cin >> k >> a >> b;
	if ((a < k && b < k) || (a % k != 0 && b < k) || (b % k != 0 && a < k))
	cout << "-1";//根本不够赢或一方赢不了整数轮另一方不够赢 
	else cout << a / k + b / k; //双方轮流赢 
} 
```
