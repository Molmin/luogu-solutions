“/”和“\”都只占半个格子，我们只需要判断“.”是在图形的里面还是外面就行了。

这道题的解法就是一行一行的过，从“/”或“\”开始统计，到另外一个“/”或“\”结束统计，其间的都在图形内部，另外的都在图形外部，用flag来判断。

```
4 4
/\/\
\../
.\.\
..\/
```
例如在样例的第三行：

1. 开始输入的“.”在外面，不计算面积。

2. 遇到第一个“\”flag=1，显示在图形内部，计算一半面积。

3. 然后遇到“.”因为flag=1所以在图形里面并计算面积

4. 最后遇到“\”flag=0，计算一半面积

那么我再发一个极简代码吧：

```
#include<bits/stdc++.h> 
using namespace std; 
int n,m,flag,area; //n和m是长和宽，flag是判断是否在图形外，area是面积（由于不想涉及小数，所以半个算作一个，一个算作两个） 
char c; //正在读入的字符，会被下一个替换 
int main(){
	cin>>n>>m;  
	for(int i=1;i<=n*m;i++)//直接n乘m，不需要二层循环 
	{
		cin>>c; //flag等于1表示在图形内部，等于0在外部 
		if(c=='/'||c=='\\')  flag=!flag,area++; //由于反斜杠的重要性，所以C++规定必须使用两个反斜杠来代表这个字符 
		else if(c=='.'&&flag) area+=2; //flag==1可以省略成flag flag!=flag就是1变0，0变1 
	}
	cout<<area/2; //最后除以二（原因第三行） 
}
```

1. 短

2. 注释全

3. 楼上两层循环，这里一层
