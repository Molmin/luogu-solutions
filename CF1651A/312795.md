这道题算是一道比较水的题。

首先先来考虑第一次比赛，第一次比赛竞争的两队 $x$ 和 $y$ 可以根据题目得知，它们的编号一定是一奇一偶，那么编号之和是奇数，就会竞选出编号较小的一队，可以看出，竞选出的一队就是它们中的编号是奇数的一队。

只剩下编号为奇数的队伍，那么其中相互比拼的两个队伍 $x$ 和 $y$，它们的编号之和一定是偶数，就会竞选出编号较大的一队。到了最后，竞选出的队伍的编号的一定是最大的奇数，也就是 $2^n-1$。


代码：
```cpp
#include <bits/stdc++.h>
using namespace std;
int t, n;
int main() {
	cin >> t;
	while(t--) {
		cin >> n;
		cout << (1 << n) - 1 << endl;
	}
	return 0;
}
```
