非常简单的思维题。
### 题目大意：

$2^n$ 名选手，两两成对。当编号为 $x$ 和 $y$ 的选手比赛时，如果他们的和是奇数，谁的编号小谁赢；当和为偶数时，谁的编号大谁赢。求最后胜利的人的编号。

以样例为例，如图：

![](https://cdn.luogu.com.cn/upload/vjudge_pic/CF1651A/5e1871b96c9101b943a5400444d0518e867eae14.png)

- 第一轮比赛时，每两个选手的编号和为奇数，每一组中奇数比偶数小，所以编号为奇数的人晋级。
- 下一轮，编号和为偶数，编号大的人晋级。
- 最后一轮，编号大的人晋级。

所以不难得出，所有的偶数都被淘汰，第二轮往后晋级的都是每组中较大的奇数，而最后的胜者一定是 $2^n$ 名奇数选手中编号最大的。

答案为 $2^n-1$。

$\text{Code}:$
```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int n,t;
	cin >> t;
	while(t--){
		cin >> n;
		cout << (1<<n)-1 << endl;
	}
	system("pause");
	return 0;
}
```
