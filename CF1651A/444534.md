## 题意
考虑场数为 $2^n$ 的运动员比赛。运动员的编号从 $1$ 到 $2^n$。

分 $n$ 个阶段举行。在每个阶段，运动员被分成两人一组。在每一对中，运动员只有一人获胜。每对选手中的获胜者进入下一阶段。

这些对的构成为相邻未淘汰的运动员。

当运动员 $x$ 和 $y$ 比赛时，优胜者的决定如下：
如果 $x+y$ 为奇数，则编号较低的运动员获胜;
如果 $x+y$ 为偶数，则编号较高的运动员获胜。

给定整数 $n$，确定赢得比赛的运动员的编号。
## 思路
在第一阶段，为相邻两个数的运动员比赛，所以和为奇数，奇数运动员获胜；其它的时候奇数加奇数和为偶数，编号较高的运动员获胜，所以最后获胜的运动员为 $2^n-1$。
## CODE
```cpp
#include<bits/stdc++.h>
using namespace std;
long long n, t;
int main(){
	cin >> t;
	while(t--){
		cin >> n;
		n = 1 << n;
		cout << (1 << n) - 1 << endl;
	}
}
```
