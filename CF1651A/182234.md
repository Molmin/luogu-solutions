## Description

有 $2^n$ 位运动员比赛。第一轮为 $1$ 号对 $2$ 号，$3$ 号对 $4$ 号，以此类推。第二轮则是 $1$、$2$ 号中的胜者对 $3$、$4$ 号中的胜者。之后的规则都是一样的，直到决出冠军。

如果 $a$ 号对 $b$ 号运动员，则：

- 若 $(a+b)$ 为奇数，则 $\min(a,b)$ 号获胜；
- 若 $(a+b)$ 为偶数，则 $\max(a,b)$ 号获胜。

问冠军会是谁。

## Solution

第一轮都是奇数对偶数，而他们的和一定是奇数，又因为第一轮的每场比赛中都是奇数较小，所以第一轮出线的是 $[1,2^n]$ 内的所有奇数。

而后面的所有轮都是奇数之间对决，每一场都是大的赢。由此可见，最后能够获得冠军的就是最大的那个奇数，也就是 $(2^n-1)$。

## Code

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int T;
	scanf("%d",&T);
	while(T--)
	{
		int n;
		scanf("%d",&n);
		printf("%d\n",(1<<n)-1);
	}
	return 0;
}
```
