# 题解

- 我们可以轻易的得出结论：最后的结果一定是奇数，因为偶数在第一轮就被筛掉了（奇数加偶数必定为奇数，在分组中奇数又是小的那个）。

- 又因为奇数加奇数必定为偶数，所以总是每组中小的那个奇数被筛掉。

- 结果即为 $2^n$ 以内最大的奇数，也就是 $2^n-1$。

# 代码

```cpp
#include <bits/stdc++.h>

using namespace std;

int main()
{
	int T;
	cin >> T;
	while (T --)
	{
		int n;
		cin >> n;
		cout << (1 << n) - 1 << endl;
	}
}
```
