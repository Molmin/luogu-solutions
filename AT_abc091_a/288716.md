(先给一下题面）



------------
 _输入$A$,$B$,$C$。_ 

 _小 K有一张$A$元的硬币和一个$B$元的硬币。_ 

 _问他的钱是否够买个$C$元的东西(不是刚好等于$C$，大于也可以)。_ 

 _如果可以输出 Yes  , 否则输出 No 。_ 


------------

#### 这一道题的题意其实就是：
比较$A+B$与$C$的值。若$C>A+B$，则输出 'No' ,否则输出  'Yes'  。

于是代码就有了：
```
#include <bits/stdc++.h>
using namespace std;
int main()
{
  int a,b,c;
  cin>>a>>b>>c;//输入
  if(a+b>=c) //关键的判断
  {
     cout<<"Yes\n";//输出
  }
  else 
  {
     cout<<"No\n";//输出
  }
  return 0;
}

```
