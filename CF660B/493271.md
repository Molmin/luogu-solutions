# 题目大意：

有一个公交车，如果人数小于 $ 2n $ 的话，那么这些人就只会坐在边上，是先坐左边，然后坐右边这样的，如果大于等于 $ 2n $ 的话，就会去坐中间，也是先坐左边，再坐右边，走的时候，就先走第二列，然后走第一列，然后走第三列，然后走第四列这样的，依次走一个这样。问你这些人走的样子是什么样子。

# 思路：

先走的奇数，然后再走偶数位置，先走大于 $ 2n $ 的，再走小于 $ 2n $ 的。

# 代码：

```cpp
  #include<bits/stdc++.h>
  using namespace std;
  long long n,m;
  int main(){
      cin>>n>>m;
      for(int i=1;i<=2*n;i++){
          if(2*n+i<=m){
              cout<<2*n+i<<" ";
          }
          if(i<=m){
              cout<<i<<" ";
          }
      }
      return 0;
  }
```
