模拟大法好！

我们每次输入一个数，判断这个数剩下的仓鼠数和要的笼子数，最后输出这些数中剩下的仓鼠数**最少**的就可以了。

但是，这道题有一个坑点，就是数据非常大，所以，统计最小数的这个数要开到**10的18次方**。

明白坑点后，代码就好写了：
```
#include<bits/stdc++.h>
using namespace std;
int main()
{
	long long n,m,i,j,k,t=1000000000000000010,t1=0,t2=0;
	cin>>n>>m;
	for(i=1;i<=m;i++)
	{
		cin>>k;
		if(n%k<t)
		{
			t1=n/k;//求盒子个数
			t2=i;//求盒子的编号
			t=n%k;
		}
	}
	cout<<t2<<" "<<t1;
}
```
