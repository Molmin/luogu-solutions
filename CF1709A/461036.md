作为 `Dev.2` 的 `A` 题，这题还是很简单的。

直接看题意，编号为 $i$ 的钥匙可以打开第 $i$ 扇门，现在你手中有一把钥匙，有 $3$ 扇门，门的编号从左到右依次为 $1,2,3$,门后可能有其他钥匙，若为 $0$，则表示没有钥匙，问你能否打开所有的门。

简单的模拟，有很多方法，我这里用的是 $3$ 次循环枚举统计打开的门的数量。最后判断一下就行了。

代码如下：
```cpp
		cin>>x>>a[1]>>a[2]>>a[3];
        if(a[x]==0){
            cout<<"NO\n";
            continue;
        }
        int s=0;s++;
        for(int i=a[x];;){
            s++;
            if(a[i])
                i=a[i];
            else
                break;
        }
        if(s==3)
            cout<<"YES\n";
        else
            cout<<"NO\n";
```
