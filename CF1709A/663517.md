## 题意

有三扇门，有两扇门后面有钥匙，一扇门后面什么都没有，你初始有一扇门的钥匙，询问是否能打开所有的门。

# 分析

根据题意直接模拟就可以：把三扇门内的东西取出来，如果是空的我们就取第 $0$ 扇门（不存在嘛），最后检查三扇门是否都打开即可。

#### 代码

```cpp
#include <bits/stdc++.h>
int a[4], x, v[4], t;//v数组记录是否开门
signed main () {
	std::cin >> t;
	while (t--) {
		memset(v, 0, sizeof(v));
		std::cin >> x >> a[1] >> a[2] >> a[3];
		v[x] = 1, v[a[x]] = 1, v[a[a[x]]] = 1;//将三扇门后面的东西取出
		puts((v[1] && v[2] && v[3]) ? "YES" : "NO");//判断是否成功
	}
	return 0;
}
```
