### 题意：
有三扇门在你前面，按 $1,2,3$ 编号，每扇门要用相同编号的钥匙打开，你有一把钥匙，每开一扇门，你就可以获得门后的钥匙（为 0 则没有钥匙），问你是否能打开所有的门？

输入格式：

一行一个整数 $T$，有 $T$ 组数据。

接下来一行一个数 $x$，表示你已经有的钥匙。

接下来一行三个整数 $a,b,c$，表示门后有的钥匙（为 0 则没有）。

输出格式：

$T$ 行，每行一个字符串，如果能打开所有的门，输出 $\texttt{YES}$，否则输出 $\texttt{NO}$。
___
### 分析：
你要开三次才能打开三扇门，所以打开的前两扇门必须都有钥匙才可以。
___
### 代码：
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[4];
int main(){
	int t;
	cin>>t;
	while(t--){
		int x;
		cin>>x>>a[1]>>a[2]>>a[3];
		if(a[x]==0||a[a[x]]==0)cout<<"NO\n";//如果打开的第一，二扇门后没有钥匙，就无法打开所有门。
		else cout<<"YES\n";
	}
	return 0;
}
```
