## 题意简述

你手中有一把钥匙，你面前有三道门。三道门中的两道后有一把钥匙。钥匙只能打开对应的门，如 $1$ 号钥匙只能打开第一道门。

给出你手中已有的钥匙编号以及三道门后的钥匙情况，请问你能否打开所有的门呢？

数据保证手中钥匙编号 $x$ 不为 $0$ ，同时集合 $\{ a,b,c,x\}=\{0,1,2,3\}.$

## 题解

如果一把钥匙藏在它的门后，那么你是一定打不开它的。

如果你一开始打开的门后没有钥匙，那么你无法获得更多线索，因此无法打开剩下的两道门。

据此简单判断即可。建议评级：`入门`

## 代码实现

```
#include <bits/stdc++.h>
using namespace std;
int main()
{
	int t;
	cin>>t;
	while(t--)//多组数据
	{
		int n,a[3];
		cin>>n>>a[0]>>a[1]>>a[2];
		if(a[n-1]==0/*打开的第一扇门后有没有钥匙*/
        || a[0]==1 || a[1]==2 || a[2]==3
        /*对应的钥匙是否藏在门后*/)
			cout<<"NO"<<endl;
		else
			cout<<"YES"<<endl;
	}
	return 0;
}
```
