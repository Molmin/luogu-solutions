## 大致题意
给你一个数 $x$，和三个数 $a$，$b$，$c$，分别代表一扇门，代号为 $1$，$2$ 和 $3$，$x$ 表示可以打开 第 $x$ 扇门，每扇门后面都藏有一个钥匙，门上的代号为里面钥匙所对应的那扇门。当门上数字为 $0$ 时，则代表里面没有钥匙。
例如： 当 $x = 3$，则代表可以打开第 $3$ 扇门。

如果能打开所有的 $3$ 扇门，输出 $\texttt{YES}$，否则输出 $\texttt{NO}$。

## 思路分析
设 $a$，$b$，$c$，分别为 $a_1$，$a_2$ 和 $a_3$。

通过分析，我们可以知道，当 $a_x = 0 $ 时，是不可能打开任意一扇门的。

而如果 $a_{a_x} = 0$ ，同样打不开三扇门。

其他情况则可以了。

## Ac Code
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[5];

int main(){
    int T;
    cin>>T;
    while(T--){
        int x;
        cin>>x>>a[1]>>a[2]>>a[3];//代表题中三个数。
        if(a[x]==0) puts("NO");//如题
        else if(a[a[x]]==0) puts("NO");//如题
        else puts("YES");
    }
}
```
