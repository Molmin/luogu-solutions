### 题目大意
- 有 $3$ 个门，分别标记为 $1$，$2$，$3$。其中 $2$ 个门背后有钥匙。开门之后就可以拿到门后面的钥匙，打开相应的门
- 现在你有第 $x$ 扇门的钥匙。请问能否打开所有的门？
- 本题有多组数据。

### 解题思路
考虑什么情况下开不了所有门。

- 一开始能打开的那扇门后面没有钥匙。

这种情况下，剩余两扇门都打不开，所以开不了所有门。

- 门后面的钥匙是开启自身的。

显然，这种情况下，这扇门一定打不开，所以开不了所有门。

除了这两种情况外，其他情况都能开启所有的门。顺着模拟即可。

### 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
long long t,x,a[100];
int main(){
	cin>>t;
	while(t--){
		cin>>x>>a[1]>>a[2]>>a[3];
		if(a[x]==0){//能开启的门背后没有钥匙。
			cout<<"NO"<<endl;
			continue;
		}
		if(a[1]==1||a[2]==2||a[3]==3){//门后面的钥匙是开启自身的。
			cout<<"NO"<<endl;
			continue;
		}
		cout<<"YES"<<endl;
	}
	return 0;
}
```
