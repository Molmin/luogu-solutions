我们可以用一个char类型的变量不断输入，以达到输入字符串的目的

接着我们只需在程序开头读入然后与每次输入的字符比较，若不相等则不用去掉它，按原样输出即可

若有不懂，详情看代码：
```
#include<bits/stdc++.h>
using namespace std;
char c,x;//c存输入的字符串中的某个字符，x为去掉的字符
int main(){
    cin>>x;//先把去掉的读进来
    while(cin>>c){//不断输入，不输入时结束
        if(c!=x){//不相等
            cout<<c;//不用处理，直接输出
        }//如果是相等的就不会执行上面的输出，相当于去掉了
    }
    return 0;
}
```
