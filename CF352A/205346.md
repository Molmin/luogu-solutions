我记得洛谷题库上有一题和这题一样

**能被90整除的数,一定能被9和10整除。**

**当一个数所有数位的和能整除9时，这个数就可以整除9**

**当一个数末位数为0时，它就能整除10**

（废话）

**也就是说，要想整除90，必须满足以上条件**

**所以满足条件的数5的个数一定是9的倍数，末尾一定是0**

知道这些就可以AC了

```cpp
#include<bits/stdc++.h>
using namespace std;

int n,t,z,f;//z用来记录0的个数，f用来记录5的个数
int main()
{
	cin>>n;
	for(int i=1;i<=n;i++)
	{
		cin>>t;
		if(t==0)z++;
		  else f++;
	}
	if(z==0){cout<<"-1"<<endl;return 0;}//如果木有0，这个数就肯定不能整除90
	  else if(f<9){cout<<"0"<<endl;return 0;}//5木有达到数量，就只有0一个可能性
	      else
	      {
	      	for(int i=1;i*9<=f;i++)
	      	{
	      		cout<<"555555555";//输出5
			  }
		for(int i=1;i<=z;i++)cout<<"0";//把0全部输出去
		  }
	return 0;//完美结束
}
```
