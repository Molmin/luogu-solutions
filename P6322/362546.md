### 思路

这道题因为数据范围不大，所以我们可以用一个 $O(n^2)$ 的方法。

我们可以在输入之后再开一个循环，在每一次循环中先判断第 $1$ 个圆环半径和第 $i$ 个圆环半径是否相等，如果相等就直接输出 $1/1$，否则就用第二层循环找出第 $1$ 个圆环半径和第 $i$ 个圆环半径之间的最大公约数，然后输出第 $1$ 个圆环半径 $\div$ 第 $1$ 个圆环半径和第 $i$ 个圆环半径之间的最大公约数和一条分数线，再输出 $i$ 个圆环半径 $\div$ 第 $1$ 个圆环半径和第 $i$ 个圆环半径之间的最大公约数，最后加一个 ```endl```，就可以了。

### 代码

```
#include<iostream>
#include<bits/stdc++.h>
using namespace std;
int n,a[105],j; 
int main() 
{
	cin>>n;
	for(int i=1;i<=n;i++) cin>>a[i];
	for(int i=2;i<=n;i++)
	{
		if(a[i]!=a[1])
		{
			for(j=a[i];j>=1;j--) if(a[i]%j==0&&a[1]%j==0) break;
			cout<<a[1]/j<<"/"<<a[i]/j<<endl;
		}
		if(a[i]==a[1]) cout<<1<<"/"<<1<<endl;
	}
	return 0;
}
```
