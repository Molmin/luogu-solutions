# 思路
由题意可知：
1. 当金额 $n \le 180000$ 时，个人所得税为 $0$。
1. 当金额 $180001 \le n \le 200000$ 时，个人所得税为 $2000$。
1. 当金额 $200001 \le n \le 480000$ 时，个人所得税为 $(k-180000)*0.1$。
1. 当金额 $480001 \le n \le 880000$ 时，个人所得税为 $30000+(k-480000)*0.15$。
1. 当金额 $880001 \le n \le 1180000$ 时，个人所得税为 $90000+(k-880000)*0.2$。
1. 当金额 $1180001 \le n \le 1000000000$ 时，个人所得税为 $150000+(k-1180000)*0.25$。

使用 $if$ 语句判断后输出即可。
# 代码
```cpp
#include <bits/stdc++.h>
using namespace std;
int t,n,ans;
int main(){
	cin>>t;
	for(int i=1;i<=t;i++){
		cin>>n;
		if(n<=180000){
			ans=0;
		}
		else if(n<=200000){
			ans=2000;
		}
		else if(n<=480000){
			ans=ceil(0.1*(n-180000));
		}
		else if(n<=880000){
			ans=30000+ceil(0.15*(n-480000));
		}
		else if(n<=1180000){
			ans=90000+ceil(0.2*(n-880000));
		}
		else{
			ans=150000+ceil(0.25*(n-1180000));
		}
		cout<<"Case "<<i<<": "<<ans<<endl;//注意输出要根据题意输出
	}
	return 0;
}
```
