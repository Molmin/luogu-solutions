# 一道典型的质数题！
### 题意：
#### 求 2~N-1 中的质数个数（只有1和本身两个因数）
### 思路：
#### 用删数做，用for循环，把一个数的倍数（除它本身）删去，剩下的数都是质数!

##### 本人为您献上一篇AC代码：

```c
#include<iostream>
using namespace std;
int a[1000000];//定义数组(要定的稍微大一点，防止数组越界)

int main()
{
	int n;
	cin >> n;//输入
	for (int i = 2;i < n ; i ++)//循环删数
	{
		if (a[i] == 0)//如果你没有被删去，就去删别人(别把自己删了)
			for (int j =  i + i; j <= n; j += i)//删数
				a[j] = 1;//被删去
	}
	int m = 0;//统计质数个数
	for(int i = 2;i < n;i ++)
	{
		if(a[i] == 0)
			m ++;
	}
	cout << m << endl;//完结撒花
    
    while(1)//防止抄袭
    
	return 0;
}
```
