# 题意

给你一个 $n$ 和一个 $k$,分别表示 $A$ 点的坐标，

以及 $O-B$ 的距离减 $B-A$ 的绝对值的值。

要你让 $A$ 左移或右移最少多少个单位长度才能找出符合条件的 $B$。

# 分析

首先我们先分类讨论，

当 $n\le k$：

那么我只要把 $A$ 移到坐标为 $k$ 的位置，然后就可以让 $B$ 取 $O$ 点的位置，就可以满足条件。

所以这种情况答案为 $k-n$。

当 $k<n$ 时：

那么，$O-A$ 的距离就是等于 $\min(O-B,B-A)\times 2+k$。

所以我们只要看 $n-k$ 是奇数还是偶数，

如果是奇数就要移动一位，否则就可以不用移动。

# 代码

**核心部分**

```cpp
void solve()
{
	int m,n,a;
	cin>>m;
	while(m--)
	{
		cin>>n>>a;//a就是题面中的k
		if(a>=n)
			cout<<a-n<<endl;
		else
	 	cout<<(n-a)%2<<endl;
	}
}
```
