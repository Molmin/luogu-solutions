#### 华丽丽的开头
------------
相信大家题目都看过了，题目中我们说到我们会在一个兵营里放置$s1$个工兵,注意，**只有一个阵营**，然后在将$s2$个士兵放到任意一个阵营里，注意**还是只有一个阵营**。两个阵营对于$100000$个来说简直可以忽略不计。所以我们先统计原来两个阵营的初始值。

```cpp
scanf("%d",&n);
for(int i=1;i<=n;i++)
	scanf("%d",&a[i]);
scanf("%d %d %d %d",&m,&p1,&s1,&s2);
a[p1]+=s1;//p1个士兵所处的阵营已经确定，我们可以放心的加上去
long long lsum=0,hsum=0;
for(int i=1;i<=m-1;i++)
	lsum+=(long long)a[i]*(m-i);//(m-i)是m号阵营到i号阵营的距离，(i-m)同上
for(int i=m+1;i<=n;i++)
	hsum+=(long long)a[i]*(i-m);
```

#### 解释一下为什么要用 $long long$

题目中说到$n<=10^5,c_i<=10^9$在极端情况下，一个阵营就会有$10^5*10^9=10^1$$^4$

再算上总共$10^5$个阵营，总共有$10^1$$^9$个阵营，略微低于$long long$范围，毒瘤数据一来立马爆炸。

我们前面提过，$s2$个工兵只会放置在一个阵营里，我们只要穷举$s2$个工兵会放到哪个阵营里，$O(n)$算法轻松过样例。

```
for(int i=1;i<=n;i++)
	{
		if(i<m)lsum+=(long long)(m-i)*s2;//将s2个工兵另外计算气势
		else if(i>m)hsum+=(long long)(i-m)*s2;
		long long c=lsum-hsum;
		if(c<0)c=-c;//我们要算的是他们的差距，不可能为负数，所以取绝对值
		if(c<minn)minn=c,mini=i;
		if(i<m)lsum-=(long long)(m-i)*s2;//把s2个工兵剔除，消除气势。
		else if(i>m)hsum-=(long long)(i-m)*s2;
	}

```

#### 彩蛋：
按上面的道理来说，要用$long long$最大值，我只用了$1e014$（也就是$10^1$$^4$）竟然$AC$了！
