# SP8785 题解

### 题目翻译

一个矿工要在 $ n $ 天内第 $ i $ 天给债主 $ i $ 单位的银条，来拿回上一天给的 $ i - 1 $ 单位银条。求出矿工分割银条的最少次数。

### 题目思路

既然要分割银条，我们不妨想想二进制。二进制的进位和这道题右很多相似之处，要让我们求出这个值，可以看成求 ``` log2(n) ```。这是数学库中的一个函数，可以求出以 $ 2 $ 为底，$n $ 的对数。但是分割 $ x $ 次会变成 $ x + 1 $ 段，所以还需要强制类型转换（向零舍入）。

### 题目代码

```cpp
signed main()
{
    long long a;
    while(cin >> a && a != 0) // 简便的写法，即读入 a 后判断 a != 0 时再进行 while 中的内容
    {
        cout << (long long) log2(a) << endl; // 强制类型转换
    }
	return 0;
}

```
