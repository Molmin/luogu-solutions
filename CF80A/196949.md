首先你先要会判断素数的代码，素数我们就要从2开始，到这个数-1为止，看看这些数是否能够被这个数整除，是的话它就不是素数，否则它就是素数。
```cpp
for(i=2;i<=n-1;i++)
{
	if(n%i==0)
	{
		cout<<"NO";//如果n是素数，就会输出NO，否则不会有输出
	}
}
```
接下来就方便了，我们先对n和m都先进行判断，如果二者有任意一个不是素数，我们可以直接输出NO，结束程序。
```cpp
for(i=2;i<=n-1;i++)//一个个逐一判断
{
	if(n%i==0)//能否整除
	{
		cout<<"NO";
		return 0;//结束程序
	}
}
for(i=2;i<=m-1;i++)//一个个逐一判断
{
	if(m%i==0)//能否整除
	{
		cout<<"NO";
		return 0;//结束程序
	}
}
```
接着我们要开始找n+1到m-1之间的素数，看看m是不是最小的一个。我们设一个变量s，一旦找到从2到这个数之间的某个数能被这个数整除，s就+1，我们再在末尾看看s是不是等于0，一旦它是0，那么代表从n+1到m-1之间有一个素数，那么显然m不是n往下的一个素数，输出NO，结束程序，
如果我们成功从头到尾都执行了一遍，那么相当于没输出过一次NO，这时我们就可以开心地输出YES，华丽丽地结束了。
完整代码：
```cpp
#include<iostream>
using namespace std;
int main()
{
	int n,m,i,s=0,j;
	cin>>n>>m;
	for(i=2;i<=n-1;i++)
	{
		if(n%i==0)
		{
			cout<<"NO";
			return 0;
		}
	}
	for(i=2;i<=m-1;i++)
	{
		if(m%i==0)
		{
			cout<<"NO";
			return 0;
		}
	}
	for(j=n+1;j<=m-1;j++)
	{
		s=0;//别忘了归0
		for(i=2;i<=j-1;i++)
		{
			if(j%i==0)
			{
				s++;
			}
		}
		if(s==0)
		{
			cout<<"NO";
			return 0;
		}
	}
	cout<<"YES";
	return 0;
 } 
```
