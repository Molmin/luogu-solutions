### 解释样例
```
3
3 9 6 4 6
6 9 3 1 1
8 8 9 3 7
```
第一列 $\max(3,6,8)=8$ 。\
第二列 $\max(9,9,8)=9$ 。\
第三列 $\max(6,3,9)=9$ 。\
第四列 $\max(4,1,3)=4$ 。\
第五列 $\max(6,1,7)=7$ 。

所以 $\min(8,9,9,4,7)=4$



### 基本思路

1. 输入时将五个值取最大值
2. 将五个最大值取最小值


这样，你就可以过前$7$个点

```cpp
#include<bits/stdc++.h>
using namespace std;
int a,b,c,d,e,n,ans=1e9,m1,m2,m3,m4,m5;
int main(){
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>a>>b>>c>>d>>e;
		m1=max(a,m1);
		m2=max(b,m2);
		m3=max(c,m3);
		m4=max(d,m4);
		m5=max(e,m5);
	}
	ans=min(ans,min(m1,min(m2,min(m3,min(m4,m5)))));
	cout<<ans;
    return 0;
}

```


那我们的错误在哪了？

3. 我们得考虑五个最优解所在的人数小于$3$，所以我们只需在取最大值时将本次取得最优解的人做一个标记，最后看标记数是否大于3，如果小于，则将剩下的数重复以上操作即可。
