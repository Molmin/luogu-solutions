首先众所周知，数字比较大小的方法是

1：比位数，位数多的大

2：位数一样时从首位往后开始比大小遇到不一样的数为止。

那么，对于这道题，每一次给定$n$个火柴棒，根据当然是位数越多越好，而耗费火柴棒最少的数字是“1”（只用2根）

如果n是偶数，显而易见，最大的数就是$\frac n 2$个“1”。

但如果n是奇数，我们就只能得就会多出一根，而且这时位数已经最大了，这是想要数再大点，就应改让它首位尽可能大。

把首位的“1”(2根）改成“7”(3根）数就大了不少，而且刚好用完火柴棒，且没有其它方法使得它更大，那么就输出这个数。

那么我们可以开开心心的上代码了！

PS:这道题的输出都是一个一个字符串的，所以用putchar()会快不少
```cpp
#include<bits/stdc++.h>
using namespace std;
int read(){//快读，相信我，会快的！
    int f=0;
    char ch=getchar();
    while(ch<'0'||ch>'9')ch=getchar();
    while(ch>='0'&&ch<='9'){
        f=(f<<1)+(f<<3)+ch-'0';
        ch=getchar();
    }
    return f;
}
int main(){
    int t,n;
    t=read();
    while(t--){
        n=read();
        if(n%2==1){//n是奇数
            if(n/3>0){//可以先放一个“7”
                putchar('7');//输出“7”
                n-=3;//取出相应的火柴数
            }
        }
        while(n){
            putchar('1');//剩下全放“1”
            n-=2;
        }
        putchar('\n');
    }
    return 0;
}
```
