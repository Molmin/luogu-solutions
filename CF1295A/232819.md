↓以下的偶数和奇数都是在说小棒个数

先从偶数下手，偶数的因数一定有谁？  ***2***  没错！

解决完偶数后奇数也很简单了只要 奇数→偶数 

那一般偶数加几会被2整除呢？ ***3***没错！

∴ 我们采取的顺序如下↓

***判断奇偶→是否加 7→疯狂加1***

你们最爱的代码↓

```c++
#include <iostream>
using namespace std;
int main(){
    int t,n;
    cin >> t;
    for (int i=1;i<=t;i++){
        cin >> n;
        if(n%2==1){//判断奇偶
            n-=3;
            cout<<7;
        }
        for(;n>=2;n-=2){
            cout<<1;
        }
        cout << endl;
    }
    return 0;
}
```
