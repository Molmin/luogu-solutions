注意：本题有多测。

### 题目分析

观察表格可以发现，第 $x$ 行中有 $x$ 个分数，那么从第 $1$ 行到第 $x$ 行就会有 $\frac{x(x+1)}{2}$ 个分数。

我们用一个变量 $x$ 从 $1$ 开始枚举，当到第 $x$ 行时累计的数的数量 $\frac{x(x+1)}{2}< i$，但到第 $x+1$ 行时 $\frac{(x+1)(x+2)}{2}\ge i$，我们就可以确定，题目要求的第 $i$ 个分数就在第 $x$ 行。

那么如何求出这个分数的列号呢？

我们可以先求出到第 $x-1$ 行为止的分数的数量（即 $\frac{x(x-1)}{2}$），再求出 $i$ 减去这个数的值，得出来的结果就是这个分数的列号，通过列号就可以求出这个数的分母（同为列号）。

再观察表格可以发现，对于一个在第 $a$ 行第 $d$ 列的分数 $\frac{c}{b}$，有 $a+1=b+c$，化简得 $c=a-b+1$，我们已经得出 $b=d$，而 $a, d$ 已经通过前面的计算得出，我们就可以进而求出 $c$ 的值。

### 代码

```cpp
#include <bits/stdc++.h>
using namespace std;
int n, s, cnt, a, b; // a 为分子，b 为分母，cnt 为行号
int main(){
	ios::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);
	while(cin >> n){
		s = 0;
		cnt = 1;
		while(s < n){ // 求出行号
			s += cnt;
			cnt++;
		}
		s -= cnt;
		cnt--; // 退回一步
		b = n - cnt * (cnt - 1) / 2;
		a = cnt - b + 1; // 求出分子分母，如上
		cout << a << "/" << b << "\n";
	}
	return 0;
}
```
