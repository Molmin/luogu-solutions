## 题意简析

瓶子里有 $N$ 块木板，从 $1$ 到 $N$ 标号。两个人轮流选择一块木板，并把这块木板上的数的所有因子（包括本身）对应的木板取走，取到最后一块木板的人获胜。给定 $N$ 和先后手，两个人都采取最优策略，求获胜者。

## 代码/解释
考虑只有 $2\sim N$ 时的必胜策略。
- 如果先手必胜，那么在原博弈中先手使用该策略第一步 $1$ 就会被一起取走；
- 否则先手必败，那么在原博弈中先手第一步把 $1$ 取走，后手就拿到了必败态。

因此原博弈中先手必胜。
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int t,n,k;
	cin>>t;
	while(t--)
	{
		cin>>n>>k;
		printf("%s wins.\n",k?"Pagfloyd":"Airborne");
	}
	return 0;
}
```
