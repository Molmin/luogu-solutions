先说一下题目的意思吧，其实一大堆名词都不需要管，直接看输入格式和输出格式就行了。
那我就直接说了，输入 $n$ 个数字，每个数字在 0~99 之间。
最后让输出非零数字和零的个数的差。

---
那么一下就有了思路，开一个数组，输入数组，并且逐个排查是不是零。
如果是零，那么```sum++```，所以非零数字就是```n-sum```了。
注意，还要判断 $n$ 是不是零。是的话跳出循环，结束。

代码如下，有注释：
```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
	int n,ans=0;//ans是为了计数用的
	int a[n];//开数组
	int sum=0;//计算0的个数
	while(1)
	{
		ans++;//为了后面输出Case ans
		cin>>n;
		if(n==0)
		break;//跳出循环
		for(int i=0;i<n;i++)
		cin>>a[i];
		for(int i=0;i<n;i++)
		{
			if(a[i]==0)//如果是0
			sum++;//加一
		}
		cout<<"Case "<<ans<<": "<<(n-sum)-sum /*非零个数减去零的个数 */ <<endl;
		sum=0;//归零，否则会混乱
	}
	return 0;//完结散花
}
```
