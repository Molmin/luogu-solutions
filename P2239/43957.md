###新人蒟蒻题解，神犇大牛勿喷

这道题可以用递归来实现，可以从四个方面去思考，在第一行，在最后一行，在第一列，在最后一列（我这种蒟蒻是画图才知道的）；其他详见代码：

```cpp
#include<bits/stdc++.h>//万能头文件 
using namespace std; 
int n,x,y;
int dg(int n,int i,int j)//递归函数，三个参数分别为当前矩阵的大小，和所求点在当前矩阵的横纵坐标 
{
    if(i==1)
    return j;//如果所求坐标在第一行，那么这个坐标上的数就是纵坐标（这个很容易理解吧） 
    if(i==n)
    return 3*n-1-j;//如果所求坐标在最后一行
    if(j==1)
    return 4*n-2-i;//如果所求坐标在第一列
    if(j==n)
    return n+i-1;//如果所求坐标在最后一列 
    return dg(n-2,i-1,j-1)+(n-1)*4;//如果都没有，说明 所求坐标在中间，递归下一层； 
}
int main()
{
    cin>>n>>x>>y;//输入矩阵尺寸，和所求点坐标 
    cout<<dg(n,x,y)<<endl;//输出 
}
```
