## no.2——分层计算
先把矩阵分层，例如：

![](https://i.loli.net/2018/09/25/5baa3232d6f47.png)

~~盗图~~

很明显，只要找出所在层，就可以把在这之前几层的数量与本层排名相加得结果(因为后层一定比前层后写)

以下是AC代码(个人喜欢循环，也可以递归)：
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,x,y,s;
int f(){//本层排名
	if(x==1)return y;//所在行为1
	if(y==1)return 4*n-2-x;//所在列为1
	if(y==n)return n-1+x;//所在行为最后
	if(x==n)return 3*n-1-y;//所在列为最后
}
int main(){
	cin>>n>>x>>y;
	while(x>1&&y>1&&y<n&&x<n){//不是当前层
		s+=4*(n-1);//之前层累加
		n-=2;//矩阵边长减2
		--x;//所在行减
		--y;//所在列减
	}
	cout<<s+f();//输出
	return 0;
}
```
