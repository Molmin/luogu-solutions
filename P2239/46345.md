首先我们把所有边框用全砍掉。画个图更容易理解。![](https://cdn.luogu.com.cn/upload/pic/33936.png)

如图，绿色为目标点，把红色地方的全部砍掉，用a与b来记录此时的边，sum砍过后为12。

用while砍到目标点已在四周为止。
```cpp
int a=1,b=n;
	int sum=0;
	while(i!=a&&j!=b&&i!=b&&j!=a)
	{
		sum+=(b-a)*4;
		a++;b--;
	}
```

这样不断砍，知道砍到目标点在边侧的时候循环4次找，每找1个sum++，找到就输出。

![](https://cdn.luogu.com.cn/upload/pic/33937.png)
如图，红色的地方为删去部分，蓝色为第一次查找，紫色为第二次查找，白色为第三次查找褐色为第四次查找。由图中可发现，绿色目标点在第二次紫色查找中已被发现，就直接输出。
```cpp
for(int x=a;x<=b;x++)
	{
		sum++;
		if(i==a&&j==x) 
		{
			cout<<sum;
			return 0;
		}
	}
	for(int x=a+1;x<=b;x++)
	{
		sum++;
		if(j==b&&i==x)
		{
			cout<<sum;
			return 0;
		}
	}
	for(int x=b-1;x>=a;x--)
	{
		sum++;
		if(i==b&&j==x)
		{
			cout<<sum;
			return 0;
		}
	}
	for(int x=b-1;x>a;x--)
	{
		sum++;
		if(i==x)
		{
			cout<<sum;
			return 0;
		}
	}
```
