> 有 $a$ 个 $1$ 元硬币，$b$ 个 $2$ 元硬币，求最小的无法付的钱数。

分类讨论。

- 当 $a = 0$ 时，显然 $1$ 就凑不出来了，所以答案就是 $1$。
- 当 $a \not = 0$ 时，显然 $[1, 2a + b]$ 之间所有数都能凑出来，所以答案是 $2a + b + 1$。

```cpp
void solve() {
  LL a, b;
  iocin >> a >> b;
  if (a == 0) {
    cout << 1;
  } else {
    cout << a + b * 2 + 1;
  }
  putchar('\n');
}
```
