#题目背景

Bessie喜欢玩棋盘游戏和RPG游戏，所以她让John开车带她去商店买了3个骰子

#题目描述

这三个骰子有分别有s1,s2,s3三个面，而每个骰子的面分别为1~S

现在Bessie将3个骰子一起扔出，将会有不同的情况，每种情况对应的会有三个骰子朝上面所示的数之和sum，而每个面朝上的概率相同

Bessie希望求出在她扔出骰子后，在出现的所有sum中出现最多的一种。如果有几种出现次数一样多，取sum较小的那种

```cpp
#include<cstdio>
int s1,s2,s3,a[100005],sum,ans;//s1s2s3对应原题目，a数组存储每种sum的出现次数，sum和ans在最后的找答案中才会派上用场 
int main()
{
    scanf("%d%d%d",&s1,&s2,&s3);//读入s1s2s3 
    for(int i=1;i<=s1;i++)
    {
        for(int j=1;j<=s2;j++)
        {
            for(int k=1;k<=s3;k++)//由于数据较水，三重循环即可，时间复杂度O(s1*s2*s3),最坏情况为O(16000)，相当小了
            {
                a[i+j+k]++;//统计每种sum出现的次数 
            }
        }
    }
    for(int i=1;i<=s1+s2+s3;i++) if(a[i]>sum)//把a数组过一遍，找到答案 
    {
        sum=a[i];
        ans=i;
    }
    printf("%d",ans);//输出，结束 
    return 0;
}
```
