# 题目意思

题目中有一张部分打乱顺序的表格，

第一部分是是行打乱的；

第二部分是列打乱的；

前面虽然行是不对的，但列是对的，后面虽然列不对，但行是对的，

最后要输出顺序正确的表格。

# 解题思路

可以用两个数组分别储存正确的行和列；

定一个一维数组来存储每个数正确的行，再定一个一维数组来存储正确的列；

此时正确的行和列都已经确定，然后将正确的行和列共同统计到一个二维数组中。

将每张表格输出即可。

# 上代码

```
#include<bits/stdc++.h>//万能头，谁用谁知道
using namespace std;

int t,n,m,a,h[250001],ny[5001][5001];//定义变量
int main(){
    cin>>t;
    for(int s=1;s<=t;s++){
		cin>>n>>m;
        for(int i=1;i<=n;i++){
            for(int j=1;j<=m;j++){
                cin>>a;
                h[a]=j;
            }
        }//记录正确的行，赋值到h
        for(int i=1;i<=m;i++){
            for(int j=1;j<=n;j++){
                cin>>a;
                ny[j][h[a]]=a;//记录正确的列，此时行和列都已经确定，直接赋值到ny 
            }
        }
        for(int i=1;i<=n;i++){ 
            for(int j=1;j<=m;j++){
				cout<<ny[i][j]<<' ';
			}
            cout<<endl; 
        }
    }
    return 0;  
}
```
