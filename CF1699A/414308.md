## Part 1 题意简述

给定一个 $n$，要求找出满足 $(a\oplus b)+(b\oplus c)+(a\oplus c)=n$的三个自然数 $a, b, c$。

## Part 2 思路

仔细观察数据范围： $ ( 0 \le a, b, c \le {10^9})$，肯定是暗示我们要去构造这样的三个数而不是暴力地去搜索。

如果你不小心被样例组给出的答案所蒙蔽，不妨先敲一个暴力，在较小的数据范围内多测一些数，看看能发现什么规律。根据探究，可以发现，当 $n$ 为奇数时，不存在这样的三个数；而 $n$ 为偶数的时候，$0, 0, \frac{n}{2}$ 就是一组方案，下文来证明此规律的正确性。

首先，你应当清楚：

- 奇$\oplus$奇$=$偶
- 偶$\oplus$偶$=$偶
- 奇$\oplus$偶$=$奇

将 $a, b, c$ 的奇偶性分成三偶，一偶两奇，两偶一奇，三奇四种情况，分别带入，发现结果肯定都是偶数。也就是说，当结果 $n$ 为奇数，那肯定没有符合原题条件的 $a, b, c$ 存在。

而，将 $a = 0, b = 0, c = \frac{n}{2}$ 带入  $(a\oplus b)+(b\oplus c)+(a\oplus c)$ 中，可以知道结果是 $n$。这是显然的。

## Part 3 Code


```cpp

#include <iostream>
using namespace std;

int t, n;

int main()
{
	cin >> t;
	while(t--)
	{
		cin >> n;
		if(n % 2 == 1)
		{
			cout << "-1" << endl;
		}
		else
		{
			cout << "0 0 " << n / 2 << endl;
		}
	}
	return 0;
}
```
