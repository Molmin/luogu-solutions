题目大意：给定三角形三条边的长度，问这个三角形是否是直角三角形。

很显然，我们可以用勾股定理：$a^2+b^2=c^2$ 来解决判断问题。

注意有多组数据。
```cpp
    int a[4]；
    while (scanf("%d %d %d", &a[1], &a[2], &a[3]) && a[1]) {
        sort(a + 1, a + 1 + 3)；
        if (a[1] * a[1] + a[2] * a[2] == a[3] * a［3]) printf("right/n")；
        else printf("wrong/n")；
    }
```
