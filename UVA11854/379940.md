### 翻译

题目背景：

很久以前，埃及人发现边长为$3,4,5$的三角形的最大角是直角。叫你你确定其他三角形是否具有类似的性质。

输入：

每次输入三个$ \le30000$的数字表示三角形边的长度。

输出：

如果是直角三角形输出“$right$”，不是输出“$wrong$”。

我们可以用勾股定理来解决这个问题.
也就是说判断$a²+b$²是否$=c²$，等于就输出$right$，不等于输出$wrong$

勾股定理：在任何一个平面直角三角形中的两直角边的平方之和一定等于斜边的平方。在$△ABC$中，$∠C$=90°，则$a²+b²=c²$ 。

### $code$：
```
#include<cstdio>//定义输入／输出函数
#include<algorithm>//STL 通用算法
//现在不喜欢用万能头文件了呢
long long a[3];//定义变量，因为我们的教练说高手的变量特别少，来装一波高手
int main(){
	while(1){//无限循环，也可以用for(int i=1;;i++)
		for(int i=1;i<=3;i++) scanf("%lld",&a[i]);//输入变量
		if(a[1]==0) return 0;//特判：如果输入的数字为0，就直接结束程序
		std::sort(a+1,a+3+1);//排序，因为没有写using namespace std，所以前面要加一个std::。
        printf((1LL*a[1]*a[1]+1LL*a[2]*a[2]==1LL*a[3]*a[3])?"right\n":"wrong\n");//三目运算符先判断a²+b²是否=c²，然后在输出。ps：要换行,否则会保龄
    }
}
```
