### 楼上的大佬这么忙约分，都不想想有最简单的方法吗-----打表

思路很简单，如果两位绅士分数为a和b(a>=b),那么小D最少要获得max(a,b)分

而小D获胜的可能数就是6-小D最少分数+1也就是7-a。而最后的约分嘛.....六个情况手打就行了

贴出代码：

```
#include <bits/stdc++.h>
using namespace std;
int a,b,c; //a开始是输入后来是小D最小分数，b是输入的，c是可能种数
int main(){
cin>>a>>b;
a=max(a,b);  //求出最小分数
c=6-a+1;     //可能种数
if(c==1) cout<<"1/6";
if(c==2) cout<<"1/3";
if(c==3) cout<<"1/2";
if(c==4) cout<<"2/3";
if(c==5) cout<<"5/6";
if(c==6) cout<<"1/1";    //以上为打表
return 0;
}

```
