- 本题的重难点在于玩偶的方向，你会发现，玩偶的方向都是由1或0来表示的。

- 我们先来判断一下方向吧。先把这个环拆成一个线段。

- 当玩偶刚开始为0时（也就是朝内），如果朝左数（0）的话，和刚开始为1时（朝外）然后往右数（1）是方向一样的。而且它们的和（0+0=（1+1）%2）。

- 同理，刚开始为0是往右数和刚开始为0时往左数方向也是一样的，和都为1.

- 由此可以推出，当和 mod 2 后，为1时则将坐标往右，为0时则将坐标往左，于是这题就圆满结束了。

```cpp
#include <bits/stdc++.h>

using namespace std;
int n,m,x,t,k,a[100001];
string s[1000001];//定义一个字符串，记录下玩偶的所有名字
int main()
{
    cin>>n>>m;
    for (int i=1;i<=n;i++)
      cin>>a[i]>>s[i];
    x=1;//坐标，表示现在已经搜到了第几个玩偶了
    for (int i=1;i<=m;i++)
    {
        cin>>t>>k;
        t=(t+a[x])%2;//上面已经解释了
        if (t==0)
        {
            x=x-k;
            if (x<1) x+=n;
        }
        else
        {
            x=x+k;
            if (x>n) x-=n;
        }
    }
    cout<<s[x];//直接输出即可
}
```
