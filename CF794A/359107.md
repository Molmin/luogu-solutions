# 题解 CF794A 【Bank Robbery】

###### ~~大家好，本奆蒟蒻又来写题解了~~

拿到这道题，一看感觉好难啊，绝对是我做不出来的题。但其实认真分析一下，就会发现这道题其实是一道大[㵘](https://baike.baidu.com/item/%E3%B5%98/15198103?fr=aladdin)题。

___

题意：

**有一排保险柜，小偷在其中的一个保险柜里，在他的左边和右边的某个保险柜的位置上有两个警察，小偷可以来回移动，但不能到警察所在的保险柜里，问小偷可以去几个保险柜（即输入一排保险柜的号码，问小偷可以去的保险柜有几个）。**

___
### 分析：

可以把这一排保险柜想象成一个数轴，先输入小偷的位置 $a$ ,那么小偷就在 $a$ 点，再输入两个警察的位置 $b$ 和 $c$ ，那么警察们就在 $b$ 点和 $c$ 点，然而，题目中又说了，$b<a<c$ ，所以，小偷的位置就无关紧要了。



___

懂了原理，开始代码实现

定义 $a$ , $b$ , $c$ , $n$ 不用说了，体面上说是 $ai$ 来表示输入的数，其实这里不用定义数组，只要一输入就判断一下就可以了。

话不多说，更详细的讲解在代码里。

## $Ac$ $Code$ 

```cpp
#include<iostream>
using namespace std;
int main()
{
	int a,b,c,g,n,money=0;  // g 为输入的柜子号， money 为偷到的钱的总数
	cin>>a>>b>>c>>n;
	for(int i=1;i<=n;i++)
	{
		cin>>g;  //输入 ai 
		if(g>b&&g<c)  //如果在警察的范围之内
		{
			money++;  //钱就加一，因为每个柜子里只有一张钞票
		}
	}
	cout<<money;  //最后输出
	return 0;  //结束程序
}
```
