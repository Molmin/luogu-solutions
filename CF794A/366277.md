## 解题思路

因为已经保证$b<a<c$，即小偷一定在中间，所以不需要考虑在警察两边的情况，也就是说小偷的位置并不影响结果。

由于不需要保存每个数据一起处理，采用一个临时变量边输入边判断即可。

### $Code:$

```
#include<bits/stdc++.h>
using namespace std;
int main (){
	int a,b,c,n,x,cnt=0;//a,b,c是三个位置
   //x是每张钞票的位置，cnt用于计数
	cin>>a>>b>>c>>n;//按题意输入
	for (int i=1;i<=n;i++){//按题意输入n次
		cin>>x;//输入
		if (b<x&&x<c)//如果钞票在两个警察中间，则小偷可以拿到
   //注意不能取等号，因为不能拿警察所在位置保险箱里的钱
		cnt++;//计数器+1
	}
	cout<<cnt<<endl;//输出可以偷到的钞票数量
	return 0;//好习惯
}
```
