题意：给很多行字符串，每行去重后输出。

思路：我们可以循环读入每个单词，每个单词读入后，再读入单词后是空格还是回车。若是回车，说明我们要判断这个单词是否重复。这里用了一个容器，叫做映射。

先解释一下映射：映射有两个参数，键和值。我们可以直接通过键来访问这个值。那么对于这道题，我们就可以把这个词出现的位置（即第几个）做为键，这个单词作为值。

继续讲思路：若是空格，那么就看映射里有没有这个单词。有就不管，没有就添加。那如果是回车呢？如果是回车，就说明我们需要输出这一行单词去重后的结果了。

但是，有一个问题，题中说，输入以 EOF 结束，但是样例中却没有体现。那么我这里是用 $-1$ 结束。

代码（加解释）：
```cpp
#include<bits/stdc++.h>
using namespace std;
map<int,string> s;
map<int,string>::iterator it; // 访问映射所用的迭代器
int cnt; // 表示我这个单词应该在映射的什么位置
string str;
char c;
int main()
{
	while((cin>>str)!="-1")
	{
		if(str=="-1") break; //如上所说
		c=getchar();
		if(c==' ') // 判断此刻需要干什么，是空格，看是否添加
		{
			bool f=0;
			for(it=s.begin();it!=s.end();it++) // 循环访问映射中是否有重复单词
				if(it->second==str) f=1; // 若有，就改标志
			if(f==0) // 没被改标志，就存入映射
			{
				cnt++;
				s[cnt]=str;
			}
		}
		if(c=='\n') // 如果是回车，就输出这一行的结果
		{
			for(it=s.begin();it!=s.end();it++) // 循环访问映射
				cout<<it->second<<" "; // 因为单词存储在键值对中的值这个位置，所以输出的是第二个，也就是值
			cout<<endl;
			s.clear(); // 记得清空
			cnt=0; // 位置计数变量也要清空
		}
	}
	return 0;
}
```
