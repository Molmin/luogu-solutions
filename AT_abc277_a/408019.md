## AT_abc277_a [ABC277A] ^{-1} 題解

### 题面

给定长度为 $n$ 的集合 $a$，求 $x$ 出现在第几位。

### 解析

考察分支语句以及循环语句的使用。

本题完全不需要数组，可以使用一个“边读边操作”的方法，输入集合中的一个数据对它进行判断：

- 若为 $x$，输出循环变量 $i$ 并直接结束；
- 否则，跳过这层循环，读入下一个数字。

### 代码

核心代码如下：

```cpp
for i = 1; i <= n; i++{
	fmt.Scan(&a)
    if a==x{
    	fmt.Println(i)
        break
    }
    else{
    	continue
    }
}
```
