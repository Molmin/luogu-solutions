这题比较简单,**思路**:

首先我们知道加一花费a,减一话费b,所以**先加一再减一或者先减一再加一是没必要的**,所以我们只要**一直加一或者一直减一**.

那么现在题目就变成了这个样子:

现在有一个数n,**求是加到m的倍数时花费的代价少还是减到m的倍数时花费的代价少**

那么代码就显而易见了,当然十分得短,如下:

```cpp
#include<bits/stdc++.h>
using namespace std;
long long n,m,a,b;//记住一定要开longlong
int main() {
	cin>>n>>m>>a>>b;
	cout<<min((n%m)*b,(m-n%m)*a)<<endl;//比哪个花费的代价少
}
```
