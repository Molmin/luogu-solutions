------------
首先应该想到的是，本题第二站上下车人数对于结果来说至关重要，所以应该从求得第二站上车（或下车）的人数入手，由于我比较懒，所以直接就把每一个站上下车人数都顺带求出来了，这里我们设第二站上车人数为i

所以本题解题的关键是一个字：“**试**”

由于第二站上下车人数无上限，所以应该用死循环，然后利用我们一个一个“试”出来的数向后推到n-1站

因为最后一站下车人数即为倒数第二站发车时剩余人数，所以当第n-1站发车时，如果人数恰好等于m，此时的i就“试”出来了



------------
上代码
```cpp
#include<iostream>
using namespace std;
int ans,a,n,m,x,ren,sh[21],xia[21];
int main()
{
	cin>>a>>n>>m>>x;
	for(int i=0;;i++)
	{
		for(int k=1;k<=n;k++)
		{
			sh[k]=0;
			xia[k]=0;
		}
		sh[1]=a;
		ren=a;
		sh[2]=i;
		xia[2]=i;
		for(int j=3;j<=n-1;j++)
		{
			xia[j]+=sh[j-1];
			sh[j]+=sh[j-1]+sh[j-2];
			ren+=sh[j]-xia[j];
			if(ren>m)
			break; 
		}
		if(ren==m)
		break;
	}
	ans=a;
	for(int k=3;k<=x;k++)
	{
		ans+=sh[k]-xia[k];
	}
	cout<<ans;
 } 
```
