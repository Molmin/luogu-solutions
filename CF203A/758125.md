## 正文
这题很简单，按照要求依次模拟即可

先放代码

```cpp
#include <bits/stdc++.h>
using namespace std;
int x,t,a,b,d_a,d_b;
int main(){
    cin>>x>>t>>a>>b>>d_a>>d_b;
    if(x==0){cout<<"YES"<<endl;return 0;}//如果是0分,那么一定是可以达到的
    if(a>=x&&(a-x)%d_a==0&&(a-x)/d_a<t){cout<<"YES"<<endl;return 0;}//见解释1
    if(b>=x&&(b-x)%d_b==0&&(b-x)/d_b<t){cout<<"YES"<<endl;return 0;}//见解释2
    for(int i=0;i<t;i++){//开始枚举,见解释3
        for(int j=0;j<t;j++){
            if(a-d_a*i+b-d_b*j==x){
                cout<<"YES"<<endl;
                return 0;
            }
        }
    }
    cout<<"NO"<<endl;//凑不出这个分数
    
    return 0;
}

```

### 解释

- 解释一

首先如果分数 $x$ 比 $a$ 小，也就是说我们要判断的分数比第一题的分数少。

其次`(a-x)%d_a==0`的意思是，每分钟减少 $d\_a$ 分数，那么想让分数 $a$ 变成分数 $x$ 就需要保证 $a$ 与 $x$ 的差值 $a-x$ 对 $d\_a$ 取模一定要是 $0$ ！

还有`(a-x)/d_a<t`的意思是，你想让 $a$ 变成 $x$ 还需要满足的一个条件就是在有限的 $t$ 分钟内，我们要能把 $a$ 分拖到 $x$ 分！比如比赛只有 $1$ 分钟，$a$ 是 $100$ ，$x$ 是 $10$ ，$d\_a$ 是 $5$ 那么虽然 $a$ 可以变成 $x$ ，但由于时间的限制，我们还是不能得到 $x$ 分。

同解释一

- 解释三

如果以上的也不行，那我们就暴力枚举。$i$ 表示等待多长时间交第一题，$j$ 表示等待多长时间交第二题，`a-d_a*i+b-d_b*j==x`的意思是等待 $i$ 分钟交第一题，等待 $j$ 分钟交第二题。然后的总分是不是 $x$ 。
