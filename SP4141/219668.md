## ~~真怀念小奥啊~~

这道题是求欧（o）拉（le）函数。

还记得奥数老师是怎么讲的吗？用它除以它的每个质因数并乘以每个质因数减一。为什么？

互质是公约数只有1的两个整数。换句话说，互质代表一个数没有另外一个数的所有质因数。

一个数对n取模有n种情况，有n-1种情况是对n取模不为零的。而把n除以每个质因数再乘以每个质因数减一，就是再求除以n的每个质因数都除不尽的数的个数。没错，就是φ(n)。~~没听懂？再看一遍，好好消化一下~~

最后一个小细节，T<=20000，n<=10^6,不能用O(n)的做法,会超时。那么枚举质因数只用枚举到n的平方根就行了，最后在判断n是否等于一。因为n不等于一代表n就是个质数。

上代码：
```cpp
#include<iostream>
using namespace std;
int main(){
	int t;
	cin>>t;
	while(t--){//t组数据
		int n;
		cin>>n;
		int ans=n;
		for(int i=2;i*i<=n;i++){
			if(n%i==0){//i是n的质因数
				ans=ans/i*(i-1);//公式
				while(n%i==0){//把n的质因数i全除完
					n/=i;
				}
			}
		}
		if(n!=1){
			ans=ans/n*(n-1);
		}
		cout<<ans<<endl;//华丽的结束
	}
	return 0;
}
```
