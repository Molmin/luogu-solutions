最大值是不可能拿到的，因为电脑一定会拿走另外一半
但正是因此以及人先选的规则，人可以直接选第二大的值中行或者列跟最大值相同的那个行或者列，这样电脑一定会去拿该行或列中的最大值，剩下的，人拿到次大值的另一半，游戏就已经赢了

既然题目已经看明白，那答案就更简单，直接输出1和次大值即可，还判断个啥啊
```cpp
#include<iostream>
#include<algorithm>
using namespace std;

int main(){
	
	int n,nn[500];
  	cin>>n;
  	int end=n*(n-1)/2;
    	for(int i=0;i<end;i++){
		cin>>nn[i];
    	}
	sort(nn,nn+end);
	cout<<1<<endl<<nn[end-2];
	return 0;
}
```
