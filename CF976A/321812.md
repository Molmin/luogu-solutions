## 前言：
第一次写题解有一点点小紧张

------------
## 题意：
- 给你一个长度为 $n$ 的数（只含 $1$ 和 $0$ ）
- 两位可以换位置，两个1可以换为一个1
	
可以得出：

   当有$1$时，$0$ 无法处理， $1$ 最后会剩一个
    
------------

## 思路
- 当只有一位时直接输出
- 如果大于$2$位，就统计$0$的个数,第一位一定是$1$，接下来是多个$0$


------------
## 代码
```cpp
#include<bits/stdc++.h>//万能头好评 
using namespace std;
int n;
char s;
int a=0;//统计0的个数 
int main() {
	cin>>n;
	for(int i=1;i<=n;i++){
		cin>>s;//读入每一位 
		if(n==1)//只有一位 
		{
			cout<<s<<endl;
			return 0;//提前结束 
		}
		if(s=='0'){//统计0 
		a++;
		}
	}
	cout<<'1';//最后剩一个1 
	for(int i=1;i<=a;i++){
		cout<<'0';
	}
	return 0;//好习惯 
}
```
