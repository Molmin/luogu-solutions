### 题目分析
这是一道求斐波那契数列的题。输入一共分为几行，每行一个数，表示查找斐波那契数列的个数。
### 思路分析
只需要列出一个斐波那契递推公式。虽然说是一个模板，但是我还是解释一下吧。
* 首先，该函数接收一个
`int` 类型的参数 $s$，表示要求数列中的第几个数。
* 接下来进行特判，如果 $s$ 的值为 1 或 2，也就是下面的循环无法考虑的情况，那么就直接返回 1，就是斐波那契数列的前两项。
* 如果不符合特判，也就是 $s \ge 3$，就需要使用循环解决。$a,b $ 的初始值为 1，然后在声明 $c$ 用来存储两个数的和。
* 下一步进行循环，每一次，将 $s$ 减 1，表示还剩的循环次数 $-1$，并将 $c$ 设定为 $a + b$，将 $a$ 设定为 $b$，将 $b$ 设定为 $c$，也就相当于斐波那契数列中 $f(i) = f(i - 1) + f(i-2)$。
* 最后一步，返回值为 $c$，就是最终的结果。
```cpp
int fibo(int s)
{
    if(s==1 or s==2)return 1;
   	 int a=1,b=1,c;
    while(s>=2)
    {
        s--;
        int c=a+b;a=b;b=c;
    }
    return c;
}
```
需要注意的是，如果不加**特判**，一定会 WA，因为循环至少执行一次，而我们不需要他去执行。
