这道题题意：高桥君参加了一次考试，共有N题，所有问题的答案都是 1 - 4 之间的整数。 高桥君考试前并没有复习， 所以全都选了同一选项， 期中考试答案发布后， 高桥不记得自己选了哪个选项了， 但是高桥想知道自己的分数。请输出高桥最高和最低的得分.


------------

思路:****先用一个字符串读入，再用a数组记录答案各选项的数量，再排序，输出最大最小即可。****


------------

代码：
```cpp
#include<iostream>
#include<algorithm>//sort的头文件 
using namespace std;
int a[5],n;//因为有4个选项，所以开5的数组 （防炸）
string s;
int main()
{
int n;
cin>>n>>s;//输入有多少道题和答案的选项 
for(int i=0;i<n;i++)a[s[i]-'0']++;//处理字符，把字符改为数字 
sort(a+1,a+5);//把各选项的选择数量 用sort排序（~~太懒了~~） ，sort默认从小到大 
cout<<a[4]<<" "<<a[1]<<endl;//输出最大最小（必须换行，不然要炸） 
return 0;
}
```
