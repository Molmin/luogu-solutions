# 经验区

2、每次的读入变量反复用。

3、提前计算好借位。

# 代码区

```
#include<cstdio>
using namespace std;
int main()
{
	int a,b,c,d,e,f;//六个时、分、秒
	for(int i=1;i<=3;i++)
	{
		scanf("%d%d%d%d%d%d",&a,&b,&c,&d,&e,&f);
		if(f<c)//秒不够减，借一分钟
		{
			e--;
			f+=60;
		}
		if(e<b)//分钟不够，借一小时
		{
			d--;
			e+=60;
		}
		printf("%d %d %d\n",d-a,e-b,f-c);
	}//别忘换行
	return 0;
}
```
