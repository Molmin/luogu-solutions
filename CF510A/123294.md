### 题解 CF510A 【Fox And Snake】

~~一道简单的模拟题~~

#### 思路：通过观察，我们可以发现，对于第n行

- #### 当n是奇数时，都是输出m个"#"

- #### 当n%4==2时，都是先输出（m-1）个"."，再输出一个"#"

- #### 当n%4==0时，都是先输出一个"#"，再输出（m-1）个"."

### 有了这个规律，我们就可以愉快的模拟了QWQ~

为了简化代码，我们可以先把（m-1）个"."用一个字符串存储下来，需要的时候输出就行了

### 代码（经过简化后只有16行）如下
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,m;
string a;
int main()
{
	cin>>n>>m;
	for(int x=1;x<m;x++)a+='.';//存储，方便输出 
	for(int x=1;x<=n;x++){
		if(x%2)for(int y=1;y<=m;y++)cout<<"#";
		else if(x%4==2)cout<<a<<"#";
		else cout<<"#"<<a;//根据思路模拟
		cout<<endl;//别忘了换行 
	}
	return 0;
}//望管理员通过
```
