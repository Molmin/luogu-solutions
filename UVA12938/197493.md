**思路一**

看到数据范围，容易知道 $n$ 都是**四位数**，所以 $\forall n$ ，修改任意一位数字，一共有 $40\ (4 \times 10)$ 种结果（如果算得细一点，只有 $35$ 种）

然后看到 $T \leq 1000$，于是对于所有的可能判断一下是否是完全平方数就可以啦。 $O(T)$，可过。

至于如何判断完全平方数，使用``if(pow((int)sqrt(n),2)==n)``判断即可（$C$++会把整型变量自动取整）

这个方法代码暂不放了

**思路二**

因为 $n$ 都是四位数，所以改出来的数字也一定是最四位数。

而容易知道四位的完全平方数只有 $68$ 个（$32^2$ ~ $99^2$）（可以打表），所以对于每一个 $n$，把它与这 $68$ 个完全平方数依次比较，如果只有一位数字不同，那么对应的 $ans$ 就 +$1$。稍微比思路一快一点（不过也没快多少就是了）。

贴上代码：

```
#include<bits/stdc++.h>
using namespace std;
int a[100]={1024,1089,1156,1225,1296,1369,1444,1521,1600,1681,1764,1849,1936,2025,2116,2209,2304,2401,2500,2601,2704,2809,2916,3025,3136,3249,3364,3481,3600,3721,3844,3969,4096,4225,4356,4489,4624,4761,4900,5041,5184,5329,5476,5625,5776,5929,6084,6241,6400,6561,6724,6889,7056,7225,7396,7569,7744,7921,8100,8281,8464,8649,8836,9025,9216,9409,9604,9801};
//cnt=68;
int Compare4(int k,int b){
	int tmp=0;
	if(k%10000/1000==b%10000/1000)++tmp;
	if(k%1000/100==b%1000/100)++tmp;
	if(k%100/10==b%100/10)++tmp;
	if(k%10==b%10)++tmp;
	return tmp;
}
signed main() {
	ios::sync_with_stdio(0);
	int T;
	cin>>T;
	for(int i=1;i<=T;++i){
		int n;
		cin>>n;
		int ansn=0;
		for(int u=0;u<68;++u)
			if(Compare4(n,a[u])==3)
				++ansn;
		cout<<"Case "<<i<<": "<<ansn<<endl;
	}
}
```
