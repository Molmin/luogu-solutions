# 红题！！！
本题思路：

∵两个面加起来==7

∴另一个面==7-输入面数据
### 具体见代码
```cpp
#include<cstdio>//scanf,printf头文件
int main(){           //主函数
    int a;            //定义输入的变量
    scanf("%d",&a);   //scanf语法:int类型使用%d输入
    printf("%d",7-a); //输出7与它的差
    return 0;}        //c/c++的main函数必须是int类型，而且最后要return 0。这不仅对洛谷其他题目有效，而且也是noip/noi比赛的要求！
```
