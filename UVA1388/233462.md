总评：一道稍稍需要思维含量的题。

首先可以自己手造几组样例，分析一下发现总是有一个雕塑没有移动。那么我们可以猜想，对于每种情况，都有一个雕塑没有移动。根据这个性质，我们可以将这个没有移动的雕塑设成原点，单位长度为$\dfrac{m+n}{n}$。

那么我们可以确定每一个雕塑初始的位置，则移动后的位置一定是整数，直接四舍五入求即可。

这样是否会出现两个雕塑移动到同一位置的情况？不会，因为初始雕塑距离都大于 $1$，四舍五入后不可能是同一个整数。

但是我们的猜想可能并不总是成立，下面来证明一下。如果所有雕塑都移动了，则可以将移动距离最近的那个点不动，其他点的移动距离总和相对于初始来说不会增加且可能减少，因此这个猜想总是成立的。

推导完成后代码非常简短：

```cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
	int n,m;
	while(scanf("%d%d",&n,&m)==2){//读入多组数据
		m+=n;//最终的雕塑数
		double x=(double)m/(double)n;
		double ans=0;
		for(int i=1;i<=n;i++){
			double a=x*(double)(i-1);
			int p=floor(a+0.5);
			ans+=abs(p-a);
		}
		printf("%.4lf\n",ans*10000/(double)m);
	}
	return 0;
}```
