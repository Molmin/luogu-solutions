所以，今天，本蒟蒻就来介绍用$for$循环的方法  
代码如下：  
```
#include <stdio.h>
int main()
{
	int a[10],i,ans=0;//题意说需要定义10个数，本蒟蒻就定义10个数（省空间（逃~~）
	for(i=0;i<10;i++)//读入
		scanf("%d",&a[i]);
	for(i=1;i<10;i++)//出了a[0]外，把其他数字求和
		ans+=a[i];
	printf("%d\n",a[0]-ans);//输出
}//完美的收尾23333
```
