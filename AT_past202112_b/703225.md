## 题目大意
日本正在使用六种硬币，分别是：$1$ 日元、 $5$ 日元、 $10$ 日元、 $50$ 日元、 $100$ 日元和 $500$ 日元。 其中，$5$ 日元和 $50$ 日元硬币上有洞。

有 $n$ 位顾客，分别给出每位顾客花的钱 $a$，以及支付的钱 $b$，问找回的前中有多少个硬币是带孔的（并且找零是用最少的面额）

## 题目解法
看到这道题目，我最先想到的是模拟，但是细细地看了一下，这道题目不用非得用模拟算法，但是需要思考一下。

我们找零要尽量找大面额，如果大面额不够再用小面额，但是我们只需要求 $50$ 日元和 $5$ 日元需要用多少张就可以了。

我们把要找零的金额定义为 $n$，也就是 $b-a$。

如果我们在用 $50$ 日元硬币之前，要先用 $100$ 日元硬币，如果用完 $100$ 日元找零之后剩余的钱 $\ge 50$，那么此时肯定会用 $1$ 个 $50$ 日元的硬币继续找零。我们用 $n \bmod 100$，就是用完 $100$ 日元找零之后剩余的钱。（因为 $500$ 日元和 $100$ 日元都可以被 $100$ 整除）。

同理，用 $5$ 日元硬币之前，要先用 $10$ 日元硬币，如果 $(n \bmod 10) \ge 5$，那么就会用 $1$ 个 $5$ 日元的硬币。

### 参考代码
```cpp
#include<iostream>
using namespace std;
int ans,a,b,n;
void solve()
{
	cin>>a>>b;
	n=b-a;
	if(n%100>=50) ans++;
    if(n%10>=5) ans++;
}
int main()
{
	int t;
	cin>>t;
	while(t--) 
		solve();
	cout<<ans<<endl;
 }
```
