入门题，只需要用到基本的贪心思想就可以做出来。

题意可以自行查看。

首先我们思考：什么情况可以用最少的硬币？

答案是：尽量用大面额的，如果当前的面额不能用就从大到小依次枚举。

接下来思考：什么情况我们会用到有洞硬币（$5$ 日元和 $50$ 日元）？

由题目可知：比 $5$ 日元的面额大的硬币中，最小的是 $10$ 日元，而比 $50$ 日元的面额大的硬币中，最小的是 $100$ 日元。

得出结论：当 $10$ 日元找不了时，会用 $5$ 日元；在 $100$ 日元找不了时，会用 $50$ 日元。

至此，本题得解。代码极易：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n; cin >> n;
    int ans = 0;
    for(int i = 1;i <= n;i++) {
        int a,b; cin >> a >> b;
        int cost = b - a; 
        
        if(cost % 100 >= 50) ans++;
        if(cost % 10 >= 5) ans++;
    }
    cout << ans << endl;
    return 0;
}
```
