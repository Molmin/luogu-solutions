# [题目传送门](https://www.luogu.com.cn/problem/AT_past202112_b)

## 题意归纳

日本正在使用六种硬币：$1$ 日元、$5$ 日元、$10$ 日元、$50$ 日元、$100$ 日元和 $500$ 日元。$5$ 日元和 $5$日元硬币上有洞。

高桥在日本某处经营一家商店。一天，$N$ 位顾客进了他的商店。第 $i$ 位顾客支付了   $B_i$ 日元买了一件价格为 $A_i$ 日元，高桥用最少的硬币去给每位顾客找零。

在当天用作找零的硬币中，有多少个硬币上有洞？

假设商店里每种硬币都足够多。

## 解题思路

本题要求“在当天用作找零的硬币中，有多少个硬币上有洞”，那么就是求 $B_i$ 与  $A_i$ 的差 $C_i$ 中用最少的硬币找零，有多少个硬币是 $5$ 日元或 $50$ 日元的。所以我们就用一个 $coinHole$ 函数来计算当前这位客人的找零硬币中有孔的数量，然后用计数器变量 $cnt$ 来累加就可以了。总体来说这道题还是比较水的。

## 函数实现

函数引入实参 $C_i$，也就是 $B_i$ 与  $A_i$ 的差。由于题目“用最少的硬币去给每位顾客找零”，所以我们要从面值大的货币来计算。

## 算法使用

模拟+递归

## 代码

```cpp
#include<bits/stdc++.h>//万能头文件
#define int  long long
#define F(i,a,b) for(int i=(a);i<=(b);i++)
using namespace std;
int n,a,b,cnt,c; 
int coinHole( int qwq) //当前客人的找零的硬币中带洞硬币的数量
{
	//用数量尽量少的硬币找零
	if(qwq>=500) return coinHole(qwq%500);//递归，求用500元找完零还需要多少
	if(qwq>=100) return coinHole(qwq%100);//递归，求用100元找完零还需要多少
	if(qwq>=50) 
	{
		cnt+=qwq/50;//由于50日元硬币有洞，所以cnt计数变量要加上找零中50日元硬币的数量
		return coinHole(qwq%50);//递归，求用50元找完零还需要多少
	}
	if(qwq>=10) return coinHole(qwq%10);//递归，求用10元找完零还需要多少
	if(qwq>=5) 
	{
		cnt+=qwq/5;//由于5日元硬币有洞，所以cnt计数变量要加上找零中5日元硬币的数量
		return coinHole(qwq%5);;//递归，求用50元找完零还需要多少
	}
	if(qwq>=1) return 0;//如果其他面值硬币都找不开，那么只能用1日元的硬币来找零，因为1日元硬币没有洞，找完零也不会剩下，所以直接返回0
	if(qwq==500||qwq==100||qwq==10||qwq==1)
		return 0;//如果刚好找的开，直接返回0
	
}
signed main()
{
	cin>>n;
	F(i,1,n)
	{
		cin>>a>>b;
		c=b-a;//A_i与B_i的差
		cnt+=coinHole(c); //累加
	}
	cout<<cnt<<endl;
	return 0;
}

```
