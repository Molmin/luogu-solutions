## AT_past202112_b 题解

### 思路

通过读题，我们可以知道高桥会每次都尽可能用面额较大的硬币去给顾客找零，并且不用考虑硬币剩余的数量，所以当高桥需要找给顾客的零钱不能被若干个面值为 $100$ 的日元所整除，并且余数 $\geq50$，就会找给顾客一个面值为 $50$ 的日元，就多了一个有洞的硬币。同理，当高桥需要找给顾客的零钱不能被若干个面值为 $10$ 的日元所整除，并且余数 $\geq5$，就会找给顾客一个面值为 $5$ 的日元，就又多了一个有洞的硬币。

### AC  Code

```c++
#include<bits/stdc++.h>
using namespace std;
int n,a,b,sum;//不用开longlong
int main(){
	scanf("%d",&n);
	while(n--){
		scanf("%d%d",&a,&b);
		if((b-a)%100>=50) sum++;//余数大于等于50的情况
		if((b-a)%10>=5) sum++;//余数大于等于5的情况
	}
	printf("%d",sum);
	return 0;
} 
```
