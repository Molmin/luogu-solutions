### 思路

我们从最大的面额开始找。

顺序是：$500 \to 100 \to 50 \to 10 \to 5 \to 1 $。（单位：日元）

有洞的硬币面额分别是 $50$ 日元和 $5$ 日元。

所以如果找零的时候剩下的钱 $\le 100$ 日元，并且 $\ge 50$ 日元，则多一枚带洞的硬币。

同理，如果找零的时候剩下的钱 $\le 10$ 日元，并且 $\ge 5$ 日元，则再多一枚带洞的硬币。

此部分代码为：

```cpp
if( ( y - x ) % 100 >= 50 )
{
	sum++;
}
if( ( y - x ) % 10 >= 5 )
{
	sum++;
}
```
### 代码

```cpp
#include<cstdio>
int main() 
{
    int n , sum = 0;
    scanf( "%d" , &n );
    for( int i = 0; i < n; i++ )
    {
        int x , y;
        scanf( "%d %d" , &x , &y );
        if( ( y - x ) % 100 >= 50 )
        {
            sum++;
        }
        if( ( y - x ) % 10 >= 5 )
        {
            sum++;
        }
    }
    printf( "%d" , sum );
    return 0;
}
```
