## 题目思路

本题要求出每位~~倒霉的~~顾客会收到多少枚有洞的硬币，高桥君要用**最少**的硬币给每位顾客找零，所以要从**最大**面值的硬币开始找零。

找零顺序是：
$100$ 日元 $\rarr 50$ 日元 $\rarr 10$ 日元 $\rarr 5$ 日元 $\rarr 1$ 日元。

而有洞的硬币是 $50$ 日元的硬币和 $5$ 日元的硬币。

什么时候需要这些有洞的硬币呢？

由于找零时是从**面值大**的硬币开始找零，在使用 $50$ 日元的硬币找零前会先使用 $100$ 日元的硬币找零，而使用 $5$ 日元的硬币找零前会使用 $10$ 日元的硬币找零。

所以如果找零完 $100$ 日元的硬币后剩下的钱 $\ge50$ 日元，或者找零完 $10$ 日元的硬币后剩下的钱 $\ge5$ 日元，找零的硬币就是有洞的硬币。

得出以上结论后，代码就好写了。

## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int main() {
    int n,ans=0;
    cin>>n;//来了n位顾客
    while(n--) {
        int a,b;
        cin>>a>>b;//物品的价格和付的钱数
        int x=b-a;//找零的钱数
        if(x%100>=50)ans++;
        if(x%10>=5)ans++;
    }
    cout<<ans<<endl;
    return 0;
}
```
