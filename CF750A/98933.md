#### 题目：Limak有n个任务，完成时间分别为1×5,2×5,……，n×5。而Limak走到他朋友家需要k分钟。问他在4小时内在确保能到达朋友家的同时最多能做多少任务。
### 大概思路：先算出一共有多少时间，然后再用循环语句算出可以做多少个任务
#### 代码如下（pascal）:
```pascal
var
 i,n,k,s,m:longint;
begin
 read(n,k);
 m:=240-k;//算出共有多少时间可以用来做任务
 for i:=1 to n do //枚举每一件任务
  if (i*5<=m) then //每一件任务的时间如果足够
   begin
    m:=m-i*5;//减去用掉的时间
    inc(s);//任务数+1
   end
   else break;
   //不然直接退出循环（因为i*5已经大于剩余时间了，所以可以直接退出，当然不退出也是可以的，因为后面已经没有一个任务的时间符合条件了）
  write(s);//输出做的任务数量
end. 
```
