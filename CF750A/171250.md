本题是一道模拟题，再加上数据很良心，只要暴力模拟就能过了。想法很简单，如果能把任务做完就输出总任务数，否则就一直循环直至时间不够，然后输出任务数减一（最后那个完成不了）。然而我一开始输入n=4竟然输出5，于是我就索性把最终答案减一，结果就对了。

 _代码如下：_ 

------------
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,k,i;//定义变量
int main()
{cin>>n>>k;//输入
k=(240-k)/5;//计算做任务的时间（为了简化就/5）
for(i=1;i<=n;i++)//循环计算最多能做几个任务
  {k-=i;//减去做任务的时间
   if(k<0) break;//时间超限就退出
  }
cout<<i-1;//输出i-1
return 0;
}
```
