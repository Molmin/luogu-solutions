这道题有一点点简单。

被$3$整除的条件是：数字和被$3$整除。

由于每$3$个连续数的数字和都被3整除，所以我们只要看最后的数字。

最后的数字有$3$种可能性：（模$3$）余$0,1,2$。

当余$0$时，原数可以被$3$整除~~（显然）~~

余$1$时则会最后也余$1$；

余$2$时则也会余$0$。$（1+2）%3=0$。

所以只要判断输入的数模$3$值。

源代码：

```
#include<iostream>
using namespace std;
int main(){
int a,z;
cin>>a;
if(a%3==0||a%3==2)cout<<"YES";
else cout<<"NO";
return 0;}
```
