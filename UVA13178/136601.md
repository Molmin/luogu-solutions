### 中文翻译：
```cpp
第1行给出一个数字t，第2-t+1行分别有1个数字n1、n2...nt。
如果一个由1到nx组成的数字能被三整除，那么输出YES并换行，否则输出NO同样换行。（如n=8，这个数字为12345678，能被3整除，所以输出YES）
```
### 大致思考：
```
一个能被三整除的数，其各位数字之和定为三的倍数，而又因为0+9b+99c+...+99...9z≡0(mod 3)，所以a+10b+.....+100...0z≡a+b+....+z（mod 3）。
所以要看这样组成的数字是不是3的倍数，只要看从1加到n是不是三的倍数。根据高斯求和，1+2+...+n-1+n=(1+n)*n/2。所以只要1+n或n是3的倍数，组成的数字就能被三整除。即如果n%3==0或2，输出YES，否则输出NO。
```
### 代码如下：
```cpp
#include<bits/stdc++.h>
using namespace std;
int n,a;
int main(){
	cin>>a;//输入数据总数
	for(int i=1;i<=n;i++){
		cin>>n;//输入一组测试数据
		if(n%3==1) cout<<"NO"<<endl;//如果n%3的值是1，那么这样的数不是3的倍数，原因见上
		else cout<<"YES"<<endl;//否则这样构成的数是3的倍数
	}
}
```
