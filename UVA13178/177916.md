题目翻译:
```
先输入n，然后输入n个数。
每一个数需要经过以下过程：
	把这个数换成从1~n，如4变成1234,12变成123456789101112
然后判断这个新数能不能被3整除。能就输出YES，否则输出NO
```
谁都知道，如果一个数的数字和能被3整除，那这个数就能被3整除。

另外，我们可以用乱切法，也就是说，判断9999，可以拆成99和99，和198能被3整除，所以9999能被3整除 (其实直接除以3就行了)

但是面对这道题就有用武之地了。就用这种方法，干起来！

**友情提示：$n<={10}^9$，当n达到最大值时，这个和将会爆`int`，但是不会爆`long long`**

求和公式还记得吗？

$1+2+3+...+n=\dfrac{n(n+1)}{2}$

**其实从1直接循环加到n会T掉**

所以直接上代码吧

```cpp
#include<bits/stdc++.h>
using namespace std;
int main()
{
    long long n;cin>>n;//读入
    long long c;long long x;
    //这里c都不用归零
    for(int i=1;i<=n;i++)
    {
        cin>>x;//一个一个处理
        c=(1+x)*(x)/2;//求和
        if(c%3==0)cout<<"YES"<<endl;//是
        else cout<<"NO"<<endl;//非
    }
    cout<<"禁止抄袭，人人有责！"<<endl;
    return 0;
}
```
