# 题目描述
对于一个01字符串，当它中间0的数量和1的数量不同时，称它为一个好的序列，否则称它为一个不好的序列。

给出一个长度为nn的01字符串，将它划分为几个字符串，使划分出的几个字符串都是好的序列。

求最少划分出几个字符串，并任意输出一种方案。


------------

# 输入输出格式
## 输入格式
第一行是一个整数n(1<=n<=100)，表示字符串长度

第二行是由01组成的长为n的字符串
## 输出格式
第一行是一个整数k，表示分割成k个字符串

第二行是分解方案，多个题解之间用空格隔开


------------

# 解题方法
## 字符串s有两种可能：
### 1.s是好的字符串
这种情况直接输出原字符串即可
### 2.s是不好的字符串
显然,k最小是2。

显然，一个单独的1或0是好的序列

显然，将不好的序列的第一个字符拿掉，剩余序列也是好的

所以把s分成两个——第一个字符单独组成的字符串和剩余的字符组成的字符串——是最佳方案

# AC代码
```cpp
#include<iostream>
using namespace std;
int main()
{
	int n,t0=0,t1=0;
	char a[101];
	cin>>n;
	for(int i=0;i<n;i++)
	{
		cin>>a[i];
		if(a[i]=='0')
			t0++;
		else
			t1++;
	}
	if(t0==t1)
	{
		cout<<2<<endl;
		cout<<a[0]<<' ';
		for(int i=1;i<n;i++)
			cout<<a[i];
	}
	else
	{
		cout<<1<<endl;
		for(int i=0;i<n;i++)
			cout<<a[i]<<' ';
	}
	return 0;
}
```
