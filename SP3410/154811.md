------------
题意：**给你一个n\*n的正方形网格图，问你其中有多少个大大小小的正方形。**

------------
看完这道题，我们可以发现一些规律：

![](https://cdn.luogu.com.cn/upload/pic/58524.png)

仔细观察上图，可以发现： _**一个n\*n的正方形网格图，包含大小正方形个数为：$1^2$+$2^2$+$3^2$+······+$n^2$**_ 。知道规律后就好办了。

------------

```cpp
#include<bits/stdc++.h>
using namespace std;
int n;
int main() {
	while(1){
		cin>>n;//输入到n
		if(n==0)return 0;//如果n为0就停止读入
		int s=0;//s表示n*n的正方形网格中大小正方形的数量
		for(int i=1;i<=n;i++)//从1~n枚举，每次加i的平方
		s+=pow(i,2);//将i的平方相加（相当于加i*i）
		cout<<s<<endl;//输出
	}
	return 0;//养成好习惯
}
```
