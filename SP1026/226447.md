对于这样一道求期望的题目,我们可以很轻易的知道要用期望 DP 。

 然而我们期望 DP 的状态一般都是有后效性的,
 
 所以我们要用逆推的方式(优先考虑得到该状态会怎么样,而不是怎样转移到该状态)
 
 然而我们观察题目发现题目中只有 $2$ 个参数:
 
**1.已经出现了的点的个数**
 
**2.投掷骰子的次数**
 
 但是我们的投掷骰子的次数,
 
 就是我们要求的结果,
 
 那么我们的 $f_i$ 的 $i$ 就表示已经出现了 $i$ 个点,
 
  $f_i$ 就表示有 $i$ 个点已经出现了,
 
 要投出剩下的 $n-i$ 个点的期望次数
 
 那么对于这个期望,我们对于每一次投掷骰子进行分析:
  
 我们每次投掷骰子都有两种情况:
 
1. 投到已经出现的数字: 概率为 $i/n$ 
 
2. 投到没有出现的数字: 概率为 $1-i/n$ 
 
 那么我们根据这两点就可以轻易的写出状态转移方程:
 
 $f_i = (i/n)*f_i+((n-i)/n)*f_{i+1}$
 
  化简可得
  
 $ f_i = f_{i+1}+(n/(n-i)) $
  
 那么我们就可以根据这个方程进行逆推;
 
 因为出现 $n$ 个数字时不用再投掷,所以 $f_n=0$ ;
 
 而我们是从一个数字都没有开始投掷的,所以最终输出就是 $f_0$ ;	
 ```cpp
#include<bits/stdc++.h>
using namespace std;
#define N 1010
int T,n;
double f[N];
int main(){
	//freopen(".in","r",stdin);
	//freopen(".out","w",stdout);
	cin>>T;
	while(T--){
		scanf("%d",&n);
		memset(f,0,sizeof(f));
		f[n]=0;
		for(int i=n-1;i>=0;--i){
			f[i]=f[i+1]+1.0*n/(n-i);
		}
		printf("%.2lf\n",f[0]);
	}
	return 0;
}
```
