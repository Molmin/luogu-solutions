本题属于贪心间隔类题目，要求用最少的稻草人覆盖所有长着谷物的土地。('.'长着谷物，'#'为废地)
#### 策略选择
1. 选中第一块长着谷物的土地，从这块土地开始连续三块土地归为一个稻草人的覆盖范围。(如果要记录稻草人的位置, 中间的位置即是稻草人的位置)
2. 下一块土地的选择，从前一个稻草人覆盖范后的下一块土地开始，跳过标记为'#'的废地，遇到'.'的第后重复第一步。
#### 分析
1. 有谷物的地方一定会被稻草人覆盖，那么第一块有谷物的情况有：".##"、".#."、"...",我们一遇到有谷物的地，就把他往后的三块地都覆盖住（稻草人插中间），可以满足任意情况，具有最优子结构。
2. 在"......"这个样例中，采用策略选择第一个位置，覆盖123号位置后，相当于把前三个位置截去，样例变成了"..."，所以每步策略做出后，不会影响下一段地的选择，无后效性。
3. 边界问题，如果选到最后，剩下"#."、"##"、".."或者单个地，等不足三块地的情况。如果是"..""#.".#""."等情况,我们的策略最终会检测到第一个'.'然后稻草人+1。而"##""#"则不会被选中。
```cpp
#include <cstdio>
int main()
{
    int t, c = 1; scanf("%d", &t);
    while(t--)
    {
        char s[110];
        int n; scanf("%d", &n);
        getchar(); gets(s);
        int i = 0, ans = 0;
        while(i < n)
        {
            if(s[i] == '#'){ i++; continue; }//跳过废地
            if(s[i] == '.' || s[(i + 1) % n] == '.' || s[(i + 2) % n] == '.')
                ans++;
            i += 3;//每隔三块地进行选择
        }
        printf("Case %d: %d\n", c++, ans);
    }
    return 0;
}
```
