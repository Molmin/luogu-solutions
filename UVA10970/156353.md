## 思路：
切的次数|增加的块数
:-:|:-:
1|1
2|2
3|3
:|:
n|n

可以发现，每切一刀就会增加1块，所以要增加几块就要切几刀。

切了N刀以后有```N*M```块，但因为最开始有1块，所以增加了```N*M-1```块，就要切```N*M-1```刀
## 代码
```
#include<bits/stdc++.h>//万能头
using namespace std;
int main()
{
	int n,m,ans;
	while(cin>>n>>m)//因为有多组输入，所以要一直输入，直到输入完为止
	{
		ans=n*m-1;//切N*M刀
		cout<<ans<<endl;//不要忘了换行
	}
	return 0;
} 
```
