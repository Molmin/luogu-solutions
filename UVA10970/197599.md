由于每切一刀就增加一块巧克力，而原先已经有一块（它本身），所以可以得出共需 $m*n-1$ 刀。

那么代码就很简单了~~（怪不得题是红色的）~~

```python
while 1==1:           #疯狂循环
    s=input().split()
    if s!='\n':       #判断，没有完
        m=int(s[0])   #m,n的定义
        n=int(s[1])
        op=m*n-1      #这里我用变量计算了，其实也可以直接在print里算
        print(op)     #输出（自带换行）
    else:             #完成输入，退出
        break
```

其实就是超简单的数论，稍加推算就得出来了。
