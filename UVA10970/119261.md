$$Ans=N*M-1$$  
这里给个简单的证明


-------------

如果我们先将巧克力切成等宽的$M$行，再把每行切成$N$列即可满足要求，一共需要切$N*M-1$次



因为每切一次，只能多增加一块巧克力，所以将一块完整的巧克力切成$N*M$块至少需要$N*M-1$次  



有一种已知的方法能在$N*M-1$次切完，而最少需要切$N*M-1$次，所以结论得证  


-------------------

```cpp
#include<bits/stdc++.h>
int ans,n,m;
int main()
{
	while(scanf("%d%d",&n,&m)!=EOF){
		printf("%d\n",n*m-1);
	}
	return 0;
}
```
