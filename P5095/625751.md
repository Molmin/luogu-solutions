# **2.题目意思及思路**
题意：给你两个数，以及书的高度和宽度。问你在每排宽度不超过指定宽度的情况下，书架高度最小为多少，每层高度为这一层最高书的宽度。

思路：我们设当前书本为 $i$，则 $f(i)$ 为到第 $i$ 本书时，书架的最小高度。我们用动态规划的思想。有两种情况。一：把当前这本书插入之前的书架。二：另开一层书架，把当前这本书放新开的书架。
     
动态转移方程为 $f(i)=min(f(i),f(j-1)+maxn)$。

**注意我们可以用前缀来加快速度，如下图。**

