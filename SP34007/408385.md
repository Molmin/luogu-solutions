## 题意简述
$T$ 组数据，在平面直角坐标系中，设一个圆的方程为 $x^2+y^2-px-qy+z=0$，每次给定 $p,q,z$ 和 $n$ 组点，每个点给定 $x,y$ 表示其坐标，输出这 $n$ 个点中，位置在圆上或圆内的点个数。
## 分析
设一个点到圆心的距离为 $d$，圆的半径为 $r$,如果 $d\le r$，说明点在圆上或圆内，设点坐标为 $(x_1,y_1)$，圆坐标为 $(x_2,y_2)$，根据平面内点距离公式可以求出 $d=\sqrt{(x_1-x_2)^2+(y_1-y_2)^2}$。

因此问题的关键就转化为求圆心的坐标 $(x_2,y_2)$ 和圆的半径 $r$。

圆的标准方程 $x^2+y^2-px-qy+z=0$ 

可以转化为 $(x-\frac{p}{2})^2+(y-\frac{q}{2})^2-\frac{p^2+q^2}{4}+z=0$

移项得到 $(x-\frac{p}{2})^2+(y-\frac{q}{2})^2=(\frac{p^2+q^2}{4}-z)$。

根据高中知识，等式右边其实就是圆的半径的平方（即 $r^2$），等式左边能满足两个平方项和为 $0$ 的 $x,y$ 值就是圆心坐标 $(x_1,y_1)$。

根据推导过程得到 $x_2=\frac{p}{2},y_2=\frac{q}{2},r=\sqrt{\frac{p^2+q^2}{4}-z}$，同时我们发现所要判断的 $d,r$ 的式子都含有根号，因此两边同时平方，最后我们只要判断 $[(x_1-\frac{p}{2})^2+(y_1-\frac{q}{2})^2]$ 和 $(\frac{p^2+q^2}{4}-z)$ 的大小关系即可。
## AC 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int T;
int main()
{
	cin>>T;
	while(T--)
	{
		int p,q,z,n,ans=0;
		cin>>p>>q>>z>>n;
		double xx=p/2.0,yy=q/2.0,r=((1.0*p*p+1.0*q*q)/4.0)-z*1.0;//注意int转double要乘上1.0 
		while(n--)
		{
			int x,y;
			cin>>x>>y;
			if((x*1.0-xx)*(x*1.0-xx)+(y*1.0-yy)*(y*1.0-yy)<=r)
				ans++;
		}
		cout<<ans<<endl;
	}
	return 0;
}  
```
