## CF851A 题解
### 题目分析：
本题题面较为复杂，导致像我这样的初学者有点糊涂。但是，静下心来仔细推一推式子，看看样例解析。很显然，只有三种情况。
- 如果 $t\le k$，就是 $t$ 个人站着
- 如果 $k\le t\le n$，就是 $k$ 个人站着
- 如果 $n\le t\le n+k$，就是 $n+k-t$ 个人站着

在了解了这三种情况的时候，就可以写出代码了。

### 题目难度及考点：
大致难度在 `入门` 左右，主要是考思维，代码较为简单。

### 做法分析：
只需要两个 $if$ 语句，先判断 $t\le k$ 成立不成立，如果成立，输出 $t$ 。否则判断 $k\le t\le n$ 成不成立，如果成立，输出 $k$ 。否则直接输出 $n+k-t$ 。

知道了上述结论及程序分析，就可以开始写代码了

### 完整代码：
```cpp
#include<iostream>
using namespace std;
long long n,k,t,ans;  //因为数据不小，所以定义long long类型
int main(){
    scanf("%lld%lld%lld",&n,&k,&t);  //输入需要用 %lld，输出也是
    if(t<=k)           //判断 $t\le k$ 成立不成立
        printf("%lld\n",t); //成立，输出 t
    else 
        if(t<=n)     //否则判断 $k\le t\le n$ 成立不成立
            printf("%lld\n",k);   //成立，输出 k 
    else 
        printf("%lld\n",k-t+n);   //否则直接输出 n+t-k
}
```
