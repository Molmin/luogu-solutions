#### 可以发现如下规律：

当t≤k时	输出t

当n≥t＞k时	输出k

当t＞n时	输出n+k-t
### **代码**
```pascal
var
 n,k,t:longint;//定义n，k和t
begin
 readln(n,k,t);//读入n、k、t
 if t<=k then writeln(t) else if (t>k) and (n>=t) then writeln(k) else writeln(n+k-t);//根据规律模拟
end.
```
