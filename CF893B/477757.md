题意说的很清楚了，因为 $n$ 的范围只有 $10^5$ ，而在这个范围里可以表示成 $(2^k-1)\times (2^{k-1})$ 的只有 $1,6,28,120,496,2016,8128,32640$，可以直接拿一个数组存起来，从大往小枚举。

## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int a[8]={1,6,28,120,496,2016,8128,32640},n;
int main(){
	cin>>n;
	for(int i=7;i>-1;i--)if(n%a[i]==0)cout<<a[i],exit(0);
}
```
