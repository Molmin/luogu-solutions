# Atcoder红题祭！

首先我们认真读题会发现，其实这道题不难；

过了这道题后，我看了各位大佬的题解，发现对新手有点不友好，于是有了这篇题解：

~~~
核心思想：

先输入n，接着循环读入

每次读入五个数，如果小于20，就sum++

全部读入之后，就可以输出了，要记得换行
~~~

那么，要注意几点：

1. 头文件和使用命名空间（using namespace）

2. 输出endl

3. 由于数据较小，可以只用int

- 代码实现

```
#include<bits/stdc++.h>
using namespace std;
bool cnt(int a,int b,int c,int d,int e)
{
	return a+b+c+d+e<20;
}
int a1,a2,a3,a4,a5;
int n,sum=0;
int main()
{
	cin>>n;
	for(int i=0;i<n;i++)
	{
		cin>>a1>>a2>>a3>>a4>>a5;
		if(cnt(a1,a2,a3,a4,a5))sum++;
	}
	cout<<sum<<endl;
	return 0;
}
```

需要注意的是，我这里使用了一个函数，函数特点如下：

~~~
这是一个bool类型的函数，bool当中只存在两个值：true和false

其次，return语句后面直接写了一个判断，没有false或者true的字样，这是因为，如果return后面写的是比较语句（compare sentence），就会自动返回布尔值，即如果符合返回真，反之返回假

利用函数可以使整体内容更加清晰
~~~

然后呢，我们如果将其判断语句放入主函数中也可。

因为这道题较为简单，这里介绍几种思路，就不写代码了

1. 可以用read快速读入，当然如果不行也可以这么写：

$#define read(x) scanf("%d",&x)$

2. 可以将代码函数实现，也就是将循环、输出全部写成函数，这样虽然代码量较大，但是功能清晰

3. 将a~e五个数的和存在sum当中，然后：

k+=(sum<20)

这样就可以进行一个十分巧妙地判断，也就是说如果小，+1否则+0，因为逻辑比较只能返回0或者1.
