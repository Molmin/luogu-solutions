## 题意理解

给出一个正整数 $n$，再给出一组由 ```*``` 和 ```.``` 组成的五行 $4 \times n$ 列字符阵。这个字符阵由 ```.``` 表示空白部分，其余部分可以组成数字 $1$、$2$ 或 $3$，共 $n$ 个数字（每个数字四列），要求输出这 $n$ 个数字。

## 思路阐述

观察每个数字所对应字符阵的特点：

```
.*..	***.	***.
.*..	..*.	..*.
.*..	***.	***.
.*..	*...	..*.
.*..	***.	***.
```

第四行是可以单独辨别数字，所以我们只看这一行。再观察发现，每个数字的第四行前两个字符就是不同的：

```
.*	*.	..
```

所以我们只需要比较这两个字符就可以判断出对应的数字了。

## 代码呈现

```cpp
#include <bits/stdc++.h>
using namespace std;

int n;
string s;

signed main(){
    
    scanf("%d",&n);
    for (int i=1;i<=4;i++)//把前三行吃掉，再读入第四行
        cin>>s;
    for (int i=0;i<n;i++){
        if (s[i*4]=='.'){//每个数字的第一个字符
            if (s[i*4+1]=='*')//第二个字符
                printf("1");
            else printf("3");
        }
        else printf("2");
    }
    puts("");//注意末尾输出换行
    return 0;
    
}
```
