## 分析

看样例，我们不难发现：在数字 $1,2,3$ 中，只有第 $4$ 行是完全不一样的。那么我们是不是就可以直接看输入的字符矩阵的第四行就行了？因为有 $n$ 个数字，所以我们的第 $i$ 个数字的起始下标在字符串中是 $(i-1) × 4$。如果这个地方的字符是 `*`，可以推算出这个数字是 $2$；如果这个地方的右边是 `*`，可以推算出这个数字是 $1$；如果这个地方的右边的右边是 `*`，可以推算出这个数字是 $3$。

## 代码
```cpp
#include<bits/stdc++.h>
using namespace std;
int n;
string s;
signed main()
{
	cin>>n;
	for(int i=1;i<=3;i++) cin>>s;
	cin>>s;
	for(int i=1;i<=n;i++)
	{
		if(s[(i-1)*4+1]=='*') cout<<1;
		else if(s[(i-1)*4]=='*') cout<<2;
		else cout<<3;
	}
}
```
