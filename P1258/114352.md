标签写的是一道二分题，  
可明显不用更快（—！@！—）  
强行用数学，列个方程很好解  
设甲先坐车做了会到c地，距离b地z  
很容易理解是乙也走了z的路程  
所以甲超过了乙s（总长）-2z  
因为这段路时间相同  
所以甲时间（s-2x+x）/b==乙时间（x/a）  
所以的t值求出  
后面就很好证明了  
废话不说见代码
```cpp
#include <bits/stdc++.h>
using namespace std;
 int main() {
    double a,b,s,z;
    cin>>s>>a>>b;
    z=(2*a*s)/(3*a+b);
    z/=a;
    z=z+(s-z)/b;
    printf("%.6lf",z);
}
```
