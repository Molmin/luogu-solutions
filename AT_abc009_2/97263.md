这题可以说是水的可以了，2<=N<=100，N个数字的范围在1~1000，很明显是让你**桶排**  
### 不解释上代码： 
```cpp
//MADE BY ZLZ
#include<iostream>
int n,z=2,a[1005],i; //z用来记第二大的数
int main()
{
	std::cin>>n;
	while(n--)
	{
		cin>>i;
		a[i]=1;
	}
	for(i=1000;z;i--)z-=a[i];
	std::cout<<i+1<<endl; //注意i要加1，别忘了换行
}
```
