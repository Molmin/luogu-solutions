### 此题的坑

- 中间的部分也可以为空
- 求和时记得开long long
## 思路
#### 双指针
$l$ :左边的部分的边界（左边的部分包括l)

$r$ :右边的部分的边界（右边的部分包括r）

如果 ```左边的和```$>$```右边的和``` 就 扩大右边的边界 $(r--)$

如果 ```左边的和```$<=$```右边的和``` 就 扩大左边的边界 $(l++)$

出现相同的情况就更新答案（由于左边和右边的部分是不断扩大的，所以不用取最大值）

## 核心代码
```cpp
  long long suml = a[0], sumr = a[n - 1];//左边和右边的和
  int l = 0, r = n - 1;
  while(l < r) {
    if(sumr == suml) { //更新答案
      ans = sumr;
    }
    if(suml > sumr) {
      sumr += a[--r];
    } else {
      suml += a[++l];
    }
  }

```
