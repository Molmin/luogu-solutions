有道题叫做 $ A+B Problem $ ，则这道题就叫做 $ A*B Problem $  ！

来看看 $ A+B Problem $ 的代码：

```cpp
#include<bits/stdc++.h>
using namespace std;
int a,b;
int main()
{
	cin>>a>>b;
	cout<<a+b;
	return 0;
}
```

所以这题只需把 $ + $ 变成 $ * $ 即可

```cpp
#include<bits/stdc++.h>
using namespace std;
int a,b;
int main()
{
	cin>>a>>b;
	cout<<a*b;
	return 0;
}
```
