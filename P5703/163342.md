看懂题目背景后快速搞定，不就是仅次于P1001 $A+B$ Problem的$A*B$ problem嘛，输出的就是$A*B$。

上代码：
```cpp
#include<bits/stdc++.h>//万能头文件
using namespace std;
int a,b;//定义变量
int main()//主函数
{
	cin>>a>>b;//输入
	cout<<a*b<<endl;//输出
	return 0;//程序结束，好习惯
 } 
```
最基本的程序，初学者每行代码都要搞清楚。

如果不是初学者觉得题目过水不想做，可以做做高精度的[P1303](https://www.luogu.com.cn/problem/P1303)
