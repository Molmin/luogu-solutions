#### 这道题很简单，只需要按题意模拟即可。

#### Python AC 代码：

```python
a = input().split()                  # 输入。
floor = abs(int(a[0]) - int(a[1]))   # 计算楼层数。由于结果可能是负数，要用 abs 函数把结果转化为正数。
go = floor * int(a[-3])              # 用楼层数乘上一层楼所需的时间，得到走楼梯所需的总时间。
wait = abs(int(a[0]) - int(a[2])) * int(a[-2]) + int(a[-1]) * 3             
                                     
                                     # 用电梯所在的楼层数减她现在所在楼层数的绝对值乘电梯上一层楼所需的时间，得到等待的时间。
                                     # 再加上电梯开关门的时间（到了目的地还要开一次，一共是三次），得到等待的总时间。
take = wait + floor * int(a[-2])     # 用等待的总时间加上电梯从 x 层跑到 y 层所需的时间，就是电梯所需的总时间。
if take <= go:                       # 判断。
    print("YES")                     # 如果电梯所需的时间不大于走楼梯需要的，输出 YES （大写）。
else:                                # 否则输出 NO （大写）。
    print("NO")
```
