第一个细节：闰年变化。   

1752（含 1752）年前与 1752 年后是不同的。

- 1752 年前：只要是 4 的倍数就是闰年。

- 1752 年后：如果是4的倍数且不是100的倍数，或者是400的倍数，那么是闰年。


```cpp
bool cheak_year(int x)
{
	if(x<=1752)
	{
		return x%4==0;
	}//改用公历前
	return (x%400==0)||(x%4==0&&x%100!=0);//改用公历后
}
```


  第二、三个细节：判断日期是否违法。  
  
  判断主要有以下几点：
- 年：只能是正整数（不能小于 1）。

- 月：介于 1 和 12 之间（不能小于 1 或大于 12）。
- 日：符合那一月份的天数。
- 日期必须存在。  

```cpp
bool cheak()
{
	if(y<1)
	{
		return 0;
	}//年只能是正整数（不能小于 1）。
	if(m<1||m>12)
	{
		return 0;
	}//月介于 1 和 12 之间（不能小于 1 或大于 12）。
	month_day[2]=28+cheak_year(y);
	if(d<1||d>month_day[m])
	{
		return 0;
	}//日符合那一月份的天数。
	if(y==1752&&m==9&&d>=3&&d<=13)
	{
		return 0;
	}//日期必须存在。 
	return 1;
}
```
