# 题意
通俗的来说，这就是给了个方块，让你涂色，得涂几次。
# 思路
首先，判断一遍这个**好点**是在哪个位置，因为并没有说一个好点能用几次，所以就当是无限用，这样只要有一个点在边上就一定比在中间次数少。
### 有三种情况

#### 第一种：
当在角上有**好点**时，我们就能知道，只需要一次，所有点就都能被覆盖。
```
if(x==1&&i==1&j==1)
{
	cout<<"1";
    return 0;
}
if(x==1&&i==1&j==m)
{
	cout<<"1";
    return 0;
}
if(x==1&&i=n&&j==1)
{
	cout<<"1";
    return 0;
}
if(x==1&&i==n&j==m)
{
	cout<<"1";
    return 0;
}
```
四个点就有四种可能
#### 第二种
当在边上有**好点**时，我们就能知道，需要两次，所有点才能被覆盖，因为需要向上来一次，向下来一次。
```
if(x==1&&(i==1||j==1||i==n||j==m))
{
	cout<<"2";
	return 0;
}
```
只要在边上就有两次
#### 第三种
当在中心有**好点**时，我们就能知道，需要四次，所有点才能被覆盖，因为需要向上下左右各来一次，因为是道入门题，而且题目也没说，就不用考虑一个**好点**也没有的情况。
```
cout<<"4";
```
结束时直接输出就行
#附上代码
```
#include<iostream>
#include<cstdio>
using namespace std;
int n,m,x;
int main()
{
	scanf("%d%d",&n,&m);
	for(int i=1 ; i<=n ; ++i)
	{
		for(int j=1 ; j<=m ; ++j)
		{
			scanf("%d",&x);
			if(x==1&&i==1&&j==1)//判断是否在角上
			{
				cout<<"1";
				return 0;
			}
			else if(x==1&&i==1&&j==m)//判断是否在角上
			{
				cout<<"1";
				return 0;
			}
			else if(x==1&&i==n&&j==1)//判断是否在角上
			{
				cout<<"1";
				return 0;
			}
			else if(x==1&&i==n&&j==m)//判断是否在角上
			{
				cout<<"1";
				return 0;
			}
			else if(x==1&&(i==1||j==1||i==n||j==m))//判断是否在边上
			{
				cout<<"2";
				return 0;
			}
		}
	}
	cout<<"4";//如果都不是就直接输出
	return 0;
}
