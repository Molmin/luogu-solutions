###   我第一次发题解，为洛谷做贡献，求通过！！！
 
  这道题蛮简单的，其实看样例也能看出来，一共2种情况；  
  1.好点在边上，结果为2；  
  2.好点在中间，结果为4；                    
  就这样，看代码：

```cpp
#include<bits/stdc++.h> //万能头 
using namespace std;

int n,m,a[100][100],ans=4;  // 因为只有2种情况，先定义为'好点'在中间，则结果为4 

int main(){
	
	cin>>n>>m;
	
	for(int i=1;i<=n;i++)
	for(int w=1;w<=m;w++){      
		cin>>a[i][w];       // 其实不用数组也行 
		if( a[i][w]==1 && (i==1 || i==n || w==1 || w==m )){   // 如果'好点'在边上，则结果为2 
			ans=2;
		}
	}
	
	cout<<ans;  //输出结果 
	return 0;   //完美结束 
}
```
